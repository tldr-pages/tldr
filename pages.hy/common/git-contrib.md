# git ներդրում

> Ցուցադրել պարտավորությունները հեղինակից:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-contrib>:.

- Ցուցադրել բոլոր commit հեշերը և դրանց համապատասխան հաղորդագրությունները կոնկրետ հեղինակից.:

`git contrib {{author}}`
