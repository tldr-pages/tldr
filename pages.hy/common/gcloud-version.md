# gcloud-տարբերակ

> Տպել տարբերակի տեղեկությունները Google Cloud CLI բաղադրիչների համար:.
> Լրացուցիչ տեղեկություններ. <https://docs.cloud.google.com/sdk/gcloud/reference/version>:.

- Ցուցադրել բոլոր տեղադրված բաղադրիչների տարբերակների մասին տեղեկությունները, ինչպես նաև դրանց հասանելի թարմացումները.:

`gcloud version`

- Ցուցադրել օգնությունը.:

`gcloud version --help`
