# git հավատարմագրերի քեշ

> Git helper գաղտնաբառերը հիշողության մեջ ժամանակավորապես պահելու համար:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-credential-cache>:.

- Պահպանեք Git հավատարմագրերը որոշակի ժամանակով.:

`git config credential.helper 'cache --timeout={{time_in_seconds}}'`
