# aws-ի պահոց

> Զարգացման միջավայրերում AWS հավատարմագրերն ապահով պահելու և մուտք գործելու պահոց:.
> Լրացուցիչ տեղեկություններ. <https://github.com/99designs/aws-vault>:.

- Ավելացրեք հավատարմագրերը անվտանգ բանալիների պահեստին.:

`aws-vault add {{profile}}`

- Կատարեք հրաման AWS հավատարմագրերով շրջակա միջավայրում.:

`aws-vault exec {{profile}} -- {{aws s3 ls}}`

- Բացեք դիտարկիչի պատուհանը և մուտք գործեք AWS Console.:

`aws-vault login {{profile}}`

- Ցուցակեք պրոֆիլները՝ դրանց հավատարմագրերի և նիստերի հետ միասին.:

`aws-vault list`

- Պտտեցնել AWS հավատարմագրերը.:

`aws-vault rotate {{profile}}`

- Հեռացրեք հավատարմագրերը անվտանգ բանալիների պահեստից.:

`aws-vault remove {{profile}}`
