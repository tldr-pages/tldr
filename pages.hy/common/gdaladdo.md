# gdaladdo

> Կառուցեք ռաստերային տվյալների հավաքածուների ընդհանուր պատկերներ:.
> Լրացուցիչ տեղեկություններ. <https://gdal.org/en/stable/programs/gdaladdo.html>:.

- Կառուցեք ռաստերային տվյալների ակնարկ պատկերներ՝ օգտագործելով «միջին» [r] նմուշառման մեթոդը.:

`gdaladdo -r average {{path/to/input.tif}}`
