# pbmtoppa

> Փոխարկեք PBM պատկերը HP Printer Performance Architecture ձևաչափի:.
> Լրացուցիչ տեղեկություններ. <https://netpbm.sourceforge.net/doc/pbmtoppa.html>:.

- Փոխակերպեք PBM պատկերը PPA ֆայլի.:

`pbmtoppa {{path/to/image.pbm}} > {{path/to/output.ppa}}`

- Նշեք ցանկալի կետերը մեկ դյույմի համար և թղթի չափը.:

`pbmtoppa -d {{300}} -s {{a4}} {{path/to/image.pbm}} > {{path/to/output.ppa}}`
