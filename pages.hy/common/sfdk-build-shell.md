# sfdk build-shell

> Կատարեք հատուկ քայլեր կառուցման շարժիչում:.
> Տես նաև՝ `sfdk config`, `sfdk build-init`:.
> Լրացուցիչ տեղեկություններ. <https://github.com/sailfishos/sailfish-qtcreator/blob/master/share/qtcreator/sfdk/modules/20-building-mb2/doc/command.build-shell.adoc>:.

- Գործարկեք ինտերակտիվ կեղևը շինարարական շարժիչում.:

`sfdk build-shell`

- Գործարկեք նշված հրամանը build shell-ում.:

`sfdk build-shell {{command}}`

- Գործարկեք ինտերակտիվ կեղևը շինարարական շարժիչում սպասարկման ռեժիմում, երբ ստուգեք կամ փոփոխեք շինարարական միջավայրը.:

`sfdk build-shell --maintain`
