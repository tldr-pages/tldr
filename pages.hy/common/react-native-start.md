# react-native start

> Սկսեք React Native սերվերը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/facebook/react-native/tree/main/packages/community-cli-plugin#start>:.

- Գործարկեք սերվերը, որը շփվում է միացված սարքերի հետ.:

`react-native start`

- Սկսեք մետրոյի փաթեթը մաքուր քեշով.:

`react-native start --reset-cache`

- Գործարկեք սերվերը հատուկ նավահանգստում (կանխադրված է 8081):

`react-native start --port {{3000}}`

- Գործարկեք սերվերը մանրամասն ռեժիմով.:

`react-native start --verbose`

- Նշեք ֆայլերի փոխակերպման համար աշխատողների առավելագույն թիվը (կանխադրվածը CPU միջուկների թիվն է).:

`react-native start --max-workers {{count}}`

- Անջատել ինտերակտիվ ռեժիմը.:

`react-native start --no-interactive`
