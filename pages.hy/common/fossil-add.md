# բրածո հավելում

> Ֆայլերը կամ գրացուցակները դրեք Fossil տարբերակի հսկողության տակ:.
> Լրացուցիչ տեղեկություններ. <https://fossil-scm.org/home/help/add>:.

- Տեղադրեք ֆայլը կամ գրացուցակը տարբերակի հսկողության տակ, այնպես որ այն կլինի ընթացիկ վճարման մեջ.:

`fossil add {{path/to/file_or_directory}}`

- Հեռացրեք բոլոր ավելացված ֆայլերը ընթացիկ վճարումից.:

`fossil add --reset`
