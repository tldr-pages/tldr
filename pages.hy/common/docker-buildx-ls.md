# docker buildx ls

> Ցուցակ ստեղծող օրինակները և հարակից հանգույցները:.
> Լրացուցիչ տեղեկություններ. <https://docs.docker.com/reference/cli/docker/buildx/ls/>:.

- Ցուցակ ստեղծող օրինակներ.:

`docker buildx ls`

- Ձևաչափեք ելքը՝ օգտագործելով Go ձևանմուշը.:

`docker buildx ls --format "{{.NAME}}: {{.DriverEndpoint}}"`
