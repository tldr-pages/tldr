# git checkout-index

> Պատճենել ֆայլերը ինդեքսից աշխատանքային ծառին:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-checkout-index>:.

- Վերականգնել ցանկացած ֆայլ, որը ջնջվել է վերջին պարտավորությունից հետո.:

`git checkout-index {{[-a|--all]}}`

- Վերականգնել ցանկացած ֆայլ, որը ջնջվել կամ փոխվել է վերջին հանձնումից հետո.:

`git checkout-index {{[-a|--all]}} {{[-f|--force]}}`

- Վերականգնել ցանկացած ֆայլ, որը փոխվել է վերջին հանձնումից հետո՝ անտեսելով ջնջված բոլոր ֆայլերը.:

`git checkout-index {{[-a|--all]}} {{[-f|--force]}} {{[-n|--no-create]}}`

- Արտահանեք ամբողջ ծառի պատճենը վերջին հանձնման ժամանակ նշված գրացուցակում (հետևող շեղը կարևոր է).:

`git checkout-index {{[-a|--all]}} {{[-f|--force]}} --prefix {{path/to/export_directory}}/`
