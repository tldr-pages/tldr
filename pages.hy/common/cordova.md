#կորդովա

> Բջջային հավելվածներ HTML, CSS և JS-ով:.
> Լրացուցիչ տեղեկություններ. <https://cordova.apache.org/docs/en/latest/guide/cli/>:.

- Ստեղծեք Կորդովայի նախագիծ.:

`cordova create {{path/to/directory}} {{package}} {{project_name}}`

- Ցուցադրել աշխատանքային տարածքի ընթացիկ կարգավիճակը.:

`cordova info`

- Ավելացնել Cordova հարթակ.:

`cordova platform add {{platform}}`

- Հեռացնել Cordova հարթակը.:

`cordova platform remove {{platform}}`

- Ավելացնել Cordova plugin.:

`cordova plugin add {{pluginid}}`

- Հեռացրեք Cordova հավելվածը.:

`cordova plugin remove {{pluginid}}`
