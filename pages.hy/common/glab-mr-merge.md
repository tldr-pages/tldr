# glab mr միաձուլում

> Միավորել GitLab-ի միաձուլման հարցումները:.
> Լրացուցիչ տեղեկություններ. <https://gitlab.com/gitlab-org/cli/-/blob/main/docs/source/mr/merge.md>:.

- Ինտերակտիվ կերպով միաձուլեք ընթացիկ մասնաճյուղի հետ կապված միաձուլման հարցումը.:

`glab mr merge`

- Միավորել նշված միաձուլման հարցումը, ինտերակտիվ կերպով.:

`glab mr merge {{mr_number}}`

- Միավորել միաձուլման հարցումը՝ հեռացնելով մասնաճյուղը և՛ տեղական, և՛ հեռակառավարման վրա.:

`glab mr merge {{[-d|--remove-source-branch]}}`

- Ընթացիկ միաձուլման հարցումը միացրեք մեկ հանձնառության մեջ հաղորդագրության մարմնի հետ և միաձուլեք.:

`glab mr merge {{[-s|--squash]}} {{[-m|--message]}} "{{commit_message_body}}"`

- Ցուցադրել օգնությունը.:

`glab mr merge --help`
