# git կոնֆիգուրացիա

> Կառավարեք Git-ի պահոցների հատուկ կազմաձևման տարբերակները:.
> Այս կազմաձևերը կարող են լինել տեղական (ներկայիս պահոցի համար) կամ գլոբալ (ներկայիս օգտագործողի համար):.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-config>:.

- Համաշխարհային սահմանեք ձեր անունը կամ էլ.փոստը (այս տեղեկատվությունը պահանջվում է պահոցը հանձնելու համար և կներառվի բոլոր պարտավորությունների մեջ).:

`git config --global {{user.name|user.email}} "{{Your Name|email@example.com}}"`

- Թվարկեք տեղական, գլոբալ կամ համակարգի կազմաձևման գրառումները և ցույց տվեք դրանց ֆայլի գտնվելու վայրը.:

`git config {{[-l|--list]}} --{{local|global|system}} --show-origin`

- Սահմանեք տվյալ կազմաձևման մուտքի գլոբալ արժեքը (այս դեպքում՝ կեղծանունը).:

`git config --global {{alias.unstage}} "reset HEAD --"`

- Ստացեք տվյալ կազմաձևման մուտքի արժեքը.:

`git config {{alias.unstage}}`

- Օգտագործեք կեղծանունը.:

`git {{unstage}}`

- Վերադարձեք գլոբալ կազմաձևման գրառումը իր լռելյայն արժեքին.:

`git config --global --unset {{alias.unstage}}`

- Խմբագրեք տեղական Git-ի կոնֆիգուրացիան (`.git/config`) լռելյայն խմբագրում.:

`git config {{[-e|--edit]}}`

- Խմբագրեք Git-ի գլոբալ կոնֆիգուրացիան (`~/.gitconfig` լռելյայն կամ `$XDG_CONFIG_HOME/git/config`, եթե այդպիսի ֆայլ կա) կանխադրված խմբագրում.:

`git config --global {{[-e|--edit]}}`
