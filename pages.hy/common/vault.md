# պահոց

> Շփվեք HashiCorp Vault-ի հետ:.
> Լրացուցիչ տեղեկություններ. <https://developer.hashicorp.com/vault/docs/commands>:.

- Միացեք Vault սերվերին և սկզբնավորեք նոր կոդավորված տվյալների պահեստը.:

`vault init`

- Ապակնքեք (բացեք) պահոցը՝ տրամադրելով գաղտնագրված տվյալների պահեստ մուտք գործելու համար անհրաժեշտ հիմնական բաժնետոմսերից մեկը.:

`vault unseal {{key-share-x}}`

- Նույնականացրեք CLI հաճախորդը Vault սերվերի դեմ՝ օգտագործելով վավերացման նշան.:

`vault auth {{authentication_token}}`

- Պահպանեք նոր գաղտնիք պահոցում՝ օգտագործելով «գաղտնի» կոչվող ընդհանուր հետին վերջը.:

`vault write secret/{{hello}} value={{world}}`

- Կարդացեք արժեք պահոցից՝ օգտագործելով «գաղտնի» կոչվող ընդհանուր հետին վերջը.:

`vault read secret/{{hello}}`

- Կարդացեք որոշակի դաշտ արժեքից.:

`vault read -field={{field_name}} secret/{{hello}}`

- Կնքեք (կողպեք) Vault սերվերը՝ հեռացնելով տվյալների պահեստի կոդավորման բանալին հիշողությունից.:

`vault seal`
