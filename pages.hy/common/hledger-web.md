# hledger-web

> Վեբ ինտերֆեյս և API `hledger`-ի համար՝ պարզ տեքստային հաշվապահական հաշվառման ամուր, հարմարավետ հավելված:.
> Լրացուցիչ տեղեկություններ. <https://hledger.org/hledger-web.html>:.

- Գործարկեք վեբ հավելվածը և հնարավորության դեպքում զննարկիչը՝ միայն տեղական դիտման և ավելացնելու համար.:

`hledger-web`

- Ինչպես վերևում, բայց նշված ֆայլով և թույլ տվեք խմբագրել առկա տվյալները.:

`hledger-web {{[-f|--file]}} {{path/to/file.journal}} --allow edit`

- Սկսեք պարզապես վեբ հավելվածը և ընդունեք մուտքային կապերը նշված հոսթին և նավահանգիստին.:

`hledger-web --serve --host {{my.host.name}} --port 8000`

- Սկսեք պարզապես վեբ հավելվածի JSON API-ն և թույլատրեք միայն կարդալու մուտքը.:

`hledger-web --serve-api --host {{my.host.name}} --allow view`

- Ցույց տվեք ձեր բազային արժույթով ընթացիկ շուկայական արժեքին փոխարկված գումարները, երբ հայտնի է.:

`hledger-web --value now --infer-market-prices`

- Ցույց տվեք ձեռնարկը Info ձևաչափով, եթե հնարավոր է.:

`hledger-web --info`

- Ցուցադրել օգնությունը.:

`hledger-web {{[-h|--help]}}`
