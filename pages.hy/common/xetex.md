# xetex

> Կազմեք PDF փաստաթուղթ XeTeX աղբյուրի ֆայլերից:.
> Լրացուցիչ տեղեկություններ. <https://www.tug.org/xetex/>:.

- Կազմեք PDF փաստաթուղթ.:

`xetex {{source.tex}}`

- Կազմեք PDF փաստաթուղթ՝ նշելով ելքային գրացուցակը.:

`xetex -output-directory={{path/to/directory}} {{source.tex}}`

- Կազմեք PDF փաստաթուղթ, դուրս գալով, եթե սխալներ լինեն.:

`xetex -halt-on-error {{source.tex}}`
