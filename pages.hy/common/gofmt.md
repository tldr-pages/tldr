# gofmt

> Ձևաչափել Go սկզբնական կոդը:.
> Լրացուցիչ տեղեկություններ. <https://pkg.go.dev/cmd/gofmt>:.

- Ձևաչափեք ֆայլը և արդյունքը ցուցադրեք վահանակում.:

`gofmt {{source.go}}`

- Ֆայլի ֆորմատավորում՝ տեղում վերագրելով բնօրինակ ֆայլը.:

`gofmt -w {{source.go}}`

- Ձևաչափեք ֆայլը, այնուհետև պարզեցրեք կոդը՝ վերագրանցելով բնօրինակ ֆայլը.:

`gofmt -s -w {{source.go}}`

- Տպել բոլոր (ներառյալ կեղծ) սխալները.:

`gofmt -e {{source.go}}`
