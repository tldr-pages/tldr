# snyk

> Գտեք ձեր կոդի խոցելիությունը և վերացրեք ռիսկերը:.
> Լրացուցիչ տեղեկություններ. <https://docs.snyk.io/developer-tools/snyk-cli/commands>:.

- Մուտք գործեք ձեր Snyk հաշիվ.:

`snyk auth`

- Փորձարկեք ձեր կոդը ցանկացած հայտնի խոցելիության համար.:

`snyk test`

- Փորձարկեք տեղական Docker պատկերը ցանկացած հայտնի խոցելիության համար.:

`snyk test --docker {{docker_image}}`

- Գրանցեք կախվածության վիճակը և ցանկացած խոցելիություն snyk.io-ում.:

`snyk monitor`

- Ավտոմատ կարկատել և անտեսել խոցելիությունները.:

`snyk wizard`
