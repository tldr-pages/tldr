# n

> Կառավարեք բազմաթիվ հանգույցների տարբերակները:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/n>:.

- Տեղադրեք հանգույցի տվյալ տարբերակը: Եթե տարբերակն արդեն տեղադրված է, այն կակտիվանա.:

`n {{version}}`

- Ցուցադրել տեղադրված տարբերակները և ինտերակտիվորեն ակտիվացնել դրանցից մեկը.:

`n`

- Հեռացնել տարբերակը.:

`n rm {{version}}`

- Գործարկեք ֆայլ տվյալ տարբերակով.:

`n use {{version}} {{file.js}}`

- Արդյունք երկուական ուղի տարբերակի համար.:

`n bin {{version}}`
