# npm ci

> Մաքրել `npm` նախագծի կախվածությունների տեղադրումը ավտոմատացված միջավայրերի համար:.
> Տեղադրում է փաթեթներ՝ հիմնված `package-lock.json` կամ `npm-shrinkwrap.json`-ի վրա:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-ci/>:.

- Տեղադրեք նախագծի կախվածությունները `package-lock.json` կամ `npm-shrinkwrap.json`-ից:

`npm ci`

- Տեղադրեք նախագծի կախվածությունը, բայց բաց թողեք կախվածության նշված տեսակը.:

`npm ci --omit {{dev|optional|peer}}`

- Տեղադրեք նախագծի կախվածությունները՝ առանց `package.json`-ում սահմանված որևէ նախնական/հետսկրիպտ գործարկելու:

`npm ci --ignore-scripts`
