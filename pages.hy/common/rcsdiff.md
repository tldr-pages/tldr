# rcsdiff

> Համեմատեք RCS-ի վերանայումները (ցույց տվեք տարբերությունները RCS ֆայլերում վերանայումների միջև):.
> Տես նաև՝ `ci`, `co`, `rcs`, `rlog`:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/rcsdiff>:.

- Համեմատեք աշխատանքային ֆայլը վերջին վերանայման հետ.:

`rcsdiff {{path/to/file}}`

- Համեմատեք աշխատանքային ֆայլը կոնկրետ վերանայման հետ.:

`rcsdiff -r{{revision}} {{path/to/file}}`

- Համեմատեք ֆայլի երկու հատուկ վերանայումներ.:

`rcsdiff -r{{revision1}} -r{{revision2}} {{path/to/file}}`

- Ցույց տալ տարբերությունները միասնական ձևաչափով.:

`rcsdiff -u {{path/to/file}}`

- Ցույց տալ տարբերությունները համատեքստի գծերի հետ.:

`rcsdiff -c {{path/to/file}}`
