# իոտոպ

> Ցուցադրել ընթացիկ I/O օգտագործման աղյուսակն ըստ գործընթացների կամ թելերի:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/iotop>:.

- Սկսեք վերևի նման I/O մոնիտորը.:

`sudo iotop`

- Ցույց տալ միայն գործընթացները կամ շղթաները, որոնք իրականում կատարում են I/O:

`sudo iotop {{[-o|--only]}}`

- Ցուցադրել I/O օգտագործումը ոչ ինտերակտիվ ռեժիմում.:

`sudo iotop {{[-b|--batch]}}`

- Ցուցադրել միայն I/O գործընթացների օգտագործումը (կանխադրված է ցույց տալ բոլոր շղթաները).:

`sudo iotop {{[-P|--processes]}}`

- Ցույց տալ տվյալ PID(ներ)ի I/O օգտագործումը.:

`sudo iotop {{[-p|--pid]}} {{pid}}`

- Ցույց տալ տվյալ օգտագործողի I/O օգտագործումը.:

`sudo iotop {{[-u|--user]}} {{user}}`

- Ցույց տալ կուտակված I/O թողունակության փոխարեն.:

`sudo iotop {{[-a|--accumulated]}}`
