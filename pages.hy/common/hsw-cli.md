# hsw-cli

> REST գործիքը Handshake դրամապանակի համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/handshake-org/hs-client>:.

- Բացեք ընթացիկ դրամապանակը (վայրկյանների ժամանակի ավարտը).:

`hsw-cli unlock {{passphrase}} {{timeout}}`

- Կողպեք ընթացիկ դրամապանակը.:

`hsw-cli lock`

- Դիտեք ընթացիկ դրամապանակի մանրամասները.:

`hsw-cli get`

- Դիտեք ընթացիկ դրամապանակի մնացորդը.:

`hsw-cli balance`

- Դիտեք ընթացիկ դրամապանակի գործարքների պատմությունը.:

`hsw-cli history`

- Նշված մետաղադրամի գումարով գործարք ուղարկեք հետևյալ հասցեով.:

`hsw-cli send {{address}} {{1.05}}`

- Դիտեք ընթացիկ դրամապանակի սպասվող գործարքները.:

`hsw-cli pending`

- Դիտեք գործարքի մանրամասները.:

`hsw-cli tx {{transaction_hash}}`
