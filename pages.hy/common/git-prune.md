# git prune

> Git հրամանը օբյեկտների տվյալների բազայից բոլոր անհասանելի օբյեկտները կտրելու համար:.
> Այս հրամանը հաճախ ուղղակիորեն չի օգտագործվում, այլ որպես ներքին հրաման, որն օգտագործվում է Git gc-ի կողմից:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-prune>:.

- Հաղորդեք, թե ինչ կհեռացվի Git Prun-ի կողմից՝ առանց այն հեռացնելու.:

`git prune {{[-n|--dry-run]}}`

- Կտրեք անհասանելի առարկաները և ցուցադրեք այն, ինչ կտրված է `stdout`-ով.:

`git prune {{[-v|--verbose]}}`

- Կտրեք անհասանելի առարկաները՝ միաժամանակ ցույց տալով առաջընթաց.:

`git prune --progress`
