# kaggle կազմաձևում

> Կառավարեք Kaggle-ի կազմաձևումը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/Kaggle/kaggle-api/blob/main/docs/README.md#config>:.

- Դիտեք ընթացիկ կազմաձևման արժեքները.:

`kaggle config view`

- Սահմանեք կազմաձևման արժեքը.:

`kaggle config set {{[-n|--name]}} {{configuration_parameter}} {{[-v|--value]}} {{parameter_value}}`

- Մաքրել կազմաձևման արժեքը.:

`kaggle config unset {{[-n|--name]}} {{configuration_parameter}}`
