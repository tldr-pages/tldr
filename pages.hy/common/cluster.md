# կլաստեր

> Գտեք կլաստերներ գրաֆիկում և ավելացրեք գրաֆիկը այս տեղեկատվությամբ:.
> Լրացուցիչ տեղեկություններ. <https://graphviz.org/pdf/cluster.1.pdf>:.

- Ստեղծեք կլաստերներ, որոնք օպտիմալացնում են մոդուլյարությունը և տպում արդյունքը `stdout`-ում:

`cluster {{input.gv}}`

- Նշեք [C] փայլերի թիրախային թիվը (մոտավոր)՝ առաջացնելու համար (0-ն ըստ լռելյայն, նշանակում է մի թիվ, որը մոտավորապես օպտիմալացնում է մոդուլյարությունը).:

`cluster -C {{5}} {{input.gv}}`

- Օգտագործեք մեկ այլ [c] փայլատման մեթոդ (0. մոդուլյարության կլաստերավորում, 1: մոդուլյարության որակ).:

`cluster -c {{0|1}} {{input.gv}}`

- Պահպանեք [o] ելքը ֆայլում՝:

`cluster -o {{output.gv}} {{input.gv}}`

- Միացնել [v]erbose ռեժիմը.:

`cluster -v {{input.gv}}`
