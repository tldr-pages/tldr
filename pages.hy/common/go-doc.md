#գնա դոկ

> Դիտեք փաթեթի կամ խորհրդանիշի փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://pkg.go.dev/cmd/go#hdr-Show_documentation_for_package_or_symbol>:.

- Դիտեք ընթացիկ փաթեթի փաստաթղթերը.:

`go doc`

- Ցույց տալ փաթեթի փաստաթղթերը և արտահանվող խորհրդանիշները.:

`go doc {{encoding/json}}`

- Ցույց տալ նաև խորհրդանիշների փաստաթղթերը.:

`go doc -all {{encoding/json}}`

- Ցույց տալ նաև աղբյուրները.:

`go doc -all -src {{encoding/json}}`

- Ցույց տալ որոշակի խորհրդանիշ.:

`go doc -all -src {{encoding/json.Number}}`
