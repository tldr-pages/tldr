# pdf տառատեսակներ

> Դյուրակիր փաստաթղթի ձևաչափի (PDF) ֆայլի տառատեսակների տեղեկատվության դիտիչ:.
> Լրացուցիչ տեղեկություններ. <https://www.xpdfreader.com/pdffonts-man.html>:.

- Տպել PDF ֆայլի տառատեսակների տեղեկատվությունը.:

`pdffonts {{path/to/file.pdf}}`

- Նշեք օգտագործողի գաղտնաբառը PDF ֆայլի համար՝ անվտանգության սահմանափակումները շրջանցելու համար.:

`pdffonts -upw {{password}} {{path/to/file.pdf}}`

- Նշեք սեփականատիրոջ գաղտնաբառը PDF ֆայլի համար՝ անվտանգության սահմանափակումները շրջանցելու համար.:

`pdffonts -opw {{password}} {{path/to/file.pdf}}`

- Տպեք լրացուցիչ տեղեկատվություն տառատեսակի գտնվելու վայրի վերաբերյալ, որը կօգտագործվի PDF ֆայլի ռաստերիզացման ժամանակ.:

`pdffonts -loc {{path/to/file.pdf}}`

- Տպեք լրացուցիչ տեղեկատվություն տառատեսակի գտնվելու վայրի մասին, որը կօգտագործվի, երբ PDF ֆայլը փոխարկվի PostScript-ի.:

`pdffonts -locPS {{path/to/file.pdf}}`
