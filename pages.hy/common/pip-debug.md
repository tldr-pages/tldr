# pip վրիպազերծում

> Ցուցադրել վրիպազերծման մասին տեղեկատվությունը ընթացիկ pip միջավայրի և կազմաձևման մասին:.
> Նախատեսված է վրիպազերծման համար; ընտրանքները և արդյունքը կարող են փոխվել առանց ծանուցման:.
> Լրացուցիչ տեղեկություններ. <https://pip.pypa.io/en/stable/cli/pip_debug/>:.

- Ցուցադրել վրիպազերծման ընդհանուր տեղեկատվությունը.:

`pip debug`

- Ցույց տալ վրիպազերծման տեղեկատվությունը որոշակի հարթակի համար.:

`pip debug --platform {{platform}}`

- Ցուցադրել վրիպազերծման տեղեկատվությունը Python-ի հատուկ տարբերակի համար.:

`pip debug --python-version {{version}}`

- Ցույց տալ վրիպազերծման տեղեկատվությունը հատուկ Python-ի իրականացման համար.:

`pip debug --implementation {{implementation}}`

- Ցույց տալ վրիպազերծման տեղեկատվությունը որոշակի ABI-ի համար.:

`pip debug --abi {{abi}}`
