# թրթիռ կրակ

> Կարգավորեք Firebase նախագիծը ձեր Flutter հավելվածի համար:.
> Նշում. տեղադրելու համար պահանջվում է Firebase CLI և FlutterFire CLI:.
> Լրացուցիչ տեղեկություններ. <https://firebase.google.com/docs/flutter/setup>:.

- Նախաձեռնեք և կարգավորեք Firebase-ը ձեր ընթացիկ Flutter հավելվածի համար.:

`flutterfire configure`

- Նշեք հարթակներ (iOS, Android, վեբ) աջակցելու համար.:

`flutterfire configure --platforms {{ios,android,...}}`

- Օգտագործեք Firebase հատուկ նախագիծ.:

`flutterfire configure --project {{project_id}}`
