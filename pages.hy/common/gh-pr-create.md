# gh pr ստեղծել

> Կառավարեք GitHub-ի ձգման հարցումները:.
> Լրացուցիչ տեղեկություններ. <https://cli.github.com/manual/gh_pr_create>:.

- Ինտերակտիվ կերպով ստեղծեք ձգման հարցում.:

`gh pr {{[new|create]}}`

- Ստեղծեք ձգման հարցում՝ որոշելով վերնագիրը և նկարագրությունը ընթացիկ մասնաճյուղի հաղորդագրություններից.:

`gh pr {{[new|create]}} {{[-f|--fill]}}`

- Ստեղծեք ձգման հայտի նախագիծ.:

`gh pr {{[new|create]}} {{[-d|--draft]}}`

- Ստեղծեք ձգման հարցում՝ նշելով բազային ճյուղը, վերնագիրը և նկարագրությունը.:

`gh pr {{[new|create]}} {{[-B|--base]}} {{base_branch}} {{[-t|--title]}} "{{title}}" {{[-b|--body]}} "{{body}}"`

- Սկսեք բացել ձգման հարցումը լռելյայն վեբ դիտարկիչում.:

`gh pr {{[new|create]}} {{[-w|--web]}}`
