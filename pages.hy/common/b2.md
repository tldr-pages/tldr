# բ2

> Հեշտությամբ մուտք գործեք Backblaze B2 Cloud Storage-ի բոլոր հնարավորությունները:.
> Լրացուցիչ տեղեկություններ. <https://www.backblaze.com/docs/cloud-storage-command-line-tools>:.

- Մուտք գործեք ձեր հաշիվ.:

`b2 authorize_account {{key_id}}`

- Թվարկեք ձեր հաշվում առկա դույլերը.:

`b2 list_buckets`

- Ստեղծեք դույլ, նշեք դույլի անունը և մուտքի տեսակը (օրինակ՝ allPublic կամ allPrivate):

`b2 create_bucket {{bucket_name}} {{allPublic|allPrivate}}`

- Վերբեռնեք ֆայլ: Ընտրեք ֆայլ, դույլ և թղթապանակ.:

`b2 upload_file {{bucket_name}} {{path/to/file}} {{folder_name}}`

- Վերբեռնեք աղբյուրի գրացուցակը Backblaze B2 դույլի նպատակակետ.:

`b2 sync {{path/to/source_file}} {{bucket_name}}`

- Պատճենել ֆայլը մի դույլից մյուս դույլ.:

`b2 copy-file-by-id {{path/to/source_file_id}} {{destination_bucket_name}} {{path/to/b2_file}}`

- Ցույց տվեք ֆայլերը ձեր դույլով.:

`b2 ls {{bucket_name}}`

- Հեռացրեք «թղթապանակը» կամ օրինաչափությանը համապատասխանող ֆայլերի մի շարք.:

`b2 rm {{path/to/folder|pattern}}`
