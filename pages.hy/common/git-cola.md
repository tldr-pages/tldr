# git cola

> Հզոր Git GUI՝ նուրբ և ինտուիտիվ ինտերֆեյսով:.
> Լրացուցիչ տեղեկություններ. <https://git-cola.readthedocs.io/en/latest/git-cola.html>:.

- Սկսեք GUI-ն.:

`git cola`

- Գործարկեք GUI-ն փոփոխման ռեժիմով.:

`git cola --amend`

- Հուշել Git պահոց: Լռելյայն ընթացիկ գրացուցակում.:

`git cola --prompt`

- Բացեք Git պահոցը նշված ճանապարհով.:

`git cola {{[-r|--repo]}} {{path/to/git-repository}}`

- Կիրառեք ուղու զտիչը կարգավիճակի վիդջեթին.:

`git cola {{[-s|--status-filter]}} {{filter}}`
