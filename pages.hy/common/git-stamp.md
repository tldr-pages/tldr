# git կնիք

> Դրոշմեք վերջին հանձնման հաղորդագրությունը՝ ձեր վրիպակների հետագծից խնդիրների համարներին հղում կատարելու կամ դրա վերանայման էջին հղում կատարելու հնարավորությամբ:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/git-stamp>:.

- Կնքեք վերջին հանձնման հաղորդագրությունը, որը վկայակոչում է դրա թողարկման համարը ձեր bug tracker-ից.:

`git stamp {{issue_number}}`

- Կնքեք վերջին հանձնման հաղորդագրությունը, որը կապում է այն իր վերանայման էջին.:

`git stamp {{Review https://example.org/path/to/review}}`

- Կնքեք վերջին հանձնման հաղորդագրությունը, որը փոխարինում է նախորդ թողարկումները նորով.:

`git stamp {{[-r|--replace]}} {{issue_number}}`
