# duc

> Սկավառակի օգտագործման ինդեքսավորման, ստուգման և պատկերացման գործիքների հավաքածու:.
> Duc-ը պահպանում է ֆայլային համակարգի դիրեկտորիաների կուտակված չափերի տվյալների բազա՝ թույլ տալով հարցումներ կատարել այս տվյալների բազայում կամ ստեղծելով շքեղ գրաֆիկներ՝ ցույց տալու, թե որտեղ են գտնվում տվյալները:.
> Լրացուցիչ տեղեկություններ. <https://htmlpreview.github.io/?https://github.com/zevv/duc/blob/master/doc/duc.1.html>:.

- Ինդեքսավորեք `/usr` գրացուցակը, գրելով տվյալների բազայի կանխադրված գտնվելու վայրը `~/.duc.db`:

`duc index {{/usr}}`

- Թվարկեք բոլոր ֆայլերը և գրացուցակները `/usr/local`-ի տակ՝ ցույց տալով ֆայլերի հարաբերական չափերը գրաֆիկում.:

`duc ls {{[-Fg|--classify --graph]}} {{/usr/local}}`

- Թվարկեք բոլոր ֆայլերը և գրացուցակները `/usr/local`-ի ներքո՝ օգտագործելով ծառի դիտումը ռեկուրսիվ կերպով.:

`duc ls {{[-Fg|--classify --graph]}} {{[-R|--recursive]}} {{/usr/local}}`

- Գործարկեք գրաֆիկական ինտերֆեյսը ֆայլային համակարգը ուսումնասիրելու համար՝ օգտագործելով արևայրուք գծապատկերներ.:

`duc gui {{/usr}}`

- Գործարկեք ncurses կոնսոլի ինտերֆեյսը ֆայլային համակարգը ուսումնասիրելու համար.:

`duc ui {{/usr}}`

- Թափել տվյալների բազայի տվյալները.:

`duc info`
