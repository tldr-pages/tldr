# pulumi նոր

> Ստեղծեք նոր Pulumi նախագիծ:.
> Լրացուցիչ տեղեկություններ. <https://www.pulumi.com/docs/iac/cli/commands/pulumi_new/>:.

- Ընտրեք ձևանմուշ ինտերակտիվ կերպով.:

`pulumi new`

- Ստեղծեք նախագիծ կոնկրետ ձևանմուշից (օրինակ՝ `azure-python`):

`pulumi new {{provided-template}}`

- Ստեղծեք նախագիծ տեղական ֆայլից.:

`pulumi new {{path/to/templates/aws-typescript}}`

- Ստեղծեք նախագիծ Git պահոցից.:

`pulumi new {{url}}`

- Օգտագործեք նշված գաղտնիքների մատակարարը <pulumi.com> հետին պլանով.:

`pulumi new --secrets-provider {{passphrase}}`
