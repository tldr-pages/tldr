# sfdk qmake

> Կատարել qmake build քայլը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/sailfishos/sailfish-qtcreator/blob/master/share/qtcreator/sfdk/modules/20-building-mb2/doc/command.qmake.adoc>:.

- Գործարկել qmake build:

`sfdk qmake`

- Գործարկեք shadow qmake build-ը նշված նախագծում.:

`sfdk qmake {{project}}`

- Գործարկել build-ը լրացուցիչ qmake փաստարկներով.:

`sfdk qmake -- {{arguments}}`

- Գործարկեք shadow qmake build-ը լրացուցիչ qmake փաստարկներով.:

`sfdk qmake {{project}} {{arguments}}`
