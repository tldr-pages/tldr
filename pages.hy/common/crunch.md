# ճռճռոց

> Բառացանկ գեներատոր:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/crunch>:.

- Արտադրեք 1-ից 3 երկարության բառերի ցանկը միայն փոքրատառ նիշերով.:

`crunch {{1}} {{3}}`

- Արտադրեք 8 երկարությամբ տասնվեցական բառերի ցանկը.:

`crunch {{8}} {{8}} {{0123456789abcdef}}`

- Արտադրեք abc-ի բոլոր փոխարկումների ցանկը (երկարությունները չեն մշակվում).:

`crunch {{1}} {{1}} -p {{abc}}`

- Արտադրեք տվյալ տողերի բոլոր փոխարկումների ցանկը (երկարությունները չեն մշակվում).:

`crunch {{1}} {{1}} -p {{abc}} {{def}} {{ghi}}`

- Արտադրեք բառերի ցանկը, որոնք ստեղծվել են ըստ տրված օրինաչափության և առավելագույն թվով կրկնօրինակ տառեր.:

`crunch {{5}} {{5}} {{abcde123}} -t {{@@@12}} -d 2@`

- Տրված չափի ֆայլերում բառերի ցանկ գրեք՝ սկսած տրված տողից.:

`crunch {{3}} {{5}} -o {{START}} -b {{10kb}} -s {{abc}}`

- Գրի՛ր տրված տողով դադարող և բառացանկը շրջելով բառերի ցանկ.:

`crunch {{1}} {{5}} -o {{START}} -e {{abcde}} -i`

- Գրեք բառերի ցանկը սեղմված կտոր ֆայլերում բառերի որոշակի քանակով.:

`crunch {{1}} {{5}} -o {{START}} -c {{1000}} -z {{gzip|bzip2|lzma|7z}}`
