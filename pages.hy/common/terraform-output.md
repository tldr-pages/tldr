# տերրաֆորմային ելք

> Արտահանեք կառուցվածքային տվյալներ ձեր Terraform ռեսուրսների մասին:.
> Լրացուցիչ տեղեկություններ. <https://developer.hashicorp.com/terraform/cli/commands/output>:.

- Ցուցադրել արմատային մոդուլի բոլոր ելքերը.:

`terraform output`

- Արտադրեք միայն որոշակի անունով արժեք.:

`terraform output {{name}}`

- Վերափոխեք ելքային արժեքը չմշակված տողի (օգտակար կեղևի սկրիպտների համար).:

`terraform output -raw`

- Ձևաչափեք ելքերը որպես JSON օբյեկտ՝ յուրաքանչյուր ելքի համար ստեղնով (օգտակար jq-ի դեպքում).:

`terraform output -json`
