# gitmoji

> Ինտերակտիվ կերպով տեղադրեք էմոջիներ պարտավորությունների վրա:.
> Լրացուցիչ տեղեկություններ. <https://github.com/carloscuesta/gitmoji-cli>:.

- Սկսեք կատարման հրաշագործը.:

`gitmoji --commit`

- Նախաձեռնեք Git կեռիկը (այսպես՝ `gitmoji`-ը կգործարկվի ամեն անգամ, երբ գործարկվի `git commit`):

`gitmoji --init`

- Հեռացրեք Git կեռիկը.:

`gitmoji --remove`

- Թվարկեք բոլոր հասանելի էմոջիները և դրանց նկարագրությունները.:

`gitmoji --list`

- Որոնեք էմոջիների ցուցակը հիմնաբառերի ցանկի համար.:

`gitmoji --search {{keyword1}} {{keyword2}}`

- Թարմացրեք էմոջիների քեշավորված ցուցակը հիմնական պահոցից.:

`gitmoji --update`

- Կազմաձևեք գլոբալ նախապատվությունները.:

`gitmoji --config`
