# git brv

> Տպել մասնաճյուղերի ցանկը՝ տեսակավորված ըստ վերջին կատարման ամսաթվի:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-brv>:.

- Թվարկեք յուրաքանչյուր մասնաճյուղ, որը ցույց է տալիս ամսաթիվը, վերջին կատարման հեշը և հաղորդագրությունը.:

`git brv`
