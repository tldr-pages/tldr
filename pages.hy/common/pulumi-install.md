# pulumi տեղադրում

> Տեղադրեք փաթեթներ և պլագիններ ընթացիկ ծրագրի կամ քաղաքականության փաթեթի համար:.
> Լրացուցիչ տեղեկություններ. <https://www.pulumi.com/docs/iac/cli/commands/pulumi_install/>:.

- Տեղադրեք փաթեթներ և պլագիններ.:

`pulumi install`

- Տեղադրեք փաթեթներ և պլագիններ, բայց բաց թողեք կախվածությունները.:

`pulumi install --no-dependencies`

- Տեղադրեք փաթեթներ և նորից տեղադրեք առկա պլագինները.:

`pulumi install --reinstall`
