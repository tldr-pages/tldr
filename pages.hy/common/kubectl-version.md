# kubectl տարբերակ

> Տպել հաճախորդի և սերվերի տարբերակի տեղեկատվությունը ընթացիկ համատեքստի համար:.
> Լրացուցիչ տեղեկություններ. <https://kubernetes.io/docs/reference/kubectl/generated/kubectl_version/>:.

- Տպեք հաճախորդի և սերվերի տարբերակի տեղեկատվությունը ընթացիկ համատեքստի համար.:

`kubectl version`

- Տպել հաճախորդի տարբերակը միայն ընթացիկ համատեքստի համար.:

`kubectl version --client`

- Արտադրեք հաճախորդի և սերվերի տարբերակի տեղեկատվությունը JSON-ում ընթացիկ համատեքստի համար.:

`kubectl version {{[-o|--output]}} json`

- Արտադրեք հաճախորդի և սերվերի տարբերակի տեղեկատվությունը YAML-ի ընթացիկ համատեքստի համար.:

`kubectl version {{[-o|--output]}} yaml`
