# գաղափար

> JetBrains Java և Kotlin IDE:.
> Լրացուցիչ տեղեկություններ. <https://www.jetbrains.com/help/idea/working-with-the-ide-features-from-command-line.html>:.

- Բացեք ընթացիկ գրացուցակը IntelliJ IDEA-ում.:

`idea {{path/to/directory}}`

- Բացեք կոնկրետ ֆայլ կամ գրացուցակ IntelliJ IDEA-ում.:

`idea {{path/to/file_or_directory}}`

- Բացեք տարբերությունների դիտիչը՝ մինչև 3 ֆայլ համեմատելու համար.:

`idea diff {{path/to/file1 path/to/file2 path/to/optional_file3}}`

- Բացեք միաձուլման երկխոսությունը՝ ֆայլերի երկկողմանի միաձուլում կատարելու համար.:

`idea merge {{path/to/file1}} {{path/to/file2}} {{path/to/output}}`

- Գործարկել կոդի ստուգումները նախագծի վրա.:

`idea inspect {{path/to/project_directory}} {{path/to/inspection_profile}} {{path/to/output}}`
