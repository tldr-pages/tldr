# tcc

> Փոքրիկ C կոմպիլյատոր, որը կարող է գործարկել C աղբյուրի ֆայլերը որպես սկրիպտներ և այլ դեպքում ունի հրամանի տողի ընտրանքներ, որոնք նման են `gcc`-ին:.
> Լրացուցիչ տեղեկություններ. <https://bellard.org/tcc/tcc-doc.html>:.

- Կազմել և կապել 2 սկզբնաղբյուր ֆայլ՝ գործարկվող ստեղծելու համար.:

`tcc -o {{executable_name}} {{path/to/file1.c}} {{path/to/file2.c}}`

- Ուղղակիորեն գործարկեք մուտքագրված ֆայլը, ինչպես սցենարը և փոխանցեք փաստարկներ դրան.:

`tcc -run {{path/to/source_file.c}} {{arguments}}`

- Մեկնաբանեք C աղբյուրի ֆայլերը ֆայլի ներսում shebang-ով.:

`#!/{{path/to/tcc}} -run`
