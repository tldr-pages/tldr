# jj կազմաձևը չեղարկված է

> Չեղարկել կազմաձևման տարբերակը:.
> Տես նաև՝ `jj config set`:.
> Լրացուցիչ տեղեկություններ. <https://docs.jj-vcs.dev/latest/cli-reference/#jj-config-unset>:.

- Անջատեք կազմաձևման տարբերակը օգտագործողի մակարդակի կազմաձևում.:

`jj config {{[u|unset]}} --user {{name}}`

- Անջատեք կազմաձևման տարբերակը ռեպո մակարդակի կազմաձևում.:

`jj config {{[u|unset]}} --repo {{name}}`

- Աշխատանքային տարածքի մակարդակի կազմաձևում անջատեք կազմաձևման տարբերակը.:

`jj config {{[u|unset]}} --workspace {{name}}`
