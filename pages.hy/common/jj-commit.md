# jj պարտավորվել

> Թարմացրեք նկարագրությունը և ստեղծեք նոր փոփոխություն վերևում:.
> Լրացուցիչ տեղեկություններ. <https://docs.jj-vcs.dev/latest/cli-reference/#jj-commit>:.

- Բացեք խմբագիրը՝ commit հաղորդագրությունը գրելու և այնուհետև վերևում ստեղծել նոր դատարկ հանձնում.:

`jj commit`

- Պարտավորվեք տրված հաղորդագրությամբ.:

`jj commit {{[-m|--message]}} "{{message}}"`

- Ինտերակտիվ կերպով ընտրեք, թե որ փոփոխությունները ներառվեն.:

`jj commit {{[-i|--interactive]}}`
