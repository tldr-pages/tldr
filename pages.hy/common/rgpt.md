# rgpt

> Կոդերի վերանայման ավտոմատացված գործիք, որն օգտագործում է GPT, որը կարող եք օգտագործել անմիջապես ձեր տերմինալից:.
> Լրացուցիչ տեղեկություններ. <https://github.com/vaibhav-venkat/review-gpt>:.

- Խնդրեք GPT-ին բարելավել կոդը առանց լրացուցիչ ընտրանքների.:

`rgpt --i "$(git diff {{path/to/file}})"`

- Ստացեք ավելի մանրամասն ծավալուն ելք `rgpt`-ից ծածկագիրը վերանայելիս.:

`rgpt --v --i "$(git diff {{path/to/file}})"`

- Խնդրեք GPT-ին բարելավել կոդը և սահմանափակել այն որոշակի քանակությամբ GPT3 նշաններով.:

`rgpt --max {{300}} --i "$(git diff {{path/to/file}})"`

- Հարցրեք GPT-ին ավելի եզակի արդյունքի համար՝ օգտագործելով 0-ից 2-ի միջև լողացող արժեքը: (ավելի բարձր = ավելի եզակի):

`rgpt --pres {{1.2}} --i "$(git diff {{path/to/file}})"`

- Խնդրեք GPT-ին վերանայել ձեր կոդը՝ օգտագործելով հատուկ մոդել.:

`rgpt --model {{davinci}} --i "$(git diff {{path/to/file}})"`

- Դարձրեք `rgpt`-ն օգտագործել JSON ելք.:

`rgpt --json --i "$(git diff {{path/to/file}})"`
