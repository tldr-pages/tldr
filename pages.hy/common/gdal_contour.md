# gdal_contour

> Ստեղծեք ուրվագծային գծեր և բազմանկյուններ թվային բարձրության մոդելից:.
> Լրացուցիչ տեղեկություններ. <https://gdal.org/en/stable/programs/gdal_contour.html>:.

- Ստեղծեք վեկտորային տվյալների հավաքածու ուրվագծային գծերով, որոնք տարածված են 100 մետր [i] ինտերվալի վրա՝ միաժամանակ [a] վերագրելով բարձրության հատկությունը որպես «ele»:

`gdal_contour -a {{ele}} -i {{100.0}} {{path/to/input.tif}} {{path/to/output.gpkg}}`

- Ստեղծեք վեկտորային տվյալների հավաքածու [p]պոլիգոններով, որոնք տարածված են 100 մետր [i] ինտերվալի վրա.:

`gdal_contour -i {{100.0}} -p {{path/to/input.tif}} {{path/to/output.gpkg}}`
