# axel

> Ներբեռնեք արագացուցիչ:.
> Աջակցում է HTTP, HTTPS, FTP և FTP-ներին:.
> Տես նաև՝ `aria2c`:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/axel>:.

- Ներբեռնեք URL ֆայլի մեջ.:

`axel {{url}}`

- Ներբեռնեք և նշեք ելքային ֆայլ.:

`axel {{url}} {{[-o|--output]}} {{path/to/file}}`

- Ներբեռնեք որոշակի թվային կապերով.:

`axel {{[-n|--num-connections]}} {{number}} {{url}}`

- Օգտագործեք որոշակի թվով հայելիներ որոնման և ներբեռնման համար.:

`axel {{[-S|--search=]}}{{number}} {{url}}`

- Սահմանափակել ներբեռնման արագությունը (բայթ վայրկյանում).:

`axel {{[-s|--max-speed]}} {{speed}} {{url}}`

- Օգտագործեք IPv4 արձանագրությունը միայն հոսթին միանալիս.:

`axel {{[-4|--ipv4]}} {{url}}`

- Սահմանափակեք ելքը `stdout`-ով և ներբեռնելիս օգտագործեք հատուկ օգտագործողի գործակալ.:

`axel {{[-q|--quiet]}} {{[-U|--user-agent]}} "{{Mozilla/5.0}}" {{url}}`
