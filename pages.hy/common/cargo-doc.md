# բեռների դոկտ

> Կառուցեք Rust փաթեթների փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://doc.rust-lang.org/cargo/commands/cargo-doc.html>:.

- Կառուցեք փաստաթղթեր ընթացիկ նախագծի և բոլոր կախվածությունների համար.:

`cargo {{[d|doc]}}`

- Կախվածությունների համար փաստաթղթեր մի ստեղծեք.:

`cargo {{[d|doc]}} --no-deps`

- Կառուցեք և բացեք փաստաթղթերը զննարկիչում.:

`cargo {{[d|doc]}} --open`

- Կառուցեք և դիտեք որոշակի փաթեթի փաստաթղթերը.:

`cargo {{[d|doc]}} --open {{[-p|--package]}} {{package}}`
