# egrep

> Գտեք նախշեր ֆայլերում՝ օգտագործելով ընդլայնված `regex`es:.
> Նշում. այս հրամանը `grep --extended-regexp`-ի կեղծանունն է:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/egrep>:.

- Որոնեք մեկ կամ մի քանի կրկնվող նիշ.:

`egrep '{{a}}+' {{path/to/file}}`

- Որոնեք նիշի զրոյական կամ մեկ երևույթ (ըստ ցանկության).:

`egrep '{{a}}?' {{path/to/file}}`

- Որոնեք կերպարի 10 կրկնություն.:

`egrep '{{a}}{10}' {{path/to/file}}`

- Որոնեք կերպարի 3-ից 7 կրկնություն.:

`egrep '{{a}}{3,7}' {{path/to/file}}`

- Որոնեք նշված տարբերակներից մեկը.:

`egrep '{{cat}}|{{dog}}|{{mouse}}' {{path/to/file}}`

- Որոնեք թվարկված տարբերակներից մեկը ավելի մեծ օրինակի ներսում.:

`egrep 'c({{a}}|{{o}}|{{u}})p' {{path/to/file}}`

- Որոնեք մեկ կամ մի քանի անգամ կրկնվող նիշերի խումբ.:

`egrep '({{aeiou}})+' {{path/to/file}}`

- Որոնում՝ օգտագործելով ստանդարտ նիշերի դասեր (ավելի մանրամասն՝ <https://www.regular-expressions.info/posixbrackets.html>):

`egrep [[{{:alnum:|:alpha:|:space:|...}}]] {{path/to/file}}`
