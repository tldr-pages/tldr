# dmypy

> Մուտքագրեք ստուգիչ Python կոդը՝ գործարկելով `mypy`-ը որպես դևոն ավելի լավ արագության համար:.
> Տես նաև՝ `mypy`:.
> Լրացուցիչ տեղեկություններ. <https://mypy.readthedocs.io/en/stable/mypy_daemon.html>:.

- Մուտքագրեք ստուգեք ֆայլը և գործարկեք դեյմոնը, եթե այն չի աշխատում.:

`dmypy run -- {{path/to/file.py}}`

- Սկսեք դեյմոնը.:

`dmypy start`

- Մուտքագրեք ստուգեք ֆայլը (պահանջում է, որ դեյմոնը գործարկվի).:

`dmypy check -- {{path/to/file.py}}`

- Դադարեցրեք դևոնին.:

`dmypy stop`
