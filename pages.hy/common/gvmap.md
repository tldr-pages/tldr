# gvmap

> Գտեք կլաստերներ և ստեղծեք աշխարհագրական քարտեզ՝ առանձնացնելով կլաստերները:.
> Լրացուցիչ տեղեկություններ. <https://graphviz.org/pdf/gvmap.1.pdf>:.

- Ստեղծեք կլաստերային քարտեզի դասավորություն DOT ձևաչափի գրաֆիկից.:

`gvmap {{graph.gv}} -o {{output.xdot}}`

- Ստեղծեք քարտեզ՝ օգտագործելով առկա կլաստերային ենթագրաֆները մուտքագրումից.:

`gvmap -D {{graph.gv}}`

- Ներառեք գրաֆիկի եզրերը [o]ելքային քարտեզում.:

`gvmap -e {{graph.gv}} -o {{output.xdot}}`

- Նշեք [c] գունային սխեման (1: պաստել, 2: կապույտ-դեղին, 3: սպիտակ-կարմիր և այլն):

`gvmap -c {{1|2|3|...}} {{graph.gv}} -o {{output.xdot}}`

- Սահմանեք [C] փայլերի առավելագույն քանակը (լռելյայն 0, այսինքն՝ սահմանափակում չկա):

`gvmap -C {{10}} {{graph.gv}} -o {{output.xdot}}`

- Նկարել միայն կլաստեր 2 (ըստ լռելյայն, բոլոր կլաստերները գծված են).:

`gvmap -highlight={{2}} {{graph.gv}} -o {{output.xdot}}`
