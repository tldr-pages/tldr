# pulumi ոչնչացնել

> Ոչնչացնել բոլոր առկա ռեսուրսները մի կույտում:.
> Լրացուցիչ տեղեկություններ. <https://www.pulumi.com/docs/iac/cli/commands/pulumi_destroy/>:.

- Ոչնչացնել բոլոր ռեսուրսները ընթացիկ փաթեթում.:

`pulumi destroy`

- Ոչնչացնել բոլոր ռեսուրսները կոնկրետ փաթեթում.:

`pulumi destroy {{[-s|--stack]}} {{stack}}`

- Նախադիտելուց հետո ավտոմատ կերպով հաստատեք և ոչնչացրեք ռեսուրսները.:

`pulumi destroy {{[-y|--yes]}}`

- Բացառել պաշտպանված ռեսուրսները ոչնչացումից.:

`pulumi destroy --exclude-protected`

- Հեռացրեք կույտը և դրա կազմաձևման ֆայլը, երբ կույտի բոլոր ռեսուրսները ջնջվեն.:

`pulumi destroy --remove`

- Շարունակեք ոչնչացնել ռեսուրսները, նույնիսկ եթե սխալ է հանդիպում.:

`pulumi destroy --continue-on-error`
