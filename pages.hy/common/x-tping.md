# x tping

> Ping հոսթին TCP-ի միջոցով՝ հասանելիությունը ստուգելու համար:.
> Լրացուցիչ տեղեկություններ. <https://x-cmd.com/mod/tping>:.

- Ping հոսթինգ լռելյայն նավահանգստի 80-ի վրա՝ բառացի ռեժիմով.:

`x tping {{host}}`

- Ping հյուրընկալող հատուկ նավահանգիստով.:

`x tping {{host}}:{{port}}`

- Ցույց տալ արդյունքները որպես ջերմային քարտեզ.:

`x tping {{[-m|--heatmap]}} {{host}}`

- Ցույց տալ արդյունքները որպես գծային գրաֆիկ.:

`x tping {{[-b|--bar]}} {{host}}:{{port}}`

- Արդյունք հում տվյալների ռեժիմում.:

`x tping {{[-r|--raw]}} {{host}}`

- Ելք կոնկրետ ձևաչափով.:

`x tping {{--csv|--tsv}} {{host}}`
