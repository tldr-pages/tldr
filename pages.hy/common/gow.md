#հագուստ

> Դիտում է Go ֆայլերը և փոփոխությունների դեպքում վերագործարկում է հավելվածը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/mitranim/gow>:.

- Սկսեք և դիտեք ընթացիկ գրացուցակը.:

`gow run .`

- Սկսեք դիմումը նշված փաստարկներով.:

`gow run . {{argument1 argument2 ...}}`

- Դիտեք ենթադիրեկտորիաները մանրամասն ռեժիմով.:

`gow -v -w={{path/to/directory1,path/to/directory2,...}} run .`

- Դիտեք նշված ֆայլերի ընդարձակումները.:

`gow -e={{go,html}} run .`

- Ցուցադրել օգնությունը.:

`gow -h`
