# git փոխարինում

> Ստեղծեք, ցուցակագրեք և ջնջեք հղումներ՝ օբյեկտները փոխարինելու համար:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-replace>:.

- Փոխարինեք ցանկացած պարտավորություն մեկ այլով, մյուս պարտավորությունները թողնելով անփոփոխ.:

`git replace {{object}} {{replacement}}`

- Ջնջել գոյություն ունեցող փոխարինող հղումները տվյալ օբյեկտների համար.:

`git replace {{[-d|--delete]}} {{object}}`

- Ինտերակտիվ կերպով խմբագրել օբյեկտի բովանդակությունը.:

`git replace --edit {{object}}`
