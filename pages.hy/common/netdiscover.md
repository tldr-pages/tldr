# netdiscover

> Ցանցային սկաներ, որն օգտագործվում է ցանցում կենդանի հոսթեր գտնելու համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/netdiscover-scanner/netdiscover>:.

- Սկանավորեք IP տիրույթը ցանցային ինտերֆեյսի վրա ակտիվ հոսթերների համար.:

`netdiscover -r {{172.16.6.0/23}} -i {{ens244}}`
