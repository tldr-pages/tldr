#վերմակ

> Կառավարեք մի շարք patches:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/quilt>:.

- Ներմուծեք գոյություն ունեցող կարկատել ֆայլից.:

`quilt import {{path/to/file.patch}}`

- Ստեղծեք նոր կարկատել.:

`quilt new {{filename.patch}}`

- Ավելացրեք ֆայլ ընթացիկ կարկատմանը.:

`quilt add {{path/to/file}}`

- Ֆայլը խմբագրելուց հետո թարմացրեք ընթացիկ կարկատումը փոփոխություններով.:

`quilt refresh`

- Կիրառեք սերիայի ֆայլի բոլոր պատչերը.:

`quilt push -a`

- Հեռացրեք բոլոր կիրառված պատչերը.:

`quilt pop -a`
