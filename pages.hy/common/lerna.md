#լեռնա

> Կառավարեք JavaScript նախագծերը բազմաթիվ փաթեթներով:.
> Լրացուցիչ տեղեկություններ. <https://lerna.js.org/docs/api-reference/commands>:.

- Նախաձեռնել նախագծի ֆայլերը (`lerna.json`, `package.json`, `.git` և այլն):

`lerna init`

- Տեղադրեք յուրաքանչյուր փաթեթի բոլոր արտաքին կախվածությունները և միացրեք տեղական կախվածությունները.:

`lerna bootstrap`

- Գործարկեք հատուկ սկրիպտ յուրաքանչյուր փաթեթի համար, որն այն պարունակում է իր `package.json`-ում:

`lerna run {{script}}`

- Կատարեք կամայական կեղևի հրաման յուրաքանչյուր փաթեթում.:

`lerna exec -- {{ls}}`

- Հրապարակեք բոլոր փաթեթները, որոնք փոխվել են վերջին թողարկումից հետո.:

`lerna publish`
