# x

> Կառավարեք մոդուլները և փաթեթները:.
> Լրացուցիչ տեղեկություններ. <https://x-cmd.com>:.

- Օգտագործեք մոդուլ (օրինակ՝ ուղու մոդուլ).:

`x {{module}}`

- Տեղադրեք փաթեթ ամբողջ աշխարհում (օրինակ՝ jq):

`x env use {{package}}`

- Գործարկեք փաթեթը ժամանակավորապես ընթացիկ shell նիստում.:

`x env try {{package}}`

- Գործարկել փաթեթը անմիջապես առանց տեղադրման.:

`x {{package}}`

- Նախաձեռնեք OpenAI API ստեղները AI հատկանիշների համար.:

`x openai init`

- Հարցրեք փաթեթի տեղադրման եղանակը և կատարեք այն.:

`x install`
