# git շոու

> Ցույց տալ տարբեր տեսակի Git օբյեկտներ (կոմիտացիաներ, պիտակներ և այլն):.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-show>:.

- Ցույց տալ տեղեկատվություն վերջին commit-ի մասին (հեշ, հաղորդագրություն, փոփոխություններ և այլ մետատվյալներ).:

`git show`

- Ցույց տալ տեղեկատվություն կոնկրետ commit-ի, պիտակի կամ ճյուղի մասին (օրինակ՝ `HEAD` վերջին commit-ի համար).:

`git show {{commit|tag|branch}}`

- Ցուցադրել միայն ավելացված, վերանվանված կամ ջնջված ֆայլերի ցանկը.:

`git show --summary {{commit}}`

- Ցույց տալ 3-րդ կատարման մասին տեղեկությունները մասնաճյուղի `HEAD`-ից.:

`git show {{branch}}~{{3}}`

- Ցուցադրել commit-ի հաղորդագրությունը մեկ տողով՝ ճնշելով տարբերությունը.:

`git show --oneline {{[-s|--no-patch]}} {{commit}}`

- Ցույց տալ միայն վիճակագրությունը (ավելացված/հեռացված նիշեր) փոփոխված ֆայլերի մասին.:

`git show --stat {{commit}}`

- Ցուցադրել բոլոր ֆայլերի պարզեցված ցուցակը, որոնք փոխվել են commit-ում (փոփոխված, ավելացված և ջնջված).:

`git show --name-only {{commit}}`

- Ցուցադրել ֆայլի բովանդակությունը այնպես, ինչպես եղել է տվյալ վերանայման ժամանակ (օրինակ՝ ճյուղ, պիտակ կամ պարտավորություն).:

`git show {{revision}}:{{path/to/file}}`
