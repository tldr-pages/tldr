# մետր

> Matt's Traceroute. համակցված traceroute և ping գործիք:.
> Տես նաև՝ `traceroute`, `ping`:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/mtr>:.

- Traceroute դեպի հյուրընկալող և շարունակաբար պինգ կատարել բոլոր միջանկյալ հոփերը.:

`mtr {{example.com}}`

- Անջատել IP հասցեի և հյուրընկալողի անվան քարտեզագրումը.:

`mtr {{[-n|--no-dns]}} {{example.com}}`

- Ստեղծեք ելք յուրաքանչյուր հոփ 10 անգամ պինգինգ կատարելուց հետո.:

`mtr {{[-w|--report-wide]}} {{example.com}}`

- Ստիպել IPv4 կամ IPv6:

`mtr -4 {{example.com}}`

- Սպասեք որոշակի ժամանակ (վայրկյաններով), նախքան մեկ այլ փաթեթ ուղարկելը նույն հոփին.:

`mtr {{[-i|--interval]}} {{10}} {{example.com}}`

- Ցուցադրել ինքնավար համակարգի համարը (ASN) յուրաքանչյուր հոփի համար.:

`mtr {{[-z|--aslookup]}} {{example.com}}`

- Ցուցադրել ինչպես IP հասցեն, այնպես էլ հակադարձ DNS անունը.:

`mtr {{[-b|--show-ips]}} {{example.com}}`
