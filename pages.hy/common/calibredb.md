# calibredb

> Կառավարեք էլեկտրոնային գրքերի տվյալների բազան:.
> Caliber էլեկտրոնային գրքերի գրադարանի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://manual.calibre-ebook.com/generated/en/calibredb.html>:.

- Գրադարանում նշեք էլեկտրոնային գրքերը լրացուցիչ տեղեկություններով.:

`calibredb list`

- Որոնեք լրացուցիչ տեղեկություններ ցուցադրող էլեկտրոնային գրքեր.:

`calibredb list --search {{search_term}}`

- Որոնեք էլեկտրոնային գրքերի միայն ID-ներ.:

`calibredb search {{search_term}}`

- Գրադարանին ավելացրեք մեկ կամ մի քանի էլեկտրոնային գրքեր.:

`calibredb add {{path/to/file1 path/to/file2 ...}}`

- Ռեկուրսիվ կերպով գրադարանին ավելացրեք բոլոր էլեկտրոնային գրքերը գրացուցակի տակ.:

`calibredb add {{[-r|--recurse]}} {{path/to/directory}}`

- Հեռացրեք մեկ կամ մի քանի էլեկտրոնային գրքեր գրադարանից: Ձեզ անհրաժեշտ են էլեկտրոնային գրքերի նույնացուցիչները (տե՛ս վերևում).:

`calibredb remove {{id1 id2 ...}}`
