# lex

> Լեքսիկական անալիզատորի գեներատոր:.
> Հաշվի առնելով լեքսիկական անալիզատորի առանձնահատկությունները, այն իրականացնում է C կոդը:.
> Նշում. հիմնական ՕՀ-երի մեծ մասում այս հրամանը `flex`-ի այլանունն է:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/lex>:.

- Ստեղծեք անալիզատոր Lex ֆայլից՝ այն պահելով `lex.yy.c` ֆայլում:

`lex {{analyzer.l}}`

- Նշեք ելքային ֆայլը.:

`lex -t {{analyzer.l}} > {{analyzer.c}}`

- Կազմեք Lex-ի կողմից ստեղծված C ֆայլ.:

`c99 {{path/to/lex.yy.c}} -o {{executable}}`
