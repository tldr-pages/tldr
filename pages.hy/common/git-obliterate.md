# git ջնջել

> Ջնջել ֆայլերը և ջնջել դրանց պատմությունը Git-ի պահոցից:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-obliterate>:.

- Ջնջել հատուկ ֆայլերի առկայությունը.:

`git obliterate {{file_1 file_2 ...}}`

- Ջնջել հատուկ ֆայլերի առկայությունը 2 պարտավորությունների միջև.:

`git obliterate {{file_1 file_2 ...}} -- {{commit_hash_1}}..{{commit_hash_2}}`
