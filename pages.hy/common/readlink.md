# կարդալ հղում

> Հետևեք սիմհղումներին և ստացեք սիմհղման տեղեկություններ:.
> Լրացուցիչ տեղեկություններ. <https://www.gnu.org/software/coreutils/manual/html_node/readlink-invocation.html>:.

- Ստացեք իրական ֆայլը, որին մատնանշում է symlink-ը.:

`readlink {{path/to/file}}`

- Ստացեք բացարձակ ուղին դեպի ֆայլ.:

`readlink {{[-f|--canonicalize]}} {{path/to/file}}`
