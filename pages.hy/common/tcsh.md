# tcsh

> C shell ֆայլի անվան լրացմամբ և հրամանի տողի խմբագրմամբ:.
> Տես նաև՝ `csh`:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/tcsh>:.

- Սկսեք ինտերակտիվ shell նստաշրջան.:

`tcsh`

- Սկսեք ինտերակտիվ կեղևի նստաշրջան առանց գործարկման կազմաձևերի բեռնման.:

`tcsh -f`

- Կատարեք հատուկ [c] հրամաններ.:

`tcsh -c "{{echo 'tcsh is executed'}}"`

- Կատարել կոնկրետ սցենար.:

`tcsh {{path/to/script.tcsh}}`

- Ստուգեք հատուկ սցենար շարահյուսական սխալների համար.:

`tcsh -n {{path/to/script.tcsh}}`

- Կատարեք հատուկ հրամաններ `stdin`-ից.:

`{{echo "echo 'tcsh is executed'"}} | tcsh`
