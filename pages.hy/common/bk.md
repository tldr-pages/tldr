# բկ

> Կառավարեք Buildkite-ի շինությունները, խողովակաշարերը և գործակալները:.
> Լրացուցիչ տեղեկություններ. <https://github.com/buildkite/cli#usage>:.

- Կարգավորեք API նշանը և կազմակերպությունը.:

`bk configure`

- Ընտրեք կազմակերպություն օգտագործելու համար.:

`bk use {{organization_slug}}`

- Նախաձեռնել `pipeline.yaml` ֆայլը՝:

`bk init`

- Թվարկեք ներկայիս կազմակերպության բոլոր խողովակաշարերը.:

`bk pipeline list`

- Խողովակաշարի կառուցման գործարկում.:

`bk build create {{pipeline_slug}}`

- Դիտեք կոնկրետ կառուցման կարգավիճակը.:

`bk build view {{build_number}}`

- Թվարկեք բոլոր գործակալներին ընթացիկ կազմակերպությունում.:

`bk agent list`

- Ցուցադրել օգնությունը.:

`bk {{[-h|--help]}}`
