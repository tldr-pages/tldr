# git push

> Push-ը պարտավորվում է դեպի հեռավոր պահեստ:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-push>:.

- Ընթացիկ մասնաճյուղի տեղական փոփոխություններն ուղարկեք իր կանխադրված հեռավոր գործընկերոջը.:

`git push`

- Փոփոխություններ ուղարկել կոնկրետ տեղական մասնաճյուղից իր հեռավոր գործընկերոջը.:

`git push {{remote_name}} {{local_branch}}`

- Փոփոխություններ ուղարկեք կոնկրետ տեղական մասնաճյուղից իր հեռավոր գործընկերոջը և սահմանեք հեռավորը որպես տեղականի լռելյայն մղում/քաշման թիրախ:

`git push {{[-u|--set-upstream]}} {{remote_name}} {{local_branch}}`

- Փոփոխություններ ուղարկել կոնկրետ տեղական մասնաճյուղից որոշակի հեռավոր մասնաճյուղ.:

`git push {{remote_name}} {{local_branch}}:{{remote_branch}}`

- Ուղարկեք փոփոխությունները բոլոր տեղական մասնաճյուղերում իրենց գործընկերներին տվյալ հեռավոր պահոցում.:

`git push --all {{remote_name}}`

- Ջնջել մասնաճյուղը հեռավոր պահոցում.:

`git push {{remote_name}} {{[-d|--delete]}} {{remote_branch}}`

- Հեռացրեք հեռավոր մասնաճյուղերը, որոնք չունեն տեղական գործընկեր.:

`git push --prune {{remote_name}}`

- Հրապարակեք պիտակներ, որոնք դեռևս չկան հեռավոր պահոցում.:

`git push --tags`
