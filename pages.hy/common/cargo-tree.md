# բեռների ծառ

> Ցուցադրել կախվածության գրաֆիկի ծառի արտացոլումը:.
> Ծանոթագրություն. Ծառի մեջ `(*)`-ով նշված փաթեթների կախվածությունն արդեն ցուցադրվել է գրաֆիկի մեկ այլ տեղ, և այդ պատճառով չեն կրկնվում:.
> Լրացուցիչ տեղեկություններ. <https://doc.rust-lang.org/cargo/commands/cargo-tree.html>:.

- Ցույց տալ ընթացիկ նախագծի կախվածության ծառը.:

`cargo tree`

- Ցուցադրել միայն կախվածությունը մինչև նշված խորությունը (օրինակ, երբ `n`-ը 1 է, ցուցադրել միայն ուղղակի կախվածությունները):

`cargo tree --depth {{n}}`

- Մի ցուցադրեք տրված փաթեթը (և դրա կախվածությունները) ծառի վրա.:

`cargo tree --prune {{package_spec}}`

- Ցույց տալ կրկնվող կախվածությունների բոլոր դեպքերը.:

`cargo tree --no-dedupe`

- Ցույց տալ միայն նորմալ/կառուցել/զարգացման կախվածությունը.:

`cargo tree {{[-e|--edges]}} {{normal|build|dev}}`
