# ցամաքային

> Ձեր Terraform CLI արգումենտները չոր պահեք:.
> Լրացուցիչ տեղեկություններ. <https://terragrunt.gruntwork.io/docs/reference/cli/>:.

- Ստեղծեք և ցուցադրեք կատարման պլան.:

`terragrunt plan`

- Կառուցել կամ փոխել ենթակառուցվածքը.:

`terragrunt apply`

- Ցույց տալ ընթացիկ տեղակայումը (պետությունից).:

`terragrunt show`

- Ցույց տալ մոդուլի ելքային արժեքները.:

`terragrunt output`

- Ոչնչացնել Terraform-ի կողմից կառավարվող ենթակառուցվածքը.:

`terragrunt destroy`

- Կառուցեք կամ փոխեք ենթակառուցվածքը Terragrunt մոդուլների ծառից (կույտ).:

`terragrunt run-all apply`
