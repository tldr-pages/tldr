# git անջատիչ

> Անցում Git ճյուղերի միջև: Պահանջում է Git տարբերակ 2.23+:.
> Տես նաև՝ `git checkout`:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-switch>:.

- Անցում գոյություն ունեցող մասնաճյուղին.:

`git switch {{branch_name}}`

- Ստեղծեք նոր մասնաճյուղ և անցեք դրան.:

`git switch {{[-c|--create]}} {{branch_name}}`

- Ստեղծեք նոր մասնաճյուղ՝ հիմնված գոյություն ունեցող commit-ի վրա և անցեք դրան.:

`git switch {{[-c|--create]}} {{branch_name}} {{commit}}`

- Անցում նախորդ մասնաճյուղին.:

`git switch -`

- Անցեք մասնաճյուղ և թարմացրեք բոլոր ենթամոդուլները՝ համապատասխանելու համար.:

`git switch --recurse-submodules {{branch_name}}`

- Անցեք մասնաճյուղ և ավտոմատ կերպով միաձուլեք ընթացիկ մասնաճյուղը և դրա մեջ ցանկացած չհաստատված փոփոխություն.:

`git switch {{[-m|--merge]}} {{branch_name}}`

- Անցեք պիտակի կամ կատարեք.:

`git switch {{[-d|--detach]}} {{tag|commit}}`

- Անցում դեպի մեկ այլ հեռակառավարման ճյուղ.:

`git switch {{[-t|--track]}} {{remote_name}}/{{branch}}`
