# կրճատել

> Նվազեցնել կամ երկարացնել ֆայլի չափը նշված չափին:.
> Լրացուցիչ տեղեկություններ. <https://www.gnu.org/software/coreutils/manual/html_node/truncate-invocation.html>:.

- Գոյություն ունեցող ֆայլի համար սահմանեք 10 ԳԲ չափ կամ ստեղծեք նոր ֆայլ նշված չափով.:

`truncate {{[-s|--size]}} 10G {{path/to/file}}`

- Երկարացրեք ֆայլի չափը 50 ՄԲ-ով, լրացրեք անցքեր (որը կարդում է որպես զրո բայթ).:

`truncate {{[-s|--size]}} +50M {{path/to/file}}`

- Նեղացրեք ֆայլը 2 ԳԲ-ով, ֆայլի վերջից տվյալները հեռացնելով.:

`truncate {{[-s|--size]}} -2G {{path/to/file}}`

- Դատարկեք ֆայլի բովանդակությունը.:

`truncate {{[-s|--size]}} 0 {{path/to/file}}`

- Դատարկեք ֆայլի բովանդակությունը, բայց մի ստեղծեք ֆայլը, եթե այն գոյություն չունի.:

`truncate {{[-s|--size]}} 0 {{[-c|--no-create]}} {{path/to/file}}`
