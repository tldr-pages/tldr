# aws codecommit

> Կառավարվող աղբյուրի կառավարման ծառայություն, որը հյուրընկալում է մասնավոր Git պահոցներ:.
> Լրացուցիչ տեղեկություններ. <https://docs.aws.amazon.com/cli/latest/reference/codecommit/>:.

- Ցուցադրել օգնությունը.:

`aws codecommit help`

- Ցուցադրել օգնություն կոնկրետ հրամանի համար.:

`aws codecommit {{command}} help`
