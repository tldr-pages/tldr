# հաշվ

> Ինտերակտիվ կամայական ճշգրիտ հաշվիչ տերմինալում:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/calc>:.

- Սկսեք `calc`-ը ինտերակտիվ ռեժիմում.:

`calc`

- Կատարեք հաշվարկ ոչ ինտերակտիվ ռեժիմով.:

`calc '{{85 * (36 / 4)}}'`

- Մի ձևաչափեք ելքը ([p]ipes-ի հետ օգտագործման համար).:

`calc -p '{{4/3 * pi() * 5^3}}'`

- Կատարեք հաշվարկ և անցեք [i] ինտերակտիվ ռեժիմի.:

`calc -i '{{sqrt(2)}}'`

- Սկսեք `calc`-ը որոշակի թույլտվության [m] ռեժիմով (0-ից 7, կանխադրվածը՝ 7):

`calc -m {{mode}}`

- Դիտեք `calc`-ի ներածությունը:

`calc help intro`

- Դիտեք `calc`-ի ակնարկը՝:

`calc help overview`

- Բացեք `calc` ձեռնարկը՝:

`calc help`
