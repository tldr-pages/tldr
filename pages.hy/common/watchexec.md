# watchexec

> Գործարկել կամայական հրամաններ, երբ ֆայլերը փոխվում են:.
> Տես նաև՝ `entr`:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/watchexec>:.

- Զանգահարեք `ls -la`-ին, երբ ընթացիկ գրացուցակի որևէ ֆայլ փոխվում է.:

`watchexec {{ls -la}}`

- Գործարկեք `make`-ը, երբ ընթացիկ գրացուցակի որևէ JavaScript, CSS և HTML ֆայլ փոխվում է.:

`watchexec {{[-e|--exts]}} {{js,css,html}} make`

- Գործարկեք `make`-ը, երբ `lib` կամ `src` գրացուցակի որևէ ֆայլ փոխվում է.:

`watchexec {{[-w|--watch]}} {{lib}} {{[-w|--watch]}} {{src}} {{make}}`

- Զանգահարեք/վերագործարկեք `my_server`-ը, երբ ընթացիկ գրացուցակի որևէ ֆայլ փոխվում է՝ ուղարկելով `SIGKILL`՝ երեխայի գործընթացը դադարեցնելու համար.:

`watchexec {{[-r|--restart]}} --stop-signal {{SIGKILL}} {{my_server}}`

- Վերագործարկեք հրամանի կատարումը, երբ ընթացիկ գրացուցակի Java աղբյուրի որևէ ֆայլ փոխվում է, ուղարկելով `SIGKILL` և միայն ստուգելով թարմացումները յուրաքանչյուր `10000` ms:

`watchexec {{[-r|--restart]}} --stop-signal {{SIGKILL}} --poll 10000 {{[-e|--exts]}} {{java}} {{command}}`
