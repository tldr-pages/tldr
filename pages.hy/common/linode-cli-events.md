# linode-cli իրադարձություններ

> Կառավարեք Linode իրադարձությունները:.
> Տես նաև՝ `linode-cli`:.
> Լրացուցիչ տեղեկություններ. <https://techdocs.akamai.com/cloud-computing/docs/cli-commands-for-viewing-events>:.

- Թվարկեք իրադարձությունները ձեր հաշվի վրա.:

`linode-cli events list`

- Դիտեք մանրամասներ կոնկրետ իրադարձության մասին.:

`linode-cli events view {{event_id}}`

- Նշեք իրադարձությունը որպես կարդացված.:

`linode-cli events mark-read {{event_id}}`
