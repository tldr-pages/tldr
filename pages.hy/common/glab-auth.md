# glab auth

> Նույնականացում GitLab հոսթի միջոցով:.
> Լրացուցիչ տեղեկություններ. <https://gitlab.com/gitlab-org/cli/-/blob/main/docs/source/auth/_index.md>:.

- Մուտք գործեք ինտերակտիվ հուշումով.:

`glab auth login`

- Մուտք գործեք նշանով.:

`glab auth login {{[-t|--token]}} {{token}}`

- Ստուգեք նույնականացման կարգավիճակը.:

`glab auth status`

- Մուտք գործեք GitLab-ի որոշակի օրինակ.:

`glab auth login {{[-h|--hostname]}} {{gitlab.example.com}}`
