# npm մուտք

> Մուտք գործեք ռեեստրի օգտատիրոջ հաշիվ:.
> Տես նաև՝ `npm logout`:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-login/>:.

- Մուտք գործեք ռեեստրի օգտատիրոջ հաշիվ և պահպանեք հավատարմագրերը `.npmrc` ֆայլում՝:

`npm login`

- Մուտք գործեք՝ օգտագործելով մաքսային ռեեստր.:

`npm login --registry {{registry_url}}`

- Մուտք գործեք՝ օգտագործելով նույնականացման հատուկ ռազմավարություն.:

`npm login --auth-type {{legacy|web}}`
