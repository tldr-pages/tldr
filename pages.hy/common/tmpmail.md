# tmpmail

> Ժամանակավոր նամակ հենց ձեր տերմինալից գրված POSIX sh.
> Լրացուցիչ տեղեկություններ. <https://github.com/sdushantha/tmpmail#usage>:.

- Ստեղծեք ժամանակավոր մուտքի արկղ.:

`tmpmail {{[-g|--generate]}}`

- Ցուցակեք հաղորդագրությունները և դրանց թվային ID-ն.:

`tmpmail`

- Ցուցադրել ամենավերջին ստացված էլ.:

`tmpmail {{[-r|--recent]}}`

- Բացեք կոնկրետ հաղորդագրություն.:

`tmpmail {{email_id}}`

- Դիտեք էլփոստը որպես չմշակված տեքստ առանց HTML պիտակների.:

`tmpmail {{[-t|--text]}}`

- Բացեք էլփոստը հատուկ դիտարկիչով (կանխադրվածը w3m է).:

`tmpmail {{[-b|--browser]}} {{browser}}`
