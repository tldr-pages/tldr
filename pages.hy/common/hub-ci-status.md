# հանգույց ci-կարգավիճակ

> Ցուցադրել GitHub ստուգումների կարգավիճակը:.
> Լրացուցիչ տեղեկություններ. <https://hub.github.com/hub-ci-status.1.html>:.

- Ստուգեք այս մասնաճյուղի CI կարգավիճակը.:

`hub ci-status {{[-v|--verbose]}}`

- Ցուցադրել GitHub-ի ստուգումների կարգավիճակը պարտավորության համար.:

`hub ci-status {{[-v|--verbose]}} {{commit_sha}}`
