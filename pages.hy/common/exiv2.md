# exiv2

> Պատկերի մետատվյալների մանիպուլյացիայի գործիք:.
> Լրացուցիչ տեղեկություններ. <https://exiv2.org/manpage.html>:.

- Տպեք պատկերի Exif մետատվյալների ամփոփագիրը.:

`exiv2 {{path/to/file}}`

- Տպեք բոլոր մետատվյալները (Exif, IPTC, XMP) մեկնաբանված արժեքներով.:

`exiv2 {{[-P|-Print]}} kt {{path/to/file}}`

- Տպել բոլոր մետատվյալները չմշակված արժեքներով.:

`exiv2 {{[-P|-Print]}} kv {{path/to/file}}`

- Ջնջել բոլոր մետատվյալները պատկերից.:

`exiv2 {{[-d|--delete]}} a {{path/to/file}}`

- Ջնջել բոլոր մետատվյալները՝ պահպանելով ֆայլի ժամանակացույցը.:

`exiv2 {{[-d|--delete]}} a {{[-k|--keep]}} {{path/to/file}}`

- Վերանվանեք ֆայլը` մետատվյալներից (ոչ թե ֆայլի ժամանակի դրոշմից) ամսաթվի և ժամի հետ կապված.:

`exiv2 {{[-r|--rename]}} '{{%Y%m%d_%H%M%S_:basename:}}' {{path/to/file}}`
