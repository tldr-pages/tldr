# բեռի որոնում

> Փաթեթների որոնում <https://crates.io> կայքում:.
> Արկղերը ցուցադրվում են նկարագրությունների հետ միասին TOML ձևաչափով, որը հարմար է `Cargo.toml`-ում պատճենելու համար:.
> Լրացուցիչ տեղեկություններ. <https://doc.rust-lang.org/cargo/commands/cargo-search.html>:.

- Փաթեթների որոնում.:

`cargo search {{query}}`

- Ցույց տալ `n` արդյունքները (կանխադրված՝ 10, առավելագույնը՝ 100):

`cargo search --limit {{n}} {{query}}`
