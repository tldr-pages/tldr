# rabbitmqctl

> Կառավարեք, կազմաձևեք և ստուգեք RabbitMQ սերվերները:.
> Լրացուցիչ տեղեկություններ. <https://www.rabbitmq.com/docs/man/rabbitmqctl.8>:.

- Ցուցադրել ընթացիկ հանգույցի կարգավիճակը.:

`rabbitmqctl status`

- Դիտեք փաստաթղթերը RabbitMQ կլաստերի հրամանների համար.:

`tldr rabbitmqctl cluster`

- Դիտեք փաստաթղթեր RabbitMQ օգտվողի հրամանների համար.:

`tldr rabbitmqctl users`

- Դիտեք փաստաթղթերը RabbitMQ vhosts հրամանների համար.:

`tldr rabbitmqctl vhosts`

- Իմացեք ավելին կոնկրետ հրամանի մասին.:

`rabbitmqctl help {{command}}`
