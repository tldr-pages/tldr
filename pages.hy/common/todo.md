# անել

> Պարզ, ստանդարտների վրա հիմնված, cli todo կառավարիչ:.
> Լրացուցիչ տեղեկություններ. <https://todoman.readthedocs.io/en/stable/man.html>:.

- Թվարկեք սկսվող առաջադրանքները.:

`todo list --startable`

- Աշխատանքային ցանկում ավելացրեք նոր առաջադրանք.:

`todo new {{thing_to_do}} {{[-l|--list]}} {{work}}`

- Տրված ID-ով առաջադրանքին տեղադրություն ավելացրեք.:

`todo edit --location {{location_name}} {{task_id}}`

- Ցույց տալ մանրամասներ առաջադրանքի մասին.:

`todo show {{task_id}}`

- Նշեք առաջադրանքները նշված ID-ներով որպես ավարտված.:

`todo done {{task_id1 task_id2 ...}}`

- Ջնջել առաջադրանքը.:

`todo delete {{task_id}}`

- Ջնջել կատարված առաջադրանքները և վերականգնել մնացած առաջադրանքների ID-ները.:

`todo flush`
