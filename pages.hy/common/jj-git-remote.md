# jj git հեռակառավարման վահանակ

> Կառավարեք Git հեռակառավարիչները:.
> Լրացուցիչ տեղեկություններ. <https://docs.jj-vcs.dev/latest/cli-reference/#jj-git-remote>:.

- Նշեք բոլոր Git հեռակառավարման սարքերը.:

`jj git remote list`

- Ավելացնել Git հեռակառավարման վահանակ.:

`jj git remote add {{remote}} {{url}}`

- Փոխեք Git հեռակառավարման URL-ը.:

`jj git remote set-url {{remote}} {{url}}`

- Հեռացրեք Git հեռակառավարիչը.:

`jj git remote remove {{remote}}`

- Վերանվանել Git հեռակառավարման վահանակը.:

`jj git remote rename {{old_name}} {{new_name}}`
