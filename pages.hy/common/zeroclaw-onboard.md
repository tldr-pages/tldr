# zeroclaw ինքնաթիռում

> Նախաձեռնել աշխատանքային տարածքը և կազմաձևումը ZeroClaw-ի համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/zeroclaw-labs/zeroclaw#quick-start>:.

- Արագ կարգավորում API բանալիով և մատակարարով.:

`zeroclaw onboard --api-key {{api_key}} --provider {{openrouter}}`

- Գործարկեք ամբողջական ինտերակտիվ հրաշագործը.:

`zeroclaw onboard --interactive`

- Վերակազմավորեք միայն ալիքները/թույլատրելի ցուցակները (արագ վերանորոգման հոսք).:

`zeroclaw onboard --channels-only`

- Արագ կարգավորում հատուկ հիշողության հետին մասով.:

`zeroclaw onboard --api-key {{api_key}} --provider {{openrouter}} --memory {{sqlite}}`

- Ցուցադրել օգնությունը.:

`zeroclaw onboard {{[-h|--help]}}`
