# git ls-remote

> Git հրաման՝ հեռավոր պահեստում տեղեկանքների ցուցակագրման համար՝ հիմնվելով անվան կամ URL-ի վրա:.
> Եթե որևէ անուն կամ URL տրված չէ, ապա կօգտագործվի կազմաձևված հոսանքային ճյուղը կամ հեռավոր սկզբնաղբյուրը, եթե առաջինը կազմաձևված չէ:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-ls-remote>:.

- Ցուցադրել բոլոր հղումները կանխադրված հեռավոր պահոցում.:

`git ls-remote`

- Ցույց տալ միայն գլուխների հղումները կանխադրված հեռավոր պահոցում.:

`git ls-remote --heads`

- Ցույց տալ միայն պիտակների հղումները կանխադրված հեռավոր պահոցում.:

`git ls-remote {{[-t|--tags]}}`

- Ցույց տալ բոլոր հղումները հեռավոր պահոցից՝ հիմնված անվան կամ URL-ի վրա.:

`git ls-remote {{repository_url}}`

- Ցույց տալ հղումները հեռավոր պահոցից, որը զտված է օրինաչափությամբ.:

`git ls-remote {{repository_name}} "{{pattern}}"`
