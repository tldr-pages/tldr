# git հետարկել

> Հետարկել վերջին պարտավորությունները:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-undo>:.

- Հեռացրեք ամենավերջին պարտավորությունը.:

`git undo`

- Հեռացրեք ամենավերջին պարտավորությունների որոշակի քանակ.:

`git undo {{3}}`
