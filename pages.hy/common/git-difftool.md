# git difftool

> Ցուցադրել ֆայլի փոփոխությունները՝ օգտագործելով արտաքին տարբերությունների գործիքները: Ընդունում է նույն տարբերակներն ու փաստարկները, ինչ `git diff`:.
> Տես նաև՝ `git diff`:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-difftool>:.

- Ցուցակեք առկա տարբեր գործիքները.:

`git difftool --tool-help`

- Սահմանեք լռելյայն տարբերության գործիքը Meld:

`git config --global diff.tool "meld"`

- Օգտագործեք լռելյայն տարբերության գործիքը՝ փուլային փոփոխությունները ցուցադրելու համար.:

`git difftool --staged`

- Օգտագործեք հատուկ գործիք՝ տվյալ պարտավորությունից հետո փոփոխությունները ցույց տալու համար.:

`git difftool {{[-t|--tool]}} {{opendiff}} {{commit}}`
