# npm օգնություն-որոնում

> Որոնեք npm օգնության փաստաթղթեր՝ օգտագործելով հիմնաբառ:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-help-search/>:.

- Որոնեք բանալի բառի հետ կապված հրամաններ կամ թեմաներ.:

`npm help-search {{keyword}}`

- Որոնել բազմաթիվ բառերով.:

`npm help-search {{keyword1 keyword2 ...}}`
