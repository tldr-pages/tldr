# mvn կոմպիլյացիա

> Կազմեք Maven նախագծի սկզբնական կոդը:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/mvn>:.

- Կազմեք նախագծի սկզբնական կոդը.:

`mvn compile`

- Մաքրել կազմված ֆայլերը և վերակազմավորել.:

`mvn clean compile`

- Կազմել հատուկ մոդուլ բազմամոդուլային նախագծում.:

`mvn compile {{[-pl|--projects]}} {{module_name}}`

- Բաց թողնել թեստերը կազմելիս.:

`mvn compile {{[-D|--define]}} skipTests`
