# nvm

> Տեղադրեք, տեղահանեք կամ անցեք Node.js տարբերակների միջև:.
> Աջակցում է տարբերակների համարներին, ինչպիսիք են «12.8» կամ «v16.13.1», և պիտակներ, ինչպիսիք են «հանգույց», «համակարգ» և այլն:.
> Տես նաև՝ `asdf`:.
> Լրացուցիչ տեղեկություններ. <https://github.com/nvm-sh/nvm#usage>:.

- Տեղադրեք/տեղահանեք Node.js-ի կոնկրետ տարբերակը.:

`nvm {{install|uninstall}} {{node_version}}`

- Օգտագործեք Node.js-ի կոնկրետ տարբերակը ընթացիկ վահանակում.:

`nvm use {{node_version}}`

- Սահմանեք Node.js-ի լռելյայն տարբերակը.:

`nvm alias default {{node_version}}`

- Ցուցադրեք տեղադրման համար հասանելի հեռավոր տարբերակները, ցուցադրեք միայն LTS (երկարաժամկետ աջակցություն) տարբերակները.:

`nvm ls-remote --lts`

- Տեղադրված տարբերակների ցանկ.:

`nvm {{[ls|list]}}`

- Գործարկեք Node.js-ի կոնկրետ տարբերակի REPL-ը.:

`nvm run {{node_version}}`

- Կատարեք սցենար Node.js-ի կոնկրետ տարբերակում.:

`nvm exec {{node_version}} node {{path/to/script.js}}`

- Թարմացրեք վերջին աշխատանքային npm տարբերակին ընթացիկ Node.js տարբերակում.:

`nvm install-latest-npm`
