# sfdk էմուլյատորի տեղադրում

> Տեղադրեք էմուլյատորներ:.
> Լրացուցիչ տեղեկություններ. <https://docs.sailfishos.org/Tools/Sailfish_SDK/Early_Access/#early-access-build-targets-and-emulator>:.

- Տեղադրեք էմուլյատոր.:

`sfdk emulator install {{name}}`

- Հեռացրեք էմուլյատորը.:

`sfdk emulator remove {{name}}`

- Ցուցադրել առկա էմուլյատորները.:

`sfdk emulator list {{[-a|--available]}}`

- Ցուցադրել տեղադրված էմուլյատորները.:

`sfdk emulator list`
