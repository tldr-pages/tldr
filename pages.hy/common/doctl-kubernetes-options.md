# doctl kubernetes ընտրանքներ

> Ստացեք արժեքներ, որոնք հասանելի են `doctl`-ի Kubernetes հրամաններով օգտագործելու համար:.
> Լրացուցիչ տեղեկություններ. <https://docs.digitalocean.com/reference/doctl/reference/kubernetes/options/>:.

- Թվարկեք տարածաշրջանները, որոնք աջակցում են Kubernetes կլաստերներին.:

`doctl {{[k|kubernetes]}} {{[o|options]}} {{[r|regions]}}`

- Թվարկեք մեքենաների չափերը, որոնք կարող են օգտագործվել Kubernetes կլաստերում.:

`doctl {{[k|kubernetes]}} {{[o|options]}} {{[s|sizes]}}`

- Թվարկեք Kubernetes-ի տարբերակները, որոնք կարող են օգտագործվել DigitalOcean կլաստերների հետ.:

`doctl {{[k|kubernetes]}} {{[o|options]}} {{[v|versions]}}`
