# ընդօրինակել

> Վերականգնել կեղևի ընտրանքները՝ որոշակի կեղևի վարքագծին համապատասխանելու համար:.
> Օգտագործվում է նաև՝ ապահովելու համար, որ shell ֆունկցիան աշխատում է լռելյայն կազմաձևով:.
> Լրացուցիչ տեղեկություններ. <https://github.com/antonio/zsh-config/blob/master/help/emulate>:.

- Տպել ընթացիկ էմուլյացիայի ռեժիմը.:

`emulate`

- Կարգավորեք ընթացիկ shell նստաշրջանի կարգավորումները՝ նմանակելու Bash-ին.:

`emulate bash`

- Գնահատեք [c] հրամանը ժամանակավոր էմուլյացիայի շրջանակներում.:

`emulate ksh -c "{{echo hi}}"`

- Ընդօրինակեք լռելյայն Zsh, [R] կարգավորումների ընտրանքները, երբ հնարավոր է.:

`emulate zsh -R`

- Ընթացիկ ֆունկցիայի շրջանակի էմուլյացիա.:

`{{my_function}}() { emulate -L {{zsh|sh|ksh|csh}}; {{ls}} }`
