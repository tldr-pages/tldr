# rustup override

> Փոփոխել գրացուցակի գործիքաշղթայի անտեսումները:.
> Գործիքների շղթաների մասին լրացուցիչ տեղեկությունների համար տես `rustup help toolchain`:.
> Լրացուցիչ տեղեկություններ. <https://rust-lang.github.io/rustup/>:.

- Ցուցակ գրացուցակի գործիքների շղթայի անտեսումները.:

`rustup override list`

- Սահմանեք ընթացիկ գրացուցակի շրջադարձային գործիքների շղթան (այսինքն՝ ասեք `rustup`-ին, որ գործարկի `cargo`, `rustc` և այլն, որոշակի գործիքաշարից, երբ այդ գրացուցակում է):

`rustup override set {{toolchain}}`

- Հեռացրեք ընթացիկ գրացուցակի համար գործիքաշարի փոխարինումը.:

`rustup override unset`

- Հեռացրեք բոլոր գործիքների շղթայի անտեսումները այն դիրեկտորիաների համար, որոնք այլևս գոյություն չունեն.:

`rustup override unset --nonexistent`
