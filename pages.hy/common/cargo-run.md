# բեռնափոխադրում

> Գործարկեք ընթացիկ բեռների փաթեթը:.
> Նշում. Կատարված երկուականի աշխատանքային գրացուցակը կսահմանվի ընթացիկ աշխատանքային գրացուցակի վրա:.
> Լրացուցիչ տեղեկություններ. <https://doc.rust-lang.org/cargo/commands/cargo-run.html>:.

- Գործարկեք լռելյայն երկուական թիրախը.:

`cargo {{[r|run]}}`

- Գործարկեք նշված երկուականը.:

`cargo {{[r|run]}} --bin {{name}}`

- Գործարկեք նշված օրինակը.:

`cargo {{[r|run]}} --example {{name}}`

- Ակտիվացրեք բացատ կամ ստորակետերով առանձնացված հատկանիշների ցանկը.:

`cargo {{[r|run]}} {{[-F|--features]}} "{{feature1 feature2 ...}}"`

- Անջատեք լռելյայն հնարավորությունները.:

`cargo {{[r|run]}} --no-default-features`

- Ակտիվացրեք բոլոր առկա հնարավորությունները.:

`cargo {{[r|run]}} --all-features`

- Գործարկել տվյալ պրոֆիլով.:

`cargo {{[r|run]}} --profile {{name}}`
