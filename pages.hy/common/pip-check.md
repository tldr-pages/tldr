# pip check

> Ստուգեք տեղադրված փաթեթները կոտրված կամ անհամատեղելի կախվածությունների համար:.
> Լրացուցիչ տեղեկություններ. <https://pip.pypa.io/en/stable/cli/pip_check/>:.

- Ստուգեք կոտրված կամ բացակայող պահանջները.:

`pip check`

- Ստուգեք և գրանցեք ելքը ֆայլում՝:

`pip check --log {{path/to/file}}`
