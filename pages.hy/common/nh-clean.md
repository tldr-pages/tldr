# nh մաքուր

> Մաքրել Nix պրոֆիլները և ջնջել չօգտագործված և անհասանելի Nix խանութի ուղիները:.
> Սերունդները կարելի է ցուցակագրել՝ օգտագործելով `nix-env --list-generations` կամ `nh os info`:.
> Լրացուցիչ տեղեկություններ. <https://github.com/nix-community/nh#usage>:.

- Խնդրեք մաքրման պլանի հաստատում, մաքրեք բոլոր պրոֆիլները և հավաքեք աղբը.:

`nh clean all {{[-a|--ask]}}`

- Պահպանեք ամենավերջին պրոֆիլների որոշակի քանակ ընթացիկ օգտագործողի համար և մաքրեք մնացած պրոֆիլները.:

`nh clean user {{[-k|--keep]}} {{number}}`

- Մաքրել որոշակի պրոֆիլ և հավաքել աղբը.:

`nh clean profile {{path/to/profile}}`
