# sfdk էմուլյատոր

> Պահպանեք և վերահսկեք էմուլյատորները:.
> Լրացուցիչ տեղեկություններ. <https://github.com/sailfishos/sailfish-qtcreator/blob/master/share/qtcreator/sfdk/modules/40-testing-maintain/doc/command.emulator.adoc>:.

- Ցուցադրել տեղադրված էմուլյատորները.:

`sfdk emulator list`

- Սկսեք էմուլյատոր.:

`sfdk emulator start {{name}}`

- Դադարեցրեք էմուլյատորը.:

`sfdk emulator stop {{name}}`

- Ցուցադրել էմուլյատորի կարգավիճակը.:

`sfdk emulator status {{name}}`

- Գործարկեք ինտերակտիվ վահանակ էմուլյատորի վրա.:

`sfdk emulator exec {{emulator}}`

- Կատարեք հրաման էմուլյատորի վրա.:

`sfdk emulator exec {{emulator}} {{command}}`

- Սահմանել սեփականություն.:

`sfdk emulator set {{name}} {{property}}={{value}}`

- Ցույց տալ էմուլյատորի հատկությունները.:

`sfdk emulator show {{name}}`
