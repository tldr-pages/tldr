# mysql_secure_installation

> Ստեղծեք MySQL՝ ավելի լավ անվտանգություն ունենալու համար:.
> Լրացուցիչ տեղեկություններ. <https://dev.mysql.com/doc/refman/en/mysql-secure-installation.html>:.

- Սկսեք ինտերակտիվ կարգավորում.:

`mysql_secure_installation`

- Օգտագործեք հատուկ հոսթ և նավահանգիստ.:

`mysql_secure_installation --host {{host}} --port {{port}}`

- Ցուցադրել օգնությունը.:

`mysql_secure_installation --help`
