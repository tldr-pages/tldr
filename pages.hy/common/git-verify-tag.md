# git verify-tag

> Ստուգեք պիտակների GPG ստուգումը:.
> Եթե պիտակը ստորագրված չէ, սխալ կառաջանա:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-verify-tag>:.

- Ստուգեք պիտակները GPG ստորագրության համար.:

`git verify-tag {{tag1 optional_tag2 ...}}`

- Ստուգեք պիտակները GPG ստորագրության համար և ցուցադրեք մանրամասներ յուրաքանչյուր պիտակի համար.:

`git verify-tag {{tag1 optional_tag2 ...}} {{[-v|--verbose]}}`

- Ստուգեք պիտակները GPG ստորագրության համար և տպեք չմշակված մանրամասները.:

`git verify-tag {{tag1 optional_tag2 ...}} --raw`
