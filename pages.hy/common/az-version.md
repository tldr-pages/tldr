# az տարբերակ

> Ցույց տալ Azure CLI մոդուլների և ընդլայնումների ընթացիկ տարբերակը:.
> `azure-cli`-ի մի մասը (հայտնի է նաև որպես `az`):.
> Լրացուցիչ տեղեկություններ. <https://learn.microsoft.com/cli/azure/reference-index#az-version>:.

- Ցուցադրել Azure CLI մոդուլների և ընդլայնումների ընթացիկ տարբերակը JSON ձևաչափով.:

`az version`

- Ցույց տալ Azure CLI մոդուլների և ընդլայնումների ընթացիկ տարբերակը տվյալ ձևաչափով.:

`az version {{[-o|--output]}} {{json|table|tsv}}`
