# pip որոնում

> Փնտրեք Python փաթեթներ անունով կամ ամփոփումով:.
> Չի աշխատում PyPI-ի հետ; կարող է աշխատել այլ փաթեթի ինդեքսների հետ:.
> Լրացուցիչ տեղեկություններ. <https://pip.pypa.io/en/stable/cli/pip_search/>:.

- Փաթեթների որոնում՝ օգտագործելով հատուկ փաթեթի ինդեքս.:

`pip search {{[-i|--index]}} {{example.com}} {{query}}`
