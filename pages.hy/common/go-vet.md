#գնա անասնաբույժ

> Ստուգեք Go-ի աղբյուրի կոդը և զեկուցեք կասկածելի կոնստրուկցիաների մասին (օրինակ՝ ձեր Go-ի սկզբնաղբյուր ֆայլերը փակցնելու համար):.
> Go vet-ը վերադարձնում է ոչ զրոյական ելքի կոդը, եթե խնդիրներ հայտնաբերվեն. վերադարձնում է զրոյական ելքի կոդը, եթե խնդիրներ չեն հայտնաբերվել:.
> Լրացուցիչ տեղեկություններ. <https://pkg.go.dev/cmd/vet>:.

- Ստուգեք Go փաթեթը ընթացիկ գրացուցակում.:

`go vet`

- Ստուգեք Go փաթեթը նշված ճանապարհով.:

`go vet {{path/to/file_or_directory}}`

- Թվարկեք առկա ստուգումները, որոնք կարող են իրականացվել go vet-ի հետ.:

`go tool vet help`

- Դիտեք մանրամասները և դրոշները որոշակի ստուգման համար.:

`go tool vet help {{check_name}}`

- Ցուցադրել վիրավորական տողեր՝ գումարած `n` տող շրջապատող ենթատեքստ.:

`go vet -c={{n}}`

- Արդյունքների վերլուծություն և սխալներ JSON ձևաչափով.:

`go vet -json`
