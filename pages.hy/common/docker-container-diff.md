# դոկեր կոնտեյների տարբերություն

> Ստուգեք կոնտեյների ֆայլային համակարգի ֆայլերի կամ գրացուցակների փոփոխությունները:.
> Լրացուցիչ տեղեկություններ. <https://docs.docker.com/reference/cli/docker/container/diff/>:.

- Ստուգեք կոնտեյների փոփոխությունները, քանի որ այն ստեղծվել է.:

`docker {{[diff|container diff]}} {{container}}`

- Ցուցադրել օգնությունը.:

`docker {{[diff|container diff]}} --help`
