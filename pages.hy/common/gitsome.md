#հիանալի

> Տերմինալի վրա հիմնված ինտերֆեյս GitHub-ի համար, որը հասանելի է `gh` հրամանի միջոցով:.
> Այն նաև տրամադրում է ընտրացանկի ոճով ավտոմատ լրացման առաջարկներ `git` հրամանների համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/donnemartin/gitsome#index>:.

- Մուտքագրեք gitsome shell-ը (ըստ ցանկության), Git (և gh) հրամանների ավտոմատ լրացումը և ինտերակտիվ օգնությունը միացնելու համար.:

`gitsome`

- Կարգավորեք GitHub-ի ինտեգրումը ընթացիկ հաշվի հետ.:

`gh configure`

- Ցուցակեք ընթացիկ հաշվի ծանուցումները (ինչպես երևում է <https://github.com/notifications>-ում):

`gh notifications`

- Թվարկեք ընթացիկ հաշվի աստղանշված ռեպոները՝ զտված որոնման տվյալ տողով.:

`gh starred "{{python 3}}"`

- Դիտեք տվյալ GitHub պահոցի վերջին գործունեության հոսքը.:

`gh feed {{tldr-pages/tldr}}`

- Դիտեք GitHub-ի տվյալ օգտատիրոջ վերջին գործունեության հոսքը՝ օգտագործելով լռելյայն փեյջերը (օրինակ՝ `less`):

`gh feed {{torvalds}} {{[-p|--pager]}}`
