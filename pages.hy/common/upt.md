# վեր

> Միասնական ինտերֆեյս տարբեր օպերացիոն համակարգերում փաթեթներ կառավարելու համար, ինչպիսիք են Windows-ը, Linux-ի շատ բաշխումներ, macOS, FreeBSD և նույնիսկ Haiku:.
> Այն պահանջում է տեղադրել OS փաթեթի կառավարիչը:.
> Տես նաև՝ `flatpak`, `brew`, `scoop`, `apt`, `dnf`:.
> Լրացուցիչ տեղեկություններ. <https://github.com/sigoden/upt>:.

- Թարմացրեք առկա փաթեթների ցանկը.:

`upt update`

- Որոնել տվյալ փաթեթը.:

`upt search {{search_term}}`

- Ցույց տալ տեղեկատվություն փաթեթի համար.:

`upt info {{package}}`

- Տեղադրեք տվյալ փաթեթը.:

`upt install {{package}}`

- Հեռացնել տրված փաթեթը.:

`upt {{remove|uninstall}} {{package}}`

- Թարմացրեք բոլոր տեղադրված փաթեթները.:

`upt upgrade`

- Թարմացրեք տվյալ փաթեթը.:

`upt upgrade {{package}}`

- Տեղադրված փաթեթների ցանկ.:

`upt list`
