# DOOM

> Դասական բաց կոդով բումեր հրաձիգ, որը ներառում է մոդինգ և բազմախաղեր:.
> `zandronum-server`-ը կարող է փոխարինվել գրեթե ցանկացած աղբյուրի հետ:.
> Լրացուցիչ տեղեկություններ. <https://doomwiki.org/wiki/Source_port_parameters>:.

- Host co-op multiplayer.:

`{{zandronum-server}} -iwad {{wad}} +map {{map}} -host {{players}}`

- Host deathmatch multiplayer.:

`{{zandronum-server}} -iwad {{wad}} +map {{map}} -host {{players}} -deathmatch`
