# dolt ավելացնել

> Ավելացնել աղյուսակի բովանդակությունը Dolt փուլային աղյուսակների ցանկում:.
> Լրացուցիչ տեղեկություններ. <https://docs.dolthub.com/cli-reference/cli#dolt-add>:.

- Ավելացրեք աղյուսակ փուլային աղյուսակների ցանկին (աղյուսակի փուլ).:

`dolt add {{table}}`

- Բեմադրել բոլոր աղյուսակները.:

`dolt add {{[-A|--all]}}`
