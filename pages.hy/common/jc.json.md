# jc

> Փոխարկեք բազմաթիվ հրամանների ելքը JSON-ի:.
> Լրացուցիչ տեղեկություններ. <https://github.com/kellyjonbrazil/jc#usage>:.

- Խողովակի միջոցով հրամանի ելքը փոխարկեք JSON.:

`{{ifconfig}} | jc {{--ifconfig}}`

- Փոխակերպեք հրամանի ելքը JSON-ի կախարդական շարահյուսության միջոցով.:

`jc {{ifconfig}}`

- Արտադրեք գեղեցիկ JSON խողովակի միջոցով.:

`{{ifconfig}} | jc {{--ifconfig}} {{[-p|--pretty]}}`

- Արտադրեք գեղեցիկ JSON կախարդական շարահյուսության միջոցով.:

`jc {{[-p|--pretty]}} {{ifconfig}}`
