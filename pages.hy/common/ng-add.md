# ng ավելացնել

> Ավելացրեք և կազմաձևեք փաթեթներ ընթացիկ աշխատանքային տարածքի նախագծի համար:.
> Լրացուցիչ տեղեկություններ. <https://angular.dev/cli/add>:.

- Ավելացրեք փաթեթ ընթացիկ նախագծին.:

`ng add {{package}}`

- Ավելացնել բազմաթիվ փաթեթներ.:

`ng add {{package1 package2 ...}}`

- Ավելացնել փաթեթի որոշակի տարբերակ.:

`ng add {{package}}@{{version}}`

- Բաց թողնել հաստատման հուշումը.:

`ng add {{package}} --skip-confirmation`

- Անջատել ինտերակտիվ հուշումները.:

`ng add {{package}} --interactive false`

- Ցուցադրել ներքին գործառնությունների վերաբերյալ մանրամասն արդյունքը.:

`ng add {{package}} --verbose`

- Կատարեք չոր վազք՝ առանց որևէ փոփոխության.:

`ng add {{package}} {{[-d|--dry-run]}}`
