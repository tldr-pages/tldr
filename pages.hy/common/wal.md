# Wal

> Ստեղծեք գունային սխեմաներ՝ հիմնվելով պաստառի գերիշխող գույների վրա:.
> Լրացուցիչ տեղեկություններ. <https://github.com/dylanaraps/pywal/wiki/Getting-Started>:.

- Նախադիտեք գունային սխեման.:

`wal --preview {{image.png}}`

- Ստեղծեք գունային սխեման.:

`wal -i {{image.png}}`

- Ստեղծեք թեթև գունային սխեման.:

`wal -i {{image.png}} -l`

- Բաց թողնել աշխատասեղանի պաստառի կարգավորումը.:

`wal -i {{image.png}} -n`

- Բաց թողնել տերմինալի գույների կարգավորումը.:

`wal -i {{image.png}} -s`

- Վերականգնել նախկինում ստեղծված գունային սխեման և պաստառ.:

`wal -R`
