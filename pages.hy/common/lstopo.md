# lstopo

> Ցույց տալ համակարգի ապարատային տոպոլոգիան:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/lstopo>:.

- Ցուցադրել ամփոփված համակարգի տոպոլոգիան գրաֆիկական պատուհանում (կամ տպել կոնսոլում, եթե գրաֆիկական ցուցադրում չկա):

`lstopo`

- Ցուցադրել համակարգի ամբողջական տոպոլոգիան առանց ամփոփումների.:

`lstopo --no-factorize`

- Ցույց տվեք համակարգի ամփոփված տոպոլոգիան միայն ֆիզիկական ցուցանիշներով (այսինքն, ինչպես երևում է ՕՀ-ի կողմից).:

`lstopo {{[-p|--physical]}}`

- Նշված ձևաչափով ֆայլում գրեք համակարգի ամբողջական տոպոլոգիան.:

`lstopo --no-factorize {{[--of|--output-format]}} {{console|ascii|tex|fig|svg|pdf|ps|png|xml}} {{path/to/file}}`

- Արդյունք մոնոխրոմ կամ մոխրագույն մասշտաբով.:

`lstopo --palette {{none|grey}}`
