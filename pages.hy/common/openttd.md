# openttd

> «Transport Tycoon Deluxe» Microprose խաղի բաց կոդով կլոն:.
> Լրացուցիչ տեղեկություններ. <https://wiki.openttd.org/en/Manual/Command%20line>:.

- Սկսեք նոր խաղ.:

`openttd -g`

- Բեռնել խնայող խաղը սկզբում.:

`openttd -g {{path/to/file}}`

- Սկսեք նշված պատուհանի լուծաչափով.:

`openttd -r {{1920x1080}}`

- Սկսեք հատուկ կազմաձևման ֆայլից.:

`openttd -c {{path/to/file}}`

- Սկսեք ընտրված տեսանյութի, ձայնի և երաժշտության վարորդներից.:

`openttd -v {{video_driver}} -s {{sound_driver}} -m {{music_driver}}`

- Սկսեք հատուկ սերվեր, որը պատառաքաղված է հետին պլանում.:

`openttd -f -D {{host}}:{{port}}`

- Միացեք գաղտնաբառով սերվերին.:

`openttd -n {{host}}:{{port}}#{{player_name}} -p {{password}}`
