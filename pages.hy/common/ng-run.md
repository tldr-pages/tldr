# նգ վազք

> Գործարկել Architect-ի թիրախը կամընտիր մաքսային շինարարի կոնֆիգուրացիայով:.
> Լրացուցիչ տեղեկություններ. <https://angular.dev/cli/run>:.

- Գործարկել Architect-ի թիրախը՝ օգտագործելով `project:target:configuration` ձևաչափը՝:

`ng run {{project_name}}:{{build|serve|test|lint|...}}:{{development|production|...}}`
