# կենդանի սերվեր

> Զարգացման պարզ HTTP սերվեր՝ ուղիղ վերաբեռնման հնարավորությամբ:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tapio/live-server>:.

- Ծառայեք `index.html` ֆայլ և վերաբեռնեք փոփոխությունները՝:

`live-server`

- Նշեք մի նավահանգիստ (կանխադրվածը 8080 է), որտեղից պետք է սպասարկվի ֆայլը.:

`live-server --port={{8081}}`

- Նշեք տրված ֆայլը ծառայելու համար.:

`live-server --open={{about.html}}`

- ROUTE դեպի URL բոլոր հարցումները վստահված անձի կողմից.:

`live-server --proxy={{/}}:{{http:localhost:3000}}`
