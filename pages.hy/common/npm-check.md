# npm-ստուգում

> Ստուգեք հնացած, սխալ և չօգտագործված npm փաթեթի կախվածությունները:.
> Լրացուցիչ տեղեկություններ. <https://github.com/dylang/npm-check>:.

- Ցուցադրել հնացած, սխալ և չօգտագործված կախվածությունների հաշվետվություն.:

`npm-check`

- Ինտերակտիվ կերպով թարմացրեք հնացած փաթեթները.:

`npm-check {{[-u|--update]}}`

- Թարմացրեք ամեն ինչ առանց հուշելու.:

`npm-check {{[-y|--update-all]}}`

- Մի ստուգեք չօգտագործված փաթեթները.:

`npm-check {{[-s|--skip-unused]}}`
