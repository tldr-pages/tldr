#զոքսիդ

> Հետևեք ամենահաճախ օգտագործվող գրացուցակներին:.
> Օգտագործում է վարկանիշային ալգորիթմ՝ նավարկելու դեպի լավագույն համընկնում:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/zoxide>:.

- Գնացեք ամենաբարձր վարկանիշ ունեցող գրացուցակը, որը պարունակում է `string` անունը.:

`zoxide query string`

- Գնացեք ամենաբարձր վարկանիշ ունեցող գրացուցակը, որը պարունակում է `string1`, ապա `string2`:

`zoxide query string1 string2`

- Սկսեք ինտերակտիվ գրացուցակի որոնում (պահանջվում է `fzf`):

`zoxide query {{[-i|--interactive]}}`

- Ավելացրեք գրացուցակ կամ ավելացրեք դրա վարկանիշը.:

`zoxide add {{path/to/directory}}`

- Հեռացրեք գրացուցակը `zoxide`-ի տվյալների բազայից.:

`zoxide remove {{path/to/directory}}`

- Ստեղծեք կեղևի կոնֆիգուրացիա հրամանների այլանունների համար (`z`, `zi`):

`zoxide init {{bash|elvish|fish|nushell|posix|powershell|tcsh|xonsh|zsh}}`
