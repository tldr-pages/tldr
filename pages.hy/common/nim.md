# նիմ

> Nim կոմպիլյատորը:.
> Մշակել, կազմել և կապել Nim լեզվի սկզբնաղբյուր ֆայլերը:.
> Լրացուցիչ տեղեկություններ. <https://nim-lang.org/docs/nimc.html>:.

- Կազմել աղբյուրի ֆայլ.:

`nim {{[c|compile]}} {{path/to/file.nim}}`

- Կազմեք և գործարկեք աղբյուրի ֆայլը.:

`nim {{[c|compile]}} {{[-r|--run]}} {{path/to/file.nim}}`

- Կազմեք սկզբնաղբյուր ֆայլ՝ միացված թողարկման օպտիմալացումներով.:

`nim {{[c|compile]}} {{[-d|--define]}}:release {{path/to/file.nim}}`

- Ստեղծեք թողարկման երկուական օպտիմիզացված ֆայլի ցածր չափի համար.:

`nim {{[c|compile]}} {{[-d|--define]}}:release --opt:size {{path/to/file.nim}}`

- Ստեղծեք HTML փաստաթղթեր մոդուլի համար (արդյունքը կտեղադրվի ընթացիկ գրացուցակում).:

`nim doc {{path/to/file.nim}}`

- Ստուգեք ֆայլը շարահյուսության և իմաստաբանության համար.:

`nim check {{path/to/file.nim}}`
