# zeroclaw deemon

> Սկսեք ZeroClaw-ի ամբողջական ինքնավար գործարկման ժամանակը (դարպաս + ալիք + սրտի բաբախյուն):.
> Լրացուցիչ տեղեկություններ. <https://github.com/zeroclaw-labs/zeroclaw#quick-start>:.

- Սկսեք դեյմոնը լռելյայն նավահանգստում (8080):

`zeroclaw daemon`

- Սկսեք դեյմոնը կոնկրետ նավահանգստում.:

`zeroclaw daemon {{[-p|--port]}} {{8080}}`

- Սկսեք դեյմոնը պատահական հասանելի նավահանգստի վրա.:

`zeroclaw daemon {{[-p|--port]}} 0`

- Սկսեք դևոնը կոնկրետ հյուրընկալողի վրա.:

`zeroclaw daemon --host {{0.0.0.0}} {{[-p|--port]}} {{8080}}`

- Ցուցադրել օգնությունը.:

`zeroclaw daemon {{[-h|--help]}}`
