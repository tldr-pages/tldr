# վիճակագրություն

> Ցուցադրել ֆայլի և ֆայլային համակարգի տեղեկատվությունը:.
> Տես նաև՝ `file`:.
> Լրացուցիչ տեղեկություններ. <https://www.gnu.org/software/coreutils/manual/html_node/stat-invocation.html>:.

- Ցուցադրել հատկություններ որոշակի ֆայլի վերաբերյալ, ինչպիսիք են չափը, թույլտվությունները, ստեղծման ամսաթիվը և մուտքի ամսաթիվը, ի թիվս այլոց.:

`stat {{path/to/file}}`

- Ցուցադրել հատկությունները կոնկրետ ֆայլի վերաբերյալ՝ ցույց տալով միայն չմշակված արդյունքների տվյալները՝ առանց պիտակների.:

`stat {{[-t|--terse]}} {{path/to/file}}`

- Ցուցադրել տեղեկատվություն ֆայլային համակարգի մասին, որտեղ գտնվում է որոշակի ֆայլ.:

`stat {{[-f|--file-system]}} {{path/to/file}}`

- Ցույց տալ միայն օկտալ ֆայլի թույլտվությունները.:

`stat {{[-c|--format]}} "%a %n" {{path/to/file}}`

- Ցույց տալ որոշակի ֆայլի սեփականատիրոջը և խմբին.:

`stat {{[-c|--format]}} "%U %G" {{path/to/file}}`

- Ցույց տալ որոշակի ֆայլի չափը բայթերով.:

`stat {{[-c|--format]}} "%s %n" {{path/to/file}}`
