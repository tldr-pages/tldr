# git reflog

> Ցույց տվեք տեղական հղումների փոփոխությունների մատյան, ինչպիսիք են HEAD-ը, մասնաճյուղերը կամ պիտակները:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-reflog>:.

- Ցուցադրել reflog-ը `HEAD`-ի համար՝:

`git reflog`

- Ցույց տալ ռեֆլոգը տվյալ ճյուղի համար.:

`git reflog {{branch_name}}`

- Ցուցադրել միայն 5 վերջին գրառումները reflog-ում.:

`git reflog {{[-n|--max-count]}} 5`
