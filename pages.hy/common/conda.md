#կոնդա

> Փաթեթի, կախվածության և շրջակա միջավայրի կառավարում ցանկացած ծրագրավորման լեզվի համար:.
> Որոշ ենթահրամաններ, ինչպիսիք են `create`-ն, ունեն իրենց օգտագործման փաստաթղթերը:.
> Տես նաև՝ `mamba`:.
> Լրացուցիչ տեղեկություններ. <https://docs.conda.io/projects/conda/en/latest/commands/index.html>:.

- Ստեղծեք նոր միջավայր՝ դրա մեջ տեղադրելով անվանված փաթեթներ.:

`conda create {{[-n|--name]}} {{environment_name}} {{python=3.9 matplotlib}}`

- Թվարկեք բոլոր միջավայրերը.:

`conda info {{[-e|--envs]}}`

- Ակտիվացրեք միջավայրը.:

`conda activate {{environment_name}}`

- Ապաակտիվացնել միջավայրը.:

`conda deactivate`

- Ջնջել միջավայրը (հեռացնել բոլոր փաթեթները).:

`conda remove {{[-n|--name]}} {{environment_name}} --all`

- Տեղադրեք փաթեթներ ընթացիկ միջավայրում.:

`conda install {{python=3.4 numpy}}`

- Ցուցակեք ներկայումս տեղադրված փաթեթները ընթացիկ միջավայրում.:

`conda list`

- Ջնջել չօգտագործված փաթեթներն ու քեշերը.:

`conda clean {{[-a|--all]}}`
