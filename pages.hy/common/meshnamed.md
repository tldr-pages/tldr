# meshname

> Բաշխված անվանման համակարգ IPv6 ցանցային ցանցերի համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/zhoreeq/meshname>:.

- Սկսեք տեղական meshname DNS սերվեր.:

`meshnamed`

- Փոխակերպեք IPv6 հասցեն ցանցի անունի.:

`meshnamed -getname {{200:6fc8:9220:f400:5cc2:305a:4ac6:967e}}`

- Փոխարկեք ցանցի անունը IPv6 հասցեի.:

`meshnamed -getip {{aiag7sesed2aaxgcgbnevruwpy}}`
