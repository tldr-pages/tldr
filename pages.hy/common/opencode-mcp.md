# բաց կոդ mcp

> Կառավարեք MCP (Model Context Protocol) սերվերները:.
> Լրացուցիչ տեղեկություններ. <https://opencode.ai/docs/cli#mcp>:.

- Ցուցակեք MCP սերվերները և դրանց կարգավիճակը.:

`opencode mcp list`

- Ինտերակտիվ կերպով ավելացրեք MCP սերվեր.:

`opencode mcp add`

- Նույնականացում OAuth-ով միացված MCP սերվերով.:

`opencode mcp auth {{server_name}}`

- Հեռացրեք OAuth հավատարմագրերը MCP սերվերի համար.:

`opencode mcp logout {{server_name}}`

- Վրիպազերծել OAuth կապը MCP սերվերի համար.:

`opencode mcp debug {{server_name}}`

- Ցուցադրել օգնությունը.:

`opencode mcp {{[-h|--help]}}`
