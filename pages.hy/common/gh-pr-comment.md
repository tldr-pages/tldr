# gh pr մեկնաբանություն

> Ավելացրեք մեկնաբանություն GitHub-ի ձգման հարցումին:.
> Լրացուցիչ տեղեկություններ. <https://cli.github.com/manual/gh_pr_comment>:.

- Մեկնաբանեք ընթացիկ մասնաճյուղի ձգման խնդրանքը.:

`gh pr comment {{[-b|--body]}} "{{LGTM}}"`

- Մեկնաբանեք հատուկ ձգողականության խնդրանքը.:

`gh pr comment {{123}} {{[-b|--body]}} "{{Thanks!}}"`

- Մեկնաբանություն ֆայլից.:

`gh pr comment {{123}} {{[-F|--body-file]}} {{path/to/file.txt}}`

- Բացեք խմբագրիչը՝ բազմատող մեկնաբանություն գրելու համար.:

`gh pr comment {{123}}`
