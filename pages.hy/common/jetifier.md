# jetifier

> Jetifier AndroidX-ի անցումային գործիք npm ձևաչափով, react-native համատեղելի ոճով:.
> Լրացուցիչ տեղեկություններ. <https://github.com/mikehardy/jetifier>:.

- Տեղափոխել նախագծի կախվածությունը AndroidX ձևաչափին.:

`jetifier`

- Տեղափոխեք նախագծի կախվածությունները AndroidX ձևաչափից.:

`jetifier reverse`
