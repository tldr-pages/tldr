# վազում-օգնություն

> Ստացեք հրահանգներ Zsh հրամանների համար:.
> Նշում. այս հրամանը լռելյայնորեն `man`-ի կեղծանունն է:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/zshcontrib>:.

- Ակտիվացրեք Zsh-ի `run-help`-ը, որն ավելացնում է հրահանգներ Zsh ներկառուցվածների համար.:

`unalias run-help && autoload run-help`

- Ցուցադրել բոլոր Zsh-ի հատուկ թեմաները.:

`run-help`

- Ցուցադրել օգնություն հրամանի համար.:

`run-help {{command}}`
