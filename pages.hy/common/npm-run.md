# npm վազում

> Գործարկել սցենար:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-run/>:.

- Ցուցակեք մատչելի սցենարները.:

`npm run`

- Գործարկել սցենար.:

`npm run {{script_name}}`

- Փաստարկներ փոխանցեք սցենարին.:

`npm run {{script_name}} -- {{argument}} {{--option}}`

- Գործարկեք `start` անունով սկրիպտը:

`npm start`

- Գործարկել `stop` անունով սկրիպտը:

`npm stop`

- Գործարկեք `restart` անունով սկրիպտը:

`npm restart`

- Գործարկել `test` անունով սկրիպտը:

`npm {{[t|test]}}`
