# git imerge

> Կատարեք միաձուլում կամ վերահիմնավորում երկու Git ճյուղերի միջև աստիճանաբար:.
> Մասնաճյուղերի միջև կոնֆլիկտները դիտարկվում են մինչև զույգ անհատական ​​պարտավորություններ՝ հակամարտությունների լուծումը պարզեցնելու համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/mhagger/git-imerge>:.

- Սկսեք imerge-ի վրա հիմնված վերահիմնավորումը (նախ ստուգեք այն ճյուղը, որը պետք է վերահիմնվի):

`git imerge rebase {{branch_to_rebase_onto}}`

- Սկսեք միաձուլման վրա հիմնված միաձուլումը (նախ ստուգեք այն ճյուղը, որին պետք է միավորել).:

`git imerge merge {{branch_to_be_merged}}`

- Ցույց տալ ընթացքի մեջ գտնվող միաձուլման կամ վերաբաժանման ASCII դիագրամը.:

`git imerge diagram`

- Շարունակեք միացման գործողությունը կոնֆլիկտները լուծելուց հետո (`git add` կոնֆլիկտային ֆայլերը, նախ).:

`git imerge continue --no-edit`

- Ամփոփեք միացման գործողությունը, բոլոր հակամարտությունները լուծվելուց հետո.:

`git imerge finish`

- Դադարեցրեք միացման գործողությունը և վերադարձեք նախորդ մասնաճյուղ.:

`git imerge remove && git checkout {{previous_branch}}`
