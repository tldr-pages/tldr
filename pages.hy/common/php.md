# php

> PHP հրամանի տողի ինտերֆեյս:.
> Լրացուցիչ տեղեկություններ. <https://www.php.net/manual/en/features.commandline.options.php>:.

- Վերլուծեք և գործարկեք PHP սկրիպտը.:

`php {{path/to/file}}`

- Ստուգեք շարահյուսությունը (այսինքն, [l]int) PHP սցենարի վրա.:

`php {{[-l|--syntax-check]}} {{path/to/file}}`

- Գործարկեք PHP-ն inter[a]ակտիվորեն.:

`php {{[-a|--interactive]}}`

- Գործարկեք PHP կոդը (Ծանոթագրություններ. Մի օգտագործեք `<? ?>` պիտակները, կրկնակի չակերտներից խուսափեք հետշեղով):

`php {{[-r|--run]}} "{{code}}"`

- Գործարկեք PHP ներկառուցված վեբ սերվեր ընթացիկ գրացուցակում.:

`php {{[-S|--server]}} {{host}}:{{port}}`

- Ցուցակեք տեղադրված PHP ընդարձակումները.:

`php {{[-m|--modules]}}`

- Ցուցադրել տեղեկատվություն ընթացիկ PHP կազմաձևման մասին.:

`php {{[-i|--info]}}`

- Ցուցադրել տեղեկատվություն որոշակի գործառույթի մասին.:

`php {{[--rf|--rfunction]}} {{function_name}}`
