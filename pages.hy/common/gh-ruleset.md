# gh կանոնակարգ

> Կառավարեք GitHub պահեստի կանոնները:.
> Լրացուցիչ տեղեկություններ. <https://cli.github.com/manual/gh_ruleset>:.

- Թվարկեք բոլոր կանոնները ընթացիկ պահոցի համար.:

`gh {{[rs|ruleset]}} {{[ls|list]}}`

- Թվարկեք բոլոր կանոնները կոնկրետ կազմակերպության համար.:

`gh {{[rs|ruleset]}} {{[ls|list]}} {{[-o|--org]}} {{organization_name}}`

- Ստուգեք ընթացիկ մասնաճյուղի նկատմամբ կիրառվող կանոնները.:

`gh {{[rs|ruleset]}} check`

- Ստուգեք կանոնները, որոնք վերաբերում են որոշակի մասնաճյուղին մեկ այլ պահեստում.:

`gh {{[rs|ruleset]}} check {{branch_name}} {{[-R|--repo]}} {{owner}}/{{repository}}`

- Ինտերակտիվ կերպով ընտրեք և դիտեք կանոնների հավաքածու ընթացիկ պահեստի համար.:

`gh {{[rs|ruleset]}} view`

- Դիտեք որոշակի կանոններ իր ID-ով.:

`gh {{[rs|ruleset]}} view {{ruleset_id}}`

- Դիտեք կազմակերպության մակարդակի կանոնակարգն իր ID-ով.:

`gh {{[rs|ruleset]}} view {{ruleset_id}} {{[-o|--org]}} {{organization_name}}`

- Բրաուզերում բացեք կանոնների ցանկը կոնկրետ պահոցի համար.:

`gh {{[rs|ruleset]}} {{[ls|list]}} {{[-R|--repo]}} {{owner}}/{{repository}} {{[-w|--web]}}`
