# tailscale up

> Միացրեք հաճախորդին Tailscale ցանցին:.
> Նշում. Գործարկեք `sudo tailscale set --operator $USER`-ը, որպեսզի ընթացիկ օգտվողին թույլ տա գործարկել այս հրամանները:.
> Այստեղ նկարագրված բոլոր տարբերակները կարող են փոխվել ավելի ուշ՝ օգտագործելով `tailscale set --option argument`: Օգտագործեք `--option=false`՝ արգումենտներ չպահանջող տարբերակներն անջատելու համար:.
> Լրացուցիչ տեղեկություններ՝ <https://tailscale.com/kb/1080/cli/#up>:.

- Միացեք Tailscale-ին.:

`tailscale up`

- Միացեք և առաջարկեք ընթացիկ մեքենան լինել ինտերնետ տրաֆիկի ելքի հանգույց.:

`tailscale up --advertise-exit-node`

- Միացեք ինտերնետ տրաֆիկի հատուկ հանգույցի միջոցով.:

`tailscale up --exit-node {{exit_node_ip}}`

- Միացրեք և արգելափակեք մուտքային կապերը ընթացիկ հանգույցին.:

`tailscale up --shields-up`

- Միացեք և մի ընդունեք DNS-ի կազմաձևումը ադմինիստրատորի վահանակից (կանխադրված է `true`):

`tailscale up --accept-dns=false`

- Միացրեք և կարգավորեք Tailscale-ը որպես ենթացանցային երթուղիչ.:

`tailscale up --advertise-routes {{10.0.0.0/24,10.0.1.0/24,...}}`

- Միացրեք և ընդունեք ենթացանցային երթուղիները Tailscale-ից.:

`tailscale up --accept-routes`

- Վերականգնել չսահմանված կարգավորումները իրենց լռելյայն արժեքներին և միացնել.:

`tailscale up --reset`
