#git mr

> Ստուգեք GitLab-ի միաձուլման հարցումները տեղում:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-mr>:.

- Ստուգեք կոնկրետ միաձուլման հարցում.:

`git mr {{mr_number}}`

- Ստուգեք միաձուլման հարցումը որոշակի հեռակառավարման վահանակից.:

`git mr {{mr_number}} {{remote}}`

- Ստուգեք միավորման հարցումն իր URL-ից.:

`git mr {{url}}`

- Մաքրել միաձուլման հարցումների հին մասնաճյուղերը.:

`git mr clean`
