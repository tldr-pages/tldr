# կոմպոզիտոր-պահանջում-ստուգիչ

> Վերլուծել կոմպոզիտորի կախվածությունը փափուկ կախվածությունների համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/maglnet/ComposerRequireChecker>:.

- Վերլուծեք Composer JSON ֆայլը.:

`composer-require-checker check {{path/to/composer.json}}`

- Վերլուծեք Composer JSON ֆայլը հատուկ կազմաձևով.:

`composer-require-checker check --config-file {{path/to/config.json}} {{path/to/composer.json}}`
