# spfquery

> Հարցրեք ուղարկողի քաղաքականության շրջանակի գրառումները՝ էլփոստ ուղարկողներին վավերացնելու համար:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/spfquery>:.

- Ստուգեք, արդյոք IP հասցեին թույլատրվում է նամակ ուղարկել նշված էլ. փոստի հասցեից.:

`spfquery -ip {{8.8.8.8}} -sender {{sender@example.com}}`

- Միացնել վրիպազերծման ելքը.:

`spfquery -ip {{8.8.8.8}} -sender {{sender@example.com}} --debug`
