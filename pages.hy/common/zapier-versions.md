# zapier տարբերակ

> Թվարկե՛ք Zapier ինտեգրման տարբերակները:.
> Լրացուցիչ տեղեկություններ. <https://github.com/zapier/zapier-platform/blob/main/packages/cli/docs/cli.md#validate>:.

- Թվարկեք ինտեգրման տարբերակները.:

`zapier versions`

- Ներառեք հնացած տարբերակները.:

`zapier versions {{[-a|--all]}}`

- Ցուցադրել վրիպազերծման լրացուցիչ ելք.:

`zapier versions {{[-d|--debug]}}`

- Թվարկեք տարբեր ելքային կառուցվածք ունեցող տարբերակները.:

`zapier versions {{[-f|--format]}} {{plain|json|raw|row|table}}`
