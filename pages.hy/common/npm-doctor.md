# npm բժիշկ

> Ստուգեք npm միջավայրի առողջությունը:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-doctor/>:.

- Գործարկեք առողջության բոլոր կանխադրված ստուգումները `npm`-ի համար՝:

`npm doctor`

- Ստուգեք կապը `npm` ռեեստրի հետ.:

`npm doctor connection`

- Ստուգեք Node.js-ի և `npm`-ի տարբերակները, որոնք օգտագործվում են.:

`npm doctor versions`

- Ստուգեք թույլտվությունների հետ կապված խնդիրներ `npm` դիրեկտորիաների և քեշի հետ:

`npm doctor permissions`

- Վավերացրեք քեշավորված փաթեթի ֆայլերը և ստուգիչ գումարները.:

`npm doctor cache`
