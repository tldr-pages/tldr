# az բարելավում

> Թարմացրեք Azure CLI-ն և ընդլայնումները:.
> `azure-cli`-ի մի մասը (հայտնի է նաև որպես `az`):.
> Լրացուցիչ տեղեկություններ. <https://learn.microsoft.com/cli/azure/reference-index#az-upgrade>:.

- Թարմացրեք Azure CLI:

`az upgrade`

- Թարմացրեք Azure CLI-ն և ընդարձակումները.:

`az upgrade --all`

- Թարմացրեք Azure CLI-ն և ընդարձակումները՝ առանց հաստատման պահանջելու.:

`az upgrade --all {{[-y|--yes]}}`
