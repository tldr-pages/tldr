# pyats տարբերակ

> Դիտեք և թարմացրեք pyATS-ի տեղադրումը:.
> Լրացուցիչ տեղեկություններ. <https://developer.cisco.com/pyats/>:.

- Ցուցադրել բոլոր փաթեթների տարբերակը.:

`pyats version check`

- Ցուցադրել հնացած փաթեթներ.:

`pyats version check --outdated`

- Թարմացրեք փաթեթները ամենավերջին տարբերակին.:

`pyats version update`

- Թարմացրեք կամ իջեցրեք փաթեթները որոշակի տարբերակի.:

`pyats version update {{version}}`
