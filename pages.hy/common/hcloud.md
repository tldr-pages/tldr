# hcloud

> Ցույց տվեք, թե ինչպես օգտագործել CLI-ը Hetzner Cloud-ի համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/hetznercloud/cli/blob/main/docs/reference/manual/hcloud.md>:.

- Ցույց տալ առկա հրամաններն ու դրոշները.:

`hcloud`

- Ցույց տալ հասանելի հրամաններն ու դրոշները `hcloud` համատեքստերի համար.:

`hcloud context`

- Ցուցադրել օգնությունը.:

`hcloud -h`
