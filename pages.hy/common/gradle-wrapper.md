# gradle փաթաթան

> Ստեղծեք Gradle wrapper ֆայլերը նախագծի համար:.
> Լրացուցիչ տեղեկություններ. <https://docs.gradle.org/current/userguide/gradle_wrapper.html>:.

- Ստեղծեք փաթաթան ընթացիկ Gradle տարբերակով.:

`gradle wrapper`

- Ստեղծեք փաթաթան հատուկ Gradle տարբերակով.:

`gradle wrapper --gradle-version {{8.5}}`

- Ստեղծեք փաթաթան հատուկ բաշխման տեսակով.:

`gradle wrapper --distribution-type {{bin|all}}`

- Ստեղծեք փաթաթան՝ օգտագործելով հատուկ բաշխման URL.:

`gradle wrapper --gradle-distribution-url {{url}}`
