# pnpm աուդիտ

> Սկանավորեք նախագծի կախվածությունները:.
> Ստուգեք տեղադրված փաթեթների հետ կապված անվտանգության հայտնի խնդիրները:.
> Լրացուցիչ տեղեկություններ. <https://pnpm.io/cli/audit>:.

- Բացահայտեք ծրագրի խոցելիությունը.:

`pnpm audit`

- Ավտոմատ կերպով շտկել խոցելիությունները.:

`pnpm audit fix`

- Ստեղծեք անվտանգության հաշվետվություն JSON ձևաչափով.:

`pnpm audit --json > {{path/to/audit-report.json}}`

- Աուդիտի ենթարկեք միայն մշակողների կախվածությունները.:

`pnpm audit {{[-D|--dev]}}`

- Աուդիտ միայն արտադրական կախվածությունները.:

`pnpm audit {{[-P|--prod]}}`

- Բացառել կամընտիր կախվածությունները աուդիտից.:

`pnpm audit --no-optional`

- Անտեսեք ռեեստրի սխալները աուդիտի գործընթացում.:

`pnpm audit --ignore-registry-errors`

- Զտեք խորհուրդներն ըստ խստության (ցածր, չափավոր, բարձր, կրիտիկական).:

`pnpm audit --audit-level {{severity}}`
