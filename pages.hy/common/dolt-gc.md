# dolt gc

> Փնտրեք պահեստում այնպիսի տվյալներ, որոնք այլևս չեն հղում և այլևս անհրաժեշտ չեն:.
> Լրացուցիչ տեղեկություններ. <https://docs.dolthub.com/cli-reference/cli#dolt-gc>:.

- Մաքրել չհղված տվյալները պահոցից.:

`dolt gc`

- Սկսել ավելի արագ, բայց ավելի քիչ մանրակրկիտ աղբահանության գործընթաց.:

`dolt gc {{[-s|--shallow]}}`
