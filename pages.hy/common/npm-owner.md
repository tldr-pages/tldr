# npm սեփականատեր

> Կառավարեք հրապարակված փաթեթների սեփականությունը:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-owner/>:.

- Ավելացրեք նոր օգտվող որպես փաթեթի պահպանող.:

`npm owner add {{username}} {{package_name}}`

- Հեռացրեք օգտվողին փաթեթի սեփականատերերի ցանկից.:

`npm owner rm {{username}} {{package_name}}`

- Նշեք փաթեթի բոլոր սեփականատերերը.:

`npm owner ls {{package_name}}`
