# բաց կոդով վեբ

> Սկսեք OpenCode սերվերը վեբ ինտերֆեյսով:.
> Լրացուցիչ տեղեկություններ. <https://opencode.ai/docs/cli#web>:.

- Սկսեք վեբ ինտերֆեյսը լռելյայն նավահանգստում.:

`opencode web`

- Սկսեք կոնկրետ նավահանգստից.:

`opencode web --port {{8080}}`

- Սկսեք կոնկրետ հյուրընկալողի անունով.:

`opencode web --hostname {{0.0.0.0}}`

- Միացնել mDNS ծառայության հայտնաբերումը.:

`opencode web --mdns`

- Ցուցադրել օգնությունը.:

`opencode web {{[-h|--help]}}`
