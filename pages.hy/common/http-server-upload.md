# http-սերվեր-վերբեռնում

> Զրոյական կոնֆիգուրացիայի HTTP սերվեր, որն ապահովում է թեթև ինտերֆեյս ֆայլեր վերբեռնելու համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/crycode-de/http-server-upload>:.

- Գործարկեք HTTP սերվերը լռելյայն նավահանգստի վրա՝ ֆայլերը ընթացիկ գրացուցակ վերբեռնելու համար.:

`http-server-upload`

- Սկսեք HTTP սերվեր ՄԲ-ով վերբեռնումների համար սահմանված առավելագույն թույլատրելի չափով (կանխադրված մինչև 200 ՄԲ):

`MAX_FILE_SIZE={{size_in_megabytes}} http-server-upload`

- Գործարկեք HTTP սերվեր որոշակի նավահանգստի վրա՝ ֆայլերը ընթացիկ գրացուցակ վերբեռնելու համար.:

`PORT={{port}} http-server-upload`

- Սկսեք HTTP սերվեր՝ բեռնված ֆայլերը պահելով որոշակի գրացուցակում.:

`UPLOAD_DIR={{path/to/directory}} http-server-upload`

- Սկսեք HTTP սերվեր՝ օգտագործելով հատուկ գրացուցակ՝ բեռնման գործընթացում ֆայլերը ժամանակավորապես պահելու համար.:

`UPLOAD_TMP_DIR={{path/to/directory}} http-server-upload`

- Սկսեք HTTP սերվեր, որը ընդունում է վերբեռնումներ HTTP գրառման մեջ հատուկ նշանային դաշտով.:

`TOKEN={{secret}} http-server-upload`
