# zeroclaw հմտություններ

> Կառավարեք օգտագործողի կողմից սահմանված հմտությունները ZeroClaw-ի համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/zeroclaw-labs/zeroclaw#quick-start>:.

- Տեղադրված հմտությունների ցուցակ.:

`zeroclaw skills list`

- Տեղադրեք հմտություն GitHub URL-ից.:

`zeroclaw skills install https://github.com/{{user}}/{{repo}}`

- Տեղադրեք հմտություն տեղական ճանապարհից.:

`zeroclaw skills install {{path/to/skill}}`

- Հեռացնել տեղադրված հմտությունը.:

`zeroclaw skills remove {{skill_name}}`

- Ցուցադրել օգնությունը.:

`zeroclaw skills {{[-h|--help]}}`
