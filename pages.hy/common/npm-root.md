# npm արմատ

> Ցուցադրել ուղին դեպի `node_modules` գրացուցակ:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-root/>:.

- Ցուցադրել ուղին դեպի տեղական `node_modules` գրացուցակը.:

`npm root`

- Ցուցադրել ուղին դեպի համաշխարհային `node_modules` գրացուցակ.:

`npm root {{[-g|--global]}}`
