# բաց կոդ ներմուծում

> Ներմուծեք նստաշրջանի տվյալները JSON ֆայլից կամ URL-ից:.
> Լրացուցիչ տեղեկություններ. <https://opencode.ai/docs/cli#import>:.

- Ներմուծում JSON ֆայլից.:

`opencode import {{path/to/session}}.json`

- Ներմուծում բաժնետոմսերի URL-ից.:

`opencode import {{url}}`

- Ցուցադրել օգնությունը.:

`opencode import {{[-h|--help]}}`
