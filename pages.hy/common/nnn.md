# nnn

> Ինտերակտիվ տերմինալային ֆայլերի կառավարիչ և սկավառակի օգտագործման անալիզատոր:.
> Լրացուցիչ տեղեկություններ. <https://github.com/jarun/nnn/wiki/Usage#program-options>:.

- Բացեք ընթացիկ գրացուցակը (կամ նշեք մեկը որպես առաջին փաստարկ).:

`nnn`

- Սկսեք [d] մանրամասն ռեժիմով.:

`nnn -d`

- Ցույց տալ [H] նույնացված ֆայլերը.:

`nnn -H`

- Բացեք գոյություն ունեցող [b]էջանիշը (սահմանված է `$NNN_BMS` միջավայրի փոփոխականում).:

`nnn -b {{bookmark_name}}`

- Տեսակավորել ֆայլերը [a]նախնական սկավառակի օգտագործման / [d]disk օգտագործման / [e]extension / [r]everse / [s]ize / [t]time / [v]ersion:

`nnn -T {{a|d|e|r|s|t|v}}`

- Բացեք ձեր ընտրած ֆայլը: Ընտրեք ֆայլը, այնուհետև սեղմեք `<o>` և մուտքագրեք ծրագիր՝ ֆայլը բացելու համար.:

`nnn -o`
