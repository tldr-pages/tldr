# git rm

> Հեռացրեք ֆայլերը պահեստի ինդեքսից և տեղական ֆայլային համակարգից:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-rm>:.

- Հեռացրեք ֆայլը պահեստի ինդեքսից և ֆայլային համակարգից.:

`git rm {{path/to/file}}`

- Հեռացրեք գրացուցակը [r] էկուրսիվ կերպով.:

`git rm -r {{path/to/directory}}`

- Հեռացրեք ֆայլը պահեստի ինդեքսից, բայց պահեք այն անձեռնմխելի տեղում.:

`git rm --cached {{path/to/file}}`

- Կրճատել պահոցը միայն `.git` գրացուցակով՝:

`git rm -r .`
