# gh թողարկում է ստեղծել

> Ստեղծեք GitHub-ի խնդիրները պահեստում:.
> Լրացուցիչ տեղեկություններ. <https://cli.github.com/manual/gh_issue_create>:.

- Ստեղծեք նոր թողարկում ընթացիկ պահեստի դեմ ինտերակտիվ կերպով.:

`gh issue {{[new|create]}}`

- Ստեղծեք նոր թողարկում `bug` պիտակով ինտերակտիվ կերպով.:

`gh issue {{[new|create]}} {{[-l|--label]}} "bug"`

- Ստեղծեք նոր թողարկում ինտերակտիվ կերպով և հանձնարարեք այն նշված օգտվողներին.:

`gh issue {{[new|create]}} {{[-a|--assignee]}} {{user1,user2,...}}`

- Ստեղծեք նոր թողարկում վերնագրով և տեքստով և այն վերագրեք ընթացիկ օգտագործողին.:

`gh issue {{[new|create]}} {{[-t|--title]}} "{{title}}" {{[-b|--body]}} "{{body}}" {{[-a|--assignee]}} "@me"`

- Ստեղծեք նոր թողարկում ինտերակտիվ կերպով՝ կարդալով հիմնական տեքստը ֆայլից.:

`gh issue {{[new|create]}} {{[-F|--body-file]}} {{path/to/file}}`

- Ստեղծեք նոր խնդիր լռելյայն վեբ դիտարկիչում.:

`gh issue {{[new|create]}} {{[-w|--web]}}`

- Ցուցադրել օգնությունը.:

`gh issue {{[new|create]}} {{[-h|--help]}}`
