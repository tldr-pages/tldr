# fastlane

> Ստեղծեք և թողարկեք բջջային հավելվածներ:.
> Լրացուցիչ տեղեկություններ. <https://docs.fastlane.tools/actions/>:.

- Կառուցեք և ստորագրեք iOS հավելվածը ընթացիկ գրացուցակում.:

`fastlane run build_app`

- Գործարկեք `pod install` նախագծի համար ընթացիկ գրացուցակում.:

`fastlane run cocoapods`

- Ջնջել ստացված տվյալները Xcode-ից.:

`fastlane run clear_derived_data`

- Հեռացրեք քեշը պատիճների համար.:

`fastlane run clean_cocoapods_cache`
