# Conda info

> Ցուցադրել կոնդերի տեղադրման մանրամասները:.
> Լրացուցիչ տեղեկություններ. <https://docs.conda.io/projects/conda/en/latest/commands/info.html>:.

- Ցույց տալ բոլոր տեղեկությունները.:

`conda info {{[-a|--all]}}`

- Ցուցադրել բազային միջավայրի ուղին.:

`conda info --base`

- Թվարկեք բոլոր կոնդա միջավայրերը.:

`conda info {{[-e|--envs]}}`

- Ցուցակել շրջակա միջավայրի փոփոխականները.:

`conda info {{[-s|--system]}}`

- Ցուցադրել ալիքների ցանկը, որոնցում առկա են նշաններ.:

`conda info --unsafe-channels`
