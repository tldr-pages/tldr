# jj կազմաձևման հավաքածու

> Սահմանեք կազմաձևման տարբերակը տվյալ արժեքի վրա:.
> Արժեքը նշված է որպես TOML արտահայտություն:.
> Տես նաև՝ `jj config unset`:.
> Լրացուցիչ տեղեկություններ. <https://docs.jj-vcs.dev/latest/cli-reference/#jj-config-set>:.

- Սահմանեք օգտվողի անունը օգտվողի մակարդակի կազմաձևում.:

`jj config {{[s|set]}} --user user.name "{{name}}"`

- Սահմանեք օգտվողի էլ.փոստը օգտվողի մակարդակի կազմաձևում.:

`jj config {{[s|set]}} --user user.email "{{email}}"`

- Սահմանեք կազմաձևման տարբերակ ռեպո մակարդակի կազմաձևում.:

`jj config {{[s|set]}} --repo {{name}} {{value}}`

- Աշխատանքային տարածքի մակարդակի կազմաձևում սահմանեք կազմաձևման տարբերակ.:

`jj config {{[s|set]}} --workspace {{name}} {{value}}`

- Սահմանեք բուլյան կազմաձևման տարբերակ.:

`jj config {{[s|set]}} --user {{name}} {{true|false}}`
