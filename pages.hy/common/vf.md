# vf

> VirtualFish-ը ձկան կեղևի գործիք է Python վիրտուալ միջավայրերը կառավարելու համար:.
> Լրացուցիչ տեղեկություններ. <https://virtualfish.readthedocs.io/en/latest/>:.

- Ստեղծեք վիրտուալ միջավայր.:

`vf new {{virtualenv_name}}`

- Ստեղծեք վիրտուալ միջավայր Python-ի հատուկ տարբերակի համար.:

`vf new --python {{/usr/local/bin/python3.8}} {{virtualenv_name}}`

- Ակտիվացրեք և օգտագործեք նշված վիրտուալ միջավայրը.:

`vf activate {{virtualenv_name}}`

- Միացրեք ընթացիկ virtualenv-ը ընթացիկ գրացուցակին, որպեսզի այն ավտոմատ կերպով ակտիվացվի հենց այն մուտքագրեք այն (և ապաակտիվացվի հենց որ դուրս գաք):

`vf connect`

- Անջատեք ընթացիկ վիրտուալ միջավայրը.:

`vf deactivate`

- Թվարկեք բոլոր վիրտուալ միջավայրերը.:

`vf ls`

- Հեռացնել վիրտուալ միջավայրը.:

`vf rm {{virtualenv_name}}`

- Ցուցադրել օգնությունը.:

`vf help`
