# ընդդեմ

> Ընդլայնման կառավարիչ Visual Studio Code-ի համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/microsoft/vscode-vsce>:.

- Թվարկեք հրատարակչի կողմից ստեղծված բոլոր ընդլայնումները.:

`vsce list {{publisher}}`

- Հրապարակեք ընդլայնում որպես հիմնական, փոքր կամ կարկատել տարբերակ.:

`vsce publish {{major|minor|patch}}`

- Չեղարկել ընդլայնման հրապարակումը.:

`vsce unpublish {{extension_id}}`

- Ներկայիս աշխատանքային գրացուցակը փաթեթավորեք որպես `.vsix` ֆայլ.:

`vsce package`

- Ցուցադրել ընդլայնման հետ կապված մետատվյալները.:

`vsce show {{extension_id}}`
