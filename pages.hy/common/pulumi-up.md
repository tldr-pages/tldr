# pulumi up

> Ստեղծեք կամ թարմացրեք ռեսուրսները կույտում:.
> Լրացուցիչ տեղեկություններ. <https://www.pulumi.com/docs/iac/cli/commands/pulumi_up/>:.

- Նախադիտեք և տեղակայեք փոփոխություններ ծրագրի և/կամ ենթակառուցվածքի մեջ.:

`pulumi up`

- Ավտոմատ կերպով հաստատեք և կատարեք թարմացումը այն նախադիտելուց հետո.:

`pulumi up {{[-y|--yes]}}`

- Նախադիտեք և տեղակայեք փոփոխությունները որոշակի կույտում.:

`pulumi up {{[-s|--stack]}} {{stack}}`

- Թարմացրեք փաթեթի ռեսուրսների վիճակը թարմացնելուց առաջ.:

`pulumi up {{[-r|--refresh]}}`

- Մի ցուցադրեք կույտի ելքերը.:

`pulumi up --suppress-outputs`

- Շարունակեք թարմացնել ռեսուրսները, նույնիսկ եթե սխալ է հանդիպում.:

`pulumi up --continue-on-error`

- Ցուցադրել օգնությունը.:

`pulumi up {{[-h|--help]}}`
