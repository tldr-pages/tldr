# dvc կոնֆիգուրացիա

> Ցածր մակարդակի հրաման՝ dvc պահեստների անհատական կազմաձևման տարբերակները կառավարելու համար:.
> Այս կոնֆիգուրացիան կարող է լինել նախագծի, տեղական, գլոբալ կամ համակարգի մակարդակով:.
> Լրացուցիչ տեղեկություններ. <https://doc.dvc.org/command-reference/config>:.

- Ստացեք լռելյայն հեռակառավարման անունը.:

`dvc config core.remote`

- Սահմանեք նախագծի լռելյայն հեռակառավարումը.:

`dvc config core.remote {{remote_name}}`

- Չեղարկեք նախագծի լռելյայն հեռակառավարումը.:

`dvc config {{[-u|--unset]}} core.remote`

- Ստացեք ընթացիկ նախագծի համար սահմանված բանալի կազմաձևման արժեքը.:

`dvc config {{key}}`

- Նախագծի մակարդակով բանալիի համար սահմանեք կազմաձևման արժեքը.:

`dvc config {{key}} {{value}}`

- Չսահմանեք նախագծի մակարդակի կազմաձևման արժեքը տվյալ բանալու համար.:

`dvc config {{[-u|--unset]}} {{key}}`

- Սահմանեք տեղական, գլոբալ կամ համակարգի մակարդակի կազմաձևման արժեքը.:

`dvc config --{{local|global|system}} {{key}} {{value}}`
