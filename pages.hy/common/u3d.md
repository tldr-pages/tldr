# u3d

> Unity-ի հետ փոխգործակցելու գործիքների մի շարք:.
> Լրացուցիչ տեղեկություններ. <https://github.com/DragonBox/u3d>:.

- Բացեք նախագիծը ընթացիկ գրացուցակում՝ օգտագործելով Unity-ի ճիշտ տարբերակը.:

`u3d`

- Ցուցակեք Unity-ի տեղադրված տարբերակները.:

`u3d list`

- Թվարկեք Unity-ի հասանելի տարբերակները, որոնք կարելի է ներբեռնել.:

`u3d available`

- Ներբեռնեք և տեղադրեք վերջին կայուն Unity տարբերակը.:

`u3d install latest_stable`

- Ներբեռնեք և տեղադրեք Unity տարբերակը և խմբագրիչի [p] փաթեթները.:

`u3d install {{2021.2.0f1}} -p {{Unity,iOS,Android}}`
