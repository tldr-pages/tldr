# zapier թեստ

> Փորձարկեք Zapier ինտեգրումը `test` սկրիպտի միջոցով ձեր `package.json`-ում:.
> Լրացուցիչ տեղեկություններ. <https://github.com/zapier/zapier-platform/blob/main/packages/cli/docs/cli.md#test>:.

- Փորձարկել ինտեգրումը.:

`zapier test`

- Ստուգեք ինտեգրումը` բաց թողնելով վավերացումը.:

`zapier test --skip-validate`

- Փորձարկել ինտեգրումը լրացուցիչ վրիպազերծման ելքի հետ.:

`zapier test {{[-d|--debug]}}`
