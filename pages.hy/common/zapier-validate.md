# zapier վավերացնել

> Վավերացրեք Zapier ինտեգրումը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/zapier/zapier-platform/blob/main/packages/cli/docs/cli.md#validate>:.

- Վավերացնել ինտեգրումը.:

`zapier validate`

- Վավերացրեք ինտեգրումը առանց ոճի ստուգումների.:

`zapier validate --without-style`

- Վավերացրեք ինտեգրումը առանց build script-ը գործարկելու.:

`zapier validate --skip-build`

- Վավերացրեք ինտեգրումը լրացուցիչ վրիպազերծման ելքով.:

`zapier validate {{[-d|--debug]}}`

- Վավերացրեք ինտեգրումը այլ ելքային կառուցվածքով.:

`zapier validate {{[-f|--format]}} {{plain|json|raw|row|table}}`
