# rustup վազք

> Գործարկեք հրաման Rust գործիքների շղթայի համար կազմաձևված միջավայրով:.
> Նշում. `rustup`-ով կառավարվող բոլոր հրամաններն ունեն սրա սղագրությունը. օրինակ՝ `cargo +nightly build`-ը համարժեք է `rustup run nightly cargo build`-ին:.
> Լրացուցիչ տեղեկություններ. <https://rust-lang.github.io/rustup/>:.

- Գործարկեք հրաման՝ օգտագործելով տրված Rust գործիքների շղթան (տե՛ս `rustup help toolchain` լրացուցիչ տեղեկությունների համար).:

`rustup run {{toolchain}} {{command}}`
