#jadx

> Dex դեպի Java decompiler:.
> Արտադրում է Java-ի աղբյուրի կոդը Android Dex և APK ֆայլերից:.
> Լրացուցիչ տեղեկություններ. <https://github.com/skylot/jadx#usage>:.

- Dex ֆայլը ապակոմպիլացրեք գրացուցակի մեջ.:

`jadx {{path/to/file}}`

- Ապակոմպիլացրեք Dex ֆայլը որոշակի գրացուցակի մեջ.:

`jadx {{[-d|--output-dir]}} {{path/to/directory}} {{path/to/file}}`
