# pip ստուգում

> Ստուգեք Python միջավայրը և պատրաստեք հաշվետվություն JSON ձևաչափով:.
> Լրացուցիչ տեղեկություններ. <https://pip.pypa.io/en/stable/cli/pip_inspect/>:.

- Ստուգեք ներկա միջավայրը.:

`pip inspect`

- Ստուգեք և պահպանեք արդյունքը ֆայլում.:

`pip inspect > {{environment_report.json}}`

- Ստուգեք միայն տեղական տեղադրված փաթեթները (ոչ գլոբալ).:

`pip inspect --local`

- Ստուգեք միայն օգտագործողի կողմից տեղադրված փաթեթները.:

`pip inspect --user`

- Ստուգեք փաթեթները որոշակի ճանապարհով.:

`pip inspect --path {{path/to/environment}}`

- Ստուգեք բառացի ելքով (Նշում. `-v` դրոշը կարող է կրկնվել՝ խոսակցությունը բարձրացնելու համար):

`pip inspect {{[-v|--verbose]}}`
