# jj կոնֆիգուրացիա ստանալ

> Ստացեք տվյալ կազմաձևման ընտրանքի արժեքը:.
> Ի տարբերություն `jj config list`-ի՝ արդյունքը տպվում է առանց լրացուցիչ ձևաչափման՝ սկրիպտներում օգտագործելու համար:.
> Տես նաև՝ `jj config list`:.
> Լրացուցիչ տեղեկություններ. <https://docs.jj-vcs.dev/latest/cli-reference/#jj-config-get>:.

- Ստացեք կազմաձևման տարբերակի արժեքը.:

`jj config {{[g|get]}} {{name}}`

- Ստացեք կազմաձևված օգտվողի անունը.:

`jj config {{[g|get]}} user.name`

- Ստացեք կազմաձևված օգտվողի էլ.:

`jj config {{[g|get]}} user.email`

- Ստացեք լռելյայն վերականգնում log հրամանի համար.:

`jj config {{[g|get]}} revsets.log`
