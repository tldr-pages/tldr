# nix flake info

> Ցույց տալ շերտավոր մետատվյալները:.
> Լրացուցիչ տեղեկություններ. <https://nix.dev/manual/nix/stable/command-ref/new-cli/nix3-flake-info>:.

- Show flake metadata of the flake in the current directory:

`nix flake info`

- Ցույց տալ շերտավոր մետատվյալները GitHub-ից որպես json մեկ տողում.:

`nix flake info {{github:owner/repo}} --json --no-pretty`

- Ցուցադրել փաթիլային մետատվյալները Github-ից որպես բազմատող մատնանշված json:

`nix flake info {{github:owner/repo}} --json --pretty`
