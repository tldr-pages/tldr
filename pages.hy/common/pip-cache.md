# pip cache

> Ստուգեք և կառավարեք pip-ի անիվի քեշը:.
> Լրացուցիչ տեղեկություններ. <https://pip.pypa.io/en/stable/cli/pip_cache/>:.

- Ցույց տալ pip cache գրացուցակի գտնվելու վայրը.:

`pip cache dir`

- Թվարկեք քեշում ներկայումս պահվող բոլոր փաթեթների անունները.:

`pip cache list`

- Հեռացրեք բոլոր ֆայլերը pip cache-ից.:

`pip cache purge`

- Հեռացրեք քեշավորված ֆայլերը, որոնք համապատասխանում են որոշակի փաթեթի անվանմանը.:

`pip cache remove {{package_name}}`
