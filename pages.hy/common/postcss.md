# postcss

> Փոխակերպեք ոճերը JS հավելվածներով:.
> Լրացուցիչ տեղեկություններ. <https://github.com/postcss/postcss-cli#usage>:.

- Վերլուծել և վերափոխել CSS ֆայլը.:

`postcss {{path/to/file}}`

- Վերլուծեք և փոխակերպեք CSS ֆայլը և թողարկեք որոշակի ֆայլի.:

`postcss {{path/to/file}} {{[-o|--output]}} {{path/to/file}}`

- Վերլուծեք և փոխակերպեք CSS ֆայլը և թողարկեք որոշակի գրացուցակ.:

`postcss {{path/to/file}} {{[-d|--dir]}} {{path/to/directory}}`

- Վերլուծեք և փոխակերպեք CSS ֆայլը տեղում.:

`postcss {{path/to/file}} {{[-r|--replace]}}`

- Նշեք հատուկ PostCSS վերլուծիչ.:

`postcss {{path/to/file}} --parser {{parser}}`

- Նշեք հատուկ PostCSS շարահյուսություն.:

`postcss {{path/to/file}} --syntax {{syntax}}`

- Հետևեք CSS ֆայլի փոփոխություններին.:

`postcss {{path/to/file}} {{[-w|--watch]}}`

- Ցուցադրել օգնությունը.:

`postcss {{[-h|--help]}}`
