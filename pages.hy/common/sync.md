# համաժամեցում

> Մաքրում է բոլոր առկախ գրելու գործողությունները համապատասխան սկավառակների վրա:.
> Լրացուցիչ տեղեկություններ. <https://www.gnu.org/software/coreutils/manual/html_node/sync-invocation.html>:.

- Լվացեք բոլոր սպասող գրելու գործողությունները բոլոր սկավառակների վրա.:

`sync`

- Սկավառակի վրա մեկ ֆայլի վրա գրելու բոլոր սպասվող գործողությունները.:

`sync {{path/to/file}}`

- Flush-ը գրում և թողնում է ֆայլային համակարգի քեշերը (միայն Linux-ը).:

`sync; echo 3 | sudo tee /proc/sys/vm/drop_caches`

- Flush disk-ը գրում և փորձում է մաքրել ոչ ակտիվ հիշողությունը և ֆայլային համակարգի քեշերը (միայն macOS).:

`sync; sudo purge`
