# git ֆորմատ-կարկատել

> Պատրաստել `.patch` ֆայլեր: Օգտակար է, երբ էլփոստով պարտավորություններ ուղարկեք այլուր:.
> Տես նաև՝ `git am`:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-format-patch>:.

- Ստեղծեք ավտոմատ անունով `.patch` ֆայլ բոլոր չհրաժարված պարտավորությունների համար.:

`git format-patch {{origin}}`

- Գրեք `.patch` ֆայլ բոլոր պարտավորությունների համար՝ `stdout`-ի 2 վերանայման միջև:

`git format-patch {{revision_1}}..{{revision_2}}`

- Գրեք `.patch` ֆայլ `n` վերջին պարտավորությունների համար.:

`git format-patch -{{n}}`
