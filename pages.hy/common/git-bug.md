# git bug

> Բաշխված սխալների հետագծում, որն օգտագործում է Git-ի ներքին հիշողությունը, այնպես որ ձեր նախագծում ոչ մի ֆայլ չի ավելացվում:.
> Դուք կարող եք ձեր խնդիրները ներկայացնել նույն Git հեռակառավարման սարքին, որն օգտագործում եք ուրիշների հետ շփվելու համար, ինչպես պարտավորությունները և ճյուղերը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/git-bug/git-bug/blob/master/doc/md/git-bug.md>:.

- Ստեղծել նոր ինքնություն.:

`git bug user create`

- Ստեղծեք նոր սխալ.:

`git bug add`

- Նոր վրիպակի մուտքագրեք հեռակառավարման վահանակ.:

`git bug push`

- Քաշեք թարմացումների համար.:

`git bug pull`

- Թվարկեք առկա սխալները.:

`git bug ls`

- Զտել և տեսակավորել սխալները՝ օգտագործելով հարցումը.:

`git bug ls "{{status}}:{{open}} {{sort}}:{{edit}}"`

- Որոնեք սխալներ ըստ տեքստի բովանդակության.:

`git bug ls "{{search_query}}" baz`
