# tlmgr որոնում

> Որոնեք TeX Live փաթեթներ՝ օգտագործելով (Perl) `regex`:.
> Լրացուցիչ տեղեկություններ. <https://www.tug.org/texlive/doc/tlmgr.html#search>:.

- Փնտրեք փաթեթի անվանումը և տեղական տեղադրված բոլոր փաթեթների նկարագրությունները կոնկրետ `regex`-ից:

`tlmgr search "{{regex}}"`

- Փնտրեք տեղական տեղադրված բոլոր փաթեթների բոլոր ֆայլերի անունները `regex`-ից:

`tlmgr search --file "{{regex}}"`

- Փնտրեք բոլոր ֆայլերի անունները, փաթեթների անունները և տեղական տեղադրված բոլոր փաթեթների նկարագրությունները `regex`-ից:

`tlmgr search --all "{{regex}}"`

- Որոնեք TeX Live տվյալների բազան, տեղական տեղադրման փոխարեն.:

`tlmgr search --global "{{regex}}"`

- Սահմանափակեք փաթեթների անունների և նկարագրությունների համընկնումները (բայց ոչ ֆայլերի անունների համար) ամբողջական բառերով.:

`tlmgr search --all --word "{{regex}}"`
