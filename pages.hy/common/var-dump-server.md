# var-dump-server

> Symfony dump սերվեր:.
> Հավաքում է Symfony VarDumper բաղադրիչի կողմից թափված տվյալները:.
> Լրացուցիչ տեղեկություններ. <https://symfony.com/doc/current/components/var_dumper.html#the-dump-server>:.

- Գործարկել սերվերը.:

`var-dump-server`

- Թափել տվյալները HTML ֆայլում.:

`var-dump-server --format=html > {{path/to/file.html}}`

- Ստիպեք սերվերին լսել որոշակի հասցեով և նավահանգստով.:

`var-dump-server --host {{127.0.0.1:9912}}`
