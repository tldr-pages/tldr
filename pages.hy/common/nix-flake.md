# nix փաթիլ

> Կառավարեք Nix փաթիլները:.
> Որոշ ենթահրամաններ, ինչպիսիք են `init`, `show`, `info`, ունեն իրենց օգտագործման փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://nix.dev/manual/nix/stable/command-ref/new-cli/nix3-flake.html>:.

- Թարմացրեք փաթիլների բոլոր մուտքերը (կախվածությունները) ընթացիկ գրացուցակում.:

`nix flake update`

- Թարմացրեք փաթիլների հատուկ մուտքագրումը (կախվածությունը) ընթացիկ գրացուցակում.:

`nix flake update {{input}}`

- Կլոնավորեք GitHub պահեստը, որը պարունակում է փաթիլ.:

`nix flake clone {{github:owner/repo#attributes}}`

- Ստեղծեք բացակայող կողպեքի ֆայլի գրառումներ փաթիլների համար նշված գրացուցակում.:

`nix flake lock {{path/to/flake}}`

- Ցուցադրել օգնությունը.:

`nix flake --help`
