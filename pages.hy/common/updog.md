# updog

> Python-ի SimpleHTTPS-ի փոխարինում:.
> Այն թույլ է տալիս վերբեռնել և ներբեռնել HTTP/S-ի միջոցով, կարող է սահմանել ժամանակավոր SSL վկայագրեր և օգտագործել HTTP հիմնական վավերացում:.
> Լրացուցիչ տեղեկություններ. <https://github.com/sc0tfree/updog>:.

- Գործարկեք HTTP սերվեր ընթացիկ գրացուցակի համար.:

`updog`

- Սկսեք HTTP սերվեր նշված գրացուցակի համար.:

`updog --directory /{{path/to/directory}}`

- Սկսեք HTTP սերվերը նշված նավահանգստում.:

`updog --port {{port}}`

- Սկսեք HTTP սերվերը գաղտնաբառով (Մուտք գործելու համար թողեք օգտվողի անունը դատարկ և մուտքագրեք գաղտնաբառը գաղտնաբառի դաշտում).:

`updog --password {{password}}`

- Միացնել տրանսպորտի կոդավորումը SSL-ի միջոցով.:

`updog --ssl`
