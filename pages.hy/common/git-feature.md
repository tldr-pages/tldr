# git հատկություն

> Ստեղծեք կամ միավորեք առանձնահատկությունների ճյուղերը:.
> Հատկությունների ճյուղերը ենթարկվում են ձևաչափի հատկանիշին/անունին:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/git-feature>:.

- Ստեղծեք և անցեք նոր գործառույթի մասնաճյուղ.:

`git feature {{feature_branch}}`

- Միավորել առանձնահատկությունների ճյուղը ընթացիկ ճյուղի մեջ՝ ստեղծելով միաձուլման պարտավորություն.:

`git feature finish {{feature_branch}}`

- Միաձուլեք առանձնահատկությունների ճյուղը ընթացիկ ճյուղի մեջ՝ սեղմելով փոփոխությունները մեկ կատարման մեջ.:

`git feature finish --squash {{feature_branch}}`

- Փոփոխություններ ուղարկեք հատուկ գործառույթի ճյուղից իր հեռավոր գործընկերոջը.:

`git feature {{feature_branch}} {{[-r|--remote]}} {{remote_name}}`
