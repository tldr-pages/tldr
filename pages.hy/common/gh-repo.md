# gh ռեպո

> Աշխատեք GitHub պահեստների հետ:.
> Լրացուցիչ տեղեկություններ. <https://cli.github.com/manual/gh_repo>:.

- Ստեղծեք նոր պահեստ ինտերակտիվ կերպով.:

`gh repo {{[new|create]}}`

- Կլոնավորել պահեստը.:

`gh repo clone {{owner}}/{{repository}}`

- Պատառաքաղել և կլոնավորել պահեստը.:

`gh repo fork {{owner}}/{{repository}} --clone`

- Դիտեք պահեստը լռելյայն վեբ բրաուզերում.:

`gh repo view {{repository}} {{[-w|--web]}}`

- Ցուցակեք պահոցները, որոնք պատկանում են որոշակի օգտվողին կամ կազմակերպությանը (եթե սեփականատերը սահմանված չէ, ապա լռելյայն սեփականատերը կլինի տվյալ պահին մուտք գործած օգտվողը).:

`gh repo {{[ls|list]}} {{owner}}`

- Թվարկեք միայն ոչ պատառաքաղային պահոցները և սահմանափակեք պահեստների քանակը ցուցակով (կանխադրված՝ 30):

`gh repo {{[ls|list]}} {{owner}} --source {{[-L|--limit]}} {{limit}}`

- Թվարկեք պահեստները հատուկ հիմնական կոդավորման լեզվով.:

`gh repo {{[ls|list]}} {{owner}} {{[-l|--language]}} {{language_name}}`

- Համաժամեցրեք ընթացիկ տեղական պահոցը իր վերին հոսանքով.:

`gh repo sync`
