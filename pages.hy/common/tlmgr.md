# tlmgr

> Կառավարեք գոյություն ունեցող TeX Live տեղադրման փաթեթները և կազմաձևման տարբերակները:.
> Որոշ ենթահրամաններ, ինչպիսիք են `paper`-ն, ունեն իրենց օգտագործման փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://www.tug.org/texlive/doc/tlmgr.html#NAME>:.

- Տեղադրեք փաթեթը և դրա կախվածությունները.:

`tlmgr install {{package}}`

- Հեռացրեք փաթեթը և դրա կախվածությունները.:

`tlmgr remove {{package}}`

- Ցուցադրել տեղեկատվություն փաթեթի մասին.:

`tlmgr info {{package}}`

- Թարմացրեք բոլոր փաթեթները.:

`tlmgr update --all`

- Ցույց տալ հնարավոր թարմացումները՝ առանց որևէ բան թարմացնելու.:

`tlmgr update --list`

- Սկսեք tlmgr-ի GUI տարբերակը.:

`tlmgr gui`

- Թվարկեք TeX Live-ի բոլոր կոնֆիգուրացիաները.:

`tlmgr conf`
