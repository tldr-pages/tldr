# պոեզիա-վրիպազերծում

> Վրիպազերծեք ձեր Poetry նախագծի հետ կապված խնդիրները:.
> Տես նաև՝ `asdf`:.
> Լրացուցիչ տեղեկություններ. <https://python-poetry.org/docs/cli/#debug>:.

- Ցույց տալ վրիպազերծման տեղեկությունները Poetry-ի և ձեր նախագծի վիրտուալ միջավայրի մասին.:

`poetry debug info`

- Փորձեք լուծել ձեր նախագծի կախվածությունները.:

`poetry debug resolve`

- Ցույց տալ աջակցվող փաթեթավորման պիտակները ձեր նախագծի ակտիվ վիրտուալ միջավայրի համար.:

`poetry debug tags`
