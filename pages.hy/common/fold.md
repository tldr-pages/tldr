# ծալք

> Փաթեթավորեք յուրաքանչյուր տողը մուտքային ֆայլի մեջ, որպեսզի համապատասխանի նշված լայնությանը և տպեք այն `stdout`-ով:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/fold.1p>:.

- Յուրաքանչյուր տող փաթեթավորեք լռելյայն լայնությամբ (80 նիշ).:

`fold {{path/to/file}}`

- Յուրաքանչյուր տող փաթաթեք «30» լայնությամբ.:

`fold -w30 {{path/to/file}}`

- Յուրաքանչյուր տող փաթաթեք «5» լայնությամբ և գիծը կտրեք բացատներով (յուրաքանչյուր բացատով առանձնացված բառը դնում է նոր տողի մեջ, 5 երկարությամբ բառերը փաթաթված են):

`fold -w5 -s {{path/to/file}}`
