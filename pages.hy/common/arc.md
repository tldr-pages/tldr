# աղեղ

> Arcanist. CLI Phabricator-ի համար:.
> Լրացուցիչ տեղեկություններ. <https://secure.phabricator.com/book/phabricator/article/arcanist/>:.

- Փոփոխություններն ուղարկեք Դիֆերենցիալ՝ վերանայման համար.:

`arc diff`

- Ցուցադրել սպասվող վերանայման տեղեկատվությունը.:

`arc list`

- Թարմացրեք Git commit հաղորդագրությունները վերանայումից հետո.:

`arc amend`

- Push Git փոփոխությունները.:

`arc land`
