# բեռների փորձարկում

> Կատարեք Rust փաթեթի միավորի և ինտեգրման թեստերը:.
> Լրացուցիչ տեղեկություններ. <https://doc.rust-lang.org/cargo/commands/cargo-test.html>:.

- Գործարկեք միայն թեստեր, որոնք պարունակում են որոշակի տող իրենց անուններով.:

`cargo {{[t|test]}} {{test_name}}`

- Սահմանեք միաժամանակյա վազքի փորձարկման դեպքերի քանակը.:

`cargo {{[t|test]}} -- --test-threads {{count}}`

- Փորձարկեք արտեֆակտները թողարկման ռեժիմում, օպտիմալացումներով.:

`cargo {{[t|test]}} {{[-r|--release]}}`

- Փորձարկեք բոլոր փաթեթները աշխատանքային տարածքում.:

`cargo {{[t|test]}} --workspace`

- Գործարկել թեստեր կոնկրետ փաթեթի համար.:

`cargo {{[t|test]}} {{[-p|--package]}} {{package}}`

- Գործարկել թեստերը, առանց թաքցնելու արդյունքը թեստային կատարումներից.:

`cargo {{[t|test]}} -- --nocapture`
