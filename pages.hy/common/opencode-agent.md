# opencode գործակալ

> Կառավարեք գործակալները OpenCode-ի համար:.
> Լրացուցիչ տեղեկություններ. <https://opencode.ai/docs/cli#agent>:.

- Թվարկեք բոլոր հասանելի գործակալները.:

`opencode agent list`

- Ստեղծեք նոր գործակալ ինտերակտիվ կերպով.:

`opencode agent create`

- Ցուցադրել օգնությունը.:

`opencode agent {{[-h|--help]}}`
