# պտղցի

> Ինտերակտիվ Haskell REPL՝ շարահյուսական ընդգծմամբ, բազմագիծ խմբագրմամբ և իրական ժամանակի տիպի ցուցադրմամբ:.
> Տես նաև՝ `ghci`:.
> Լրացուցիչ տեղեկություններ. <https://github.com/litxio/ptghci>:.

- Սկսեք ինտերակտիվ նիստ.:

`ptghci`

- Սկսեք ինտերակտիվ նիստ և բեռնեք Haskell ֆայլը.:

`ptghci {{path/to/file.hs}}`

- [Ինտերակտիվ] Ցուցադրել անցյալի հրամանները ընթացիկ նստաշրջանից.:

`%past`

- [Ինտերակտիվ] Ցուցակեք անցյալ հրամանները, ներառյալ նախորդ նիստերը.:

`%past -n {{number}}`

- [Ինտերակտիվ] Վերագործարկեք հատուկ հրամաններ պատմությունից.:

`%rerun {{3,4-5,p8}}`

- [Ինտերակտիվ] Որոնել Hoogle նույնացուցիչի համար.:

`%hoogle {{identifier}}`

- [Ինտերակտիվ] Փոխեք շարահյուսության ընդգծման ոճը.:

`%style {{style_name}}`
