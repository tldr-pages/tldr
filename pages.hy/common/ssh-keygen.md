# ssh-keygen

> Ստեղծեք SSH ստեղներ, որոնք օգտագործվում են նույնականացման, առանց գաղտնաբառի մուտքերի և այլ բաների համար:.
> Տես նաև՝ `ssh-copy-id`:.
> Լրացուցիչ տեղեկություններ. <https://man.openbsd.org/ssh-keygen>:.

- Ստեղծեք բանալին ինտերակտիվ կերպով.:

`ssh-keygen`

- Ստեղծեք ed25519 բանալի 32 ստեղնավորման ֆունկցիայի փուլերով և պահեք բանալին որոշակի ֆայլում.:

`ssh-keygen -f {{~/.ssh/filename}} -t ed25519 -a 32`

- Ստեղծեք RSA 4096-բիթանոց բանալի էլեկտրոնային փոստով որպես մեկնաբանություն.:

`ssh-keygen -t rsa -b 4096 -C "{{comment|email}}"`

- Հեռացրեք հոսթի ստեղները `known_hosts` ֆայլից (օգտակար է, երբ հայտնի հոսթն ունի նոր բանալի):

`ssh-keygen -R {{remote_host}}`

- Առբերեք բանալիի մատնահետքը MD5 Hex-ում.:

`ssh-keygen -f {{~/.ssh/filename}} -l -E md5`

- Փոխեք բանալիի գաղտնաբառը.:

`ssh-keygen -f {{~/.ssh/filename}} -p`

- Փոխեք բանալի ձևաչափի տեսակը (օրինակ՝ OPENSSH ձևաչափից PEM), ֆայլը տեղում կվերագրվի.:

`ssh-keygen -f {{~/.ssh/OpenSSH_private_key}} -p -m PEM`

- Առբերեք հանրային բանալին մասնավոր բանալիից.:

`ssh-keygen -f {{~/.ssh/OpenSSH_private_key}} -y`
