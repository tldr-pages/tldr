# zstd

> Սեղմել կամ ապասեղմել ֆայլերը Zstandard սեղմումով:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/zstd>:.

- Կոմպրես ֆայլը նոր ֆայլի մեջ `.zst` վերջածանցով.:

`zstd {{path/to/file}}`

- Ապասեղմել ֆայլը.:

`zstd {{[-d|--decompress]}} {{path/to/file.zst}}`

- Ապասեղմել `stdout`:

`zstd {{[-d|--decompress]}} {{[-c|--stdout]}} {{path/to/file.zst}}`

- Սեղմեք ֆայլը, որը նշում է սեղմման մակարդակը, որտեղ 1=ամենաարագ, 19=ամենադանդաղ և 3=կանխադրված:

`zstd -{{level}} {{path/to/file}}`

- Կոմպրես ֆայլը՝ օգտագործելով ծայրահեղ արագ սեղմման մակարդակ, որտեղ 1=կանխադրված:

`zstd --fast={{level}} {{path/to/file}}`

- Բացեք սեղմման ավելի բարձր մակարդակները (մինչև 22)՝ օգտագործելով ավելի շատ հիշողություն (և՛ սեղմման, և՛ ապասեղմման համար).:

`zstd --ultra -{{level}} {{path/to/file}}`

- Սահմանեք աշխատանքային թելերի քանակը ֆիզիկական պրոցեսորի միջուկների քանակին.:

`zstd {{[-T|--threads]}} 0`
