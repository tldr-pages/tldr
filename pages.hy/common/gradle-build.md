# gradle build

> Կառուցեք նախագիծ Gradle-ի միջոցով:.
> Լրացուցիչ տեղեկություններ. <https://docs.gradle.org/current/userguide/command_line_interface.html#common_tasks>:.

- Կառուցեք նախագիծը.:

`gradle build`

- Կատարել մաքուր կառուցում.:

`gradle clean build`

- Կառուցեք նախագիծը թեստերը բաց թողնելով.:

`gradle build {{[-x|--exclude-task]}} test`

- Կառուցեք ավելի մանրամասն անտառահատումներով.:

`gradle build {{[-i|--info]}}`
