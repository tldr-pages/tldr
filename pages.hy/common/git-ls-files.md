# git ls-ֆայլեր

> Ցուցադրել ինդեքսում և աշխատող ծառի ֆայլերի մասին տեղեկատվություն:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-ls-files>:.

- Ցույց տալ ջնջված ֆայլերը.:

`git ls-files {{[-d|--deleted]}}`

- Ցուցադրել փոփոխված և ջնջված ֆայլերը.:

`git ls-files {{[-m|--modified]}}`

- Ցույց տալ անտեսված և չհետևված ֆայլերը.:

`git ls-files {{[-o|--others]}}`

- Ցուցադրել չհետևված ֆայլերը՝ չանտեսված.:

`git ls-files {{[-o|--others]}} --exclude-standard`
