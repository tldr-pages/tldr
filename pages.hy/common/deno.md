#դենո

> JavaScript-ի, TypeScript-ի և WebAssembly-ի անվտանգ գործարկման ժամանակ:.
> Ներառում է կախվածության կառավարում, օգտագործելով `npm` կամ `jsr`, ինչպես նաև գործիքներ, ինչպիսիք են նստարանը, փաթեթը, փաստաթուղթը և ծածկույթը:.
> Լրացուցիչ տեղեկություններ. <https://docs.deno.com/runtime/reference/cli/>:.

- Սկսեք REPL (ինտերակտիվ կեղև, որը նաև հայտնի է որպես Read-Eval-Print Loop):

`deno`

- Սկսեք նմուշ անունով նոր նախագիծ և փորձարկեք այն.:

`deno init sample && cd sample && deno test`

- Ապահով գործարկել ֆայլը: Այն կխնդրի (անհրաժեշտության դեպքում) թույլատրել ցանցը, կարդալ և այլն:

`deno run {{path/to/file.ts}}`

- Գործարկեք բացահայտ թույլտվություններով ֆայլ կամ թույլատրեք բոլորը (միայն եթե վստահում եք աղբյուրին).:

`deno run {{[--allow-env|--allow-net|--allow-write|--allow-all]}} {{jsr:@deno/deployctl}}`

- Թվարկեք և գործարկեք առաջադրանքները `deno.json`-ից կամ սկրիպտներ `package.json`-ից:

`deno task`

- Տեղադրեք `deno.json` կամ `package.json`-ում թվարկված կախվածությունները (նաև կողպեք ֆայլերը).:

`deno install`

- Ստուգեք տեսակները, ձևաչափը և ծածկույթը (հնարավորության դեպքում ուղղեք).:

`deno check && deno fmt && deno lint --fix`

- Կազմեք սցենարը, ներմուծված կախվածությունները և գործարկման ժամանակը ինքնուրույն պարունակվող գործարկիչի մեջ.:

`deno compile {{path/to/file.ts}}`
