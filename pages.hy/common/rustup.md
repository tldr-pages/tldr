#ժանգոտվել

> Տեղադրեք, կառավարեք և թարմացրեք Rust գործիքների շղթաները:.
> Որոշ ենթահրամաններ, ինչպիսիք են `toolchain`, `target`, `update` և այլն, ունեն իրենց օգտագործման փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://rust-lang.github.io/rustup/>:.

- Տեղադրեք գիշերային գործիքների շղթան ձեր համակարգի համար.:

`rustup install nightly`

- Անցեք լռելյայն գործիքաշարը գիշերայինի, որպեսզի `cargo` և `rustc` հրամաններն օգտագործեն այն.:

`rustup default nightly`

- Օգտագործեք գիշերային գործիքների շղթան ընթացիկ նախագծի ներսում, բայց գլոբալ կարգավորումները թողեք անփոփոխ.:

`rustup override set nightly`

- Թարմացրեք բոլոր գործիքների շղթաները.:

`rustup update`

- Տեղադրված գործիքների շղթաների ցանկ.:

`rustup show`

- Գործարկեք `cargo build`-ը որոշակի գործիքաշարով.:

`rustup run {{toolchain}} cargo build`

- Բացեք տեղական Rust փաստաթղթերը լռելյայն վեբ բրաուզերում.:

`rustup doc`
