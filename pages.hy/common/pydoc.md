# pydoc

> Ցուցադրել անցանց Python փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://docs.python.org/library/pydoc.html>:.

- Տպել փաստաթղթերը թեմայի վերաբերյալ (Python հիմնաբառ, թեմա, գործառույթ, մոդուլ, փաթեթ և այլն):

`pydoc {{subject}}`

- Սկսեք HTTP սերվերը կամայական չօգտագործված նավահանգստի վրա և բացեք [b] զննարկիչը՝ փաստաթղթերը տեսնելու համար.:

`pydoc -b`

- Ցուցադրել օգնությունը.:

`pydoc`
