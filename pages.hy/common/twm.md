# twm

> X Window համակարգի պատուհանների կառավարիչ:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/twm>:.

- Միացեք լռելյայն X սերվերին.:

`twm`

- Միացեք կոնկրետ X սերվերին.:

`twm {{[-d|-display]}} {{display}}`

- Կառավարեք միայն լռելյայն էկրանը.:

`twm {{[-s|-single]}}`

- Օգտագործեք հատուկ մեկնարկային ֆայլ.:

`twm {{[-f|-file]}} {{path/to/file}}`

- Միացնել բառացի ռեժիմը և տպել անսպասելի սխալներ X-ում:

`twm {{[-v|-verbose]}}`
