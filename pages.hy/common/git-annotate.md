# git ծանոթագրություն

> Ցույց տալ commit hash-ը և վերջին հեղինակը ֆայլի յուրաքանչյուր տողում:.
> Տես `git blame`-ը, որը նախընտրելի է `git annotate`-ից:.
> `git annotate` տրամադրված է տարբերակների կառավարման այլ համակարգերին ծանոթ մարդկանց համար:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-annotate>:.

- Տպեք ֆայլ հեղինակի անունով և կատարեք հեշ, որը կցված է յուրաքանչյուր տողին.:

`git annotate {{path/to/file}}`

- Տպեք ֆայլ հեղինակի էլ.փոստով և կատարեք հեշ, որը կցված է յուրաքանչյուր տողին.:

`git annotate {{[-e|--show-email]}} {{path/to/file}}`

- Տպեք միայն այն տողերը, որոնք համապատասխանում են `regex`-ին:

`git annotate -L :{{regexp}} {{path/to/file}}`
