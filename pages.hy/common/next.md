# հաջորդ

> React շրջանակը, որն օգտագործում է սերվերի կողմից օպտիմիզացված վեբ հավելվածներ կառուցելու համար:.
> Լրացուցիչ տեղեկություններ. <https://nextjs.org/docs/app/api-reference/cli/next>:.

- Գործարկեք ընթացիկ հավելվածը զարգացման ռեժիմում.:

`next dev`

- Սկսեք ընթացիկ հավելվածը և լսեք որոշակի նավահանգստում.:

`next dev {{[-p|--port]}} {{port}}`

- Կառուցեք արտադրության համար օպտիմիզացված ընթացիկ հավելվածը.:

`next build`

- Սկսեք կազմված հավելվածը արտադրության ռեժիմում.:

`next start`

- Սկսեք կազմված հավելվածը և լսեք որոշակի նավահանգստում.:

`next start {{[-p|--port]}} {{port}}`

- Արտահանել ընթացիկ հավելվածը ստատիկ HTML էջերին.:

`next export`

- Ցուցադրել Next.js հեռաչափության կարգավիճակը.:

`next telemetry`

- Ցուցադրել օգնություն ենթահրամանի համար.:

`next {{build|dev|export|start|telemetry}} {{[-h|--help]}}`
