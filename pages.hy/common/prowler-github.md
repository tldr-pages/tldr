# prowler github

> Գնահատեք GitHub հաշվի, պահոցի և կազմակերպության անվտանգության լավագույն փորձը:.
> Տես նաև՝ `prowler`, `prowler-aws`, `prowler-azure`, `prowler-gcp`, `prowler-kubernetes`, `prowler-m365`:.
> Լրացուցիչ տեղեկություններ. <https://docs.prowler.com/user-guide/cli/tutorials/misc>:.

- Գործարկեք բոլոր կանխադրված GitHub անվտանգության ստուգումները.:

`prowler github`

- Նույնականացրեք՝ օգտագործելով GitHub Անձնական մուտքի նշան.:

`prowler github --personal-access-token {{pat}}`

- Նույնականացրեք GitHub OAuth հավելվածի նշանի միջոցով.:

`prowler github --oauth-app-token {{oauth_token}}`

- Նույնականացրեք GitHub հավելվածի ID-ի և անձնական բանալիի միջոցով.:

`prowler github --github-app-id {{app_id}} --github-app-key {{app_key}}`
