#բնակարան մաքուր

> Ջնջել ժամանակավոր կամ չօգտագործված ֆայլերը՝ ինդեքսի քեշը, կողպված ֆայլերը, չօգտագործված քեշի փաթեթները, tarballs և log ֆայլերը:.
> Լրացուցիչ տեղեկություններ. <https://docs.conda.io/projects/conda/en/stable/commands/clean.html>:.

- Բառացիորեն ջնջեք բոլոր ժամանակավոր կամ չօգտագործված ֆայլերը և այո ասեք բոլոր հաստատումներին.:

`conda clean {{[-avy|--all --verbose --yes]}}`

- Ջնջել միայն ինդեքսի քեշը, tarballs և log ֆայլերը.:

`conda clean {{[-itl|--index-cache --tarballs --logfiles]}}`

- Ջնջեք միայն ժամանակավոր [c]ache ֆայլերը, որոնք հնարավոր չէին ավելի վաղ ջնջել՝ օգտագործման պատճառով.:

`conda clean {{[-c|--tempfiles]}} {{path/to/tempfiles}}`

- Ջնջել միայն չօգտագործված փաթեթները: Կարող է ջնջել փափուկ հղումներով տեղադրված փաթեթները.:

`conda clean {{[-p|--packages]}}`

- Ստիպել ջնջել բոլոր գրավոր փաթեթները: Ավելի լայն, քան `--all` տարբերակը: Կջնջի փափուկ հղումներով տեղադրված փաթեթները.:

`conda clean {{[-f|--force-pkgs-dirs]}}`

- Ցուցադրել օգնությունը.:

`conda clean {{[-h|--help]}}`
