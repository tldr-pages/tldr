# git diff-ֆայլեր

> Համեմատեք ֆայլերը՝ օգտագործելով իրենց sha1 հեշերը և ռեժիմները:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-diff-files>:.

- Համեմատեք բոլոր փոխված ֆայլերը.:

`git diff-files`

- Համեմատեք միայն նշված ֆայլերը.:

`git diff-files {{path/to/file}}`

- Ցույց տալ միայն փոխված ֆայլերի անունները.:

`git diff-files --name-only`

- Արտադրեք ընդլայնված վերնագրի տեղեկատվության ամփոփագիրը.:

`git diff-files --summary`
