# ruff ձևաչափ

> Չափազանց արագ Python կոդերի ձևաչափ:.
> Եթե ֆայլեր կամ գրացուցակներ նշված չեն, ընթացիկ աշխատանքային գրացուցակը օգտագործվում է լռելյայն:.
> Լրացուցիչ տեղեկություններ. <https://docs.astral.sh/ruff/formatter/>:.

- Տրված ֆայլերի կամ գրացուցակների ձևաչափը տեղում.:

`ruff format {{path/to/file_or_directory1 path/to/file_or_directory2 ...}}`

- Տպեք, թե որ ֆայլերը կփոփոխվեին և վերադարձրեք ոչ զրոյական ելքի կոդ, եթե կան ֆայլեր՝ վերաֆորմատավորելու համար, իսկ հակառակ դեպքում՝ զրո.:

`ruff format --check`

- Տպեք, թե ինչ փոփոխություններ կկատարվեն առանց ֆայլերը փոփոխելու.:

`ruff format --diff`
