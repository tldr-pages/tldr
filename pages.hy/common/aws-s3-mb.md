# aws s3 mb

> Ստեղծեք S3 դույլեր:.
> Լրացուցիչ տեղեկություններ. <https://docs.aws.amazon.com/cli/latest/reference/s3/mb.html>:.

- Ստեղծեք S3 դույլ.:

`aws s3 mb s3://{{bucket_name}}`

- Ստեղծեք S3 դույլ որոշակի տարածաշրջանում.:

`aws s3 mb s3://{{bucket_name}} --region {{region}}`

- Ցուցադրել օգնությունը.:

`aws s3 mb help`
