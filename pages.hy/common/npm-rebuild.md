# npm վերակառուցում

> Վերակառուցեք հայրենի Node.js փաթեթները Node-ի կամ կախվածության փոփոխություններից հետո:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-rebuild/>:.

- Վերակառուցեք հատուկ փաթեթ.:

`npm {{[rb|rebuild]}} {{package}}`

- Վերակառուցեք բոլոր տեղադրված փաթեթները.:

`npm {{[rb|rebuild]}}`

- Վերակառուցել մանրամասն ելքով.:

`npm {{[rb|rebuild]}} --verbose`

- Վերակառուցեք փաթեթը հատուկ գրացուցակում.:

`npm {{[rb|rebuild]}} --prefix {{path/to/dir}} {{package}}`

- Վերակառուցեք առանց npm քեշի օգտագործման.:

`npm {{[rb|rebuild]}} --no-cache`

- Վերակառուցել գլոբալ ռեժիմով.:

`npm {{[rb|rebuild]}} {{[-g|--global]}}`
