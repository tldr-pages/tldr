# git bugreport

> Գտեք վրիպազերծման տեղեկատվությունը համակարգից և օգտատերից՝ ստեղծելով տեքստային ֆայլ, որը կօգնի տեղեկացնել Git-ում սխալի մասին:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-bugreport>:.

- Ստեղծեք վրիպակի հաշվետվության նոր ֆայլ ընթացիկ գրացուցակում.:

`git bugreport`

- Ստեղծեք սխալի մասին նոր ֆայլ նշված գրացուցակում, ստեղծելով այն, եթե այն գոյություն չունի.:

`git bugreport {{[-o|--output-directory]}} {{path/to/directory}}`

- Ստեղծեք վրիպակի հաշվետվության նոր ֆայլ՝ նշված ֆայլի անվան վերջածանցով `strftime` ձևաչափով.:

`git bugreport {{[-s|--suffix]}} {{%m%d%y}}`
