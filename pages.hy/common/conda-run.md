# conda run

> Գործարկեք գործարկվող հրամանը conda միջավայրում:.
> Լրացուցիչ տեղեկություններ. <https://docs.conda.io/projects/conda/en/latest/commands/run.html>:.

- Գործարկեք հրաման ներկայիս ակտիվ միջավայրում.:

`conda run {{command}}`

- Թիրախավորեք միջավայրը անունով.:

`conda run {{[-n|--name]}} {{environment_name}} {{command}}`

- Թիրախավորել միջավայրն իր ճանապարհով (այսինքն՝ նախածանցով).:

`conda run {{[-p|--prefix]}} {{path/to/env}} {{command}}`
