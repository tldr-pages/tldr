# pbmtobbnbg

> Փոխարկել PBM պատկերը BitGraph գրաֆիկայի:.
> Լրացուցիչ տեղեկություններ. <https://netpbm.sourceforge.net/doc/pbmtobbnbg.html>:.

- Փոխակերպեք PBM պատկերը BitGraph տերմինալի Ցուցադրման պիքսել տվյալների հաջորդականության.:

`pbmtobbnbg < {{path/to/image.pbm}} > {{path/to/output.dpd}}`

- Նշեք rasterop-ը.:

`pbmtobbnbg < {{path/to/image.pbm}} {{3}} > {{path/to/output.dpd}}`
