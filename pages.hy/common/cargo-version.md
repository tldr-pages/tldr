# բեռնատար տարբերակ

> Ցուցադրել `cargo` տարբերակի մասին տեղեկությունները:.
> Լրացուցիչ տեղեկություններ. <https://doc.rust-lang.org/cargo/commands/cargo-version.html>:.

- Ցուցադրման տարբերակը:

`cargo version`

- Ցուցադրել լրացուցիչ շինարարական տեղեկատվություն.:

`cargo version {{[-v|--verbose]}}`
