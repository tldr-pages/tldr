# k6

> Բաց կոդով բեռնվածության փորձարկման գործիք և SaaS ինժեներական թիմերի համար:.
> Լրացուցիչ տեղեկություններ. <https://grafana.com/docs/k6/latest/using-k6/k6-options/reference/>:.

- Տեղադրեք բեռնվածության թեստը.:

`k6 run {{script.js}}`

- Գործարկեք բեռնվածության թեստը տեղական վիրտուալ օգտագործողների որոշակի քանակով և տևողությամբ.:

`k6 run {{[-u|--vus]}} {{10}} {{[-d|--duration]}} {{30s}} {{script.js}}`

- Գործարկեք բեռնվածության թեստը տեղում՝ տվյալ միջավայրի փոփոխականով.:

`k6 run {{[-e|--env]}} {{HOSTNAME=example.com}} {{script.js}}`

- Գործարկեք բեռնվածության թեստը տեղում՝ օգտագործելով InfluxDB՝ արդյունքները պահելու համար.:

`k6 run {{[-o|--out]}} influxdb={{http://localhost:8086/k6db}} {{script.js}}`

- Գործարկեք բեռնվածության թեստը տեղում և մերժեք արձագանքման մարմինները (զգալիորեն ավելի արագ).:

`k6 run --discard-response-bodies {{script.js}}`

- Տեղադրեք բեռնվածության թեստը՝ օգտագործելով JavaScript-ի բազային համատեղելիության ռեժիմը (զգալիորեն ավելի արագ).:

`k6 run --compatibility-mode=base {{script.js}}`

- Մուտք գործեք ամպային ծառայություն՝ օգտագործելով գաղտնի նշան.:

`k6 login cloud --token {{secret}}`

- Գործարկել բեռնվածության փորձարկումը ամպային ենթակառուցվածքի վրա.:

`k6 cloud {{script.js}}`
