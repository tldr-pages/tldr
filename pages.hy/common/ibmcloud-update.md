# ibmcloud թարմացում

> Թարմացրեք IBM Cloud CLI-ն վերջին հասանելի տարբերակին:.
> Լրացուցիչ տեղեկություններ. <https://cloud.ibm.com/docs/cli?topic=cli-ibmcloud_cli#ibmcloud_update>:.

- Թարմացրեք CLI-ն ամենավերջին տարբերակին.:

`ibmcloud update`

- Ստիպել թարմացում առանց հաստատման հուշումների.:

`ibmcloud update -f`
