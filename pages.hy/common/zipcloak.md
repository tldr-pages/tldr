# zipcloak

> Գաղտնագրեք բովանդակությունը Zip արխիվում:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/zipcloak>:.

- Գաղտնագրեք Zip արխիվի բովանդակությունը.:

`zipcloak {{path/to/archive.zip}}`

- Zip արխիվի բովանդակությունը վերծանել.:

`zipcloak {{[-d|--decrypt]}} {{path/to/archive.zip}}`

- Արտադրեք կոդավորված բովանդակությունը նոր Zip արխիվի մեջ.:

`zipcloak {{path/to/archive.zip}} {{[-O|--output-file]}} {{path/to/encrypted.zip}}`
