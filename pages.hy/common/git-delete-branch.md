# git ջնջել-ճյուղ

> Ջնջել տեղական և հեռավոր Git մասնաճյուղերը:.
> `git-extras`-ի մի մասը: Ստուգված մասնաճյուղը ջնջելու դեպքում կջնջվի միայն հեռավոր մասնաճյուղը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-delete-branch>:.

- Ջնջել մեկ կամ մի քանի տեղական և հեռավոր Git մասնաճյուղեր.:

`git delete-branch {{branch_name1 branch_name2 ...}}`
