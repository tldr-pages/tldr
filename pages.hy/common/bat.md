#չղջիկ

> Տպել և միացնել ֆայլերը:.
> `cat` կլոն՝ շարահյուսական ընդգծմամբ և Git ինտեգրմամբ:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/bat>:.

- Գեղեցիկ տպեք մեկ կամ մի քանի ֆայլերի բովանդակությունը `stdout`-ում:

`bat {{path/to/file1 path/to/file2 ...}}`

- Միացրեք մի քանի ֆայլ թիրախային ֆայլի մեջ.:

`bat {{path/to/file1 path/to/file2 ...}} > {{path/to/target_file}}`

- Հեռացրեք զարդերը և անջատեք էջավորումը.:

`bat {{[-pp|--style plain --pager never]}} {{path/to/file}}`

- Նշեք որոշակի գիծ կամ գծերի մի շարք տարբեր ֆոնի գույնով.:

`bat {{[-H|--highlight-line]}} {{10|5:10|:10|10:|10:+5}} {{path/to/file}}`

- Ցույց տալ չտպվող նիշերը, ինչպիսիք են բացատը, ներդիրը կամ նոր տողը.:

`bat {{[-A|--show-all]}} {{path/to/file}}`

- Հեռացրեք բոլոր զարդերը, բացառությամբ ելքի գծերի համարների.:

`bat {{[-n|--number]}} {{path/to/file}}`

- Շարահյուսությունը ընդգծում է JSON ֆայլը` հստակ սահմանելով լեզուն.:

`bat {{[-l|--language]}} json {{path/to/file.json}}`

- Ցուցադրել բոլոր աջակցվող լեզուները.:

`bat {{[-L|--list-languages]}}`
