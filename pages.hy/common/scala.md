# սկալա

> Scala հավելվածի գործարկիչ և ինտերակտիվ թարգմանիչ:.
> Լրացուցիչ տեղեկություններ. <https://scala-cli.virtuslab.org/docs/reference/cli-options/>:.

- Սկսեք REPL (ինտերակտիվ կեղև).:

`scala`

- Սկսեք թարգմանիչը [c]lass[p]ath-ում կախվածությամբ.:

`scala {{[-cp|--extra-jars]}} {{filename.jar}} {{command}}`

- Կատարեք Scala սցենար.:

`scala {{script.scala}}`

- Կատարեք `.jar` ծրագիր.:

`scala {{filename.jar}}`

- Կատարեք մեկ Scala հրաման.:

`scala {{[-e|--execute-script]}} {{command}}`
