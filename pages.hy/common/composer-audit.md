# կոմպոզիտորական աուդիտ

> Վերլուծեք PHP նախագծի կախվածությունները՝ հայտնաբերելու անվտանգության հայտնի խոցելիությունը և թվարկեք տուժած փաթեթները:.
> Տես նաև՝ `composer`:.
> Լրացուցիչ տեղեկություններ. <https://getcomposer.org/doc/03-cli.md#audit>:.

- Ստուգեք անվտանգության խոցելիությունը ձեր ընթացիկ նախագծում.:

`composer audit`

- Բաց թողեք մշակողի կախվածությունը աուդիտում.:

`composer audit --no-dev`

- Զտեք խոցելիությունը ելքային ձևաչափով.:

`composer audit --format {{table|plain|json|summary}}`

- Արտադրեք աուդիտի արդյունքները ֆայլի JSON ձևաչափով.:

`composer audit --format json > audit_report.json`

- Ստուգեք, թե արդյոք ձեր նախագծի կոնկրետ փաթեթի վրա ազդում են անվտանգության խնդիրները.:

`composer audit {{vendor}}/{{package}}`
