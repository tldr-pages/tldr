# mycli

> CLI MySQL-ի, MariaDB-ի և Percona-ի համար, որը կարող է կատարել ավտոմատ լրացում և շարահյուսական ընդգծում:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/mycli>:.

- Միացեք 3306 նավահանգստի տեղական տվյալների բազային՝ օգտագործելով ընթացիկ օգտվողի օգտանունը.:

`mycli {{database_name}}`

- Միացեք տվյալների շտեմարանին (օգտատիրոջը կառաջարկվի գաղտնաբառ).:

`mycli {{[-u|--user]}} {{username}} {{database_name}}`

- Միացեք տվյալների բազայի մեկ այլ հոսթի վրա.:

`mycli {{[-h|--host]}} {{database_host}} {{[-P|--port]}} {{port}} {{[-u|--user]}} {{username}} {{database_name}}`
