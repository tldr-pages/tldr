# nc

> Վերահղեք մուտքը/ելքը դեպի ցանցային հոսք այս բազմակողմանի գործիքի միջոցով:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/nc>:.

- Գործարկեք [l]istener նշված TCP [p]ort-ում և ֆայլ ուղարկեք դրա մեջ.:

`nc < {{filename}} -l -p {{port}}`

- Միացեք թիրախային լսողին նշված նավահանգստում և ստացեք ֆայլ դրանից.:

`nc {{host}} {{port}} > {{received_filename}}`

- Սկանավորեք նշված հոսթի բաց TCP նավահանգիստները.:

`nc -v -z -w {{timeout_in_seconds}} {{host}} {{start_port}}-{{end_port}}`

- Սկսեք [l]istener նշված TCP [p]ort-ում և տրամադրեք ձեր տեղական պատյան մուտքը միացված կողմին (սա վտանգավոր է և կարող է չարաշահվել).:

`nc -l -p {{port}} -e {{shell_executable}}`

- Միացեք թիրախային լսողին և ապահովեք ձեր տեղական կեղևի հասանելիությունը հեռավոր կողմին (սա վտանգավոր է և կարող է չարաշահվել).:

`nc {{host}} {{port}} -e {{shell_executable}}`

- Գործեք որպես վստահված անձ և փոխանցեք տվյալները տեղական TCP պորտից տվյալ հեռավոր հոսթին.:

`nc -l -p {{local_port}} | nc {{host}} {{remote_port}}`

- Ուղարկեք HTTP GET հարցում.:

`echo -e "GET / HTTP/1.1\nHost: {{host}}\n\n" | nc {{host}} 80`
