# npm ապատեղադրել

> Հեռացրեք փաթեթը:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-uninstall/>:.

- Հեռացրեք փաթեթը ընթացիկ նախագծից.:

`npm {{[r|uninstall]}} {{package_name}}`

- Հեռացրեք փաթեթը գլոբալ.:

`npm {{[r|uninstall]}} {{package_name}} {{[-g|--global]}}`

- Հեռացրեք մի քանի փաթեթներ միանգամից.:

`npm {{[r|uninstall]}} {{package_name1 package_name2 ...}}`
