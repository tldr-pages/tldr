# տրամաչափի-սերվեր

> Սերվերային հավելված՝ էլեկտրոնային գրքերը ցանցով տարածելու համար:.
> Նշում. էլեկտրոնային գրքերն արդեն պետք է ներմուծվեն գրադարան՝ օգտագործելով GUI կամ `calibredb` CLI:.
> Caliber էլեկտրոնային գրքերի գրադարանի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://manual.calibre-ebook.com/generated/en/calibre-server.html>:.

- Սկսեք սերվեր՝ էլեկտրոնային գրքեր տարածելու համար: Մուտք գործեք <http://localhost:8080> հասցեով՝:

`calibre-server`

- Սկսեք սերվերը տարբեր նավահանգստում: Մուտք գործեք <http://localhost:port>:

`calibre-server --port {{port}}`

- Գաղտնաբառը պաշտպանում է սերվերը.:

`calibre-server --username {{username}} --password {{password}}`
