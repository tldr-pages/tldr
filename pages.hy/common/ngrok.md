# ngrok

> Հակադարձ վստահված անձ, որը ստեղծում է անվտանգ թունել հանրային վերջնակետից մինչև տեղական գործող վեբ ծառայություն:.
> Լրացուցիչ տեղեկություններ. <https://ngrok.com/docs/agent/cli>:.

- Բացահայտեք տեղական HTTP ծառայությունը տվյալ նավահանգստում.:

`ngrok http {{80}}`

- Բացահայտեք տեղական HTTP ծառայությունը կոնկրետ հոսթի վրա.:

`ngrok http {{example.com}}:{{80}}`

- Բացահայտեք տեղական HTTPS սերվերը.:

`ngrok http https://localhost`

- Բացահայտեք TCP տրաֆիկը տվյալ նավահանգստում.:

`ngrok tcp {{22}}`

- Բացահայտեք TLS տրաֆիկը որոշակի հյուրընկալողի և նավահանգստի համար.:

`ngrok tls -hostname={{example.com}} {{443}}`
