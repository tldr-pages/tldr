# ղ պր

> Կառավարեք GitHub-ի ձգման հարցումները:.
> Որոշ ենթահրամաններ, ինչպիսիք են `create`-ն, ունեն իրենց օգտագործման փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://cli.github.com/manual/gh_pr>:.

- Ստեղծեք ձգման հարցում.:

`gh pr {{[new|create]}}`

- Ստուգեք տեղական ձգողականության հատուկ հարցում.:

`gh {{[co|pr checkout]}} {{pr_number|url|branch}}`

- Դիտեք ընթացիկ մասնաճյուղի ձգման հարցումում կատարված փոփոխությունները.:

`gh pr diff`

- Հաստատեք ընթացիկ մասնաճյուղի ձգման հարցումը.:

`gh pr review {{[-a|--approve]}}`

- Ինտերակտիվ կերպով միաձուլեք ընթացիկ մասնաճյուղի հետ կապված ձգման հարցումը.:

`gh pr merge`

- Ինտերակտիվ կերպով խմբագրել ձգման հարցումը.:

`gh pr edit`

- Խմբագրել ձգման հարցումի բազային ճյուղը.:

`gh pr edit {{[-B|--base]}} {{branch_name}}`

- Ստուգեք ընթացիկ պահեստի ձգման հարցումների կարգավիճակը.:

`gh pr status`
