# pytest

> Գործարկել Python թեստերը:.
> Լրացուցիչ տեղեկություններ. <https://docs.pytest.org/en/latest/how-to/usage.html>:.

- Գործարկել թեստեր հատուկ ֆայլերից.:

`pytest {{path/to/test_file1.py path/to/test_file2.py ...}}`

- Կատարեք թեստեր անուններով, որոնք համապատասխանում են կոնկրետ [k] հիմնաբառի արտահայտությանը.:

`pytest -k {{expression}}`

- Դուրս եկեք հենց որ թեստը ձախողվի կամ սխալի հանդիպի.:

`pytest {{[-x|--exitfirst]}}`

- Գործարկել թեստերը, որոնք համապատասխանում են կամ բացառում են մարկերները.:

`pytest -m {{marker_name1 and not marker_name2}}`

- Գործարկել մինչև թեստի ձախողումը, շարունակելով վերջին ձախողված թեստից.:

`pytest {{[--sw|--stepwise]}}`

- Գործարկել թեստերը առանց ելք վերցնելու.:

`pytest {{[-s|--capture=no]}}`

- Գործարկել թեստերը բարձրացված խոսակցությամբ՝ ցուցադրելով առանձին թեստի անուններ.:

`pytest {{[-v|--verbose]}}`
