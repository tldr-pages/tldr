# zeroclaw միգրացիա

> Տեղափոխեք տվյալները այլ գործակալների գործարկման ժամանակներից դեպի ZeroClaw:.
> Լրացուցիչ տեղեկություններ. <https://github.com/zeroclaw-labs/zeroclaw#quick-start>:.

- Նախադիտեք միգրացիան OpenClaw-ից առանց տվյալների գրելու.:

`zeroclaw migrate openclaw --dry-run`

- Տեղափոխել հիշողությունը OpenClaw աշխատանքային տարածքից.:

`zeroclaw migrate openclaw`

- Ցուցադրել օգնությունը.:

`zeroclaw migrate {{[-h|--help]}}`
