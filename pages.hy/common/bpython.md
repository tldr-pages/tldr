# bpython

> Շքեղ ինտերֆեյս Python թարգմանչի համար:.
> Ապահովում է շարահյուսության ընդգծում և շատ այլ հաճելի տարբերակներ REPL ռեժիմում:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/bpython>:.

- Սկսեք REPL (ինտերակտիվ կեղև).:

`bpython`

- Կատարեք հատուկ Python ֆայլ.:

`bpython {{path/to/file.py}}`

- Կատարեք հատուկ Python ֆայլ և սկսեք REPL.:

`bpython {{[-i|--interactive]}} {{path/to/file.py}}`

- Օգտագործեք նշված կազմաձևման ֆայլը լռելյայն կազմաձևման փոխարեն.:

`bpython --config {{path/to/file.conf}}`
