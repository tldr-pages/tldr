# rtl_sdr

> Հում տվյալների ձայնագրիչ RTL-SDR ընդունիչների համար:.
> Տվյալները կոդավորված են I/Q նմուշառման միջոցով (այսպես կոչված քառակուսի նմուշառում):.
> Լրացուցիչ տեղեկություններ. <https://osmocom.org/projects/rtl-sdr/wiki/Rtl-sdr>:.

- Պահպանեք RAW տվյալները հաճախականությունից (նշված է Հց-ով) ֆայլում.:

`rtl_sdr -f {{100000000}} {{path/to/file}}`

- Տվյալները փոխանցեք մեկ այլ ծրագրի.:

`rtl_sdr -f {{100000000}} - | {{aplay}}`

- Կարդացեք որոշակի քանակությամբ նմուշներ.:

`rtl_sdr -f {{100000000}} -n {{20}} -`

- Նշեք ընտրանքի արագությունը Հց-ով (225001-300000 և 900001-3200000 միջակայքերը).:

`rtl_sdr -f {{100000000}} -s {{2400000}} -`

- Նշեք սարքը իր ինդեքսով.:

`rtl_sdr -f {{100000000}} -d {{0}} -`

- Նշեք շահույթը.:

`rtl_sdr -f {{100000000}} -g {{20}} -`

- Նշեք ելքային բլոկի չափը.:

`rtl_sdr -f {{100000000}} -b {{9999999}} -`

- Օգտագործեք համաժամանակյա ելք.:

`rtl_sdr -f {{100000000}} -S -`
