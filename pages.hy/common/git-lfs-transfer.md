# git-lfs-փոխանցում

> Git LFS մաքուր SSH փոխանցման արձանագրության սերվերի կողմից իրականացում:.
> Թույլ է տալիս Git LFS-ին վերբեռնել և ներբեռնել մեծ ֆայլեր SSH-ով HTTPS-ի փոխարեն:.
> Լրացուցիչ տեղեկություններ. <https://github.com/charmbracelet/git-lfs-transfer#usage>:.

- Վերբեռնեք Git LFS-ի կողմից հետևված մեծ ֆայլերը պահեստ.:

`git-lfs-transfer {{path/to/repo.git}} upload`

- Ներբեռնեք Git LFS-ի կողմից հետևված մեծ ֆայլերը պահեստից.:

`git-lfs-transfer {{path/to/repo.git}} download`
