# jj էջանիշ

> Կառավարեք էջանիշները `jj` պահոցում:.
> Git backend-ն օգտագործելիս էջանիշները համապատասխանում են Git ճյուղերին:.
> Լրացուցիչ տեղեկություններ. <https://docs.jj-vcs.dev/latest/cli-reference/#jj-bookmark>:.

- Ստեղծեք նոր էջանիշ տվյալ վերանայման դեպքում.:

`jj {{[b|bookmark]}} {{[c|create]}} {{[-r|--revision]}} {{revision}} {{name}}`

- Թվարկեք բոլոր առկա էջանիշները և դրանց թիրախները.:

`jj {{[b|bookmark]}} {{[l|list]}}`

- Տեղափոխեք գոյություն ունեցող էջանիշը մեկ այլ վերանայում.:

`jj {{[b|bookmark]}} {{[m|move]}} {{[-t|--to]}} {{revision}} {{name}}`

- Հետևեք տրված հեռավոր էջանիշներին.:

`jj {{[b|bookmark]}} {{[t|track]}} {{name}}@{{remote}}`

- Ջնջեք էջանիշը և տարածեք ջնջումը հեռակառավարման սարքերին հաջորդ սեղմումով.:

`jj {{[b|bookmark]}} {{[d|delete]}} {{name}}`

- Մոռացեք էջանիշը տեղում, առանց նշելու, որ դրա ջնջումը պետք է մղվի.:

`jj {{[b|bookmark]}} {{[f|forget]}} {{name}}`
