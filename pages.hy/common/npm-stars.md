# npm աստղ

> Դիտեք փաթեթները, որոնք նշված են որպես ընտրյալ:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-stars/>:.

- Ցուցադրել բոլոր աստղանշված փաթեթները ներկայումս մուտք գործած օգտվողի կողմից.:

`npm stars`

- Ցույց տվեք ձեր աստղանշված փաթեթները որոշակի ռեեստրից.:

`npm stars --registry {{registry_url}}`

- Ցույց տալ փաթեթները, որոնք աստղանշված են որոշակի օգտվողի կողմից.:

`npm stars {{username}}`

- Ցույց տալ փաթեթները, որոնք աստղանշված են որոշակի օգտվողի կողմից որոշակի ռեեստրից.:

`npm stars {{username}} --registry {{registry_url}}`

- Ցուցադրել օգնությունը.:

`npm stars {{[-h|--help]}}`
