# git stash

> Պահպանեք տեղական Git-ի փոփոխությունները ժամանակավոր տարածքում:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-stash>:.

- Պահպանեք ընթացիկ չկատարված փոփոխությունները.:

`git stash`

- Պահպանեք ընթացիկ փոփոխությունները, ներառյալ նոր չհետևված ֆայլերը.:

`git stash {{[-u|--include-untracked]}}`

- Ինտերակտիվ կերպով ընտրեք փոփոխված ֆայլերի մասերը պահելու համար.:

`git stash {{[-p|--patch]}}`

- Թվարկեք բոլոր ստեղները.:

`git stash list`

- Ցույց տվեք փոփոխությունները որպես stash-ի և commit-ի միջև, երբ առաջին անգամ ստեղծվել է stash մուտքագրումը:

`git stash show {{[-p|--patch]}}`

- Կիրառեք պահոց և հեռացրեք այն պահոցների ցանկից, եթե դիմելը հակասություններ չի առաջացնում.:

`git stash pop`

- Ջնջել վերջին պահոցը.:

`git stash drop`

- Ջնջել բոլոր ստեղները.:

`git stash clear`
