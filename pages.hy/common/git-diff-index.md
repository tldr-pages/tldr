# git diff-index

> Համեմատեք աշխատանքային գրացուցակը commit կամ ծառի օբյեկտի հետ:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-diff-index>:.

- Համեմատեք աշխատանքային գրացուցակը կոնկրետ commit-ի հետ.:

`git diff-index {{commit}}`

- Համեմատեք աշխատանքային գրացուցակի որոշակի ֆայլ կամ գրացուցակ commit-ի հետ.:

`git diff-index {{commit}} {{path/to/file_or_directory}}`

- Համեմատեք աշխատանքային գրացուցակը ինդեքսի (բեմականացման տարածքի) հետ՝ փուլային փոփոխությունները ստուգելու համար.:

`git diff-index --cached {{commit}}`

- Ճեղքեք ելքը և վերադարձրեք ելքի կարգավիճակ՝ տարբերությունները ստուգելու համար.:

`git diff-index --quiet {{commit}}`
