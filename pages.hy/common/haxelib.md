# haxelib

> Haxe գրադարանի կառավարիչ:.
> Լրացուցիչ տեղեկություններ. <https://lib.haxe.org/documentation/using-haxelib/>:.

- Որոնել Haxe գրադարան.:

`haxelib search {{keyword}}`

- Տեղադրեք Haxe գրադարան.:

`haxelib install {{libname}}`

- Տեղադրեք Haxe գրադարանի հատուկ տարբերակը.:

`haxelib install {{libname}} {{version}}`

- Թարմացրեք բոլոր տեղադրված Haxe գրադարանները.:

`haxelib upgrade`

- Տեղադրեք գրադարանի զարգացման տարբերակը Git պահոցից.:

`haxelib git {{libname}} {{git_url}}`

- Տեղահանել Haxe գրադարանը.:

`haxelib remove {{libname}}`

- Տպեք տեղական տեղադրված Haxe գրադարանների ծառը.:

`haxelib list`
