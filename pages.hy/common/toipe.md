# toipe

> Եվս մեկ տպագրական թեստ, բայց խեցգետնի համով:.
> Վստահելի տերմինալի մուտքագրման փորձարկիչ:.
> Լրացուցիչ տեղեկություններ. <https://github.com/Samyak2/toipe>:.

- Սկսեք մուտքագրման թեստը լռելյայն բառացանկով.:

`toipe`

- Օգտագործեք կոնկրետ բառացանկ.:

`toipe {{[-w|--wordlist]}} {{wordlist_name}}`

- Օգտագործեք հատուկ բառացանկ.:

`toipe {{[-f|--file]}} {{path/to/file}}`

- Նշեք յուրաքանչյուր թեստի բառերի քանակը.:

`toipe {{[-n|--num]}} {{number_of_words}}`

- Ներառեք կետադրական նշաններ.:

`toipe {{[-p|--punctuation]}}`
