# git կողպեք

> Կողպեք ֆայլը Git-ի պահոցում, որպեսզի չփոփոխվի commit-ի կողմից:.
> `git-extras`-ի մի մասը:.
> Տես նաև՝ `git-unlock`:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-lock>:.

- Անջատեք տեղական ֆայլում փոփոխություններ կատարելու հնարավորությունը.:

`git lock {{path/to/file}}`
