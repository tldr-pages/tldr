# guacd

> Apache Guacamole վստահված անձի դաեմոն:.
> Աջակցեք բեռնիչին հաճախորդի հավելումների համար՝ Guacamole արձանագրության և ցանկացած կամայական հեռավոր աշխատասեղանի արձանագրության (օրինակ՝ RDP, VNC, Այլ) միջև ինտերֆեյսի համար:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/guacd>:.

- Կապվեք localhost-ի կոնկրետ նավահանգստի հետ.:

`guacd -b {{127.0.0.1}} -l {{4823}}`

- Սկսեք վրիպազերծման ռեժիմում՝ գործընթացը պահելով առաջին պլանում.:

`guacd -f -L {{debug}}`

- Սկսեք TLS աջակցությամբ.:

`guacd -C {{my-cert.crt}} -K {{my-key.pem}}`

- PID-ը գրեք ֆայլում՝:

`guacd -p {{path/to/file.pid}}`
