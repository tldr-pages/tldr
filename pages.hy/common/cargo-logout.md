# բեռի դուրս գալ

> Հեռացրեք API նշանը գրանցամատյանից տեղական մակարդակում:.
> Նշանն օգտագործվում է փաթեթի ռեեստրում իսկությունը հաստատելու համար: Դուք կարող եք այն նորից ավելացնել՝ օգտագործելով `cargo login`:.
> Լրացուցիչ տեղեկություններ. <https://doc.rust-lang.org/cargo/commands/cargo-logout.html>:.

- Հեռացրեք API նշանը տեղական հավատարմագրերի պահեստից (գտնվում է `$CARGO_HOME/credentials.toml`-ում).:

`cargo logout`

- Օգտագործեք նշված ռեեստրը (ռեգիստրի անունները կարող են սահմանվել կազմաձևում - լռելյայն է <https://crates.io>):

`cargo logout --registry {{name}}`
