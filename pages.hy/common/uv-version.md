# ուլտրամանուշակագույն տարբերակ

> Կարդացեք կամ թարմացրեք նախագծի տարբերակը:.
> Լրացուցիչ տեղեկություններ. <https://docs.astral.sh/uv/reference/cli/#uv-version>:.

- Ցուցադրել ծրագրի ընթացիկ տարբերակը.:

`uv version`

- Նախագծի տարբերակը սահմանեք որոշակի արժեքի.:

`uv version {{1.2.3}}`

- Բացեք նախագծի տարբերակը՝ օգտագործելով իմաստային տարբերակավորում.:

`uv version --bump {{major|minor|patch}}`

- Նախադիտեք տարբերակի փոփոխությունները՝ առանց գրելու `pyproject.toml`:

`uv version --bump {{patch}} --dry-run`

- Թարմացրեք տարբերակը որոշակի փաթեթի համար աշխատանքային տարածքում.:

`uv version --package {{package_name}} {{1.2.3}}`

- Ցուցադրել տարբերակը JSON ձևաչափով.:

`uv version --output-format json`
