# ng նոր

> Ստեղծեք և սկզբնավորեք նոր Angular հավելված:.
> Լրացուցիչ տեղեկություններ. <https://angular.dev/cli/new>:.

- Ստեղծեք նոր Angular հավելված.:

`ng {{[n|new]}} {{app_name}}`

- Նախադիտեք գործողությունները առանց ֆայլեր ստեղծելու.:

`ng {{[n|new]}} {{app_name}} {{[-d|--dry-run]}}`

- Բաց թողնել գեներացնող միավորի թեստային (`spec.ts`) ֆայլերը.:

`ng {{[n|new]}} {{app_name}} {{[-S|--skip-tests]}}`

- Բաց թողնել փաթեթի ավտոմատ տեղադրումը.:

`ng {{[n|new]}} {{app_name}} --skip-install`

- Բաց թողնել Git պահեստի սկզբնավորումը՝:

`ng {{[n|new]}} {{app_name}} {{[-g|--skip-git]}}`

- Կազմաձևեք AI գործիքավորումը նախագծի համար.:

`ng {{[n|new]}} {{app_name}} --ai-config {{claude|copilot|cursor|gemini|jetbrains|none|windsurf}}`

- Անջատել ինտերակտիվ հուշումները.:

`ng {{[n|new]}} {{app_name}} --interactive false`
