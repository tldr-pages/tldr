# cs

> Scala լեզվի հավելվածի և արտեֆակտի կառավարիչ:.
> Տեղադրում է Scala հավելվածները և ստեղծում Scala-ի զարգացման միջավայր:.
> Որոշ ենթահրամաններ, ինչպիսիք են `install`, `launch`, `java`, `fetch`, `resolve`, `complete-dep` և այլն, ունեն իրենց օգտագործման փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://get-coursier.io/docs/overview>:.

- Տեղադրված հավելվածների ցանկ.:

`cs list`

- Տեղադրեք հատուկ հավելված.:

`cs install {{application_name}}`

- Տեղահանել հատուկ հավելված.:

`cs uninstall {{application_name}}`

- Scala-ի մշակման մեքենա տեղադրելու համար.:

`cs setup`

- Թարմացրեք բոլոր տեղադրված հավելվածները.:

`cs update`

- Ցուցադրման տարբերակը:

`cs version`
