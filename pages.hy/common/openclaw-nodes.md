# openclaw հանգույցներ

> Կառավարեք OpenClaw հանգույցները (iOS, Android, macOS սարքեր):.
> Լրացուցիչ տեղեկություններ. <https://docs.openclaw.ai/cli/nodes>:.

- Թվարկեք բոլոր միացված և սպասող հանգույցները.:

`openclaw nodes list`

- Ցուցակ սպասող զուգավորման հարցումները.:

`openclaw nodes pending`

- Հաստատել առկախ հանգույցների զուգակցման հարցումը.:

`openclaw nodes approve {{request_id}}`

- Դիտեք հանգույցի կարգավիճակը.:

`openclaw nodes status`

- Ցուցադրել օգնությունը.:

`openclaw nodes {{[-h|--help]}}`
