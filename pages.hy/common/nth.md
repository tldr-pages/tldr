# n-րդ

> Name That Hash - Անմիջապես անվանեք ցանկացած հեշի տեսակը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/bee-san/Name-That-Hash>:.

- Անվանեք հաշ.:

`nth {{[-t|--text]}} {{5f4dcc3b5aa765d61d8327deb882cf99}}`

- Անվանական հեշեր ֆայլում.:

`nth {{[-f|--file]}} {{path/to/hashes}}`

- Տպել JSON ձևաչափով.:

`nth {{[-t|--text]}} {{5f4dcc3b5aa765d61d8327deb882cf99}} {{[-g|--greppable]}}`

- Վերծանեք հեշը Base64-ում նախքան այն անվանելը.:

`nth {{[-t|--text]}} {{NWY0ZGNjM2I1YWE3NjVkNjFkODMyN2RlYjg4MmNmOTkK}} {{[-b64|--base64]}}`
