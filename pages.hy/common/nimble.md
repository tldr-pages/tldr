# ճարպիկ

> Փաթեթի կառավարիչ Nim ծրագրավորման լեզվի համար:.
> Կառավարեք Nim նախագծերը և դրանց կախվածությունները:.
> Լրացուցիչ տեղեկություններ. <https://github.com/nim-lang/nimble>:.

- Փաթեթների որոնում.:

`nimble search {{search_string}}`

- Տեղադրեք փաթեթ.:

`nimble install {{package}}`

- Տեղադրված փաթեթների ցանկ.:

`nimble list {{[-i|--installed]}}`

- Ստեղծեք նոր Nimble փաթեթ ընթացիկ գրացուցակում.:

`nimble init`

- Կառուցեք ճարպիկ փաթեթ.:

`nimble build`

- Տեղադրեք Nimble փաթեթ.:

`nimble install`
