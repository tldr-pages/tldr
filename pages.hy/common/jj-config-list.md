# jj կազմաձևման ցուցակ

> Ցուցադրեք կազմաձևման ֆայլերում սահմանված փոփոխականները՝ դրանց արժեքների հետ միասին:.
> Տես նաև՝ `jj config get`:.
> Լրացուցիչ տեղեկություններ. <https://docs.jj-vcs.dev/latest/cli-reference/#jj-config-list>:.

- Թվարկեք բոլոր կազմաձևման փոփոխականները և դրանց արժեքները.:

`jj config {{[l|list]}}`

- Նշեք որոշակի կազմաձևման տարբերակ.:

`jj config {{[l|list]}} {{name}}`

- Ցուցակեք օգտագործողի մակարդակի կազմաձևման փոփոխականները.:

`jj config {{[l|list]}} --user`

- Ցուցակեք ռեպո մակարդակի կազմաձևման փոփոխականները.:

`jj config {{[l|list]}} --repo`

- Ցուցակեք կազմաձևման փոփոխականները, ներառյալ ներկառուցված լռելյայն արժեքները.:

`jj config {{[l|list]}} --include-defaults`

- Ցուցակեք կազմաձևման փոփոխականները, ներառյալ վերացված արժեքները.:

`jj config {{[l|list]}} --include-overridden`

- Ցուցակեք կազմաձևման փոփոխականները հատուկ ձևանմուշով.:

`jj config {{[l|list]}} {{[-T|--template]}} {{template}}`
