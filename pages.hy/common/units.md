# միավոր

> Փոխարկել երկու չափման միավորների միջև:.
> Լրացուցիչ տեղեկություններ. <https://www.gnu.org/software/units/manual/units.html>:.

- Գործարկել ինտերակտիվ ռեժիմով.:

`units`

- ինտերակտիվ ռեժիմում նշեք հատուկ տող պարունակող բոլոր միավորները.:

`search {{string}}`

- Ցույց տվեք փոխակերպումը երկու պարզ միավորների միջև.:

`units {{quarts}} {{tablespoons}}`

- Փոխարկել միավորների միջև քանակություններով.:

`units "{{15 pounds}}" {{kilograms}}`

- Ցույց տալ փոխակերպումը երկու բարդ միավորների միջև.:

`units "{{meters / second}}" "{{inches / hour}}"`

- Ցույց տալ փոխակերպումը տարբեր չափսերով միավորների միջև.:

`units "{{acres}}" "{{ft^2}}"`

- Ցույց տալ բայթերի բազմապատկիչների փոխակերպումը.:

`units "{{15 megabytes}}" {{bytes}}`
