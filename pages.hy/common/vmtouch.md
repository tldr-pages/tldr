# vmtouch

> Կառավարեք ֆայլային համակարգի քեշը:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/vmtouch>:.

- Տպել ֆայլի քեշի կարգավիճակը.:

`vmtouch {{path/to/file}}`

- Բեռնել ֆայլը քեշի մեջ.:

`vmtouch -t {{path/to/file}}`

- Վտարել ֆայլը քեշից.:

`vmtouch -e {{path/to/file}}`

- Կողպեք ֆայլը քեշում՝ հիշողությունից վտարումը կանխելու համար.:

`vmtouch -l {{path/to/file}}`

- Կողպեք ֆայլը և դևոնացրեք ծրագիրը.:

`vmtouch -ld {{path/to/file}}`
