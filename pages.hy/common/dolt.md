#դոլթ

> SQL տվյալների բազա, որը կարող եք պատառաքաղել, կլոնավորել, ճյուղավորել, միաձուլել, մղել և քաշել ճիշտ այնպես, ինչպես Git պահոցը:.
> Որոշ ենթահրամաններ, ինչպիսիք են `commit`-ն, ունեն իրենց օգտագործման փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://docs.dolthub.com/cli-reference/cli>:.

- Նախաձեռնեք նոր Dolt տվյալների շտեմարան.:

`dolt init`

- Ցույց տալ ընթացիկ աշխատանքային ծառի կարգավիճակը.:

`dolt status`

- Ավելացնել հեռավոր Dolt պահոց.:

`dolt remote add {{repository_name}} {{url}}`

- Սեղանների փուլային փոփոխություններ.:

`dolt add {{table_name}}`

- Գրանցեք շտեմարանային փոփոխությունները պահեստում.:

`dolt commit {{[-m|--message]}} "{{commit_message}}"`

- Տեղական փոփոխությունները տեղափոխեք հեռակառավարման վահանակ.:

`dolt push`

- Փոփոխությունները հեռակառավարման վահանակից հանեք.:

`dolt pull`
