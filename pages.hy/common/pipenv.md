# pipenv

> Պարզ և միասնական Python-ի մշակման աշխատանքային հոսք:.
> Կառավարեք փաթեթները և վիրտուալ միջավայրը նախագծի համար:.
> Լրացուցիչ տեղեկություններ. <https://pypi.org/project/pipenv>:.

- Ստեղծեք նոր նախագիծ.:

`pipenv`

- Ստեղծեք նոր նախագիծ՝ օգտագործելով Python 3:

`pipenv --three`

- Տեղադրեք փաթեթ.:

`pipenv install {{package}}`

- Տեղադրեք բոլոր կախվածությունները նախագծի համար.:

`pipenv install`

- Տեղադրեք բոլոր կախվածությունները նախագծի համար (ներառյալ մշակող փաթեթները).:

`pipenv install --dev`

- Ապատեղադրել փաթեթը.:

`pipenv uninstall {{package}}`

- Ստեղծված վիրտուալ միջավայրում սկսեք վահանակ.:

`pipenv shell`

- Ստեղծեք `requirements.txt` (կախվածությունների ցանկ) նախագծի համար.:

`pipenv lock --requirements`
