# մայրամուտ

> Կարգավորեք գունային ջերմաստիճանը Wayland կոմպոզիտորներում՝ կախված մայրամուտից/արևածագից:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/wlsunset>:.

- Ավտոմատ հաշվարկել մայրամուտի/արևածագի ժամանակները՝ հիմնվելով նշված վայրի վրա.:

`wlsunset -l {{latitude}} -L {{longitude}}`

- Ձեռքով սահմանեք մայրամուտի/արևածագի ժամերը (ժամանակի ձևաչափը՝ `HH:MM`):

`wlsunset -s {{sunset_time}} -S {{sunrise_time}}`

- Սահմանեք բարձր և ցածր ջերմաստիճանները (կանխադրված՝ բարձր՝ 6500, ցածր՝ 4000):

`wlsunset -T {{high_temp}} -t {{low_temp}}`

- Անիմացիայի ժամանակը սահմանեք `n` վայրկյանի (կիրառելի է միայն ձեռքով մայրամուտի/արևածագի ժամանակների օգտագործման դեպքում).:

`wlsunset -d {{n}} -s {{sunset_time}} -S {{sunrise_time}}`

- Սահմանեք գամմա (կանխադրված՝ 1.0):

`wlsunset -g {{gamma_value}}`
