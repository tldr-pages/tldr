# ipcalc

> Հաշվարկել IP տեղեկատվությունը (ենթացանց, հեռարձակում, հաղորդավարի տիրույթ) IP հասցեից և ցանցի դիմակից:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/ipcalc>:.

- Ցուցադրել ցանցի տվյալները IP հասցեի համար.:

`ipcalc {{192.168.0.1}}`

- Ցուցադրել ցանցի տվյալները՝ օգտագործելով CIDR նշումը.:

`ipcalc {{192.168.0.1}}/{{24}}`

- Ցուցադրել ցանցի տվյալները՝ օգտագործելով կետավոր տասնորդական ցանցային դիմակ.:

`ipcalc {{192.168.0.1}} {{255.255.255.0}}`

- Ճնշել բիթային ելքը.:

`ipcalc {{[-b|--nobinary]}} {{192.168.0.1}}`

- Ցանցը բաժանեք նշված չափի բլոկների.:

`ipcalc {{[-s|--split]}} {{size1 size2 size3 ...}} {{192.168.0.1}}`

- Ցուցադրման տարբերակը:

`ipcalc {{[-v|--version]}}`
