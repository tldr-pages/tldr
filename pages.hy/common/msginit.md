# msginit

> Ստեղծեք լեզվի հատուկ թարգմանչական ֆայլեր՝ հիմնված դյուրակիր օբյեկտների կաղապարների վրա:.
> Լրացուցիչ տեղեկություններ. <https://www.gnu.org/software/gettext/manual/gettext.html#msginit-Invocation>:.

- Ստեղծեք շարժական օբյեկտների ֆայլեր համակարգի տեղակայման վայրում՝ `messages.pot`-ից՝:

`msginit`

- Սահմանեք տեղայնությունը՝ որոշակի ձևանմուշից առաջացնելու համար.:

`msginit {{[-l|--locale]}} {{locale}} {{[-i|--input]}} {{path/to/messages.pot}}`

- Ցուցադրել օգնությունը.:

`msginit {{[-h|--help]}}`
