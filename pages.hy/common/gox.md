# gox

> Խաչաձև կազմեք Go ծրագրերը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/mitchellh/gox>:.

- Կազմեք Go ծրագիրը ընթացիկ գրացուցակում բոլոր օպերացիոն համակարգերի և ճարտարապետության համակցությունների համար.:

`gox`

- Ներբեռնեք և կազմեք Go ծրագիր հեռավոր URL-ից.:

`gox {{url_1 url_2 ...}}`

- Կազմել ընթացիկ գրացուցակը որոշակի օպերացիոն համակարգի համար.:

`gox -os="{{os}}"`

- Կազմել ընթացիկ գրացուցակը մեկ օպերացիոն համակարգի և ճարտարապետության համակցության համար.:

`gox -osarch="{{os}}/{{arch}}"`
