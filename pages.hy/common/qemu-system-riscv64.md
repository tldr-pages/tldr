# qemu-system-riscv64

> Ընդօրինակել `riscv64` ճարտարապետությունը:.
> Լրացուցիչ տեղեկություններ. <https://www.qemu.org/docs/master/system/target-riscv.html>:.

- Բեռնեք միջուկը, որը նմանակում է `riscv64` ճարտարապետությանը.:

`qemu-system-riscv64 {{[-M|-machine]}} virt -bios none -kernel {{kernel.elf}} -nographic`

- Ցուցակեք աջակցվող մեքենաների տեսակները.:

`qemu-system-riscv64 {{[-M|-machine]}} help`
