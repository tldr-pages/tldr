# zipnote

> Դիտեք, ավելացրեք կամ խմբագրեք Zip արխիվի մեկնաբանությունները:.
> Ֆայլերը կարող են նաև վերանվանվել Zip արխիվում:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/zipnote>:.

- Դիտեք մեկնաբանությունները Zip արխիվում.:

`zipnote {{path/to/file.zip}}`

- Քաղեք Zip արխիվի մեկնաբանությունները ֆայլի մեջ.:

`zipnote {{path/to/file.zip}} > {{path/to/file.txt}}`

- Ավելացնել/թարմացնել մեկնաբանությունները Zip արխիվում ֆայլից.:

`zipnote < {{path/to/file.txt}} -w {{path/to/file.zip}}`
