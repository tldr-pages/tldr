# yadm կոնֆիգուրացիա

> Փոխանցեք ընտրանքները yadm-ի կազմաձևման ֆայլին: Փոխեք yadm-ի կողմից կառավարվող պահեստի `.config`-ը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/TheLocehiliosan/yadm/blob/master/yadm.md#configuration>:.

- Սահմանեք կամ թարմացրեք yadm-ի Git կոնֆիգուրացիան.:

`yadm config {{key.inner-key}} {{value}}`

- Ստացեք արժեք yadm-ի Git կոնֆիգուրացիայից.:

`yadm config --get {{key}}`

- Չեղարկեք արժեքը yadm-ի Git-ի կազմաձևում.:

`yadm config --unset {{key}}`

- Թվարկեք բոլոր արժեքները yadm-ի Git կազմաձևման մեջ.:

`yadm config --list`
