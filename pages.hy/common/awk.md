#Ահկ

> Ֆայլերի վրա աշխատելու բազմակողմանի ծրագրավորման լեզու:.
> Նշում. AWK-ի տարբեր իրականացումները հաճախ սա դարձնում են իրենց երկուականի սիմվոլիկ:.
> Տես նաև՝ `gawk`:.
> Լրացուցիչ տեղեկություններ. <https://github.com/onetrueawk/awk>:.

- Տպեք հինգերորդ սյունակը (a.k.a. դաշտ) բացատով առանձնացված ֆայլում.:

`awk '{print $5}' {{path/to/file}}`

- Տպեք «foo» պարունակող տողերի երկրորդ սյունակը բացատով առանձնացված ֆայլում.:

`awk '/{{foo}}/ {print $2}' {{path/to/file}}`

- Տպեք յուրաքանչյուր տողի վերջին սյունակը ֆայլում, օգտագործելով ստորակետը (տարածքի փոխարեն) որպես դաշտի բաժանարար.:

`awk -F ',' '{print $NF}' {{path/to/file}}`

- Գումարեք ֆայլի առաջին սյունակի արժեքները և տպեք ընդհանուրը.:

`awk '{s+=$1} END {print s}' {{path/to/file}}`

- Տպեք յուրաքանչյուր երրորդ տողը՝ սկսած առաջին տողից.:

`awk 'NR%3==1' {{path/to/file}}`

- Տպել տարբեր արժեքներ՝ ելնելով պայմաններից.:

`awk '{if ($1 == "foo") print "Exact match foo"; else if ($1 ~ "bar") print "Partial match bar"; else print "Baz"}' {{path/to/file}}`

- Տպեք բոլոր այն տողերը, որոնց 10-րդ սյունակի արժեքը գտնվում է րոպեի և առավելագույնի միջև.:

`awk '($10 >= {{min_value}} && $10 <= {{max_value}})' {{path/to/file}}`

- Տպել օգտատերերի աղյուսակը >=1000 UID-ով վերնագրով և ֆորմատավորված ելքով՝ օգտագործելով երկու կետ որպես բաժանարար (`%-20s` միջինը՝ 20 ձախ հավասարեցված տողային նիշ, `%6s` նշանակում է՝ 6 տողերի աջ հավասարեցման նիշ):

`awk 'BEGIN {FS=":";printf "%-20s %6s %25s\n", "Name", "UID", "Shell"} $4 >= 1000 {printf "%-20s %6d %25s\n", $1, $4, $7}' /etc/passwd`
