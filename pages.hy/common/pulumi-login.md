# pulumi մուտք

> Մուտք գործեք Pulumi ամպ:.
> Լրացուցիչ տեղեկություններ. <https://www.pulumi.com/docs/iac/cli/commands/pulumi_login/>:.

- Մուտք գործեք կառավարվող Pulumi Cloud backend-ը, որը կանխադրված է <https://api.pulumi.com>:

`pulumi login`

- Մուտք գործեք ինքնակառավարվող Pulumi Cloud backend-ը նշված URL-ով.:

`pulumi login {{url}}`

- Օգտագործեք Pulumi-ն տեղական մակարդակում՝ անկախ ամպային հետևից.:

`pulumi login {{[-l|--local]}}`

- Ցուցադրել օգնությունը.:

`pulumi login {{[-h|--help]}}`
