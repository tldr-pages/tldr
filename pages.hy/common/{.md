# {

> Բազմաֆունկցիոնալ կեղևի շարահյուսություն:.
> Լրացուցիչ տեղեկություններ. <https://www.gnu.org/software/bash/manual/bash.html>:.

- Մեկուսացնել փոփոխականների անունները.:

`echo ${{{HOME}work}}`

- Հյուսվածքների ընդլայնման հաջորդականությունը.:

`echo {{{1..3}}} {{{a..c}}}{{{dir1,dir2,dir3}}}`

- Ստուգեք, արդյոք `variable`-ը սահմանված է նախքան տեքստը վերադարձնելը:

`echo ${{{variable:+variable is set and contains $variable}}}`

- Սահմանեք լռելյայն արժեքներ, եթե `variable`-ը կարգավորված չէ.:

`echo ${{{variable:-default}}}`

- Վերադարձրեք `variable` երկարությունը նիշերով՝:

`echo ${{{#variable}}}`

- Վերադարձեք լարային հատված.:

`echo ${{{variable:3:7}}}`

- Ռեկուրսիվորեն ընդլայնել `variable`:

`echo ${{{!variable}}}`

- Խմբի հրամանի արդյունքը միասին.:

`{ {{command1; command2; ...}} } | {{another_command}}`
