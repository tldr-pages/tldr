# թափառական կարգավիճակ

> Ցուցադրել մեքենաների վիճակը ներկայիս Vagrant միջավայրում:.
> Տես նաև՝ `vagrant`, `vagrant global-status`:.
> Լրացուցիչ տեղեկություններ. <https://developer.hashicorp.com/vagrant/docs/cli/status>:.

- Դիտեք մեքենաների կարգավիճակը ընթացիկ գրացուցակում.:

`vagrant status`

- Թիրախավորեք մեքենան անունով կամ ID-ով.:

`vagrant status {{name|id}}`
