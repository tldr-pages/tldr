# docker պատկերի պիտակ

> Նշանակել առկա Docker պատկերներին:.
> Լրացուցիչ տեղեկություններ. <https://docs.docker.com/reference/cli/docker/image/tag/>:.

- Նշեք անուն և պիտակ հատուկ պատկերի ID-ին.:

`docker {{[tag|image tag]}} {{id}} {{name}}:{{tag}}`

- Նշանակեք որոշակի պատկերին.:

`docker {{[tag|image tag]}} {{image}}:{{current_tag}} {{image}}:{{new_tag}}`

- Ցուցադրել օգնությունը.:

`docker {{[tag|image tag]}}`
