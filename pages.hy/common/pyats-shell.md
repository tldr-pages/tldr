# pyats shell

> Սկսեք նախապես բեռնված pyATS ինտերակտիվ Python Shell-ը` ժամանակ խնայելու նախատիպերի մշակման մեջ:.
> Լրացուցիչ տեղեկություններ. <https://pubhub.devnetcloud.com/media/genie-docs/docs/cli/genie_shell.html>:.

- Բացեք pyATS shell-ը սահմանված Testbed ֆայլով.:

`pyats shell --testbed-file {{path/to/testbed.yaml}}`

- Բացեք pyATS shell-ը սահմանված Pickle ֆայլով.:

`pyats shell --pickle-file {{path/to/pickle.file}}`

- Բացեք pyATS-ն անջատված IPython-ով.:

`pyats shell --no-ipython`
