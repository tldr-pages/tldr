# ղ օրգ

> Աշխատեք GitHub կազմակերպությունների հետ:.
> Լրացուցիչ տեղեկություններ. <https://cli.github.com/manual/gh_org>:.

- Նշեք կազմակերպությունները վավերացված օգտվողի համար.:

`gh org {{[ls|list]}}`

- Նշեք որոշակի քանակությամբ կազմակերպություններ (կանխադրված է 30).:

`gh org {{[ls|list]}} {{[-L|--limit]}} {{100}}`

- Ցուցադրել օգնությունը.:

`gh org --help`
