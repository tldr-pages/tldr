# rustup թարմացում

> Թարմացրեք Rust գործիքների շղթաները և `rustup`-ը (եթե տեղադրված չէ փաթեթների կառավարչի միջոցով):.
> Լրացուցիչ տեղեկություններ. <https://rust-lang.github.io/rustup/>:.

- Թարմացրեք բոլոր տեղադրված գործիքների շղթաները և `rustup`:

`rustup update`

- Տեղադրեք կամ թարմացրեք հատուկ գործիքաշար (տե՛ս `rustup help toolchain` լրացուցիչ տեղեկությունների համար).:

`rustup update {{toolchain}}`
