#կկատ

> Apache Kafka արտադրող և սպառողի գործիք:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/kcat>:.

- Սպառեք հաղորդագրությունները՝ սկսած նորագույն օֆսեթից.:

`kcat -C -t {{topic}} -b {{brokers}}`

- Սպառեք հաղորդագրությունները, որոնք սկսվում են ամենահին օֆսեթից և դուրս եկեք վերջին հաղորդագրությունը ստանալուց հետո.:

`kcat -C -t {{topic}} -b {{brokers}} -o beginning -e`

- Սպառեք հաղորդագրությունները որպես Կաֆկա սպառողների խումբ.:

`kcat -G {{group_id}} {{topic}} -b {{brokers}}`

- Հրապարակեք հաղորդագրությունը՝ կարդալով `stdin`-ից՝:

`echo {{message}} | kcat -P -t {{topic}} -b {{brokers}}`

- Հրապարակեք հաղորդագրությունները՝ կարդալով ֆայլից.:

`kcat -P -t {{topic}} -b {{brokers}} {{path/to/file}}`

- Ցուցակեք մետատվյալները բոլոր թեմաների և բրոքերների համար.:

`kcat -L -b {{brokers}}`

- Ցուցակե՛ք մետատվյալները կոնկրետ թեմայի համար.:

`kcat -L -t {{topic}} -b {{brokers}}`

- Շեղվեք թեմայի/բաժանման համար որոշակի ժամանակի համար.:

`kcat -Q -t {{topic}}:{{partition}}:{{unix_timestamp}} -b {{brokers}}`
