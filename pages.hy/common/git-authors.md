# git հեղինակներ

> Ստեղծեք Git պահեստի կոմիտերների ցուցակ:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/git-authors>:.

- Տպեք կոմիտերների ամբողջական ցանկը `stdout`-ում՝ `AUTHORS` ֆայլի փոխարեն:

`git authors {{[-l|--list]}}`

- Կոմիտատորների ցանկը ավելացրեք `AUTHORS` ֆայլին և բացեք այն լռելյայն խմբագրիչում.:

`git authors`

- Կոմիտատորների ցանկը, բացառությամբ էլ. նամակների, ավելացրեք `AUTHORS` ֆայլին և բացեք այն լռելյայն խմբագրիչում.:

`git authors --no-email`
