# bun-info

> Ցուցադրել փաթեթի մետատվյալները npm ռեեստրից:.
> Լրացուցիչ տեղեկություններ. <https://bun.com/docs/pm/cli/info>:.

- Ցուցադրել փաթեթի մետատվյալները.:

`bun info {{package_name}}`

- Ցուցադրել հատուկ տարբերակի մետատվյալները.:

`bun info {{package_name}}@{{version}}`

- Ցույց տալ փաթեթի որոշակի հատկություն.:

`bun info {{package_name}} {{property}}`

- Արդյունքները JSON ձևաչափով.:

`bun info {{package_name}} --json`
