# [[

> Ստուգեք ֆայլերի տեսակները և համեմատեք արժեքները:.
> Վերադարձնում է 0 կարգավիճակը, եթե պայմանը գնահատվում է true, 1, եթե այն գնահատվում է false:.
> Լրացուցիչ տեղեկություններ. <https://gnu.org/software/bash/manual/bash.html#index-_005b_005b>:.

- Ստուգեք, թե արդյոք տվյալ փոփոխականը հավասար է նշված տողի կամ գլոբին.:

`[[ ${{variable}} == "{{string}}" ]]`

- Փորձեք, արդյոք տվյալ տողը համապատասխանում է նշված `regex`-ին:

`[[ ${{variable}} =~ {{pattern}} ]]`

- Ստուգեք, թե արդյոք տվյալ փոփոխականը [e]qual/[n]ot [e]qual/[g]reater [t]han/[l]ess [t]han/[g]reater քան կամ [e]qual/[l]ss քան կամ [e]հավասար է նշված թվին.:

`[[ ${{variable}} -{{eq|ne|gt|lt|ge|le}} {{integer}} ]]`

- Ստուգեք, արդյոք նշված փոփոխականն ունի [n]դատարկ արժեք.:

`[[ -n ${{variable}} ]]`

- Ստուգեք, արդյոք նշված փոփոխականն ունի դատարկ արժեք ([z]ero երկարություն):

`[[ -z ${{variable}} ]]`

- Ստուգեք, արդյոք նշված [f]ile գոյություն ունի.:

`[[ -f {{path/to/file}} ]]`

- Ստուգեք, արդյոք նշված [d] գրացուցակը գոյություն ունի.:

`[[ -d {{path/to/directory}} ]]`

- Ստուգեք, արդյոք նշված ֆայլը կամ գրացուցակը [կա].:

`[[ -e {{path/to/file_or_directory}} ]]`
