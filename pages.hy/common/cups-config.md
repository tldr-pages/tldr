# cups-config

> Ցույց տալ տեխնիկական տեղեկատվությունը ձեր CUPS տպագիր սերվերի տեղադրման մասին:.
> Լրացուցիչ տեղեկություններ. <https://openprinting.github.io/cups/doc/man-cups-config.html>:.

- Ցույց տալ, թե որտեղ է ներկայումս տեղադրված CUPS-ը.:

`cups-config --serverbin`

- Ցույց տվեք CUPS-ի կազմաձևման գրացուցակի գտնվելու վայրը.:

`cups-config --serverroot`

- Ցույց տալ CUPS-ի տվյալների գրացուցակի գտնվելու վայրը.:

`cups-config --datadir`

- Ցուցադրել օգնությունը.:

`cups-config --help`

- Ցուցադրման տարբերակը:

`cups-config --version`
