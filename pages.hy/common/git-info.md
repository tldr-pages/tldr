# git տեղեկատվություն

> Ցուցադրել Git պահեստի տեղեկատվությունը:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/git-info>:.

- Ցուցադրել հեռավոր վայրերը, հեռավոր և տեղական մասնաճյուղերը, կատարման վերջին տվյալները և `.git/config` կարգավորումները.:

`git info`

- Ցուցադրել հեռավոր վայրերը, հեռավոր և տեղական մասնաճյուղերը և ամենավերջին պարտավորությունների տվյալները.:

`git info --no-config`
