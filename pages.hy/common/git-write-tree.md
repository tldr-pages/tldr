# git գրել-ծառ

> Ցածր մակարդակի օգտակար ծրագիր՝ ընթացիկ ինդեքսից ծառի օբյեկտ ստեղծելու համար:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-write-tree>:.

- Ստեղծեք ծառի օբյեկտ ընթացիկ ինդեքսից.:

`git write-tree`

- Ստեղծեք ծառի օբյեկտ՝ առանց ստուգելու, թե արդյոք գրացուցակի կողմից նշված օբյեկտները գոյություն ունեն օբյեկտների տվյալների բազայում.:

`git write-tree --missing-ok`

- Ստեղծեք ծառի օբյեկտ, որը ներկայացնում է ենթատեղեկատու (օգտագործվում է ենթածրագրի համար ծառի օբյեկտը գրելու համար նշված ենթագրքում).:

`git write-tree --prefix {{subdirectory}}/`
