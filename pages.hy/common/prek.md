#շփում

> Ստեղծեք Git կեռիկներ, որոնք աշխատում են նախքան պարտավորությունները:.
> Լրացուցիչ տեղեկություններ. <https://prek.j178.dev/reference/cli/>:.

- Տեղադրեք prek-ը ձեր Git կեռիկների մեջ.:

`prek install`

- Գործարկեք պրեկ կեռիկներ բոլոր փուլային ֆայլերի վրա.:

`prek run`

- Գործարկեք կեռիկներ բոլոր ֆայլերի վրա՝ բեմականացված կամ չբեմադրված.:

`prek run {{[-a|--all-files]}}`

- Մաքրել պրեկ քեշը.:

`prek cache clean`

- Թարմացրեք prek կազմաձևման ֆայլը ռեպոների վերջին տարբերակներին.:

`prek auto-update`
