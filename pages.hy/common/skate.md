# չմուշկ

> Պարզ և հզոր բանալի-արժեքի խանութ:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/skate>:.

- Պահպանեք բանալի և արժեք լռելյայն տվյալների բազայում.:

`skate set "{{key}}" "{{value}}"`

- Ցույց տվեք ձեր բանալիները, որոնք պահպանված են կանխադրված տվյալների բազայում.:

`skate list`

- Ջնջել բանալին և արժեքը կանխադրված տվյալների բազայից.:

`skate delete "{{key}}"`

- Ստեղծեք նոր բանալի և արժեք նոր տվյալների բազայում.:

`skate set "{{key}}"@"{{database_name}}" "{{value}}"`

- Ցույց տվեք ձեր բանալիները, որոնք պահպանված են ոչ լռելյայն տվյալների բազայում.:

`skate list @"{{database_name}}"`

- Ջնջել բանալին և արժեքը հատուկ տվյալների բազայից.:

`skate delete "{{key}}"@"{{database_name}}"`

- Ցույց տալ առկա տվյալների բազաները.:

`skate list-dbs`

- Ջնջեք տեղական db-ն և քաշեք թարմ պատճենը Charm Cloud-ից.:

`skate reset @"{{database_name}}"`
