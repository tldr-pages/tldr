# խումբ որը

> Ցուցադրել ցանցի ընթացիկ օգտագործումը ըստ գործընթացի, կապի կամ հեռավոր IP/հոսթի անվան:.
> Լրացուցիչ տեղեկություններ. <https://github.com/imsnif/bandwhich#usage>:.

- Ցույց տալ միայն հեռավոր հասցեների աղյուսակը.:

`bandwhich {{[-a|--addresses]}}`

- Ցույց տալ DNS հարցումները.:

`bandwhich {{[-s|--show-dns]}}`

- Ցույց տալ ընդհանուր (կուտակային) օգտագործումը.:

`bandwhich {{[-t|--total-utilization]}}`

- Ցուցադրել ցանցի օգտագործումը որոշակի ցանցային ինտերֆեյսի համար.:

`bandwhich {{[-i|--interface]}} {{eth0}}`

- Ցույց տալ DNS հարցումները տվյալ DNS սերվերով.:

`bandwhich {{[-s|--show-dns]}} {{[-d|--dns-server]}} {{dns_server_ip}}`
