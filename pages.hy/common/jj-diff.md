# jj տարբերություն

> Համեմատեք ֆայլի բովանդակությունը երկու վերանայումների միջև:.
> Լրացուցիչ տեղեկություններ. <https://docs.jj-vcs.dev/latest/cli-reference/#jj-diff>:.

- Ցույց տալ ընթացիկ վերանայման փոփոխությունները.:

`jj diff`

- Ցույց տալ տրված վերափոխումների փոփոխությունները (օրինակ՝ `B::D`, `A..D`, `B|C|D` և այլն):

`jj diff {{[-r|--revisions]}} {{revsets}}`

- Ցույց տալ փոփոխությունները տվյալ վերանայումից տվյալ վերանայում.:

`jj diff {{[-f|--from]}} {{from_revset}} {{[-t|--to]}} {{to_revset}}`

- Ցույց տալ տարբեր վիճակագրությունը.:

`jj diff --stat`

- Ցույց տալ Git-ֆորմատի տարբերությունը.:

`jj diff --git`
