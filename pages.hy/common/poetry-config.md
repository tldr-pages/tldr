# պոեզիայի կազմաձև

> Խմբագրել պոեզիայի կազմաձևման կարգավորումները և պահոցները:.
> Լրացուցիչ տեղեկություններ. <https://python-poetry.org/docs/cli/#config>:.

- Ցուցակ ընթացիկ կոնֆիգուրացիան.:

`poetry config --list`

- Հեռացրեք նախկինում սահմանված պարամետրը.:

`poetry config {{config_key}} --unset`

- Տեսեք որոշակի պարամետրի արժեքը.:

`poetry config {{config_key}}`

- Փոխեք կամ ավելացրեք նոր կազմաձևման կարգավորում՝ պարամետրի անունից հետո արժեք փոխանցելով.:

`poetry config {{config_key}} {{config_value}}`

- Տեղափոխել հնացած կազմաձևման կարգավորումները.:

`poetry config --migrate`

- Սահմանեք/ստացեք պարամետրեր, որոնք հատուկ են նախագծին.:

`poetry config --local`
