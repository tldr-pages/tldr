# բեկորներ

> Կախվածության կառավարման գործիք Crystal լեզվի համար:.
> Լրացուցիչ տեղեկություններ. <https://crystal-lang.org/reference/the_shards_command>:.

- Ստեղծեք կմախք `shard.yml` ֆայլ.:

`shards init`

- Տեղադրեք կախվածությունները `shard.yml` ֆայլից.:

`shards install`

- Թարմացրեք բոլոր կախվածությունները.:

`shards update`

- Թվարկեք բոլոր տեղադրված կախվածությունները.:

`shards list`

- Ցուցադրել կախվածության տարբերակը.:

`shards version {{path/to/dependency_directory}}`
