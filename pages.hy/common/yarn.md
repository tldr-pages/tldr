# մանվածք

> JavaScript և Node.js փաթեթների կառավարչի այլընտրանք:.
> Լրացուցիչ տեղեկություններ. <https://yarnpkg.com/cli>:.

- Տեղադրեք մոդուլ ամբողջ աշխարհում.:

`yarn global add {{module_name}}`

- Տեղադրեք բոլոր կախվածությունները, որոնք նշված են `package.json` ֆայլում (`install`-ը պարտադիր չէ):

`yarn install`

- Տեղադրեք մոդուլ և պահեք այն որպես կախվածություն `package.json` ֆայլին (ավելացրեք `--dev`՝ որպես մշակողի կախվածություն պահպանելու համար):

`yarn add {{module_name}}@{{version}}`

- Տեղահանեք մոդուլը և հեռացրեք այն `package.json` ֆայլից.:

`yarn remove {{module_name}}`

- Ինտերակտիվ կերպով ստեղծեք `package.json` ֆայլ.:

`yarn init`

- Բացահայտեք, թե արդյոք մոդուլը կախվածություն է և թվարկեք այլ մոդուլներ, որոնք կախված են դրանից.:

`yarn why {{module_name}}`
