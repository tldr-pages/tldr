# jj պիտակ

> Կառավարեք պիտակները `jj` պահոցում:.
> Որոշ ենթահրամաններ, ինչպիսիք են `delete`, `list`, `set`, ունեն իրենց օգտագործման փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://docs.jj-vcs.dev/latest/cli-reference/#jj-tag>:.

- Ստեղծեք պիտակ, որը ցույց է տալիս ընթացիկ աշխատանքային պատճենի վերանայումը.:

`jj tag {{[s|set]}} {{tag_name}}`

- Ստեղծեք պիտակ, որը ցույց է տալիս որոշակի վերանայում.:

`jj tag {{[s|set]}} {{tag_name}} {{[-r|--revision]}} {{revision}}`

- Նշեք բոլոր պիտակները.:

`jj tag {{[l|list]}}`

- Ցուցակել պիտակները, որոնք համապատասխանում են օրինաչափությանը, տեսակավորված ըստ կատարողի ամսաթվի (նախ ամենանորը).:

`jj tag {{[l|list]}} --sort committer-date- "{{pattern}}"`

- Տեղափոխեք գոյություն ունեցող պիտակը մեկ այլ վերանայման.:

`jj tag {{[s|set]}} {{tag_name}} {{[-r|--revision]}} {{revision}} --allow-move`

- Ջնջել պիտակը.:

`jj tag {{[d|delete]}} {{tag_name}}`

- Ջնջել գլոբալ օրինաչափությանը համապատասխանող պիտակները.:

`jj tag {{[d|delete]}} "{{glob:v1.*}}"`

- Ցուցադրել օգնությունը.:

`jj tag {{[-h|--help]}}`
