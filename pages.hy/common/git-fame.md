# git փառք

> Գեղեցիկ տպագիր Git պահեստի ներդրումներ:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/git-fame>:.

- Հաշվեք ներդրումները ընթացիկ Git պահոցի համար.:

`git fame`

- Բացառեք ֆայլերը/տեղեկատուները, որոնք համապատասխանում են նշված `regex`-ին.:

`git fame --excl "{{regex}}"`

- Հաշվարկել նշված ամսաթվից հետո կատարված ներդրումները.:

`git fame --since "{{3 weeks ago|2021-05-13}}"`

- Ցուցադրել ներդրումները նշված ձևաչափով.:

`git fame --format {{pipe|yaml|json|csv|tsv}}`

- Ցուցադրել ներդրումները մեկ ֆայլի ընդլայնման համար.:

`git fame {{[-t|--bytype]}}`

- Անտեսել բացատների փոփոխությունները.:

`git fame {{[-w|--ignore-whitespace]}}`

- Հայտնաբերեք միջֆայլերի տողերի շարժումները և պատճենները.:

`git fame -C`

- Հայտնաբերել ներֆայլի տողերի շարժումները և պատճենները.:

`git fame -M`
