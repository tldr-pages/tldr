# rustfmt

> Ձևաչափել Rust սկզբնական կոդը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/rust-lang/rustfmt>:.

- Ֆայլի ֆորմատավորում՝ տեղում վերագրելով բնօրինակ ֆայլը.:

`rustfmt {{path/to/source.rs}}`

- Ստուգեք ֆայլի ձևաչափումը և ցուցադրեք ցանկացած փոփոխություն վահանակի վրա.:

`rustfmt --check {{path/to/source.rs}}`

- Կրկնօրինակեք ցանկացած փոփոխված ֆայլ մինչև ձևաչափումը (բնօրինակ ֆայլը վերանվանվել է `.bk` ընդլայնմամբ):

`rustfmt --backup {{path/to/source.rs}}`

- Ձևաչափեք կոդը՝ օգտագործելով Rust ոճի հատուկ հրատարակություն (ձևաչափման կանոններ) բացառապես.:

`rustfmt --style-edition {{2015|2018|2021|2024}} {{[-v|--verbose]}} {{path/to/source1.rs path/to/source2.rs ...}}`

- Ձևաչափեք կոդը՝ օգտագործելով Rust-ի հատուկ հրատարակություն (լեզվի առանձնահատկություններ և վերլուծություն).:

`rustfmt --edition {{2015|2018|2021|2024}} {{path/to/source1.rs path/to/source2.rs ...}}`
