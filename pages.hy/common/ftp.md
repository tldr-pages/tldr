# ftp

> Գործիքներ սերվերի հետ Ֆայլերի փոխանցման արձանագրության միջոցով փոխազդելու համար:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/ftp>:.

- Միացեք FTP սերվերին և գործարկեք ինտերակտիվ ռեժիմով.:

`ftp {{ftp.example.com}}`

- Միացեք FTP սերվերին՝ նշելով դրա IP հասցեն և նավահանգիստը.:

`ftp {{ip_address}} {{port}}`

- [Ինտերակտիվ] Անցում երկուական փոխանցման ռեժիմին (գրաֆիկա, սեղմված ֆայլեր և այլն).:

`binary`

- [Ինտերակտիվ] Տեղափոխեք բազմաթիվ ֆայլեր՝ առանց յուրաքանչյուր ֆայլի վրա հաստատում պահանջելու.:

`prompt off`

- [Ինտերակտիվ] Ներբեռնեք բազմաթիվ ֆայլեր (գլոբալ արտահայտություն).:

`mget {{*.png}}`

- [Ինտերակտիվ] Վերբեռնեք բազմաթիվ ֆայլեր (գլոբալ արտահայտություն).:

`mput {{*.zip}}`

- [Ինտերակտիվ] Ջնջել բազմաթիվ ֆայլեր հեռավոր սերվերի վրա.:

`mdelete {{*.txt}}`

- [Ինտերակտիվ] Վերանվանել ֆայլը հեռավոր սերվերի վրա.:

`rename {{original_filename}} {{new_filename}}`
