# tokei

> Ցուցադրել վիճակագրությունը կոդի վերաբերյալ:.
> Լրացուցիչ տեղեկություններ. <https://github.com/XAMPPRocky/tokei>:.

- Ցուցադրեք հաշվետվություն կոդի համար գրացուցակում և բոլոր ենթագրքերում.:

`tokei {{path/to/directory}}`

- Ցուցադրել հաշվետվություն գրացուցակի համար, բացառությամբ `.min.js` ֆայլերի:

`tokei {{path/to/directory}} {{[-e|--exclude]}} {{*.min.js}}`

- Ցուցադրել վիճակագրությունը առանձին ֆայլերի համար գրացուցակում.:

`tokei {{path/to/directory}} {{[-f|--files]}}`

- Ցուցադրել հաշվետվություն Rust և Markdown տիպի բոլոր ֆայլերի համար.:

`tokei {{path/to/directory}} {{[-t|--type]}} {{Rust}},{{Markdown}}`
