# nodenv

> Կառավարեք Node.js տարբերակները:.
> Լրացուցիչ տեղեկություններ. <https://github.com/nodenv/nodenv>:.

- Տեղադրեք Node.js-ի կոնկրետ տարբերակը.:

`nodenv install {{version}}`

- Ցուցադրել առկա տարբերակների ցանկը.:

`nodenv install --list`

- Օգտագործեք Node.js-ի հատուկ տարբերակը ամբողջ համակարգում.:

`nodenv global {{version}}`

- Օգտագործեք Node.js-ի հատուկ տարբերակը գրացուցակով.:

`nodenv local {{version}}`

- Ցուցադրել Node.js տարբերակը ընթացիկ գրացուցակի համար.:

`nodenv version`

- Ցուցադրել Node.js տեղադրված հրամանի գտնվելու վայրը (օրինակ՝ `npm`):

`nodenv which {{command}}`
