# բեռների մուտք

> Պահպանեք API նշանը գրանցամատյանից տեղում:.
> Նշանն օգտագործվում է փաթեթի ռեեստրում իսկությունը հաստատելու համար: Դուք կարող եք հեռացնել այն՝ օգտագործելով `cargo logout`:.
> Լրացուցիչ տեղեկություններ. <https://doc.rust-lang.org/cargo/commands/cargo-login.html>:.

- Ավելացրեք API նշան տեղական հավատարմագրերի պահեստին (գտնվում է `$CARGO_HOME/credentials.toml`-ում):

`cargo login`

- Օգտագործեք նշված ռեեստրը (ռեգիստրի անունները կարող են սահմանվել կազմաձևում - լռելյայն է <https://crates.io>):

`cargo login --registry {{name}}`
