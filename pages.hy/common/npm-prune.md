# npm սալորաչիր

> Հեռացրեք կողմնակի փաթեթները `node_modules`-ից:.
> Ծանոթագրություն․ կողմնակի փաթեթներն այն փաթեթներն են, որոնք առկա են node_modules պանակում, որոնք նշված չեն որպես որևէ փաթեթի կախվածության ցանկ։
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-prune/>:.

- Հեռացրեք բոլոր կողմնակի փաթեթները, որոնք նշված չեն կախվածության մեջ.:

`npm prune`

- Հեռացրեք կողմնակի փաթեթները և devDependencies (օգտակար արտադրության համար).:

`npm prune --production`

- Ցույց տալ, թե ինչ կհեռացվի առանց որևէ փոփոխություն կատարելու.:

`npm prune --dry-run`

- Փոփոխությունները թողարկեք որպես JSON:

`npm prune --json`

- Հեռացրեք հատուկ փաթեթներ անունով.:

`npm prune {{package_name}}`
