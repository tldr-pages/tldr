# dvc

> Տվյալների տարբերակի կառավարման համակարգ մեքենայական ուսուցման նախագծերի համար:.
> Որոշ ենթահրամաններ, ինչպիսիք են `commit`-ն, ունեն իրենց օգտագործման փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://doc.dvc.org/command-reference>:.

- Նախաձեռնեք նոր DVC նախագիծ.:

`dvc init`

- Կարգավորեք հեռավոր պահեստավորման վայրը.:

`dvc remote add {{storage_name}} {{url}}`

- Ավելացրեք մեկ կամ մի քանի տվյալների ֆայլ կամ գրացուցակ հետևելու համար.:

`dvc add {{path/to/file_or_directory}}`

- Ցույց տալ ծրագրի կարգավիճակը.:

`dvc status`

- Վերբեռնեք հետևված ֆայլերը հեռավոր պահեստում.:

`dvc push`

- Ներբեռնեք հետևված ֆայլերը հեռավոր պահեստից.:

`dvc pull`

- Ցուցադրել օգնությունը.:

`dvc {{[-h|--help]}}`

- Ցուցադրման տարբերակը:

`dvc --version`
