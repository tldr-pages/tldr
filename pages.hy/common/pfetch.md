# pfetch

> Ցուցադրել համակարգի տեղեկատվությունը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/dylanaraps/pfetch>:.

- Ցուցադրել ASCII արվեստի և լռելյայն դաշտերը.:

`pfetch`

- Ցուցադրել միայն ASCII արվեստի և գունային պալիտրա դաշտերը.:

`PF_INFO="{{ascii palette}}" pfetch`

- Ցուցադրել բոլոր հնարավոր դաշտերը.:

`PF_INFO="{{ascii title os host kernel uptime pkgs memory shell editor wm de palette}}" pfetch`

- Ցուցադրել այլ օգտվողի անուն և հյուրընկալող անուն.:

`USER="{{user}}" HOSTNAME="{{hostname}}" pfetch`

- Ցուցադրել առանց գույների.:

`PF_COLOR={{0}} pfetch`
