# npm-տուն

> Բացեք վեբ բրաուզերում փաթեթի `npm` էջը, Yarn էջը կամ GitHub-ի պահոցը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/sindresorhus/npm-home>:.

- Բացեք որոշակի փաթեթի `npm` էջը վեբ բրաուզերում.:

`npm-home {{package}}`

- Բացեք հատուկ փաթեթի GitHub պահոցը վեբ բրաուզերում.:

`npm-home {{[-g|--github]}} {{package}}`

- Բացեք կոնկրետ փաթեթի Yarn էջը վեբ բրաուզերում.:

`npm-home {{[-y|--yarn]}} {{package}}`
