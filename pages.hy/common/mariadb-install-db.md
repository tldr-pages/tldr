# mariadb-install-db

> Նախաձեռնել MariaDB տվյալների բազա:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/mariadb-install-db>:.

- Նախաձեռնել տվյալների բազա.:

`sudo mariadb-install-db --user {{user}} --basedir {{/usr}} --datadir {{/var/lib/mysql}}`

- Ցուցադրել օգնությունը.:

`mariadb-install-db --help`
