#կոնան

> Բաց կոդով, ապակենտրոնացված և միջպլատֆորմային փաթեթների կառավարիչ՝ ստեղծելու և համօգտագործելու ձեր բոլոր բնիկ երկուականները:.
> Որոշ ենթահրամաններ, ինչպիսիք են `frogarian`-ն, ունեն իրենց օգտագործման փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://docs.conan.io/2/reference/commands.html>:.

- Տեղադրեք փաթեթներ՝ հիմնված `conanfile.txt`-ի վրա.:

`conan install {{.}}`

- Տեղադրեք փաթեթներ և ստեղծեք կազմաձևման ֆայլեր որոշակի գեներատորի համար.:

`conan install -g {{generator}}`

- Տեղադրեք փաթեթներ, կառուցել աղբյուրից.:

`conan install {{.}} --build`

- Տեղական տեղադրված փաթեթների որոնում.:

`conan search {{package}}`

- Որոնեք հեռավոր փաթեթներ.:

`conan search {{package}} -r {{remote}}`

- Ցուցակ հեռակառավարման սարքեր.:

`conan remote list`
