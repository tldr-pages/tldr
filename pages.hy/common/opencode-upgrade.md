# բաց կոդի թարմացում

> Թարմացրեք OpenCode-ը վերջին տարբերակին կամ կոնկրետ տարբերակին:.
> Լրացուցիչ տեղեկություններ. <https://opencode.ai/docs/cli#upgrade>:.

- Թարմացրեք վերջին տարբերակին.:

`opencode upgrade`

- Թարմացնել/նվազեցնել որոշակի տարբերակի.:

`opencode upgrade {{1.1.10}}`

- Թարմացրեք տեղադրման հատուկ մեթոդով.:

`opencode upgrade {{[-m|--method]}} {{curl|npm|pnpm|bun|brew|choco|scoop}}`

- Ցուցադրել օգնությունը.:

`opencode upgrade {{[-h|--help]}}`
