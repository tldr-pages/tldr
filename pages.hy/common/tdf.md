# tdf

> Tui PDF դիտող:.
> Լրացուցիչ տեղեկություններ. <https://github.com/itsjunetime/tdf>:.

- Բացեք հատուկ ֆայլ.:

`tdf {{path/to/file}}.pdf`

- Բացեք լիաէկրան ռեժիմով.:

`tdf {{[-f|--fullscreen]}} {{path/to/file}}.pdf`

- Սահմանեք սովորական սպիտակ գույն.:

`tdf {{[-w|--white-color]}} {{ffffff}} {{path/to/file}}.pdf`

- Սահմանեք սովորական սև գույն.:

`tdf {{[-b|--black-color]}} "rgb({{0}}, {{0}}, {{0}})" {{path/to/file}}.pdf`
