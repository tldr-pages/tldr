# ibmcloud թիրախ

> Սահմանեք կամ դիտեք թիրախային հաշիվը, տարածաշրջանը կամ ռեսուրսների խումբը:.
> Լրացուցիչ տեղեկություններ. <https://cloud.ibm.com/docs/cli?topic=cli-ibmcloud_cli#ibmcloud_target>:.

- Դիտեք ընթացիկ թիրախային հաշիվը և տարածաշրջանը.:

`ibmcloud target`

- Սահմանեք թիրախային հաշիվը.:

`ibmcloud target -c {{account_id}}`

- Անցում դեպի որոշակի տարածաշրջան.:

`ibmcloud target -r {{region_name}}`

- Սահմանեք թիրախային ռեսուրսների խումբը.:

`ibmcloud target -g {{resource_group_name}}`

- Մաքրել թիրախային շրջանը.:

`ibmcloud target --unset-region`

- Մաքրել թիրախային ռեսուրսների խումբը.:

`ibmcloud target --unset-resource-group`
