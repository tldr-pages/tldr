# բեռների մետատվյալներ

> Արտադրեք աշխատանքային տարածքի անդամները և ընթացիկ փաթեթի լուծված կախվածությունները որպես JSON:.
> Նշում. ելքային ձևաչափը ենթակա է փոփոխման Cargo-ի հետագա տարբերակներում:.
> Լրացուցիչ տեղեկություններ. <https://doc.rust-lang.org/cargo/commands/cargo-metadata.html>:.

- Տպեք աշխատանքային տարածքի անդամները և ընթացիկ փաթեթի լուծված կախվածությունները.:

`cargo metadata`

- Տպեք միայն աշխատանքային տարածքի անդամները և մի առեք կախվածություն.:

`cargo metadata --no-deps`

- Տպել մետատվյալները որոշակի ձևաչափով` հիմնված նշված տարբերակի վրա.:

`cargo metadata --format-version {{version}}`

- Տպեք մետատվյալները `resolve` դաշտով, ներառյալ կախվածությունը միայն տվյալ թիրախային եռակի համար (Նշում. `packages` զանգվածը դեռևս կներառի բոլոր թիրախների կախվածությունները).:

`cargo metadata --filter-platform {{target_triple}}`
