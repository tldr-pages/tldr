# git ժայռ

> Բարձր կարգավորելի փոփոխության մատյան գեներատոր:.
> Լրացուցիչ տեղեկություններ. <https://git-cliff.org/docs/usage/args/>:.

- Ստեղծեք փոփոխության մատյան բոլոր պարտավորություններից Git պահոցում և պահեք այն `CHANGELOG.md`-ում:

`git cliff > {{CHANGELOG.md}}`

- Ստեղծեք փոփոխությունների գրանցամատյան՝ սկսած վերջին պիտակից, և տպեք այն `stdout`:

`git cliff {{[-l|--latest]}}`

- Ստեղծեք փոփոխության մատյան այն հանձնառություններից, որոնք պատկանում են ընթացիկ պիտակին (օգտագործեք `git checkout` պիտակի վրա մինչ այս):

`git cliff --current`

- Ստեղծեք փոփոխության մատյան այն պարտավորություններից, որոնք չեն պատկանում պիտակին.:

`git cliff {{[-u|--unreleased]}}`

- Ներկայիս գրացուցակում գրեք կանխադրված կազմաձևման ֆայլը `cliff.toml`.:

`git cliff {{[-i|--init]}}`
