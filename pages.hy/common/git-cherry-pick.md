# git cherry-pick

> Կիրառել առկա պարտավորությունների կողմից ներկայացված փոփոխությունները ընթացիկ մասնաճյուղում:.
> Մեկ այլ մասնաճյուղում փոփոխություններ կիրառելու համար նախ օգտագործեք `git checkout`՝ ցանկալի մասնաճյուղին անցնելու համար:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-cherry-pick>:.

- Կիրառեք պարտավորություն ընթացիկ մասնաճյուղում.:

`git cherry-pick {{commit}}`

- Կիրառեք մի շարք պարտավորություններ ընթացիկ մասնաճյուղում (տես նաև՝ `git rebase --onto`):

`git cherry-pick {{start_commit}}~..{{end_commit}}`

- Կիրառեք բազմաթիվ (ոչ հաջորդական) պարտավորություններ ընթացիկ մասնաճյուղում.:

`git cherry-pick {{commit1 commit2 ...}}`

- Աշխատանքային գրացուցակում ավելացրեք commit-ի փոփոխությունները՝ առանց commit ստեղծելու.:

`git cherry-pick {{[-n|--no-commit]}} {{commit}}`
