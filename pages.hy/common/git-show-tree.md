# git show-tree

> Ցույց տվեք զարդարված ծառի գրաֆիկը Git պահեստի բոլոր ճյուղերով՝ ցույց տալով ծանոթագրություններ:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-show-tree>:.

- Ցույց տվեք զարդարված ծառի գրաֆիկ բոլոր ճյուղերի համար, որոնք նշում են պիտակներով և ճյուղերի անուններով.:

`git show-tree`
