# բուլկի

> JavaScript-ի գործարկման ժամանակ և գործիքակազմ:.
> Ներառում է փաթեթավորող, փորձնական վազող և փաթեթի կառավարիչ:.
> Լրացուցիչ տեղեկություններ. <https://bun.com/docs>:.

- Ստեղծեք նոր Bun նախագիծ ընթացիկ գրացուցակում.:

`bun init`

- Գործարկեք JavaScript ֆայլ կամ `package.json` սկրիպտ՝:

`bun run {{path/to/file|script_name}}`

- Գործարկել միավորի թեստերը.:

`bun test`

- Ներբեռնեք և տեղադրեք բոլոր փաթեթները, որոնք նշված են որպես կախվածություն `package.json`-ում:

`bun {{[i|install]}}`

- Ավելացրեք կախվածություն `package.json`-ին:

`bun {{[a|add]}} {{module_name}}`

- Հեռացրեք կախվածությունը `package.json`-ից:

`bun {{[rm|remove]}} {{module_name}}`

- Սկսեք REPL (ինտերակտիվ կեղև).:

`bun repl`

- Թարմացրեք Bun-ը վերջին տարբերակին.:

`bun upgrade`
