# tofu ելք

> Արտահանեք կառուցվածքային տվյալներ ձեր OpenTofu ռեսուրսների մասին:.
> Լրացուցիչ տեղեկություններ. <https://opentofu.org/docs/cli/commands/output/>:.

- Ցուցադրել արմատային մոդուլի բոլոր ելքերը.:

`tofu output`

- Արտադրեք միայն որոշակի անունով արժեք.:

`tofu output {{name}}`

- Վերափոխեք ելքային արժեքը չմշակված տողի (օգտակար կեղևի սկրիպտների համար).:

`tofu output -raw`

- Ձևաչափեք ելքերը որպես JSON օբյեկտ՝ յուրաքանչյուր ելքի համար ստեղնով (օգտակար `jq`-ի հետ):

`tofu output -json`
