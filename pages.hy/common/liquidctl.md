# հեղուկքլ

> Կառավարեք հեղուկ հովացուցիչներ:.
> Լրացուցիչ տեղեկություններ. <https://github.com/liquidctl/liquidctl>:.

- Ցուցակեք առկա սարքերը.:

`liquidctl list`

- Նախաձեռնել բոլոր աջակցվող սարքերը.:

`sudo liquidctl initialize all`

- Տպել առկա հեղուկ հովացուցիչների կարգավիճակը.:

`liquidctl status`

- Համապատասխանեցրեք արտադրանքի անվանման տողը սարք ընտրելու և դրա արագությունը 50% սահմանելու համար.:

`liquidctl {{[-m|--match]}} {{string}} set {{fan|pump|...}} speed 50`

- Սահմանեք օդափոխիչի արագության աստիճանական կորը, որը 0% է 20°C-ում, 50% 50°C և 100% 70°C-ում:

`liquidctl set {{fan|pump|...}} speed 20 0 50 50 70 100`

- Սահմանեք հեղուկ հովացուցիչի RGB լույսերը (աջակցվող ռեժիմները կախված են մոդելից).:

`liquidctl set {{fan|pump|...}} color {{fixed|fading|rainbow|...}} {{00ff00}}`
