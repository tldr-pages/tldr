# հիմք64

> Կոդավորել կամ վերծանել ֆայլը կամ `stdin`-ից/base64-ից մինչև `stdout`:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/base64>:.

- Կոդավորել ֆայլը.:

`base64 {{path/to/file}}`

- Փաթաթել կոդավորված ելքը որոշակի լայնությամբ (`0`-ն անջատում է փաթաթումը):

`base64 {{[-w|--wrap]}} {{0|76|...}} {{path/to/file}}`

- Վերծանել ֆայլը.:

`base64 {{[-d|--decode]}} {{path/to/file}}`

- Կոդավորում `stdin`-ից՝:

`{{command}} | base64`

- Վերծանել `stdin`-ից՝:

`{{command}} | base64 {{[-d|--decode]}}`
