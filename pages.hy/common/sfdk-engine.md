# sfdk շարժիչ

> Պահպանեք և վերահսկեք շինարարական շարժիչը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/sailfishos/sailfish-qtcreator/blob/master/share/qtcreator/sfdk/modules/30-building-maintain/doc/command.engine.adoc>:.

- Գործարկել շինարարական շարժիչը.:

`sfdk engine start`

- Դադարեցրեք շինարարական շարժիչը.:

`sfdk engine stop`

- Ցույց տալ շինարարական շարժիչի կարգավիճակը.:

`sfdk engine status`

- Սահմանել շարժիչի հատկությունը.:

`sfdk engine set {{property}}={{value}}`

- Ցույց տալ շարժիչի հատկությունները.:

`sfdk engine show`

- Կատարեք հրաման կառուցման շարժիչի վրա.:

`sfdk engine exec {{command}}`

- Սկսեք ինտերակտիվ կեղևը շինարարական շարժիչի վրա.:

`sfdk engine exec`
