# git init

> Նախաձեռնեք նոր տեղական Git պահոց:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-init>:.

- Նախաձեռնեք նոր տեղական պահեստ.:

`git init`

- Նախաձեռնեք պահեստ սկզբնական ճյուղի համար նշված անունով.:

`git init {{[-b|--initial-branch]}} {{branch_name}}`

- Նախաձեռնեք պահեստ՝ օգտագործելով SHA256 օբյեկտների հեշերի համար (պահանջում է Git տարբերակ 2.29+):

`git init --object-format sha256`

- Նախաձեռնեք barebones պահեստը, որը հարմար է SSH-ի վրա որպես հեռակառավարման վահանակ օգտագործելու համար.:

`git init --bare`
