# pbmtozinc

> Փոխակերպեք PBM պատկերը ցինկ բիթքարտեզի, ինչպես օգտագործվում է Zinc Interface Library 1.0 տարբերակի կողմից:.
> Լրացուցիչ տեղեկություններ. <https://netpbm.sourceforge.net/doc/pbmtozinc.html>:.

- Փոխակերպեք PBM պատկերը ցինկի բիտ քարտեզի.:

`pbmtozinc {{path/to/image.pbm}} > {{path/to/output.zinc}}`
