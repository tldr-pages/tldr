# ուլտրամանուշակագույն կառուցում

> Կառուցեք Python փաթեթներ աղբյուրի բաշխումների և անիվների մեջ:.
> Լրացուցիչ տեղեկություններ. <https://docs.astral.sh/uv/reference/cli/#uv-build>:.

- Կառուցեք փաթեթ ընթացիկ գրացուցակում.:

`uv build`

- Կառուցեք փաթեթ հատուկ գրացուցակից.:

`uv build {{path/to/directory}}`

- Կառուցեք միայն անիվ (բաց թողեք աղբյուրի բաշխումը).:

`uv build --wheel`

- Կառուցեք միայն աղբյուրի բաշխում (բաց թողնել անիվը).:

`uv build --sdist`

- Կառուցեք և թողարկեք որոշակի գրացուցակ.:

`uv build {{[-o|--out-dir]}} {{path/to/output}}`

- Ստեղծեք հատուկ փաթեթ աշխատանքային տարածքում.:

`uv build --package {{package_name}}`

- Ստեղծեք բոլոր փաթեթները աշխատանքային տարածքում.:

`uv build {{[--all|--all-packages]}}`

- Կառուցեք հատուկ Python թարգմանիչով.:

`uv build {{[-p|--python]}} {{python3.11}}`
