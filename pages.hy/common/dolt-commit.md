# dolt commit

> Կատարել փուլային փոփոխություններ աղյուսակներում:.
> Լրացուցիչ տեղեկություններ. <https://docs.dolthub.com/cli-reference/cli#dolt-commit>:.

- Կատարեք բոլոր փուլային փոփոխությունները՝ բացելով `$EDITOR`-ով նշված խմբագրիչը՝ հանձնման հաղորդագրությունը մուտքագրելու համար.:

`dolt commit`

- Կատարեք բոլոր փուլային փոփոխությունները նշված հաղորդագրությամբ.:

`dolt commit {{[-m|--message]}} "{{commit_message}}"`

- Բեմադրեք բոլոր չբեմադրված փոփոխությունները աղյուսակներում՝ նախքան կատարելը.:

`dolt commit {{[-a|--all]}}`

- Օգտագործեք նշված ISO 8601-ի կատարման ամսաթիվը (կանխադրված է ընթացիկ ամսաթվին և ժամին).:

`dolt commit --date "{{2021-12-31T00:00:00}}"`

- Օգտագործեք նշված հեղինակը պարտավորության համար.:

`dolt commit --author "{{author_name}} <{{author_email}}>"`

- Թույլ տվեք ստեղծել դատարկ հանձնառություն՝ առանց փոփոխությունների.:

`dolt commit --allow-empty`

- Անտեսեք արտաքին հիմնական նախազգուշացումները.:

`dolt commit {{[-f|--force]}}`
