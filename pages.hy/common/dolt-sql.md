# dolt sql

> Գործարկել SQL հարցում: Բազմաթիվ SQL հայտարարություններ պետք է առանձնացվեն ստորակետերով:.
> Լրացուցիչ տեղեկություններ. <https://docs.dolthub.com/cli-reference/cli#dolt-sql>:.

- Գործարկել մեկ հարցում.:

`dolt sql {{[-q|--query]}} "{{INSERT INTO t values (1, 3);}}"`

- Նշեք բոլոր պահպանված հարցումները.:

`dolt sql {{[-l|--list-saved]}}`
