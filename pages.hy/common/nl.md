# nl

> Թվային տողեր ֆայլից կամ `stdin`-ից:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/nl.1p>:.

- Թվարկել ոչ դատարկ տողերը ֆայլում.:

`nl {{path/to/file}}`

- Կարդացեք `stdin`-ից՝:

`{{command}} | nl -`

- Համարեք [a]բոլոր [b]ody տողերը, ներառյալ դատարկ տողերը, կամ [n]չհամարեք [b]ody տողերը.:

`nl -b {{a|n}} {{path/to/file}}`

- Համարակալեք միայն [b]ody տողերը, որոնք համընկնում են հիմնական `regex` (BRE) [p]ատրոնի հետ.:

`nl -b p'FooBar[0-9]' {{path/to/file}}`

- Օգտագործեք հատուկ [i] հավելում տողերի համարակալման համար.:

`nl -i {{increment}} {{path/to/file}}`

- Նշեք տողերի համարակալման ձևաչափը [r]աջ կամ [l]ձախ հիմնավորված՝ պահպանելով [z]eros կամ [n]ot առաջատարը:

`nl -n {{rz|ln|rn}}`

- Նշեք տողերի համարակալման [w]idth-ը (կանխադրված 6).:

`nl -w {{col_width}} {{path/to/file}}`

- Օգտագործեք հատուկ տող՝ [s] տողերի համարները տողերից առանձնացնելու համար (`TAB` ըստ լռելյայն).:

`nl -s {{separator}} {{path/to/file}}`
