# մինիսպասարկում

> Պարզ HTTP ֆայլերի սերվեր:.
> Լրացուցիչ տեղեկություններ. <https://github.com/svenstaro/miniserve#usage>:.

- Ծառայել գրացուցակ.:

`miniserve {{path/to/directory}}`

- Ծառայել մեկ ֆայլ.:

`miniserve {{path/to/file}}`

- Ծառայել գրացուցակ՝ օգտագործելով HTTP հիմնական նույնականացումը.:

`miniserve {{[-a|--auth]}} {{username}}:{{password}} {{path/to/directory}}`
