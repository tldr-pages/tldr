# sui

> Շփվեք Sui ցանցի հետ:.
> Լրացուցիչ տեղեկություններ. <https://docs.sui.io/references/cli/cheatsheet>:.

- Կատարեք Sui ենթահրաման.:

`sui {{subcommand}}`

- Ստեղծեք գործիքներ խելացի պայմանագրի համար.:

`sui move {{subcommand}}`

- Հրապարակեք խելացի պայմանագրեր, ստացեք օբյեկտի տեղեկատվություն, կատարեք գործարքներ և ավելին.:

`sui client {{subcommand}}`

- Սկսեք տեղական ցանց.:

`sui start`

- Թարմացում աղբյուրից.:

`cargo install --locked --git https://github.com/MystenLabs/sui.git --branch testnet sui`
