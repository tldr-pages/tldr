# dtrx

> «Կատարեք ճիշտ հանում» - ցանկացած արխիվ հանեք նոր գրացուցակում՝ գործիքը գուշակելով ընդլայնումից:.
> Լրացուցիչ տեղեկություններ. <https://github.com/dtrx-py/dtrx>:.

- Արտահանեք արխիվը՝ գուշակելով արդյունահանման գործիքը ընդլայնումից.:

`dtrx {{path/to/file}}`

- Արտահանեք արխիվը, վերագրեք առկա ցանկացած թիրախային արդյունք.:

`dtrx --overwrite {{path/to/file}}`

- Քաղեք արխիվը, ամեն ինչ դրեք ընթացիկ գրացուցակում.:

`dtrx --flat {{path/to/file}}`
