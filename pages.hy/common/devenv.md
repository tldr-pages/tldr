# devenv

> Արագ, դեկլարատիվ, վերարտադրվող և կազմվող ծրագրավորող միջավայրեր՝ օգտագործելով Nix:.
> Լրացուցիչ տեղեկություններ. <https://devenv.sh/getting-started/#commands>:.

- Նախաձեռնել շրջակա միջավայրը.:

`devenv init`

- Մուտք գործեք զարգացման միջավայր հանգիստ հերմետիկությամբ (հերմետիկ լինելու վիճակ).:

`devenv shell --impure`

- Ստացեք մանրամասն տեղեկատվություն ընթացիկ միջավայրի մասին.:

`devenv info --verbose`

- Գործընթացներ սկսել `devenv`-ով.:

`devenv up --config {{path/to/file}}`

- Մաքրել շրջակա միջավայրի փոփոխականները և կրկին մուտքագրել կեղևը անցանց ռեժիմում.:

`devenv --clean --offline`

- Ջնջել կեղևի նախորդ սերունդները.:

`devenv gc`
