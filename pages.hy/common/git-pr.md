# git pr

> Ստուգեք GitHub-ի ձգման հարցումները տեղական մակարդակում:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-pr>:.

- Ստուգեք հատուկ ձգման հարցում.:

`git pr {{pr_number}}`

- Ստուգեք ձգման հարցումը որոշակի հեռակառավարման վահանակից.:

`git pr {{pr_number}} {{remote}}`

- Ստուգեք ձգման հարցումը դրա URL-ից.:

`git pr {{url}}`

- Մաքրել ձգման պահանջի հին ճյուղերը.:

`git pr clean`
