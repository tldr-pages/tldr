#լզոպ

> Սեղմել կամ ապասեղմել ֆայլերը LZO սեղմումով:.
> Լրացուցիչ տեղեկություններ. <https://www.lzop.org/lzop_man.php>:.

- Կոմպրես ֆայլը նոր ֆայլի մեջ `.lzo` վերջածանցով.:

`lzop {{path/to/file}}`

- Ապասեղմել ֆայլը.:

`lzop {{[-d|--decompress]}} {{path/to/file.lzo}}`

- Սեղմեք ֆայլը, միաժամանակ նշելով սեղմման մակարդակը: 0 = Վատագույն, 9 = Լավագույն (Լռակյաց մակարդակը 3 է):

`lzop -{{level}} {{path/to/file}}`

- Սեղմեք ֆայլը լավագույն սեղմման մակարդակով.:

`lzop {{[-9|--best]}} {{path/to/file}}`

- Սեղմեք ֆայլը ամենաարագ սեղմման մակարդակով.:

`lzop {{[-1|--fast]}} {{path/to/file}}`
