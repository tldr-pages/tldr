# golangci-lint

> Զուգահեռացված, խելացի և արագ Go linters վազող, որը ինտեգրվում է բոլոր հիմնական IDE-ների հետ և աջակցում է YAML կազմաձևմանը:.
> Լրացուցիչ տեղեկություններ. <https://golangci-lint.run/welcome/quick-start/>:.

- Գործարկել linters ընթացիկ թղթապանակում:

`golangci-lint run`

- Ցուցակեք միացված և անջատված լինտերները (Ծանոթագրություն. հաշմանդամ լինտերները ցուցադրվում են վերջինը, մի շփոթեք դրանք միացվածների հետ):

`golangci-lint linters`

- Միացրեք հատուկ լցոն այս գործարկման համար.:

`golangci-lint run {{[-E|--enable]}} {{linter}}`
