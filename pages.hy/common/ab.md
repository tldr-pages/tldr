#աբ

> Apache HTTP սերվերի չափման գործիք:.
> Լրացուցիչ տեղեկություններ. <https://httpd.apache.org/docs/current/programs/ab.html>:.

- Կատարեք 100 HTTP GET հարցում տվյալ URL-ին.:

`ab -n 100 {{url}}`

- Կատարեք 100 HTTP GET հարցումներ, 10-ի համաժամանակյա խմբաքանակներով, URL հասցեով.:

`ab -n 100 -c 10 {{url}}`

- Կատարեք 100 HTTP POST հարցումներ դեպի URL՝ օգտագործելով ֆայլից JSON ծանրաբեռնվածություն.:

`ab -n 100 -T {{application/json}} -p {{path/to/file.json}} {{url}}`

- Օգտագործեք HTTP [k]eep-Alive, այսինքն՝ կատարեք բազմաթիվ հարցումներ մեկ HTTP նստաշրջանի ընթացքում.:

`ab -k {{url}}`

- Սահմանեք առավելագույն թվով վայրկյաններ ([t]timeout), որոնք պետք է ծախսվեն համեմատության համար (30 լռելյայն).:

`ab -t {{60}} {{url}}`

- Արդյունքները գրեք CSV ֆայլում.:

`ab -e {{path/to/file.csv}}`
