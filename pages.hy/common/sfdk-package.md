# sfdk փաթեթ

> Կատարել փաթեթի ստեղծման կառուցման քայլը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/sailfishos/sailfish-qtcreator/blob/master/share/qtcreator/sfdk/modules/20-building-mb2/doc/command.package.adoc>:.

- Ստեղծեք փաթեթ.:

`sfdk package`

- Ստեղծեք փաթեթ՝ առանց RPM SPEC ֆայլում ստուգման բաժինը կատարելու.:

`sfdk package --no-check`

- Ստեղծեք և ստորագրեք փաթեթ.:

`sfdk package --sign`
