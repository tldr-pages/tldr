# arp-սկան

> Ուղարկեք ARP փաթեթներ հոսթներին (նշված են որպես IP հասցեներ կամ հոսթների անուններ) տեղական ցանցը սկանավորելու համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/royhills/arp-scan>:.

- Սկանավորեք ընթացիկ տեղական ցանցը.:

`arp-scan {{[-l|--localnet]}}`

- Սկանավորեք կոնկրետ հոսթ.:

`arp-scan {{10.0.0.1}}`

- Սկանավորեք IP ցանցը հատուկ բիտդիմակով.:

`arp-scan {{192.168.1.1}}/{{24}}`

- Սկանավորեք IP ցանցը հատուկ տիրույթում.:

`arp-scan {{127.0.0.0}}-{{127.0.0.31}}`

- Սկանավորեք IP ցանցը հատուկ ցանցային դիմակով.:

`arp-scan {{10.0.0.0}}:{{255.255.255.0}}`

- Նշեք ինտերֆեյս, որը կօգտագործվի սկանավորման համար.:

`arp-scan {{10.0.0.1}} {{[-I|--interface]}} {{interface_name}}`
