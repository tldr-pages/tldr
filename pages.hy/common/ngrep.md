# ngrep

> Զտել ցանցի երթևեկության փաթեթները՝ օգտագործելով `regex`:.
> Լրացուցիչ տեղեկություններ. <https://github.com/jpr5/ngrep/blob/master/EXAMPLES.md>:.

- Գրավել բոլոր ինտերֆեյսների երթևեկությունը.:

`ngrep -d any`

- Գրավել հատուկ ինտերֆեյսի երթևեկությունը.:

`ngrep -d {{eth0}}`

- Ինտերֆեյսի eth0-ի երթևեկության հատման նավահանգիստը գրավեք.:

`ngrep -d {{eth0}} port {{22}}`

- Գրանցեք երթևեկությունը հյուրընկալողից կամ դեպի հյուրընկալող.:

`ngrep host {{www.example.com}}`

- Զտել «User-Agent:» ինտերֆեյսի eth0 հիմնաբառը:

`ngrep -d {{eth0}} '{{User-Agent:}}'`
