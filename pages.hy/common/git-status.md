# git կարգավիճակ

> Ցույց տալ փոփոխությունները ֆայլերում Git պահոցում:.
> Ցուցակը փոփոխված, ավելացված և ջնջված ֆայլերի համեմատ՝ ներկայումս ստուգված կատարման հետ:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-status>:.

- Ցույց տալ չհետևված և փոփոխված ֆայլերը, որոնք դեռ չեն ավելացվել կատարման համար.:

`git status`

- Տվեք ելք կարճ ձևաչափով.:

`git status {{[-s|--short]}}`

- Ցույց տալ ելքը կարճ ձևաչափով մասնաճյուղի տեղեկատվության հետ միասին.:

`git status {{[-sb|--short --branch]}}`

- Ցուցադրել փուլային ֆայլերի փոփոխությունները (ինչպես `git diff --cached`):

`git status {{[-v|--verbose]}}`

- Ցույց տալ փոփոխությունները բոլոր հետևված ֆայլերում (օրինակ՝ `git diff HEAD`):

`git status {{[-vv|--verbose --verbose]}}`

- Ցույց տալ ներկայումս թաքցված գրառումների քանակը.:

`git status --show-stash`

- Մի ցուցադրեք չհետևված ֆայլերը ելքի մեջ.:

`git status {{[-uno|--untracked-files=no]}}`
