# git verify-pack

> Ստուգեք փաթեթավորված Git արխիվային ֆայլերը:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-verify-pack>:.

- Ստուգեք փաթեթավորված Git արխիվի ֆայլը.:

`git verify-pack {{path/to/pack-file}}`

- Ստուգեք փաթեթավորված Git արխիվային ֆայլը և ցույց տվեք մանրամասն մանրամասներ.:

`git verify-pack {{[-v|--verbose]}} {{path/to/pack-file}}`

- Ստուգեք փաթեթավորված Git արխիվային ֆայլը և ցուցադրեք միայն վիճակագրությունը.:

`git verify-pack {{[-s|--stat-only]}} {{path/to/pack-file}}`
