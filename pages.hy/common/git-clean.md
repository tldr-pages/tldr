#git մաքուր

> Հեռացրեք Git-ի կողմից չհետևված ֆայլերը աշխատանքային ծառից:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-clean>:.

- Ինտերակտիվ կերպով ջնջեք չհետևված ֆայլերը.:

`git clean {{[-i|--interactive]}}`

- Ցույց տալ, թե որ ֆայլերը կջնջվեն առանց դրանք իրականում ջնջելու.:

`git clean {{[-n|--dry-run]}}`

- Անմիջապես ստիպել ջնջել բոլոր չհետևված ֆայլերը.:

`git clean {{[-f|--force]}}`

- Ջնջել չհետևված [d] գրացուցակները.:

`git clean {{[-f|--force]}} -d`

- Ջնջեք միայն չհետևված ֆայլերը, որոնք համապատասխանում են հատուկ ուղիներին կամ գլոբալ օրինաչափություններին.:

`git clean {{[-f|--force]}} -- {{path/to/directory}} '{{*.ext}}'`

- Ջնջել չհետևված ֆայլերը, բացառությամբ նրանց, որոնք համապատասխանում են տվյալ օրինաչափություններին.:

`git clean {{[-f|--force]}} {{[-e|--exclude]}} '{{*.ext}}' {{[-e|--exclude]}} {{path/to/directory}}/`

- Ջնջել չհետևված ֆայլերը և e[x]ներառյալ ֆայլերը (նրանք, որոնք նշված են `.gitignore` և `.git/info/exclude`-ում):

`git clean {{[-f|--force]}} -x`
