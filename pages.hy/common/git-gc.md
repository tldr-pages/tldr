# git gc

> Օպտիմալացնել տեղական պահեստը` մաքրելով ավելորդ ֆայլերը:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-gc>:.

- Օպտիմալացնել պահեստը.:

`git gc`

- Ագրեսիվորեն օպտիմիզացնել, ավելի շատ ժամանակ է պահանջում.:

`git gc --aggressive`

- Մի էտեք չամրացված առարկաները (սալորաչիր ըստ լռելյայն).:

`git gc --no-prune`

- ճնշել բոլոր ելքերը.:

`git gc --quiet`

- Ցուցադրել օգնությունը.:

`git gc --help`
