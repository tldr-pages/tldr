# pipx

> Տեղադրեք և գործարկեք Python հավելվածները մեկուսացված միջավայրերում:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/pipx>:.

- Գործարկել հավելվածը ժամանակավոր վիրտուալ միջավայրում.:

`pipx run {{pycowsay}} {{moo}}`

- Տեղադրեք փաթեթ վիրտուալ միջավայրում և մուտքի կետեր ավելացրեք ճանապարհին.:

`pipx install {{package}}`

- Ապատեղադրել փաթեթը.:

`pipx uninstall {{package}}`

- Տեղադրված փաթեթների ցանկ.:

`pipx list`

- Գործարկեք հավելված ժամանակավոր վիրտուալ միջավայրում՝ գործարկվողից տարբերվող փաթեթի անունով.:

`pipx run --spec {{httpx-cli}} {{httpx}} {{http://www.github.com}}`

- Ներարկեք կախվածությունը գոյություն ունեցող վիրտուալ միջավայրում.:

`pipx inject {{package}} {{dependency1 dependency2 ...}}`

- Տեղադրեք փաթեթ վիրտուալ միջավայրում pip արգումենտներով.:

`pipx install --pip-args='{{pip-args}}' {{package}}`

- Թարմացրեք/վերատեղադրեք/տեղահանեք բոլոր տեղադրված փաթեթները.:

`pipx {{upgrade-all|uninstall-all|reinstall-all}}`
