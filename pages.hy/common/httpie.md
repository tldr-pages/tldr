# httpie

> Կառավարման միջերես HTTPie-ի համար:.
> Տես նաև՝ `http`:.
> Լրացուցիչ տեղեկություններ. <https://httpie.io/docs/cli/plugin-manager>:.

- Ստուգեք թարմացումները `http`-ի համար՝:

`httpie cli check-updates`

- Տեղադրված `http` հավելվածների ցանկ՝:

`httpie cli plugins list`

- Տեղադրեք/թարմացրեք/տեղահանեք պլագինները.:

`httpie cli plugins {{install|upgrade|uninstall}} {{plugin_name}}`
