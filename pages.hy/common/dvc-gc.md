# dvc gc

> Հեռացրեք չօգտագործված ֆայլերը և գրացուցակները քեշից կամ հեռավոր պահեստից:.
> Լրացուցիչ տեղեկություններ. <https://doc.dvc.org/command-reference/gc>:.

- Աղբը հավաքվում է քեշից՝ պահպանելով միայն ընթացիկ աշխատանքային տարածքի կողմից հղում արված տարբերակները.:

`dvc gc {{[-w|--workspace]}}`

- Աղբը հավաքվում է քեշից՝ պահելով միայն այն տարբերակները, որոնք հղում են կատարում մասնաճյուղի, պիտակների և պարտավորությունների.:

`dvc gc {{[-a|--all-branches]}} {{[-T|--all-tags]}} {{[-a|--all-commits]}}`

- Աղբը հավաքվում է քեշից, ներառյալ լռելյայն ամպային հեռավոր պահեստավորումը (եթե սահմանված է).:

`dvc gc {{[-a|--all-commits]}} {{[-c|--cloud]}}`

- Աղբը հավաքվում է քեշից, ներառյալ հատուկ ամպային հեռավոր պահեստավորում.:

`dvc gc {{[-a|--all-commits]}} {{[-c|--cloud]}} {{[-r|--remote]}} {{remote_name}}`
