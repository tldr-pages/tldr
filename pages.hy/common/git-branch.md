# git մասնաճյուղ

> Main Git հրամանը ճյուղերի հետ աշխատելու համար:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-branch>:.

- Թվարկեք բոլոր մասնաճյուղերը (տեղական և հեռավոր; ընթացիկ մասնաճյուղը ընդգծված է `*`-ով):

`git branch {{[-a|--all]}}`

- Նշեք, թե որ մասնաճյուղերն են իրենց պատմության մեջ ներառում կոնկրետ Git commit.:

`git branch {{[-a|--all]}} --contains {{commit_hash}}`

- Ցույց տալ ընթացիկ մասնաճյուղի անունը.:

`git branch --show-current`

- Ստեղծեք նոր մասնաճյուղ՝ հիմնվելով ընթացիկ պարտավորության վրա.:

`git branch {{branch_name}}`

- Ստեղծեք նոր մասնաճյուղ՝ հիմնվելով որոշակի պարտավորության վրա.:

`git branch {{branch_name}} {{commit_hash}}`

- Վերանվանեք մասնաճյուղը (դա անելուց առաջ դուք պետք է անցնեք այլ մասնաճյուղի).:

`git branch {{[-m|--move]}} {{old_branch_name}} {{new_branch_name}}`

- Ջնջեք տեղական մասնաճյուղը (դա անելուց առաջ դուք պետք է անցնեք մեկ այլ մասնաճյուղ).:

`git branch {{[-d|--delete]}} {{branch_name}}`

- Ջնջել հեռավոր մասնաճյուղը.:

`git push {{remote_name}} {{[-d|--delete]}} {{remote_branch_name}}`
