# gh օգնություն

> Ցուցադրել օգնություն GitHub CLI հրամանի վերաբերյալ:.
> Լրացուցիչ տեղեկություններ. <https://cli.github.com/manual/gh_help>:.

- Ցուցադրել ընդհանուր օգնությունը.:

`gh help`

- Ցուցադրել օգնություն `gh help` ենթահրամանի համար.:

`gh help {{[-h|--help]}}`

- Ցուցադրել օգնություն շրջակա միջավայրի փոփոխականների մասին, որոնք կարող են օգտագործվել `gh`-ով:

`gh help environment`

- Ցուցադրել բոլոր `gh` հրամանների նշագրման հղումը՝:

`gh help reference`

- Ցուցադրել օգնություն `gh`-ից JSON ելքի ձևաչափման վերաբերյալ՝ օգտագործելով `jq`:

`gh help formatting`

- Ցուցադրել օգնություն `gh` MinTTY-ի հետ օգտագործելու վերաբերյալ՝:

`gh help mintty`

- Ցուցադրել օգնություն ենթահրամանի համար.:

`gh help {{subcommand}}`

- Ցուցադրել օգնություն ենթահրամանի գործողության համար.:

`gh help {{pr}} {{create}}`
