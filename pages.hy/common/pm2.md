# pm2

> Գործընթացների կառավարիչ Node.js-ի համար:.
> Օգտագործվում է գրանցամատյանների կառավարման, մոնիտորինգի և կազմաձևման գործընթացների համար:.
> Լրացուցիչ տեղեկություններ. <https://pm2.keymetrics.io/docs/usage/quick-start/>:.

- Գործընթաց սկսել անունով, որը կարող է օգտագործվել հետագա գործողությունների համար.:

`pm2 start {{app.js}} --name {{application_name}}`

- Գործընթացների ցուցակ.:

`pm2 list`

- Դիտեք բոլոր գործընթացները.:

`pm2 monit`

- Դադարեցնել գործընթացը.:

`pm2 stop {{application_name}}`

- Վերագործարկեք գործընթացը.:

`pm2 restart {{application_name}}`

- Թափել բոլոր գործընթացները՝ դրանք հետագայում վերակենդանացնելու համար.:

`pm2 save`

- Վերականգնել նախկինում թափված գործընթացները.:

`pm2 resurrect`
