# flox

> Հեշտ օգտագործվող Nix փաթեթի և շրջակա միջավայրի կառավարիչ:.
> Լրացուցիչ տեղեկություններ. <https://flox.dev/docs/reference/command-reference/flox/>:.

- Ստեղծեք նոր միջավայր ընթացիկ գրացուցակում.:

`flox init`

- Մուտքագրեք միջավայր կամ ստեղծեք այն, եթե այն գոյություն չունի.:

`flox activate`

- Փաթեթների որոնում FloxHub կատալոգում.:

`flox search {{package}}`

- Տեղադրեք փաթեթ ընթացիկ միջավայրում.:

`flox {{[i|install]}} {{package}}`

- Հեռացրեք փաթեթը ընթացիկ միջավայրից.:

`flox uninstall {{package}}`

- Դիտեք ընթացիկ միջավայրում տեղադրված բոլոր փաթեթների ցանկը.:

`flox {{[l|list]}}`

- Տեղական միջավայր մղեք FloxHub՝ ուրիշների հետ կիսվելու համար.:

`flox push`

- Քաշեք ընդհանուր միջավայր FloxHub-ից.:

`flox pull {{environment_name}}`
