# mvn կայք

> Ստեղծեք նախագծի կայք՝ հիմնվելով `pom.xml` ֆայլի տեղեկատվության վրա:.
> Լրացուցիչ տեղեկություններ. <https://maven.apache.org/plugins/maven-site-plugin/usage.html>:.

- Ստեղծեք նախագծի կայք՝ օգտագործելով կայքի հավելումը.:

`mvn site`

- Ստեղծեք կայք հատուկ Maven նախագծի համար (բազմամոդուլային կառուցում).:

`mvn site {{[-pl|--projects]}} {{module_name}}`

- Մաքրեք կայքի նախորդ ելքը՝ նախքան նորը ստեղծելը.:

`mvn clean site`

- Բաց թողնել թեստերը կայքը ստեղծելիս.:

`mvn site {{[-D|--define]}} skipTests`

- Ստեղծեք և տեղադրեք կայքը հեռավոր սերվերի վրա.:

`mvn site-deploy`
