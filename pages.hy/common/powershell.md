# powershell

> Այս հրամանը կարող է սխալվել որպես PowerShell-ի միջպլատֆորմային տարբերակ (նախկինում հայտնի է որպես PowerShell Core), որն օգտագործում է `pwsh` `powershell`-ի փոխարեն:.
> Բնօրինակ `powershell` հրամանը Windows-ում դեռ հասանելի է PowerShell-ի հին Windows տարբերակը օգտագործելու համար (տարբերակ 5.1 և ավելի ցածր):.
> Լրացուցիչ տեղեկություններ. <https://learn.microsoft.com/powershell/module/microsoft.powershell.core/about/about_pwsh>:.

- Դիտեք հրամանի փաստաթղթերը, որոնք վերաբերում են PowerShell-ի վերջին, միջպլատֆորմային տարբերակին (տարբերակ 6 և ավելի):

`tldr pwsh`

- Դիտեք հրամանի փաստաթղթերը, որոնք վերաբերում են հին Windows PowerShell-ին (տարբերակ 5.1 և ստորև).:

`tldr powershell {{[-p|--platform]}} windows`
