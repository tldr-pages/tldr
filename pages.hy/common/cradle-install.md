# օրորոցի տեղադրում

> Տեղադրեք Cradle PHP շրջանակի բաղադրիչները:.
> Լրացուցիչ տեղեկություններ. <https://cradlephp.github.io/docs/3.B.-Reference-Command-Line-Tools.html#install>:.

- Տեղադրեք Cradle-ի բաղադրիչները (օգտագործողին կառաջարկվի լրացուցիչ մանրամասներ).:

`cradle install`

- Հարկադիր վերագրանցել ֆայլերը.:

`cradle install {{[-f|--force]}}`

- Բաց թողնել գործարկվող SQL միգրացիաները.:

`cradle install --skip-sql`

- Բաց թողեք փաթեթի թարմացումները.:

`cradle install --skip-versioning`

- Օգտագործեք տվյալների բազայի հատուկ մանրամասներ.:

`cradle install -h {{hostname}} -u {{username}} -p {{password}}`
