# npm նշան

> Կառավարեք և ստեղծեք նույնականացման նշաններ npm ռեեստրի համար:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-token/>:.

- Ստեղծեք նույնականացման նոր նշան.:

`npm token create`

- Թվարկեք հաշվի հետ կապված բոլոր նշանները.:

`npm token list`

- Ջնջել կոնկրետ նշան՝ օգտագործելով իր նշանի ID-ն.:

`npm token revoke {{token_id}}`

- Ստեղծեք նշան միայն կարդալու հասանելիությամբ.:

`npm token create --read-only`

- Ստեղծեք նշան հրապարակման հասանելիությամբ.:

`npm token create --publish`

- Ավտոմատ կերպով կազմաձևեք npm նշանը ձեր գլոբալ `.npmrc` ֆայլում, երբ մուտք եք գործում՝:

`npm login`

- Հեռացրեք նշանը գլոբալ կազմաձևից.:

`npm token revoke {{token_id}}`
