# մ.թ.ա

> Կամայական ճշգրիտ հաշվիչի լեզու:.
> Տես նաև՝ `dc`, `qalc`:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/bc>:.

- Սկսեք ինտերակտիվ նիստ.:

`bc`

- Սկսեք ինտերակտիվ նիստ՝ միացված ստանդարտ մաթեմատիկական գրադարանով.:

`bc {{[-i|--interactive]}} {{[-l|--mathlib]}}`

- Հաշվել արտահայտությունը.:

`echo '{{5 / 3}}' | bc`

- Կատարել սցենար.:

`bc {{path/to/script.bc}}`

- Հաշվեք արտահայտությունը նշված մասշտաբով.:

`echo 'scale = {{10}}; {{5 / 3}}' | bc`

- Հաշվեք սինուս/կոսինուս/հարկտանգենս/բնական լոգարիթմ/էքսպոնենցիալ ֆունկցիա՝ օգտագործելով `mathlib`:

`echo '{{s|c|a|l|e}}({{1}})' | bc {{[-l|--mathlib]}}`

- Կատարեք ներկառուցված գործոնային սցենար.:

`echo "define factorial(n) { if (n <= 1) return 1; return n*factorial(n-1); }; factorial({{10}})" | bc`
