# pixi գլոբալ

> Կառավարեք գլոբալ փաթեթները:.
> Լրացուցիչ տեղեկություններ. <https://pixi.sh/latest/reference/cli/#global>:.

- Տեղադրեք փաթեթ գլոբալ և ավելացրեք ճանապարհին.:

`pixi global install {{package1 package2 ...}}`

- Ապատեղադրեք փաթեթը գլոբալ.:

`pixi global remove {{package1 package2 ...}}`

- Թվարկեք բոլոր գլոբալ տեղադրված փաթեթները.:

`pixi global list`

- Թարմացրեք գլոբալ տեղադրված փաթեթը.:

`pixi global upgrade {{package}}`

- Թարմացրեք բոլոր գլոբալ տեղադրված փաթեթները.:

`pixi global upgrade-all`
