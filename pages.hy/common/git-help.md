#git օգնություն

> Ցուցադրել օգնության տեղեկատվությունը Git-ի մասին:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-help>:.

- Ցուցադրել օգնություն կոնկրետ Git ենթահրամանի վերաբերյալ.:

`git help {{subcommand}}`

- Ցուցադրել օգնություն հատուկ Git ենթահրամանի վերաբերյալ վեբ բրաուզերում.:

`git help {{[-w|--web]}} {{subcommand}}`

- Ցուցադրել բոլոր հասանելի Git ենթահրամանների ցանկը.:

`git help {{[-a|--all]}}`

- Թվարկեք առկա ուղեցույցները.:

`git help {{[-g|--guides]}}`

- Թվարկեք բոլոր հնարավոր կազմաձևման փոփոխականները.:

`git help {{[-c|--config]}}`
