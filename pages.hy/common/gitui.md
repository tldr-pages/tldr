# gitui

> Թեթև ստեղնաշարի համար նախատեսված TUI Git-ի համար:.
> Տես նաև՝ `tig`, `git-gui`:.
> Լրացուցիչ տեղեկություններ. <https://github.com/gitui-org/gitui>:.

- Սկսեք `gitui` պահոցը ընթացիկ գրացուցակում.:

`gitui`

- Նշեք կազմաձևման գրացուցակից բեռնված գունային թեմայի ֆայլի անունը (կանխադրված է `theme.ron`):

`gitui {{[-t|--theme]}} {{theme2.ron}}`

- Պահպանեք մուտքի ելքը որոշակի ֆայլում.:

`gitui --logfile {{path/to/file}}`

- Ստուգեք որոշակի ֆայլ պահեստի ներսում ընթացիկ գրացուցակում.:

`gitui {{[-f|--file]}} {{path/to/file}}`

- Օգտագործեք ծանուցումների վրա հիմնված ֆայլային համակարգի դիտորդ՝ տիզերի վրա հիմնված թարմացման փոխարեն.:

`gitui --watcher`

- Ստեղծեք սխալի մասին հաշվետվություն.:

`gitui --bugreport`

- Օգտագործեք հատուկ Git գրացուցակ.:

`gitui {{[-d|--directory]}} {{path/to/directory}}`

- Օգտագործեք հատուկ աշխատանքային գրացուցակ.:

`gitui {{[-w|--workdir]}} {{path/to/directory}}`
