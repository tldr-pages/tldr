# rustup շոու

> Ցուցադրել տեղադրված գործիքների շղթաները, թիրախները և `rustc` տարբերակը:.
> Լրացուցիչ տեղեկություններ. <https://rust-lang.github.io/rustup/>:.

- Ցույց տալ բոլոր տեղեկությունները.:

`rustup show`

- Ցույց տալ ակտիվ գործիքների շղթան.:

`rustup show active-toolchain`

- Ցույց տալ rustup տվյալների գրացուցակը.:

`rustup show home`
