# limactl

> Վիրտուալ մեքենայի կառավարիչ Linux-ի հյուրերի համար՝ հասանելի բազմաթիվ VM կաղապարներով:.
> Կարող է օգտագործվել macOS-ում կոնտեյներներ գործարկելու համար, ինչպես նաև macOS-ի և Linux-ի հոսթերների ընդհանուր վիրտուալ մեքենաների օգտագործման դեպքերի համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/lima-vm/lima>:.

- Ցուցակ VM-ներ.:

`limactl list`

- Ստեղծեք VM՝ օգտագործելով լռելյայն կարգավորումները և ընտրովի տրամադրեք անուն և/կամ ձևանմուշ (տե՛ս `limactl create --list-templates` հասանելի ձևանմուշների համար).:

`limactl create --name {{vm_name}} template:{{debian|fedora|ubuntu|...}}`

- Սկսեք VM (սա կարող է տեղադրել որոշ կախվածություններ դրա մեջ և տևել մի քանի րոպե).:

`limactl start {{vm_name}}`

- Բացեք հեռակառավարման վահանակը VM-ի ներսում.:

`limactl shell {{vm_name}}`

- Գործարկեք հրաման VM-ի ներսում.:

`limactl shell {{vm_name}} {{command}}`

- Դադարեցնել/անջատել VM-ը.:

`limactl stop {{vm_name}}`

- Ջնջել VM-ը.:

`limactl remove {{vm_name}}`
