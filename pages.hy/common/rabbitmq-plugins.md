# rabbitmq-plugins

> Կառավարեք RabbitMQ հավելվածները:.
> Լրացուցիչ տեղեկություններ. <https://www.rabbitmq.com/docs/man/rabbitmq-plugins.8>:.

- Ցուցակ պլագիններ.:

`rabbitmq-plugins list`

- Միացնել plugin-ը.:

`rabbitmq-plugins enable {{plugin}}`

- Անջատել plugin:

`rabbitmq-plugins disable {{plugin}}`

- Միացրեք plugin-ը և անջատեք մնացածը.:

`rabbitmq-plugins set {{plugin}}`
