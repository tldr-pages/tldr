# git հավելված

> Կառավարեք ֆայլերը Git-ի միջոցով՝ առանց դրանց բովանդակությունը ստուգելու:.
> Երբ ֆայլը կցվում է, դրա բովանդակությունը տեղափոխվում է բանալի-արժեքի պահեստ, և ստեղծվում է սիմհղում, որը ցույց է տալիս բովանդակությունը:.
> Լրացուցիչ տեղեկություններ. <https://git-annex.branchable.com/git-annex/>:.

- Նախաձեռնեք ռեպո Git հավելվածով.:

`git annex init`

- Ավելացնել ֆայլ.:

`git annex add {{path/to/file_or_directory}}`

- Ցույց տալ ֆայլի կամ գրացուցակի ընթացիկ կարգավիճակը.:

`git annex status {{path/to/file_or_directory}}`

- Համաժամացրեք տեղական պահեստը հեռակառավարման հետ.:

`git annex {{remote}}`

- Ստացեք ֆայլ կամ գրացուցակ.:

`git annex get {{path/to/file_or_directory}}`

- Ցուցադրել օգնությունը.:

`git annex help`
