# npm հարցում

> Տպեք կախվածության օբյեկտների զանգված՝ օգտագործելով CSS-ի նման ընտրիչներ:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-query/>:.

- Տպել ուղղակի կախվածությունները.:

`npm query ':root > *'`

- Տպել բոլոր ուղղակի արտադրության/զարգացման կախվածությունները.:

`npm query ':root > .{{prod|dev}}'`

- Տպել կախվածությունը կոնկրետ անունով.:

`npm query '#{{package}}'`

- Տպել կախվածությունը կոնկրետ անունով և իմաստային տարբերակման տիրույթում.:

`npm query '#{{package}}@{{semantic_version}}'`

- Տպել կախվածություն, որը կախվածություն չունի.:

`npm query ':empty'`

- Գտեք բոլոր կախվածությունները հետտեղադրման սկրիպտներով և տեղահանեք դրանք.:

`npm query ":attr(scripts, [postinstall])" | jq 'map(.name) | join("\n")' {{[-r|--raw-output]}} | xargs -I _ npm uninstall _`

- Գտեք Git-ի բոլոր կախվածությունները և տպեք, թե որ հավելվածն է դրանք պահանջում.:

`npm query ":type(git)" | jq 'map(.name)' | xargs -I _ npm why _`
