# ուլտրամանուշակագույն օգնություն

> Ցուցադրել մանրամասն փաստաթղթեր `uv` հրամանների համար:.
> Լրացուցիչ տեղեկություններ. <https://docs.astral.sh/uv/reference/cli/#uv-help>:.

- Ցուցադրել ընդհանուր օգնություն `uv`-ի համար.:

`uv help`

- Ցուցադրել օգնություն կոնկրետ հրամանի համար.:

`uv help {{command}}`

- Ցուցադրել օգնություն ենթահրամանի համար.:

`uv help {{command}} {{subcommand}}`

- Ցուցադրել օգնությունը առանց փեյջեր օգտագործելու.:

`uv help --no-pager {{command}}`
