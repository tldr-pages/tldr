# VBoxManage ցուցակ

> Ցուցակե՛ք Oracle VM VirtualBox ծրագրաշարի և հարակից ծառայության մասին տեղեկությունները:.
> Լրացուցիչ տեղեկություններ. <https://www.virtualbox.org/manual/ch08.html#vboxmanage-list>:.

- Թվարկեք բոլոր VirtualBox վիրտուալ մեքենաները.:

`VBoxManage list vms`

- Ցուցադրել հյուրընկալող համակարգում հասանելի DHCP սերվերները.:

`VBoxManage list dhcpservers`

- Ցույց տալ Oracle VM VirtualBox ընդլայնման փաթեթները, որոնք ներկայումս տեղադրված են.:

`VBoxManage list extpacks`

- Ցույց տալ բոլոր վիրտուալ մեքենաների խմբերը.:

`VBoxManage list groups`

- Ցույց տալ վիրտուալ սկավառակի կարգավորումները, որոնք ներկայումս օգտագործվում են VirtualBox-ի կողմից.:

`VBoxManage list hdds`

- Ցույց տալ միայն հոսթինգի ցանցային միջերեսները, որոնք հասանելի են հոսթ համակարգում.:

`VBoxManage list hostonlyifs`

- Ցուցադրել ներկայումս գործող վիրտուալ մեքենաների ցանկը.:

`VBoxManage list runningvms`

- Ցուցադրել հյուրընկալող համակարգի տեղեկատվությունը.:

`VBoxManage list hostinfo`
