# gh pr դիտում

> Դիտեք GitHub-ի ձգման հարցումի մանրամասները:.
> Լրացուցիչ տեղեկություններ. <https://cli.github.com/manual/gh_pr_view>:.

- Դիտեք ընթացիկ մասնաճյուղի հետ կապված ձգման հարցումը.:

`gh pr view`

- Դիտեք հատուկ ձգման հարցում.:

`gh pr view {{123}}`

- Բացեք ձգման հարցումը լռելյայն վեբ դիտարկիչում.:

`gh pr view {{[-w|--web]}}`

- Ցույց տալ մեկնաբանությունները տերմինալում.:

`gh pr view {{123}} {{[-c|--comments]}}`
