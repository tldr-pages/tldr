#git whatchanged

> Այս հրամանը `git log --raw --no-merges`-ի կեղծանունն է (յուրաքանչյուր commit-ի համար չմշակված տարբերություններով մուտք):.
> Նշում. այս հրամանը հնացած է և ապագայում կհեռացվի:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-whatchanged>:.

- Դիտեք փաստաթղթերը բնօրինակ հրամանի համար.:

`tldr git log`
