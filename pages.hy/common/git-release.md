# git թողարկում

> Ստեղծեք Git թեգ թողարկման համար:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/git-release>:.

- Ստեղծեք և մղեք թողարկում.:

`git release {{tag_name}}`

- Ստեղծեք և մղեք ստորագրված թողարկում.:

`git release {{tag_name}} -s`

- Ստեղծեք և մղեք թողարկում հաղորդագրությամբ.:

`git release {{tag_name}} -m "{{message}}"`
