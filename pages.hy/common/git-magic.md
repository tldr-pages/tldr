#git magic

> Ավտոմատացրեք ավելացման, կատարման և առաջխաղացման ռեժիմները:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/git-magic>:.

- Փոփոխություններ կատարեք ստեղծված հաղորդագրության միջոցով.:

`git magic`

- [a] ավելացրեք չհետևված ֆայլեր և փոփոխություններ կատարեք ստեղծված հաղորդագրության միջոցով.:

`git magic -a`

- Փոփոխություններ կատարեք սովորական [m] հաղորդագրությամբ.:

`git magic -m "{{custom_commit_message}}"`

- [e] խմբագրել commit [m] հաղորդագրությունը կատարելուց առաջ՝:

`git magic -em "{{custom_commit_message}}"`

- Փոփոխություններ կատարեք և [p] սեղմեք հեռակա՝:

`git magic -p`

- Փոփոխություններ կատարեք՝ [f]force [p] սեղմեք հեռակառավարման վրա՝:

`git magic -fp`
