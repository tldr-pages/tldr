# rsstail

> `tail` RSS հոսքերի համար:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/rsstail>:.

- Ցույց տվեք տվյալ URL-ի հոսքը և սպասեք, որ ներքևում հայտնվեն նոր գրառումներ.:

`rsstail -u {{url}}`

- Ցուցադրել թարմացվող բովանդակությունը հակառակ ժամանակագրական հաջորդականությամբ (ավելի նոր՝ ներքևում).:

`rsstail -r -u {{url}}`

- Ներառեք հրապարակման ամսաթիվը և հղումը.:

`rsstail -pl -u {{url}}`

- Սահմանեք թարմացման միջակայքը.:

`rsstail -u {{url}} -i {{interval_in_seconds}}`

- Ցույց տալ հոսքը և ելքը.:

`rsstail -1 -u {{url}}`
