# git shortlog

> Ամփոփում է `git log` ելքը:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-shortlog>:.

- Դիտեք կատարված բոլոր պարտավորությունների ամփոփագիրը՝ խմբավորված այբբենական կարգով՝ ըստ հեղինակի անունով.:

`git shortlog`

- Դիտեք կատարված բոլոր պարտավորությունների ամփոփագիրը՝ դասավորված ըստ կատարված պարտավորությունների քանակի.:

`git shortlog {{[-n|--numbered]}}`

- Դիտեք կատարված բոլոր պարտավորությունների ամփոփագիրը՝ խմբավորված ըստ կատարողի ինքնության (անունը և էլ. փոստը).:

`git shortlog {{[-c|--committer]}}`

- Դիտեք վերջին 5 պարտավորությունների ամփոփագիրը (այսինքն՝ նշեք վերանայման շրջանակը).:

`git shortlog HEAD~5..HEAD`

- Դիտեք բոլոր օգտվողներին, էլ. նամակները և ընթացիկ մասնաճյուղում կատարվող պարտավորությունների քանակը.:

`git shortlog {{[-sne|--summary --numbered --email]}}`

- Դիտեք բոլոր օգտվողներին, էլ. նամակները և պարտավորությունների քանակը բոլոր մասնաճյուղերում.:

`git shortlog {{[-sne|--summary --numbered --email]}} --all`
