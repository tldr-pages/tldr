# jj մատյան

> Ցուցադրել վերանայման պատմությունը որպես գրաֆիկ:.
> Լրացուցիչ տեղեկություններ. <https://docs.jj-vcs.dev/latest/cli-reference/#jj-log>:.

- Ցուցադրել վերանայման պատմությունը որպես գրաֆիկ.:

`jj log`

- Ցույց տալ միայն տրված վերափոխումները (օրինակ՝ `B::D`, `A..D`, `B|C|D` և այլն):

`jj log {{[-r|--revisions]}} {{revsets}}`

- Ցուցադրել գրանցամատյանը յուրաքանչյուր տողի համար որոշակի ձևանմուշով (օրինակ՝ 5 նիշ՝ commit hash-ով և հեղինակով).:

`jj log {{[-T|--template]}} 'commit_id.shortest(5) ++ " " ++ author'`
