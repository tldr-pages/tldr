# mdp

> Կատարեք շնորհանդեսներ Markdown ֆայլերից:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/mdp>:.

- Գործարկեք շնորհանդես տերմինալում Markdown ֆայլից.:

`mdp {{presentation.md}}`

- Անջատել մարող անցումները.:

`mdp {{[-f|--nofade]}} {{presentation.md}}`

- Փոխեք տառատեսակի գույները՝ բաց ֆոնով տերմինալներում օգտագործելու համար.:

`mdp {{[-i|--invert]}} {{presentation.md}}`

- Անջատել թափանցիկությունը թափանցիկ տերմինալներում.:

`mdp {{[-t|--notrans]}} {{presentation.md}}`
