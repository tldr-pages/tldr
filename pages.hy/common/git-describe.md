# git նկարագրություն

> Օբյեկտին տվեք մարդու համար ընթեռնելի անուն՝ հիմնվելով հասանելի հղումի վրա:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-describe>:.

- Ստեղծեք եզակի անուն ընթացիկ commit-ի համար (անունը պարունակում է ամենավերջին ծանոթագրված պիտակը, լրացուցիչ պարտավորությունների քանակը և կրճատված commit hash-ը).:

`git describe`

- Ստեղծեք անուն 4 նիշով կրճատված commit hash-ի համար.:

`git describe --abbrev={{4}}`

- Ստեղծեք անուն պիտակի հղման ճանապարհով.:

`git describe --all`

- Նկարագրեք Git թեգը.:

`git describe {{v1.0.0}}`

- Ստեղծեք անուն տվյալ ճյուղի վերջին կատարման համար.:

`git describe {{branch_name}}`
