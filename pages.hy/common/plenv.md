# plenv

> Անցում Perl-ի բազմաթիվ տարբերակների միջև:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tokuhirom/plenv#name>:.

- Ցույց տվեք ներկայումս ընտրված Perl տարբերակը և ինչպես է այն ընտրվել.:

`plenv version`

- Թվարկեք բոլոր առկա տեղադրված Perl տարբերակները.:

`plenv versions`

- Սահմանեք գլոբալ Perl տարբերակը (օգտագործվում է, եթե տեղական կամ shell տարբերակը առաջնահերթություն չի ստանում).:

`plenv global {{version}}`

- Սահմանեք տեղական հավելվածին հատուկ Perl տարբերակը (օգտագործվում է ընթացիկ գրացուցակում և դրա տակ գտնվող բոլոր դիրեկտորիաներում).:

`plenv local {{version}}`

- Սահմանեք կեղևի հատուկ Perl տարբերակը (օգտագործվում է միայն ընթացիկ նստաշրջանի համար).:

`plenv shell {{version}}`

- Ցուցադրել օգնությունը.:

`plenv`

- Ցուցադրել օգնություն հրամանի համար.:

`plenv help {{command}}`
