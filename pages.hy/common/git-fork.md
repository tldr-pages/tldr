# git պատառաքաղ

> Պատառաքաղել GitHub ռեպո: Հավանել `git clone`-ը, բայց առաջինը պատառաքաղել:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-fork>:.

- Պատառաքաղեք և կլոնավորեք GitHub-ի պահոցն իր URL-ով.:

`git fork {{https://github.com/tldr-pages/tldr}}`

- Պատառաքաղել և կլոնավորել GitHub-ի պահոցն իր slug-ով.:

`git fork {{tldr-pages/tldr}}`
