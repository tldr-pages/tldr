#git grep

> Գտեք տողեր պահոցում հետևված ֆայլերի ներսում:.
> Ընդունում է շատ նույն դրոշները, ինչ սովորական `grep`:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-grep>:.

- Փնտրեք տող ֆայլերում ընթացիկ `HEAD`-ում:

`git grep "{{search_string}}"`

- Ներկայիս `HEAD`-ում գլոբի օրինաչափությանը համապատասխանող ֆայլերում տող փնտրեք:

`git grep "{{search_string}}" -- "{{*.ext}}"`

- Փնտրեք տող, ներառյալ ենթամոդուլները.:

`git grep --recurse-submodules "{{search_string}}"`

- Փնտրեք տող պատմության որոշակի կետում.:

`git grep "{{search_string}}" {{HEAD~2}}`

- Փնտրեք տող բոլոր ճյուղերում և ամբողջ պատմության մեջ.:

`git grep "{{search_string}}" $(git rev-list --all)`
