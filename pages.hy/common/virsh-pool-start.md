# virsh pool-start

> Սկսեք նախկինում կազմաձևված, բայց ոչ ակտիվ վիրտուալ մեքենայի պահեստավորման լողավազան:.
> Տես նաև՝ `virsh`, `virsh-pool-define-as`, `virsh-pool-destroy`:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/virsh>:.

- Սկսեք անվանմամբ կամ UUID-ով նշված պահեստային ավազանը (որոշեք՝ օգտագործելով `virsh pool-list`) և ստեղծեք հիմքում ընկած պահեստային համակարգը, եթե այն գոյություն չունի.:

`virsh pool-start --pool {{name|uuid}} --build`
