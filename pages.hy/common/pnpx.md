# pnpx

> Ուղղակիորեն գործարկեք երկուականներ npm փաթեթներից՝ օգտագործելով `pnpm` `npm`-ի փոխարեն:.
> Նշում. այս հրամանը հնացած է: Փոխարենը օգտագործեք `pnpm exec` և `pnpm dlx`:.
> Լրացուցիչ տեղեկություններ. <https://cuyl.github.io/pnpm.github.io/pnpx-cli/>:.

- Կատարեք երկուականը տրված `npm` մոդուլից.:

`pnpx {{module_name}}`

- Կատարեք որոշակի երկուական տվյալ `npm` մոդուլից, եթե մոդուլն ունի բազմաթիվ երկուականներ.:

`pnpx --package {{package_name}} {{module_name}}`

- Ցուցադրել օգնությունը.:

`pnpx --help`
