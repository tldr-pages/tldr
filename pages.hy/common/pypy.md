# pypy

> Python լեզվի արագ և համապատասխան այլընտրանքային իրականացում:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/pypy>:.

- Սկսեք REPL (ինտերակտիվ կեղև).:

`pypy`

- Կատարել սկրիպտը տվյալ Python ֆայլում.:

`pypy {{path/to/file.py}}`

- Կատարեք սկրիպտը որպես ինտերակտիվ կեղևի մաս.:

`pypy -i {{path/to/file.py}}`

- Կատարեք Python արտահայտություն.:

`pypy -c "{{expression}}"`

- Գործարկեք գրադարանի մոդուլը որպես սցենար (վերջացնում է ընտրանքների ցանկը).:

`pypy -m {{module}} {{arguments}}`

- Տեղադրեք փաթեթ՝ օգտագործելով pip.:

`pypy -m pip install {{package}}`

- Ինտերակտիվ կարգաբերել Python սկրիպտը.:

`pypy -m pdb {{path/to/file.py}}`
