#ջատ

> Java կույտի վերլուծության գործիք:.
> Լրացուցիչ տեղեկություններ. <https://docs.oracle.com/javase/8/docs/technotes/tools/unix/jhat.html>:.

- Վերլուծեք կույտային աղբավայրը (`jmap`-ից), դիտեք HTTP-ի միջոցով 7000 նավահանգստում.:

`jhat {{dump_file.bin}}`

- Վերլուծեք կույտային աղբարկղը՝ նշելով այլընտրանքային նավահանգիստ HTTP սերվերի համար.:

`jhat {{[-p|-port]}} {{port}} {{dump_file.bin}}`

- Վերլուծեք աղբավայրը, որը թույլ է տալիս `jhat`-ն օգտագործել մինչև 8 ԳԲ RAM (խորհուրդ է տրվում 2-4 անգամ աղբահանության չափը):

`jhat -J-mx8G {{dump_file.bin}}`
