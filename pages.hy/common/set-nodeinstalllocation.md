# Set-NodeInstallLocation

> Սահմանեք Node.js-ի նախնական տեղադրման գրացուցակը `ps-nvm`-ի համար:.
> Այս հրամանը `ps-nvm`-ի մասն է և կարող է գործարկվել միայն PowerShell-ի տակ:.
> Լրացուցիչ տեղեկություններ. <https://github.com/aaronpowell/ps-nvm>:.

- Փոխեք Node.js-ի տեղադրման վայրը նշված գրացուցակի մեջ (`ps-nvm`-ը կստեղծի նոր `.nvm` ենթացանց՝ դրանք տեղադրելու համար):

`Set-NodeInstallLocation {{path/to/directory}}`
