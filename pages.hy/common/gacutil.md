# gacutil

> Global Assembly Cache (GAC) կառավարման կոմունալ:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/gacutil>:.

- Տեղադրեք նշված ժողովը GAC-ում.:

`gacutil -i {{path/to/assembly.dll}}`

- Տեղահանեք նշված ժողովը GAC-ից.:

`gacutil -u {{assembly_display_name}}`

- Տպել GAC-ի բովանդակությունը.:

`gacutil -l`
