# գնալ fmt

> Ձևաչափեք Go սկզբնաղբյուր ֆայլերը՝ տպելով փոխված ֆայլերի անունները:.
> Նշում. `./...`-ը Go փաթեթի օրինաչափություն է, որը հասկանալի է Go tooling-ով: Այն համապատասխանում է ընթացիկ փաթեթին և բոլոր փաթեթներին ռեկուրսիվորեն ընթացիկ գրացուցակի տակ:.
> Լրացուցիչ տեղեկություններ. <https://pkg.go.dev/cmd/go#hdr-Gofmt__reformat__package_sources>:.

- Ձևաչափեք Go աղբյուրի ֆայլերը ընթացիկ գրացուցակում.:

`go fmt`

- Ձևաչափեք որոշակի Go փաթեթ ձեր ներմուծման ճանապարհին (`$GOPATH/src`):

`go fmt {{path/to/package}}`

- Ձևաչափեք փաթեթը ընթացիկ գրացուցակում և բոլոր ենթագրքերում (նշեք `...`-ը).:

`go fmt {{./...}}`

- Տպեք, թե ինչ ձևաչափի հրամաններ կգործարկվեին, առանց որևէ բան փոփոխելու.:

`go fmt -n`

- Տպեք, թե որ ձևաչափի հրամաններն են գործարկվում, երբ դրանք գործարկվում են.:

`go fmt -x`
