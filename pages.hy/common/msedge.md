# msedge

> Microsoft Edge հրամանի տողի օգտակար ծրագիրը հասանելի է որպես `msedge` Windows-ում և `microsoft-edge` այլ հարթակների համար:.
> Լրացուցիչ տեղեկություններ. <https://microsoft.com/edge>:.

- Դիտեք փաստաթղթերը Microsoft Edge-ի համար Windows-ի համար.:

`tldr {{[-p|--platform]}} windows msedge`

- Դիտեք Microsoft Edge-ի փաստաթղթերը այլ հարթակների համար.:

`tldr {{[-p|--platform]}} common microsoft-edge`
