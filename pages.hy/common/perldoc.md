# perldoc

> Փնտրեք Perl փաստաթղթերը `.pod` ձևաչափով:.
> Լրացուցիչ տեղեկություններ. <https://perldoc.perl.org/perldoc>:.

- Դիտեք փաստաթղթերը ներկառուցված [f] ֆունկցիայի, [v] փոփոխականի կամ [a]PI-ի համար.:

`perldoc -{{f|v|a}} {{name}}`

- Որոնեք Perl FAQ-ի [q]հարցերի վերնագրերում.:

`perldoc -q {{regex}}`

- Ուղարկեք ելքը անմիջապես `stdout`-ին (ըստ լռելյայն, այն ուղարկվում է փեյջերին).:

`perldoc -T {{page|module|program|url}}`

- Նշեք ցանկալի թարգմանության լեզվի կոդը.:

`perldoc -L {{language_code}} {{page|module|program|url}}`
