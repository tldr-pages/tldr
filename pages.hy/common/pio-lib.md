# pio lib

> Կառավարեք PlatformIO գրադարանները:.
> Լրացուցիչ տեղեկություններ. <https://docs.platformio.org/en/latest/core/userguide/lib/>:.

- Տեղադրված գրադարանների ցանկ.:

`pio lib list`

- Ցուցակե՛ք ներկառուցված գրադարանները՝ հիմնված տեղադրված զարգացման հարթակների և դրանց շրջանակների վրա.:

`pio lib builtin`

- Գոյություն ունեցող գրադարանների որոնում.:

`pio lib search {{keyword}}`

- Ցույց տալ գրադարանի մասին մանրամասներ.:

`pio lib show {{library}}`

- Տեղադրեք գրադարան.:

`pio lib install {{library}}`

- Թարմացրեք տեղադրված գրադարանները.:

`pio lib update`

- Տեղահանել գրադարանը.:

`pio lib uninstall {{library}}`

- Ցույց տալ PlatformIO գրադարանի ռեեստրի վիճակագրությունը.:

`pio lib stats`
