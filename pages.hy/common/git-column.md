# git սյունակ

> Ցուցադրել տվյալները սյունակներում:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-column>:.

- Ձևաչափեք `stdin`-ը որպես բազմաթիվ սյունակներ.:

`ls | git column --mode={{column}}`

- Ձևաչափեք `stdin`-ը որպես `100` առավելագույն լայնությամբ բազմաթիվ սյունակներ:

`ls | git column --mode=column --width={{100}}`

- Ձևաչափեք `stdin`-ը որպես մի քանի սյունակներ՝ `30` առավելագույն լրացումով:

`ls | git column --mode=column --padding={{30}}`
