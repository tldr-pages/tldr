# git worktree

> Կառավարեք մի քանի աշխատող ծառեր, որոնք կցված են նույն պահեստին:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-worktree>:.

- Ավելացնել նոր աշխատատեղի գրացուցակ նույն անունով մասնաճյուղի համար (ստեղծվել է, եթե բացակայում է).:

`git worktree add {{branch}}`

- Ստեղծեք նոր գրացուցակ, որի մեջ նշված մասնաճյուղը ստուգված է.:

`git worktree add {{path/to/directory}} {{branch}}`

- Ստեղծեք նոր գրացուցակ դրանում ստուգված նոր մասնաճյուղով.:

`git worktree add {{path/to/directory}} -b {{new_branch}}`

- Թվարկեք բոլոր աշխատանքային գրացուցակները (ներառյալ առաջնայինը).:

`git worktree list`

- Տեղափոխեք գոյություն ունեցող աշխատանքային ծառը նոր վայր.:

`git worktree move {{path/to/worktree}} {{new/path}}`

- Հեռացրեք worktree գրացուցակը և դրա մետատվյալները (միայն այն դեպքում, եթե այն չկատարված փոփոխություններ չունի).:

`git worktree remove {{path/to/worktree}}`

- Հեռացրեք մետատվյալները ձեռքով ջնջված worktree գրացուցակի համար.:

`git worktree prune`
