# git կեղծանուն

> Ստեղծեք դյուրանցումներ Git հրամանների համար:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-alias>:.

- Թվարկեք բոլոր կեղծանունները.:

`git alias`

- Ստեղծեք նոր կեղծանուն.:

`git alias "{{name}}" "{{command}}"`

- Որոնեք գոյություն ունեցող կեղծանունը.:

`git alias ^{{name}}`
