# nh որոնում

> Փաթեթների որոնում Nixpkgs-ում:.
> Լրացուցիչ տեղեկություններ. <https://github.com/nix-community/nh#usage>:.

- Փաթեթ փնտրեք Nixpkgs-ում, սահմանափակելով արդյունքները.:

`nh search {{[-l|--limit]}} {{number}} {{name}}`

- Փաթեթ փնտրեք նշված Nixpkgs ալիքում.:

`nh search {{[-c|--channel]}} {{nixos-unstable}} {{name}}`

- Փնտրեք փաթեթ Nixpkgs-ում, ցույց տալով աջակցվող հարթակներ յուրաքանչյուր փաթեթի համար.:

`nh search {{[-P|--platforms]}} {{name}}`
