# ուլտրամանուշակագույն ծառ

> Ցուցադրել նախագծի կախվածությունները ծառի ձևաչափով:.
> Լրացուցիչ տեղեկություններ. <https://docs.astral.sh/uv/reference/cli/#uv-tree>:.

- Ցույց տալ կախվածության ծառը ընթացիկ միջավայրի համար.:

`uv tree`

- Ցույց տալ կախվածության ծառը բոլոր միջավայրերի համար.:

`uv tree --universal`

- Ցույց տալ կախվածության ծառը մինչև որոշակի խորություն.:

`uv tree {{[-d|--depth]}} {{n}}`

- Ցույց տալ վերջին հասանելի տարբերակը բոլոր հնացած փաթեթների համար.:

`uv tree --outdated`

- Բացառել կախվածությունները մշակող խմբից.:

`uv tree --no-dev`

- Ցույց տվեք շրջված ծառը, այնպես որ երեխաները կախվածության փոխարեն կախված են.:

`uv tree --invert`
