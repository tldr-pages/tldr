# docker կոնտեյների վերանվանում

> Վերանվանել կոնտեյները:.
> Լրացուցիչ տեղեկություններ. <https://docs.docker.com/reference/cli/docker/container/rename/>:.

- Վերանվանել կոնտեյները.:

`docker {{[rename|container rename]}} {{container}} {{new_name}}`

- Ցուցադրել օգնությունը.:

`docker {{[rename|container rename]}} --help`
