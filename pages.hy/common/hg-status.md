# hg կարգավիճակ

> Ցույց տալ ֆայլերը, որոնք փոխվել են աշխատանքային գրացուցակում:.
> Լրացուցիչ տեղեկություններ. <https://www.mercurial-scm.org/help/commands/status>:.

- Ցուցադրել փոփոխված ֆայլերի կարգավիճակը.:

`hg status`

- Ցուցադրել միայն փոփոխված ֆայլերը.:

`hg status {{[-m|--modified]}}`

- Ցուցադրել միայն ավելացված ֆայլերը.:

`hg status {{[-a|--added]}}`

- Ցուցադրել միայն հեռացված ֆայլերը.:

`hg status {{[-r|--removed]}}`

- Ցուցադրել միայն ջնջված (բայց հետևված) ֆայլերը.:

`hg status {{[-d|--deleted]}}`

- Ցուցադրել փոփոխությունները աշխատանքային գրացուցակում՝ համեմատած նշված փոփոխությունների փաթեթի հետ.:

`hg status --rev {{revision}}`

- Ցուցադրել միայն նշված գլոբուսի օրինաչափությանը համապատասխանող ֆայլեր.:

`hg status {{[-I|--include]}} {{pattern}}`

- Ցուցադրել ֆայլերը, բացառությամբ նրանց, որոնք համապատասխանում են նշված գլոբուսի օրինակին.:

`hg status {{[-X|--exclude]}} {{pattern}}`
