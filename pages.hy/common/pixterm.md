# pixterm

> Պատկերի տպում տերմինալում:.
> Տես նաև՝ `chafa`, `catimg`:.
> Լրացուցիչ տեղեկություններ. <https://github.com/eliukblau/pixterm#cool-screenshots>:.

- Ներկայացրեք ստատիկ պատկեր անմիջապես տերմինալում.:

`pixterm {{path/to/file}}`

- Օգտագործեք պատկերի բնօրինակ կողմի հարաբերակցությունը.:

`pixterm -s 2 {{path/to/file}}`

- Նշեք հարմարեցված կողմի հարաբերակցությունը, օգտագործելով որոշակի թվով [t] տերմինալ [r] տողեր և [c] սյունակներ.:

`pixterm -tr {{24}} -tc {{80}} {{path/to/file}}`

- Զտեք ելքը [m]atte ֆոնի գույնով և [d] նիշով.:

`pixterm -m {{000000}} -d 2 {{path/to/file}}`
