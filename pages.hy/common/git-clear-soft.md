# git պարզ-փափուկ

> Մաքրել Git-ի աշխատանքային գրացուցակը, կարծես այն թարմ կլոնավորված է ընթացիկ մասնաճյուղով, բացառությամբ `.gitignore`-ի ֆայլերի:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-clear-soft>:.

- Վերականգնել բոլոր հետևված ֆայլերը և ջնջել բոլոր չհետևված ֆայլերը.:

`git clear-soft`
