# npm դիտում

> Դիտեք ռեեստրի տեղեկատվությունը փաթեթի մասին:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-view/>:.

- Դիտեք տեղեկատվություն փաթեթի վերջին տարբերակի մասին.:

`npm {{[v|view]}} {{package}}`

- Դիտեք տեղեկատվություն փաթեթի որոշակի տարբերակի մասին.:

`npm {{[v|view]}} {{package}}@{{version}}`

- Դիտեք փաթեթի բոլոր հասանելի տարբերակները.:

`npm {{[v|view]}} {{package}} versions`

- Դիտեք փաթեթի նկարագրությունը.:

`npm {{[v|view]}} {{package}} description`

- Դիտեք փաթեթի վերջին տարբերակի կախվածությունները.:

`npm {{[v|view]}} {{package}} dependencies`

- Դիտեք փաթեթի պահեստի URL-ը.:

`npm {{[v|view]}} {{package}} repository`

- Դիտեք փաթեթի սպասարկողներին.:

`npm {{[v|view]}} {{package}} maintainers`
