# բեռնափոխադրումներ

> Հեռացրեք հրված տուփը ցուցիչից: Սա պետք է օգտագործվի միայն այն դեպքում, երբ դուք պատահաբար բաց եք թողնում զգալիորեն կոտրված վանդակը:.
> Նշում. սա չի հեռացնում որևէ տվյալ: Արկղը դեռևս առկա է անկումից հետո, դա պարզապես թույլ չի տալիս նոր նախագծերին օգտագործել այն:.
> Լրացուցիչ տեղեկություններ. <https://doc.rust-lang.org/cargo/commands/cargo-yank.html>:.

- Անկացրեք տուփի նշված տարբերակը.:

`cargo yank {{crate}}@{{version}}`

- Հետարկել անկումը (այսինքն՝ թույլ տալ նորից ներբեռնել այն).:

`cargo yank --undo {{crate}}@{{version}}`

- Օգտագործեք նշված ռեեստրը (ռեգիստրի անունները կարող են սահմանվել կազմաձևում - լռելյայն է <https://crates.io>):

`cargo yank --registry {{name}} {{crate}}@{{version}}`
