# պոեզիայի համաժամացում

> Համաժամացնում է ձեր նախագծի միջավայրը `poetry.lock` ֆայլի հետ:.
> Լրացուցիչ տեղեկություններ. <https://python-poetry.org/docs/cli/#sync>:.

- Համաժամացրեք ձեր նախագծի միջավայրը `poetry.lock` ֆայլի հետ.:

`poetry sync`

- Բացառեք մեկ կամ մի քանի կախվածության խմբեր տեղադրման համար.:

`poetry sync --without {{test|docs|...}}`

- Ընտրեք կամընտիր կախվածության խմբեր.:

`poetry sync --with {{test|docs|...}}`

- Տեղադրեք բոլոր կախվածության խմբերը, ներառյալ կամընտիր խմբերը.:

`poetry sync --all-groups`

- Տեղադրեք հատուկ կախվածության խմբեր.:

`poetry sync --only {{test|docs|...}}`

- Տեղադրեք նախագիծը առանց կախվածության.:

`poetry sync --only-root`

- Նշեք լրացուցիչները՝ տեղադրելու համար.:

`poetry sync {{[-E|--extras]}}`

- Բաց թողեք ձեր նախագծի համար լռելյայն փաթեթի տեղադրումը.:

`poetry sync --no-root`
