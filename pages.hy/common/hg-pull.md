# hg քաշեք

> Տեղափոխեք փոփոխությունները նշված պահոցից դեպի տեղական պահեստ:.
> Լրացուցիչ տեղեկություններ. <https://www.mercurial-scm.org/help/commands/pull>:.

- Քաշեք «կանխադրված» աղբյուրի ուղուց.:

`hg pull`

- Քաշեք նշված աղբյուրի պահոցից.:

`hg pull {{path/to/source_repository}}`

- Թարմացրեք տեղական պահոցը հեռակառավարման ղեկավարին.:

`hg pull {{[-u|--update]}}`

- Քաշեք փոփոխությունները նույնիսկ այն դեպքում, երբ հեռավոր պահեստը կապ չունի.:

`hg pull {{[-f|--force]}}`

- Նշեք որոշակի վերանայման փոփոխությունների հավաքածու՝ հասնելու համար՝:

`hg pull {{[-r|--rev]}} {{revision}}`

- Նշեք հատուկ ճյուղ, որը պետք է քաշվի.:

`hg pull {{[-b|--branch]}} {{branch}}`

- Նշեք հատուկ էջանիշ քաշելու համար.:

`hg pull {{[-B|--bookmark]}} {{bookmark}}`
