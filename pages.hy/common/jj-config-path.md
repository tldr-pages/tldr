# jj կազմաձևման ուղի

> Տպեք կոնֆիգուրացիայի ֆայլերի ուղիները:.
> Տես նաև՝ `jj config edit`:.
> Լրացուցիչ տեղեկություններ. <https://docs.jj-vcs.dev/latest/cli-reference/#jj-config-path>:.

- Տպեք ուղին դեպի օգտվողի մակարդակի կազմաձևման ֆայլ.:

`jj config {{[p|path]}} --user`

- Տպեք ռեպո մակարդակի կազմաձևման ֆայլի ուղին.:

`jj config {{[p|path]}} --repo`

- Տպեք աշխատանքային տարածքի մակարդակի կազմաձևման ֆայլի ուղին.:

`jj config {{[p|path]}} --workspace`
