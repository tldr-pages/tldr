#սասս

> Փոխարկել SCSS կամ Sass ֆայլերը CSS:.
> Լրացուցիչ տեղեկություններ. <https://sass-lang.com/documentation/cli/dart-sass/>:.

- Փոխակերպեք SCSS կամ Sass ֆայլը CSS և տպեք արդյունքը.:

`sass {{inputfile.scss|inputfile.sass}}`

- Փոխակերպեք SCSS կամ Sass ֆայլը CSS և արդյունքը պահեք ֆայլի մեջ.:

`sass {{inputfile.scss|inputfile.sass}} {{outputfile.css}}`

- Դիտեք SCSS կամ Sass ֆայլը փոփոխությունների համար և թողարկեք կամ թարմացրեք CSS ֆայլը նույն ֆայլի անունով.:

`sass --watch {{inputfile.scss|inputfile.sass}}`

- Դիտեք SCSS կամ Sass ֆայլը փոփոխությունների համար և թողարկեք կամ թարմացրեք CSS ֆայլը տվյալ ֆայլի անունով.:

`sass --watch {{inputfile.scss|inputfile.sass}}:{{outputfile.css}}`
