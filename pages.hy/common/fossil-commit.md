# բրածո պարտավորություն

> Հանձնել ֆայլերը Fossil պահեստում:.
> Լրացուցիչ տեղեկություններ. <https://fossil-scm.org/home/help/commit>:.

- Ստեղծեք նոր տարբերակ, որը պարունակում է ընթացիկ վճարման բոլոր փոփոխությունները. օգտվողին կառաջարկվի մեկնաբանություն.:

`fossil {{[ci|commit]}}`

- Ստեղծեք նոր տարբերակ, որը պարունակում է ընթացիկ վճարման բոլոր փոփոխությունները՝ օգտագործելով նշված [m] հաղորդագրությունը.:

`fossil {{[ci|commit]}} {{[-m|--comment]}} "{{comment}}"`

- Ստեղծեք նոր տարբերակ, որը պարունակում է ընթացիկ վճարման բոլոր փոփոխությունները հատուկ ֆայլից ընթերցված մեկնաբանությամբ.:

`fossil {{[ci|commit]}} {{[-M|--message-file]}} {{path/to/commit_message_file}}`

- Ստեղծեք նոր տարբերակ, որը պարունակում է փոփոխություններ նշված ֆայլերից; օգտվողին կառաջարկվի մեկնաբանություն.:

`fossil {{[ci|commit]}} {{path/to/file1 path/to/file2 ...}}`
