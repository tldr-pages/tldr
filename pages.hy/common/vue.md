#վյու

> Բազմաֆունկցիոնալ CLI Vue.js-ի համար:.
> Որոշ ենթահրամաններ, ինչպիսիք են `build`-ն, ունեն իրենց օգտագործման փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://cli.vuejs.org/guide/>:.

- Ստեղծեք նոր Vue նախագիծ ինտերակտիվ կերպով.:

`vue create {{project_name}}`

- Ստեղծեք նոր նախագիծ վեբ UI-ով.:

`vue ui`
