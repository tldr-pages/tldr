# aws պատմություն

> Տպել հրամանի տողի պատմությունը AWS CLI հրամանների համար (AWS CLI հրամանների պատմության գրառումը պետք է միացված լինի):.
> Լրացուցիչ տեղեկություններ. <https://docs.aws.amazon.com/cli/latest/reference/history/>:.

- Ցուցակ հրամանների պատմությունը հրամանների ID-ներով.:

`aws history list`

- Ցուցադրել իրադարձությունները, որոնք կապված են հատուկ հրամանի հետ, որը տրված է հրամանի ID.:

`aws history show {{command_id}}`
