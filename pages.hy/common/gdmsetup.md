# gdmsetup

> Գործարկեք GUI՝ GNOME Display Manager (GDM) կարգավորելու համար:.
> Արդյունքները խմբագրված ֆայլերում՝ `/etc/X11/gdm/gdm.conf` կամ `/etc/gdm/gdm.conf`՝ կախված բաշխումից և միջավայրից:.
> Տես նաև՝ `gdm`, `gdm-binary`, `gdm-stop`, `gdm-restart`, `gdm-safe-restart`:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/gdm>:.

- Գործարկեք GNOME Display Manager GUI-ի կազմաձևման հավելվածը.:

`gdmsetup`
