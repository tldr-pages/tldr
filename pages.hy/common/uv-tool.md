# ուլտրամանուշակագույն գործիք

> Տեղադրեք և գործարկեք Python փաթեթների կողմից տրամադրված հրամանները:.
> Լրացուցիչ տեղեկություններ. <https://docs.astral.sh/uv/reference/cli/#uv-tool>:.

- Գործարկեք հրաման փաթեթից՝ առանց այն տեղադրելու.:

`uv tool run {{command}}`

- Տեղադրեք Python փաթեթ համակարգով.:

`uv tool install {{package}}`

- Թարմացրեք տեղադրված Python փաթեթը.:

`uv tool upgrade {{package}}`

- Տեղահանել Python փաթեթը.:

`uv tool uninstall {{package}}`

- Ցուցակեք Python փաթեթները, որոնք տեղադրված են համակարգով.:

`uv tool list`
