# ssh-գործակալ

> Ստեղծեք SSH գործակալի գործընթաց:.
> Նշում. SSH գործակալը պահում է SSH ստեղները գաղտնազերծված հիշողության մեջ, մինչև չհեռացվի կամ գործընթացը չավարտվի:.
> Տես նաև՝ `ssh-add`:.
> Լրացուցիչ տեղեկություններ. <https://man.openbsd.org/ssh-agent>:.

- Գործարկեք SSH գործակալ ընթացիկ կեղևի համար.:

`eval $(ssh-agent)`

- Սպանեք ներկայումս գործող գործակալին.:

`ssh-agent -k`
