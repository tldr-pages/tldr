# կռունկի ցուցիչի հավելված

> Կցեք մանիֆեստը հեռավոր ինդեքսին:.
> Այս ենթահրամանը մղում է ինդեքս, որը հիմնված է (կամընտիր) բազային ինդեքսի վրա՝ կից մանիֆեստներով:.
> Կցված մանիֆեստների հարթակը ենթադրվում է կազմաձևման ֆայլից կամ բաց է թողնվում, եթե դա անիրագործելի է:.
> Լրացուցիչ տեղեկություններ. <https://github.com/google/go-containerregistry/blob/main/cmd/crane/doc/crane_index_append.md>:.

- Կցեք մանիֆեստը հեռավոր ինդեքսին՝:

`crane index append`

- Հղում մանիֆեստներին՝ բազային ինդեքսին ավելացնելու համար.:

`crane index append {{[-m|--manifest]}} {{manifest_name1 manifest_name2 ...}}`

- Ստացված պատկերին կիրառելու համար նշեք.:

`crane index append {{[-t|--tag]}} {{tag_name}}`

- Դատարկ բազային ինդեքսը կունենա Docker մեդիա տեսակներ՝ OCI-ի փոխարեն.:

`crane index append --docker-empty-base`

- Կցեք իր երեխաներից յուրաքանչյուրին, այլ ոչ թե բուն ինդեքսը (կանխադրվածը ճշմարիտ է).:

`crane index append --flatten`

- Ցուցադրել օգնությունը.:

`crane index append {{[-h|--help]}}`
