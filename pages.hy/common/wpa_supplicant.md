# wpa_supplicant

> Կառավարեք պաշտպանված անլար ցանցերը:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/wpa_supplicant.1>:.

- Միացեք պաշտպանված անլար ցանցին.:

`wpa_supplicant -i {{interface}} -c {{path/to/wpa_supplicant_conf.conf}}`

- Միացեք պաշտպանված անլար ցանցին և գործարկեք այն դեյմոնով.:

`wpa_supplicant -B -i {{interface}} -c {{path/to/wpa_supplicant_conf.conf}}`
