# git local-commits

> Ցույց տալ տեղական պարտավորությունները, որոնք չեն առաջացել: Ցանկացած լրացուցիչ փաստարկ կփոխանցվի անմիջապես `git log`-ին:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-local-commits>:.

- Ցույց տալ պարտավորությունները, որոնք չեն դրվել.:

`git local-commits`
