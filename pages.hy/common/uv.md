#ուլտրամանուշակագույն

> Արագ Python փաթեթ և նախագծի կառավարիչ:.
> Որոշ ենթահրամաններ, ինչպիսիք են `tool`-ը և `python`-ն, ունեն իրենց օգտագործման փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://docs.astral.sh/uv/reference/cli/>:.

- Ստեղծեք նոր Python նախագիծ ընթացիկ գրացուցակում.:

`uv init`

- Ստեղծեք նոր Python նախագիծ նշված ճանապարհով.:

`uv init {{path/to/directory}}`

- Նախագծին ավելացրեք նոր կախվածություն.:

`uv add {{package}}`

- Հեռացրեք կախվածությունը նախագծից.:

`uv remove {{package}}`

- Գործարկեք սցենար կամ հրաման նախագծի միջավայրում.:

`uv run {{path/to/script.py|command}}`

- Թարմացրեք նախագծի միջավայրը `pyproject.toml`-ից՝:

`uv sync`

- Ստեղծեք կողպեքի ֆայլ՝ նախագծի կախվածությունների համար.:

`uv lock`

- Կառուցեք նախագիծը աղբյուրի և երկուական բաշխումների մեջ.:

`uv build`
