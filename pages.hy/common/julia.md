# Ջուլիա

> Բարձր մակարդակի, բարձր կատարողականության դինամիկ ծրագրավորման լեզու տեխնիկական հաշվարկների համար:.
> Լրացուցիչ տեղեկություններ. <https://docs.julialang.org/en/v1/manual/getting-started/>:.

- Սկսեք REPL (ինտերակտիվ կեղև).:

`julia`

- Կատարեք Julia ծրագիրը և դուրս եկեք.:

`julia {{program.jl}}`

- Գործարկեք Julia ծրագիր, որն ընդունում է փաստարկներ.:

`julia {{program.jl}} {{arguments}}`

- Գնահատեք Ջուլիայի կոդը պարունակող տողը.:

`julia {{[-e|--eval]}} '{{julia_code}}'`

- Գնահատեք Ջուլիայի կոդի տողը, դրան փաստարկներ փոխանցելով.:

`julia {{[-e|--eval]}} '{{for x in ARGS; println(x); end}}' {{arguments}}`

- Գնահատեք արտահայտությունը և տպեք արդյունքը.:

`julia {{[-E|--print]}} '{{(1 - cos(pi/4))/2}}'`

- Սկսեք Julia-ն բազմաթելային ռեժիմով՝ օգտագործելով `n` թելերը.:

`julia {{[-t|--threads]}} {{n}}`
