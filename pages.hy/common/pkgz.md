# pkgz

> Միասնական փաթեթի կառավարիչ բազմաթիվ աղբյուրների համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/roguehashrate/pkgz#-usage>:.

- Տեղադրեք փաթեթ.:

`pkgz install {{package_name}}`

- Հեռացնել փաթեթը.:

`pkgz remove {{package_name}}`

- Փաթեթի որոնում.:

`pkgz search {{package_name}}`

- Ցույց տալ, թե քանի փաթեթ է տեղադրված մեկ աղբյուրի համար.:

`pkgz info`

- Ցույց տալ, թե որտեղ է տեղադրված կամ հասանելի փաթեթը.:

`pkgz info {{package_name}}`

- Թարմացրեք բոլոր փաթեթները բոլոր աղբյուրներում.:

`pkgz update`

- Հեռացրեք բոլոր աջակցվող աղբյուրների քեշը.:

`pkgz clean`

- Ցուցադրման տարբերակը:

`pkgz --version`
