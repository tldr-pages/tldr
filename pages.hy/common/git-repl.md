# git repl

> Git REPL (read-evaluate-print-loop) - ինտերակտիվ Git կեղև:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-repl>:.

- Սկսեք ինտերակտիվ Git shell.:

`git repl`

- Գործարկեք Git հրամանը, երբ գտնվում եք ինտերակտիվ Git վահանակում.:

`{{git_subcommand}} {{command_arguments}}`

- Գործարկեք արտաքին (ոչ Git) հրաման, երբ գտնվում եք ինտերակտիվ Git վահանակում.:

`!{{command}} {{command_arguments}}`

- Դուրս եկեք ինտերակտիվ Git shell-ից (կամ սեղմեք `<Ctrl d>`):

`exit`
