# ipython

> Python-ի կեղև՝ ավտոմատ պատմությամբ, օբյեկտների դինամիկ ներդիտումով, ավելի հեշտ կոնֆիգուրացիայով, հրամանի ավարտով, մուտք դեպի համակարգի պատյան և այլն:.
> Լրացուցիչ տեղեկություններ. <https://ipython.readthedocs.io/en/stable/config/options/index.html>:.

- Սկսեք REPL (ինտերակտիվ կեղև).:

`ipython`

- Մուտքագրեք ինտերակտիվ IPython նիստը Python սկրիպտը գործարկելուց հետո.:

`ipython -i {{script.py}}`

- Ստեղծեք լռելյայն IPython պրոֆիլ.:

`ipython profile create`

- Տպեք ուղին դեպի գրացուցակ լռելյայն IPython պրոֆիլի համար.:

`ipython locate profile`

- Մաքրել IPython պատմության տվյալների բազան՝ ջնջելով բոլոր գրառումները.:

`ipython history clear`
