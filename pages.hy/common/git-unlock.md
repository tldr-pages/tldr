# git unlock

> Ապակողպեք ֆայլը Git-ի պահոցում, որպեսզի այն հնարավոր լինի փոփոխել commit-ի միջոցով:.
> `git-extras`-ի մի մասը:.
> Տես նաև՝ `git lock`:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-unlock>:.

- Միացնել նախկինում կողպված տեղական ֆայլի փոփոխությունները կատարելու հնարավորությունը.:

`git unlock {{path/to/file}}`
