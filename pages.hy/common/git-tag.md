# git tag

> Ստեղծեք, ցուցակագրեք, ջնջեք կամ հաստատեք պիտակներ:.
> Պիտակը ստատիկ հղում է ստանձնման:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-tag>:.

- Նշեք բոլոր պիտակները.:

`git tag`

- Ստեղծեք պիտակ տվյալ անունով՝ մատնանշելով ընթացիկ պարտավորությունը.:

`git tag {{tag_name}}`

- Ստեղծեք պիտակ՝ տրված անունով մատնանշելով տվյալ commit-ը.:

`git tag {{tag_name}} {{commit}}`

- Ստեղծեք ծանոթագրված պիտակ տվյալ հաղորդագրությամբ.:

`git tag {{tag_name}} {{[-m|--message]}} {{tag_message}}`

- Ջնջել տվյալ անունով պիտակը.:

`git tag {{[-d|--delete]}} {{tag_name}}`

- Ստացեք թարմացված պիտակներ հեռակաից.:

`git fetch {{[-t|--tags]}}`

- Հրել պիտակը հեռակառավարման վրա.:

`git push origin tag {{tag_name}}`

- Թվարկեք բոլոր պիտակները, որոնք պարունակում են տվյալ commit (`HEAD` եթե նշված չէ):

`git tag --contains {{commit}}`
