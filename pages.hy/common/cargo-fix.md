# բեռնափոխադրումներ

> Ավտոմատ կերպով շտկել `rustc`-ով հաղորդված ցողունային նախազգուշացումները:.
> Լրացուցիչ տեղեկություններ. <https://doc.rust-lang.org/cargo/commands/cargo-fix.html>:.

- Ուղղեք կոդը, նույնիսկ եթե այն արդեն ունի կոմպիլյատորի սխալներ.:

`cargo fix --broken-code`

- Ուղղել կոդը, նույնիսկ եթե աշխատանքային գրացուցակը փոխվել է.:

`cargo fix --allow-dirty`

- Տեղափոխեք փաթեթը հաջորդ Rust հրատարակությանը.:

`cargo fix --edition`

- Ամրագրել փաթեթի գրադարանը.:

`cargo fix --lib`

- Ուղղեք նշված ինտեգրման թեստը.:

`cargo fix --test {{name}}`

- Ամրագրեք բոլոր անդամներին աշխատանքային տարածքում.:

`cargo fix --workspace`
