#սքամփեր

> Ակտիվորեն զննում է ինտերնետը, որպեսզի վերլուծի տոպոլոգիան և կատարողականությունը:.
> Ներառում է որոշ գործիքներ, որոնք սկսվում են `sc_`-ով, օրինակ՝ `sc_warts2text` կամ `sc_ttlexp`:.
> Լրացուցիչ տեղեկություններ. <https://www.caida.org/catalog/software/scamper/>:.

- Կատարեք ստանդարտ տարբերակը (traceroute) դեպի նպատակակետ.:

`scamper -i {{192.0.2.1}}`

- Կատարեք երկու գործողություն (ping և traceroute) երկու տարբեր թիրախների վրա.:

`scamper -I "{{ping}} {{192.0.2.1}}" -I "{{trace}} {{192.0.2.2}}"`

- Ping մի քանի հոսթ UDP-ով, օգտագործեք հատուկ պորտի համար առաջին պինգի համար և ավելացրեք այն յուրաքանչյուր հաջորդ պինգի համար.:

`scamper -c "{{ping}} -P {{UDP-dport}} -d {{33434}}" -i {{192.0.2.1}} -i {{192.0.2.2}}`

- Օգտագործեք Multipath Discovery Algorithm (MDA)՝ որոշելու համար դեպի նպատակակետ բեռի հավասարակշռված ուղիները և օգտագործեք ICMP echo փաթեթներ՝ առավելագույնը երեք փորձով նմուշառելու համար, արդյունքը գրեք `.warts` ֆայլում.:

`scamper -O {{warts}} -o {{path/to/output.warts}} -I "{{tracelb}} -P {{ICMP-echo}} -q {{3}} {{192.0.2.1}}"`

- Կատարեք Փարիզի հետագծում ICMP-ով դեպի նպատակակետ և պահպանեք արդյունքը սեղմված `.warts` ֆայլում.:

`scamper -O {{warts.gz}} -o {{path/to/output.warts}} -I "{{trace}} -P {{icmp-paris}} {{2001:db8:dead:beaf::4}}"`

- Գրանցեք բոլոր ICMP փաթեթները, որոնք հասնում են որոշակի IP հասցե և ունեն հատուկ ICMP ID `.warts` ֆայլում.:

`scamper -O {{warts}} -o {{path/to/output.warts}} -I "sniff -S {{2001:db8:dead:beef::6}} icmp[icmpid] == {{101}}"`
