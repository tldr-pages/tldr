#լպ

> Տպել ֆայլերը:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/lp>:.

- Տպեք հրամանի ելքը լռելյայն տպիչի վրա (տես `lpstat` հրամանը).:

`echo "test" | lp`

- Տպել ֆայլը լռելյայն տպիչի մեջ.:

`lp {{path/to/file}}`

- Տպեք ֆայլ անունով տպիչի վրա (տես `lpstat` հրամանը).:

`lp -d {{printer_name}} {{path/to/file}}`

- Տպեք ֆայլի `n` պատճենները լռելյայն տպիչում.:

`lp -n {{n}} {{path/to/file}}`

- Տպեք միայն որոշակի էջեր լռելյայն տպիչում (տպեք էջերը 1, 3-5 և 16):

`lp -P 1,3-5,16 {{path/to/file}}`

- Շարունակեք տպել աշխատանքը.:

`lp -i {{job_id}} -H resume`
