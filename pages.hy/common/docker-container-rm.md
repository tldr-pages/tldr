# դոկեր կոնտեյներ rm

> Հեռացրեք տարաները:.
> Լրացուցիչ տեղեկություններ. <https://docs.docker.com/reference/cli/docker/container/rm/>:.

- Հեռացրեք տարաները.:

`docker {{[rm|container rm]}} {{container1 container2 ...}}`

- Ստիպեք հեռացնել տարան.:

`docker {{[rm|container rm]}} {{[-f|--force]}} {{container1 container2 ...}}`

- Հեռացրեք տարան և դրա ծավալները.:

`docker {{[rm|container rm]}} {{[-v|--volumes]}} {{container}}`

- Ցուցադրել օգնությունը.:

`docker {{[rm|container rm]}} --help`
