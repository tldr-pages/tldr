# վրկ

> Ստուգեք գործադիրների անվտանգության հատկությունները:.
> Նշում. Ավելի հին տարբերակների համար օգտագործեք դրոշներ (օրինակ՝ `--file` `file`-ի փոխարեն):.
> Լրացուցիչ տեղեկություններ. <https://github.com/slimm609/checksec>:.

- Նշեք գործարկվող երկուական ֆայլի անվտանգության հատկությունները.:

`checksec file {{path/to/binary}}`

- ցուցակագրեք անվտանգության հատկությունները ռեկուրսիվորեն բոլոր գործարկվող ֆայլերի գրացուցակում.:

`checksec dir {{path/to/directory}}`

- Թվարկեք գործընթացի անվտանգության հատկությունները.:

`checksec proc {{pid}}`

- Թվարկեք գործող միջուկի անվտանգության հատկությունները.:

`checksec kernel`
