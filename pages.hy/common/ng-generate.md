# ng առաջացնում

> Ստեղծեք և/կամ փոփոխեք ֆայլեր՝ հիմնվելով սխեմայի վրա:.
> Լրացուցիչ տեղեկություններ. <https://angular.dev/cli/generate>:.

- Ստեղծեք AI կազմաձևման ֆայլեր Angular նախագծի համար.:

`ng {{[g|generate]}} ai-config`

- Ստեղծեք նոր ծառայություն.:

`ng {{[g|generate]}} {{[s|service]}} {{service_name}}`

- Ստեղծեք նոր հավելված աշխատանքային տարածքում.:

`ng {{[g|generate]}} {{[app|application]}} {{app_name}}`

- Ստեղծեք նոր դաս.:

`ng {{[g|generate]}} {{[cl|class]}} {{class_name}}`

- Ստեղծեք նոր բաղադրիչ.:

`ng {{[g|generate]}} {{[c|component]}} {{component_name}}`

- Ստեղծեք նոր հրահանգ.:

`ng {{[g|generate]}} {{[d|directive]}} {{directive_name}}`

- Ստեղծեք նոր ցուցակ.:

`ng {{[g|generate]}} {{[e|enum]}} {{enum_name}}`

- Ստեղծեք նոր խողովակ.:

`ng {{[g|generate]}} {{[p|pipe]}} {{pipe_name}}`
