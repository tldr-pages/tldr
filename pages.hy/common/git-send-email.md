# git ուղարկել-էլ

> Ուղարկեք կարկատների հավաքածու որպես էլ.
> Կարկատները կարող են նշվել որպես ֆայլեր, ուղղություններ կամ վերանայման ցուցակ:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-send-email>:.

- Ինտերակտիվ կերպով ուղարկեք ընթացիկ մասնաճյուղում վերջին պարտավորությունը.:

`git send-email -1`

- Ուղարկեք տրված պարտավորություն.:

`git send-email -1 {{commit}}`

- Ուղարկեք բազմաթիվ (օրինակ՝ 10) պարտավորություններ ընթացիկ մասնաճյուղում.:

`git send-email {{-10}}`

- Ուղարկեք ներածական էլփոստի հաղորդագրություն կարկատների շարքի համար.:

`git send-email -{{number_of_commits}} --compose`

- Վերանայեք և խմբագրեք էլփոստի հաղորդագրությունը յուրաքանչյուր պատչի համար, որը պատրաստվում եք ուղարկել.:

`git send-email -{{number_of_commits}} --annotate`
