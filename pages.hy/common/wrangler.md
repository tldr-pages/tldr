# վիճաբան

> Cloudflare Workers գործիք:.
> Լրացուցիչ տեղեկություններ. <https://developers.cloudflare.com/workers/wrangler/commands/>:.

- Նախաձեռնեք նախագիծը կմախքի կազմաձևով.:

`wrangler init {{project_name}}`

- Նույնականացրեք Cloudflare-ով.:

`wrangler login`

- Սկսեք տեղական զարգացման սերվեր.:

`wrangler dev --host {{hostname}}`

- Հրապարակեք աշխատողի սցենարը.:

`wrangler publish`

- Համախառն տեղեկամատյաններ արտադրության աշխատողից.:

`wrangler tail`
