#ժիրա

> Ինտերֆեյս (երրորդ կողմ) Ժիրայի հետ շփվելու համար:.
> Նշում. Jira API նշանի ձեռքբերումը և այն որպես `$JIRA_API_TOKEN` փոփոխական արտահանելը պարտադիր է:.
> Լրացուցիչ տեղեկություններ. <https://github.com/ankitpokhrel/jira-cli#commands>:.

- Ստեղծեք կազմաձևման ֆայլ (պահանջվում է `jira`-ն օգտագործելուց առաջ):

`jira init`

- Թվարկեք վերջին խնդիրները.:

`jira issue {{[ls|list]}}`

- Թվարկեք չնշանակված խնդիրները բարձր առաջնահերթությամբ.:

`jira issue {{[ls|list]}} {{[-a|--assignee]}} x {{[-y|--priority]}} High`

- Թվարկե՛ք ինձ հանձնարարված ընթացիկ սպրինտի խնդիրները.:

`jira sprint {{[ls|list]}} --current {{[-a|--assignee]}} $(jira me)`

- Ստեղծեք նոր թողարկում ծնողական խնդրի հետ.:

`jira issue create {{[-P|--parent]}} {{parent}}`

- Բացեք խնդիրը դիտարկիչում.:

`jira open {{123}}`
