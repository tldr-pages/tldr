#գոհար

> Ruby ծրագրավորման լեզվի փաթեթների կառավարիչ:.
> Լրացուցիչ տեղեկություններ. <https://guides.rubygems.org/command-reference/>:.

- Որոնեք հեռավոր գոհար(ներ) և ցույց տվեք բոլոր հասանելի տարբերակները.:

`gem search {{regex}} {{[-a|--all]}}`

- Տեղադրեք գոհարի վերջին տարբերակը.:

`gem install {{gem_name}}`

- Տեղադրեք գոհարի հատուկ տարբերակ.:

`gem install {{gem_name}} {{[-v|--version]}} {{1.0.0}}`

- Տեղադրեք գոհարի վերջին համապատասխանող (SemVer) տարբերակը.:

`gem install {{gem_name}} {{[-v|--version]}} '~> {{1.0}}'`

- Թարմացրեք գոհարը.:

`gem update {{gem_name}}`

- Թվարկեք բոլոր տեղական գոհարները.:

`gem list`

- Տեղահանել գոհարը.:

`gem uninstall {{gem_name}}`

- Տեղահանեք գոհարի հատուկ տարբերակը.:

`gem uninstall {{gem_name}} {{[-v|--version]}} {{1.0.0}}`
