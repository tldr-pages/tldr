# դոկեր կոնտեյների վերև

> Ցուցադրել կոնտեյների գործող գործընթացները:.
> Լրացուցիչ տեղեկություններ. <https://docs.docker.com/reference/cli/docker/container/top/>:.

- Ցուցադրել կոնտեյների գործող գործընթացները.:

`docker {{[top|container top]}} {{container}}`

- Ցուցադրել օգնությունը.:

`docker {{[top|container top]}} --help`
