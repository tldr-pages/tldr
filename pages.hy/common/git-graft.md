# git փոխպատվաստում

> Միաձուլումը կատարում է ճյուղից մեկ այլ մասնաճյուղ և ջնջում աղբյուրի ճյուղը:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-graft>:.

- Թիրախային ճյուղում չներկայացված բոլոր պարտավորությունները միաձուլեք սկզբնաղբյուր ճյուղից դեպի թիրախային ճյուղ և ջնջեք սկզբնաղբյուր ճյուղը.:

`git graft {{source_branch}} {{target_branch}}`
