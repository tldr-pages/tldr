# npm խմբագրում

> Ընտրեք կախվածություն ընթացիկ նախագծում և բացեք փաթեթի պանակը կանխադրված խմբագրիչում (`$EDITOR`):.
> Խմբագրելուց հետո փաթեթը վերակառուցվում է՝ հավաքագրված փաթեթներում ցանկացած փոփոխություն վերցնելու համար:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-edit/>:.

- Խմբագրել տեղական տեղադրված փաթեթը լռելյայն խմբագրում.:

`npm edit {{package_name}}`
