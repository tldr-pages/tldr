# թափառաշրջիկ նավահանգիստ

> Ցուցակեք քարտեզագրումները հյուրի և հյուրընկալող նավահանգիստների միջև:.
> Տես նաև՝ `vagrant`:.
> Լրացուցիչ տեղեկություններ. <https://developer.hashicorp.com/vagrant/docs/cli/port>:.

- Թվարկեք ընթացիկ գրացուցակում աշխատող մեքենայի(ների) բոլոր նավահանգիստների քարտեզագրումները.:

`vagrant port`

- Ցուցակեք քարտեզագրումները կոնկրետ մեքենայի համար (եթե `Vagrantfile`-ը բազմաֆունկցիոնալ է).:

`vagrant port {{machine_name}}`

- Ցուցադրել տվյալ հյուրի հատուկ նավահանգստի համար.:

`vagrant port --guest {{port_number}}`

- Ցուցադրել մեքենանընթեռնելի արդյունքը.:

`vagrant port --machine-readable`
