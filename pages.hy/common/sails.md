# առագաստներ

> Իրական ժամանակի ձեռնարկության մակարդակի MVC շրջանակ, որը կառուցված է Node.js-ի վերևում:.
> Լրացուցիչ տեղեկություններ. <https://sailsjs.com/documentation/reference/command-line-interface>:.

- Սկսել առագաստները.:

`sails lift`

- Ստեղծեք նոր Sails նախագիծ.:

`sails new {{projectName}}`

- Ստեղծեք Sails API:

`sails generate {{name}}`

- Ստեղծեք առագաստների վերահսկիչ.:

`sails generate controller {{name}}`

- Ստեղծեք առագաստների մոդել.:

`sails generate model {{name}}`
