# cfv

> Command-line File Verify, գործիք՝ ստուգիչ գումարի ֆայլեր փորձարկելու և ստեղծելու համար:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/cfv>:.

- [T] ստուգիչ գումարի ֆայլերը ընթացիկ գրացուցակում.:

`cfv -T`

- [T]est [v]inbosely, re[n]ame bad, եւ ցուցադրեք [u]nverted ֆայլերը՝ օգտագործելով checksum [f]ile:

`cfv -Tvnu -f {{path/to/file}}`

- [C]ընտրված ֆայլերի համար ստեղծել որոշակի [t] տիպի ստուգիչ գումարի ֆայլ.:

`cfv -C -t "{{sfv|sha256|md5|...}}" {{file1 file2 ...}}`

- [C]ստեղծել առանձին ստուգիչ ֆայլեր յուրաքանչյուր [r] էկուրսիվորեն ընտրված գրացուցակի համար.:

`cfv -Cr`

- [C]ստեղծել մեկ `sha256` տեսակի [f] ֆայլ բոլոր [rr]էկուրսիվորեն ընտրված ֆայլերի համար.:

`cfv -C -f {{sums.sha256}} -rr`

- [C]ստեղծեք `g[z]ipped` `.sfv` ֆայլ [v] մանրամասնորեն՝ հետևելով `sym[l]inks`-ին և օգտագործելով `path/to/directory` որպես հիմք [p]ath:

`cfv -Czvl -p {{path/to/directory}}`
