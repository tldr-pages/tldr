#արագ

> Ստեղծեք, գործարկեք և կառուցեք Swift նախագծեր:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/swiftc>:.

- Սկսեք REPL (ինտերակտիվ կեղև).:

`swift repl`

- Գործարկել ծրագիր.:

`swift {{file.swift}}`

- Սկսեք նոր նախագիծ փաթեթի կառավարչի հետ.:

`swift package init`

- Թարմացնել կախվածությունները.:

`swift package update`

- Կազմեք նախագիծ թողարկման համար.:

`swift build {{[-c|--configuration]}} {{debug|release}}`
