# onefetch

> Ցուցադրել ծրագրի տեղեկատվությունը և կոդերի վիճակագրությունը տեղական Git պահեստի համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/o2sh/onefetch/wiki/command-line-options>:.

- Ցուցադրել Git պահոցի վիճակագրությունը ընթացիկ աշխատանքային գրացուցակում.:

`onefetch`

- Ցուցադրել վիճակագրությունը Git պահոցի համար նշված գրացուցակում.:

`onefetch {{path/to/directory}}`

- Անտեսեք բոտերի կողմից կատարված պարտավորությունները.:

`onefetch --no-bots`

- Անտեսել միաձուլման պարտավորությունները.:

`onefetch --no-merges`

- Մի տպեք լեզվի ASCII արվեստը.:

`onefetch --no-art`

- Ցույց տալ `n` հեղինակներին, լեզուներին կամ ֆայլերի խափանումները (կանխադրված՝ համապատասխանաբար 3, 6 և 3).:

`onefetch --number-of-{{authors|languages|file-churns}} {{n}}`

- Անտեսեք նշված ֆայլերը և գրացուցակները.:

`onefetch {{[-e|--exclude]}} {{path/to/file_or_directory|regex}}`

- Հայտնաբերել լեզուներ միայն նշված կատեգորիաներից (կանխադրված՝ ծրագրավորում և նշում).:

`onefetch {{[-T|--type]}} {{programming|markup|prose|data}}`
