#հաջորդհոսք

> Գործարկել հաշվողական խողովակաշարերը: Հիմնականում օգտագործվում է կենսաինֆորմատիկայի աշխատանքային հոսքերի համար:.
> Լրացուցիչ տեղեկություններ. <https://www.nextflow.io/docs/latest/cli.html>:.

- Գործարկել խողովակաշարը, օգտագործել քեշավորված արդյունքները նախորդ գործարկումներից.:

`nextflow run {{main.nf}} -resume`

- Գործարկեք GitHub-ից հեռավոր աշխատանքային հոսքի հատուկ թողարկում.:

`nextflow run {{user/repo}} -revision {{release_tag}}`

- Գործարկել տվյալ աշխատանքային գրացուցակով միջանկյալ ֆայլերի համար, պահպանել կատարման հաշվետվությունը.:

`nextflow run {{workflow}} -work-dir {{path/to/directory}} -with-report {{report.html}}`

- Ցույց տալ նախորդ գործարկումների մանրամասները ընթացիկ գրացուցակում.:

`nextflow log`

- Հեռացրեք քեշը և միջանկյալ ֆայլերը որոշակի գործարկման համար.:

`nextflow clean -force {{run_name}}`

- Թվարկեք բոլոր ներբեռնված նախագծերը.:

`nextflow list`

- Հեռավոր աշխատանքային հոսքի վերջին տարբերակը քաշեք Bitbucket-ից.:

`nextflow pull {{user/repo}} -hub bitbucket`

- Թարմացրեք Nextflow:

`nextflow self-update`
