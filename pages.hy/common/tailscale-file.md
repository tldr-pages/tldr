# tailscale ֆայլ

> Ուղարկեք ֆայլեր միացված սարքերով Tailscale ցանցում:.
> Այն ներկայումս չի աջակցում ֆայլերի ուղարկումը սարքեր, որոնք պատկանում են այլ օգտատերերին նույնիսկ նույն Tailscale ցանցում:.
> Լրացուցիչ տեղեկություններ. <https://tailscale.com/kb/1106/taildrop>:.

- Ուղարկեք ֆայլ կոնկրետ հանգույց.:

`tailscale file cp {{path/to/file}} {{hostname|ip}}:`

- Պահպանեք ընթացիկ հանգույցին ուղարկված ֆայլերը որոշակի գրացուցակում.:

`tailscale file get {{path/to/directory}}`
