# pnpm

> Արագ, սկավառակի տարածության արդյունավետ փաթեթների կառավարիչ Node.js-ի համար:.
> Կառավարեք Node.js նախագծերը և դրանց մոդուլային կախվածությունները:.
> Լրացուցիչ տեղեկություններ. <https://pnpm.io/pnpm-cli>:.

- Ստեղծեք `package.json` ֆայլ՝:

`pnpm init`

- Ներբեռնեք բոլոր փաթեթները, որոնք նշված են որպես կախվածություն `package.json`-ում:

`pnpm {{[i|install]}}`

- Ներբեռնեք փաթեթի որոշակի տարբերակ և ավելացրեք այն `package.json`-ի կախվածությունների ցանկում՝:

`pnpm add {{module_name}}@{{version}}`

- Ներբեռնեք փաթեթ և ավելացրեք այն մշակողների կախվածությունների ցանկում `package.json`-ում:

`pnpm add {{module_name}} {{[-D|--save-dev]}}`

- Ներբեռնեք փաթեթ և տեղադրեք այն ամբողջ աշխարհում.:

`pnpm add {{module_name}} {{[-g|--global]}}`

- Տեղահանեք փաթեթը և հեռացրեք այն `package.json`-ի կախվածությունների ցանկից՝:

`pnpm {{[rm|remove]}} {{module_name}}`

- Տպեք տեղական տեղադրված մոդուլների ծառը.:

`pnpm {{[ls|list]}}`

- Ցուցակե՛ք բարձր մակարդակի գլոբալ տեղադրված մոդուլները.:

`pnpm {{[ls|list]}} {{[-g|--global]}} --depth {{0}}`
