# gh պիտակ

> Աշխատեք GitHub պիտակների հետ:.
> Լրացուցիչ տեղեկություններ. <https://cli.github.com/manual/gh_label>:.

- Ցուցադրեք պահեստի պիտակները ընթացիկ գրացուցակում.:

`gh label {{[ls|list]}}`

- Դիտեք պահեստի պիտակները ընթացիկ գրացուցակում կանխադրված վեբ բրաուզերում.:

`gh label {{[ls|list]}} {{[-w|--web]}}`

- Ստեղծեք պիտակ հատուկ անունով, նկարագրությամբ և գույնով տասնվեցական ձևաչափով ընթացիկ գրացուցակի պահոցի համար.:

`gh label create {{name}} {{[-d|--description]}} "{{description}}" {{[-c|--color]}} {{color_hex}}`

- Ջնջեք պահեստի պիտակը ընթացիկ գրացուցակում, որը հաստատում է.:

`gh label delete {{name}}`

- Թարմացրեք ընթացիկ գրացուցակում գտնվող պահեստի հատուկ պիտակի անվանումը և նկարագրությունը.:

`gh label edit {{name}} {{[-n|--name]}} {{new_name}} {{[-d|--description]}} "{{description}}"`

- Կլոնավորեք պիտակները կոնկրետ պահոցից ընթացիկ գրացուցակի պահեստում.:

`gh label clone {{owner}}/{{repository}}`

- Ցուցադրել օգնություն ենթահրամանի համար.:

`gh label {{subcommand}} {{[-h|--help]}}`
