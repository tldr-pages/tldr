# խոր

> Տեղադրեք PHP հավելվածներ:.
> Նշում. նույն անունով Go հրամանը `dep` հնացած և արխիվացված է:.
> Լրացուցիչ տեղեկություններ. <https://deployer.org/docs/8.x/cli>:.

- Ինտերակտիվորեն նախաստորագրեք տեղակայողը տեղական ուղու վրա (օգտագործեք շրջանակի ձևանմուշ `--template=template`-ով):

`dep init`

- Տեղադրեք հավելված հեռավոր հոսթին.:

`dep deploy {{hostname}}`

- Վերադարձ դեպի նախորդ աշխատանքային թողարկում.:

`dep rollback`

- Միացեք հեռավոր հոսթին SSH-ի միջոցով.:

`dep ssh {{hostname}}`

- Ցուցակ հրամաններ.:

`dep list`

- Գործարկեք ցանկացած կամայական հրաման հեռավոր հոսթերների վրա.:

`dep run "{{command}}"`

- Ցուցադրել օգնություն հրամանի համար.:

`dep help {{command}}`
