# nvm

> Տեղադրեք, տեղահանեք կամ անցեք Node.js տարբերակների միջև ձկան կեղևի տակ:.
> Աջակցում է տարբերակների համարներին, ինչպիսիք են «12.8» կամ «v16.13.1» և «կայուն», «համակարգ» և այլն:.
> Լրացուցիչ տեղեկություններ. <https://github.com/jorgebucaran/nvm.fish>:.

- Տեղադրեք Node.js-ի կոնկրետ տարբերակը.:

`nvm install {{node_version}}`

- Օգտագործեք Node.js-ի կոնկրետ տարբերակը ընթացիկ վահանակում.:

`nvm use {{node_version}}`

- Սահմանեք Node.js-ի լռելյայն տարբերակը.:

`set nvm_default_version {{node_version}}`

- Թվարկե՛ք Node.js-ի բոլոր հասանելի տարբերակները և ընդգծե՛ք լռելյայնը՝:

`nvm list`

- Տեղահանեք տրված Node.js տարբերակը.:

`nvm uninstall {{node_version}}`
