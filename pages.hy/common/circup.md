# շրջան

> CircuitPython գրադարանի թարմացում:.
> Լրացուցիչ տեղեկություններ. <https://github.com/adafruit/circup#usage>:.

- Ինտերակտիվորեն թարմացրեք մոդուլները սարքի վրա.:

`circup update`

- Տեղադրեք նոր գրադարան.:

`circup install {{library_name}}`

- Որոնել գրադարան.:

`circup show {{partial_name}}`

- Ցուցակե՛ք բոլոր գրադարանները միացված սարքում `requirements.txt` ձևաչափով.:

`circup freeze`

- Պահպանեք բոլոր գրադարանները միացված սարքում `requirements.txt`-ում ընթացիկ գրացուցակում.:

`circup freeze {{[-r|--requirement]}}`
