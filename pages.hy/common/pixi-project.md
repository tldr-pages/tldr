# pixi նախագիծ

> Փոփոխել նախագծի կազմաձևման ֆայլը:.
> Լրացուցիչ տեղեկություններ. <https://pixi.sh/latest/reference/cli/#project>:.

- Կառավարեք նախագծի ալիքները.:

`pixi project channel {{command}}`

- Կառավարեք ծրագրի նկարագրությունը.:

`pixi project description {{command}}`

- Կառավարեք ծրագրի հարթակը.:

`pixi project platform {{command}}`

- Կառավարեք նախագծի տարբերակը.:

`pixi project version {{command}}`

- Կառավարեք ծրագրի միջավայրը.:

`pixi project environment {{command}}`
