# git rename-remote

> Փոխեք հեռակառավարիչը՝ քաշելու և հրելու համար:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-rename-remote>:.

- Փոխեք հոսանքով վերևի հեռակառավարումը սկզբնականի.:

`git rename-remote {{upstream}} {{origin}}`
