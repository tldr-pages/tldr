# pip-ի տեղահանում

> Տեղահանել Python փաթեթները:.
> Լրացուցիչ տեղեկություններ. <https://pip.pypa.io/en/stable/cli/pip_uninstall/>:.

- Ապատեղադրել փաթեթը.:

`pip uninstall {{package}}`

- Տեղահանեք հատուկ ֆայլում թվարկված փաթեթները.:

`pip uninstall {{[-r|--requirement]}} {{path/to/requirements.txt}}`

- Տեղահանեք փաթեթը՝ առանց հաստատում խնդրելու.:

`pip uninstall {{[-y|--yes]}} {{package}}`
