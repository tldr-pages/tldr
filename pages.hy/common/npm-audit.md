# npm աուդիտ

> Սկանավորեք նախագծերի կախվածությունը անվտանգության հայտնի խոցելիության համար:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-audit/>:.

- Սկանավորեք նախագծի կախվածությունները հայտնի խոցելիությունների համար.:

`npm audit`

- Ավտոմատ կերպով շտկել խոցելիությունները նախագծի կախվածության մեջ.:

`npm audit fix`

- Ստիպել ավտոմատ շտկել խոցելիություններ ունեցող կախվածությունները.:

`npm audit fix {{[-f|--force]}}`

- Թարմացրեք կողպեքի ֆայլը առանց փոփոխելու `node_modules` գրացուցակը.:

`npm audit fix --package-lock-only`

- Մոդելավորեք շտկման գործընթացը՝ առանց որևէ փոփոխության.:

`npm audit fix --dry-run`

- Աուդիտի արդյունքները JSON ձևաչափով.:

`npm audit --json`

- Կազմաձևեք աուդիտը այնպես, որ ձախողվի միայն խոցելիության դեպքում՝ նշված խստության աստիճանից բարձր.:

`npm audit --audit-level {{info|low|moderate|high|critical}}`
