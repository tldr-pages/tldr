# pyenv virtualenv

> Ստեղծեք վիրտուալ միջավայրեր՝ հիմնվելով մեկի կողմից տեղադրված Python բաշխումների վրա:.
> Լրացուցիչ տեղեկություններ. <https://github.com/pyenv/pyenv-virtualenv#usage>:.

- Ստեղծեք նոր Python 3.6.6 վիրտուալ միջավայր.:

`pyenv virtualenv 3.6.6 {{virtualenv_name}}`

- Թվարկեք բոլոր գոյություն ունեցող վիրտուալ միջավայրերը.:

`pyenv virtualenvs`

- Ակտիվացրեք վիրտուալ միջավայրը.:

`pyenv activate {{virtualenv_name}}`

- Անջատեք վիրտուալ միջավայրը.:

`pyenv deactivate`
