# dvc commit

> Նախագծում գրանցեք փոփոխություններ DVC-ով հետևված ֆայլերում:.
> Լրացուցիչ տեղեկություններ. <https://doc.dvc.org/command-reference/commit>:.

- Փոփոխություններ կատարեք DVC-ով հետևված բոլոր ֆայլերում և գրացուցակներում.:

`dvc commit`

- Փոփոխություններ կատարեք նշված DVC-ով հետևվող թիրախում.:

`dvc commit {{target}}`

- Recursively կատարել բոլոր DVC-հետագծված ֆայլերը գրացուցակում.:

`dvc commit {{[-R|--recursive]}} {{path/to/directory}}`
