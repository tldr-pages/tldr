#էքսպո

> Ստեղծեք, մշակեք և գործարկեք React Native հավելվածներ:.
> Լրացուցիչ տեղեկություններ. <https://docs.expo.dev/eas/>:.

- Ստեղծեք Android APK կամ AAB:

`expo build:android`

- Կառուցեք iOS IPA.:

`expo build:ios`

- Գործարկեք հավելվածը Expo Go-ում.:

`expo start`

- Տեղադրեք կախվածություն.:

`expo install {{package_name}}`

- Նախակառուցեք բնիկ Android/iOS նախագծեր.:

`expo prebuild`

- Գործարկեք հավելվածը Android-ում.:

`expo run:android`

- Գործարկեք հավելվածը iOS-ում.:

`expo run:ios`
