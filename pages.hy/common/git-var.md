# git var

> Տպել Git տրամաբանական փոփոխականի արժեքը:.
> Տես `git config`-ը, որը նախընտրելի է `git var`-ից:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-var>:.

- Տպեք Git տրամաբանական փոփոխականի արժեքը.:

`git var {{GIT_AUTHOR_IDENT|GIT_COMMITTER_IDENT|GIT_EDITOR|GIT_PAGER}}`

- [l] թվարկեք բոլոր Git տրամաբանական փոփոխականները.:

`git var -l`
