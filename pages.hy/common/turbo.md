# տուրբո

> JavaScript-ի և TypeScript-ի կոդերի բազաների համար բարձր արդյունավետության կառուցման համակարգ:.
> Տես նաև՝ `nx`:.
> Լրացուցիչ տեղեկություններ. <https://turborepo.com/docs/reference>:.

- Մուտք գործեք՝ օգտագործելով լռելյայն վեբ զննարկիչը Vercel հաշիվով.:

`turbo login`

- Կապեք ընթացիկ գրացուցակը Vercel կազմակերպության հետ և միացրեք հեռավոր քեշավորումը.:

`turbo link`

- Կառուցեք ընթացիկ նախագիծը.:

`turbo run build`

- Գործարկել առաջադրանքը առանց միաժամանակյաության.:

`turbo run {{task_name}} --concurrency={{1}}`

- Գործարկեք առաջադրանք՝ անտեսելով պահված արտեֆակտները և բռնի կերպով նորից կատարեք բոլոր առաջադրանքները.:

`turbo run {{task_name}} --force`

- Գործարկել առաջադրանքը զուգահեռ փաթեթներով.:

`turbo run {{task_name}} --parallel --no-cache`

- Անջատեք ընթացիկ գրացուցակը ձեր Vercel կազմակերպությունից և անջատեք Remote Caching-ը.:

`turbo unlink`

- Ստեղծեք կոնկրետ առաջադրանքի կատարման կետային գրաֆիկ (ելքային ֆայլի ձևաչափը կարելի է կառավարել ֆայլի անունով).:

`turbo run {{task_name}} --graph={{path/to/file.html|jpg|json|pdf|png|svg}}`
