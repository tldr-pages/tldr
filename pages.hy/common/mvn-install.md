# mvn տեղադրում

> Տեղադրեք երրորդ կողմի Maven կախվածությունները և կառուցեք նախագիծը:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/mvn>:.

- Կազմեք, փորձարկեք, փաթեթավորեք և տեղադրեք նախագիծը տեղական պահեստում.:

`mvn install`

- Բաց թողնել թեստերը տեղադրման ժամանակ.:

`mvn install {{[-D|--define]}} skipTests`

- Կախվածությունների պարտադիր թարմացում տեղադրելուց առաջ.:

`mvn install {{[-U|--update-snapshots]}}`

- Բաց թողնել թեստի կազմումը և կատարումը.:

`mvn install {{[-D|--define]}} maven.test.skip=true`
