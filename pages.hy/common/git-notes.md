# git նշումներ

> Ավելացնել կամ ստուգել օբյեկտի նշումները:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-notes>:.

- Թվարկեք բոլոր նշումները և այն առարկաները, որոնց կցված են.:

`git notes list`

- Թվարկե՛ք տվյալ օբյեկտին կցված բոլոր նշումները (կանխադրված՝ `HEAD`):

`git notes list {{object}}`

- Ցույց տվեք տվյալ օբյեկտին կցված նշումները (կանխադրված՝ `HEAD`):

`git notes show {{object}}`

- Նշում նշեք նշված օբյեկտին (բացում է լռելյայն տեքստային խմբագրիչը).:

`git notes append {{object}}`

- Նշված օբյեկտին ավելացրեք նշում՝ նշելով հաղորդագրությունը.:

`git notes append --message="{{message_text}}"`

- Խմբագրել գոյություն ունեցող նշումը (կանխադրված է `HEAD`):

`git notes edit {{object}}`

- Պատճենել նշումը մի օբյեկտից մյուսը.:

`git notes copy {{source_object}} {{target_object}}`

- Հեռացրեք նշված օբյեկտին ավելացված բոլոր նշումները.:

`git notes remove {{object}}`
