# aws օգնություն

> Ցուցադրել օգնության տեղեկատվություն AWS CLI-ի մասին:.
> Լրացուցիչ տեղեկություններ. <https://docs.aws.amazon.com/cli/latest/userguide/cli-usage-help.html>:.

- Ցուցադրել օգնությունը.:

`aws help`

- Թվարկեք բոլոր առկա թեմաները.:

`aws help topics`

- Ցուցադրել օգնություն կոնկրետ թեմայի վերաբերյալ.:

`aws help {{topic_name}}`
