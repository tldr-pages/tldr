#git

> Բաշխված տարբերակի կառավարման համակարգ:.
> Որոշ ենթահրամաններ, ինչպիսիք են `commit`, `add`, `branch`, `switch`, `push` և այլն, ունեն իրենց օգտագործման փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git>:.

- Ստեղծեք դատարկ Git պահոց.:

`git init`

- Կլոնավորեք Git-ի հեռավոր պահոցը ինտերնետից.:

`git clone {{https://example.com/repo.git}}`

- Դիտեք տեղական պահեստի կարգավիճակը.:

`git status`

- Բեմադրեք բոլոր փոփոխությունները պարտավորության համար.:

`git add {{[-A|--all]}}`

- Փոփոխություններ կատարեք տարբերակի պատմության մեջ.:

`git commit {{[-m|--message]}} {{message_text}}`

- Տեղական պարտավորությունները մղեք հեռավոր պահոց.:

`git push`

- Քաշեք ցանկացած փոփոխություն, որը կատարվել է հեռակառավարման վահանակում.:

`git pull`

- Վերականգնել ամեն ինչ այնպես, ինչպես եղել է վերջին commit-ում.:

`git reset --hard; git clean {{[-f|--force]}}`
