#lolcat

> Տեղադրեք ծիածան այն ամենի մեջ, ինչ դուք `cat` սարքում եք:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/lolcat>:.

- Տպեք ֆայլը վահանակի վրա ծիածանի գույներով.:

`lolcat {{path/to/file}}`

- Տպեք տեքստ արտադրող հրամանի արդյունքը ծիածանի գույներով.:

`{{fortune}} | lolcat`

- Օգտագործեք սերմ՝ հետևողական գույներ ստեղծելու համար (կանխադրված՝ `0` ինչպես պատահական):

`lolcat {{[-S|--seed]}} {{number}} {{path/to/file}}`

- Կառավարեք ծիածանի հաճախականությունը (կանխադրված՝ `0.1`):

`lolcat {{[-F|--frequency]}} {{number}} {{path/to/file}}`

- Կառավարեք ծիածանի հարթեցումը (կանխադրված՝ `3`):

`lolcat {{[-p|--spread]}} {{number}} {{path/to/file}}`

- Տպեք ֆայլ մխիթարիչում անիմացիոն ծիածանի գույներով.:

`lolcat {{[-a|--animate]}} {{path/to/file}}`

- Տպեք ֆայլ կոնսոլում 24-բիթանոց (truecolor) ծիածանի գույներով.:

`lolcat {{[-t|--truecolor]}} {{path/to/file}}`
