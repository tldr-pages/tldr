# Vue build

> `@vue/cli`-ի և `@vue/cli-service-global`-ի կողմից տրամադրված ենթահրաման, որը հնարավորություն է տալիս արագ նախատիպեր պատրաստել:.
> Լրացուցիչ տեղեկություններ. <https://cli.vuejs.org/guide/cli-service.html#vue-cli-service-serve>:.

- Ստեղծեք `.js` կամ `.vue` ֆայլ արտադրական ռեժիմում՝ զրոյական կազմաձևով.:

`vue build {{filename}}`
