# npm կազմաձևում

> Կառավարեք `npm` կազմաձևման կարգավորումները:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-config/>:.

- Ցուցադրել բոլոր կազմաձևման կարգավորումները.:

`npm {{[c|config]}} list`

- Նշեք բոլոր կազմաձևման կարգավորումները որպես `JSON`:

`npm {{[c|config]}} list --json`

- Ստացեք որոշակի կազմաձևման բանալի արժեքը.:

`npm {{[c|config]}} get {{key}}`

- Սահմանեք կոնֆիգուրացիայի բանալին որոշակի արժեքի.:

`npm {{[c|config]}} set {{key}} {{value}}`

- Ջնջել կազմաձևման բանալին.:

`npm {{[c|config]}} delete {{key}}`

- Խմբագրեք npm կազմաձևման ֆայլը լռելյայն խմբագրում (օգտագործեք `--global` դրոշը՝ գլոբալ կազմաձևը խմբագրելու համար).:

`npm {{[c|config]}} edit`

- Անվավեր կազմաձևման տարրերը վերանորոգելու փորձ.:

`npm {{[c|config]}} fix`
