# gpg-tui

> Տերմինալի ինտերֆեյս GNU Privacy Guard-ի համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/orhun/gpg-tui>:.

- Սկսել `gpg-tui`:

`gpg-tui`

- Սկսեք `gpg-tui`-ը գունավոր և ASCII զրահապատ ելքով.:

`gpg-tui {{[-s|--style]}} {{colored}} {{[-a|--armor]}}`

- Դուրս գալ `gpg-tui`:

`<q>`

- Ինտերակտիվ կերպով ստեղծեք նոր բանալի.:

`<g>`

- Արտահանել ընտրված բանալին.:

`<x>`

- Սահմանեք մանրամասների մակարդակը ընտրված բանալու համար.:

`<1>|<2>|<3>`

- Թարմացնել `gpg-tui`:

`<r>`

- Ցուցադրել օգնությունը `gpg-tui`-ում՝:

`<?>`
