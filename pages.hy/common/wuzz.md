# wuzz

> Ինտերակտիվ կերպով ստուգեք HTTP հարցումներն ու պատասխանները:.
> Լրացուցիչ տեղեկություններ. <https://github.com/asciimoo/wuzz#commands>:.

- Սկսել `wuzz`:

`wuzz`

- Ուղարկեք HTTP հարցում.:

`<Ctrl r>`

- Անցեք հաջորդ տեսքին.:

`<Ctrl j><Tab>`

- Անցեք նախորդ տեսքին.:

`<Ctrl k><Shift Tab>`

- Ցուցադրել օգնությունը.:

`<F1>`
