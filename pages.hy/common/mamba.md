#մամբա

> Արագ, միջպլատֆորմային փաթեթների կառավարիչ, որը նախատեսված է որպես կոնդերի փոխարինող:.
> Որոշ ենթահրամաններ, ինչպիսիք են `repoquery`-ն, ունեն իրենց օգտագործման փաստաթղթերը:.
> Տես նաև՝ `conda`:.
> Լրացուցիչ տեղեկություններ. <https://mamba.readthedocs.io/en/latest/user_guide/mamba.html>:.

- Ստեղծեք նոր միջավայր՝ դրանում տեղադրելով նշված փաթեթները.:

`mamba create {{[-n|--name]}} {{environment_name}} {{python=3.10 matplotlib}}`

- Տեղադրեք փաթեթներ ընթացիկ միջավայրում՝ նշելով փաթեթի ալիքը.:

`mamba install {{[-c|--channel]}} {{conda-forge}} {{python=3.6 numpy}}`

- Թարմացրեք բոլոր փաթեթները ընթացիկ միջավայրում.:

`mamba update {{[-a|--all]}}`

- Որոնեք հատուկ փաթեթ պահեստարաններում.:

`mamba repoquery search {{numpy}}`

- Թվարկեք բոլոր միջավայրերը.:

`mamba info {{[-e|--envs]}}`

- Հեռացրեք չօգտագործված փաթեթներն ու տաբատները քեշից.:

`mamba clean {{[-pt|--packages --tarballs]}}`

- Ակտիվացրեք միջավայրը.:

`mamba activate {{environment_name}}`

- Թվարկեք բոլոր տեղադրված փաթեթները ներկայումս ակտիվացված միջավայրում.:

`mamba list`
