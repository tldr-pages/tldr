# git standup

> Տե՛ս նշված օգտվողի պարտավորությունները:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/git-standup>:.

- Ցույց տալ տվյալ հեղինակի պարտավորությունները վերջին 10 օրվա ընթացքում.:

`git standup -a {{name|email}} -d {{10}}`

- Ցույց տալ տվյալ հեղինակի պարտավորությունները վերջին 10 օրվա ընթացքում և արդյոք դրանք ստորագրված են GPG:

`git standup -a {{name|email}} -d {{10}} -g`

- Ցույց տալ բոլոր պարտավորությունները բոլոր ներդրողների կողմից վերջին 10 օրվա ընթացքում.:

`git standup -a all -d {{10}}`

- Ցուցադրել օգնությունը.:

`git standup -h`
