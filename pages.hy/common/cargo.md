#բեռ

> Կառավարեք Rust նախագծերը և դրանց մոդուլային կախվածությունները (վանդակներ):.
> Որոշ ենթահրամաններ, ինչպիսիք են `build`-ն, ունեն իրենց օգտագործման փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://doc.rust-lang.org/stable/cargo/>:.

- Որոնեք տուփեր.:

`cargo search {{search_string}}`

- Տեղադրեք երկուական վանդակ.:

`cargo install {{crate_name}}`

- Տեղադրված երկուական արկղերի ցուցակ.:

`cargo install --list`

- Ստեղծեք նոր երկուական կամ գրադարանային Rust նախագիծ նշված գրացուցակում (կամ ընթացիկ աշխատանքային գրացուցակը ըստ լռելյայն).:

`cargo init --{{bin|lib}} {{path/to/directory}}`

- Ավելացրեք կախվածություն `Cargo.toml`-ին ընթացիկ գրացուցակում.:

`cargo add {{dependency}}`

- Կառուցեք Rust նախագիծը ընթացիկ գրացուցակում, օգտագործելով թողարկման պրոֆիլը.:

`cargo {{[b|build]}} {{[-r|--release]}}`

- Գործարկեք կառուցված երկուականը (կկառուցվի այն, եթե այն արդեն կառուցված չէ):

`cargo {{[r|run]}}`

- Կառուցեք Rust նախագիծը ընթացիկ գրացուցակում, օգտագործելով գիշերային կոմպիլյատորը (պահանջում է `rustup`):

`cargo +nightly {{[b|build]}}`
