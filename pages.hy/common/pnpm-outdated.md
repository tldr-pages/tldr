# pnpm հնացած է

> Ստուգեք հնացած փաթեթների առկայությունը:.
> Ստուգումը կարող է սահմանափակվել տեղադրված փաթեթների ենթաբազմությամբ՝ տրամադրելով փաստարկներ (նախշերն ապահովված են):.
> Լրացուցիչ տեղեկություններ. <https://pnpm.io/cli/outdated>:.

- Ստուգեք հնացած փաթեթների համար.:

`pnpm outdated`

- Ստուգեք յուրաքանչյուր աշխատանքային տարածքի փաթեթում հայտնաբերված հնացած կախվածությունները.:

`pnpm outdated {{[-r|--recursive]}}`

- Զտեք հնացած փաթեթները՝ օգտագործելով փաթեթի ընտրիչ.:

`pnpm outdated --filter {{package_selector}}`

- Թվարկեք հնացած փաթեթները ամբողջ աշխարհում.:

`pnpm outdated {{[-g|--global]}}`

- Տպել հնացած փաթեթների մանրամասները.:

`pnpm outdated --long`

- Տպել հնացած կախվածությունները հատուկ ձևաչափով.:

`pnpm outdated --format {{format}}`

- Տպել միայն այն տարբերակները, որոնք բավարարում են `package.json`-ի բնութագրերը.:

`pnpm outdated --compatible`

- Ստուգեք միայն հնացած մշակողի կախվածությունները.:

`pnpm outdated {{[-D|--dev]}}`
