# hexedit

> Ֆայլերի խմբագիր տասնվեցական և ASCII բովանդակության համար:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/hexedit>:.

- Բացել ֆայլը.:

`hexedit {{path/to/file}}`

- Ցուցադրել ֆայլը հատվածներով.:

`hexedit {{[-s|--sector]}} {{path/to/file}}`

- Առավելագույնի հասցնել ֆայլի ցուցադրումը.:

`hexedit {{[-m|--maximize]}} {{path/to/file}}`

- Ցուցադրել օգնությունը.:

`hexedit {{[-h|--help]}}`
