# xml pyx

> Փոխակերպեք XML փաստաթուղթը PYX (ESIS - ISO 8879) ձևաչափի:.
> Լրացուցիչ տեղեկություններ. <https://xmlstar.sourceforge.net/doc/UG/xmlstarlet-ug.html#idm47077139550832>:.

- Վերափոխեք XML փաստաթուղթը PYX ձևաչափի.:

`xml pyx {{path/to/input.xml|uri}} > {{path/to/output.pyx}}`

- Փոխակերպեք XML փաստաթուղթը `stdin`-ից PYX ձևաչափի.:

`cat {{path/to/input.xml}} | xml pyx > {{path/to/output.pyx}}`

- Ցուցադրել օգնությունը.:

`xml pyx --help`
