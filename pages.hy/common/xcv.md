# xcv

> Կտրեք, պատճենեք և տեղադրեք հրամանի տողում:.
> Լրացուցիչ տեղեկություններ. <https://github.com/busterc/xcv#xcv>:.

- Կտրեք ֆայլը.:

`xcv x {{input_file}}`

- Պատճենել ֆայլը.:

`xcv c {{input_file}}`

- Տեղադրեք ֆայլ.:

`xcv v {{output_file}}`

- Ցուցակեք տեղադրման համար հասանելի ֆայլերը.:

`xcv l`
