# պոեզիա ավելացնել

> Պոեզիայի `pyproject.toml` ֆայլին ավելացրեք պահանջվող փաթեթները:.
> Տես նաև՝ `asdf`:.
> Լրացուցիչ տեղեկություններ. <https://python-poetry.org/docs/cli/#add>:.

- Ավելացնել պահանջվող փաթեթներ.:

`poetry add {{package_name}}`

- Պահանջվող փաթեթներ ավելացրեք կախվածությունների որոշակի խմբին.:

`poetry add {{package_name}} --group {{group_name}}`

- Ավելացնել փաթեթի որոշակի տարբերակ.:

`poetry add {{package_name}}=={{version}}`

- Ավելացնել փաթեթի որոշակի տարբերակ, որը հավասար է կամ ավելի վաղ, քան տվյալ տարբերակը.:

`poetry add {{package_name}}<={{version}}`

- Ավելացնել փաթեթի որոշակի տարբերակ, որը հավասար է կամ ավելի ուշ, քան տվյալ տարբերակը.:

`poetry add {{package_name}}>={{version}}`
