# openstack օգնություն

> Ցուցադրել օգնության տեղեկատվություն openstackclient-ի մասին:.
> Լրացուցիչ տեղեկություններ. <https://docs.openstack.org/python-openstackclient/latest/cli/man/openstack.html>:.

- Ցուցադրել որոշակի հրամանի նկարագրությունը.:

`openstack help {{command_name}}`

- Ստացեք օգնություն Identity v3-ի համար.:

`openstack --os-identity-api-version 3 --help`

- Ցուցադրել օգնությունը.:

`openstack --help`
