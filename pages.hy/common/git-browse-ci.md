# git թերթել-ci

> Բացեք ընթացիկ Git պահոցի CI կայքը կանխադրված վեբ բրաուզերում:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-browse-ci>:.

- Բացեք ընթացիկ պահոցի CI կոնֆիգուրացիան նրա վերին հոսանքով գտնվող կայքում.:

`git browse-ci`

- Բացեք ընթացիկ պահոցի CI կոնֆիգուրացիան իր վերին վեբ կայքում որոշակի հեռակառավարման համար.:

`git browse-ci {{remote}}`
