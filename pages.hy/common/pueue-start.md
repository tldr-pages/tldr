# pueue start

> Վերսկսել առաջադրանքների կամ առաջադրանքների խմբերի աշխատանքը:.
> Տես նաև՝ `pueue pause`:.
> Լրացուցիչ տեղեկություններ. <https://github.com/Nukesor/pueue#how-to-use-it>:.

- Վերսկսեք բոլոր առաջադրանքները լռելյայն խմբում.:

`pueue start`

- Վերսկսել կոնկրետ առաջադրանքը.:

`pueue start {{task_id}}`

- Միանգամից վերսկսեք բազմաթիվ առաջադրանքներ.:

`pueue start {{task_id}} {{task_id}}`

- Վերսկսեք բոլոր առաջադրանքները և սկսեք իրենց երեխաներին.:

`pueue start {{[-a|--all]}} --children`

- Վերսկսեք բոլոր առաջադրանքները որոշակի խմբի մեջ.:

`pueue start group {{group_name}}`
