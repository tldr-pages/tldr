# git repack

> Փաթեթավորեք չփաթեթավորված օբյեկտները Git պահոցում:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-repack>:.

- Փաթեթավորեք չփաթեթավորված օբյեկտները ընթացիկ գրացուցակում.:

`git repack`

- Հեռացրեք ավելորդ առարկաները փաթեթավորումից հետո.:

`git repack -d`

- Վերալիցքավորեք բոլոր առարկաները մեկ փաթեթի մեջ.:

`git repack -a`

- Վերափաթեթը սահմանափակեք միայն տեղական օբյեկտներով.:

`git repack -l`
