#ծնձան

> Կազմել և գործարկել Elm աղբյուրի ֆայլերը:.
> Լրացուցիչ տեղեկություններ. <https://guide.elm-lang.org/install/elm.html>:.

- Նախաձեռնեք Elm նախագիծը, ստեղծեք `elm.json` ֆայլ.:

`elm init`

- Սկսեք ինտերակտիվ Elm shell-ը.:

`elm repl`

- Կազմեք Elm ֆայլ, ստացեք արդյունքը `index.html` ֆայլում.:

`elm make {{source}}`

- Կազմեք Elm ֆայլ, արդյունքը դուրս բերեք JavaScript ֆայլ.:

`elm make {{source}} --output={{destination}}.js`

- Սկսեք տեղական վեբ սերվերը, որը կազմում է Elm ֆայլերը էջի բեռնման ժամանակ.:

`elm reactor`

- Տեղադրեք Elm փաթեթը <https://package.elm-lang.org>-ից.:

`elm install {{author}}/{{package}}`
