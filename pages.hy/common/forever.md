# ընդմիշտ

> Սերվերի կողմից JavaScript հավելված, որը վստահեցնում է, որ Node.js հավելվածները անժամկետ են աշխատում (վերագործարկում է ելքից հետո):.
> Լրացուցիչ տեղեկություններ. <https://github.com/foreversd/forever#usage>:.

- Սկսեք գործարկել ֆայլը ընդմիշտ (որպես դևոն).:

`forever {{script}}`

- «Հավերժ» գործող գործընթացների ցանկ (ID-ների և «հավերժ» գործընթացների այլ մանրամասների հետ միասին).:

`forever list`

- Դադարեցրեք գործող «ընդմիշտ» գործընթացը.:

`forever stop {{id|pid|script}}`
