# npm թարմացում

> Թարմացրեք փաթեթները ընթացիկ նախագծում:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-update/>:.

- Թարմացրեք բոլոր փաթեթները ընթացիկ նախագծում.:

`npm {{[up|update]}}`

- Թարմացրեք կոնկրետ փաթեթ ընթացիկ նախագծում.:

`npm {{[up|update]}} {{package}}`

- Թարմացրեք փաթեթը ամբողջ աշխարհում.:

`npm {{[up|update]}} {{package}} {{[-g|--global]}}`

- Թարմացրեք մի քանի փաթեթներ միանգամից.:

`npm {{[up|update]}} {{package1 package2 ...}}`
