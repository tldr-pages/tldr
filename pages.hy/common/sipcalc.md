# sipcalc

> Հաշվարկել IP ենթացանցային տեղեկատվությունը IPv4 և IPv6 հասցեների համար:.
> Տես նաև՝ `ipcalc`:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/sipcalc>:.

- Ցուցադրել ենթացանցային տեղեկատվությունը IPv4 CIDR հասցեի համար.:

`sipcalc {{192.168.1.0/24}}`

- Ցուցադրել բոլոր հասանելի տեղեկությունները հասցեի համար.:

`sipcalc {{[-a|--all]}} {{192.168.1.0/24}}`

- Բաժանեք IPv4 ցանցը նշված դիմակի ենթացանցերի մեջ.:

`sipcalc {{[-s|--v4split]}} {{255.255.255.128}} {{192.168.1.0/24}}`

- Ցուցադրել IPv6 հակադարձ DNS տեղեկատվությունը.:

`sipcalc {{[-r|--v6rev]}} {{2001:db8::/32}}`

- Միացնել DNS անվան լուծումը.:

`sipcalc {{[-d|--resolve]}} {{192.168.1.0/24}}`

- Ցուցադրել օգնությունը.:

`sipcalc {{[-h|--help]}}`
