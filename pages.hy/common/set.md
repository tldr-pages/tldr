# հավաքածու

> Փոխարկել վահանակի ընտրանքները կամ սահմանել դիրքային պարամետրերի արժեքները:.
> Լրացուցիչ տեղեկություններ. <https://www.gnu.org/software/bash/manual/bash.html#The-Set-Builtin>:.

- Ցուցադրել կեղևի փոփոխականների անուններն ու արժեքները.:

`set`

- Արտահանել նոր նախաստորագրված փոփոխականները երեխայի գործընթացներին.:

`set -a`

- Գրեք ձևաչափված հաղորդագրություններ `stderr`-ին, երբ աշխատանքները ավարտվեն.:

`set -b`

- Գրեք և խմբագրեք տեքստը հրամանի տողում՝ `vi`-ի նման ստեղնաշարերով (օրինակ՝ `yy`):

`set -o vi`

- Վերադառնալ լռելյայն (`emacs`) ռեժիմին՝:

`set -o emacs`

- Նշեք բոլոր ռեժիմները.:

`set -o`

- Դուրս եկեք կեղևից, երբ (որոշ) հրամանը ձախողվի.:

`set -e`

- Վերականգնել կեղևի բոլոր պարամետրերը և նշանակել նորերը.:

`set -- {{argument1 argument2 ...}}`
