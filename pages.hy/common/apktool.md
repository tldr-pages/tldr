# apktool

> Հակադարձ ինժեների APK ֆայլեր:.
> Լրացուցիչ տեղեկություններ. <https://ibotpeaches.github.io/Apktool/>:.

- Ապակոդավորել APK ֆայլը.:

`apktool d {{path/to/file}}.apk`

- Ստեղծեք APK ֆայլ գրացուցակից.:

`apktool b {{path/to/directory}}`

- Տեղադրեք և պահեք շրջանակը.:

`apktool if {{path/to/framework}}.apk`
