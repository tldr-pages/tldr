# hexdump

> Ցուցադրել ֆայլի բովանդակությունը տասնվեցական, տասնորդական, օկտալ կամ ASCII:.
> Օգտակար է աղբավայրի ֆայլի, երկուական տվյալների կամ վրիպազերծման ելքի ստուգման համար:.
> Տես նաև՝ `hexyl`, `od`, `xxd`:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/man/freebsd/hexdump.1>:.

- Տպեք ֆայլի տասնվեցական պատկերը՝ փոխարինելով կրկնօրինակ տողերը `*`-ով:

`hexdump {{path/to/file}}`

- Ցուցադրել մուտքային օֆսեթը տասնվեցականում և դրա ASCII ներկայացումը երկու սյունակում.:

`hexdump -C {{path/to/file}}`

- Ցուցադրել ֆայլի տասնվեցական պատկերը, բայց մեկնաբանել մուտքի միայն որոշակի քանակությամբ բայթ.:

`hexdump -C -n {{number_of_bytes}} {{path/to/file}}`

- Բազմակողմանի - կրկնվող տողերի վրա `*`-ի կողմից ոչ մի ճնշում.:

`hexdump -v {{path/to/file}}`

- Ձևաչափեք ելքը՝ օգտագործելով printf-ի նման ձևաչափի տողը.:

`hexdump -e '{{element_format .. end_format}}' {{path/to/file}}`
