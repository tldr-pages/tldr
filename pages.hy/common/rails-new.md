# ռելսեր նոր

> Ստեղծեք նոր Rails հավելված:.
> Լրացուցիչ տեղեկություններ. <https://guides.rubyonrails.org/command_line.html#rails-new>:.

- Ստեղծեք Rails հավելված՝ `blog` անունով ընթացիկ գրացուցակում.:

`rails new blog`

- Ստեղծեք Rails հավելված միայն API-ի կոնֆիգուրացիայով.:

`rails new {{app_name}} --api`

- Ստեղծեք Rails հավելված՝ `postgresql` որպես տվյալների բազա.:

`rails new {{app_name}} {{[-d|--database]}} postgresql`

- Ստեղծեք Rails հավելված՝ առանց JavaScript ֆայլեր ստեղծելու.:

`rails new {{app_name}} {{[-J|--skip-javascript]}}`

- Ցուցադրել օգնությունը.:

`rails new {{[-h|--help]}}`
