# npm տեղադրում-փորձարկում

> Համարժեք է `npm install` գործարկմանը, որին հաջորդում է `npm test`:.
> Նշում. `it`-ը կարող է օգտագործվել որպես `install-test`-ի սղագրություն:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-install-test/>:.

- Տեղադրեք բոլոր կախվածությունները և այնուհետև գործարկեք թեստերը.:

`npm {{[it|install-test]}}`

- Տեղադրեք որոշակի փաթեթ և այնուհետև կատարեք թեստեր.:

`npm {{[it|install-test]}} {{package_name}}`

- Տեղադրեք փաթեթ և պահպանեք այն որպես կախվածություն՝ նախքան թեստերը կատարելը.:

`npm {{[it|install-test]}} {{package_name}} {{[-S|--save]}}`

- Տեղադրեք կախվածությունները գլոբալ և այնուհետև կատարեք թեստեր.:

`npm {{[it|install-test]}} {{[-g|--global]}}`
