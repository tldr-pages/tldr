# npm հղում

> Համակցեք տեղական փաթեթը գլոբալ `node_modules`-ին կամ զարգացման այլ նախագծին:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-link/>:.

- Համակցեք ընթացիկ փաթեթը գլոբալ.:

`npm {{[ln|link]}}`

- Կցեք գլոբալ կապակցված փաթեթը մեկ այլ նախագծի `node_modules`-ին.:

`npm {{[ln|link]}} {{package_name}}`

- Անջատեք փաթեթը ընթացիկ նախագծից.:

`npm {{[r|unlink]}} {{package_name}}`
