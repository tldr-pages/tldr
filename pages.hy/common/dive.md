# սուզվել

> Բացահայտեք Docker պատկերը, շերտի բովանդակությունը և հայտնաբերեք այն փոքրացնելու ուղիներ:.
> Լրացուցիչ տեղեկություններ. <https://github.com/wagoodman/dive>:.

- Վերլուծեք Docker պատկերը.:

`dive {{your_image_tag}}`

- Կառուցեք պատկեր և սկսեք վերլուծել այն.:

`dive build -t {{some_tag}}`
