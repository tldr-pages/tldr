# pueue ուղարկել

> Ուղարկեք տվյալ առաջադրանքը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/Nukesor/pueue#how-to-use-it>:.

- Ուղարկեք մուտքագրում գործող հրամանին.:

`pueue send {{task_id}} "{{input}}"`

- Ուղարկեք հաստատում y/N սպասող առաջադրանքին (օրինակ՝ APT, cp):

`pueue send {{task_id}} {{y}}`
