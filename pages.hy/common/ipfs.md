# ipfs

> Միջմոլորակային ֆայլային համակարգ:.
> Հիպերմեդիա պրոտոկոլ հավասարազորից: Նպատակ ունի համացանցն ավելի բաց դարձնել:.
> Լրացուցիչ տեղեկություններ. <https://docs.ipfs.tech/reference/kubo/cli/>:.

- Ավելացնել ֆայլ տեղականից ֆայլային համակարգին, ամրացնել այն և տպել հարաբերական հեշը.:

`ipfs add {{path/to/file}}`

- Ավելացնել գրացուցակ և դրա ֆայլերը ռեկուրսիվորեն տեղականից ֆայլային համակարգ և տպել հարաբերական հեշը.:

`ipfs add {{[-r|--recursive]}} {{path/to/directory}}`

- Պահպանեք հեռավոր ֆայլը և անուն տվեք, բայց մի ամրացրեք այն.:

`ipfs get {{hash}} {{[-o|--output]}} {{path/to/file}}`

- Տեղայնորեն ամրացրեք հեռավոր ֆայլը.:

`ipfs pin add {{hash}}`

- Ցուցադրել ամրացված ֆայլերը.:

`ipfs pin ls`

- Անջատեք ֆայլը տեղական պահեստից.:

`ipfs pin rm {{hash}}`

- Հեռացրեք չամրացված ֆայլերը տեղական պահեստից.:

`ipfs repo gc`
