# բաց կոդի վրիպազերծում

> Վրիպազերծման և անսարքությունների վերացման գործիքներ OpenCode-ի համար:.
> Լրացուցիչ տեղեկություններ. <https://opencode.ai/docs/cli#debug>:.

- Ցույց տալ լուծված կոնֆիգուրացիան.:

`opencode debug config`

- Ցույց տալ գլոբալ ուղիները (տվյալներ, կազմաձև, քեշ, վիճակ).:

`opencode debug paths`

- Թվարկեք բոլոր հայտնի նախագծերը.:

`opencode debug scrap`

- Թվարկեք բոլոր մատչելի հմտությունները.:

`opencode debug skill`

- Ցույց տալ գործակալի կազմաձևման մանրամասները.:

`opencode debug agent {{agent_name}}`

- Ցուցադրել օգնությունը.:

`opencode debug {{[-h|--help]}}`
