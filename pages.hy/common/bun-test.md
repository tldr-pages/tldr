# bun test

> Կատարեք թեստեր՝ օգտագործելով Bun-ի ներկառուցված թեստային վազորդը:.
> Դա արագ, Jest-ի հետ համատեղելի թեստային վազորդ է, որը փնտրում է `*.test.ts` (և նմանատիպ) ֆայլեր:.
> Լրացուցիչ տեղեկություններ. <https://bun.com/docs/test>:.

- Գործարկեք նախագծում հայտնաբերված բոլոր թեստային ֆայլերը.:

`bun test`

- Գործարկեք հատուկ թեստային ֆայլ կամ գրացուցակ.:

`bun test {{path/to/file.test.ts}}`

- Գործարկել թեստերը «դիտելու» ռեժիմում (վերագործարկվում է ավտոմատ կերպով ֆայլի փոփոխության դեպքում).:

`bun test --watch`

- Գործարկեք թեստեր և ստեղծեք ծածկագրի հաշվետվություն.:

`bun test --coverage`
