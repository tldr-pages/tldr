#մեղադրել

> Ցույց տալ, թե ինչ commit-ը և հեղինակը վերջին անգամ փոփոխել են ֆայլի յուրաքանչյուր տող:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-blame>:.

- Տպեք ֆայլ հեղինակային տեղեկություններով (հեղինակի անունը և կատարեք հեշ).:

`git blame {{path/to/file}}`

- Տպել հեղինակի էլ. նամակը նրա անվան փոխարեն.:

`git blame {{[-e|--show-email]}} {{path/to/file}}`

- Տպել ֆայլ հեղինակային տեղեկություններով, որպես կոնկրետ պարտավորություն.:

`git blame {{commit}} {{path/to/file}}`

- Հեղինակային տեղեկություններով ֆայլ տպեք նախքան կոնկրետ կատարումը.:

`git blame {{commit}}~ {{path/to/file}}`

- Տպել ֆայլ հեղինակային տեղեկություններով՝ սկսած տվյալ տողից.:

`git blame -L {{123}} {{path/to/file}}`

- Նշեք ֆայլի որոշակի տողերի տիրույթ.:

`git blame -L {{start_line}},{{end_line}} {{path/to/file}}`

- Նշեք ֆայլի 10 տող, սկսած առաջին տողից, որը համապատասխանում է տվյալ տեքստին.:

`git blame -L '/{{text}}/',+10 {{path/to/file}}`

- Նշեք ֆայլը, որը անտեսում է բացատները և տողերի շարժումները.:

`git blame -w -C -C -C {{path/to/file}}`
