# git update-server-info

> Թարմացրեք օժանդակ պահեստի տեղեկատվությունը համր սերվերների համար (սովորական վեբ սերվեր, որը սպասարկում է միայն ֆայլեր):.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-update-server-info>:.

- Թարմացրեք Git-ի օժանդակ ֆայլերը համր HTTP սպասարկման համար.:

`git update-server-info`

- Վերակառուցեք օժանդակ տեղեկատվական ֆայլերը զրոյից.:

`git update-server-info {{[-f|--force]}}`
