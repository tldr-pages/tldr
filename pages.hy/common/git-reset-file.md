# git reset-file

> Վերադարձեք ֆայլը `HEAD` կամ commit:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-reset-file>:.

- Վերականգնել ֆայլը `HEAD`:

`git reset-file {{path/to/file}}`

- Վերականգնել ֆայլը կոնկրետ commit-ի.:

`git reset-file {{path/to/file}} {{commit_hash}}`
