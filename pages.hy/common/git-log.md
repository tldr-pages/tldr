# git log

> Ցույց տալ պարտավորությունների պատմությունը:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-log>:.

- Ցույց տվեք կատարման հաջորդականությունը՝ սկսած ընթացիկից, ընթացիկ աշխատանքային գրացուցակում Git պահոցի հակառակ ժամանակագրական կարգով.:

`git log`

- Ցույց տալ որոշակի ֆայլի կամ գրացուցակի պատմությունը, ներառյալ տարբերությունները.:

`git log {{[-p|--patch]}} {{path/to/file_or_directory}}`

- Ցույց տվեք ակնարկ այն մասին, թե որ ֆայլ(ներ)ը փոխվել է յուրաքանչյուր commit-ում.:

`git log --stat`

- Ցուցադրել պարտավորությունների գրաֆիկը ընթացիկ մասնաճյուղում, օգտագործելով միայն յուրաքանչյուր commit հաղորդագրության առաջին տողը.:

`git log --oneline --graph`

- Ցույց տվեք բոլոր պարտավորությունների, պիտակների և ճյուղերի գրաֆիկը ամբողջ ռեպոում.:

`git log --oneline --decorate --all --graph`

- Ցուցադրել միայն որոշակի տող պարունակող հաղորդագրություններով՝ անտեսելով գործը.:

`git log {{[-i|--regexp-ignore-case]}} --grep {{search_string}}`

- Ցուցադրել որոշակի հեղինակի վերջին `n` թվի պարտավորությունները.:

`git log {{[-n|--max-count]}} {{n}} --author "{{author}}"`

- Ցույց տալ պարտավորությունները երկու ամսաթվերի միջև (տտտտ-մմ-դդ).:

`git log --before "{{2017-01-29}}" --after "{{2017-01-17}}"`
