# git փոփոխության լոգ

> Ստեղծեք փոփոխությունների մատյանների հաշվետվություն պահեստի պարտավորություններից և պիտակներից:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/git-changelog>:.

- Թարմացրեք առկա ֆայլը կամ ստեղծեք նոր `History.md` ֆայլ՝ commit հաղորդագրություններով՝ սկսած վերջին Git թեգից.:

`git changelog`

- Ցուցակ կատարեք ընթացիկ տարբերակից.:

`git changelog {{[-l|--list]}}`

- Թվարկեք մի շարք պարտավորություններ՝ `2.1.0` անունով պիտակից մինչև հիմա.:

`git changelog {{[-l|--list]}} {{[-s|--start-tag]}} 2.1.0`

- Թվարկեք պարտավորությունների բավականին ձևաչափված շրջանակ `0.5.0` և `1.0.0` թեգի միջև:

`git changelog {{[-s|--start-tag]}} 0.5.0 {{[-f|--final-tag]}} 1.0.0`

- Նշեք պարտավորությունների բավականին ձևաչափված շրջանակը `0b97430`-ի և `1.0.0` պիտակի միջև:

`git changelog --start-commit 0b97430 {{[-f|--final-tag]}} 1.0.0`

- Նշեք `CHANGELOG.md` որպես ելքային ֆայլ՝:

`git changelog CHANGELOG.md`

- Ամբողջովին փոխարինեք ընթացիկ փոփոխության ֆայլի բովանդակությունը.:

`git changelog {{[-p|--prune-old]}}`
