# wapm

> WebAssembly փաթեթի կառավարիչ:.
> Լրացուցիչ տեղեկություններ. <https://wasmer.io/help/reference>:.

- Ինտերակտիվ կերպով ստեղծեք նոր `wapm.toml` ֆայլ.:

`wapm init`

- Ներբեռնեք բոլոր փաթեթները, որոնք նշված են որպես կախվածություն `wapm.toml`-ում:

`wapm install`

- Ներբեռնեք փաթեթի որոշակի տարբերակ և ավելացրեք այն `wapm.toml`-ի կախվածությունների ցանկում՝:

`wapm install {{package}}@{{version}}`

- Ներբեռնեք փաթեթ և տեղադրեք այն ամբողջ աշխարհում.:

`wapm install --global {{package}}`

- Ապատեղադրեք փաթեթը և հեռացրեք այն `wapm.toml`-ի կախվածությունների ցանկից՝:

`wapm uninstall {{package}}`

- Տպեք տեղական տեղադրված կախվածությունների ծառը.:

`wapm list`

- Ցուցակե՛ք բարձր մակարդակի գլոբալ տեղադրված փաթեթները.:

`wapm list --global`

- Կատարեք փաթեթի հրաման՝ օգտագործելով Wasmer գործարկման ժամանակը.:

`wapm run {{command_name}} {{arguments}}`
