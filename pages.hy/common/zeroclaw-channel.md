# zeroclaw ալիք

> Կառավարեք ալիքները (Telegram, Discord, Slack և այլն) ZeroClaw-ի համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/zeroclaw-labs/zeroclaw#quick-start>:.

- Նշեք բոլոր կազմաձևված ալիքները.:

`zeroclaw channel list`

- Սկսեք բոլոր կազմաձևված ալիքները.:

`zeroclaw channel start`

- Գործարկել առողջական ստուգումներ կազմաձևված ալիքների համար.:

`zeroclaw channel doctor`

- Ավելացնել նոր ալիք.:

`zeroclaw channel add {{telegram|discord|whatsapp|slack|webhook|...}} '{{json_config}}'`

- Հեռացնել ալիքը.:

`zeroclaw channel remove {{channel_name}}`

- Ցուցադրել օգնությունը.:

`zeroclaw channel {{[-h|--help]}}`
