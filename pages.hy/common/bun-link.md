# bun link

> Գրանցեք տեղական փաթեթը որպես հղման ենթակա կամ գրանցված փաթեթը միացրեք նախագծին:.
> Տես նաև՝ `bun unlink`:.
> Լրացուցիչ տեղեկություններ. <https://bun.com/docs/pm/cli/link>:.

- Կապեք ընթացիկ փաթեթը գլոբալ.:

`bun link`

- Տեղական փաթեթը կապեք նախագծին.:

`bun link {{package_name}}`

- Միացրեք փաթեթը որոշակի գրացուցակում.:

`bun link --cwd {{path/to/package}}`

- Կատարեք չոր վազք առանց իրականում կապելու.:

`bun link --dry-run`

- Ցուցադրել օգնությունը.:

`bun link {{[-h|--help]}}`
