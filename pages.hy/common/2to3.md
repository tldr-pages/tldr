# 2-ից 3

> Python 2-ից 3-ի ավտոմատ փոխակերպում:.
> Նշում. այս մոդուլը հնացել է 3.11-ում և հեռացվել 3.13-ում:.
> Հղումների համար տես՝ <https://github.com/python/cpython/blob/8d42e2d915c3096e7eac1c649751d1da567bb7c3/Doc/whatsnew/3.13.rst?plain=1#L188>:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/2to3>:.

- Ցուցադրել փոփոխությունները, որոնք կկատարվեն առանց դրանք կատարելու (չոր գործարկում).:

`2to3 {{path/to/file.py}}`

- Փոխարկել Python 2 ֆայլը Python 3:

`2to3 {{[-w|--write]}} {{path/to/file.py}}`

- Փոխակերպեք Python 2-ի հատուկ լեզվի առանձնահատկությունները Python 3-ի.:

`2to3 {{[-w|--write]}} {{path/to/file.py}} {{[-f|--fix]}} {{raw_input}} {{[-f|--fix]}} {{print}}`

- Փոխակերպեք Python 2 լեզվի բոլոր հատկանիշները, բացառությամբ նշվածների, Python 3-ի:

`2to3 {{[-w|--write]}} {{path/to/file.py}} {{[-x|--nofix]}} {{has_key}} {{[-x|--nofix]}} {{isinstance}}`

- Թվարկեք բոլոր հասանելի լեզվական հատկանիշները, որոնք կարող են փոխարկվել Python 2-ից Python 3:

`2to3 {{[-l|--list-fixes]}}`

- Փոխակերպեք բոլոր Python 2 ֆայլերը գրացուցակում Python 3:

`2to3 {{[-o|--output-dir]}} {{path/to/python3_directory}} {{[-W|--write-unchanged-files]}} {{[-n|--nobackups]}} {{path/to/python2_directory}}`

- Գործարկել 2-ից 3-ը մի քանի թելերով.:

`2to3 {{[-j|--processes]}} {{4}} {{[-o|--output-dir]}} {{path/to/python3_directory}} {{[-w|--write]}} {{[-n|--nobackups]}} --no-diffs {{path/to/python2_directory}}`
