# glab պարոն ստեղծագործել

> Կառավարեք GitLab-ի միաձուլման հարցումները:.
> Լրացուցիչ տեղեկություններ. <https://gitlab.com/gitlab-org/cli/-/blob/main/docs/source/mr/create.md>:.

- Ինտերակտիվ կերպով ստեղծեք միաձուլման հարցում.:

`glab mr create`

- Ստեղծեք միաձուլման հարցում՝ որոշելով վերնագիրը և նկարագրությունը ընթացիկ մասնաճյուղի commit հաղորդագրություններից.:

`glab mr create {{[-f|--fill]}}`

- Ստեղծեք միաձուլման հայտի նախագիծ.:

`glab mr create --draft`

- Ստեղծեք միաձուլման հարցում՝ նշելով նպատակային մասնաճյուղը, անվանումը և նկարագրությունը.:

`glab mr create {{[-b|--target-branch]}} {{target_branch}} {{[-t|--title]}} "{{title}}" {{[-d|--description]}} "{{description}}"`

- Սկսեք բացել միաձուլման հարցումը լռելյայն վեբ դիտարկիչում.:

`glab mr create {{[-w|--web]}}`
