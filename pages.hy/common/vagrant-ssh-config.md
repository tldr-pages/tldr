# թափառաշրջիկ ssh-config

> Ցուցադրել SSH-ի կոնֆիգուրացիան Vagrant մեքենայի մեջ ձեռքով (`vagrant ssh`-ի փոխարեն):.
> Տես նաև՝ `vagrant`:.
> Լրացուցիչ տեղեկություններ. <https://developer.hashicorp.com/vagrant/docs/cli/ssh_config>:.

- Ստացեք SSH կոնֆիգուրացիան ընթացիկ գրացուցակում աշխատող մեքենայի համար.:

`vagrant ssh-config`

- Թիրախավորեք մեքենան անունով կամ ID-ով.:

`vagrant ssh-config {{name|id}}`
