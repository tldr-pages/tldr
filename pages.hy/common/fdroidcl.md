# fdroidcl

> Կառավարեք ԱԶԲ-ի միջոցով միացված սարքերի F-Droid հավելվածները:.
> Լրացուցիչ տեղեկություններ. <https://github.com/Hoverth/fdroidcl#commands>:.

- Ստացեք F-Droid ինդեքսը.:

`fdroidcl update`

- Ցուցադրել տեղեկություններ հավելվածի մասին.:

`fdroidcl show {{app_id}}`

- Ներբեռնեք հավելվածի APK ֆայլը.:

`fdroidcl download {{app_id}}`

- Որոնեք հավելված ինդեքսում.:

`fdroidcl search {{search_pattern}}`

- Տեղադրեք հավելված միացված սարքի վրա.:

`fdroidcl install {{app_id}}`

- Ավելացնել պահեստ.:

`fdroidcl repo add {{repo_name}} {{url}}`

- Հեռացնել, միացնել կամ անջատել պահեստը.:

`fdroidcl repo {{remove|enable|disable}} {{repo_name}}`
