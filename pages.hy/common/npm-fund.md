# npm հիմնադրամ

> Ստացեք ֆինանսավորման մասին տեղեկատվություն փաթեթներից:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-fund/>:.

- Ցուցակեք կախվածությունները ծրագրի ֆինանսավորման URL-ով ընթացիկ գրացուցակում.:

`npm fund`

- Բացեք ֆինանսավորման URL-ը որոշակի փաթեթի համար կանխադրված վեբ բրաուզերում.:

`npm fund {{package}}`

- Ցուցակեք կախվածությունը ընթացիկ գրացուցակում նախագծի կոնկրետ աշխատանքային տարածքի ֆինանսավորման URL-ով.:

`npm fund {{[-w|--workspace]}} {{workspace}}`
