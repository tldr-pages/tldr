# httpry

> Թեթև փաթեթների ախտահանիչ՝ HTTP տրաֆիկը ցուցադրելու և գրանցելու համար:.
> Այն կարող է գործարկվել իրական ժամանակում` ցուցադրելով երթևեկը, քանի որ այն վերլուծվում է, կամ որպես դևոն գործընթաց, որը մուտքագրվում է ելքային ֆայլ:.
> Լրացուցիչ տեղեկություններ. <https://dumpsterventures.com/jason/httpry/>:.

- Պահպանեք արդյունքը ֆայլում.:

`httpry -o {{path/to/file.log}}`

- Լսեք հատուկ ինտերֆեյսի վրա և պահեք արդյունքը երկուական PCAP ձևաչափի ֆայլում.:

`httpry {{eth0}} -b {{path/to/file.pcap}}`

- Զտել ելքը HTTP բայերի ստորակետերով բաժանված ցանկով.:

`httpry -m {{get|post|put|head|options|delete|trace|connect|patch}}`

- Կարդացեք մուտքագրման ֆայլից և զտեք IP-ով.:

`httpry -r {{path/to/file.log}} '{{host 192.168.5.25}}'`

- Գործարկել որպես դեյմոն գործընթաց.:

`httpry -d -o {{path/to/file.log}}`
