# pio կարգավորումներ

> Դիտեք և փոփոխեք PlatformIO-ի կարգավորումները:.
> Լրացուցիչ տեղեկություններ. <https://docs.platformio.org/en/latest/core/userguide/cmd_settings.html>:.

- Ցուցադրել բոլոր PlatformIO պարամետրերի անունները, արժեքները և նկարագրությունները.:

`pio settings get`

- Ցուցադրել կոնկրետ PlatformIO պարամետրի անունը, արժեքը և նկարագրությունը.:

`pio settings get {{setting}}`

- Սահմանեք հատուկ պարամետրերի արժեք.:

`pio settings set {{setting}} {{value}}`

- Վերականգնել բոլոր փոփոխված պարամետրերի արժեքները իրենց գործարանային ստանդարտներին.:

`pio settings reset`
