# py-լրտես

> Python ծրագրերի նմուշառման պրոֆիլավորիչ:.
> Լրացուցիչ տեղեկություններ. <https://github.com/benfred/py-spy>:.

- Ցույց տվեք այն գործառույթների կենդանի տեսքը, որոնք պահանջում են գործող գործընթացի առավելագույն կատարման ժամանակը.:

`py-spy top {{[-p|--pid]}} {{pid}}`

- Սկսեք ծրագիր և ցուցադրեք այն գործառույթների կենդանի տեսքը, որոնք ամենաշատ ժամանակն են պահանջում.:

`py-spy top -- python {{path/to/file.py}}`

- Ստեղծեք ֆունկցիայի կանչի կատարման ժամանակի SVG կրակի գրաֆիկ.:

`py-spy record {{[-o|--output]}} {{path/to/profile.svg}} {{[-p|--pid]}} {{pid}}`

- Թափել գործող գործընթացի զանգերի փաթեթը.:

`py-spy dump {{[-p|--pid]}} {{pid}}`
