#պիկո

> Տեքստի խմբագիր՝ ոճավորված Alpine Composer-ից հետո:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/pico>:.

- Սկսեք խմբագրիչը.:

`pico {{path/to/file}}`

- Սկսեք խմբագրիչը կուրսորով, որը գտնվում է n տող ֆայլում.:

`pico +{{n}} {{path/to/file}}`

- Սկսեք խմբագրիչը ընթացիկ ընտրությունից առաջ ցուցադրված կուրսորով.:

`pico -g {{path/to/file}}`

- Սահմանեք մեջբերումների տողը այնպիսի ֆայլերի համար, ինչպիսիք են էլ.:

`pico -Q "{{quotestring}}" {{path/to/file}}`

- Միացնել մկնիկի գործառույթը, երբ գործարկվում է `xterm` պատուհանում.:

`pico -m {{path/to/file}}`

- Սահմանեք գործառնական գրացուցակը `pico`-ի համար.:

`pico -o {{path/to/directory}}`

- Միացնել «միայն դիտել» ռեժիմը, որն արգելում է ցանկացած խմբագրում.:

`pico -v {{path/to/file}}`

- Ցուցադրել բոլոր ֆայլերը, ներառյալ նրանք, որոնք սկսվում են կետով.:

`pico -a`
