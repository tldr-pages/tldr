# sshuttle

> Թափանցիկ վստահված սերվեր, որը թունելավորում է երթևեկությունը SSH կապի միջոցով:.
> Չի պահանջում root կամ որևէ հատուկ կարգավորում հեռավոր SSH սերվերի վրա, թեև տեղական մեքենայի վրա արմատային մուտքի թույլտվություն է պահանջվում:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/sshuttle>:.

- Փոխանցել ամբողջ IPv4 TCP տրաֆիկը հեռավոր SSH սերվերի միջոցով.:

`sshuttle {{[-r|--remote]}} {{username}}@{{sshserver}} {{0.0.0.0/0}}`

- Նաև փոխանցեք ամբողջ DNS տրաֆիկը սերվերի լռելյայն DNS լուծիչին.:

`sshuttle --dns {{[-r|--remote]}} {{username}}@{{sshserver}} {{0.0.0.0/0}}`

- Փոխանցել ամբողջ տրաֆիկը, բացառությամբ այն, որը կապված է որոշակի ենթացանցով.:

`sshuttle {{[-r|--remote]}} {{username}}@{{sshserver}} {{0.0.0.0/0}} {{[-x|--exclude]}} {{192.168.0.1/24}}`

- Օգտագործեք tproxy մեթոդը՝ բոլոր IPv4 և IPv6 տրաֆիկները փոխանցելու համար.:

`sshuttle --method tproxy {{[-r|--remote]}} {{username}}@{{sshserver}} {{0.0.0.0/0}} {{::/0}} {{[-x|--exclude]}} {{your_local_ip_address}} {{[-x|--exclude]}} {{ssh_server_ip_address}}`
