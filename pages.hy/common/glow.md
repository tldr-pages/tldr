#փայլ

> Ցուցադրել Markdown-ը տերմինալում:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/glow>:.

- Գործարկեք փայլը և ընտրեք ֆայլ՝ դիտելու համար.:

`glow`

- Ներկայացրեք Markdown ֆայլը տերմինալին.:

`glow {{path/to/file}}`

- Դիտեք Markdown ֆայլը, օգտագործելով paginator:

`glow {{[-p|--pager]}} {{path/to/file}}`

- Դիտեք ֆայլը URL-ից.:

`glow {{https://example.com/file.md}}`

- Դիտեք GitHub/GitLab README:

`glow {{github.com/owner/repository}}`
