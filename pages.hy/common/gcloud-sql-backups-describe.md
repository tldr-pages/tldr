# gcloud sql կրկնօրինակները նկարագրում են

> Ստացեք տեղեկատվությունը կրկնօրինակի մասին:.
> Տես նաև՝ `gcloud`:.
> Լրացուցիչ տեղեկություններ. <https://docs.cloud.google.com/sdk/gcloud/reference/sql/backups/describe>:.

- Ստացեք տեղեկությունները կրկնօրինակի մասին.:

`gcloud sql backups describe {{backup_id}} --instance={{instance_id}}`
