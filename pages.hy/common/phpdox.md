# phpdox

> PHP փաստաթղթերի գեներատոր:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/phpdox>:.

- Ցուցադրել ծանոթագրված կմախքի կազմաձևման XML ֆայլ.:

`phpdox --skel`

- Ստեղծեք փաստաթղթեր ընթացիկ աշխատանքային գրացուցակի համար.:

`phpdox`

- Ստեղծեք փաստաթղթեր, օգտագործելով հատուկ կազմաձևման ֆայլ.:

`phpdox {{[-f|--file]}} {{path/to/phpdox.xml}}`

- Գործարկեք միայն մետատվյալների հավաքագրման գործընթացը.:

`phpdox {{[-c|--collector]}}`

- Գործարկեք միայն փաստաթղթերի ստեղծման գործընթացը.:

`phpdox {{[-g|--generator]}}`
