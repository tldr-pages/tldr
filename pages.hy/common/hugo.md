#հյուգո

> Կաղապարի վրա հիմնված ստատիկ կայքի գեներատոր:.
> Օգտագործում է մոդուլներ, բաղադրիչներ և թեմաներ:.
> Որոշ ենթահրամաններ, ինչպիսիք են `server`-ն, ունեն իրենց օգտագործման փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://gohugo.io/commands/>:.

- Ստեղծեք նոր Hugo կայք.:

`hugo new site {{path/to/site}}`

- Ստեղծեք նոր Hugo թեմա (թեմաները կարելի է ներբեռնել նաև <https://themes.gohugo.io/>-ից):

`hugo new theme {{theme_name}}`

- Ստեղծեք նոր էջ.:

`hugo new {{section_name}}/{{page_name}}`

- Ստեղծեք կայք `public` գրացուցակում՝:

`hugo`

- Կառուցեք կայք՝ ներառելով էջեր, որոնք նշված են որպես «սևագիր».:

`hugo {{[-D|--buildDrafts]}}`

- Ստեղծեք կայք ձեր տեղական IP-ի վրա.:

`hugo server --bind {{local_ip}} {{[-b|--baseURL]}} {{http://local_ip}}`

- Ստեղծեք կայք տվյալ գրացուցակում.:

`hugo {{[-d|--destination]}} {{path/to/destination}}`

- Կառուցեք կայք, գործարկեք վեբսերվեր այն սպասարկելու համար և ավտոմատ կերպով վերաբեռնեք, երբ էջերը խմբագրվում են.:

`hugo server`
