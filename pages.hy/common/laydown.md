# laydown

> Պատրաստվեք ամենօրյա Standup-ին:.
> Լրացուցիչ տեղեկություններ. <https://github.com/badjr13/laydown>:.

- Ավելացնել տարր DID բաժնում.:

`laydown did {{item}}`

- Ավելացնել տարր DOING բաժնում.:

`laydown doing {{item}}`

- Մաքրել բոլոր տարրերը.:

`laydown clear`

- Օգտագործեք խմբագիր ընթացիկ տվյալները խմբագրելու համար.:

`laydown edit`

- Արխիվացնել և մաքրել ընթացիկ տվյալները.:

`laydown archive`
