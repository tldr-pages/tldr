# pio հարթակ

> Կառավարեք PlatformIO զարգացման հարթակները:.
> Լրացուցիչ տեղեկություններ. <https://docs.platformio.org/en/latest/core/userguide/platforms/>:.

- Թվարկեք բոլոր տեղադրված զարգացման հարթակները.:

`pio platform list`

- Որոնել գոյություն ունեցող զարգացման հարթակներ.:

`pio platform search {{platform}}`

- Ցույց տալ մանրամասներ զարգացման հարթակի մասին.:

`pio platform show {{platform}}`

- Տեղադրեք զարգացման հարթակ.:

`pio platform install {{platform}}`

- Թարմացրեք տեղադրված զարգացման հարթակները.:

`pio platform update`

- Տեղահանել զարգացման հարթակը.:

`pio platform uninstall {{platform}}`

- Թվարկեք բոլոր աջակցվող շրջանակները.:

`pio platform frameworks`
