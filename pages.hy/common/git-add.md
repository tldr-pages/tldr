# git ավելացնել

> Փոփոխված ֆայլերը ստանձնելու համար:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-add>:.

- Հանձնարարականի համար ֆայլ բեմադրեք.:

`git add {{path/to/file}}`

- Ավելացնել բոլոր ֆայլերը (հետևված և չհետևված).:

`git add {{[-A|--all]}}`

- Ավելացրեք բոլոր ֆայլերը ռեկուրսիվորեն՝ սկսած ընթացիկ թղթապանակից.:

`git add .`

- Ավելացրեք միայն արդեն հետևված ֆայլերը.:

`git add {{[-u|--update]}}`

- Ավելացնել անտեսված ֆայլ.:

`git add {{[-f|--force]}} {{path/to/file}}`

- Ինտերակտիվ կերպով բեմադրեք ֆայլերի մասերը.:

`git add {{[-p|--patch]}}`

- Ինտերակտիվ կերպով բեմադրեք տվյալ ֆայլի մասերը.:

`git add {{[-p|--patch]}} {{path/to/file}}`

- Ինտերակտիվ կերպով բեմադրեք ֆայլը.:

`git add {{[-i|--interactive]}}`
