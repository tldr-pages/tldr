# թունավոր

> Ավտոմատացրեք Python-ի փորձարկումը Python-ի բազմաթիվ տարբերակներում:.
> Օգտագործեք tox.ini միջավայրերը կարգավորելու և փորձարկման հրամանը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tox-dev/tox>:.

- Գործարկել թեստերը բոլոր թեստային միջավայրերում.:

`tox`

- Ստեղծեք `tox.ini` կոնֆիգուրացիա.:

`tox-quickstart`

- Թվարկեք մատչելի միջավայրերը.:

`tox {{[-a|--listenvs-all]}}`

- Գործարկել թեստերը կոնկրետ միջավայրում (օրինակ՝ Python 3.6).:

`tox -e {{py36}}`

- Ստիպել վերստեղծել վիրտուալ միջավայրը.:

`tox {{[-r|--recreate]}} -e {{py27}}`
