# openclaw դարպաս

> Գործարկեք OpenClaw gateway սերվերը (կառավարման հարթություն):.
> Լրացուցիչ տեղեկություններ. <https://docs.openclaw.ai/cli/gateway>:.

- Սկսեք դարպասը լռելյայն նավահանգստում (18789):

`openclaw gateway --allow-unconfigured`

- Սկսեք դարպասը որոշակի նավահանգստի վրա.:

`openclaw gateway --allow-unconfigured --port {{18789}}`

- Սկսեք դարպասը մանրամասն գրանցումով.:

`openclaw gateway --allow-unconfigured --verbose`

- Ցուցադրել օգնությունը.:

`openclaw gateway {{[-h|--help]}}`
