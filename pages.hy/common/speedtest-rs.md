# speedtest-rs

> Rust-ի վրա հիմնված ոչ պաշտոնական գործիք՝ speedtest.net-ի միջոցով ցանցի արագությունը ստուգելու համար, որը սահմանափակվում է HTTP Legacy Fallback-ով:.
> Լրացուցիչ տեղեկություններ. <https://github.com/nelsonjchen/speedtest-rs>:.

- Գործարկել ամբողջական արագության թեստ (ներբեռնել և վերբեռնել).:

`speedtest-rs`

- Ցուցադրել `speedtest.net` սերվերների ցանկը՝ դասավորված ըստ հեռավորության.:

`speedtest-rs --list`

- Գործարկեք միայն ներբեռնման թեստ.:

`speedtest-rs --no-upload`

- Գործարկել միայն վերբեռնման թեստը.:

`speedtest-rs --no-download`

- Ստեղծեք փորձարկման արդյունքների պատկերի համօգտագործվող հղում.:

`speedtest-rs --share`

- Ցուցադրել միայն հիմնական ելքային տեղեկատվությունը.:

`speedtest-rs --simple`
