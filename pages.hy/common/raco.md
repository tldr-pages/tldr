#ռակո

> Ռակետային գործիքներ:.
> Լրացուցիչ տեղեկություններ. <https://docs.racket-lang.org/raco/>:.

- Տեղադրեք փաթեթ՝ ավտոմատ կերպով տեղադրելով կախվածությունները.:

`raco pkg install --auto {{package_source}}`

- Տեղադրեք ընթացիկ գրացուցակը որպես փաթեթ.:

`raco pkg install`

- Ստեղծեք (կամ վերակառուցեք) բայթկոդ, փաստաթղթեր, գործարկվող նյութեր և մետատվյալների ինդեքսներ հավաքածուների համար.:

`raco setup {{collection1 collection2 ...}}`

- Գործարկել թեստերը ֆայլերում.:

`raco test {{path/to/tests1.rkt path/to/tests2.rkt ...}}`

- Որոնել տեղական փաստաթղթեր.:

`raco docs {{search_terms}}`

- Ցուցադրել օգնությունը.:

`raco help`
