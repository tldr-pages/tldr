# առաջադրանք

> Անելիքների ցանկի կառավարիչ:.
> Լրացուցիչ տեղեկություններ. <https://taskwarrior.org/docs/>:.

- Ավելացրեք նոր առաջադրանք, որը նախատեսված է վաղը.:

`task add {{description}} due:{{tomorrow}}`

- Թարմացրեք առաջադրանքի առաջնահերթությունը.:

`task {{task_id}} modify priority:{{H|M|L}}`

- Կատարել առաջադրանք.:

`task {{task_id}} done`

- Ջնջել առաջադրանքը.:

`task {{task_id}} delete`

- Թվարկեք բոլոր բաց առաջադրանքները.:

`task list`

- Թվարկեք բաց առաջադրանքները, որոնք նախատեսված են մինչև շաբաթվա վերջ.:

`task list due.before:{{eow}}`

- Ցույց տվեք այրման գրաֆիկական աղյուսակը, ըստ օրվա.:

`task burndown.daily`

- Թվարկեք բոլոր հաշվետվությունները.:

`task reports`
