# բեռների տվյալներ

> Ցուցադրել Rust փաթեթի մասին տեղեկատվություն:.
> Լրացուցիչ տեղեկություններ. <https://doc.rust-lang.org/cargo/commands/cargo-info.html>:.

- Ցուցադրել փաթեթի մասին տեղեկատվություն <https://crates.io> կայքում:

`cargo info {{package}}`

- Ցուցադրել տեղեկատվություն փաթեթի որոշակի տարբերակի մասին.:

`cargo info {{package}}@{{version}}`

- Ցուցադրել լրացուցիչ տեղեկատվություն փաթեթի մասին.:

`cargo info {{[-v|--verbose]}} {{package}}`
