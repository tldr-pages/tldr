#UV venv

> Ստեղծեք մեկուսացված Python միջավայր փաթեթներ տեղադրելու համար:.
> Լրացուցիչ տեղեկություններ. <https://docs.astral.sh/uv/reference/cli/#uv-venv>:.

- Ստեղծեք վիրտուալ միջավայր լռելյայն վայրում (`.venv`):

`uv venv`

- Ստեղծեք վիրտուալ միջավայր որոշակի ճանապարհով.:

`uv venv {{path/to/venv}}`

- Ստեղծեք Python-ի հատուկ տարբերակի միջոցով.:

`uv venv {{[-p|--python]}} {{3.12}}`

- Ներառեք սերմերի փաթեթներ, ինչպիսիք են `pip` վիրտուալ միջավայրը ստեղծելիս.:

`uv venv --seed`

- Ստեղծեք հատուկ հուշման նախածանցով.:

`uv venv --prompt {{my_project}}`

- Ստեղծեք և թույլատրեք վերագրանցել գոյություն ունեցող միջավայրը.:

`uv venv --allow-existing {{venv_name}}`
