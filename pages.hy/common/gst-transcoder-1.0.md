# gst-transcoder-1.0

> Փոխարկել լրատվամիջոցները երկու ֆայլի տեսակների միջև:.
> Լրացուցիչ տեղեկություններ. <https://wiki.st.com/stm32mpu/wiki/Gst-transcoder>:.

- Թվարկե՛ք տրանսկոդավորման հասանելի թիրախները.:

`gst-transcoder-1.0 {{[-l|--list-targets]}}`

- Փոխակերպել մեկ լրատվամիջոցից մյուսը.:

`gst-transcoder-1.0 {{path/to/input_file.webm}} {{path/to/output_file.mp4}}`
