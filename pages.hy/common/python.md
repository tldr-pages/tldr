# python

> Python լեզվի թարգմանիչ:.
> Լրացուցիչ տեղեկություններ. <https://docs.python.org/using/cmdline.html>:.

- Սկսեք REPL (ինտերակտիվ կեղև).:

`python`

- Կատարեք հատուկ Python ֆայլ.:

`python {{path/to/file.py}}`

- Կատարեք հատուկ Python ֆայլ և սկսեք REPL.:

`python -i {{path/to/file.py}}`

- Կատարեք Python արտահայտություն.:

`python -c "{{expression}}"`

- Գործարկեք նշված գրադարանային մոդուլի սցենարը.:

`python -m {{module}} {{arguments}}`

- Տեղադրեք փաթեթ՝ օգտագործելով `pip`՝:

`python -m pip install {{package}}`

- Ինտերակտիվ կարգաբերել Python սկրիպտը.:

`python -m pdb {{path/to/file.py}}`

- Գործարկեք ներկառուցված HTTP սերվերը 8000 նավահանգստում ընթացիկ գրացուցակում.:

`python -m http.server`
