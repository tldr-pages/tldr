# gitstats

> Git պահեստի վիճակագրության գեներատոր:.
> Լրացուցիչ տեղեկություններ. <https://gitstats.readthedocs.io/en/stable/getting-started.html#command-line-usage>:.

- Ստեղծեք վիճակագրություն տեղական պահեստի համար.:

`gitstats {{path/to/git_repo/.git}} {{path/to/output_folder}}`

- Դիտեք ստեղծված վիճակագրությունը վեբ բրաուզերում Windows (PowerShell)/macOS/Linux-ում:

`{{Invoke-Item|open|xdg-open}} {{path/to/output_folder/index.html}}`

- Ցուցադրել օգնությունը.:

`gitstats {{[-h|--help]}}`
