# փնթփնթալ

> PHP Composer հավելված, որը հնարավորություն է տալիս սկզբնական կոդի որակի ստուգում:.
> Լրացուցիչ տեղեկություններ. <https://github.com/phpro/grumphp#commands>:.

- Գրանցեք Git կեռիկները.:

`grumphp git:init`

- Ձեռքով գործարկեք նախնական կեռիկը.:

`grumphp git:pre-commit`

- Ստուգեք յուրաքանչյուր տարբերակված ֆայլ.:

`grumphp run`
