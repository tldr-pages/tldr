# ps-nvm

> PowerShell-ի վրա հիմնված օգտակար ծրագիր՝ Node.js-ի բազմաթիվ տարբերակները կառավարելու համար՝ ոգեշնչված `nvm`-ով:.
> Այս գործիքը տրամադրում է բազմաթիվ հրամաններ, որոնք բոլորը կարող են գործարկվել միայն PowerShell-ի միջոցով:.
> Լրացուցիչ տեղեկություններ. <https://github.com/aaronpowell/ps-nvm>:.

- Դիտեք փաստաթղթերը `Get-NodeInstallLocation`-ի համար, գործիք՝ Node.js-ի ընթացիկ տեղադրման վայրը ստանալու համար.:

`tldr get-nodeinstalllocation`

- Դիտեք փաստաթղթերը `Get-NodeVersions`-ի համար, գործիք՝ բոլոր հասանելի և ներկայումս տեղադրված Node.js տարբերակները ցուցակագրելու համար:

`tldr get-nodeversions`

- Դիտեք փաստաթղթերը `Install-NodeVersion`-ի համար՝ Node.js-ի գործարկման ժամանակի տարբերակները տեղադրելու գործիք.:

`tldr install-nodeversion`

- Դիտեք փաստաթղթերը `Remove-NodeVersion`-ի համար, գործիք՝ գոյություն ունեցող Node.js տարբերակը հեռացնելու համար:

`tldr remove-nodeversion`

- Դիտեք փաստաթղթերը `Set-NodeInstallLocation`-ի համար, գործիք՝ Node.js-ի տեղադրման վայրը սահմանելու համար.:

`tldr set-nodeinstalllocation`

- Դիտեք փաստաթղթերը `Set-NodeVersion`-ի համար, գործիք՝ Node.js-ի լռելյայն տարբերակը սահմանելու համար:

`tldr set-nodeversion`
