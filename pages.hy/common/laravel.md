# laravel

> Տեղադրող Laravel շրջանակի համար:.
> Լրացուցիչ տեղեկություններ. <https://laravel.com/docs/installation>:.

- Ստեղծեք նոր Laravel հավելված.:

`laravel new {{name}}`

- Օգտագործեք զարգացման վերջին թողարկումը.:

`laravel new {{name}} --dev`

- Վերագրեք, եթե գրացուցակը արդեն գոյություն ունի.:

`laravel new {{name}} --force`

- Տեղադրեք Laravel Jetstream փայտամածը.:

`laravel new {{name}} --jet`

- Տեղադրեք Laravel Jetstream փայտամածը հատուկ կույտով.:

`laravel new {{name}} --jet --stack {{livewire|inertia}}`

- Տեղադրեք Laravel Jetstream փայտամածը՝ թիմերի աջակցությամբ.:

`laravel new {{name}} --jet --teams`

- Թվարկեք տեղադրողի հասանելի հրամանները.:

`laravel list`
