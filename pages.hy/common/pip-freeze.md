# pip սառեցում

> Ցուցակեք տեղադրված փաթեթները պահանջների ձևաչափով:.
> Լրացուցիչ տեղեկություններ. <https://pip.pypa.io/en/stable/cli/pip_freeze/>:.

- Տեղադրված փաթեթների ցանկ.:

`pip freeze`

- Տեղադրված փաթեթները գրեք `requirements.txt` ֆայլում՝:

`pip freeze > requirements.txt`

- Ցուցակե՛ք տեղադրված փաթեթները վիրտուալ միջավայրում՝ բացառելով գլոբալ տեղադրված փաթեթները.:

`pip freeze {{[-l|--local]}}`

- Ցուցակեք տեղադրված փաթեթները օգտագործողի կայքում.:

`pip freeze --user`

- Թվարկեք բոլոր փաթեթները, ներառյալ `pip`, `distribute`, `setuptools` և `wheel` (նրանք բաց են թողնվել լռելյայնորեն):

`pip freeze --all`
