# git delta

> Ցուցակեք այլ ճյուղերից տարբերվող ֆայլերը:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-delta>:.

- Թվարկե՛ք ընթացիկ ստուգված մասնաճյուղի ֆայլերը, որոնք տարբերվում են `main` մասնաճյուղից.:

`git delta {{main}}`

- Թվարկեք ֆայլերը որոշակի մասնաճյուղից, որոնք տարբերվում են մեկ այլ ճյուղից.:

`git delta {{branch_1}} {{branch_2}}`
