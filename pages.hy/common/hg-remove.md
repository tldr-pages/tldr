# hg հեռացնել

> Հեռացրեք նշված ֆայլերը բեմադրության տարածքից:.
> Լրացուցիչ տեղեկություններ. <https://www.mercurial-scm.org/help/commands/remove>:.

- Հեռացրեք ֆայլերը կամ գրացուցակները բեմադրության տարածքից.:

`hg remove {{path/to/file}}`

- Հեռացրեք բոլոր փուլային ֆայլերը, որոնք համապատասխանում են նշված օրինակին.:

`hg remove {{[-I|--include]}} {{pattern}}`

- Հեռացրեք բոլոր բեմադրված ֆայլերը, բացառությամբ նրանց, որոնք համապատասխանում են նշված օրինակին.:

`hg remove {{[-X|--exclude]}} {{pattern}}`

- Ռեկուրսիվորեն հեռացնել ենթապահոցները.:

`hg remove {{[-S|--subrepos]}}`

- Հեռացրեք ֆայլերը պահոցից, որոնք ֆիզիկապես հեռացվել են.:

`hg remove {{[-A|--after]}}`
