# dropuser

> Հեռացրեք գոյություն ունեցող PostgreSQL օգտվողին (դերը):.
> Լրացուցիչ տեղեկություններ. <https://www.postgresql.org/docs/current/app-dropuser.html>:.

- Հաստատման և օգտվողի անունի հուշում մինչև օգտագործողի հեռացումը.:

`dropuser {{[-i|--interactive]}}`

- Անմիջապես հեռացնել օգտվողին.:

`dropuser {{username}}`

- Սխալ չկա, եթե հեռացվող օգտվողը գոյություն չունի.:

`dropuser --if-exists {{username}}`

- Հեռացրեք օգտվողին սերվերից 127.0.0.1 հասցեով 4321 նավահանգստում.:

`dropuser {{[-h|--host]}} 127.0.0.1 {{[-p|--port]}} 4321 {{username}}`

- Հեռացրեք օգտվողին սերվերից 127.0.0.1 հասցեով 4321 նավահանգստում որպես օգտագործողի «admin»:

`dropuser {{[-U|--username]}} admin {{[-h|--host]}} 127.0.0.1 {{[-p|--port]}} 4321 {{username}}`
