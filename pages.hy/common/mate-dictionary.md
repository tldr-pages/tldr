# mate-բառարան

> Փնտրեք բառեր բառարաններում:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/mate-dictionary>:.

- Տպել կոնկրետ բառի սահմանում.:

`mate-dictionary --no-window --look-up '{{word}}'`

- Ցույց տալ նմանատիպ բառեր կոնկրետ մեկի համար նոր պատուհանում.:

`mate-dictionary --match '{{word}}'`
