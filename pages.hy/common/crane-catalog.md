# կռունկների կատալոգ

> Թվարկեք պահեստները գրանցամատյանում:.
> Լրացուցիչ տեղեկություններ. <https://github.com/google/go-containerregistry/blob/main/cmd/crane/doc/crane_catalog.md>:.

- Թվարկեք ռեեստրի պահեստները.:

`crane catalog {{registry_address}}`

- Տպել պատկերի ամբողջական հղումը.:

`crane catalog {{registry_address}} --full-ref`

- Ցուցադրել օգնությունը.:

`crane catalog {{[-h|--help]}}`
