# phpstorm

> PHP-ի միջպլատֆորմային IDE՝ հիմնված JetBrains IntelliJ հարթակի վրա:.
> Լրացուցիչ տեղեկություններ. <https://www.jetbrains.com/help/phpstorm/working-with-the-ide-features-from-command-line.html#arguments>:.

- Բացեք հատուկ գրացուցակ.:

`phpstorm {{path/to/directory}}`

- Բացեք ֆայլ.:

`phpstorm {{path/to/file}}`

- Բացեք ֆայլ որոշակի տողում.:

`phpstorm --line {{line_number}} {{path/to/file}}`

- Դիտեք երկու ֆայլերի միջև եղած տարբերությունները.:

`phpstorm diff {{path/to/left_file}} {{path/to/right_file}}`
