# rustup doc

> Բացեք Rust-ի անցանց փաստաթղթերը ընթացիկ գործիքների շղթայի համար:.
> Կան շատ ավելի շատ փաստաթղթերի էջեր, որոնք այստեղ նշված չեն: Լրացուցիչ տեղեկությունների համար տես `rustup help doc`:.
> Լրացուցիչ տեղեկություններ. <https://rust-lang.github.io/rustup/>:.

- Բացեք հիմնական էջը.:

`rustup doc`

- Բացեք փաստաթղթերը կոնկրետ թեմայի համար (մոդուլ ստանդարտ գրադարանում, տեսակ, հիմնաբառ և այլն).:

`rustup doc {{std::fs|usize|fn|...}}`

- Բացեք Rust Programming Language գիրքը.:

`rustup doc --book`

- Բացեք բեռների գիրքը.:

`rustup doc --cargo`

- Բացեք Rust Reference-ը.:

`rustup doc --reference`
