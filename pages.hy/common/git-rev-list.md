# git rev-ցուցակ

> Թվարկեք վերանայումները (պարտավորությունները) հակառակ ժամանակագրական հաջորդականությամբ:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-rev-list>:.

- Թվարկեք ընթացիկ մասնաճյուղի բոլոր պարտավորությունները.:

`git rev-list HEAD`

- Տպեք վերջին commit-ը, որը փոխել է (ավելացնել/խմբագրել/հեռացնել) որոշակի ֆայլ ընթացիկ մասնաճյուղում.:

`git rev-list {{[-n|--max-count]}} 1 HEAD -- {{path/to/file}}`

- Ցանկը կատարում է ավելի վերջին, քան կոնկրետ ամսաթվից, կոնկրետ մասնաճյուղում.:

`git rev-list --since "{{2019-12-01 00:00:00}}" {{branch_name}}`

- Թվարկեք բոլոր միաձուլման պարտավորությունները կոնկրետ հանձնառության վրա.:

`git rev-list --merges {{commit}}`

- Տպեք պարտավորությունների քանակը որոշակի պիտակից.:

`git rev-list {{tag_name}}..HEAD --count`
