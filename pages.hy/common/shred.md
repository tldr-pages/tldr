# մանրացնել

> Վերագրեք ֆայլերը՝ տվյալները ապահով կերպով ջնջելու համար:.
> Լրացուցիչ տեղեկություններ. <https://www.gnu.org/software/coreutils/manual/html_node/shred-invocation.html>:.

- Վերագրանցել ֆայլը.:

`shred {{path/to/file}}`

- Վերագրեք ֆայլը և ցուցադրեք առաջընթացը էկրանին.:

`shred {{[-v|--verbose]}} {{path/to/file}}`

- Վերագրեք ֆայլը՝ պատահական տվյալների փոխարեն զրոներ թողնելով.:

`shred {{[-z|--zero]}} {{path/to/file}}`

- Վերագրանցեք ֆայլը որոշակի քանակությամբ անգամ.:

`shred {{[-n|--iterations]}} {{25}} {{path/to/file}}`

- Վերագրեք ֆայլը և հեռացրեք այն.:

`shred {{[-u|--remove]}} {{path/to/file}}`

- Վերագրեք ֆայլը 100 անգամ, ավելացրեք վերջնական վերագրումը զրոներով, հեռացրեք ֆայլը այն վերագրելուց հետո և ցուցադրեք էկրանի առաջընթացը:

`shred {{[-vzu|--verbose --zero --remove]}} {{[-n|--iterations]}} 100 {{path/to/file}}`
