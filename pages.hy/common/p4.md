# p4

> Perforce Version Control System:.
> Լրացուցիչ տեղեկություններ. <https://help.perforce.com/helix-core/server-apps/cmdref/current/Content/CmdRef/commands.html>:.

- Մուտք գործեք Perforce ծառայություն.:

`p4 login -a`

- Ստեղծեք հաճախորդ.:

`p4 client`

- Պատճենեք ֆայլերը դեպոյից հաճախորդի աշխատանքային տարածք.:

`p4 sync`

- Ստեղծեք կամ խմբագրեք փոփոխությունների ցանկի նկարագրությունը.:

`p4 change`

- Բացեք ֆայլ՝ խմբագրելու համար.:

`p4 edit -c {{changelist_number}} {{path/to/file}}`

- Բացեք նոր ֆայլ՝ այն պահեստում ավելացնելու համար.:

`p4 add`

- Ցուցադրել փոփոխությունների ցանկով փոփոխված ֆայլերի ցանկը.:

`p4 describe -c {{changelist_number}}`

- Ներկայացրեք փոփոխությունների ցուցակը դեպոյին.:

`p4 submit -c {{changelist_number}}`
