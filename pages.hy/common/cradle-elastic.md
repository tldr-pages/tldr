# օրորոցի էլաստիկ

> Կառավարեք Elasticsearch-ի օրինակները Cradle-ի օրինակի համար:.
> Լրացուցիչ տեղեկություններ. <https://cradlephp.github.io/docs/3.B.-Reference-Command-Line-Tools.html#elastic>:.

- Կտրեք Elasticsearch ինդեքսը.:

`cradle elastic flush`

- Կտրեք Elasticsearch ինդեքսը որոշակի փաթեթի համար.:

`cradle elastic flush {{package}}`

- Ներկայացրեք Elasticsearch սխեման.:

`cradle elastic map`

- Ներկայացրեք Elasticsearch սխեման որոշակի փաթեթի համար.:

`cradle elastic map {{package}}`

- Լրացրեք Elasticsearch ինդեքսները բոլոր փաթեթների համար.:

`cradle elastic populate`

- Լրացրեք Elasticsearch ինդեքսները որոշակի փաթեթի համար.:

`cradle elastic populate {{package}}`
