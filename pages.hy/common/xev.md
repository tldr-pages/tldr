# xev

> Տպել X իրադարձությունների բովանդակությունը:.
> Լրացուցիչ տեղեկություններ. <https://gitlab.freedesktop.org/xorg/app/xev>:.

- Դիտեք բոլոր տեղի ունեցող X իրադարձությունները.:

`xev`

- Դիտեք արմատային պատուհանի բոլոր X իրադարձությունները՝ նորը ստեղծելու փոխարեն.:

`xev -root`

- Դիտեք որոշակի պատուհանի բոլոր X իրադարձությունները.:

`xev -id {{window_id}}`

- Դիտեք X իրադարձությունները տվյալ կատեգորիայից (կարելի է մի քանի անգամ նշել).:

`xev -event {{event_category}}`
