# r2e

> Փոխանցել RSS հոսքերը էլփոստի հասցեին:.
> Պահանջվում է կազմաձևված `sendmail` կամ SMTP կարգավորում:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/r2e>:.

- Ստեղծեք նոր թարմացվող տվյալների բազա, որը էլփոստի հասցե է ուղարկում.:

`r2e new {{email_address}}`

- Բաժանորդագրվեք հոսքին.:

`r2e add {{feed_name}} {{feed_uri}}`

- Ուղարկեք նոր պատմություններ էլփոստի հասցեով.:

`r2e run`

- Թվարկեք բոլոր հոսքերը.:

`r2e list`

- Ջնջել հոսքը նշված ցուցանիշով.:

`r2e delete {{index}}`

- Ցուցադրել օգնությունը.:

`r2e {{[-h|--help]}}`
