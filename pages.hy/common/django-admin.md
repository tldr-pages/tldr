# django-admin

> Django-ի օգտակար ադմինիստրատիվ առաջադրանքների համար:.
> Լրացուցիչ տեղեկություններ. <https://docs.djangoproject.com/en/stable/ref/django-admin/>:.

- Ստեղծեք նոր Django նախագիծ.:

`django-admin startproject {{project_name}}`

- Ստեղծեք նոր հավելված ընթացիկ նախագծի համար.:

`django-admin startapp {{app_name}}`

- Ցուցադրել օգնություն կոնկրետ հրամանի համար.:

`django-admin help {{command}}`

- Ցուցադրման տարբերակը:

`django-admin --version`
