# sgpt

> Արտադրողականության գործիք, որն աշխատում է OpenAI-ի GPT մոդելներով:.
> Լրացուցիչ տեղեկություններ. <https://github.com/TheR1D/shell_gpt#readme>:.

- Օգտագործեք այն որպես որոնման համակարգ՝ խնդրելով արևի զանգվածը.:

`sgpt "{{mass of the sun}}"`

- Կատարեք Shell հրամանները և կիրառեք `chmod 444` ընթացիկ գրացուցակի բոլոր ֆայլերը.:

`sgpt --shell "{{make all files in current directory read only}}"`

- Ստեղծեք կոդ՝ լուծելով դասական բզզոցի խնդիրը.:

`sgpt --code "{{solve fizz buzz problem using Python}}"`

- Սկսեք զրույցի նստաշրջան եզակի նստաշրջանի անունով.:

`sgpt --chat {{session_name}} "{{please remember my favorite number: 4}}"`

- Սկսեք `REPL` (Read-eval-print loop) նիստ.:

`sgpt --repl {{command}}`

- Ցուցադրել օգնությունը.:

`sgpt --help`
