# git հավատարմագիր

> Առբերեք և պահեք օգտվողի հավատարմագրերը:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-credential>:.

- Ցուցադրել հավատարմագրային տեղեկությունները, մուտքագրելով օգտվողի անունը և գաղտնաբառը կազմաձևման ֆայլերից.:

`echo "{{url=http://example.com}}" | git credential fill`

- Ուղարկեք հավատարմագրային տեղեկությունները բոլոր կազմաձևված հավատարմագրային օգնականներին՝ հետագայում օգտագործման համար պահելու համար.:

`echo "{{url=http://example.com}}" | git credential approve`

- Ջնջել նշված հավատարմագրային տեղեկատվությունը բոլոր կազմաձևված հավատարմագրային օգնականներից.:

`echo "{{url=http://example.com}}" | git credential reject`
