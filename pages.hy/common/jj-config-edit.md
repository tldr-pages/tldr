# jj կազմաձևման խմբագրում

> Սկսեք խմբագրիչը կազմաձևման ֆայլի վրա:.
> Տես նաև՝ `jj config path`:.
> Լրացուցիչ տեղեկություններ. <https://docs.jj-vcs.dev/latest/cli-reference/#jj-config-edit>:.

- Սկսեք խմբագիր օգտագործողի մակարդակի կազմաձևման ֆայլում.:

`jj config {{[e|edit]}} --user`

- Սկսեք խմբագիր ռեպո մակարդակի կազմաձևման ֆայլում.:

`jj config {{[e|edit]}} --repo`

- Սկսեք խմբագիր աշխատանքային տարածքի մակարդակի կազմաձևման ֆայլում.:

`jj config {{[e|edit]}} --workspace`
