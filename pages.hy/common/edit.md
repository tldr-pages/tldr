# խմբագրել

> Microsoft-ի տերմինալների վրա հիմնված տեքստային խմբագիր:.
> Լրացուցիչ տեղեկություններ. <https://github.com/microsoft/edit>:.

- Բացեք ֆայլ.:

`edit {{path/to/file}}`

- Որոնել օրինակ.:

`<Ctrl f>{{pattern}}<Enter>`

- Գտեք և փոխարինեք օրինակ.:

`<Ctrl r>{{pattern}}<Enter>`

- Պատճենել տեքստը.:

`<Ctrl c>`

- Կպցնել տեքստը.:

`<Ctrl v>`

- Բացեք Ֆայլի ընտրացանկը.:

`<Alt f>`

- Դուրս գալ.:

`<Ctrl q>`
