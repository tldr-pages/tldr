# bmptopnm

> Փոխակերպեք BMP ֆայլը PBM, PGM կամ PNM պատկերի:.
> Լրացուցիչ տեղեկություններ. <https://netpbm.sourceforge.net/doc/bmptopnm.html>:.

- Ստեղծեք PBM, PGM կամ PNM պատկերը որպես ելք, Windows կամ OS/2 BMP ֆայլի համար որպես մուտքագրում.:

`bmptopnm {{path/to/file.bmp}}`

- Հաղորդեք BMP վերնագրի բովանդակությունը `stderr` հասցեին՝:

`bmptopnm {{[-verb|-verbose]}} {{path/to/file.bmp}}`

- Ցուցադրման տարբերակը:

`bmptopnm {{[-v|-version]}}`
