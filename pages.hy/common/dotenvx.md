# dotenvx

> Ավելի լավ `dotenv`՝ `dotenv`-ի ստեղծողից:.
> Լրացուցիչ տեղեկություններ. <https://dotenvx.com/docs/>:.

- Գործարկեք հրաման շրջակա միջավայրի փոփոխականներով `.env` ֆայլից.:

`dotenvx run -- {{command}}`

- Գործարկեք հրաման շրջակա միջավայրի փոփոխականներով կոնկրետ `.env` ֆայլից.:

`dotenvx run -f {{path/to/file.env}} -- {{command}}`

- Սահմանեք շրջակա միջավայրի փոփոխական գաղտնագրմամբ.:

`dotenvx set {{key}} {{value}}`

- Սահմանեք շրջակա միջավայրի փոփոխական առանց կոդավորման.:

`dotenvx set {{key}} {{value}} --plain`

- Վերադարձրեք `.env` ֆայլում սահմանված միջավայրի փոփոխականները.:

`dotenvx get`

- Վերադարձրեք `.env` ֆայլում սահմանված միջավայրի փոփոխականի արժեքը.:

`dotenvx get {{key}}`

- Վերադարձրեք շրջակա միջավայրի բոլոր փոփոխականները `.env` ֆայլերից և ՕՀ-ից:

`dotenvx get --all`
