#գնա թեստ

> Test Go փաթեթներ (ֆայլերը պետք է ավարտվեն `_test.go`-ով):.
> Նշում. `./...`-ը Go փաթեթի օրինաչափություն է, որը հասկանալի է Go tooling-ով: Այն համապատասխանում է ընթացիկ փաթեթին և բոլոր փաթեթներին ռեկուրսիվորեն ընթացիկ գրացուցակի տակ:.
> Լրացուցիչ տեղեկություններ. <https://pkg.go.dev/cmd/go#hdr-Testing_flags>:.

- Փորձարկեք ընթացիկ գրացուցակում հայտնաբերված փաթեթը.:

`go test`

- [v]խստորեն փորձարկեք փաթեթը ընթացիկ գրացուցակում.:

`go test -v`

- Փորձարկեք փաթեթները ընթացիկ գրացուցակում և բոլոր ենթագրքերում.:

`go test -v ./...`

- Փորձարկեք փաթեթը ընթացիկ գրացուցակում և գործարկեք բոլոր հենանիշները.:

`go test -v -bench .`

- Փորձարկեք փաթեթը ընթացիկ գրացուցակում և գործարկեք բոլոր հենանիշերը 50 վայրկյան.:

`go test -v -bench . -benchtime 50s`

- Փորձարկեք փաթեթը ծածկույթի վերլուծությամբ.:

`go test -cover`
