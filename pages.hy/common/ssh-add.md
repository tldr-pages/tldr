# ssh-ավելացնել

> Կառավարեք բեռնված SSH ստեղները `ssh-agent`-ում:.
> Համոզվեք, որ `ssh-agent`-ը գործարկված է և աշխատում է, որպեսզի ստեղները բեռնվեն դրանում:.
> Լրացուցիչ տեղեկություններ. <https://man.openbsd.org/ssh-add>:.

- Ավելացրեք կանխադրված SSH ստեղները `~/.ssh`-ում ssh-գործակալին.:

`ssh-add`

- Ավելացրեք հատուկ բանալի `ssh-agent`-ին.:

`ssh-add {{path/to/private_key}}`

- Թվարկեք ներկայումս բեռնված ստեղների մատնահետքերը.:

`ssh-add -l`

- Ջնջել բանալին `ssh-agent`-ից:

`ssh-add -d {{path/to/private_key}}`

- Ջնջել բոլոր ներկայումս բեռնված բանալիները `ssh-agent`-ից:

`ssh-add -D`

- Ավելացրեք բանալի `ssh-agent`-ին և բանալիների շղթային.:

`ssh-add -K {{path/to/private_key}}`
