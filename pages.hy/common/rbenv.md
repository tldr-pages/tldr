#ռբենվ

> Հեշտությամբ տեղադրեք Ruby-ի տարբերակները և կառավարեք հավելվածների միջավայրերը:.
> Տես նաև՝ `asdf`:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/rbenv>:.

- Տեղադրեք Ruby տարբերակը.:

`rbenv install {{version}}`

- Ցուցադրել յուրաքանչյուր Ruby-ի վերջին կայուն տարբերակների ցանկը.:

`rbenv install --list`

- Ցուցադրել տեղադրված Ruby տարբերակների ցանկը.:

`rbenv versions`

- Օգտագործեք Ruby-ի հատուկ տարբերակ ամբողջ համակարգում.:

`rbenv global {{version}}`

- Օգտագործեք Ruby-ի հատուկ տարբերակը հավելվածի/նախագծի գրացուցակի համար.:

`rbenv local {{version}}`

- Ցուցադրել ներկայումս ընտրված Ruby տարբերակը.:

`rbenv version`

- Տեղահանեք Ruby տարբերակը.:

`rbenv uninstall {{version}}`

- Ցուցադրել Ruby-ի բոլոր տարբերակները, որոնք պարունակում են նշված գործադիրը.:

`rbenv whence {{executable}}`
