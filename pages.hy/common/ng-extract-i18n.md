# ng քաղվածք-i18n

> Քաղեք i18n հաղորդագրությունները սկզբնական կոդից:.
> Լրացուցիչ տեղեկություններ. <https://angular.dev/cli/extract-i18n>:.

- Քաղեք i18n հաղորդագրությունները.:

`ng extract-i18n`

- Արտահանեք i18n հաղորդագրությունները հատուկ ձևաչափով.:

`ng extract-i18n --format {{arb|json|xlf|...}}`

- Արտահանեք i18n հաղորդագրությունները որոշակի ելքային ֆայլի մեջ.:

`ng extract-i18n --out-file {{path/to/file}}`

- Արտահանեք i18n հաղորդագրությունները որոշակի գրացուցակում.:

`ng extract-i18n --output-path {{path/to/directory}}`

- Վերահսկեք, թե ինչպես են մշակվում կրկնօրինակ թարգմանությունները.:

`ng extract-i18n --i18n-duplicate-translation {{error|ignore|warning}}`

- Մուտքագրեք առաջընթացը դեպի վահանակ.:

`ng extract-i18n --progress`
