# մինի նշան

> Մեռած պարզ գործիք ֆայլերը ստորագրելու և ստորագրությունները ստուգելու համար:.
> Լրացուցիչ տեղեկություններ. <https://jedisct1.github.io/minisign/>:.

- Ստեղծեք նոր բանալիների զույգ լռելյայն վայրում.:

`minisign -G`

- Ստորագրեք ֆայլը.:

`minisign -Sm {{path/to/file}}`

- Ստորագրեք ֆայլ՝ ստորագրության մեջ ավելացնելով վստահելի (ստորագրված) և անվստահելի (չստորագրված) մեկնաբանություն.:

`minisign -Sm {{path/to/file}} -c "{{Untrusted comment}}" -t "{{Trusted comment}}"`

- Ստուգեք ֆայլը և վստահելի մեկնաբանությունները նրա ստորագրության մեջ՝ օգտագործելով նշված հանրային բանալի ֆայլը.:

`minisign -Vm {{path/to/file}} -p {{path/to/publickey.pub}}`

- Ստուգեք ֆայլը և վստահելի մեկնաբանությունները դրա ստորագրության մեջ՝ նշելով հանրային բանալին որպես Base64 կոդավորված բառացի.:

`minisign -Vm {{path/to/file}} -P "{{public_key_base64}}"`
