# fls

> Թվարկեք ֆայլերը և գրացուցակները պատկերային ֆայլում կամ սարքում:.
> Լրացուցիչ տեղեկություններ. <https://wiki.sleuthkit.org/index.php?title=Fls>:.

- Սարքի վրա կառուցեք ռեկուրսիվ fls ցուցակ, ելքային ուղիները կսկսվեն C-ով:

`fls -r -m {{C:}} {{/dev/loop1p1}}`

- Վերլուծեք մեկ բաժանմունք՝ ապահովելով այն հատվածի օֆսեթը, որով սկսվում է ֆայլային համակարգը նկարում.:

`fls -r -m {{C:}} -o {{sector}} {{path/to/image_file}}`

- Վերլուծեք մեկ բաժանմունք՝ տրամադրելով սկզբնական համակարգի ժամային գոտին.:

`fls -r -m {{C:}} -z {{timezone}} {{/dev/loop1p1}}`
