# ընդ

> Ինտերֆեյս VirusTotal-ի համար:.
> Այս հրամանի համար պահանջվում է VirusTotal հաշվի API բանալի:.
> Լրացուցիչ տեղեկություններ. <https://github.com/VirusTotal/vt-cli>:.

- Սկանավորեք հատուկ ֆայլ վիրուսների համար.:

`vt scan file {{path/to/file}}`

- Սկանավորեք URL վիրուսների համար.:

`vt scan url {{url}}`

- Ցուցադրել տեղեկատվություն կոնկրետ վերլուծությունից.:

`vt analysis {{file_id|analysis_id}}`

- Ներբեռնեք ֆայլերը կոդավորված Zip ձևաչափով (պահանջվում է պրեմիում հաշիվ).:

`vt download {{file_id}} --output {{path/to/directory}} --zip --zip-password {{password}}`

- Նախաստորագրեք կամ նորից սկզբնավորեք `vt`-ը՝ API բանալին ինտերակտիվ կերպով մուտքագրելու համար.:

`vt init`

- Ցուցադրել տեղեկատվություն տիրույթի մասին.:

`vt domain {{url}}`

- Ցուցադրել տեղեկատվություն կոնկրետ URL-ի համար.:

`vt url {{url}}`

- Ցուցադրել տեղեկատվություն որոշակի IP հասցեի համար.:

`vt domain {{ip_address}}`
