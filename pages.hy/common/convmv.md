# convmv

> Փոխարկել ֆայլերի անունները (ՈՉ ֆայլի բովանդակությունը) մեկ կոդավորումից մյուսը:.
> Լրացուցիչ տեղեկություններ. <https://www.j3e.de/linux/convmv/man/>:.

- Փորձարկեք ֆայլի անվան կոդավորման փոխարկումը (իրականում մի փոխեք ֆայլի անունը).:

`convmv -f {{from_encoding}} -t {{to_encoding}} {{input_file}}`

- Փոխակերպեք ֆայլի անվան կոդավորումը և վերանվանեք ֆայլը նոր կոդավորման.:

`convmv -f {{from_encoding}} -t {{to_encoding}} --notest {{input_file}}`
