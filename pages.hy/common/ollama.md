#օլամա

> Լեզուների մեծ մոդելի վազորդ:.
> Հասանելի մոդելների ցանկի համար տես <https://ollama.com/library>:.
> Լրացուցիչ տեղեկություններ. <https://docs.ollama.com/cli>:.

- Սկսեք այլ հրամաններ գործարկելու համար պահանջվող դեյմոնը.:

`ollama serve`

- Գործարկեք մոդելը և զրուցեք դրա հետ (ավտոմատ կերպով կներբեռնվի մոդելը, եթե այն ներբեռնված չէ):

`ollama run {{model}}`

- Գործարկեք մոդել մեկ հուշումով և մտածողությամբ անջատված.:

`ollama run {{model}} --think=false "{{prompt}}"`

- Ներբեռնված մոդելների ցուցակ.:

`ollama {{[ls|list]}}`

- Քաշեք կոնկրետ մոդել.:

`ollama pull {{model}}`

- Գործող մոդելների ցուցակ.:

`ollama ps`

- Ջնջել մոդելը.:

`ollama rm {{model}}`

- Ստեղծեք մոդել `Modelfile`-ից.:

`ollama create {{new_model_name}} {{[-f|--file]}} {{path/to/Modelfile}}`
