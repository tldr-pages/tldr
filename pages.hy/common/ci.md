# ci

> Ստուգեք RCS-ի վերանայումները (պահեք ֆայլի փոփոխությունները Revision Control System-ում):.
> Տես նաև՝ `co`, `rcs`, `rcsdiff`, `rlog`:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/ci>:.

- Ստուգեք ֆայլը և պահեք աշխատանքային ֆայլը ապակողպված.:

`ci -u {{path/to/file}}`

- Ստուգեք ֆայլը և պահեք աշխատանքային ֆայլը կողպված.:

`ci -l {{path/to/file}}`

- Ստուգեք ֆայլի մեջ հատուկ տեղեկամատյան հաղորդագրությամբ.:

`ci -m"{{log_message}}" {{path/to/file}}`

- Ստուգեք ֆայլը՝ բացելով այն, բայց թողնելով աշխատանքային ֆայլը միայն կարդալու համար.:

`ci {{path/to/file}}`

- Հարկադիր գրանցում, նույնիսկ եթե փոփոխություններ չկան.:

`ci -f -u {{path/to/file}}`
