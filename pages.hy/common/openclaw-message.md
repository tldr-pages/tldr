# openclaw հաղորդագրություն

> Ուղարկեք հաղորդագրություններ OpenClaw ալիքներով:.
> Լրացուցիչ տեղեկություններ. <https://docs.openclaw.ai/cli/message>:.

- Ուղարկեք հաղորդագրություն հեռախոսահամարին (WhatsApp).:

`openclaw message send {{[-t|--target]}} {{+1234567890}} {{[-m|--message]}} "{{Hello}}"`

- Ուղարկեք հաղորդագրություն կոնկրետ ալիքին.:

`openclaw message send --channel {{telegram}} {{[-t|--target]}} {{user_id}} {{[-m|--message]}} "{{Hello}}"`

- Ցուցադրել օգնությունը.:

`openclaw message {{[-h|--help]}}`
