# git count

> Տպել պարտավորությունների ընդհանուր թիվը:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-count>:.

- Տպեք պարտավորությունների ընդհանուր քանակը.:

`git count`

- Տպեք յուրաքանչյուր մասնակցի պարտավորությունների քանակը և պարտավորությունների ընդհանուր քանակը.:

`git count --all`
