# npm ռեպո

> Բացեք բրաուզերում փաթեթի պահեստի էջը:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-repo/>:.

- Բացեք ընթացիկ նախագծի պահեստային էջը (հիմնվելով `package.json`-ի վրա).:

`npm repo`

- Բացեք ռեեստրից որոշակի փաթեթի պահեստային էջը.:

`npm repo {{package_name}}`

- Բացեք պահեստային էջերը բազմաթիվ փաթեթների համար.:

`npm repo {{package_name1 package_name2 ...}}`

- Տպեք պահեստի URL-ը զննարկիչում բացելու փոխարեն.:

`npm repo --browser false`

- Բացեք պահեստի էջը որոշակի բրաուզերում փաթեթի համար.:

`npm repo --browser {{browser_name}}`

- Բացեք փաթեթի պահեստային էջը որոշակի աշխատանքային տարածքում.:

`npm repo --workspace {{workspace_name}}`
