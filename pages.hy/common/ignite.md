#բոցավառվել

> CLI React Native կաթսաների, պլագինների, գեներատորների և այլնի համար:.
> Լրացուցիչ տեղեկություններ. <https://infinite.red/ignite>:.

- Ստեղծեք նոր React Native նախագիծ.:

`ignite new {{project_name}}`

- Ստեղծեք ֆայլ plugin-ից.:

`ignite generate {{plugin_name}} {{path/to/file}}`

- Նախագծին ավելացրեք Ignite հավելված.:

`ignite add {{plugin_name}}`

- Հեռացրեք Ignite հավելվածը նախագծից.:

`ignite remove {{plugin_name}}`
