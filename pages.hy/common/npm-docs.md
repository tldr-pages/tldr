# npm փաստաթղթեր

> Բացեք փաստաթղթերը մեկ կամ մի քանի փաթեթների համար կանխադրված վեբ դիտարկիչում:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-docs/>:.

- Բացեք որոշակի փաթեթի փաստաթղթերի էջը.:

`npm docs {{package}}`

- Բացեք փաստաթղթերը միանգամից մի քանի փաթեթների համար.:

`npm docs {{package1 package2 ...}}`
