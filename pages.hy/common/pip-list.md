# առաջարկների ցուցակ

> Ցուցակեք տեղադրված Python փաթեթները:.
> Լրացուցիչ տեղեկություններ. <https://pip.pypa.io/en/stable/cli/pip_list/>:.

- Տեղադրված փաթեթների ցանկ.:

`pip list`

- Թվարկեք հնացած փաթեթները, որոնք կարող են թարմացվել.:

`pip list {{[-o|--outdated]}}`

- Թվարկեք արդի փաթեթները.:

`pip list {{[-u|--uptodate]}}`

- Ցուցակեք JSON ձևաչափով փաթեթները.:

`pip list --format json`

- Թվարկեք փաթեթները, որոնք այլ փաթեթների կողմից չեն պահանջվում.:

`pip list --not-required`

- Ցուցակեք միայն օգտագործողի կայքում տեղադրված փաթեթները.:

`pip list --user`

- Թվարկեք փաթեթները և բացառեք խմբագրվող փաթեթները արտադրանքից.:

`pip list --exclude-editable`

- Ցուցակեք փաթեթները սառեցված ձևաչափով (ի տարբերություն `pip freeze`-ի, չի ցուցադրում խմբագրվող տեղադրման տեղեկատվությունը).:

`pip list --format freeze`
