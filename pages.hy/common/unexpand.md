# չընդլայնել

> Փոխարկեք բացատները ներդիրների:.
> Լրացուցիչ տեղեկություններ. <https://www.gnu.org/software/coreutils/manual/html_node/unexpand-invocation.html>:.

- Յուրաքանչյուր ֆայլի բացերը փոխարկեք ներդիրների՝ գրելով `stdout`:

`unexpand {{path/to/file}}`

- Դատարկները փոխարկեք ներդիրների՝ կարդալով `stdout`-ից:

`unexpand`

- Փոխակերպեք բոլոր դատարկությունները, պարզապես սկզբնական դատարկությունների փոխարեն.:

`unexpand {{[-a|--all]}} {{path/to/file}}`

- Փոխակերպեք բացթողումների միայն առաջատար հաջորդականությունները (փոխանցում է -a):

`unexpand --first-only {{path/to/file}}`

- Ունեն ներդիրներ որոշակի թվով նիշերի միմյանցից տարբեր, ոչ թե 8 (միացնում է -a):

`unexpand {{[-t|--tabs]}} {{number}} {{path/to/file}}`
