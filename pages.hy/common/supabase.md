# subbase

> Մշակել, տեղակայել և կառավարել Supabase նախագծերը տեղական և ամպի վրա:.
> Լրացուցիչ տեղեկություններ. <https://supabase.com/docs/reference/cli>:.

- Մուտք գործեք Supabase՝ օգտագործելով մուտքի նշան.:

`supabase login`

- Նախաձեռնեք նոր Supabase նախագիծը ընթացիկ գրացուցակում.:

`supabase init`

- Սկսեք տեղական Supabase-ի զարգացման փաթեթը.:

`supabase start`

- Դադարեցրեք տեղական Supabase-ի զարգացման կույտը.:

`supabase stop`

- Ցուցադրել տեղական Supabase-ի զարգացման կույտի կարգավիճակը.:

`supabase status`

- Կապեք ընթացիկ գրացուցակը հեռավոր Supabase նախագծին.:

`supabase link --project-ref {{project_id}}`

- Թվարկեք բոլոր Supabase նախագծերը.:

`supabase projects list`

- Ստեղծեք ավտոմատ լրացման սցենար հատուկ կեղևի համար.:

`supabase completion {{bash|fish|zsh|...}}`
