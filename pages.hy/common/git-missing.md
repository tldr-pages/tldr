# git բացակայում է

> Ցույց տալ պարտավորությունները, որոնք չեն կիսվում երկու ճյուղերի միջև:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-missing>:.

- Ցույց տալ պարտավորությունները, որոնք չեն համօգտագործվում ներկայումս դուրս գրված մասնաճյուղի և մեկ այլ մասնաճյուղի միջև.:

`git missing {{branch}}`

- Ցույց տալ պարտավորությունները, որոնք չեն կիսվում երկու ճյուղերի միջև.:

`git missing {{branch_1}} {{branch_2}}`
