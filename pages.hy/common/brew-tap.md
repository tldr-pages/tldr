# եփել թակել

> Հպեք Homebrew բանաձևի պահեստին: Եթե ​​արգումենտներ չկան, նշեք Homebrew-ի բոլոր տեղադրված ծորակները:.
> Լրացուցիչ տեղեկություններ. <https://docs.brew.sh/Manpage#tap-options-userrepo-url>:.

- Տեղադրված Homebrew ծորակների ցուցակ.:

`brew tap`

- Հպեք Git պահեստին, որը տեղակայված է GitHub-ում.:

`brew tap {{github_username}}/{{github_repository_name}}`

- Հպեք Git պահեստին, որը տեղակայված է GitHub-ից դուրս.:

`brew tap {{username}}/{{repository_name}} {{git_clone_url}}`

- Հպեք GitLab-ում տեղակայված պահեստին.:

`brew tap {{username}}/{{repository_name}} https://gitlab.com/{{username}}/{{repository_name}}.git`

- Ցուցադրել օգնությունը.:

`brew tap {{[-h|--help]}}`
