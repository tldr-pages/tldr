# dolt կոնֆիգուրացիա

> Կարդացեք և գրեք տեղական (ըստ պահեստի) և գլոբալ (մեկ օգտվողի համար) Dolt-ի կազմաձևման փոփոխականները:.
> Լրացուցիչ տեղեկություններ. <https://docs.dolthub.com/cli-reference/cli#dolt-config>:.

- Թվարկեք բոլոր տեղական և գլոբալ կազմաձևման տարբերակները և դրանց արժեքները.:

`dolt config --list`

- Ցուցադրել տեղական կամ գլոբալ կազմաձևման փոփոխականի արժեքը.:

`dolt config --get {{name}}`

- Փոփոխեք տեղական կազմաձևման փոփոխականի արժեքը՝ ստեղծելով այն, եթե այն գոյություն չունի.:

`dolt config --add {{name}} {{value}}`

- Փոփոխեք գլոբալ կազմաձևման փոփոխականի արժեքը՝ ստեղծելով այն, եթե այն գոյություն չունի.:

`dolt config --global --add {{name}} {{value}}`

- Ջնջել տեղական կազմաձևման փոփոխականը.:

`dolt config --unset {{name}}`

- Ջնջել գլոբալ կազմաձևման փոփոխականը.:

`dolt config --global --unset {{name}}`
