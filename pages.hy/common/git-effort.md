# git ջանք

> Ցուցադրել ֆայլի ակտիվության ծավալը՝ ցույց տալով յուրաքանչյուր ֆայլի պարտավորությունները և «ակտիվ օրերը», այսինքն՝ ֆայլին նպաստած օրերի ընդհանուր թիվը:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-efort>:.

- Ցուցադրել յուրաքանչյուր ֆայլ պահեստում՝ ցույց տալով պարտավորություններ և ակտիվ օրեր.:

`git effort`

- Ցուցադրել ֆայլեր, որոնք փոփոխվել են որոշակի թվով կամ ավելի ստանձնումների կողմից, ցույց տալով պարտավորություններ և ակտիվ օրեր.:

`git effort --above {{5}}`

- Ցուցադրել որոշակի հեղինակի կողմից փոփոխված ֆայլեր՝ ցույց տալով պարտավորություններ և ակտիվ օրեր.:

`git effort -- --author="{{username}}"`

- Ցուցադրել որոշակի ժամանակից/ամսաթվից փոփոխված ֆայլերը՝ ցույց տալով պարտավորություններ և ակտիվ օրեր.:

`git effort -- --since="{{last month}}"`

- Ցուցադրել միայն նշված ֆայլերը կամ գրացուցակները՝ ցույց տալով պարտավորություններ և ակտիվ օրեր.:

`git effort {{path/to/file_or_directory1 path/to/file_or_directory2 ...}}`

- Ցուցադրել բոլոր ֆայլերը որոշակի գրացուցակում, ցույց տալով պարտավորություններ և ակտիվ օրեր.:

`git effort {{path/to/directory/*}}`
