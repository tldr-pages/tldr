# bun init

> Փայտամած դատարկ փունջ նախագիծ:.
> Լրացուցիչ տեղեկություններ. <https://bun.com/docs/runtime/templating/init>:.

- Նախաձեռնեք Bun նախագիծը ընթացիկ գրացուցակում.:

`bun init`

- Նախաձեռնել կոնկրետ գրացուցակում.:

`bun init {{path/to/directory}}`

- Նախաձեռնել կոնկրետ նախագծի անունով.:

`bun init {{[-n|--name]}} {{project_name}}`

- Նախաձեռնեք նախագիծը լռելյայն հուշումներով.:

`bun init {{[-y|--yes]}}`

- Նախաձեռնեք նվազագույն նախագիծ.:

`bun init {{[-m|--minimal]}}`

- Նախաձեռնեք React նախագիծը.:

`bun init {{[-r|--react]}}`

- Նախաձեռնեք React նախագիծը Tailwind CSS-ով.:

`bun init --react=tailwind`
