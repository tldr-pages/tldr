#գիտա

> Կառավարեք բազմաթիվ Git պահոցներ կողք կողքի:.
> Տես նաև՝ `git`:.
> Լրացուցիչ տեղեկություններ. <https://github.com/nosarthur/gita>:.

- Ցուցադրել [ll] գրանցված պահեստների ամփոփագիրը.:

`gita ll`

- Ցույց տալ բոլոր գրանցված պահեստների [st]atus-ը.:

`gita st`

- Գրանցեք մեկ կամ մի քանի պահեստ gita-ով.:

`gita add {{path/to/repo1 path/to/repo2 ...}}`

- Քաշեք թարմացումները բոլոր գրանցված պահեստների համար.:

`gita pull`

- Գործարկեք Git հրամանը կոնկրետ պահոցում.:

`gita super {{repo_name}} {{git_command}}`

- Ստեղծեք պահեստների խումբ և սահմանեք այն որպես ակտիվ համատեքստ.:

`gita group add {{[-n|--name]}} {{group_name}} {{repo1 repo2 ...}} && gita context {{group_name}}`

- Ստացեք թարմացումներ մեկ կամ մի քանի պահեստների կամ խմբերի համար.:

`gita fetch {{repo1 repo2 ...|group1 group2 ...}}`

- Ցուցադրել բոլոր պահեստների տեղեկատվությունը.:

`gita freeze`
