# gt

> Ստեղծեք և կառավարեք Git-ի և GitHub-ի համար կախված կոդերի փոփոխությունների (կույտերի) հաջորդականությունները:.
> Լրացուցիչ տեղեկություններ. <https://graphite.com/docs/get-started>:.

- Նախաձեռնեք `gt` պահոցը ընթացիկ գրացուցակում.:

`gt init`

- Ստեղծեք նոր մասնաճյուղ, որը դրված է ընթացիկ մասնաճյուղի վերևում և կատարեք փուլային փոփոխություններ.:

`gt create {{branch_name}}`

- Ստեղծեք նոր հանձնառություն և շտկեք վերին ճյուղերը.:

`gt modify -cam {{commit_message}}`

- Ընթացիկ կույտի բոլոր ճյուղերը ստիպեք մղել GitHub և ստեղծել կամ թարմացնել PR-ները.:

`gt stack submit`

- Ստուգեք տարբեր մասնաճյուղեր (հուշում է ինտերակտիվ ռեժիմ, երբ մասնաճյուղի անունը բաց է թողնվում).:

`gt co {{branch_name}}`

- Համաժամեցրեք կույտը հեռավոր տարբերակի հետ (նաև ջնջում է միավորված ճյուղերը).:

`gt sync`

- Գրանցեք բոլոր հետևված կույտերը.:

`gt log short`

- Ցուցադրել օգնությունը նշված ենթահրամանի համար.:

`gt {{subcommand}} --help`
