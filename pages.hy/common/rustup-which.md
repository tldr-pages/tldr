# rustup որը

> Ցուցադրել, թե որ երկուական տարբերակն է գործարկվելու `rustup`-ով կառավարվող հրամանի համար:.
> Հավանել `which`-ը, բայց որոնում է Rust գործիքների շղթան `$PATH`-ի փոխարեն:.
> Լրացուցիչ տեղեկություններ. <https://rust-lang.github.io/rustup/>:.

- Ցուցադրել երկուականի ուղին լռելյայն գործիքաշարում.:

`rustup which {{command}}`

- Ցուցադրեք երկուականի ուղին նշված գործիքների շղթայում (տե՛ս `rustup help toolchain` լրացուցիչ տեղեկությունների համար).:

`rustup which --toolchain {{toolchain}} {{command}}`
