# sshpass

> SSH գաղտնաբառի մատակարար:.
> Այն աշխատում է՝ ստեղծելով TTY, գաղտնաբառ մուտքագրելով դրա մեջ, այնուհետև `stdin`-ը վերահղելով դեպի SSH նիստ:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/sshpass>:.

- Միացեք հեռավոր սերվերին, օգտագործելով գաղտնաբառ, որը տրված է ֆայլի նկարագրիչի վրա (այս դեպքում՝ `stdin`):

`sshpass -d {{0}} ssh {{user}}@{{hostname}}`

- Միացեք հեռավոր սերվերին որպես տարբերակ տրամադրված գաղտնաբառով և ավտոմատ կերպով ընդունեք անհայտ SSH ստեղները.:

`sshpass -p {{password}} ssh -o StrictHostKeyChecking=no {{user}}@{{hostname}}`

- Միացեք հեռավոր սերվերին, օգտագործելով ֆայլի առաջին տողը որպես գաղտնաբառ, ավտոմատ կերպով ընդունեք անհայտ SSH ստեղները և գործարկեք հրաման.:

`sshpass -f {{path/to/file}} ssh -o StrictHostKeyChecking=no {{user}}@{{hostname}} "{{command}}"`
