# sqlite3

> Ինտերֆեյս SQLite 3-ի համար, որն իրենից ներկայացնում է ֆայլերի վրա հիմնված ներկառուցված SQL շարժիչ:.
> Լրացուցիչ տեղեկություններ. <https://sqlite.org/cli.html>:.

- Սկսեք ինտերակտիվ վահանակ նոր տվյալների բազայով.:

`sqlite3`

- Բացեք ինտերակտիվ վահանակ գոյություն ունեցող տվյալների բազայի դեմ.:

`sqlite3 {{path/to/database.sqlite3}}`

- Կատարեք SQL հայտարարություն տվյալների բազայի դեմ և այնուհետև դուրս եկեք.:

`sqlite3 {{path/to/database.sqlite3}} '{{SELECT * FROM some_table;}}'`
