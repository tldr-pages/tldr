# գնալ ցուցակ

> Ցուցակեք փաթեթները կամ մոդուլները:.
> Նշում. `./...`-ը Go փաթեթի օրինաչափություն է, որը հասկանալի է Go tooling-ով: Այն համապատասխանում է ընթացիկ փաթեթին և բոլոր փաթեթներին ռեկուրսիվորեն ընթացիկ գրացուցակի տակ:.
> Լրացուցիչ տեղեկություններ. <https://pkg.go.dev/cmd/go#hdr-List_packages_or_modules>:.

- Ցուցակ փաթեթներ.:

`go list ./...`

- Նշեք ստանդարտ փաթեթներ.:

`go list std`

- Ցուցակեք փաթեթները JSON ձևաչափով.:

`go list -json time net/http`

- Ցուցակեք մոդուլի կախվածությունները և հասանելի թարմացումները.:

`go list -m -u all`
