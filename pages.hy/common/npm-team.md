# npm թիմ

> Կառավարեք թիմերը կազմակերպությունում `npm` գրանցամատյանում:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-team/>:.

- Ավելացրեք օգտվողին կազմակերպության թիմում.:

`npm team add {{organization:team}} {{username}}`

- Հեռացրեք օգտվողին թիմից.:

`npm team rm {{organization:team}} {{username}}`

- Ստեղծեք նոր թիմ կազմակերպությունում.:

`npm team create {{organization:team}}`

- Ջնջել թիմը կազմակերպությունից.:

`npm team destroy {{organization:team}}`

- Նշեք կազմակերպության բոլոր թիմերը.:

`npm team ls {{organization}}`

- Նշեք բոլոր օգտվողներին հատուկ թիմում.:

`npm team ls {{organization:team}}`
