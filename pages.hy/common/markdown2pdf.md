# markdown2pdf

> Փոխարկել նշումը PDF-ի:.
> Լրացուցիչ տեղեկություններ. <https://github.com/theiskaa/markdown2pdf>:.

- Փոխակերպեք Markdown ֆայլը PDF-ի.:

`markdown2pdf {{[-p|--path]}} {{path/to/input_file.md}}`

- Փոխակերպեք Markdown ֆայլը PDF-ի հատուկ ճանապարհով.:

`markdown2pdf {{[-p|--path]}} {{path/to/input_file.md}} {{[-o|--output]}} {{path/to/output_file.pdf}}`

- Փոխակերպեք Markdown-ի բովանդակությունը որպես տող.:

`markdown2pdf {{[-s|--string]}} {{markdown_text}} {{[-o|--output]}} {{path/to/output_file.pdf}}`

- Փոխակերպեք URL-ից (սա այդ URL-ում Markdown ֆայլը կվերածի տեղական PDF ֆայլի).:

`markdown2pdf {{[-u|--url]}} {{url}} {{[-o|--output]}} {{path/to/output_file.pdf}}`
