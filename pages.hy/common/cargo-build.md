# բեռների կառուցում

> Կազմեք տեղական փաթեթ և դրա բոլոր կախվածությունները:.
> Լրացուցիչ տեղեկություններ. <https://doc.rust-lang.org/cargo/commands/cargo-build.html>:.

- Կառուցեք փաթեթը կամ փաթեթները, որոնք սահմանված են `Cargo.toml` մանիֆեստի ֆայլով տեղական ճանապարհով.:

`cargo {{[b|build]}}`

- Ստեղծեք արտեֆակտներ թողարկման ռեժիմում, օպտիմալացումներով.:

`cargo {{[b|build]}} {{[-r|--release]}}`

- Պահանջեք, որ `Cargo.lock`-ը արդիական լինի՝:

`cargo {{[b|build]}} --locked`

- Ստեղծեք բոլոր փաթեթները աշխատանքային տարածքում.:

`cargo {{[b|build]}} --workspace`

- Կառուցեք հատուկ փաթեթ.:

`cargo {{[b|build]}} {{[-p|--package]}} {{package}}`

- Կառուցեք միայն նշված երկուականը.:

`cargo {{[b|build]}} --bin {{name}}`

- Կառուցեք միայն նշված թեստային թիրախը.:

`cargo {{[b|build]}} --test {{test_name}}`
