#մոնոլիտ

> Պահպանեք վեբ էջը որպես մեկ HTML ֆայլ:.
> Լրացուցիչ տեղեկություններ. <https://github.com/Y2Z/monolith>:.

- Պահպանեք վեբ էջը որպես մեկ HTML ֆայլ.:

`monolith {{url}}`

- Պահպանեք վեբ էջը որպես մեկ HTML ֆայլ, բացառությամբ աուդիո.:

`monolith {{url}} {{[-a|--no-audio]}}`

- Պահպանեք վեբ էջը որպես մեկ HTML ֆայլ, բացառությամբ CSS-ի.:

`monolith {{url}} {{[-c|--no-css]}}`

- Պահպանեք վեբ էջը որպես մեկ HTML ֆայլ, բացառությամբ պատկերների.:

`monolith {{url}} {{[-i|--no-images]}}`

- Պահպանեք վեբ էջը որպես մեկ HTML ֆայլ, բացառությամբ տեսանյութերի.:

`monolith {{url}} {{[-v|--no-video]}}`

- Պահպանեք վեբ էջը որպես մեկ HTML ֆայլ, բացառությամբ JavaScript-ի.:

`monolith {{url}} {{[-j|--no-js]}}`

- Պահպանեք վեբ էջը որպես մեկ HTML ֆայլ՝ ընդունելով անվավեր TLS վկայականներ.:

`monolith {{url}} {{[-k|--insecure]}}`

- Պահպանեք վեբ էջը որպես մեկ HTML ֆայլ՝ նշելով որոշակի ելքային ֆայլ.:

`monolith {{url}} {{[-o|--output]}} {{path/to/file.html}}`
