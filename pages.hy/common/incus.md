# incus

> Ժամանակակից, անվտանգ և հզոր համակարգի կոնտեյների և վիրտուալ մեքենայի կառավարիչ:.
> Լրացուցիչ տեղեկություններ. <https://linuxcontainers.org/incus/docs/main/>:.

- Թվարկեք բոլոր բեռնարկղերը և վիրտուալ մեքենաները (ինչպես աշխատող, այնպես էլ դադարեցված).:

`incus list`

- Ստեղծեք կոնտեյներ պատկերից, հատուկ անունով.:

`incus create {{image}} {{container_name}}`

- Սկսել կամ դադարեցնել գոյություն ունեցող կոնտեյները.:

`incus {{start|stop}} {{container_name}}`

- Բացեք պատյան արդեն հոսող տարայի մեջ.:

`incus shell {{container_name}}`

- Հեռացրեք կանգնեցված տարան.:

`incus delete {{container_name}}`

- Քաշեք պատկերը պատկերների պահոցից (հեռավոր) դեպի տեղական.:

`incus copy {{remote}}:{{image}} local:{{custom_image_name}}`

- Թվարկեք բոլոր հասանելի պատկերները պաշտոնական `images:` հեռակառավարման վահանակում.:

`incus image list images:`

- Թվարկեք բոլոր պատկերները, որոնք արդեն ներբեռնվել են `local:` հեռակառավարման վահանակում՝:

`incus image list local:`
