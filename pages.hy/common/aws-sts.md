# ավս ս

> Անվտանգության նշանների ծառայությունը (STS) թույլ է տալիս ժամանակավոր հավատարմագրեր պահանջել (IAM) օգտագործողների կամ դաշնային օգտատերերի համար:.
> Լրացուցիչ տեղեկություններ. <https://docs.aws.amazon.com/cli/latest/reference/sts/>:.

- Ստացեք ժամանակավոր անվտանգության հավատարմագրեր՝ հատուկ AWS ռեսուրսներ մուտք գործելու համար.:

`aws sts assume-role --role-arn {{aws_role_arn}}`

- Ստացեք IAM օգտվող կամ դեր, որի հավատարմագրերն օգտագործվում են գործողությունը կանչելու համար.:

`aws sts get-caller-identity`
