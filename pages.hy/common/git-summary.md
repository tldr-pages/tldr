# git ամփոփում

> Ցուցադրել տեղեկատվություն Git պահեստի մասին:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/git-summary>:.

- Ցուցադրել տվյալները Git պահեստի մասին.:

`git summary`

- Ցուցադրել տվյալները Git պահեստի մասին, քանի որ պարտավորվել է.:

`git summary {{commit|branch_name|tag_name}}`

- Ցուցադրել տվյալները Git պահեստի մասին՝ միավորելով կոմիտերները՝ օգտագործելով տարբեր էլ. նամակներ յուրաքանչյուր հեղինակի համար 1 վիճակագրության մեջ.:

`git summary --dedup-by-email`

- Ցուցադրել տվյալները Git պահեստի մասին՝ ցույց տալով յուրաքանչյուր ներդրողի կողմից փոփոխված տողերի քանակը.:

`git summary --line`
