# ռեզյումե

> Հեշտությամբ կարգավորեք նոր ռեզյումե:.
> Լրացուցիչ տեղեկություններ. <https://github.com/jsonresume/resume-cli>:.

- Ստեղծեք նոր `resume.json` ֆայլ ընթացիկ աշխատանքային գրացուցակում.:

`resume init`

- Վավերացրեք `resume.json`-ը սխեմայի թեստերի դեմ՝ համոզվելու համար, որ այն համապատասխանում է ստանդարտին.:

`resume validate`

- Արտահանեք ռեզյումեն տեղական ոճավորված HTML կամ PDF ձևաչափով.:

`resume export {{path/to/html_or_pdf}}`

- Սկսեք վեբ սերվեր, որը սպասարկում է տեղական `resume.json`:

`resume serve`
