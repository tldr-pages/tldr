# git range-diff

> Համեմատեք երկու կատարման տիրույթներ (օրինակ՝ ճյուղի երկու տարբերակ):.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-range-diff>:.

- Տարբերեք երկու անհատական պարտավորությունների փոփոխությունները.:

`git range-diff {{commit_1}}^! {{commit_2}}^!`

- Տարբերեք մեր և նրանց փոփոխությունները իրենց ընդհանուր նախահայրից, օրինակ. ինտերակտիվ վերանայումից հետո.:

`git range-diff {{theirs}}...{{ours}}`

- Տարբերեք երկու կատարման միջակայքերի փոփոխությունները, օրինակ. ստուգելու համար, թե արդյոք կոնֆլիկտները պատշաճ կերպով լուծվել են, երբ պարտավորությունները վերահիմնավորվում են `base1`-ից `base2`-ի.:

`git range-diff {{base1}}..{{rev1}} {{base2}}..{{rev2}}`
