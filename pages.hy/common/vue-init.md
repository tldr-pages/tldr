# vue init

> Vue.js շրջանակի հին նախագծի սկզբնավորման ենթահրաման:.
> Լրացուցիչ տեղեկություններ. <https://cli.vuejs.org/guide/creating-a-project.html#pulling-2-x-templates-legacy>:.

- Ստեղծեք նոր նախագիծ՝ օգտագործելով լռելյայն ձևանմուշներից մեկը.:

`vue init {{webpack|webpack-simple|browserify|browserify-simple|simple}} {{project_name}}`

- Ստեղծեք նոր նախագիծ՝ օգտագործելով տեղական ձևանմուշը.:

`vue init {{path/to/template_directory}} {{project_name}}`

- Ստեղծեք նոր նախագիծ՝ օգտագործելով GitHub-ի ձևանմուշը.:

`vue init {{username}}/{{repository}} {{project_name}}`
