# hugo սերվեր

> Կառուցեք և սպասարկեք կայք Hugo-ի ներկառուցված վեբսերվերով:.
> Լրացուցիչ տեղեկություններ. <https://gohugo.io/commands/hugo_server/>:.

- Կառուցեք և սպասարկեք կայք.:

`hugo server`

- Կառուցեք և սպասարկեք կայք նշված նավահանգստի համարով.:

`hugo server {{[-p|--port]}} {{port_number}}`

- Կառուցեք և սպասարկեք կայք՝ նվազագույնի հասցնելով աջակցվող ելքային ձևաչափերը (HTML, XML և այլն).:

`hugo server --minify`

- Կառուցեք և սպասարկեք կայք արտադրական միջավայրում՝ ամբողջական վերարտադրումներով՝ նվազագույնի հասցնելով աջակցվող ձևաչափերը.:

`hugo server {{[-e|--environment]}} {{production}} --disableFastRender --minify`

- Ցուցադրել օգնությունը.:

`hugo server {{[-h|--help]}}`
