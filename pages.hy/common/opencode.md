# բաց կոդ

> AI կոդավորման գործակալ:.
> Որոշ ենթահրամաններ, ինչպիսիք են `auth`, `models`, `web` և այլն, ունեն իրենց օգտագործման փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://opencode.ai/docs/cli/>:.

- Սկսեք ինտերակտիվ TUI.:

`opencode`

- Շարունակեք ամենավերջին նիստը.:

`opencode {{[-c|--continue]}}`

- Գործարկեք opencode-ը ոչ ինտերակտիվ ռեժիմով՝ ուղղակիորեն հուշում ուղարկելով.:

`opencode run "{{message}}"`

- Օգտագործեք հատուկ մոդել և գործակալ.:

`opencode run {{[-m|--model]}} {{provider}}/{{model}} --agent {{agent_name}} "{{message}}"`

- Թվարկեք բոլոր առկա մոդելները կազմաձևված մատակարարներից.:

`opencode models`

- Կառավարեք հավատարմագրերը և մուտքը մատակարարների համար.:

`opencode auth login`

- Սկսեք անգլուխ բաց կոդով սերվեր API մուտք գործելու համար.:

`opencode serve {{[-h|--hostname]}} {{hostname}} {{[-p|--port]}} {{port}}`

- Ստեղծեք նոր գործակալ մաքսային կազմաձևով.:

`opencode agent create`
