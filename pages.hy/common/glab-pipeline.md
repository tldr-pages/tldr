# գլաբ խողովակաշար

> Ցուցակել, դիտել և գործարկել GitLab CI/CD խողովակաշարերը:.
> Լրացուցիչ տեղեկություններ. <https://gitlab.com/gitlab-org/cli/-/blob/main/docs/source/_index.md>:.

- Դիտեք ընթացիկ ճյուղի ընթացիկ խողովակաշարը.:

`glab pipeline status`

- Դիտեք գործող խողովակաշարը որոշակի ճյուղի վրա.:

`glab pipeline status --branch {{branch_name}}`

- Ստացեք խողովակաշարերի ցանկը.:

`glab pipeline list`

- Գործարկել ձեռքով խողովակաշար ընթացիկ ճյուղի վրա.:

`glab pipeline run`

- Գործարկել ձեռքով խողովակաշար կոնկրետ ճյուղի վրա.:

`glab pipeline run --branch {{branch_name}}`
