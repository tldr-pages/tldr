# openssl արագություն

> Հենանիշ OpenSSL ալգորիթմներ:.
> Լրացուցիչ տեղեկություններ. <https://docs.openssl.org/master/man1/openssl-speed/>:.

- Հենանիշային բոլոր առկա ալգորիթմները.:

`openssl speed`

- Հենանիշավորեք ամենատարածված ալգորիթմները.:

`openssl speed {{md5 sha1 sha256 aes rsa}}`

- Հենանիշային հատուկ EVP ծածկագիրը.:

`openssl speed -evp {{aes-256-cbc}}`

- Հենանիշային ալգորիթմներ որոշակի տևողության համար (վայրկյաններով).:

`openssl speed -seconds {{10}}`

- Հենանիշ՝ օգտագործելով պատի ժամացույցի ժամանակը՝ պրոցեսորի օգտագործողի ժամանակի փոխարեն.:

`openssl speed -elapsed`

- Արդյունք մեքենայական ընթերցվող ձևաչափով.:

`openssl speed -mr`

- Զուգահեռաբար նշեք բազմաթիվ ալգորիթմներ՝ օգտագործելով բոլոր առկա պրոցեսորային միջուկները.:

`openssl speed -multi $(nproc)`
