# բեռների msrv

> Կառավարեք նախագծի նվազագույն աջակցվող Rust տարբերակը (MSRV):.
> Նշում. սա ներկառուցված Cargo հրաման չէ, նախ պետք է այն տեղադրել:.
> Լրացուցիչ տեղեկություններ. <https://gribnau.dev/cargo-msrv/>:.

- Ցուցադրել կախվածությունների MSRV-ները (ինչպես նշված է դրանց `Cargo.toml`-ում):

`cargo msrv list`

- Գտեք նախագծի MSRV-ն՝ փորձելով այն կազմել տարբեր գործիքների շղթաներով.:

`cargo msrv find`

- Ցույց տալ նախագծի MSRV-ն, ինչպես նշված է `Cargo.toml`-ում:

`cargo msrv show`

- Սահմանեք MSRV-ն `Cargo.toml`-ում Rust-ի տվյալ տարբերակին.:

`cargo msrv set {{version}}`

- Ստուգեք, արդյոք MSRV-ն բավարարում է նախագիծը կազմելով Rust-ի նշված տարբերակով.:

`cargo msrv verify`
