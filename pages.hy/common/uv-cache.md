# ուլտրամանուշակագույն քեշ

> Կառավարեք `uv`-ի գլոբալ քեշի գրացուցակը:.
> Լրացուցիչ տեղեկություններ. <https://docs.astral.sh/uv/reference/cli/#uv-cache>:.

- Ցույց տալ քեշի գրացուցակի ուղին.:

`uv cache dir`

- Մաքրել ամբողջ քեշը (հեռացնում է բոլոր պահված փաթեթներն ու միջավայրերը).:

`uv cache clean`

- Մաքրել քեշը հատուկ փաթեթների համար.:

`uv cache clean {{package1 package2 ...}}`

- Քեշից կտրեք բոլոր անհասանելի առարկաները.:

`uv cache prune`

- Կտրեք քեշը օպտիմիզացված CI միջավայրերի համար, ինչպիսիք են GitHub Actions-ը.:

`uv cache prune --ci`

- Օգտագործեք հատուկ քեշի գրացուցակ.:

`uv cache clean --cache-dir {{path/to/cache}}`

- Մաքրել քեշը մանրամասն ելքով.:

`uv cache clean {{[-v|--verbose]}}`
