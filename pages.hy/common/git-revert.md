# git վերադարձ

> Ստեղծեք նոր պարտավորություններ, որոնք հակադարձում են նախկինների ազդեցությունը:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-revert>:.

- Վերադարձեք ամենավերջին պարտավորությունը.:

`git revert HEAD`

- Վերադարձեք 5-րդ վերջին պարտավորությունը.:

`git revert HEAD~4`

- Վերադարձեք որոշակի պարտավորություն.:

`git revert {{0c01a9}}`

- Վերադարձեք բազմաթիվ պարտավորություններ.:

`git revert {{branch_name~5}}..{{branch_name~2}}`

- Մի ստեղծեք նոր պարտավորություններ, պարզապես փոխեք աշխատող ծառը.:

`git revert {{[-n|--no-commit]}} {{0c01a9}}..{{9a1743}}`

- Չեղարկել Git-ի վերադարձը միաձուլման հակամարտությունից հետո.:

`git revert --abort`
