# կռունկի կոնֆիգուրացիա

> Ստացեք պատկերի կոնֆիգուրացիան:.
> Լրացուցիչ տեղեկություններ. <https://github.com/google/go-containerregistry/blob/main/cmd/crane/doc/crane_config.md>:.

- Ստացեք պատկերի կոնֆիգուրացիան.:

`crane config {{image_name}}`

- Ցուցադրել օգնությունը.:

`crane config {{[-h|--help]}}`
