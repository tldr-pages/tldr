# pixi կոնֆիգուրացիա

> Կառավարեք կազմաձևման ֆայլը:.
> Լրացուցիչ տեղեկություններ. <https://pixi.sh/latest/reference/cli/#config>:.

- Խմբագրել կազմաձևման ֆայլը.:

`pixi config edit`

- Նշեք բոլոր կոնֆիգուրացիաները.:

`pixi config list`

- Նշեք արժեք ցուցակի կազմաձևման բանալի.:

`pixi config prepend {{key}} {{value}}`

- Ավելացնել արժեք ցուցակի կազմաձևման բանալի.:

`pixi config append {{key}} {{value}}`

- Սահմանեք կազմաձևման բանալի արժեքի վրա.:

`pixi config set {{key}} {{value}}`

- Չեղարկել կազմաձևման ստեղնը՝:

`pixi config unset {{key}}`
