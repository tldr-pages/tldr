# QOwnNotes

> Նշումներ կատարելու ծրագիր:.
> Ընտրովի ինտեգրվում է Nextcloud-ի և ownCloud-ի նշումներ անելու հավելվածներին:.
> Տես նաև՝ `qc`:.
> Լրացուցիչ տեղեկություններ. <https://www.qownnotes.org/getting-started/cli-parameters.html>:.

- Գործարկել շարժական ռեժիմով.:

`QOwnNotes --portable`

- Թափել կարգավորումները և հավելվածի և միջավայրի մասին այլ տեղեկություններ GitHub Markdown-ում.:

`QOwnNotes --dump-settings`

- Նշեք այլ համատեքստ կարգավորումների և ներքին ֆայլերի համար.:

`QOwnNotes --session {{test}}`

- Գործարկեք ցանկի գործողությունը հավելվածը սկսելուց հետո.:

`QOwnNotes --action {{actionShow_Todo_List}}`
