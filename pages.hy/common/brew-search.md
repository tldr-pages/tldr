# եփել որոնում

> Որոնեք տակառներ և բանաձևեր:.
> Լրացուցիչ տեղեկություններ. <https://docs.brew.sh/Manpage#search--s-options-textregex->:.

- Որոնեք տակառներ և բանաձևեր՝ օգտագործելով հիմնաբառ.:

`brew search {{keyword}}`

- Որոնեք տակառներ և բանաձևեր՝ օգտագործելով `regex`:

`brew search /{{regex}}/`

- Միացնել որոնումը նկարագրությունների միջոցով.:

`brew search --desc {{keyword}}`

- Որոնեք միայն բանաձևեր.:

`brew search --formula {{keyword}}`

- Որոնեք միայն տակառներ.:

`brew search --cask {{keyword}}`
