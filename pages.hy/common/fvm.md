# fvm

> Flutter տարբերակի կառավարիչ:.
> Լրացուցիչ տեղեկություններ. <https://fvm.app/documentation/guides/basic-commands>:.

- Տեղադրեք Flutter SDK-ի տարբերակը: Ծրագրի կարգավորումների համար օգտագործեք առանց `version`-ի.:

`fvm install {{version}}`

- Նախագծում սահմանեք Flutter SDK-ի հատուկ տարբերակը.:

`fvm use {{version}} {{options}}`

- Սահմանեք Flutter SDK-ի գլոբալ տարբերակը.:

`fvm global {{version}}`

- Ջնջել FVM քեշը.:

`fvm destroy`

- Հեռացրեք Flutter SDK-ի հատուկ տարբերակը.:

`fvm remove {{version}}`

- Թվարկեք Flutter SDK-ի բոլոր տեղադրված տարբերակները.:

`fvm list`

- Թվարկեք Flutter SDK-ի բոլոր թողարկումները.:

`fvm releases`
