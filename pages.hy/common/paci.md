#խաղաղ

> Փաթեթի կառավարիչ Bash սկրիպտների համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tradebyte/paci#usage>:.

- Թարմացրեք հասանելի փաթեթների և տարբերակների ցանկը (խորհուրդ է տրվում այն գործարկել նախքան այլ `paci` հրամանները):

`paci refresh`

- Կարգավորեք դրա վարքագիծը.:

`paci configure`

- Որոնել տվյալ փաթեթը.:

`paci search {{package}}`

- Տեղադրեք փաթեթ.:

`paci install {{package}}`

- Թարմացրեք փաթեթը.:

`paci update {{package}}`
