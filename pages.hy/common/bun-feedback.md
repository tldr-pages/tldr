# bun արձագանք

> Հետադարձ կապ ուղարկեք Bun-ին:.
> Լրացուցիչ տեղեկություններ. <https://bun.com/docs/feedback#use-bun-feedback>:.

- Ուղարկեք տեքստը որպես արձագանք.:

`bun feedback "{{Feedback text!}}"`

- Ուղարկեք մեկ կամ մի քանի ֆայլ որպես արձագանք.:

`bun feedback {{path/to/file1 path/to/file2 ...}}`

- Հետադարձ կապ ուղարկեք կից էլ.:

`bun feedback {{path/to/file|text}} {{[-e|--email]}} {{email@address}}`
