# git վճարում

> Ստուգեք ճյուղը կամ ճանապարհները դեպի աշխատող ծառ:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-checkout>:.

- Ստեղծեք և անցեք նոր մասնաճյուղ.:

`git checkout -b {{branch_name}}`

- Ստեղծեք և անցեք նոր մասնաճյուղ՝ հիմնվելով հատուկ տեղեկանքի վրա (մասնաճյուղը, հեռակառավարման/ճյուղը, պիտակը վավեր հղումների օրինակներ են).:

`git checkout -b {{branch_name}} {{reference}}`

- Անցում գոյություն ունեցող տեղական մասնաճյուղին.:

`git checkout {{branch_name}}`

- Անցեք նախկինում ստուգված մասնաճյուղին.:

`git checkout -`

- Անցում գոյություն ունեցող հեռավոր մասնաճյուղին.:

`git checkout {{[-t|--track]}} {{remote_name}}/{{branch_name}}`

- Հրաժարվեք ընթացիկ գրացուցակի բոլոր չբեմադրված փոփոխություններից (տե՛ս `git reset`՝ չեղարկելու նման հրամանների համար).:

`git checkout .`

- Վերացնել չբեմադրված փոփոխությունները տվյալ ֆայլում.:

`git checkout {{path/to/file}}`

- Փոխարինեք ընթացիկ գրացուցակում գտնվող ֆայլը տվյալ ճյուղում կատարված տարբերակով.:

`git checkout {{branch_name}} -- {{path/to/file}}`
