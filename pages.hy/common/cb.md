# կբ

> Կտրեք, պատճենեք և տեղադրեք որևէ բան տերմինալում:.
> Լրացուցիչ տեղեկություններ. <https://github.com/Slackadays/Clipboard#-7>:.

- Ցուցադրել բոլոր սեղմատախտակները.:

`cb`

- Պատճենել ֆայլը clipboard-ում.:

`cb copy {{path/to/file}}`

- Պատճենեք որոշ տեքստ clipboard-ում.:

`cb copy "{{Some example text}}"`

- Պատճենեք խողովակային տվյալները clipboard-ում.:

`echo "{{Some example text}}" | cb`

- Տեղադրեք clipboard բովանդակությունը.:

`cb paste`

- Խողովակով դուրս բերեք clipboard բովանդակությունը.:

`cb | cat`

- Ցուցադրել clipboard պատմությունը:

`cb history`

- Ցուցադրել clipboard տեղեկատվությունը.:

`cb info`
