# պոեզիա-նախանձ

> Կառավարեք վիրտուալ միջավայրերը՝ կապված Poetry նախագծի հետ:.
> Տես նաև՝ `asdf`:.
> Լրացուցիչ տեղեկություններ. <https://python-poetry.org/docs/cli/#env>:.

- Տպեք հրամանը վիրտուալ միջավայրն ակտիվացնելու համար.:

`poetry env activate`

- Ցուցադրել տեղեկատվություն ընթացիկ միջավայրի մասին.:

`poetry env info`

- Ցուցադրել ընթացիկ միջավայրի ուղին.:

`poetry env info {{[-p|--path]}}`

- Ցուցադրել ընթացիկ միջավայրի Python գործարկվող ուղին.:

`poetry env info {{[-e|--executable]}}`

- Թվարկե՛ք ընթացիկ նախագծի հետ կապված բոլոր վիրտուալ միջավայրերը (ըստ ցանկության՝ ցույց տալով ամբողջական ուղին).:

`poetry env list --full-path`

- Հեռացրեք ընթացիկ նախագծի հետ կապված կոնկրետ կամ բոլոր վիրտուալ միջավայրերը.:

`poetry env remove python {{path/to/executable|environment_name}} | poetry env remove --all`

- Ակտիվացրեք կամ ստեղծեք վիրտուալ միջավայր նախագծի համար՝ օգտագործելով նշված Python գործարկիչը.:

`poetry env use python {{path/to/executable}}`
