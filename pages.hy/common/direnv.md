# direnv

> Shell-ի ընդլայնում միջավայրի փոփոխականները բեռնելու և բեռնաթափելու համար՝ կախված ընթացիկ գրացուցակից:.
> Լրացուցիչ տեղեկություններ. <https://github.com/direnv/direnv/blob/master/man/direnv.1.md>:.

- Ներկայիս գրացուցակում առկա `.envrc`-ը բեռնելու թույլտվություն տվեք direnv-ին.:

`direnv allow {{.}}`

- Չեղարկեք ընթացիկ գրացուցակում առկա `.envrc`-ը բեռնելու թույլտվությունը.:

`direnv deny {{.}}`

- Խմբագրեք `.envrc` ֆայլը լռելյայն տեքստային խմբագրիչում և վերաբեռնեք միջավայրը ելքի ժամանակ.:

`direnv edit {{.}}`

- Գործարկել շրջակա միջավայրի վերաբեռնումը.:

`direnv reload`

- Տպել կարգաբերման կարգավիճակի որոշ տեղեկություններ.:

`direnv status`
