# ֆայլ

> Որոշեք ֆայլի տեսակը:.
> Տես նաև՝ `stat`:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/file>:.

- Տվեք նշված ֆայլի տեսակի նկարագրությունը.:

`file {{path/to/file}}`

- Մի տպեք ֆայլի ուղին.:

`file {{[-b|--brief]}} {{path/to/file}}`

- Նայեք սեղմված ֆայլի ներսում և որոշեք ներսում գտնվող ֆայլի տեսակ(ներ).:

`file {{[-z|--uncompress]}} {{path/to/file.zip}}`

- Թույլ տվեք `file`-ին աշխատել հատուկ կամ սարքի ֆայլերի հետ՝:

`file {{[-s|--special-files]}} {{path/to/file}}`

- Մի կանգնեք առաջին ֆայլի տեսակի համընկնման վրա; շարունակեք մինչև ֆայլի ավարտը.:

`file {{[-k|--keep-going]}} {{path/to/file}}`

- Որոշեք ֆայլի MIME կոդավորման տեսակը.:

`file {{[-i|--mime]}} {{path/to/file}}`
