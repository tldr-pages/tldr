# npm-name

> Ստուգեք, արդյոք փաթեթը կամ կազմակերպության անունը հասանելի է npm-ում:.
> Լրացուցիչ տեղեկություններ. <https://github.com/sindresorhus/npm-name-cli>:.

- Ստուգեք, արդյոք որոշակի փաթեթի անուն հասանելի է `npm` ռեեստրում.:

`npm-name {{package}}`

- Գտեք նմանատիպ փաթեթների անուններ `npm` ռեեստրում.:

`npm-name --similar {{package}}`
