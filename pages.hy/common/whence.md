# որտեղից

> Zsh ներկառուցված՝ ցույց տալու, թե ինչպես պետք է մեկնաբանվի հրամանը:.
> Լրացուցիչ տեղեկություններ. <https://zsh.sourceforge.io/Doc/Release/Shell-Builtin-Commands.html#index-whence>:.

- Մեկնաբանեք `command`-ը, ընդարձակմամբ, եթե սահմանվում է որպես `alias` (նման `command -v` ներկառուցված).:

`whence "{{command}}"`

- Ցուցադրել `command`-ի տեսակը՝ գտնվելու վայրով, եթե սահմանված է որպես ֆունկցիա կամ երկուական (համարժեք `type` և `command -V` ներկառուցվածներին):

`whence -v "{{command}}"`

- Նույնը, ինչ վերևում, բացառությամբ տեղադրության փոխարեն shell ֆունկցիաների բովանդակության ցուցադրման (համարժեք `which` ներկառուցված).:

`whence -c "{{command}}"`

- Նույնը, ինչ վերևում, բայց ցույց տվեք հրամանի ուղու վրա եղած բոլոր դեպքերը (համարժեք `where` ներկառուցված).:

`whence -ca "{{command}}"`

- Որոնեք միայն `$PATH` `command`-ի համար՝ անտեսելով ներկառուցվածները, կեղծանունները կամ shell ֆունկցիաները (համարժեք `where` հրամանին):

`whence -p "{{command}}"`
