# jj պիտակների հավաքածու

> Ստեղծեք կամ թարմացրեք պիտակներ `jj` պահոցում:.
> Տես նաև՝ `jj tag delete`, `jj tag list`:.
> Լրացուցիչ տեղեկություններ. <https://docs.jj-vcs.dev/latest/cli-reference/#jj-tag-set>:.

- Ստեղծեք պիտակ, որը ցույց է տալիս ընթացիկ աշխատանքային պատճենի վերանայումը.:

`jj tag {{[s|set]}} {{tag_name}}`

- Ստեղծեք պիտակ, որը ցույց է տալիս որոշակի վերանայում.:

`jj tag {{[s|set]}} {{tag_name}} {{[-r|--revision]}} {{revision}}`

- Ստեղծեք մի քանի պիտակներ, որոնք մատնանշում են նույն վերանայումը.:

`jj tag {{[s|set]}} {{tag1 tag2 ...}} {{[-r|--revision]}} {{revision}}`

- Տեղափոխեք գոյություն ունեցող պիտակը մեկ այլ վերանայման.:

`jj tag {{[s|set]}} {{tag_name}} {{[-r|--revision]}} {{revision}} --allow-move`

- Ստեղծեք պիտակ, որը ցույց է տալիս ընթացիկ վերանայման մայրը.:

`jj tag {{[s|set]}} {{tag_name}} {{[-r|--revision]}} @-`
