# mysqld

> Սկսեք MySQL տվյալների բազայի սերվերը:.
> Լրացուցիչ տեղեկություններ. <https://dev.mysql.com/doc/refman/en/mysqld.html>:.

- Սկսեք MySQL տվյալների բազայի սերվերը.:

`mysqld`

- Գործարկեք սերվերը, տպելով սխալի հաղորդագրությունները վահանակում.:

`mysqld --console`

- Գործարկեք սերվերը՝ պահպանելով գրանցման ելքը հատուկ գրանցամատյանի ֆայլում.:

`mysqld --log={{path/to/file.log}}`

- Տպեք լռելյայն արգումենտները և դրանց արժեքները և դուրս եկեք.:

`mysqld --print-defaults`

- Սկսեք սերվերը, կարդալով փաստարկներ և արժեքներ ֆայլից.:

`mysqld --defaults-file={{path/to/file}}`

- Գործարկեք սերվերը և լսեք հատուկ նավահանգիստով.:

`mysqld --port={{port}}`

- Ցուցադրել օգնությունը.:

`mysqld --verbose --help`
