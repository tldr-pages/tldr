# միկրոմամբա

> Արագ, նվազագույն, ինքնուրույն փաթեթ և շրջակա միջավայրի կառավարիչ `conda` փաթեթների համար:.
> Բացվող փոխարինում `conda`-ի համար, իդեալական CI, Docker և թեթև կարգավորումների համար:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/micromamba>:.

- Ստեղծեք նոր միջավայր որոշակի ուղու վրա՝ դրա մեջ տեղադրելով անվանված փաթեթներ.:

`micromamba create {{[-p|--prefix]}} /{{path/to/env}} {{python=3.11 numpy}}`

- Ակտիվացրեք միջավայրը անունով կամ ճանապարհով.:

`micromamba activate {{[-p|--prefix]}} /{{path/to/env}}`

- Գործարկեք հրաման միջավայրի ներսում՝ առանց այն shell-ում ակտիվացնելու.:

`micromamba run {{[-p|--prefix]}} /{{path/to/env}} {{pytest tests/}}`

- Տեղադրեք փաթեթներ ներկայիս ակտիվ միջավայրում.:

`micromamba install {{scipy pandas}}`

- Թվարկեք բոլոր տեղադրված փաթեթները ընթացիկ միջավայրում.:

`micromamba list`

- Փաթեթների որոնում ալիքներում կամ ընթացիկ միջավայրում.:

`micromamba search {{package_name}}`

- Փաթեթի ծառի նման կախվածությունների հարցում.:

`micromamba repoquery depends {{[-t|--tree]}} {{package_name}}`

- Ցույց տալ տեղեկություններ ընթացիկ `micromamba` կարգավորումների մասին.:

`micromamba info`
