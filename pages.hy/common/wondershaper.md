# wondershaper

> Թույլ է տալիս օգտվողին սահմանափակել ցանցային ադապտերների թողունակությունը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/magnific0/wondershaper#usage>:.

- Ցույց տալ որոշակի [a] ադապտերի ներկայիս [s] կարգավիճակը.:

`wondershaper -s -a {{adapter_name}}`

- Մաքրել սահմանները կոնկրետ [a]dapter-ից.:

`wondershaper -c -a {{adapter_name}}`

- Սահմանեք կոնկրետ առավելագույն [d]ներբեռնման արագություն (կբիթ/վրկ-ով).:

`wondershaper -a {{adapter_name}} -d {{1024}}`

- Սահմանեք կոնկրետ առավելագույն [u] բեռնման արագություն (կբիթ/վրկ-ով).:

`wondershaper -a {{adapter_name}} -u {{512}}`

- Սահմանեք որոշակի առավելագույն [d]ներբեռնման արագություն և [u] բեռնման արագություն (Kpbs-ով):

`wondershaper -a {{adapter_name}} -d {{1024}} -u {{512}}`

- Ցուցադրել [h] օգնություն:

`wondershaper -h`
