# օրորոցի փաթեթ

> Կառավարեք փաթեթները Cradle-ի օրինակի համար:.
> Լրացուցիչ տեղեկություններ. <https://cradlephp.github.io/docs/3.B.-Reference-Command-Line-Tools.html#package>:.

- Ցուցադրել առկա փաթեթների ցանկը.:

`cradle package list`

- Փաթեթի որոնում.:

`cradle package search {{package}}`

- Տեղադրեք փաթեթ Packagist-ից.:

`cradle package install {{package}}`

- Տեղադրեք փաթեթի որոշակի տարբերակ.:

`cradle package install {{package}} {{version}}`

- Թարմացրեք փաթեթը.:

`cradle package update {{package}}`

- Թարմացրեք փաթեթը որոշակի տարբերակի.:

`cradle package update {{package}} {{version}}`

- Հեռացրեք որոշակի փաթեթ.:

`cradle package remove {{package}}`
