# ռելսեր

> Սերվերի կողմից MVC շրջանակ, որը գրված է Ruby-ով:.
> Որոշ ենթահրամաններ, ինչպիսիք են `generate`-ն, ունեն իրենց օգտագործման փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://guides.rubyonrails.org/command_line.html>:.

- Ստեղծեք նոր ռելսերի նախագիծ.:

`rails new "{{project_name}}"`

- Ստեղծեք փայտամած Փոստ անունով մոդելի համար՝ նախապես սահմանելով ատրիբուտների անվանումը և մարմինը.:

`rails generate scaffold Post title:string body:text`

- Գործարկել միգրացիաները.:

`rails db:migrate`

- Նշեք բոլոր երթուղիները.:

`rails routes`

- Գործարկել տեղական սերվերը ընթացիկ նախագծի համար 3000 նավահանգստում.:

`rails server`

- Գործարկեք տեղական սերվերը ընթացիկ նախագծի համար նշված նավահանգստում.:

`rails server {{[-p|--port]}} "{{port}}"`

- Բացեք վահանակը հրամանի տողից հավելվածի հետ փոխազդելու համար.:

`rails console`

- Ցուցադրման տարբերակը:

`rails {{[-v|--version]}}`
