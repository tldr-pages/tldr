# git commit

> Հանձնել ֆայլերը պահեստում:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-commit>:.

- Բացեք խմբագիր՝ հաղորդագրություն գրելու և բեմադրված ֆայլերը պահոցում տեղադրելու համար.:

`git commit`

- Բեմադրված ֆայլերը պահեք նշված հաղորդագրությամբ.:

`git commit {{[-m|--message]}} "{{message}}"`

- Ներկայացրեք բեմադրված ֆայլերը ֆայլից կարդացված հաղորդագրությամբ.:

`git commit {{[-F|--file]}} {{path/to/commit_message_file}}`

- Ավտոմատ բեմադրեք բոլոր փոփոխված և ջնջված ֆայլերը և կատարեք.:

`git commit {{[-a|--all]}} {{[-m|--message]}} "{{message}}"`

- Կատարեք փուլային ֆայլեր և ստորագրեք դրանք նշված GPG ստեղնով (կամ այն, որը սահմանված է կազմաձևման ֆայլում, եթե նշված չէ `key_id`):

`git commit {{[-S|--gpg-sign]}} {{key_id}} {{[-m|--message]}} "{{message}}"`

- Թարմացրեք վերջին commit-ը` ավելացնելով ընթացիկ փուլային փոփոխությունները, փոխելով հանձնման հեշը և բացեք խմբագիր՝ հաղորդագրությունը փոխելու համար.:

`git commit --amend`

- Կատարել միայն կոնկրետ (արդեն բեմադրված) ֆայլեր.:

`git commit {{path/to/file1 path/to/file2 ...}}`

- Ստեղծեք commit նշված հաղորդագրությամբ և նկարագրությամբ.:

`git commit {{[-m|--message]}} "{{message}}" {{[-m|--message]}} "{{description}}"`
