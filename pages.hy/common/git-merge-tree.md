# git merge-tree

> Միավորել մասնաճյուղերը՝ առանց ֆայլերը փոխելու:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-merge-tree>:.

- Ցույց տալ երկու ճյուղերի միաձուլման արդյունքը.:

`git merge-tree {{branch1}} {{branch2}}`

- Կատարեք միաձուլում և գրեք ստացված ծառը.:

`git merge-tree --write-tree {{branch1}} {{branch2}}`
