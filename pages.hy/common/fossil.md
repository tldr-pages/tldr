#բրածո

> Տարբերակների վերահսկման բաշխված համակարգ՝ ներկառուցված վիքիով, սխալների հետագծով և վեբ ինտերֆեյսով:.
> Որոշ ենթահրամաններ, ինչպիսիք են `commit`-ն, ունեն իրենց օգտագործման փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://fossil-scm.org/home/help>:.

- Նախաձեռնեք նոր դատարկ Fossil պահեստ.:

`fossil init {{repository_name.fossil}}`

- Ստեղծեք հեռավոր պահեստի տեղական պատճեն.:

`fossil clone {{remote_url}}`

- Ցույց տալ պահոցի ընթացիկ վիճակի ակնարկ.:

`fossil status`

- Բեմադրեք նոր ֆայլ.:

`fossil add {{path/to/file}}`

- Բեմադրել ֆայլը հեռացնելու համար.:

`fossil {{[rm|delete]}} {{path/to/file}}`

- Ստուգեք բոլոր փուլային փոփոխությունները.:

`fossil {{[ci|commit]}} {{[-m|--comment]}} "{{comment}}"`

- Փոփոխությունները տեղային պահոցից դեպի հեռավոր պահեստ.:

`fossil push {{remote_url}}`

- Փոփոխությունները հեռավոր պահոցից տեղափոխեք տեղական պահոց.:

`fossil pull {{remote_url}}`
