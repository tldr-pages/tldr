# npm բացատրել

> Բացատրեք, թե ինչպես է տեղադրվում փաթեթը՝ մանրամասնելով դրա կախվածությունը և ներառման պատճառները:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/explain/>:.

- Բացատրեք, թե ինչու է տեղադրվել կոնկրետ փաթեթ.:

`npm {{[why|explain]}} {{package_name}}`

- Ցույց տալ բացատրությունը JSON ձևաչափով.:

`npm {{[why|explain]}} {{package_name}} --json`

- Բացատրության մեջ ներառեք հասակակիցների կախվածությունը.:

`npm {{[why|explain]}} {{package_name}} --include peer`

- Սահմանափակեք բացատրության խորությունը 2 մակարդակի խորության վրա.:

`npm {{[why|explain]}} {{package_name}} --depth 2`
