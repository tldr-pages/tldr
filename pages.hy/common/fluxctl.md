# fluxctl

> Գործիք Flux v1-ի համար:.
> Լրացուցիչ տեղեկություններ. <https://fluxcd.io/flux/migration/>:.

- Ցուցակել աշխատանքային բեռները, որոնք ներկայումս աշխատում են կլաստերում որոշակի անվանումների տարածքում.:

`fluxctl --k8s-fwd-ns={{namespace}} list-workloads`

- Ցուցադրել տեղակայված և հասանելի պատկերները.:

`fluxctl list-images`

- Համաժամացրեք կլաստերը Git պահեստի հետ.:

`fluxctl sync`

- Միացրեք ավտոմատ տեղակայումը ծանրաբեռնվածության համար.:

`fluxctl automate`
