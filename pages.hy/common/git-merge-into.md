# git միաձուլվել

> Միավորե՛ք մի ճյուղը մյուս ճյուղի մեջ:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-merge-into>:.

- Միավորել աղբյուրի ճյուղը կոնկրետ նպատակակետ ճյուղի մեջ.:

`git merge-into {{source_branch}} {{destination_branch}}`

- Միավորել ընթացիկ մասնաճյուղը կոնկրետ նպատակակետ ճյուղի մեջ.:

`git merge-into {{destination_branch}}`
