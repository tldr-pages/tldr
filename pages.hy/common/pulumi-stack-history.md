# pulumi stack պատմություն

> Ցուցադրել պատմությունը կույտի համար:.
> Լրացուցիչ տեղեկություններ. <https://www.pulumi.com/docs/iac/cli/commands/pulumi_stack_history/>:.

- Ցուցադրել պատմությունը ընթացիկ փաթեթի համար.:

`pulumi stack {{[hist|history]}}`

- Ցուցադրել պատմությունը ընթացիկ փաթեթի համար, որը ցույց է տալիս ամբողջական ամսաթվերը հարաբերական ամսաթվերի փոխարեն.:

`pulumi stack {{[hist|history]}} --full-dates`

- Ցուցադրել պատմությունը ընթացիկ փաթեթի համար JSON ձևաչափով.:

`pulumi stack {{[hist|history]}} {{[-j|--json]}}`

- Ցուցադրել պատմությունը որոշակի կույտի համար.:

`pulumi stack {{[hist|history]}} {{[-s|--stack]}} {{stack_name}}`

- Ցուցադրել օգնությունը.:

`pulumi stack {{[hist|history]}} {{[-h|--help]}}`
