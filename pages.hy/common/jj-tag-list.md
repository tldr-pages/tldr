# jj պիտակների ցուցակ

> Ցուցակել պիտակները `jj` պահոցում:.
> Տես նաև՝ `jj tag delete`, `jj tag set`:.
> Լրացուցիչ տեղեկություններ. <https://docs.jj-vcs.dev/latest/cli-reference/#jj-tag-list>:.

- Նշեք բոլոր պիտակները.:

`jj tag {{[l|list]}}`

- Ցուցակել պիտակները, որոնք համապատասխանում են օրինակին.:

`jj tag {{[l|list]}} "{{pattern}}"`

- Ցուցակ պիտակները, որոնք համապատասխանում են ենթալարի օրինակին.:

`jj tag {{[l|list]}} "{{substring:release}}"`

- Ցուցակ պիտակները դասավորված ըստ հանձնակատարի ամսաթվի (նախ ամենանորը).:

`jj tag {{[l|list]}} --sort committer-date-`

- Նշեք պիտակները՝ տեսակավորված ըստ անվանման՝ նվազման կարգով.:

`jj tag {{[l|list]}} --sort name-`

- Ցուցակեք պիտակները հատուկ ձևանմուշով.:

`jj tag {{[l|list]}} {{[-T|--template]}} "{{template}}"`
