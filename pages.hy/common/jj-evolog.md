# jj evolog

> Ցույց տվեք, թե ինչպես է փոփոխությունը զարգացել ժամանակի ընթացքում՝ թվարկելով այն նախորդ պարտավորությունները, որոնց մասին նա մատնանշել է:.
> Լրացուցիչ տեղեկություններ. <https://docs.jj-vcs.dev/latest/cli-reference/#jj-evolog>:.

- Ցույց տվեք, թե ինչպես է վերանայվել ժամանակի ընթացքում.:

`jj evolog {{[-r|--revisions]}} {{revsets}}`

- Ցույց տալ տարբեր վիճակագրություն էվոլյուցիայի մատյանում.:

`jj evolog {{[-r|--revisions]}} {{revsets}} --stat`

- Ցույց տվեք էվոլյուցիայի մատյանում յուրաքանչյուր փոփոխության ամփոփագիրը.:

`jj evolog {{[-r|--revisions]}} {{revsets}} {{[-s|--summary]}}`
