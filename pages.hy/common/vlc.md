# vlc

> Cross-platform մուլտիմեդիա նվագարկիչ:.
> Տես նաև՝ `mpv`, `mplayer`, `ytfzf`:.
> Լրացուցիչ տեղեկություններ. <https://wiki.videolan.org/Documentation:Command_line/>:.

- Նվագարկել ֆայլ.:

`vlc {{path/to/file}}`

- Խաղացեք ամբողջ էկրանով.:

`vlc --fullscreen {{path/to/file}}`

- Խաղալ անջատված:

`vlc --no-audio {{path/to/file}}`

- Բազմիցս խաղալ.:

`vlc --loop {{path/to/file}}`

- Նվագարկեք տեսանյութը URL-ից.:

`vlc {{https://www.youtube.com/watch?v=oHg5SJYRHA0}}`
