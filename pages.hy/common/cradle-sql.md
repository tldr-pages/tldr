# օրորոց sql

> Կառավարեք Cradle SQL տվյալների բազաները:.
> Լրացուցիչ տեղեկություններ. <https://cradlephp.github.io/docs/3.B.-Reference-Command-Line-Tools.html#sql>:.

- Վերակառուցեք տվյալների բազայի սխեման.:

`cradle sql build`

- Վերակառուցեք տվյալների բազայի սխեման որոշակի փաթեթի համար.:

`cradle sql build {{package}}`

- Դատարկեք ամբողջ տվյալների բազան.:

`cradle sql flush`

- Դատարկեք տվյալների բազայի աղյուսակները որոշակի փաթեթի համար.:

`cradle sql flush {{package}}`

- Լրացրեք աղյուսակները բոլոր փաթեթների համար.:

`cradle sql populate`

- Լրացրեք աղյուսակները որոշակի փաթեթի համար.:

`cradle sql populate {{package}}`
