# բեռնափոխադրումներ

> Ստեղծեք նոր Cargo փաթեթ:.
> Համարժեք է `cargo new`-ին, սակայն գրացուցակի նշումը պարտադիր չէ:.
> Լրացուցիչ տեղեկություններ. <https://doc.rust-lang.org/cargo/commands/cargo-init.html>:.

- Նախաձեռնեք Rust նախագիծը երկուական թիրախով ընթացիկ գրացուցակում.:

`cargo init`

- Նախաձեռնեք Rust նախագիծը երկուական թիրախով նշված գրացուցակում.:

`cargo init {{path/to/directory}}`

- Նախաձեռնեք Rust նախագիծը գրադարանի թիրախով ընթացիկ գրացուցակում.:

`cargo init --lib`

- Նախաձեռնեք տարբերակի վերահսկման համակարգի պահեստը նախագծի գրացուցակում (կանխադրված՝ `git`):

`cargo init --vcs {{git|hg|pijul|fossil|none}}`

- Սահմանեք փաթեթի անունը (կանխադրված՝ գրացուցակի անուն).:

`cargo init --name {{name}}`
