# git diff

> Ցույց տալ փոփոխությունները հետևված ֆայլերում:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-diff>:.

- Ցուցադրել չբեմադրված փոփոխությունները՝:

`git diff`

- Ցուցադրել բոլոր չընդունված փոփոխությունները (ներառյալ փուլայինները).:

`git diff HEAD`

- Ցուցադրել միայն բեմադրված (ավելացված, բայց դեռ չկատարված) փոփոխությունները.:

`git diff --staged`

- Ցույց տալ փոփոխությունները բոլոր պարտավորություններից սկսած տվյալ ամսաթվից/ժամից (ամսաթվի արտահայտություն, օրինակ՝ «1 շաբաթ 2 օր» կամ ISO ամսաթիվ).:

`git diff 'HEAD@{{{3 months|weeks|days|hours|seconds ago}}}'`

- Ցույց տալ տարբեր վիճակագրություն, ինչպես, օրինակ, փոխված ֆայլերը, հիստոգրամը և տողերի ընդհանուր տեղադրումները/ջնջումները.:

`git diff --stat {{commit}}`

- Արտադրեք ֆայլերի ստեղծման, վերանվանումների և ռեժիմի փոփոխությունների համառոտ նկարագրությունը տվյալ պարտավորությունից հետո.:

`git diff --summary {{commit}}`

- Համեմատեք մեկ ֆայլ երկու ճյուղերի կամ պարտավորությունների միջև.:

`git diff {{branch_1}}..{{branch_2}} {{path/to/file}}`

- Համեմատեք տարբեր ֆայլեր ընթացիկ մասնաճյուղից մեկ այլ մասնաճյուղ.:

`git diff {{other_branch}}:{{path/to/file2}} {{path/to/file1}}`
