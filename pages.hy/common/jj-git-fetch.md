# jj git fetch

> Վերցրեք Git հեռակառավարման վահանակից՝ ներբեռնելով օբյեկտներ և ռեֆերատներ հեռավոր պահոցից:.
> Լրացուցիչ տեղեկություններ. <https://docs.jj-vcs.dev/latest/cli-reference/#jj-git-fetch>:.

- Ստացեք վերջին փոփոխությունները լռելյայն հեռավոր պահոցից.:

`jj git fetch`

- Ստացեք վերջին փոփոխությունները տվյալ հեռավոր պահոցից.:

`jj git fetch --remote {{remote}}`

- Ստացեք վերջին փոփոխությունները միայն տվյալ մասնաճյուղերից.:

`jj git fetch {{[-b|--branch]}} {{branch}}`

- Ստացեք վերջին փոփոխությունները բոլոր հեռակառավարիչներից.:

`jj git fetch --all-remote`
