# դոկեր համակարգ

> Կառավարեք Docker-ի տվյալները և ցուցադրեք ամբողջ համակարգի տեղեկատվությունը:.
> Լրացուցիչ տեղեկություններ. <https://docs.docker.com/reference/cli/docker/system/>:.

- Ցույց տալ Docker սկավառակի օգտագործումը.:

`docker system df`

- Ցույց տալ մանրամասն տեղեկատվություն սկավառակի օգտագործման վերաբերյալ.:

`docker system df {{[-v|--verbose]}}`

- Հեռացրեք չօգտագործված տվյալները (ավելացրեք `--volumes`՝ չօգտագործված ծավալները նույնպես հեռացնելու համար).:

`docker system prune`

- Հեռացրեք չօգտագործված տվյալները, որոնք ստեղծվել են ավելի քան սահմանված ժամանակում անցյալում.:

`docker system prune --filter "until={{hours}}h{{minutes}}m"`

- Հեռացրեք բոլոր չօգտագործված տվյալները.:

`docker system prune {{[-a|--all]}} --volumes`

- Ցուցադրել իրական ժամանակի իրադարձությունները Docker daemon-ից.:

`docker system events`

- Ցուցադրել իրական ժամանակի իրադարձությունները բեռնարկղերից, որոնք հոսում են որպես վավեր JSON գծեր.:

`docker system events {{[-f|--filter]}} 'type=container' --format '{{json .}}'`

- Ցուցադրել համակարգային տեղեկատվություն.:

`docker system info`
