# hg

> Mercurial - բաշխված աղբյուրների կառավարման կառավարման համակարգ:.
> Որոշ ենթահրամաններ, ինչպիսիք են `commit`-ն, ունեն իրենց օգտագործման փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://www.mercurial-scm.org/help/commands>:.

- Ստեղծեք դատարկ Mercurial պահոց.:

`hg init`

- Կլոնավորեք հեռավոր Mercurial պահեստը ինտերնետից.:

`hg clone {{https://example.com/repo}}`

- Դիտեք տեղական պահեստի կարգավիճակը.:

`hg status`

- Ավելացնել բոլոր նոր ֆայլերը հաջորդ commit-ին.:

`hg add`

- Փոփոխություններ կատարեք տարբերակի պատմության մեջ.:

`hg commit {{[-m|--message]}} {{message_text}}`

- Տեղական փոփոխությունները մղեք հեռավոր պահոց.:

`hg push`

- Քաշեք ցանկացած փոփոխություն, որը կատարվել է հեռակառավարման վահանակում.:

`hg pull`

- Վերականգնել ամեն ինչ այնպես, ինչպես եղել է վերջին commit-ում.:

`hg update {{[-C|--clean]}}; hg purge`
