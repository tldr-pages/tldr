# traefik

> HTTP հակադարձ վստահված անձ և բեռի հավասարակշռող:.
> Լրացուցիչ տեղեկություններ. <https://doc.traefik.io/traefik/v2.11/operations/cli/>:.

- Սկսեք սերվերը լռելյայն կազմաձևով.:

`traefik`

- Սկսեք սերվերը հատուկ կազմաձևման ֆայլով.:

`traefik --ConfigFile {{config_file.toml}}`

- Գործարկեք սերվերը միացված կլաստերի ռեժիմով.:

`traefik --cluster`

- Գործարկեք սերվերը միացված վեբ միջերեսով.:

`traefik --web`
