# ժամացույցի մեխանիզմ-cli

> Ինտերֆեյս Clockwork PHP կարգաբերման շրջանակի համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/ptrofimov/clockwork-cli>:.

- Դիտեք ժամացույցի տեղեկամատյանները ընթացիկ նախագծի համար.:

`clockwork-cli`

- Դիտեք ժամացույցի տեղեկամատյանները որոշակի նախագծի համար.:

`clockwork-cli {{path/to/directory}}`

- Դիտեք ժամացույցի տեղեկամատյանները բազմաթիվ նախագծերի համար.:

`clockwork-cli {{path/to/directory1 path/to/directory2 ...}}`
