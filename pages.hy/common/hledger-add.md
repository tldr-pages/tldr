# hledger ավելացնել

> Գրանցեք նոր գործարքներ ինտերակտիվ հուշումներով վահանակում:.
> Լրացուցիչ տեղեկություններ. <https://hledger.org/hledger.html#add>:.

- Գրանցեք նոր գործարքներ՝ պահպանելով լռելյայն ամսագրի ֆայլում՝:

`hledger add`

- Ավելացրեք գործարքներ `2024.journal`-ին, բայց նաև բեռնեք `2023.journal` ավարտի համար.:

`hledger add {{[-f|--file]}} {{path/to/2024.journal}} {{[-f|--file]}} {{path/to/2023.journal}}`

- Տրամադրեք պատասխաններ առաջին չորս հուշումների համար.:

`hledger add {{today}} '{{best buy}}' {{expenses:supplies}} '{{$20}}'`

- Ցույց տալ `add`-ի ընտրանքները և փաստաթղթերը `$PAGER`-ով:

`hledger add {{[-h|--help]}}`

- Ցույց տալ `add`-ի փաստաթղթերը՝ `info` կամ `man`-ով, եթե առկա է.:

`hledger help add`
