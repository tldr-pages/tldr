#ղ

> Անխափան աշխատեք GitHub-ի հետ:.
> Որոշ ենթահրամաններ, ինչպիսիք են `config`-ն, ունեն իրենց օգտագործման փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://cli.github.com/manual/gh>:.

- Տեղականորեն կլոնավորեք GitHub պահեստը.:

`gh repo clone {{owner}}/{{repository}}`

- Ստեղծեք նոր թողարկում.:

`gh issue {{[new|create]}}`

- Դիտեք և զտեք ընթացիկ պահոցի բաց թողարկումները.:

`gh issue {{[ls|list]}}`

- Դիտեք խնդիրը լռելյայն վեբ դիտարկիչում.:

`gh issue view {{[-w|--web]}} {{issue_number|url}}`

- Ստեղծեք ձգման հարցում.:

`gh pr {{[new|create]}}`

- Դիտեք քաշման հարցումը լռելյայն վեբ բրաուզերում.:

`gh pr view {{[-w|--web]}} {{pr_number|url|branch}}`

- Ստուգեք տեղական ձգողականության հատուկ հարցում.:

`gh {{[co|pr checkout]}} {{pr_number|url|branch}}`

- Ստուգեք պահեստի ձգման հարցումների կարգավիճակը.:

`gh pr status`
