# տուփ

> PHP հավելված՝ Phars-ի կառուցման և կառավարման համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/box-project/box>:.

- Կազմեք նոր Phar ֆայլ.:

`box compile`

- Կազմեք նոր Phar ֆայլ՝ օգտագործելով հատուկ կազմաձևման ֆայլ.:

`box compile {{[-c|--config]}} {{path/to/config}}`

- Ցուցադրել տեղեկատվություն PHAR PHP ընդլայնման մասին.:

`box info`

- Ցուցադրել տեղեկատվություն որոշակի Phar ֆայլի մասին.:

`box info {{path/to/phar_file}}`

- Վավերացրեք առաջին հայտնաբերված կազմաձևման ֆայլը աշխատանքային գրացուցակում.:

`box validate`

- Ստուգեք կոնկրետ Phar ֆայլի ստորագրությունը.:

`box verify {{path/to/phar_file}}`

- Ցուցադրել օգնությունը.:

`box help`
