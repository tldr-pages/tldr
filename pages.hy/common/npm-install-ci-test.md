# npm install-ci-test

> Տեղադրեք նախագծի կախվածությունները և կատարեք թեստերը մեկ հրամանով, որը հիմնականում օգտագործվում է շարունակական ինտեգրման (CI) միջավայրերում:.
> Համարժեք է `npm ci && npm test` գործարկմանը:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/install-ci-test/>:.

- Տեղադրեք կախվածություններ և կատարեք թեստեր.:

`npm {{[cit|install-ci-test]}}`

- Ցուցադրել մանրամասն տեղեկամատյանները տեղադրման և փորձարկման ընթացքում.:

`npm {{[cit|install-ci-test]}} --verbose`
