# հանգույց

> Սերվերի կողմից JavaScript հարթակ (Node.js):.
> Լրացուցիչ տեղեկություններ. <https://nodejs.org/docs/latest/api/cli.html#options>:.

- Գործարկել JavaScript ֆայլը.:

`node {{path/to/file}}`

- Սկսեք REPL (ինտերակտիվ կեղև).:

`node`

- Կատարեք նշված ֆայլը՝ վերսկսելով գործընթացը, երբ ներմուծված ֆայլը փոխվում է (պահանջվում է Node.js 18.11+ տարբերակը):

`node --watch {{path/to/file}}`

- Գնահատեք JavaScript կոդը՝ փոխանցելով այն որպես փաստարկ.:

`node {{[-e|--eval]}} "{{code}}"`

- Գնահատեք և տպեք արդյունքը, որն օգտակար է հանգույցի կախվածության տարբերակները տպելու համար.:

`node {{[-p|--print]}} "process.versions"`

- Ակտիվացրեք տեսուչը՝ դադարեցնելով կատարումը, մինչև միացվի վրիպազերծիչը, երբ սկզբնական կոդը ամբողջությամբ վերլուծվի.:

`node --no-lazy --inspect-brk {{path/to/file}}`
