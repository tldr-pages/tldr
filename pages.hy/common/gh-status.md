# gh կարգավիճակ

> Ստուգեք բաժանորդագրված պահեստների կարգավիճակը GitHub-ում:.
> Լրացուցիչ տեղեկություններ. <https://cli.github.com/manual/gh_status>:.

- Տպեք տեղեկատվությունը բոլոր պահեստների մասին, որոնց բաժանորդագրված եք (հարցեր, պահանջներ, ակնարկներ, հիշատակումներ, գործունեություն և այլն):

`gh status`

- Բացառել որոշակի պահեստներ.:

`gh status {{[-e|--exclude]}} {{repositories}}`

- Կազմակերպությանը պատկանող պահեստների կարգավիճակը տպել.:

`gh status {{[-o|--org]}} {{organization}}`
