# git ignore-io

> Ստեղծեք `.gitignore` ֆայլեր նախապես սահմանված ձևանմուշներից:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-ignore-io>:.

- Ցուցակեք մատչելի ձևանմուշները.:

`git ignore-io list`

- Ստեղծեք `.gitignore` ձևանմուշ.:

`git ignore-io {{item_a,item_b,...}}`
