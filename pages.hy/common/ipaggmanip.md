# ipaggmanip

> Շահարկել `ipaggcreate`-ի կողմից ստեղծված համախառն վիճակագրությունը:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/ipaggmanip>:.

- Միավորել պիտակները, որոնք հավասար են իրենց բարձր կարգի բիթերին.:

`ipaggmanip {{[-p|--prefix]}} {{16}} {{path/to/file}}`

- Հեռացրեք բայթերի տվյալ քանակից փոքր քանակով պիտակները և դուրս բերեք այդպիսի պիտակների պատահական նմուշ.:

`ipaggmanip --cut-smaller {{100}} --cull-labels {{5}} {{path/to/file}}`

- Փոխարինեք յուրաքանչյուր պիտակի թիվը 1-ով, եթե այն զրոյական չէ.:

`ipaggmanip {{[-P|--posterize]}} {{path/to/file}}`
