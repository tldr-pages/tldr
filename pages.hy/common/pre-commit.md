# նախապես պարտավորվել

> Ստեղծեք Git կեռիկներ, որոնք աշխատում են նախքան պարտավորությունները:.
> Լրացուցիչ տեղեկություններ. <https://pre-commit.com/#command-line-interface>:.

- Տեղադրեք pre-commit ձեր Git կեռիկներում՝:

`pre-commit install`

- Գործարկեք նախնական կատարման կեռիկներ բոլոր փուլային ֆայլերի վրա.:

`pre-commit run`

- Գործարկեք նախապես կատարվող կեռիկներ բոլոր ֆայլերի վրա՝ բեմադրված կամ չբեմադրված.:

`pre-commit run --all-files`

- Մաքրել նախնական կատարման քեշը.:

`pre-commit clean`

- Թարմացրեք pre-commit կազմաձևման ֆայլը ռեպո-ների վերջին տարբերակներին.:

`pre-commit autoupdate`
