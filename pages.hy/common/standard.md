# ստանդարտ

> JavaScript Ստանդարտ ոճի գործիքը JavaScript-ի կոդն ամրացնելու և ամրացնելու համար:.
> Լրացուցիչ տեղեկություններ. <https://standardjs.com/#usage>:.

- Տեղադրեք JavaScript-ի բոլոր սկզբնաղբյուր ֆայլերը ընթացիկ գրացուցակում.:

`standard`

- Lint հատուկ JavaScript ֆայլ(ներ).:

`standard {{path/to/file1 path/to/file2 ...}}`

- Կիրառել ավտոմատ շտկումներ երեսպատման ժամանակ.:

`standard --fix`

- Հայտարարեք ցանկացած հասանելի գլոբալ փոփոխական.:

`standard --global {{variable}}`

- Օգտագործեք հատուկ ESLint պլագին, երբ պատում եք.:

`standard --plugin {{plugin}}`

- Օգտագործեք հատուկ JS վերլուծիչ, երբ պատում եք.:

`standard --parser {{parser}}`

- Օգտագործեք հատուկ ESLint միջավայր, երբ պատում եք.:

`standard --env {{environment}}`
