# ածխածին-հիմա

> Ստեղծեք կոդի գեղեցիկ պատկերներ:.
> Լրացուցիչ տեղեկություններ. <https://github.com/mixn/carbon-now-cli>:.

- Ստեղծեք պատկեր ֆայլից՝ օգտագործելով լռելյայն կարգավորումները.:

`carbon-now {{path/to/file}}`

- Ստեղծեք պատկեր տեքստից clipboard-ում՝ օգտագործելով լռելյայն կարգավորումները.:

`carbon-now --from-clipboard`

- Ստեղծեք պատկեր `stdin`-ից՝ օգտագործելով լռելյայն կարգավորումները և պատճենեք սեղմատախտակում՝:

`{{input}} | carbon-now --to-clipboard`

- Ստեղծեք պատկերներ ինտերակտիվ կերպով մաքսային պարամետրերի համար և կամայականորեն պահպանեք նախադրվածը.:

`carbon-now {{[-i|--interactive]}} {{path/to/file}}`

- Ստեղծեք պատկերներ նախկինում պահպանված նախադրյալից.:

`carbon-now {{[-p|--preset]}} {{preset}} {{path/to/file}}`

- Սկսեք տեքստի նշված տողից.:

`carbon-now {{[-s|--start]}} {{line}} {{path/to/file}}`

- Ավարտեք տեքստի որոշակի տողով.:

`carbon-now {{[-e|--end]}} {{line}} {{path/to/file}}`

- Պահելու փոխարեն բացեք պատկերը դիտարկիչում.:

`carbon-now --open {{path/to/file}}`
