# ուլտրամանուշակագույն կողպեք

> Թարմացրեք նախագծի կողպեքի ֆայլը:.
> Լրացուցիչ տեղեկություններ. <https://docs.astral.sh/uv/reference/cli/#uv-lock>:.

- Ստեղծեք կամ թարմացրեք նախագծի կողպեքի ֆայլը.:

`uv lock`

- Ստուգեք, արդյոք կողպեքի ֆայլը արդիական է առանց այն թարմացնելու.:

`uv lock --check`

- Հաստատեք, որ կողպեքի ֆայլը գոյություն ունի՝ առանց ստուգելու, թե արդյոք այն ընթացիկ է.:

`uv lock --check-exists`

- Նախադիտեք, թե ինչ կկողպվի առանց կողպեքի ֆայլը գրելու.:

`uv lock --dry-run`

- Կողպեք Python-ի կոնկրետ սցենար ընթացիկ նախագծի փոխարեն.:

`uv lock --script {{path/to/script.py}}`

- Թարմացրեք բոլոր փաթեթները իրենց վերջին համատեղելի տարբերակներին.:

`uv lock --upgrade`

- Թարմացրեք միայն հատուկ փաթեթներ.:

`uv lock --upgrade-package {{package1}} --upgrade-package {{package2}}`
