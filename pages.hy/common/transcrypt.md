# transcrypt

> Թափանցիկ գաղտնագրեք ֆայլերը Git պահոցում:.
> Լրացուցիչ տեղեկություններ. <https://github.com/elasticdog/transcrypt#command-line-options>:.

- Նախաձեռնեք չկարգավորված պահոց.:

`transcrypt`

- Թվարկեք ներկայումս գաղտնագրված ֆայլերը.:

`git ls-crypt`

- Ցուցադրել կազմաձևված պահեստի հավատարմագրերը.:

`transcrypt {{[-d|--display]}}`

- Նախաձեռնեք և վերծանեք կազմաձևված պահոցի թարմ կլոնը.:

`transcrypt {{[-c|--cipher]}} {{cipher}}`

- Վերաբանալ կոդավորման ծածկագիրը կամ գաղտնաբառը փոխելու համար.:

`transcrypt {{[-r|--rekey]}}`
