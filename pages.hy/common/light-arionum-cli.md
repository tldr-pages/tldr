# light-arionum-cli

> PHP թեթև դրամապանակ Arionum կրիպտոարժույթի համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/arionum/lightWalletCLI#usage>:.

- Ստեղծեք նոր հանրային/մասնավոր բանալիների զույգ.:

`light-arionum-cli`

- Ցուցադրել ընթացիկ հասցեի մնացորդը.:

`light-arionum-cli balance`

- Ցուցադրել նշված հասցեի մնացորդը.:

`light-arionum-cli balance {{address}}`

- Գործարք ուղարկեք կամընտիր հաղորդագրությամբ.:

`light-arionum-cli send {{address}} {{value}} {{optional_message}}`

- Արտահանել ընթացիկ դրամապանակի տեղեկատվությունը.:

`light-arionum-cli export`

- Ցուցադրել տեղեկատվություն ընթացիկ բլոկի մասին.:

`light-arionum-cli block`

- Ցուցադրել տեղեկատվություն ընթացիկ հասցեի գործարքների մասին.:

`light-arionum-cli transactions`

- Ցուցադրել տեղեկատվություն կոնկրետ գործարքի մասին.:

`light-arionum-cli transaction {{transaction_id}}`
