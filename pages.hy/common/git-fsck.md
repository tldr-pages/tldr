# git fsck

> Ստուգեք հանգույցների վավերականությունը և կապը Git պահեստի ինդեքսում:.
> Նշում. Չի կատարում որևէ փոփոխություն:.
> Տես նաև՝ `git gc`:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-fsck>:.

- Ստուգեք ընթացիկ պահոցը.:

`git fsck`

- Ցուցակե՛ք գտնված բոլոր պիտակները.:

`git fsck --tags`

- Թվարկեք հայտնաբերված բոլոր արմատային հանգույցները.:

`git fsck --root`

- Ցույց տալ բոլոր անհասանելի և կախված առարկաները, անտեսել հետադարձ կապերը և կատարել ամբողջականության ամբողջական ստուգում.:

`git fsck --dangling --no-reflogs --unreachable --full`

- Ստուգեք միայն կապը (բաց թողնել օբյեկտի ամբողջականության ստուգումը).:

`git fsck --connectivity-only`
