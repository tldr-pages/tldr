# npm ավարտ

> Միացնել կամ կարգավորել ներդիրների լրացումը npm հրամանների համար:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-completion/>:.

- Միացնել ներդիրների լրացումը ընթացիկ կեղևի նստաշրջանի համար.:

`npm completion`

- Ավարտել կեղևի պրոֆիլին (Bash):

`npm completion >> ~/.bashrc`

- Ավարտումը ավելացրեք կեղևի պրոֆիլին (Zsh):

`npm completion >> ~/.zshrc`

- Գրեք ավարտման սցենարը որոշակի ֆայլում.:

`npm completion > {{path/to/file}}`
