# mat2

> Անանունացրեք տարբեր ֆայլերի ձևաչափեր՝ հեռացնելով մետատվյալները:.
> Լրացուցիչ տեղեկություններ. <https://0xacab.org/jvoisin/mat2>:.

- Ցուցակեք աջակցվող ֆայլերի ձևաչափերը.:

`mat2 --list`

- Հեռացրեք մետատվյալները ֆայլից.:

`mat2 {{path/to/file}}`

- Հեռացրեք մետատվյալները ֆայլից և տպեք մանրամասն արդյունքը վահանակում.:

`mat2 --verbose {{path/to/file}}`

- Ցուցադրել մետատվյալները ֆայլում՝ առանց այն հեռացնելու.:

`mat2 --show {{path/to/file}}`

- Մասամբ հեռացնել մետատվյալները ֆայլից.:

`mat2 --lightweight {{path/to/file}}`

- Հեռացրեք մետատվյալները տեղում ֆայլից՝ առանց կրկնօրինակ ստեղծելու.:

`mat2 --inplace {{path/to/file}}`
