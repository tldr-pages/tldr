# ipscan

> Արագ ցանցային սկաներ, որը նախատեսված է օգտագործման համար պարզ:.
> Հայտնի է նաև որպես Angry IP սկաներ:.
> Լրացուցիչ տեղեկություններ. <https://www.aldeid.com/wiki/Angry-IPScan#CLI>:.

- Սկանավորեք որոշակի IP հասցե.:

`ipscan {{192.168.0.1}}`

- Սկանավորեք մի շարք IP հասցեներ.:

`ipscan {{192.168.0.1-254}}`

- Սկանավորեք մի շարք IP հասցեներ և արդյունքները պահեք ֆայլում.:

`ipscan {{192.168.0.1-254}} -o {{path/to/output.txt}}`

- Սկանավորեք IP-ները հատուկ նավահանգիստների հավաքածուով.:

`ipscan {{192.168.0.1-254}} -p {{80,443,22}}`

- Ցանցի գերբեռնվածությունից խուսափելու համար հարցումների միջև ուշացումով սկանավորեք.:

`ipscan {{192.168.0.1-254}} -d {{200}}`

- Ցուցադրել օգնությունը.:

`ipscan --help`
