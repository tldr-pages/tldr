# pyenv

> Հեշտությամբ անցեք Python-ի բազմաթիվ տարբերակների միջև:.
> Տես նաև՝ `asdf`:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/pyenv>:.

- Թվարկեք բոլոր հասանելի հրամանները.:

`pyenv commands`

- Թվարկեք Python-ի բոլոր տարբերակները `${PYENV_ROOT}/versions` գրացուցակում.:

`pyenv versions`

- Թվարկեք Python-ի բոլոր տարբերակները, որոնք կարող են տեղադրվել վերին հոսանքով.:

`pyenv install --list`

- Տեղադրեք Python տարբերակը `${PYENV_ROOT}/versions` գրացուցակում.:

`pyenv install {{2.7.10}}`

- Տեղահանեք Python-ի տարբերակը `${PYENV_ROOT}/versions` գրացուցակում.:

`pyenv uninstall {{2.7.10}}`

- Սահմանեք Python-ի տարբերակը գլոբալ օգտագործման համար ընթացիկ մեքենայում.:

`pyenv global {{2.7.10}}`

- Սահմանեք Python տարբերակը, որն օգտագործվի ընթացիկ գրացուցակում և դրա տակ գտնվող բոլոր դիրեկտորիաներում.:

`pyenv local {{2.7.10}}`
