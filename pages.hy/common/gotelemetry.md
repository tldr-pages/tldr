# Գոտելաչափություն

> Կառավարեք Go հեռաչափության տվյալները և կարգավորումները:.
> Լրացուցիչ տեղեկություններ. <https://telemetry.go.dev/privacy#collection>:.

- Միացնել հեռաչափության վերբեռնումը.:

`gotelemetry on`

- Անջատել հեռաչափության վերբեռնումը.:

`gotelemetry off`

- Գործարկեք Web Viewer տեղական հեռաչափության տվյալների համար.:

`gotelemetry view`

- Տպել ընթացիկ հեռաչափական միջավայրը.:

`gotelemetry env`

- Ցուցադրել օգնություն կոնկրետ ենթահրամանի համար.:

`gotelemetry help {{subcommand}}`
