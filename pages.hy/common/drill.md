# փորվածք

> Կատարել տարբեր DNS հարցումներ:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/drill>:.

- Փնտրեք հյուրընկալողի անվան հետ կապված IP(ներ)ը (A գրառումներ).:

`drill {{example.com}}`

- Փնտրեք փոստի սերվեր(ները)՝ կապված տվյալ տիրույթի անվան հետ (MX գրառում).:

`drill mx {{example.com}}`

- Ստացեք բոլոր տեսակի գրառումները տվյալ տիրույթի անվան համար.:

`drill any {{example.com}}`

- Նշեք այլընտրանքային DNS սերվեր հարցման համար.:

`drill {{example.com}} @{{8.8.8.8}}`

- Կատարեք հակադարձ DNS որոնում IP հասցեի վրա (PTR գրառում).:

`drill -x {{8.8.8.8}}`

- Կատարեք DNSSEC հետք արմատային սերվերներից մինչև տիրույթի անուն.:

`drill -TD {{example.com}}`

- Ցույց տալ DNSKEY գրառում(ներ) դոմենի անվան համար.:

`drill -s dnskey {{example.com}}`
