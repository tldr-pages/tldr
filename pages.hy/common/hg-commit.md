# hg պարտավորվել

> Տեղադրեք բոլոր փուլային կամ նշված ֆայլերը պահոց:.
> Լրացուցիչ տեղեկություններ. <https://www.mercurial-scm.org/help/commands/commit>:.

- Բեմադրված ֆայլերը պահեք պահոց.:

`hg commit`

- Կատարեք որոշակի ֆայլ կամ գրացուցակ.:

`hg commit {{path/to/file_or_directory}}`

- Պարտավորվեք կոնկրետ հաղորդագրությամբ.:

`hg commit {{[-m|--message]}} {{message}}`

- Ներկայացրեք բոլոր ֆայլերը, որոնք համապատասխանում են նշված օրինակին.:

`hg commit {{[-I|--include]}} {{pattern}}`

- Կատարել բոլոր ֆայլերը, բացառությամբ նրանց, որոնք համապատասխանում են նշված օրինակին.:

`hg commit {{[-X|--exclude]}} {{pattern}}`

- Պարտավորվեք օգտագործելով ինտերակտիվ ռեժիմը.:

`hg commit {{[-i|--interactive]}}`
