# npm

> JavaScript և Node.js փաթեթների կառավարիչ:.
> Կառավարեք Node.js նախագծերը և դրանց մոդուլային կախվածությունները:.
> Որոշ ենթահրամաններ, ինչպիսիք են `install`, `run` և այլն, ունեն իրենց օգտագործման փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm/>:.

- Ստեղծեք `package.json` ֆայլ լռելյայն արժեքներով (բաց թողեք `--yes` դա ինտերակտիվ կերպով անելու համար):

`npm init {{[-y|--yes]}}`

- Ներբեռնեք բոլոր փաթեթները, որոնք նշված են որպես կախվածություն `package.json`-ում:

`npm {{[i|install]}}`

- Ներբեռնեք փաթեթի որոշակի տարբերակ և ավելացրեք այն `package.json`-ի կախվածությունների ցանկում՝:

`npm {{[i|install]}} {{package_name}}@{{version}}`

- Ներբեռնեք փաթեթի վերջին տարբերակը և ավելացրեք այն մշակողի կախվածությունների ցանկում `package.json`-ում:

`npm {{[i|install]}} {{package_name}} {{[-D|--save-dev]}}`

- Ներբեռնեք փաթեթի վերջին տարբերակը և տեղադրեք այն գլոբալ (տեղադրման վայրը սահմանեք `npm config set prefix`-ով):

`npm {{[i|install]}} {{package_name}} {{[-g|--global]}}`

- Ապատեղադրեք փաթեթը և հեռացրեք այն `package.json`-ի կախվածությունների ցանկից՝:

`npm {{[r|uninstall]}} {{package_name}}`

- Թվարկեք տեղական տեղադրված բոլոր կախվածությունները.:

`npm {{[ls|list]}}`

- Թվարկեք բոլոր վերին մակարդակի գլոբալ տեղադրված փաթեթները.:

`npm {{[ls|list]}} {{[-g|--global]}} --depth {{0}}`
