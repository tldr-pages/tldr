# նգ քեշ

> Կազմաձևեք սկավառակի մշտական քեշը և առբերեք քեշի վիճակագրությունը:.
> Լրացուցիչ տեղեկություններ. <https://angular.dev/cli/cache>:.

- Ջնջել մշտական սկավառակի քեշը.:

`ng cache clean`

- Անջատեք մշտական սկավառակի քեշը աշխատանքային տարածքում բոլոր նախագծերի համար.:

`ng cache {{[off|disable]}}`

- Միացնել մշտական սկավառակի քեշը աշխատանքային տարածքում բոլոր նախագծերի համար.:

`ng cache {{[on|enable]}}`

- Տպել մշտական սկավառակի քեշի կազմաձևումը և վիճակագրությունը.:

`ng cache info`
