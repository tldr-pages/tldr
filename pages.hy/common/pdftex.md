# pdftex

> Կազմեք PDF փաստաթուղթ TeX աղբյուրի ֆայլերից:.
> Լրացուցիչ տեղեկություններ. <https://www.tug.org/applications/pdftex/>:.

- Կազմեք PDF փաստաթուղթ.:

`pdftex {{source.tex}}`

- Կազմեք PDF փաստաթուղթ՝ նշելով ելքային գրացուցակը.:

`pdftex -output-directory={{path/to/directory}} {{source.tex}}`

- Կազմեք PDF փաստաթուղթ, դուրս գալով յուրաքանչյուր սխալից.:

`pdftex -halt-on-error {{source.tex}}`
