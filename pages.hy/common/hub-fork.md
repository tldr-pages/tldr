# հանգույց պատառաքաղ

> Պատառաքաղել GitHub ռեպո: Հավանել `git fork` `git-extras`-ից:.
> Լրացուցիչ տեղեկություններ. <https://hub.github.com/hub-fork.1.html>:.

- Պատառաքաղեք GitHub-ի պահոցն իր slug-ով.:

`hub fork {{tldr-pages/tldr}}`

- Պատառաքաղեք GitHub պահեստը իր URL-ով.:

`hub fork {{https://github.com/tldr-pages/tldr}}`

- Fork ընթացիկ GitHub պահոցը, սահմանեք հեռավոր անունը սկզբնաղբյուր.:

`hub fork --remote-name {{origin}}`
