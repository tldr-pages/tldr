# ծույլ

> Տերմինալային միջերես Docker կոնտեյներների, պատկերների, ծավալների և այլնի կառավարման համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/jesseduffield/lazydocker>:.

- Բաց Lazydocker:

`lazydocker`

- Ցուցադրել ընթացիկ լռելյայն կոնֆիգուրացիան.:

`lazydocker --config`

- Գործարկեք Lazydocker-ը վրիպազերծման ռեժիմում.:

`lazydocker --debug`

- Նշեք այլընտրանքային Docker Compose ֆայլը.:

`lazydocker --file {{path/to/docker-compose.yml}}`

- Ցուցադրել օգնությունը.:

`lazydocker --help`

- Ցուցադրման տարբերակը:

`lazydocker --version`
