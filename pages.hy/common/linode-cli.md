# linode-cli

> Կառավարեք Linode ամպային ծառայությունները:.
> Որոշ ենթահրամաններ, ինչպիսիք են `events`-ն, ունեն իրենց օգտագործման փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://techdocs.akamai.com/cloud-computing/docs/getting-started-with-the-linode-cli>:.

- Թվարկեք բոլոր լինոդները.:

`linode-cli linodes list`

- Դիտեք փաստաթղթերը Linode հաշիվները կառավարելու համար.:

`tldr linode-cli account`

- Դիտեք փաստաթղթերը Linodes-ի կառավարման համար.:

`tldr linode-cli linodes`

- Դիտեք փաստաթղթերը Linode Kubernetes Engine (LKE) կլաստերների կառավարման համար.:

`tldr linode-cli lke`

- Դիտեք փաստաթղթերը NodeBalancers-ի կառավարման համար.:

`tldr linode-cli nodebalancers`

- Դիտեք փաստաթղթերը օբյեկտների պահեստավորման կառավարման համար.:

`tldr linode-cli object-storage`

- Դիտեք փաստաթղթերը տիրույթների և DNS կազմաձևման կառավարման համար.:

`tldr linode-cli domains`

- Դիտեք փաստաթղթերը Linode ծավալները կառավարելու համար.:

`tldr linode-cli volumes`
