# պոչի կշեռքի հավաքածու

> Փոխեք նշված նախապատվությունները:.
> Այս ընտրանքները կարող են փոխվել նաև սկզբնական տեղադրման ժամանակ:.
> Այս հրամանը կարող է սահմանել էջում փաստաթղթավորված բոլոր տարբերակները՝ `tailscale up`-ի համար:.
> Լրացուցիչ տեղեկություններ. <https://tailscale.com/kb/1080/cli#set>:.

- Դիտեք փաստաթղթերը `tailscale up`-ի համար.:

`tldr tailscale up`
