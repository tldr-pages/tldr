# php yii

> Yii Framework-ի հրամանի տող ինտերֆեյսը:.
> Լրացուցիչ տեղեկություններ. <https://www.yiiframework.com/doc/guide/2.0/en/tutorial-console>:.

- Գործարկեք PHP-ի ներկառուցված վեբ սերվերը ընթացիկ Yii հավելվածի համար.:

`php yii {{serve}}`

- Ստեղծեք վերահսկիչ, դիտումներ և հարակից ֆայլեր CRUD գործողությունների համար նշված մոդելի դասի վրա.:

`php yii {{gii/crud}} --modelClass={{ModelName}} --controllerClass={{ControllerName}}`

- Ցուցադրել օգնությունը.:

`php yii {{help}}`
