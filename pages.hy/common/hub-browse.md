# հանգույց զննել

> Բացեք GitHub պահոց բրաուզերում կամ տպեք URL-ը:.
> Լրացուցիչ տեղեկություններ. <https://hub.github.com/hub-browse.1.html>:.

- Բացեք ընթացիկ պահոցի գլխավոր էջը լռելյայն վեբ բրաուզերում.:

`hub browse`

- Բացեք որոշակի պահեստի գլխավոր էջը լռելյայն վեբ բրաուզերում.:

`hub browse {{owner}}/{{repository}}`

- Բացեք որոշակի պահեստի ենթաէջը լռելյայն վեբ բրաուզերում, ենթաէջը կարող է լինել «wiki», «commits», «issues» կամ այլ (լռելյայն՝ «tree»):

`hub browse {{owner}}/{{repository}} {{subpage}}`
