# npm սկզբից

> Ստեղծեք `package.json` ֆայլ:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-init/>:.

- Նախաձեռնեք նոր փաթեթ հուշումներով.:

`npm init`

- Նախաձեռնեք նոր փաթեթ լռելյայն արժեքներով.:

`npm init {{[-y|--yes]}}`

- Նախաձեռնեք նոր փաթեթ՝ օգտագործելով հատուկ սկզբնավորիչ.:

`npm init {{create-react-app}} {{my-app}}`
