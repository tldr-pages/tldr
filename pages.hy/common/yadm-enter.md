# յադմ մտեք

> Գործարկեք ենթաշելլը բոլոր Git փոփոխականներով: Այս ենթափեղկը կարող է օգտագործվել Git հրամանների միջոցով հեշտությամբ փոխազդելու տեղական yadm պահեստի հետ:.
> Սա կարող է օգտակար լինել, եթե դուք օգտագործում եք գործիք, որն ուղղակիորեն օգտագործում է Git-ը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/TheLocehiliosan/yadm/blob/master/yadm.md#commands>:.

- Գործարկեք ենթահոլովակը բոլոր Git փոփոխականներով.:

`yadm enter`

- Դուրս եկեք ենթափեղկից.:

`exit`
