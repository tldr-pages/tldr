# npm որոնում

> Փաթեթների որոնում `npm` ռեեստրում:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-search/>:.

- Փաթեթի որոնում անունով.:

`npm {{[s|search]}} {{package}}`

- Փաթեթների որոնում որոշակի հիմնաբառով.:

`npm {{[s|search]}} {{keyword}}`

- Փաթեթների որոնում, ներառյալ մանրամասն տեղեկություններ (օրինակ՝ նկարագրություն, հեղինակ, տարբերակ).:

`npm {{[s|search]}} {{package}} --long`

- Որոնեք որոշակի հեղինակի կողմից պահպանվող փաթեթներ.:

`npm {{[s|search]}} --author {{author}}`

- Փաթեթների որոնում կոնկրետ կազմակերպության հետ.:

`npm {{[s|search]}} --scope {{organization}}`

- Փնտրեք փաթեթներ տերմինների հատուկ համակցությամբ.:

`npm {{[s|search]}} {{term1 term2 ...}}`
