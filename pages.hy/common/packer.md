# փաթեթավորող

> Կառուցեք ավտոմատ մեքենաների պատկերներ:.
> Լրացուցիչ տեղեկություններ. <https://developer.hashicorp.com/packer/docs/commands>:.

- Կառուցեք պատկեր.:

`packer build {{path/to/config.json}}`

- Ստուգեք Packer պատկերի կազմաձևի շարահյուսությունը.:

`packer validate {{path/to/config.json}}`

- Ձևաչափեք Packer պատկերի կազմաձևը.:

`packer fmt {{path/to/config.pkr.hcl}}`
