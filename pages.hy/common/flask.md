# տափաշիշ

> Ընդհանուր օգտակար սցենար Flask հավելվածների համար: Բեռնում է `$FLASK_APP` միջավայրի փոփոխականում սահմանված հավելվածը:.
> Լրացուցիչ տեղեկություններ. <https://flask.palletsprojects.com/en/stable/cli/>:.

- Գործարկեք զարգացման սերվեր.:

`flask run`

- Ցույց տալ երթուղիները հավելվածի համար.:

`flask routes`

- Գործարկեք Python ինտերակտիվ կեղևը հավելվածի համատեքստում.:

`flask shell`
