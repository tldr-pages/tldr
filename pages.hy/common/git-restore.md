# git վերականգնում

> Վերականգնել աշխատող ծառի ֆայլերը: Պահանջում է Git տարբերակ 2.23+:.
> Տես նաև՝ `git checkout`, `git reset`:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-restore>:.

- Վերականգնել չբեմադրված ֆայլը փուլային տարբերակին.:

`git restore {{path/to/file}}`

- Վերականգնել չբեմադրված ֆայլը կոնկրետ commit-ի տարբերակին.:

`git restore {{[-s|--source]}} {{commit}} {{path/to/file}}`

- Հեռացրեք հետևված ֆայլերի բոլոր չբեմադրված փոփոխությունները.:

`git restore :/`

- Բեմադրել ֆայլը՝:

`git restore {{[-S|--staged]}} {{path/to/file}}`

- Հեռացնել բոլոր ֆայլերը.:

`git restore {{[-S|--staged]}} :/`

- Հեռացրեք ֆայլերի բոլոր փոփոխությունները՝ և՛ բեմական, և՛ չբեմադրված:

`git restore {{[-W|--worktree]}} {{[-S|--staged]}} :/`

- Ինտերակտիվ կերպով ընտրեք ֆայլերի հատվածները վերականգնելու համար.:

`git restore {{[-p|--patch]}}`
