# tlmgr shell

> Սկսեք հայրենի TeX Live մենեջերի ինտերակտիվ կեղևը:.
> Լրացուցիչ տեղեկություններ. <https://www.tug.org/texlive/doc/tlmgr.html#shell>:.

- Սկսեք `tlmgr`-ի ինտերակտիվ կեղևը՝:

`tlmgr shell`

- [Ինտերակտիվ] Գործարկեք ցանկացած `tlmgr` ենթահրաման ինտերակտիվ վահանակում.:

`{{subcommand}} {{arguments}}`

- [Ինտերակտիվ] Դուրս գալ ինտերակտիվ կեղևից.:

`quit`

- [Ինտերակտիվ] Թվարկեք TeX Live բոլոր փոփոխականները.:

`get`

- [Ինտերակտիվ] Ստացեք TeX Live փոփոխականի արժեքը.:

`get {{variable}}`

- [Ինտերակտիվ] Սահմանեք TeX Live փոփոխականի արժեքը.:

`set {{variable}} {{value}}`

- [Ինտերակտիվ] Վերագործարկեք ինտերակտիվ կեղևը.:

`restart`

- [Ինտերակտիվ] Ցուցադրել ընթացիկ արձանագրության տարբերակը.:

`protocol`
