# git ստեղծել-մասնաճյուղ

> Ստեղծեք Git մասնաճյուղը պահոցում:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/git-create-branch>:.

- Ստեղծեք տեղական մասնաճյուղ.:

`git create-branch {{branch_name}}`

- Ստեղծեք մասնաճյուղ տեղական և ծագմամբ.:

`git create-branch {{[-r|--remote]}} {{branch_name}}`

- Ստեղծեք մասնաճյուղ տեղական և վերևում (պատառաքաղների միջոցով).:

`git create-branch {{[-r|--remote]}} upstream {{branch_name}}`
