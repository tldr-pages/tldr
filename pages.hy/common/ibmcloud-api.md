# ibmcloud api

> Սահմանեք կամ դիտեք IBM Cloud API-ի վերջնակետը:.
> Լրացուցիչ տեղեկություններ. <https://cloud.ibm.com/docs/cli?topic=cli-ibmcloud_cli#ibmcloud_api>:.

- Դիտեք ընթացիկ API-ի վերջնակետը.:

`ibmcloud api`

- Սահմանեք API-ի վերջնակետը `cloud.ibm.com`:

`ibmcloud api cloud.ibm.com`

- Սահմանեք մասնավոր API վերջնակետ.:

`ibmcloud api private.cloud.ibm.com`

- Օգտագործեք VPC կապ մասնավոր վերջնակետի համար.:

`ibmcloud api private.cloud.ibm.com --vpc`

- Շրջանցեք HTTP հարցումների SSL վավերացումը.:

`ibmcloud api https://cloud.ibm.com --skip-ssl-validation`

- Հեռացրեք API-ի վերջնակետի կարգավորումը.:

`ibmcloud api --unset`
