# git cherry

> Գտեք պարտավորություններ, որոնք դեռ պետք է կիրառվեն հոսանքին հակառակ:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-cherry>:.

- Ցույց տալ պարտավորությունները (և դրանց հաղորդագրությունները) համարժեք պարտավորություններով դեպի վերև՝:

`git cherry {{[-v|--verbose]}}`

- Նշեք տարբեր վերին և թեմատիկ ճյուղ.:

`git cherry {{origin}} {{topic}}`

- Limit-ը պարտավորվում է նրանց նկատմամբ, ովքեր գտնվում են տվյալ սահմանաչափի մեջ.:

`git cherry {{origin}} {{topic}} {{base}}`
