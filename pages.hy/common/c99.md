# c99

> Կազմել C ծրագրեր ըստ ISO C ստանդարտի:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/c99>:.

- Կազմել աղբյուրի ֆայլ(ներ) և ստեղծել գործարկվող.:

`c99 {{file.c}}`

- Կազմեք սկզբնաղբյուր ֆայլ(ներ) և նշեք գործարկվող [o]utput ֆայլի անունը.:

`c99 -o {{executable_name}} {{file.c}}`

- Կազմել սկզբնաղբյուր ֆայլ(ներ) և ստեղծել օբյեկտի ֆայլ(ներ).:

`c99 -c {{file.c}}`

- Կազմել սկզբնաղբյուր ֆայլ(ներ), կապել օբյեկտի ֆայլ(ներ) և ստեղծել գործարկվող.:

`c99 {{file.c}} {{file.o}}`
