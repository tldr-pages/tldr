# մատուցել

> Ստատիկ ֆայլերի սպասարկում և գրացուցակի ցուցակ:.
> Լրացուցիչ տեղեկություններ. <https://github.com/vercel/serve>:.

- Սկսեք HTTP սերվերը, որը լսում է լռելյայն նավահանգիստը՝ ընթացիկ գրացուցակը սպասարկելու համար.:

`serve`

- Սկսեք HTTP սերվերը որոշակի [p]ort-ի վրա՝ որոշակի գրացուցակ սպասարկելու համար.:

`serve -p {{port}} {{path/to/directory}}`

- Սկսեք HTTP սերվերը CORS-ով միացված՝ ներառելով `Access-Control-Allow-Origin: *` վերնագիրը բոլոր պատասխաններում.:

`serve {{[-C|--cors]}}`

- Սկսեք HTTP սերվերը լռելյայն միացքի վրա՝ վերագրելով բոլոր չգտնված հարցումները `index.html` ֆայլին.:

`serve {{[-s|--single]}}`

- Սկսեք HTTPS սերվերը լռելյայն նավահանգստի վրա՝ օգտագործելով նշված վկայագիրը.:

`serve --ssl-cert {{path/to/cert.pem}} --ssl-key {{path/to/key.pem}}`

- Սկսեք HTTP սերվերը լռելյայն նավահանգստում, օգտագործելով հատուկ կազմաձևման ֆայլ.:

`serve {{[-c|--config]}} {{path/to/serve.json}}`

- Ցուցադրել օգնությունը.:

`serve --help`
