# դոպլերային նախագծեր

> Կառավարեք Դոպլերային նախագծերը:.
> Լրացուցիչ տեղեկություններ. <https://docs.doppler.com/docs/cli>:.

- Ստացեք բոլոր նախագծերը.:

`doppler projects`

- Ստացեք տեղեկատվություն նախագծի համար.:

`doppler projects get {{name|project_id}}`

- Ստեղծել նախագիծ.:

`doppler projects create {{name}}`

- Թարմացրեք նախագծի անվանումը և նկարագրությունը.:

`doppler projects update {{name|project_id}} --name "{{new_name}}" --description "{{new_description}}"`

- Ջնջել նախագիծը.:

`doppler projects delete {{name|project_id}}`
