# sdk

> Կառավարեք ծրագրային ապահովման մշակման բազմաթիվ փաթեթների զուգահեռ տարբերակները:.
> Աջակցում է Java, Groovy, Scala, Kotlin, Gradle, Maven, Vert.x և շատ ուրիշներ:.
> Լրացուցիչ տեղեկություններ. <https://sdkman.io/usage/>:.

- Տեղադրեք SDK տարբերակը.:

`sdk install {{sdk_name}} {{sdk_version}}`

- Օգտագործեք SDK-ի հատուկ տարբերակ ընթացիկ տերմինալի նստաշրջանի համար.:

`sdk use {{sdk_name}} {{sdk_version}}`

- Ցույց տալ ցանկացած հասանելի SDK-ի կայուն տարբերակը.:

`sdk current {{sdk_name}}`

- Ցուցադրել բոլոր տեղադրված SDK-ների կայուն տարբերակները.:

`sdk current`

- Թվարկեք բոլոր հասանելի SDK-ները.:

`sdk list`

- Թվարկեք SDK-ի բոլոր տարբերակները.:

`sdk list {{sdk_name}}`

- Թարմացրեք SDK-ն վերջին կայուն տարբերակին.:

`sdk upgrade {{sdk_name}}`

- Տեղահանեք հատուկ SDK տարբերակ.:

`sdk rm {{sdk_name}} {{sdk_version}}`
