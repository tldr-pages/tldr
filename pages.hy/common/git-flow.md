# git հոսք

> Git ընդլայնումների հավաքածու՝ պահեստի բարձր մակարդակի գործառնություններ ապահովելու համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/nvie/gitflow>:.

- Նախաձեռնեք այն գոյություն ունեցող Git պահոցի ներսում.:

`git flow init`

- Սկսեք զարգանալ առանձնահատկությունների մասնաճյուղում, որը հիմնված է `develop`-ի վրա.:

`git flow feature start {{feature}}`

- Ավարտեք մշակումը գործառույթի ճյուղի վրա՝ միացնելով այն `develop` մասնաճյուղին և ջնջելով այն՝:

`git flow feature finish {{feature}}`

- Հրապարակեք գործառույթ հեռավոր սերվերի վրա.:

`git flow feature publish {{feature}}`

- Ստացեք մեկ այլ օգտվողի կողմից հրապարակված հատկություն.:

`git flow feature pull origin {{feature}}`
