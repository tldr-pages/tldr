# git mergetool

> Գործարկեք միաձուլման կոնֆլիկտների լուծման գործիքներ՝ միաձուլման կոնֆլիկտները լուծելու համար:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-mergetool>:.

- Գործարկեք լռելյայն միաձուլման գործիքը՝ կոնֆլիկտները լուծելու համար.:

`git mergetool`

- Ցուցակեք միաձուլման վավեր գործիքներ.:

`git mergetool --tool-help`

- Գործարկեք միաձուլման գործիքը, որը նույնականացվում է անունով.:

`git mergetool {{[-t|--tool]}} {{tool_name}}`

- Միաձուլման գործիքի յուրաքանչյուր կանչից առաջ մի հուշեք.:

`git mergetool {{[-y|--no-prompt]}}`

- Հստակորեն օգտագործեք GUI-ի միաձուլման գործիքը (տես `merge.guitool` կազմաձևման փոփոխականը).:

`git mergetool {{[-g|--gui]}}`

- Հստակորեն օգտագործեք սովորական միաձուլման գործիքը (տես `merge.tool` կազմաձևման փոփոխականը).:

`git mergetool --no-gui`
