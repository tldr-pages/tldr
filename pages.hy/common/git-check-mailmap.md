# git check-mailmap

> Ցույց տալ կոնտակտների կանոնական անուններն ու էլ. հասցեները:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-check-mailmap>:.

- Փնտրեք էլփոստի հասցեի հետ կապված կանոնական անունը.:

`git check-mailmap {{email@example.com}}`

- Օգտագործեք նշված փոստի քարտեզի ֆայլը, ի լրումն կանխադրվածների.:

`git check-mailmap --mailmap-file {{path/to/file}} {{email@example.com}}`
