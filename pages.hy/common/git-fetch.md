# git fetch

> Ներբեռնեք օբյեկտները և ռեֆերատները հեռավոր պահոցից:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-fetch>:.

- Ստացեք վերջին փոփոխությունները կանխադրված հեռավոր վերին շտեմարանից (եթե սահմանված է).:

`git fetch`

- Վերցրեք նոր մասնաճյուղեր որոշակի հեռավոր վերին շտեմարանից.:

`git fetch {{remote_name}}`

- Ստացեք վերջին փոփոխությունները բոլոր հեռավոր վերին շտեմարաններից.:

`git fetch --all`

- Վերցրեք նաև պիտակները հեռավոր վերընթաց պահոցից.:

`git fetch {{[-t|--tags]}}`

- Ջնջել տեղական հղումները հեռավոր մասնաճյուղերին, որոնք ջնջվել են հոսանքին հակառակ.:

`git fetch {{[-p|--prune]}}`

- Խորացրեք ընթացիկ մակերեսային ճյուղը 2 պարտավորությամբ.:

`git fetch --deepen 2`

- Թարմացրեք `main` մասնաճյուղը՝ առանց դրան անցնելու (համարժեք `git pull`-ին):

`git fetch {{origin}} main:main`
