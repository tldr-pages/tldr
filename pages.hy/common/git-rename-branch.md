# git rename-branch

> Վերանվանել Git մասնաճյուղը:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-rename-branch>:.

- Վերանվանեք մասնաճյուղը, որտեղ այժմ գտնվում եք.:

`git rename-branch {{new_branch_name}}`

- Վերանվանել կոնկրետ մասնաճյուղ.:

`git rename-branch {{old_branch_name}} {{new_branch_name}}`
