#կայծ

> Laravel Spark գործիքը:.
> Լրացուցիչ տեղեկություններ. <https://spark.laravel.com/>:.

- Գրանցեք ձեր API նշանը.:

`spark register {{token}}`

- Ցուցադրել ներկայումս գրանցված API նշանը.:

`spark token`

- Ստեղծեք նոր Spark նախագիծ.:

`spark new {{project_name}}`

- Ստեղծեք նոր Spark նախագիծ Braintree կոճղերով.:

`spark new {{project_name}} --braintree`

- Ստեղծեք նոր Spark նախագիծ՝ թիմի վրա հիմնված վճարային կոճակներով.:

`spark new {{project_name}} --team-billing`
