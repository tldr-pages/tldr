# wasm2c

> Փոխակերպեք ֆայլը WebAssembly երկուական ձևաչափից C աղբյուրի ֆայլի և վերնագրի:.
> Լրացուցիչ տեղեկություններ. <https://webassembly.github.io/wabt/doc/wasm2c.1.html>:.

- Ֆայլը փոխարկեք C աղբյուրի ֆայլի և վերնագրի և ցուցադրեք այն վահանակում.:

`wasm2c {{file.wasm}}`

- Գրեք ելքը տվյալ ֆայլում (`file.h` լրացուցիչ ձևավորվում է).:

`wasm2c {{file.wasm}} {{[-o|--output]}} {{file.c}}`
