# mpremote

> Հեռակառավարել MicroPython սարքերը:.
> Լրացուցիչ տեղեկություններ. <https://docs.micropython.org/en/latest/reference/mpremote.html>:.

- Նշեք բոլոր միացված MicroPython սարքերը.:

`mpremote connect list`

- Բացեք ինտերակտիվ REPL նիստ միացված սարքով.:

`mpremote connect {{device}}`

- Գործարկեք տեղական սցենար միացված սարքի վրա.:

`mpremote run {{path/to/script.py}}`

- Սարքի վրա տեղադրեք տեղական գրացուցակ.:

`mpremote mount {{path/to/directory}}`

- Տեղադրեք mip փաթեթ սարքի վրա.:

`mpremote mip install {{package}}`
