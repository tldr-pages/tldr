# omz

> Oh My Zsh հրամանի տող գործիք:.
> Լրացուցիչ տեղեկություններ. <https://github.com/ohmyzsh/ohmyzsh>:.

- Թարմացրեք Oh My Zsh:

`omz update`

- Տպեք փոփոխությունները Oh My Zsh-ի վերջին թարմացումից.:

`omz changelog`

- Վերագործարկեք ընթացիկ Zsh նիստը և Oh My Zsh:

`omz reload`

- Թվարկեք բոլոր հասանելի հավելվածները.:

`omz plugin list`

- Թվարկեք բոլոր միացված պլագինները.:

`omz plugin list --enabled`

- Միացնել/անջատել Oh My Zsh հավելվածը.:

`omz plugin {{enable|disable}} {{plugin}}`

- Թվարկեք բոլոր առկա թեմաները.:

`omz theme list`

- Սահմանեք Oh My Zsh թեմա `~/.zshrc`-ում:

`omz theme set {{theme}}`
