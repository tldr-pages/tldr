# git rscp

> Հակադարձ `git scp` - պատճենեք ֆայլերը հեռավոր պահեստի աշխատանքային գրացուցակից ընթացիկ աշխատող ծառին:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-scp>:.

- Պատճենեք որոշակի ֆայլեր հեռակառավարման վահանակից.:

`git rscp {{remote_name}} {{path/to/file1 path/to/file2 ...}}`

- Պատճենեք որոշակի գրացուցակ հեռակառավարման վահանակից.:

`git rscp {{remote_name}} {{path/to/directory}}`
