# kiterunner

> Համատեքստային վեբ սկաներ API-ի ուղիները և վեբ վերջնակետերը հայտնաբերելու համար՝ օգտագործելով բառացանկեր և kitebuilder սխեմաներ:.
> Լրացուցիչ տեղեկություններ. <https://github.com/assetnote/kiterunner#usage>:.

- Դիտեք փաստաթղթերը bruteforcing API ուղիների և վեբ վերջնակետերի համար.:

`tldr kiterunner brute`

- Դիտեք փաստաթղթերը kitebuilder բառացանկերի հետ միաժամանակ սկանավորելու համար.:

`tldr kiterunner scan`

- Դիտեք փաստաթղթերը kitebuilder սխեմաները շահարկելու համար.:

`tldr kiterunner kb`

- Դիտեք փաստաթղթերը պահված և հեռավոր բառացանկերը կառավարելու համար.:

`tldr kiterunner wordlist`
