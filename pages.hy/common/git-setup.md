# git կարգավորում

> Ստեղծեք Git պահոց գրացուցակում և կատարեք բոլոր ֆայլերը:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-setup>:.

- Ստեղծեք Git պահոց ընթացիկ գրացուցակում և կատարեք բոլոր ֆայլերը.:

`git setup`

- Ստեղծեք Git պահոց հատուկ գրացուցակում և կատարեք բոլոր ֆայլերը.:

`git setup {{path/to/directory}}`
