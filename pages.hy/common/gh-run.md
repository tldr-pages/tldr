# gh վազում

> Դիտեք, գործարկեք և դիտեք GitHub Actions-ի վերջին գործարկումները:.
> Լրացուցիչ տեղեկություններ. <https://cli.github.com/manual/gh_run>:.

- Ինտերակտիվ կերպով ընտրեք վազք՝ աշխատանքների մասին տեղեկությունները տեսնելու համար.:

`gh run view`

- Ցուցադրել տեղեկատվություն կոնկրետ վազքի մասին.:

`gh run view {{workflow_run_number}}`

- Ցուցադրել տեղեկատվություն աշխատանքի քայլերի մասին.:

`gh run view {{[-j|--job]}} {{job_number}}`

- Ցուցադրել աշխատանքի գրանցամատյանը.:

`gh run view {{[-j|--job]}} {{job_number}} --log`

- Ստուգեք որոշակի աշխատանքային հոսք և դուրս եկեք ոչ զրոյական կարգավիճակով, եթե գործարկումը ձախողվեց.:

`gh run view {{workflow_run_number}} --exit-status && {{echo "run pending or passed"}}`

- Ինտերակտիվ կերպով ընտրեք ակտիվ գործարկում և սպասեք, մինչև այն ավարտվի.:

`gh run watch`

- Ցուցադրեք առաջադրանքները և սպասեք, մինչև այն ավարտվի.:

`gh run watch {{workflow_run_number}}`

- Վերագործարկեք որոշակի աշխատանքային հոսք.:

`gh run rerun {{workflow_run_number}}`
