# git-sizer

> Հաշվում է Git պահեստի չափերի տարբեր չափորոշիչներ և զգուշացնում ձեզ ցանկացածի մասին, որը կարող է խնդիրներ կամ անհարմարություններ առաջացնել:.
> Լրացուցիչ տեղեկություններ. <https://github.com/github/git-sizer>:.

- Հաղորդեք միայն այն վիճակագրությունը, որն ունի 0-ից ավելի մտահոգության մակարդակ.:

`git-sizer`

- Հաղորդել բոլոր վիճակագրությունը.:

`git-sizer -v`

- Տես լրացուցիչ ընտրանքներ.:

`git-sizer -h`
