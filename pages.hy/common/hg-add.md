# hg ավելացնել

> Ավելացնում է նշված ֆայլերը բեմականացման տարածքում՝ Mercurial-ի հաջորդ կատարման համար:.
> Լրացուցիչ տեղեկություններ. <https://www.mercurial-scm.org/help/commands/add>:.

- Ավելացնել ֆայլեր կամ գրացուցակներ բեմադրության տարածքում.:

`hg add {{path/to/file}}`

- Ավելացնել բոլոր չբեմադրված ֆայլերը, որոնք համապատասխանում են նշված օրինակին.:

`hg add {{[-I|--include]}} {{pattern}}`

- Ավելացրեք բոլոր չբեմադրված ֆայլերը՝ բացառությամբ նրանց, որոնք համապատասխանում են նշված օրինակին.:

`hg add {{[-X|--exclude]}} {{pattern}}`

- Ռեկուրսիվ կերպով ավելացրեք ենթապահոցներ.:

`hg add {{[-S|--subrepos]}}`

- Կատարեք թեստային վազք՝ առանց որևէ գործողություն կատարելու.:

`hg add {{[-n|--dry-run]}}`
