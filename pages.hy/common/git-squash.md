# git squash

> Squash մի քանի պարտավորություններ մեկ commit-ի մեջ:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/git-squash>:.

- Միավորել բոլոր պարտավորությունները կոնկրետ ճյուղից ընթացիկ ճյուղի մեջ որպես մեկ պարտավորություն.:

`git squash {{source_branch}}`

- Squash բոլոր commit-ները՝ սկսած ընթացիկ ճյուղի կոնկրետ commit-ից.:

`git squash {{commit}}`

- Փակեք `n` վերջին պարտավորությունները և կատարեք հաղորդագրություն՝:

`git squash HEAD~{{n}} "{{message}}"`

- Փակեք `n` վերջին պարտավորությունները և միացրեք բոլոր առանձին հաղորդագրությունները.:

`git squash --squash-msg HEAD~{{n}}`
