# bun թարմացում

> Թարմացրեք կախվածությունները Bun նախագծում:.
> Լրացուցիչ տեղեկություններ. <https://bun.com/docs/pm/cli/update>:.

- Թարմացրեք բոլոր կախվածությունները.:

`bun update`

- Թարմացրեք վերջին տարբերակին, անկախ դրա համատեղելիությունից.:

`bun update --latest`

- Թարմացրեք հատուկ կախվածությունը.:

`bun update {{package_name}}`

- Թարմացրեք կախվածությունը որոշակի տարբերակի.:

`bun update {{package_name}}@{{version}}`

- Թարմացրեք փաթեթները ինտերակտիվ կերպով.:

`bun update {{[-i|--interactive]}}`

- Թարմացրեք կախվածությունները ռեկուրսիվորեն բոլոր աշխատանքային տարածքներում.:

`bun update {{[-r|--recursive]}}`

- Թարմացրեք կախվածությունները ինտերակտիվ և ռեկուրսիվ.:

`bun update {{[-i|--interactive]}} {{[-r|--recursive]}}`
