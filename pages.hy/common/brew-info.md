# եփելու տեղեկություն

> Ցուցադրել Homebrew-ի տեղադրման և բանաձևերի մասին տեղեկությունները:.
> Լրացուցիչ տեղեկություններ. <https://docs.brew.sh/Manpage#info-abv-options-formulacask->:.

- Ցուցադրել վիճակագրությունը Homebrew-ի տեղադրման համար.:

`brew info`

- Ցուցադրել ավելի շատ տեղեկատվություն բանաձևի կամ տակառի մասին.:

`brew info {{formula|cask}}`

- Ցուցադրել բանաձևի կամ տակառի մասին մանրամասն տեղեկատվություն.:

`brew info {{[-v|--verbose]}} {{formula|cask}}`

- Ցուցադրել բանաձևի կամ տակառի մասին տեղեկատվություն JSON ձևաչափով.:

`brew info --json {{formula|cask}}`

- Ներկա պահին տեղադրված բանաձևերի JSON տպեք.:

`brew info --json --installed`

- Ցուցադրել օգնությունը.:

`brew info {{[-h|--help]}}`
