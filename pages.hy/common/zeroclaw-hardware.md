# zeroclaw սարքավորում

> Բացահայտեք և ներքաշեք USB ապարատը ZeroClaw-ի համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/zeroclaw-labs/zeroclaw#quick-start>:.

- Ցուցակեք հասանելի USB ապարատային սարքերը.:

`zeroclaw hardware discover`

- Ցուցադրել օգնությունը.:

`zeroclaw hardware {{[-h|--help]}}`
