# կռունկի ինդեքս

> Փոփոխել պատկերի ինդեքսը:.
> Որոշ ենթահրամաններ, ինչպիսիք են `append`-ը և `filter`-ն, ունեն իրենց օգտագործման փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/google/go-containerregistry/blob/main/cmd/crane/doc/crane_index.md>:.

- Փոփոխել պատկերի ինդեքսը.:

`crane index`

- Փոփոխեք պատկերի ինդեքսը ենթահրամանով.:

`crane index {{subcommand}}`

- Ցուցադրել օգնությունը.:

`crane index {{[-h|--help]}}`
