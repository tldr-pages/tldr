# git թերթել

> Դիտեք վերին հոսքի պահոցը լռելյայն դիտարկիչում:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-browse>:.

- Բացեք առաջին վերին հոսքը լռելյայն դիտարկիչում.:

`git browse`

- Լռելյայն բրաուզերում բացեք որոշակի վերին հոսք.:

`git browse {{upstream}}`
