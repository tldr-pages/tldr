# gh pr ստուգումներ

> Դիտեք CI ստուգումները GitHub-ի ձգման հարցումի համար:.
> Լրացուցիչ տեղեկություններ. <https://cli.github.com/manual/gh_pr_checks>:.

- Ցույց տալ ընթացիկ մասնաճյուղի ձգման խնդրանքի չեկերը.:

`gh pr checks`

- Ցույց տալ չեկեր հատուկ ձգման խնդրանքի համար.:

`gh pr checks {{pr_number}}`

- Դիտեք ստուգումները և թարմացրեք իրական ժամանակում մինչև ավարտը.:

`gh pr checks {{pr_number}} --watch`

- Ցույց տալ միայն պահանջվող ստուգումները.:

`gh pr checks {{pr_number}} --required`
