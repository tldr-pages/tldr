#վիտեստ

> Vite-ի համար կառուցված արագ, ժամանակակից փորձարկման շրջանակ, որն առաջարկում է անխափան ինտեգրում, TypeScript-ի աջակցություն և Jest-ի հետ համատեղելի API միավորի, ինտեգրման և լուսանկարի փորձարկման համար:.
> Լրացուցիչ տեղեկություններ. <https://vitest.dev/guide/cli.html>:.

- Գործարկել բոլոր առկա թեստերը.:

`vitest run`

- Գործարկեք թեստային փաթեթները տվյալ ֆայլերից.:

`vitest run {{path/to/file1 path/to/file2 ...}}`

- Գործարկեք թեստային փաթեթները ընթացիկ և ենթագրքերում գտնվող ֆայլերից, որոնց ուղիները համապատասխանում են տրված `regex`-ին:

`vitest run {{regex1 regex2 ...}}`

- Գործարկեք թեստերը, որոնց անունները համապատասխանում են նշված `regex`-ին:

`vitest run {{[-t|--testNamePattern]}} {{regex}}`

- Դիտեք ֆայլերը փոփոխությունների համար և ավտոմատ կերպով վերագործարկեք համապատասխան թեստերը.:

`vitest`

- Գործարկել թեստերը ծածկույթով.:

`vitest run --coverage`

- Կատարեք բոլոր թեստերը, բայց դադարում է առաջին թեստի ձախողումից անմիջապես հետո.:

`vitest run --bail=1`

- Ցուցադրել օգնությունը.:

`vitest --help`
