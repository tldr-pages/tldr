# git pull

> Վերցրեք մասնաճյուղը հեռավոր պահոցից և միացրեք այն տեղական պահեստին:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-pull>:.

- Ներբեռնեք փոփոխությունները լռելյայն հեռավոր պահոցից և միացրեք այն.:

`git pull`

- Ներբեռնեք փոփոխությունները լռելյայն հեռավոր պահոցից և օգտագործեք արագ առաջընթաց.:

`git pull {{[-r|--rebase]}}`

- Ներբեռնեք փոփոխությունները տվյալ հեռավոր պահոցից և ճյուղից, այնուհետև դրանք միացրեք `HEAD`-ին:

`git pull {{remote_name}} {{branch}}`
