# git utimes

> Փոխեք ֆայլերի փոփոխման ժամանակը մինչև դրանց վերջին կատարման ամսաթիվը: Չի դիպչում ֆայլերին, որոնք գտնվում են աշխատանքային ծառի կամ ինդեքսում:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-utimes>:.

- Փոփոխեք բոլոր ֆայլերի փոփոխման ժամանակը մինչև դրանց վերջին կատարման ամսաթիվը.:

`git utimes`

- Փոխեք ֆայլերի փոփոխման ժամանակը, որոնք ավելի նոր են, քան դրանց վերջին կատարման ամսաթիվը, պահպանելով տեղական պահոցից կատարված ֆայլերի փոփոխման սկզբնական ժամանակը.:

`git utimes --newer`
