# git cp

> Պատճենեք գոյություն ունեցող ֆայլը նոր վայրում՝ պահպանելով պատմությունը:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-cp>:.

- Պատճենեք գոյություն ունեցող ֆայլը Git ռեպոում՝ մնալով նույն գրացուցակում.:

`git cp {{file}} {{new_file}}`

- Պատճենեք գոյություն ունեցող ֆայլը Git ռեպոում և տեղադրեք այն մեկ այլ տեղ.:

`git cp {{path/to/file}} {{path/to/new_file}}`
