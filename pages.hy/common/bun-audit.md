# բուլկի աուդիտ

> Ստուգեք տեղադրված փաթեթները անվտանգության հայտնի խոցելիության համար:.
> Լրացուցիչ տեղեկություններ. <https://bun.com/docs/pm/cli/audit>:.

- Աուդիտ `bun.lock` ֆայլով նախագծում առկա բոլոր կախվածությունները.:

`bun audit`

- Ցույց տալ միայն խոցելիությունը որոշակի խստության մակարդակի վրա կամ ավելի բարձր.:

`bun audit --audit-level {{low|moderate|high|critical}}`

- Աուդիտ միայն արտադրական կախվածությունները.:

`bun audit --prod`

- Անտեսեք հատուկ CVE ID.:

`bun audit --ignore {{CVE-XXXX-YYYY}}`

- Արտադրեք չմշակված JSON զեկույցը.:

`bun audit --json`
