# հիմք32

> Կոդավորել կամ վերծանել ֆայլը կամ `stdin` Base32-ից մինչև `stdout`:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/base32>:.

- Կոդավորել ֆայլը.:

`base32 {{path/to/file}}`

- Փաթաթել կոդավորված ելքը որոշակի լայնությամբ (`0`-ն անջատում է փաթաթումը):

`base32 {{[-w|--wrap]}} {{0|76|...}} {{path/to/file}}`

- Վերծանել ֆայլը.:

`base32 {{[-d|--decode]}} {{path/to/file}}`

- Կոդավորում `stdin`-ից՝:

`{{command}} | base32`

- Վերծանել `stdin`-ից՝:

`{{command}} | base32 {{[-d|--decode]}}`
