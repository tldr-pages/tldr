# age-keygen

> Ստեղծեք `age` բանալիների զույգեր:.
> Տես նաև՝ `age`, `age-inspect`:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/age-keygen>:.

- Ստեղծեք բանալիների զույգ, պահեք այն չգաղտնագրված ֆայլում և տպեք հանրային բանալին `stdout`:

`age-keygen {{[-o|--output]}} {{path/to/file}}`

- Ստեղծեք հետքվանտային բանալիների զույգ, պահեք այն չգաղտնագրված ֆայլում և տպեք հանրային բանալին `stdout`:

`age-keygen -pq {{[-o|--output]}} {{path/to/file}}`

- Փոխակերպեք ինքնությունը[y]-ը ստացողի և տպեք հանրային բանալին `stdout`:

`age-keygen -y {{path/to/file}}`
