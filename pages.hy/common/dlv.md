# dlv

> Վրիպազերծիչ Go ծրագրավորման լեզվի համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/go-delve/delve/blob/master/Documentation/usage/dlv.md>:.

- Կազմեք և սկսեք վրիպազերծել հիմնական փաթեթը ընթացիկ գրացուցակում (լռելյայն, առանց փաստարկների).:

`dlv debug`

- Կազմեք և սկսեք վրիպազերծել որոշակի փաթեթ.:

`dlv debug {{package}} {{arguments}}`

- Կազմեք թեստային երկուական և սկսեք վրիպազերծել կազմված ծրագիրը.:

`dlv test`

- Միացեք անգլուխ կարգաբերման սերվերին.:

`dlv connect {{ip_address}}`

- Կցեք գործող գործընթացին և սկսեք վրիպազերծումը.:

`dlv attach {{pid}}`

- Կազմել և սկսել հետագծել ծրագիր.:

`dlv trace {{package}} --regexp '{{regex}}'`
