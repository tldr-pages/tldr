# pg_config

> Ցուցադրել տեղեկատվությունը տեղադրված PostgreSQL տարբերակի մասին:.
> Լրացուցիչ տեղեկություններ. <https://www.postgresql.org/docs/current/app-pgconfig.html>:.

- Ցույց տալ գրացուցակը, որտեղ տեղադրված են PostgreSQL երկուականները.:

`pg_config --bindir`

- Ցույց տալ գրադարանի ֆայլերի գրացուցակը.:

`pg_config --libdir`

- Ցույց տալ ներառման ֆայլերի գրացուցակը (C վերնագրեր).:

`pg_config --includedir`

- Ցուցադրել ընդհանուր տվյալների ֆայլերի գրացուցակը.:

`pg_config --sharedir`

- Ցույց տալ տեղանքի տվյալների գրացուցակը.:

`pg_config --localedir`

- Ցուցադրել օգնությունը.:

`pg_config {{[-?|--help]}}`

- Ցուցադրման տարբերակը:

`pg_config --version`
