# npm ls

> Տպել տեղադրված փաթեթները `stdout`-ում:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-ls/>:.

- Տպեք ընթացիկ նախագծում ուղղակի կախվածությունների բոլոր տարբերակները `stdout`-ում:

`npm {{[ls|list]}}`

- Տպել բոլոր տեղադրված փաթեթները, ներառյալ հասակակիցների կախվածությունը.:

`npm {{[ls|list]}} {{[-a|--all]}}`

- Տպել բոլոր գլոբալ տեղադրված փաթեթները.:

`npm {{[ls|list]}} {{[-g|--global]}}`

- Տպել կախվածությունները ընդլայնված տեղեկություններով.:

`npm {{[ls|list]}} {{[-l|--long]}}`

- Տպել կախվածությունները վերլուծելի ձևաչափով.:

`npm {{[ls|list]}} {{[-p|--parseable]}}`

- Տպել կախվածությունները JSON ձևաչափով.:

`npm {{[ls|list]}} --json`
