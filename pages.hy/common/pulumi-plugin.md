# pulumi plugin

> Ձեռքով կառավարեք լեզվի և ռեսուրսների մատակարարի հավելվածները:.
> Այլ հրամանները կառավարում են դրանք ինքնաբերաբար:.
> Լրացուցիչ տեղեկություններ. <https://www.pulumi.com/docs/iac/cli/commands/pulumi_plugin/>:.

- Թվարկեք բոլոր պլագինները ներբեռնված քեշի վրա.:

`pulumi plugin ls`

- Ցուցակեք ընթացիկ նախագծի կողմից օգտագործվող հավելումները JSON ձևաչափով.:

`pulumi plugin {{[-p|--project]}} {{[-j|--json]}}`

- Տեղադրեք plugin տեսակի (օրինակ՝ ռեսուրս) վերջին տարբերակով կամ կոնկրետ մեկով.:

`pulumi plugin install {{kind}} {{name}} {{version}}`

- Հեռացրեք plugin-ի տեսակը (օրինակ՝ ռեսուրսը) և ինտերակտիվ կերպով ընտրեք տարբերակ կամ տրամադրեք կոնկրետ մեկը.:

`pulumi plugin rm {{kind}} {{name}} {{version}}`

- Ցուցադրել օգնությունը.:

`pulumi plugin {{[-h|--help]}}`
