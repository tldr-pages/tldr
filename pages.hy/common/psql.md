# psql

> PostgreSQL հաճախորդ:.
> Լրացուցիչ տեղեկություններ. <https://www.postgresql.org/docs/current/app-psql.html>:.

- Միացեք տվյալների բազային: Լռելյայնորեն, այն միանում է տեղական վարդակից՝ օգտագործելով 5432 նավահանգիստը ներկայումս մուտք գործած օգտատիրոջ հետ.:

`psql {{database}}`

- Միացեք տվյալների շտեմարանին տվյալ սերվերի հոսթի վրա, որն աշխատում է տվյալ նավահանգստում, տվյալ օգտվողի անունով, առանց գաղտնաբառի հուշման.:

`psql {{[-h|--host]}} {{host}} {{[-p|--port]}} {{port}} {{[-U|--username]}} {{username}} {{database}}`

- Միացեք տվյալների բազային; օգտվողին կպահանջվի գաղտնաբառը.:

`psql {{[-h|--host]}} {{host}} {{[-p|--port]}} {{port}} {{[-U|--username]}} {{username}} {{[-W|--password]}} {{database}}`

- Կատարեք մեկ SQL հարցում կամ PostgreSQL հրաման տվյալ տվյալների բազայում (օգտակար է shell-ի սցենարներում).:

`psql {{[-c|--command]}} '{{query}}' {{database}}`

- Կատարել հրամաններ տվյալ տվյալների բազայի ֆայլից.:

`psql {{database}} {{[-f|--file]}} {{path/to/file.sql}}`
