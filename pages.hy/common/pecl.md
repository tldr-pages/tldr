# պեկլ

> Կառավարեք PHP ընդլայնումները PHP Extension համայնքի գրադարանից:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/pecl>:.

- Ցուցադրել առկա հրամանների ցանկը.:

`pecl`

- Տեղադրեք փաթեթ.:

`pecl install {{package}}`

- Տեղադրված փաթեթների ցանկ.:

`pecl list`

- Թարմացրեք բոլոր փաթեթները.:

`pecl upgrade`

- Թարմացրեք որոշակի փաթեթ.:

`pecl upgrade {{package}}`

- Տեղահանել կոնկրետ փաթեթ.:

`pecl uninstall {{package}}`

- Ցուցադրել տեղեկատվություն կոնկրետ փաթեթի մասին.:

`pecl info {{package}}`

- Թարմացրեք առկա փաթեթի աղբյուրները.:

`pecl update-channels`
