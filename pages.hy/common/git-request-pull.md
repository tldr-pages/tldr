# git հարցում-քաշեք

> Ստեղծեք հարցում՝ խնդրելով վերընթաց նախագծին փոփոխություններ կատարել իր ծառի մեջ:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-request-pull>:.

- Ստեղծեք հարցում, որն ամփոփում է փոփոխությունները v1.1 թողարկման և նշված ճյուղի միջև.:

`git request-pull {{v1.1}} {{https://example.com/project}} {{branch_name}}`

- Ստեղծեք հարցում, որն ամփոփում է փոփոխությունները v0.1 թողարկման միջև `foo` մասնաճյուղում և տեղական `bar` մասնաճյուղում:

`git request-pull {{v0.1}} {{https://example.com/project}} {{foo:bar}}`
