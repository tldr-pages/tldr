# git scp

> Պատճենեք ֆայլերը ընթացիկ աշխատանքային ծառից հեռավոր պահեստի աշխատանքային գրացուցակում:.
> `git-extras`-ի մի մասը: Օգտագործում է `rsync` ֆայլերը փոխանցելու համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-scp>:.

- Պատճենեք չբեմադրված ֆայլերը որոշակի հեռակառավարման վրա.:

`git scp {{remote_name}}`

- Պատճենեք բեմադրված և չբեմադրված ֆայլերը հեռակառավարման վահանակի վրա.:

`git scp {{remote_name}} HEAD`

- Պատճենել ֆայլերը, որոնք փոխվել են վերջին commit-ում և ցանկացած բեմադրված կամ չբեմադրված ֆայլեր հեռակառավարման վահանակում.:

`git scp {{remote_name}} HEAD~1`

- Պատճենեք հատուկ ֆայլեր հեռակառավարման վահանակի վրա.:

`git scp {{remote_name}} {{path/to/file1 path/to/file2 ...}}`

- Պատճենեք որոշակի գրացուցակ հեռակառավարման վահանակում.:

`git scp {{remote_name}} {{path/to/directory}}`
