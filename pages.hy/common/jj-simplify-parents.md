# jj պարզեցնել-ծնողներ

> Պարզեցրեք մայր եզրերը նշված վերանայման(ների) համար:.
> Օրինակ, «A -> B -> C | A -> C»-ը պարզեցվում է «A -> B -> C»-ի:.
> Լրացուցիչ տեղեկություններ. <https://docs.jj-vcs.dev/latest/cli-reference/#jj-simplify-parents>:.

- Պարզեցնել տրված վերանայումների մայր եզրերը.:

`jj simplify-parents {{[-r|--revisions]}} {{revsets}}`

- Պարզեցրեք տրված վերանայումների մայր եզրերը և նրանց ժառանգների ծառերը.:

`jj simplify-parents {{[-s|--source]}} {{revsets}}`
