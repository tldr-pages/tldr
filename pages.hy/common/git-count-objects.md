# git count-objects

> Հաշվեք չփաթեթավորված օբյեկտների քանակը և դրանց սկավառակի սպառումը:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-count-objects>:.

- Հաշվեք բոլոր օբյեկտները և ցուցադրեք սկավառակի ընդհանուր օգտագործումը.:

`git count-objects`

- Ցուցադրել բոլոր օբյեկտների քանակը և դրանց ընդհանուր սկավառակի օգտագործումը՝ ցուցադրելով չափերը մարդու համար ընթեռնելի միավորներով.:

`git count-objects {{[-H|--human-readable]}}`

- Ցուցադրել ավելի մանրամասն տեղեկատվություն.:

`git count-objects {{[-v|--verbose]}}`

- Ցուցադրել ավելի մանրամասն տեղեկատվություն՝ ցուցադրելով չափերը մարդու համար ընթեռնելի միավորներով.:

`git count-objects {{[-H|--human-readable]}} {{[-v|--verbose]}}`
