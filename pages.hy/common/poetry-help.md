# պոեզիայի օգնություն

> Ցուցադրել գլոբալ օգնությունը կամ օգնություն կոնկրետ `poetry` հրամանի համար:.
> Լրացուցիչ տեղեկություններ. <https://python-poetry.org/docs/cli/#help>:.

- Ցուցադրել գլոբալ օգնությունը.:

`poetry help`

- Ցուցադրել օգնություն կոնկրետ հրամանի համար.:

`poetry help {{config|show|...}}`
