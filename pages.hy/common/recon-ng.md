# վերանայում

> Ավտոմատ հետախուզության և տեղեկատվության հավաքման գործիք:.
> Լրացուցիչ տեղեկություններ. <https://github.com/lanmaster53/recon-ng/wiki>:.

- Գործիքը գործարկեք ինտերակտիվ ռեժիմով.:

`recon-ng`

- [Ինտերակտիվ] Ստեղծեք աշխատանքային տարածք.:

`workspaces create {{workspace_name}}`

- [Ինտերակտիվ] Փնտրեք շուկայում մոդուլներ, որոնք օգտագործվում են տարբեր հետախուզական առաջադրանքներ կատարելու համար.:

`marketplace search`

- [Ինտերակտիվ] Տեղադրեք բոլոր հասանելի մոդուլները (որոշներին կարող են անհրաժեշտ լինել API ստեղներ՝ ամբողջությամբ գործելու համար).:

`marketplace install all`

- [Ինտերակտիվ] Բեռնել պրոֆիլավորողի մոդուլը: Այն օգտագործվում է ցանցը սկանավորելու նպատակին համապատասխանող պրոֆիլներ գտնելու, դրանք քերելու և պահելու համար.:

`modules load profiler`

- [Ինտերակտիվ] Տեղադրեք թիրախի օգտանունը: Այս հրամանը մուտքագրելուց հետո մուտքագրեք որոնման ցանկալի օգտվողի անունը և մնացած տարբերակները թողեք դատարկ.:

`db insert profiles`

- [Ինտերակտիվ] Գործարկեք ընթացիկ մոդուլը.:

`run`
