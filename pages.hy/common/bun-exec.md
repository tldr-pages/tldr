# bun exec

> Կատարեք կեղևի հրամաններ կամ սցենարի ֆայլեր՝ օգտագործելով Bun-ի գործարկման ժամանակը:.
> Նշում. Երբ վազում եք պատյանից, հիշեք խուսափել մեջբերումներից:.
> Լրացուցիչ տեղեկություններ. <https://bun.com/docs/runtime/shell>:.

- Գործարկեք պարզ հրաման.:

`bun exec "echo hello"`

- Գործարկեք հրաման դրոշներով.:

`bun exec "ls -la"`

- Գործարկեք չակերտներ պարունակող հրաման.:

`bun exec "echo \"hello friends\""`

- Գործարկել համակցված կեղևի հրամանը.:

`bun exec "mkdir test && cd test"`

- Գործարկել սցենարի ֆայլը.:

`bun exec {{path/to/script}}`
