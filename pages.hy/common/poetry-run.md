# պոեզիայի վազք

> Գործարկել հրաման նախագծի վիրտուալ միջավայրում:.
> Լրացուցիչ տեղեկություններ. <https://python-poetry.org/docs/cli/#run>:.

- Գործարկեք հրաման վիրտուալ միջավայրում.:

`poetry run {{command}}`

- Գործարկեք հրաման արգումենտներով.:

`poetry run {{command}} {{argument1 argument2 ...}}`

- Գործարկել `pyproject.toml`-ով սահմանված սկրիպտը՝:

`poetry run {{script_name}}`

- Գործարկել Python սցենար.:

`poetry run python {{path/to/script.py}}`
