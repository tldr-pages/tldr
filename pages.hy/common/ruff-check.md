# ruff ստուգում

> Չափազանց արագ Python լցոնիչ: `check`-ը լռելյայն հրամանն է. այն կարելի է բաց թողնել ամենուր:.
> Եթե ֆայլեր կամ գրացուցակներ նշված չեն, ընթացիկ աշխատանքային գրացուցակը օգտագործվում է լռելյայն:.
> Լրացուցիչ տեղեկություններ. <https://docs.astral.sh/ruff/linter/>:.

- Գործարկեք linter-ը տվյալ ֆայլերի կամ գրացուցակների վրա.:

`ruff check {{path/to/file_or_directory1 path/to/file_or_directory2 ...}}`

- Կիրառեք առաջարկվող ուղղումները՝ տեղում փոփոխելով ֆայլերը.:

`ruff check --fix`

- Գործարկեք թաղանթը և նորից փաթաթեք փոփոխության դեպքում.:

`ruff check --watch`

- Միացրեք միայն նշված կանոնները (կամ բոլոր կանոնները)՝ անտեսելով կազմաձևման ֆայլը.:

`ruff check --select {{ALL|rule_code1,rule_code2,...}}`

- Լրացուցիչ միացրեք նշված կանոնները.:

`ruff check --extend-select {{rule_code1,rule_code2,...}}`

- Անջատեք նշված կանոնները.:

`ruff check --ignore {{rule_code1,rule_code2,...}}`

- Անտեսեք կանոնի բոլոր առկա խախտումները՝ ավելացնելով `# noqa` հրահանգները բոլոր տողերին, որոնք խախտում են այն.:

`ruff check --select {{rule_code}} --add-noqa`
