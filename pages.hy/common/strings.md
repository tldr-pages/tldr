# տող

> Գտեք տպվող տողերը օբյեկտի ֆայլում կամ երկուականում:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/strings>:.

- Տպեք բոլոր տողերը երկուական տարբերակով.:

`strings {{path/to/file}}`

- Սահմանափակեք արդյունքները առնվազն n նիշ երկարությամբ տողերով.:

`strings {{[-n|--bytes]}} {{n}} {{path/to/file}}`

- Յուրաքանչյուր արդյունքի նախածանցով իր օֆսեթով ֆայլում.:

`strings {{[-t|--radix]}} d {{path/to/file}}`

- Յուրաքանչյուր արդյունքի նախածանցով իր օֆսեթով ֆայլի ներսում տասնվեցական ձևով.:

`strings {{[-t|--radix]}} x {{path/to/file}}`
