# regedit

> Microsoft Windows `regedit` ծառայության գինու ռեեստրի գործընկեր:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/regedit>:.

- Արտահանել որոշակի ռեեստրի բանալի և բովանդակություն ֆայլ.:

`regedit -E {{path/to/file}} {{path/to/registry_key}}`

- Ջնջել նշված ռեեստրի բանալի.:

`regedit -D {{path/to/registry_key}}`

- Բացեք գրաֆիկական ռեեստրի կառավարիչը.:

`regedit`

- Ցուցադրել օգնությունը.:

`regedit -?`
