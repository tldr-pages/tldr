# եփուկի թարմացում

> Թարմացրեք հնացած բանաձևերը և տակառները:.
> Լրացուցիչ տեղեկություններ. <https://docs.brew.sh/Manpage#upgrade-options-installed_formulainstalled_cask->:.

- Թարմացրեք բոլոր հնացած տակառները և բանաձևերը.:

`brew upgrade`

- Թարմացրեք հատուկ բանաձևը / տակառը.:

`brew upgrade {{formula|cask}}`

- Տպեք այն, ինչ կթարմացվի, բայց իրականում ոչինչ մի թարմացրեք.:

`brew upgrade {{[-n|--dry-run]}}`
