# գարեջրի տեղադրում

> Տեղադրեք Homebrew բանաձեւը կամ տակառը:.
> Լրացուցիչ տեղեկություններ. <https://docs.brew.sh/Manpage#install-options-formulacask->:.

- Տեղադրեք բանաձև / տակառ.:

`brew install {{formula|cask}}`

- Ստեղծեք և տեղադրեք բանաձև աղբյուրից (կախվածությունները դեռ կտեղադրվեն շշերից).:

`brew install {{[-s|--build-from-source]}} {{formula}}`

- Ներբեռնեք մանիֆեստը, տպեք այն, ինչ կտեղադրվեր, բայց իրականում ոչինչ մի տեղադրեք.:

`brew install {{[-n|--dry-run]}} {{formula|cask}}`
