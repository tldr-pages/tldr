# svgtopam

> Փոխարկել SVG ֆայլը Netpbm ձևաչափի:.
> Լրացուցիչ տեղեկություններ. <https://netpbm.sourceforge.net/doc/svgtopam.html>:.

- Փոխարկել SVG ֆայլը Netpbm ձևաչափի.:

`svgtopam {{path/to/file.svg}} > {{path/to/output.pam}}`

- Թողարկեք տեքստային ելք, որը նկարագրում է մուտքային պատկերը.:

`svgtopam {{[-t|-trace]}} {{path/to/file.svg}}`
