# airmon-ng

> Ակտիվացրեք մոնիտորի ռեժիմը անլար ցանցային սարքերում:.
> Aircrack-ng ցանցային ծրագրային փաթեթի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://www.aircrack-ng.org/doku.php?id=airmon-ng>:.

- Թվարկեք անլար սարքերը և դրանց կարգավիճակները.:

`sudo airmon-ng`

- Միացրեք մոնիտորի ռեժիմը որոշակի սարքի համար.:

`sudo airmon-ng start {{wlan0}}`

- Սպանել անլար սարքեր օգտագործող անհանգստացնող գործընթացները.:

`sudo airmon-ng check kill`

- Անջատեք մոնիտորի ռեժիմը հատուկ ցանցային ինտերֆեյսի համար.:

`sudo airmon-ng stop {{wlan0mon}}`
