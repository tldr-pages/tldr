# ռելսերը քանդում են

> Ոչնչացնել Rails-ի ռեսուրսները:.
> Լրացուցիչ տեղեկություններ. <https://guides.rubyonrails.org/command_line.html#bin-rails-destroy>:.

- Թվարկեք բոլոր հասանելի գեներատորները ոչնչացնելու համար.:

`rails destroy`

- Ոչնչացնել Փոստ անունով մոդելին.:

`rails destroy model {{Post}}`

- Ոչնչացնել Posts անունով վերահսկիչ.:

`rails destroy controller {{Posts}}`

- Ոչնչացնել միգրացիան, որը ստեղծում է գրառումներ.:

`rails destroy migration {{CreatePosts}}`

- Քանդեք Փոստ անունով մոդելի փայտամածը.:

`rails destroy scaffold {{Post}}`
