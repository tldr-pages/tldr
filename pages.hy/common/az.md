# ազ

> Microsoft Azure-ի պաշտոնական CLI գործիքը:.
> Որոշ ենթահրամաններ, ինչպիսիք են `login`-ն, ունեն իրենց օգտագործման փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://learn.microsoft.com/cli/azure/reference-index>:.

- Մուտք գործեք Azure:

`az login`

- Կառավարեք azure բաժանորդագրության տեղեկատվությունը.:

`az account`

- Թվարկեք բոլոր Azure կառավարվող սկավառակները.:

`az disk list`

- Թվարկեք բոլոր Azure վիրտուալ մեքենաները.:

`az vm list`

- Կառավարեք Azure Kubernetes ծառայությունները.:

`az aks`

- Կառավարեք Azure ցանցի ռեսուրսները.:

`az network`

- Սկսեք ինտերակտիվ ռեժիմով.:

`az interactive`

- Ցուցադրել օգնությունը.:

`az --help`
