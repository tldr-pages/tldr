# vinmap

> Բազմաթելային Nmap սկաներ, որը IP-ի տիրույթները բաժանում է մասերի, կատարում զուգահեռ սկանավորումներ և միավորում XML կամ JSON արդյունքները:.
> Լրացուցիչ տեղեկություններ. <https://pypi.org/project/vinmap>:.

- Կատարեք ենթացանցերի հիմնական սկանավորում.:

`vinmap {{[-ip|--ip_range]}} {{192.168.1.0/24}}`

- Սկանավորեք տիրույթը տարբերակի և ՕՀ-ի հայտնաբերմամբ՝ արդյունքները պահպանելով որոշակի ֆայլում.:

`vinmap {{[-ip|--ip_range]}} {{example.com}} {{[-s|--scan_type]}} "-sV -O" -o {{path/to/scan_results.xml}}`

- Սկանավորեք IP միջակայքը՝ օգտագործելով 10 կտոր և 20 միաժամանակյա շղթաներ, օգտագործում է համակարգի պրոցեսորի միջուկների կեսը, եթե նշված չէ.:

`vinmap {{[-ip|--ip_range]}} {{10.0.0.1-10.0.0.255}} {{[-n|--num_chunks]}} 10 {{[-t|--threads]}} 20`

- Ելքային սկանավորման արդյունքները JSON ձևաչափով.:

`vinmap {{[-ip|--ip_range]}} {{192.168.1.1-192.168.1.100}} {{[-f|--format]}} json`

- Սկանավորեք բազմաթիվ IP-ներ լռելյայն կարգավորումներով և պահպանեք միավորված XML ելքը.:

`vinmap {{[-ip|--ip_range]}} {{192.168.1.1,192.168.1.2,...}}`
