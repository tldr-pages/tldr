# bun add

> Ավելացրեք և տեղադրեք նոր կախվածություններ ընթացիկ նախագծին:.
> Լրացուցիչ տեղեկություններ. <https://bun.com/docs>:.

- Տեղադրեք մեկ փաթեթ.:

`bun {{[a|add]}} {{package}}`

- Տեղադրեք բազմաթիվ փաթեթներ.:

`bun {{[a|add]}} {{package1 package2 ...}}`

- Տեղադրեք Git պահոցից.:

`bun {{[a|add]}} {{git_url}}`

- Տեղադրեք հատուկ տարբերակ.:

`bun {{[a|add]}} {{package}}@{{version}}`

- Տեղադրեք տեղական ֆայլից կամ գրացուցակից.:

`bun {{[a|add]}} file:{{path/to/file_or_directory}}`

- Ավելացրեք մշակողի կախվածություն.:

`bun {{[a|add]}} {{[-d|--dev]}} {{package}}`

- Ավելացրեք փաթեթ ամբողջ աշխարհում.:

`bun {{[a|add]}} {{[-g|--global]}} {{package}}`
