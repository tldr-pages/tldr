# csv-տարբերակ

> Դիտեք տարբերությունները երկու CSV, TSV կամ JSON ֆայլերի միջև:.
> Լրացուցիչ տեղեկություններ. <https://github.com/simonw/csv-diff>:.

- Ցուցադրել մարդու կողմից ընթեռնելի ֆայլերի միջև եղած տարբերությունների ամփոփագիրը՝ օգտագործելով հատուկ սյունակ՝ որպես եզակի նույնացուցիչ.:

`csv-diff {{path/to/file1.csv}} {{path/to/file2.csv}} --key {{column_name}}`

- Ցուցադրել մարդու կողմից ընթեռնելի ֆայլերի միջև եղած տարբերությունների ամփոփագիրը, որը ներառում է անփոփոխ արժեքներ տողերում առնվազն մեկ փոփոխությամբ.:

`csv-diff {{path/to/file1.csv}} {{path/to/file2.csv}} --key {{column_name}} --show-unchanged`

- Ցուցադրեք JSON ձևաչափով ֆայլերի միջև եղած տարբերությունների ամփոփագիրը՝ օգտագործելով հատուկ սյունակ՝ որպես եզակի նույնացուցիչ.:

`csv-diff {{path/to/file1.csv}} {{path/to/file2.csv}} --key {{column_name}} --json`
