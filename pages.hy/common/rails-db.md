# ռելսեր դբ

> Ruby on Rails-ի տվյալների բազայի հետ կապված տարբեր ենթահրամաններ:.
> Լրացուցիչ տեղեկություններ. <https://guides.rubyonrails.org/active_record_migrations.html>:.

- Ստեղծեք տվյալների բազաներ, բեռնեք սխեման և սկզբնավորեք սերմերի տվյալները.:

`rails db:setup`

- Մուտք գործեք տվյալների բազայի վահանակ.:

`rails db`

- Ստեղծեք ընթացիկ միջավայրում սահմանված տվյալների բազաները.:

`rails db:create`

- Ոչնչացնել ընթացիկ միջավայրում սահմանված տվյալների բազաները.:

`rails db:drop`

- Գործարկել սպասվող միգրացիաները.:

`rails db:migrate`

- Դիտեք յուրաքանչյուր միգրացիոն ֆայլի կարգավիճակը.:

`rails db:migrate:status`

- Հետ վերադարձնել վերջին միգրացիան.:

`rails db:rollback`

- Լրացրեք ընթացիկ տվյալների բազան `db/seeds.rb`-ում սահմանված տվյալներով.:

`rails db:seed`
