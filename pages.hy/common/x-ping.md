# x պինգ

> Ընդլայնված մոդուլներ պինգի համար:.
> Լրացուցիչ տեղեկություններ. <https://x-cmd.com/mod/ping>:.

- Ping կոնկրետ հոսթին (կանխադրված է bing.com-ին, եթե բաց թողնվի):

`x ping {{host}}`

- Ցույց տալ ping արդյունքները որպես ջերմային քարտեզ.:

`x ping {{[-m|--heatmap]}} {{host}}`

- Ցուցադրել պինգի արդյունքները որպես գծային գրաֆիկ.:

`x ping {{[-b|--bar]}} {{host}}`

- Պինգի արդյունքների մշակում և ցուցադրում որպես ջերմային քարտեզ.:

`ping {{host}} | x ping vis {{[-m|--heatmap]}}`

- Ցուցադրել օգնությունը.:

`x ping {{[-h|--help]}}`
