#ծույլ

> Պարզ տերմինալային միջերես Git հրամանների համար, որն ապահովում է ինտուիտիվ ինտերֆեյս պահեստները կառավարելու համար:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/lazygit>:.

- Բացեք Lazygit-ը ընթացիկ պահոցում.:

`lazygit`

- Բացեք Lazygit-ը որոշակի Git պահոցի համար.:

`lazygit {{[-p|--path]}} {{path/to/repository}}`

- Սկսեք Lazygit-ը՝ կենտրոնանալով կոնկրետ վահանակի վրա.:

`lazygit {{status|branch|log|stash|...}}`

- Տպել Lazygit-ի լռելյայն կոնֆիգուրացիան.:

`lazygit {{[-c|--config]}}`

- Պահեք Lazygit տեղեկամատյանները (օգտակար այլ տերմինալում վրիպազերծման ռեժիմի դեպքում).:

`lazygit {{[-l|--logs]}}`

- Գործարկեք Lazygit-ը վրիպազերծման ռեժիմում.:

`lazygit {{[-d|--debug]}}`

- Տպեք կազմաձևման գրացուցակը.:

`lazygit {{[-cd|--print-config-dir]}}`
