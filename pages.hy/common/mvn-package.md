# mvn փաթեթ

> Փաթեթավորեք Maven նախագծի կազմված կոդը իր տարածվող ձևաչափով (օրինակ՝ `.jar` կամ `.war`):.
> Լրացուցիչ տեղեկություններ. <https://manned.org/mvn>:.

- Փաթեթավորեք նախագիծը.:

`mvn package`

- Փաթեթավորեք նախագիծ՝ բաց թողնելով թեստի կատարումը.:

`mvn package {{[-D|--define]}} skipTests`

- Փաթեթավորեք նախագիծը և ստիպեք Maven-ին թարմացնել բոլոր կախվածությունները.:

`mvn package {{[-U|--update-snapshots]}}`
