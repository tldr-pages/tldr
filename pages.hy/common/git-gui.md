# git gui

> Git-ի միջերեսային միջերես՝ ճյուղերը, պարտավորությունները և հեռակառավարման սարքերը կառավարելու և տեղական միաձուլումներ կատարելու համար:.
> Տես նաև՝ `git-cola`, `gitk`:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-gui>:.

- Գործարկեք GUI-ն.:

`git gui`

- Ցույց տվեք հեղինակի անունով հատուկ ֆայլ և կատարեք հեշ յուրաքանչյուր տողում.:

`git gui blame {{path/to/file}}`

- Բացեք `git gui blame`-ը հատուկ վերանայման մեջ.:

`git gui blame {{revision}} {{path/to/file}}`

- Բացեք `git gui blame`-ը և ոլորեք տեսարանը՝ կենտրոնանալով որոշակի գծի վրա.:

`git gui blame --line={{line}} {{path/to/file}}`

- Բացեք պատուհանը, որպեսզի մեկը պարտավորվի և վերադառնա պատյան, երբ այն ավարտվի.:

`git gui citool`

- Բացեք `git gui citool`-ը «Փոփոխել վերջին պարտավորությունը» ռեժիմում՝:

`git gui citool --amend`

- Բացեք `git gui citool`-ը միայն կարդալու ռեժիմով.:

`git gui citool --nocommit`

- Ցույց տվեք զննարկիչը կոնկրետ ճյուղի ծառի համար՝ բացելով մեղադրական գործիքը ֆայլերի վրա սեղմելիս.:

`git gui browser maint`
