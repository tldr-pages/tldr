# npm սխալներ

> Հաղորդել վրիպակների մասին փաթեթի համար վեբ բրաուզերում:.
> Փաթեթի սխալների հետագծման URL-ը կամ աջակցության էլ.փոստը բացելու փորձեր:.
> Լրացուցիչ տեղեկություններ. <https://docs.npmjs.com/cli/npm-bugs/>:.

- Հաղորդեք սխալների մասին որոշակի փաթեթի համար՝ բացելով սխալների որոնիչը նշված փաթեթի համար.:

`npm bugs {{package_name}}`

- Բացեք ընթացիկ փաթեթի սխալների որոնիչը՝ փնտրելով `package.json` ֆայլ և օգտագործելով դրա անունը.:

`npm bugs`

- Կարգավորեք զննարկիչը, որն օգտագործվում է URL-ներ բացելու համար՝ ձեր նախընտրած դիտարկիչը սահմանելով `npm` հրամանների համար.:

`npm {{[c|config]}} set browser {{browser_name}}`

- Կառավարեք URL-ի բացումը. սահմանեք `browser`-ը `true` համակարգի URL բացողի համար, կամ `false`՝ տերմինալում URL-ներ տպելու համար:

`npm {{[c|config]}} set browser {{true|false}}`
