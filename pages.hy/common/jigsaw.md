# ոլորահատ սղոց

> Laravel-ի վրա հիմնված ստատիկ կայքի ստեղծող PHP-ի համար:.
> Լրացուցիչ տեղեկություններ. <https://jigsaw.tighten.co/>:.

- Նախաձեռնել նախագիծ.:

`jigsaw init`

- Նախաձեռնեք նախագիծը, օգտագործելով մեկնարկային ձևանմուշ.:

`jigsaw init {{template_name}}`

- Ստեղծեք կայք զարգացման համար.:

`jigsaw build`

- Նախադիտեք կայքը «build_local» գրացուցակից.:

`jigsaw serve`

- Կառուցեք կայքի արտադրության համար.:

`jigsaw build production`

- Նախադիտեք կայքը «build_production» գրացուցակից.:

`jigsaw serve {{build_production}}`
