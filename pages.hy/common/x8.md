# x8

> Թաքնված պարամետրերի հայտնաբերման հավաքածու՝ խոցելի կամ հետաքրքիր վեբ պարամետրերը բացահայտելու համար:.
> Լրացուցիչ տեղեկություններ. <https://github.com/Sh1Yo/x8/blob/main/docs.md>:.

- Ստուգեք թաքնված պարամետրերը URL հարցման մեջ.:

`x8 {{[-u|--url]}} {{https://example.com/}} {{[-w|--wordlist]}} {{path/to/wordlist.txt}}`

- Ստուգեք պարամետրերը հատուկ հարցման ներարկման կետով (`%s`):

`x8 {{[-u|--url]}} {{https://example.com/?something=1%26%s}} {{[-w|--wordlist]}} {{path/to/wordlist.txt}}`

- Ուղարկեք պարամետրերը POST մարմնի միջոցով JSON ձևաչափով.:

`x8 {{[-u|--url]}} {{https://example.com/}} {{[-X|--method]}} {{POST}} {{[-b|--body]}} '{{{"x":{%s\}\}}}' {{[-w|--wordlist]}} {{path/to/wordlist.txt}}`

- Ստուգեք պարամետրերը հատուկ ձևանմուշով (`%k` բանալիի համար, `%v` արժեքի համար):

`x8 {{[-u|--url]}} {{https://example.com/}} {{[-P|--param-template]}} {{user[%k]=%v}} {{[-w|--wordlist]}} {{path/to/wordlist.txt}}`

- Կոդավորեք պարամետրերը հարցումներում հատուկ նիշերի համար.:

`x8 {{[-u|--url]}} {{https://example.com/?path=..%2faction.php%3f%s%23}} --encode {{[-w|--wordlist]}} {{path/to/wordlist.txt}}`

- Բացահայտեք URL-ի թաքնված վերնագրերը.:

`x8 {{[-u|--url]}} {{https://example.com/}} --headers {{[-w|--wordlist]}} {{path/to/headers.txt}}`

- Ստուգեք բազմաթիվ URL-ներ զուգահեռաբար բարձր համաժամանակությամբ և ստուգեք գտնված պարամետրերը.:

`x8 {{[-u|--url]}} {{https://example.com/}} {{https://4rt.one/}} {{[-W|--workers]}} {{0}} -c {{3}} --verify`

- Պահպանեք հարցումների և պատասխանների տվյալները հայտնաբերված պարամետրերի համար գրացուցակում.:

`x8 {{[-u|--url]}} {{https://example.com/}} {{[-w|--wordlist]}} {{path/to/wordlist.txt}} --save-responses {{path/to/output_directory}}`
