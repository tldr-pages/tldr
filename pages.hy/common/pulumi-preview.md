# pulumi նախադիտում

> Ցույց տալ փաթեթի ռեսուրսների թարմացումների նախադիտումը:.
> Լրացուցիչ տեղեկություններ. <https://www.pulumi.com/docs/iac/cli/commands/pulumi_preview/>:.

- Ցուցադրել փաթեթի ռեսուրսների թարմացումների նախադիտումը.:

`pulumi preview`

- Ցուցադրել փաթեթի ռեսուրսների թարմացումների նախադիտումը JSON ձևաչափով.:

`pulumi preview {{[-j|--json]}}`

- Նախադիտեք թարմացումները որպես հարուստ տարբերություն, որը ցույց է տալիս ընդհանուր փոփոխությունները.:

`pulumi preview --diff`

- Նախադիտեք թարմացումները՝ օգտագործելով քաղաքականության փաթեթը (առանց Pulumi Cloud-ի, լավագույնը CI/CD-ում):

`pulumi preview --policy-pack {{path/to/directory}}`

- Ցուցադրել օգնությունը.:

`pulumi preview {{[-h|--help]}}`
