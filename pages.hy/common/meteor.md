# երկնաքար

> Full stack JavaScript հարթակ վեբ հավելվածներ ստեղծելու համար:.
> Լրացուցիչ տեղեկություններ. <https://docs.meteor.com/cli/>:.

- Գործարկեք մետեորի նախագիծն իր արմատային գրացուցակից զարգացման ռեժիմում.:

`meteor`

- Ստեղծեք նախագիծ տվյալ գրացուցակի տակ.:

`meteor create {{path/to/directory}}`

- Ցուցադրել փաթեթների ցանկը, որն այժմ օգտագործում է նախագիծը.:

`meteor list`

- Ծրագրին փաթեթ ավելացրեք.:

`meteor add {{package}}`

- Հեռացրեք փաթեթը նախագծից.:

`meteor remove {{package}}`

- Տվյալ գրացուցակի տակ ստեղծեք նախագծի արտադրական կառուցվածքը որպես tarball.:

`meteor build {{path/to/directory}}`
