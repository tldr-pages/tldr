# ռելսերի սերվեր

> Ծառայել Rails հավելվածը ընթացիկ գրացուցակում՝ օգտագործելով Puma վեբ սերվերը, որը միացված է Rails-ին:.
> Լրացուցիչ տեղեկություններ. <https://guides.rubyonrails.org/command_line.html#bin-rails-server>:.

- Գործարկեք վեբ սերվերը.:

`rails server`

- Գործարկեք վեբ սերվերը նշված նավահանգստում.:

`rails server {{[-p|--port]}} {{port_number}}`

- Գործարկեք վեբ սերվերը նշված IP հասցեով.:

`rails server {{[-b|--binding]}} {{ip_address}}`

- Գործարկեք վեբ սերվերը նշված միջավայրում.:

`rails server {{[-e|--environment]}} {{environment}}`

- Ցուցադրել օգնությունը.:

`rails server {{[-h|--help]}}`
