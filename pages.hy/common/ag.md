# ագ

> Արծաթե որոնողը: Հավանել `ack`-ը, բայց նպատակ ունի ավելի արագ լինել:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/ag>:.

- Գտեք `string` պարունակող ֆայլեր և տպեք տողերի համընկնումները համատեքստում.:

`ag string`

- Գտեք `string` պարունակող ֆայլեր որոշակի գրացուցակում.:

`ag string {{path/to/directory}}`

- Գտեք `string` պարունակող ֆայլեր, բայց նշեք միայն ֆայլերի անունները.:

`ag {{[-l|--files-with-matches]}} string`

- Գտեք ֆայլեր, որոնք պարունակում են `STRING` մեծատառերը, և տպեք միայն համընկնումը, այլ ոչ թե ամբողջ տողը:

`ag {{[-i|--ignore-case]}} {{[-o|--only-matching]}} STRING`

- Գտեք `string` `file_name` անունով ֆայլերում:

`ag string {{[-G|--file-search-regex]}} file_name`

- Գտեք ֆայլեր, որոնց բովանդակությունը համապատասխանում է `regex`-ին:

`ag '{{^ca(t|r)$}}'`

- Գտեք `string` անունով ֆայլեր՝:

`ag {{[-g|--filename-pattern]}} string`
