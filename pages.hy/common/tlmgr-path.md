# tlmgr ճանապարհ

> Ավելացրեք կամ հեռացրեք սիմհղումներ TeX Live գործարկվող սարքերի, մարդ էջերի և տեղեկատվական էջերի համար:.
> Այս հրամանը պետք է նորից գործարկվի ապագայում ավելացված ֆայլերի համար:.
> Լրացուցիչ տեղեկություններ. <https://www.tug.org/texlive/doc/tlmgr.html#path>:.

- Ավելացրեք սիմհղումներ TeX Live ֆայլերին.:

`sudo tlmgr path add`

- Հեռացրեք TeX Live ֆայլերի սիմհղումները.:

`sudo tlmgr path remove`
