# git sed

> Փոխարինեք նախշերը git-ով կառավարվող ֆայլերում՝ օգտագործելով sed:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/git-sed>:.

- Փոխարինեք նշված տեքստը ընթացիկ պահոցում.:

`git sed '{{find_text}}' '{{replace_text}}'`

- Փոխարինեք նշված տեքստը և այնուհետև կատարեք ստացված փոփոխությունները ստանդարտ commit հաղորդագրությամբ.:

`git sed -c '{{find_text}}' '{{replace_text}}'`

- Փոխարինեք նշված տեքստը՝ օգտագործելով `regex`:

`git sed -f g '{{find_text}}' '{{replace_text}}'`

- Փոխարինեք որոշակի տեքստ բոլոր ֆայլերում տվյալ գրացուցակի տակ.:

`git sed '{{find_text}}' '{{replace_text}}' -- {{path/to/directory}}`
