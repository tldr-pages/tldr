# ng տեղակայում

> Աշխատանքային տարածքում գտնվող նախագծի համար կանչեք deploy builder-ը:.
> Նշում. տեղակայումը պահանջում է կարգավորում `ng add`-ի միջոցով:.
> Լրացուցիչ տեղեկություններ. <https://angular.dev/cli/deploy>:.

- Տեղադրեք լռելյայն նախագիծը.:

`ng deploy`

- Տեղադրել կոնկրետ նախագիծ.:

`ng deploy {{project_name}}`

- Տեղակայել՝ օգտագործելով հատուկ կոնֆիգուրացիա.:

`ng deploy {{[-c|--configuration]}} {{development|production|...}}`
