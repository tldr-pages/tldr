# mkdocs

> Ծրագրի փաստաթղթեր Markdown-ով:.
> Լրացուցիչ տեղեկություններ. <https://www.mkdocs.org/user-guide/cli/>:.

- Ստեղծեք նոր mkdocs նախագիծ.:

`mkdocs new {{project_name}}`

- Ծառայել նախագիծը ընթացիկ գրացուցակում, օգտագործելով mkdocs dev-server.:

`mkdocs serve`

- Կառուցեք փաստաթղթերը ընթացիկ գրացուցակում.:

`mkdocs build`

- Տեղադրեք փաստաթղթերը ընթացիկ գրացուցակում GitHub էջերում.:

`mkdocs gh-deploy`
