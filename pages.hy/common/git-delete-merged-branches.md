# git delete-merged-branches

> Ջնջել մասնաճյուղերը, որոնք նշված են `git branch --merged`-ում՝ բացառությամբ հիմնականի:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-delete-merged-branches>:.

- Ջնջել միավորված մասնաճյուղերը.:

`git delete-merged-branches`
