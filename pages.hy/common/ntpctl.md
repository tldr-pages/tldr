# ntpctl

> Ցուցադրել տեղեկատվություն OpenNTPD-ի գործող օրինակի մասին:.
> Լրացուցիչ տեղեկություններ. <https://man.openbsd.org/ntpctl>:.

- Ցուցադրել բոլոր տվյալները.:

`ntpctl -s {{[a|all]}}`

- Ցույց տալ տեղեկատվություն յուրաքանչյուր հասակակիցի մասին.:

`ntpctl -s {{[p|peers]}}`

- Ցույց տալ հասակակիցների և սենսորների կարգավիճակը և արդյոք համակարգի ժամացույցը համաժամեցված է.:

`ntpctl -s {{[s|status]}}`

- Ցույց տալ տեղեկատվություն յուրաքանչյուր սենսորի մասին.:

`ntpctl -s {{[S|Sensors]}}`
