# git համաժամեցում

> Համաժամացրեք տեղական մասնաճյուղերը հեռավոր մասնաճյուղերի հետ:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/git-sync>:.

- Համաժամացրեք ընթացիկ տեղական մասնաճյուղը իր հեռավոր մասնաճյուղի հետ.:

`git sync`

- Համաժամացրեք ընթացիկ տեղական մասնաճյուղը հեռավոր հիմնական մասնաճյուղի հետ.:

`git sync origin main`

- Համաժամեցեք առանց չհետևված ֆայլերը մաքրելու.:

`git sync {{[-s|--soft]}} {{remote_name}} {{branch_name}}`
