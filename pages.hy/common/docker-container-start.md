# դոկեր կոնտեյների մեկնարկ

> Սկսեք դադարեցված բեռնարկղերը:.
> Լրացուցիչ տեղեկություններ. <https://docs.docker.com/reference/cli/docker/container/start/>:.

- Սկսեք Docker կոնտեյներ.:

`docker {{[start|container start]}} {{container}}`

- Գործարկեք կոնտեյներ՝ կցելով `stdout` և `stderr` և փոխանցելով ազդանշաններ.:

`docker {{[start|container start]}} {{[-a|--attach]}} {{container}}`

- Սկսեք մեկ կամ մի քանի կոնտեյներ.:

`docker {{[start|container start]}} {{container1 container2 ...}}`

- Ցուցադրել օգնությունը.:

`docker {{[start|container start]}} --help`
