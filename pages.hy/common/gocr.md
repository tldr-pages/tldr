# gocr

> Նիշերի օպտիկական ճանաչման գործիք:.
> Ճանաչել նիշերը՝ օգտագործելով իր շարժիչը, և օգտվողին հուշել անհայտ օրինաչափություններ՝ դրանք տվյալների բազայում պահելու համար:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/gocr>:.

- Ճանաչեք [i]nput պատկերի նիշերը և [o]արտադրեք այն տվյալ ֆայլում: Տեղադրեք տվյալների բազան ([p]) `path/to/db_directory`-ում (հաստատեք, որ թղթապանակը գոյություն ունի, կամ DB-ի օգտագործումը լուռ բաց կթողնի): [m]ode 130 նշանակում է ստեղծել + օգտագործել + ընդլայնել տվյալների բազան.:

`gocr -m 130 -p {{path/to/db_directory}} -i {{path/to/input_image.png}} -o {{path/to/output_file.txt}}`

- Ճանաչեք նիշերը և ենթադրեք, որ բոլոր [C] նիշերը թվեր են.:

`gocr -m 130 -p {{path/to/db_directory}} -i {{path/to/input_image.png}} -o {{path/to/output_file.txt}} -C "{{0..9}}"`

- Ճանաչել 100% վկայականով նիշերը (նիշերը անհայտ համարվելու ավելի մեծ հնարավորություն ունեն).:

`gocr -m 130 -p {{path/to/db_directory}} -i {{path/to/input_image.png}} -o {{path/to/output_file.txt}} -a 100`
