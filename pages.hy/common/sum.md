# գումար

> Հաշվարկել ստուգիչ գումարները և ֆայլի բլոկների քանակը:.
> Ավելի ժամանակակից `cksum`-ի նախորդը:.
> Լրացուցիչ տեղեկություններ. <https://www.gnu.org/software/coreutils/manual/html_node/sum-invocation.html>:.

- Հաշվեք ստուգիչ գումարը BSD-ի հետ համատեղելի ալգորիթմով և 1024 բայթ բլոկներով.:

`sum {{path/to/file}}`

- Հաշվեք ստուգման գումարը System V-ի հետ համատեղելի ալգորիթմով և 512 բայթ բլոկներով.:

`sum {{[-s|--sysv]}} {{path/to/file}}`
