# pixi առաջադրանք

> Կառավարեք առաջադրանքները ծրագրի միջավայրում:.
> Լրացուցիչ տեղեկություններ. <https://pixi.sh/latest/reference/cli/#task>:.

- Ստեղծեք նոր առաջադրանք.:

`pixi task add {{task_name}} {{task_command}}`

- Թվարկեք բոլոր առաջադրանքները նախագծում.:

`pixi task list`

- Հեռացնել առաջադրանքը.:

`pixi task remove {{task_name}}`

- Ստեղծեք այլանուն առաջադրանքի համար.:

`pixi task alias {{alias_name}} {{task1 task2 ...}}`
