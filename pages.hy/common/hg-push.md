# hg հրում

> Փոփոխությունները տեղային պահոցից տեղափոխել նշված նպատակակետ:.
> Լրացուցիչ տեղեկություններ. <https://www.mercurial-scm.org/help/commands/push>:.

- Հպեք փոփոխությունները «կանխադրված» հեռավոր ուղու վրա.:

`hg push`

- Փոփոխությունները մղել նշված հեռավոր պահոց.:

`hg push {{path/to/destination_repository}}`

- Հրել նոր մասնաճյուղ, եթե այն գոյություն չունի (կանխադրված անջատված է).:

`hg push --new-branch`

- Նշեք որոշակի վերանայման փոփոխությունների հավաքածու՝ մղելու համար.:

`hg push {{[-r|--rev]}} {{revision}}`

- Նշեք որոշակի ճյուղ՝ մղելու համար.:

`hg push {{[-b|--branch]}} {{branch}}`

- Նշեք հատուկ էջանիշ՝ մղելու համար.:

`hg push {{[-B|--bookmark]}} {{bookmark}}`
