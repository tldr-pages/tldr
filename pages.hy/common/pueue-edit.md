# pueue խմբագրում

> Խմբագրել թաքցված կամ հերթագրված առաջադրանքի հրամանը կամ ուղին:.
> Լրացուցիչ տեղեկություններ. <https://github.com/Nukesor/pueue#how-to-use-it>:.

- Խմբագրեք առաջադրանքը, տես `pueue status` առաջադրանքի ID-ն ստանալու համար.:

`pueue edit {{task_id}}`

- Խմբագրել ուղին, որից առաջադրանքը կատարվում է.:

`pueue edit {{task_id}} --path`

- Խմբագրել հրամանը նշված խմբագրիչով.:

`EDITOR={{nano}} pueue edit {{task_id}}`
