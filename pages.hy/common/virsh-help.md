# virsh օգնություն

> Ցուցադրել տեղեկություններ `virsh` հրամանների կամ հրամանների խմբերի մասին:.
> Տես նաև՝ `virsh`:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/virsh>:.

- Թվարկեք `virsh` հրամանները՝ խմբավորված համապատասխան կատեգորիաներով.:

`virsh help`

- Թվարկեք հրամանների կատեգորիաները.:

`virsh help | grep "keyword"`

- Թվարկեք հրամանները կատեգորիայում.:

`virsh help {{category_keyword}}`

- Ցուցադրել օգնություն հրամանի համար.:

`virsh help {{command}}`
