# cdecl

> Կազմել և վերծանել C և C++ տիպի հայտարարությունները:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/cdecl>:.

- Կազմեք անգլերեն արտահայտությունը C հռչակագրի մեջ և ստեղծեք [c]compilable ելք (ներառյալ `;` և `{}`):

`cdecl -c {{phrase}}`

- Բացատրեք C հռչակագիրը անգլերենով.:

`cdecl explain {{C_declaration}}`

- Փոփոխական փոխանցել մեկ այլ տեսակի.:

`cdecl cast {{variable_name}} to {{type}}`

- Գործարկել [i] ինտերակտիվ ռեժիմով.:

`cdecl -i`
