# որոնման սպլիտ

> Որոնել Exploit տվյալների շտեմարան՝ շահագործումների, shellcodes-ի և/կամ փաստաթղթերի համար:.
> Եթե հայտնի տարբերակների համարներն օգտագործվում են որպես որոնման տերմիններ, ցուցադրվում են ինչպես ճշգրիտ տարբերակի, այնպես էլ մյուսների համար, որոնց տարբերակների տիրույթը ընդգրկում է նշվածը:.
> Լրացուցիչ տեղեկություններ. <https://www.exploit-db.com/searchsploit>:.

- Որոնեք շահագործում, shellcode կամ թուղթ.:

`searchsploit {{search_terms}}`

- Որոնել հայտնի կոնկրետ տարբերակ, օրինակ. sudo տարբերակ 1.8.27:

`searchsploit sudo 1.8.27`

- Ցույց տվեք exploit-db հղումը հայտնաբերված ռեսուրսներին.:

`searchsploit {{[-w|--www]}} {{search_terms}}`

- Պատճենեք ռեսուրսը ընթացիկ գրացուցակում (պահանջում է շահագործման համարը).:

`searchsploit {{[-m|--mirror]}} {{exploit_number}}`

- Ուսումնասիրեք ռեսուրսը, օգտագործելով `$PAGER` միջավայրի փոփոխականում սահմանված փեյջերը.:

`searchsploit {{[-x|--examine]}} {{exploit_number}}`

- Թարմացրեք տեղական Exploit տվյալների բազան.:

`searchsploit {{[-u|--update]}}`

- Որոնեք [c]սովորական [v]խոցելիություններ և [e]բացահայտումներ (CVE) արժեքը.:

`searchsploit --cve {{2021-44228}}`

- Ստուգեք արդյունքները `nmap`-ի XML ելքում՝ ծառայության տարբերակով (`nmap -sV -oX nmap-output.xml`)՝ հայտնի շահագործումների համար.:

`searchsploit --nmap {{path/to/nmap-output.xml}}`
