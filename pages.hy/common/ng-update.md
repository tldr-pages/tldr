# ng թարմացում

> Թարմացրեք Angular աշխատանքային տարածքը և դրա կախվածությունները:.
> Լրացուցիչ տեղեկություններ. <https://angular.dev/cli/update>:.

- Թարմացրեք բոլոր կախվածությունները աշխատանքային տարածքում.:

`ng update`

- Թարմացրեք որոշակի փաթեթ.:

`ng update {{package}}`

- Թարմացրեք բազմաթիվ փաթեթներ.:

`ng update {{package1 package2 ...}}`

- Անտեսեք հասակակիցների կախվածության տարբերակների անհամապատասխանությունները.:

`ng update {{package}} --force`

- Թույլատրել թարմացումը, երբ պահոցը պարունակում է փոփոխված կամ չհետևված ֆայլեր.:

`ng update {{package}} --allow-dirty`

- Թարմացրեք նախնական թողարկման տարբերակներին, ներառյալ բետա և թողարկման թեկնածուներին.:

`ng update {{package}} --next`

- Ցուցադրել լրացուցիչ մանրամասներ ներքին գործառնությունների վերաբերյալ.:

`ng update {{package}} --verbose`
