# գնալ գործիք

> Գործարկել Go գործիքը կամ հրամանը:.
> Կատարեք Go հրամանը որպես առանձին երկուական, սովորաբար վրիպազերծման համար:.
> Լրացուցիչ տեղեկություններ. <https://pkg.go.dev/cmd/go#hdr-Run_specified_go_tool>:.

- Թվարկեք առկա գործիքները.:

`go tool`

- Գործարկեք go link գործիքը.:

`go tool link {{path/to/main.o}}`

- Տպեք հրամանը, որը կկատարվեր, բայց մի կատարեք այն (նման `whereis`):

`go tool -n {{command}} {{arguments}}`

- Դիտեք փաստաթղթերը նշված գործիքի համար.:

`go tool {{command}} --help`

- Թվարկեք բոլոր հասանելի խաչաձև կազմման թիրախները.:

`go tool dist list`
