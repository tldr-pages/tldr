# հանգույցի ստեղծում

> Ստեղծեք նոր պահոց GitHub-ում:.
> Լրացուցիչ տեղեկություններ. <https://hub.github.com/hub-create.1.html>:.

- Ներբեռնեք ընթացիկ (միայն տեղական) պահոցը ձեր GitHub հաշվին որպես հանրային.:

`hub create`

- Ստեղծեք մասնավոր պահոց և բացեք նոր պահոցը վեբ բրաուզերում.:

`hub create {{[-p|--private]}} {{[-o|--browse]}}`
