#մեղքի զգացում

> Ցույց տալ մեղադրանքի ընդհանուր թիվը չբեմադրված փոփոխություններով ֆայլերի համար կամ հաշվարկել մեղքի փոփոխությունը երկու վերանայումների միջև:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/git-guilt>:.

- Ցույց տալ մեղքի ընդհանուր քանակը.:

`git guilt`

- Հաշվարկեք մեղքի փոփոխությունը երկու վերանայումների միջև.:

`git guilt {{first_revision}} {{last_revision}}`

- Ցույց տալ հեղինակի նամակները անունների փոխարեն.:

`git guilt {{[-e|--email]}}`

- Անտեսեք բացատը միայն մեղքը վերագրելիս փոխվում.:

`git guilt {{[-w|--ignore-whitespace]}}`

- Գտեք մեղադրական դելտան վերջին երեք շաբաթվա ընթացքում.:

`git guilt 'git log --until "3 weeks ago" --format "%H" {{[-n|--max-count]}} 1'`

- Գտեք մեղադրական դելտան վերջին երեք շաբաթվա ընթացքում (Git 1.8.5+).:

`git guilt @{3.weeks.ago}`
