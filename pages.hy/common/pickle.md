#թթու վարունգ

> PHP ընդլայնման տեղադրիչ՝ հիմնված Կոմպոզիտորի վրա:.
> Լրացուցիչ տեղեկություններ. <https://github.com/FriendsOfPHP/pickle#usage>:.

- Տեղադրեք հատուկ PHP ընդլայնում.:

`pickle install {{extension_name}}`

- Վերափոխեք գոյություն ունեցող PECL ընդլայնման կոնֆիգուրացիան Pickle կազմաձևման ֆայլի.:

`pickle convert {{path/to/directory}}`

- Վավերացնել PECL ընդլայնումը.:

`pickle validate {{path/to/directory}}`

- Փաթեթավորեք PECL ընդլայնում թողարկման համար.:

`pickle release {{path/to/directory}}`
