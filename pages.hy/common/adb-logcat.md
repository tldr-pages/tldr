# adb logcat

> Թափել համակարգի հաղորդագրությունների մատյանը:.
> Լրացուցիչ տեղեկություններ. <https://developer.android.com/tools/logcat>:.

- Ցուցադրել համակարգի տեղեկամատյանները.:

`adb logcat`

- Ցուցադրել տողեր, որոնք համապատասխանում են `reg[e]x`-ին.:

`adb logcat -e {{regex}}`

- Ցուցադրել պիտակի տեղեկամատյանները հատուկ ռեժիմով ([V]erbose, [D]bug, [I]nfo, [W]Զգուշացում, [E]սխալ, [F]atal, [S]անձուկ), զտելով այլ պիտակներ.:

`adb logcat {{tag}}:{{mode}} *:S`

- Ցուցադրել React Native հավելվածների տեղեկամատյանները [V]erbose ռեժիմում [S]լռեցնելով այլ պիտակներ.:

`adb logcat ReactNative:V ReactNativeJS:V *:S`

- Ցուցադրել տեղեկամատյանները բոլոր պիտակների համար՝ առաջնահերթության մակարդակով [W]զգուշացումով և ավելի բարձր՝:

`adb logcat *:W`

- Ցուցադրել տեղեկամատյանները որոշակի PID-ի համար.:

`adb logcat --pid {{pid}}`

- Ցուցադրել տեղեկամատյանները որոշակի փաթեթի գործընթացի համար.:

`adb logcat --pid $(adb shell pidof -s {{package}})`

- Գունավորեք տեղեկամատյանը (սովորաբար օգտագործվում է զտիչներով).:

`adb logcat -v color`
