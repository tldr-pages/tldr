# openclaw գործակալ

> Խոսեք OpenClaw AI օգնականի հետ:.
> Լրացուցիչ տեղեկություններ. <https://docs.openclaw.ai/cli/agent>:.

- Ուղարկեք մեկ հաղորդագրություն օգնականին.:

`openclaw agent {{[-m|--message]}} "{{Hello, what is the weather?}}"`

- Սկսեք ինտերակտիվ զրույցի ռեժիմը.:

`openclaw agent`

- Ուղարկեք հաղորդագրություն կարգավորելի մտածողության մակարդակով.:

`openclaw agent {{[-m|--message]}} "{{Analyze this}}" --thinking {{off|minimal|low|medium|high}}`

- Ցուցադրել օգնությունը.:

`openclaw agent {{[-h|--help]}}`
