# git արմատ

> Տպել ընթացիկ Git պահոցի արմատային գրացուցակը:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-root>:.

- Տպեք ընթացիկ Git պահոցի բացարձակ ուղին.:

`git root`

- Տպեք ընթացիկ աշխատանքային գրացուցակը ընթացիկ Git պահոցի արմատի համեմատ.:

`git root {{[-r|--relative]}}`
