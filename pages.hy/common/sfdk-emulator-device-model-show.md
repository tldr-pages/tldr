# sfdk emulator սարք-մոդել-ցուցադրել

> Ցույց տալ նմանակված սարքի մոդելի հատկությունները:.
> Լրացուցիչ տեղեկություններ. <https://github.com/sailfishos/sailfish-qtcreator/blob/master/share/qtcreator/sfdk/modules/40-testing-maintain/doc/command.emulator.adoc>:.

- Ցույց տալ մոդելի հատկությունները.:

`sfdk emulator device-model-show {{model}}`

- Ցուցադրել հասանելի սարքերի մոդելները.:

`sfdk emulator device-model-list`
