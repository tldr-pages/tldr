#փորել

> DNS որոնման կոմունալ:.
> Տես նաև՝ `resolvectl`, `nslookup`, `host`:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/dig>:.

- Փնտրեք հյուրընկալողի անվան հետ կապված IP(ներ)ը (A գրառումներ).:

`dig +short {{example.com}}`

- Ստացեք մանրամասն պատասխան տվյալ տիրույթի համար (A գրառումներ).:

`dig +noall +answer {{example.com}}`

- Հարցրեք որոշակի DNS գրառման տեսակի՝ կապված տվյալ տիրույթի անվան հետ.:

`dig +short {{example.com}} {{A|MX|TXT|CNAME|NS}}`

- Նշեք այլընտրանքային DNS սերվեր՝ հարցման և ընտրովի օգտագործելու համար DNS TLS (DoT) միջոցով.:

`dig {{+tls}} @{{1.1.1.1|8.8.8.8|9.9.9.9|...}} {{example.com}}`

- Կատարեք հակադարձ DNS որոնում IP հասցեի վրա (PTR գրառում).:

`dig -x {{8.8.8.8}}`

- Գտնել հեղինակավոր անունների սերվերներ գոտու համար և ցուցադրել SOA գրառումները.:

`dig +nssearch {{example.com}}`

- Կատարեք կրկնվող հարցումներ և ցուցադրեք ամբողջ հետագծային ուղին տիրույթի անունը լուծելու համար.:

`dig +trace {{example.com}}`

- Հարցրեք DNS սերվերին ոչ ստանդարտ [p]ort-ի միջոցով՝ օգտագործելով TCP արձանագրությունը.:

`dig +tcp -p {{port}} @{{dns_server_ip}} {{example.com}}`
