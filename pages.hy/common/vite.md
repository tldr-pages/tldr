#Վիտե

> Ստեղծեք Vite նախագիծ:.
> Օգտագործվում է JavaScript նախագծեր ստեղծելու համար:.
> Հասանելի ձևանմուշներ՝ վանիլ, վանիլ-ց, վյու, վուե-ց, ռեակտ, ռեակտ-ց, ռեակտ-սվկ, ռեակտ-սվկ-ց, նախաակտ, պրեակտ-ց, լիտ, լիտ-ց, սվելթ, սվելթ-ց:.
> Լրացուցիչ տեղեկություններ. <https://vite.dev/guide/>:.

- Կարգավորում՝ օգտագործելով `npm` 6.x:

`npm create vite@latest my-react-app --template react-ts`

- Կարգավորեք `npm` 7+-ի միջոցով, անհրաժեշտ է լրացուցիչ կրկնակի գծիկ՝:

`npm create vite@latest my-react-app -- --template react-ts`

- Կարգավորում՝ օգտագործելով `yarn`:

`yarn create vite my-react-app --template react-ts`

- Կարգավորում՝ օգտագործելով `pnpm`:

`pnpm create vite my-react-app --template react-ts`
