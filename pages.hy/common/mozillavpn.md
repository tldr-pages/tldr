# mozillavpn

> Վիրտուալ մասնավոր ցանց Firefox-ի ստեղծողների կողմից:.
> Տես նաև՝ `fastd`, `ivpn`, `mullvad`, `warp-cli`:.
> Լրացուցիչ տեղեկություններ. <https://github.com/mozilla-mobile/mozilla-vpn-client/wiki/Command-line-interface>:.

- Մուտք գործեք ինտերակտիվ հուշումով.:

`mozillavpn login`

- Միացեք Mozilla VPN-ին.:

`mozillavpn activate`

- Ցուցադրել կապի կարգավիճակը.:

`mozillavpn status`

- Ցուցակեք առկա սերվերները.:

`mozillavpn servers`

- Ընտրեք հատուկ սերվեր.:

`mozillavpn select {{server_name}}`

- Անջատեք Mozilla VPN-ից.:

`mozillavpn deactivate`

- Դուրս գալ.:

`mozillavpn logout`

- Ցուցադրել օգնություն ենթահրամանի համար.:

`mozillavpn {{subcommand}} --help`
