# git փոստի տեղեկատվություն

> Քաղեք կարկատան և հեղինակային տեղեկությունները մեկ էլփոստի հաղորդագրությունից:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-mailinfo>:.

- Քաղեք կարկատակի և հեղինակի տվյալները էլփոստի հաղորդագրությունից.:

`git mailinfo {{message|patch}}`

- Քաղեք, բայց հեռացրեք առաջատար և հետին բացատը.:

`git mailinfo -k {{message|patch}}`

- Հեռացրեք մարմնից ամեն ինչ մկրատի գծից առաջ (օրինակ՝ «-->* --») և առբերեք հաղորդագրությունը կամ կարկատակը.:

`git mailinfo --scissors {{message|patch}}`
