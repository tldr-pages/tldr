# firebase

> Փորձարկել, կառավարել և տեղադրել Firebase նախագծերը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/firebase/firebase-tools>:.

- Մուտք գործեք <https://console.firebase.google.com>:

`firebase login`

- Թվարկեք առկա Firebase նախագծերը.:

`firebase projects:list`

- Սկսեք ինտերակտիվ հրաշագործ՝ ընթացիկ գրացուցակում Firebase նախագիծ ստեղծելու համար.:

`firebase init`

- Տեղադրեք կոդը և ակտիվները ընթացիկ Firebase նախագծին.:

`firebase deploy`

- Սկսեք տեղական սերվեր՝ ընթացիկ Firebase նախագծի ակտիվները ստատիկ կերպով հյուրընկալելու համար.:

`firebase serve`

- Սկսեք ինտերակտիվ հրաշագործ՝ լռելյայն վեբ բրաուզերում բացելու ընթացիկ Firebase նախագծի բազմաթիվ հղումներից մեկը.:

`firebase open`
