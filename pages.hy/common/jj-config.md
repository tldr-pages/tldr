# jj կոնֆիգուրացիա

> Կառավարեք կազմաձևման ընտրանքները:.
> Որոշ ենթահրամաններ, ինչպիսիք են `edit`, `get`, `list`, `path`, `set`, `unset`, ունեն իրենց օգտագործման փաստաթղթերը:.
> Լրացուցիչ տեղեկություններ. <https://docs.jj-vcs.dev/latest/cli-reference/#jj-config>:.

- Սկսեք խմբագիր օգտագործողի մակարդակի կազմաձևման ֆայլում.:

`jj config {{[e|edit]}} --user`

- Ստացեք կազմաձևման տարբերակի արժեքը.:

`jj config {{[g|get]}} {{name}}`

- Թվարկեք բոլոր կազմաձևման փոփոխականները և դրանց արժեքները.:

`jj config {{[l|list]}}`

- Տպեք ուղին դեպի օգտվողի մակարդակի կազմաձևման ֆայլ.:

`jj config {{[p|path]}} --user`

- Սահմանեք կազմաձևման տարբերակ օգտագործողի մակարդակի կազմաձևում.:

`jj config {{[s|set]}} --user {{name}} {{value}}`

- Անջատեք կազմաձևման տարբերակը օգտագործողի մակարդակի կազմաձևում.:

`jj config {{[u|unset]}} --user {{name}}`
