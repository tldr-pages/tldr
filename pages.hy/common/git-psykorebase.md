# git psykorebase

> Վերակայեք ճյուղը մյուսի վրա՝ օգտագործելով միաձուլման հանձնարարականը և միայն մեկ կոնֆլիկտային լուծում:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/git-psykorebase>:.

- Վերահաստատեք ընթացիկ ճյուղը մյուսի վրա՝ օգտագործելով միաձուլման հանձնարարականը և միայն մեկ կոնֆլիկտային լուծում.:

`git psykorebase {{upstream_branch}}`

- Շարունակեք հակամարտությունները լուծելուց հետո.:

`git psykorebase --continue`

- Նշեք այն մասնաճյուղը, որը պետք է վերահիմնավորվի.:

`git psykorebase {{upstream_branch}} {{target_branch}}`
