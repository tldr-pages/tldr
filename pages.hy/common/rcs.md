# rcs

> Վերանայման կառավարման համակարգ - կառավարել RCS ֆայլի ատրիբուտները:.
> Տես նաև՝ `ci`, `co`, `rcsdiff`, `rlog`:.
> Լրացուցիչ տեղեկություններ. <https://manned.org/rcs>:.

- Բացեք մեկ այլ օգտատիրոջ կողմից կողպված տարբերակը (մուտքի կարիք ունի).:

`rcs -u {{path/to/file}}`

- Կողպեք ֆայլի հատուկ վերանայումը.:

`rcs -l{{revision}} {{path/to/file}}`

- Սահմանեք խիստ կողպման ռեժիմը (գրանցման համար պահանջվում է կողպեք).:

`rcs -L {{path/to/file}}`

- Ավելացնել կամ փոխարինել տեղեկամատյան հաղորդագրություն հատուկ վերանայման համար.:

`rcs -m{{revision}}:"{{new_message}}" {{path/to/file}}`

- Ջնջել հատուկ վերանայում RCS ֆայլից.:

`rcs -o{{revision}} {{path/to/file}}`
