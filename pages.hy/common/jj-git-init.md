# jj git init

> Ստեղծեք նոր Git-ի աջակցությամբ Jujutsu ռեպո:.
> Նշում. Եթե `--colocate`-ը չի օգտագործվում, այն վավեր Git պահոց չէ, և `git` հրամանները չեն կարող օգտագործվել դրա վրա:.
> Լրացուցիչ տեղեկություններ. <https://docs.jj-vcs.dev/latest/cli-reference/#jj-git-init>:.

- Ստեղծեք նոր Git ապահովված ռեպո ընթացիկ գրացուցակում.:

`jj git init`

- Ստեղծեք նոր Git ապահովված ռեպո տվյալ գրացուցակում.:

`jj git init {{path/to/directory}}`

- Նախաձեռնեք Jujutsu-ի պահոցը որպես վավեր Git պահոց (թույլ տալով օգտագործել երկուսն էլ `jj` և `git` հրամանները նույն գրացուցակում):

`jj git init --colocate`

- Նախաձեռնեք Jujutsu պահեստը, որն ապահովված է գոյություն ունեցող Git պահոցով.:

`jj git init --git-repo {{git_repo}}`
