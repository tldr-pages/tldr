# git show-ref

> Git հրամանը հղումների ցուցակման համար:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-show-ref>:.

- Ցուցադրել պահոցում գտնվող բոլոր հղումները.:

`git show-ref`

- Ցույց տալ միայն գլուխների հղումները.:

`git show-ref --branches`

- Ցույց տալ միայն պիտակների հղումները.:

`git show-ref --tags`

- Ստուգեք, որ տվյալ հղումը գոյություն ունի.:

`git show-ref --verify {{path/to/ref}}`
