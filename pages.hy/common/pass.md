# անցում

> Պահպանեք և կարդացեք գաղտնաբառերը կամ այլ զգայուն տվյալներ:.
> Բոլոր տվյալները գաղտնագրված են GPG-ով և կառավարվում են Git պահոցով:.
> Լրացուցիչ տեղեկություններ. <https://git.zx2c4.com/password-store/about/>:.

- Նախաձեռնեք (կամ վերագաղտնագրեք) պահեստը՝ օգտագործելով մեկ կամ մի քանի GPG ID.:

`pass init {{gpg_id_1 gpg_id_2 ...}}`

- Պահպանեք նոր գաղտնաբառ և լրացուցիչ տեղեկություններ (լրացնելու համար սեղմեք `<Ctrl d>` նոր տողում):

`pass insert {{[-m|--multiline]}} {{path/to/data}}`

- Խմբագրել գրառումը.:

`pass edit {{path/to/data}}`

- Պատճենեք գաղտնաբառը (տվյալների ֆայլի առաջին տողը) clipboard-ում.:

`pass {{[-c|--clip]}} {{path/to/data}}`

- Թվարկեք ամբողջ խանութի ծառը.:

`pass`

- Ստեղծեք նոր պատահական գաղտնաբառ տվյալ երկարությամբ և պատճենեք այն clipboard-ում.:

`pass generate {{[-c|--clip]}} {{path/to/data}} {{num}}`

- Նախաձեռնեք նոր Git պահոց (անցումով կատարված ցանկացած փոփոխություն ինքնաբերաբար կկատարվի).:

`pass git init`

- Գործարկեք Git հրամանը գաղտնաբառի պահպանման անունից.:

`pass git {{command}}`
