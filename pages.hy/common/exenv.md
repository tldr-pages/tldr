# exenv

> Հեշտությամբ տեղադրեք Elixir-ի տարբերակները և կառավարեք հավելվածների միջավայրերը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/mururu/exenv>:.

- Ցուցադրել տեղադրված տարբերակների ցանկը.:

`exenv versions`

- Օգտագործեք Elixir-ի հատուկ տարբերակը ամբողջ համակարգում.:

`exenv global {{version}}`

- Օգտագործեք Elixir-ի հատուկ տարբերակը ընթացիկ հավելվածի/նախագծի գրացուցակի համար.:

`exenv local {{version}}`

- Ցույց տալ ներկայումս ընտրված Elixir տարբերակը.:

`exenv {{version}}`

- Տեղադրեք Elixir-ի տարբերակը (պահանջվում է `elixir-build` plugin <https://github.com/mururu/elixir-build>):

`exenv install {{version}}`
