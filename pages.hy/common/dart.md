#տեգեր

> Կառավարեք Dart նախագծերը:.
> Լրացուցիչ տեղեկություններ. <https://dart.dev/tools/dart-tool>:.

- Նախաձեռնեք նոր Dart նախագիծը նույն անունով գրացուցակում.:

`dart create {{project_name}}`

- Գործարկել Dart ֆայլը.:

`dart run {{path/to/file.dart}}`

- Ներբեռնեք կախվածությունը ընթացիկ նախագծի համար.:

`dart pub get`

- Գործարկեք միավորի թեստեր ընթացիկ նախագծի համար.:

`dart test`

- Թարմացրեք հնացած նախագծի կախվածությունները՝ զրոյական անվտանգության ապահովման համար.:

`dart pub upgrade --null-safety`

- Կազմեք Dart ֆայլը բնիկ երկուականի մեջ.:

`dart compile exe {{path/to/file.dart}}`

- Կիրառեք ավտոմատ շտկումներ ընթացիկ նախագծի համար.:

`dart fix --apply`
