# պար2

> Ֆայլերի ստուգում և վերանորոգում` օգտագործելով PAR 2.0 համատեղելի հավասարության արխիվները (.par2 ֆայլեր):.
> Լրացուցիչ տեղեկություններ. <https://github.com/Parchive/par2cmdline/>:.

- Ստեղծեք հավասարության արխիվ՝ ավելորդության սահմանված տոկոսային մակարդակով.:

`par2 create -r{{1..100}} -- {{path/to/file}}`

- Ստեղծեք հավասարաչափ արխիվ ընտրված թվով ծավալային ֆայլերով (ի լրումն ինդեքսի ֆայլի).:

`par2 create -n{{1..32768}} -- {{path/to/file}}`

- Ստուգեք ֆայլը հավասարության արխիվով.:

`par2 verify -- {{path/to/file.par2}}`

- Վերականգնել ֆայլը հավասարության արխիվով.:

`par2 repair -- {{path/to/file.par2}}`
