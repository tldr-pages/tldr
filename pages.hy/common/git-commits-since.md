# git commits-ince

> Ցուցադրել պարտավորությունները ժամանակից կամ ամսաթվից:.
> `git-extras`-ի մի մասը:.
> Լրացուցիչ տեղեկություններ. <https://github.com/tj/git-extras/blob/main/Commands.md#git-commits-since>:.

- Ցուցադրել երեկվանից.:

`git commits-since {{yesterday}}`

- Ցուցադրել պարտավորությունները անցյալ շաբաթվանից.:

`git commits-since {{last week}}`

- Ցուցադրել պարտավորությունները անցյալ ամսից սկսած.:

`git commits-since {{last month}}`

- Ցուցադրումը պարտավորվում է երեկ երեկոյան ժամը 2-ից.:

`git commits-since {{yesterday 2pm}}`
