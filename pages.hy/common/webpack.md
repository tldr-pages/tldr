# վեբ փաթեթ

> Համատեղեք վեբ նախագծի js ֆայլերը և այլ ակտիվները մեկ ելքային ֆայլի մեջ:.
> Լրացուցիչ տեղեկություններ. <https://webpack.js.org/api/cli/>:.

- Ստեղծեք մեկ ելքային ֆայլ մուտքի կետի ֆայլից.:

`webpack {{app.js}} {{bundle.js}}`

- Ներբեռնեք CSS ֆայլերը նաև JavaScript ֆայլից (սա օգտագործում է CSS բեռնիչը CSS ֆայլերի համար).:

`webpack {{app.js}} {{bundle.js}} --module-bind '{{css=css}}'`

- Անցեք կազմաձևման ֆայլ (օրինակ՝ մուտքի սկրիպտով և ելքային ֆայլի անունով) և ցույց տվեք կազմման առաջընթացը.:

`webpack {{[-c|--config]}} {{webpack.config.js}} --progress`

- Ավտոմատ վերակազմավորել նախագծի ֆայլերի փոփոխությունները.:

`webpack {{[-w|--watch]}} {{app.js}} {{bundle.js}}`
