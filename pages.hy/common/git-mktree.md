# git mktree

> Կառուցեք ծառի օբյեկտ՝ օգտագործելով `ls-tree` ձևաչափված տեքստ:.
> Լրացուցիչ տեղեկություններ. <https://git-scm.com/docs/git-mktree>:.

- Կառուցեք ծառի օբյեկտ և ստուգեք, որ յուրաքանչյուր ծառի մուտքի հեշը նույնացնում է գոյություն ունեցող օբյեկտը.:

`git mktree`

- Թույլատրել բացակայող օբյեկտները.:

`git mktree --missing`

- Կարդացեք ծառի օբյեկտի (`git ls-tree -z`) ավարտված ելքը NUL ([z]ero նիշ):

`git mktree -z`

- Թույլատրել մի քանի ծառի օբյեկտների ստեղծում.:

`git mktree --batch`

- Տեսակավորեք և կառուցեք ծառ `stdin`-ից (պահանջվում է ոչ ռեկուրսիվ `git ls-tree` ելքային ձևաչափ):

`git < {{path/to/tree.txt}} mktree`
