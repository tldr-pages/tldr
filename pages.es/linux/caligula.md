# caligula

> Una TUI ligera y fácil de usar para imágenes de disco.
> Vea también: `dd`.
> Más información: <https://github.com/ifd3f/caligula>.

- Graba una imagen ISO en una unidad:

`caligula burn {{ruta/a/imagen.iso}}`

- Graba una imagen ISO en una unidad sin introducir el hash de forma interactiva:

`caligula burn {{ruta/a/imagen.iso}} {{[-s|--hash]}} {{hash}}`
