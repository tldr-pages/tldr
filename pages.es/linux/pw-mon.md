# pw-mon

> Supervisa objetos en la instancia de PipeWire.
> Más información: <https://docs.pipewire.org/page_man_pw-mon_1.html>.

- Supervisa la instancia predeterminada de PipeWire:

`pw-mon`

- Supervisa una instancia remota específica:

`pw-mon {{[-r|--remote]}} {{nombre_remoto}}`

- Supervisar la instancia predeterminada especificando una configuración de color:

`pw-mon {{[-N|--color]}} {{never|always|auto}}`

- Muestra la ayuda:

`pw-mon {{[-h|--help]}}`
