# informant

> Lea las noticias sobre paquetes Arch Linux.
> Más información: <https://github.com/bradford-smith94/informant>.

- Lee todas las noticias no leídas:

`sudo informant read`

- Busca noticias:

`informant check`

- Lista las últimas noticias:

`informant list`

- Muestra la ayuda:

`informant --help`
