# groupmod

> Modifica los grupos de usuarios existentes en el sistema.
> Vea también: `groups`, `groupadd`, `groupdel`.
> Más información: <https://manned.org/groupmod>.

- Cambia el nombre del grupo:

`sudo groupmod {{[-n|--new-name]}} {{nuevo_grupo}} {{nombre_del_grupo}}`

- Cambia el ID del grupo:

`sudo groupmod {{[-g|--gid]}} {{nuevo_id}} {{nombre_grupo}}`
