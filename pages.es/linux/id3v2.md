# id3v2

> Gestiona etiquetas id3v2, convierte y muestra una lista de id3v1.
> Más información: <https://manned.org/id3v2>.

- Muestra todos los géneros:

`id3v2 {{[-L|--list-genres]}}`

- Muestra todas las etiquetas de archivos específicos:

`id3v2 {{[-l|--list]}} {{ruta/al/archivo1 ruta/al/archivo2 ...}}`

- Elimina todas las etiquetas `id3v2` o `id3v1` de archivos específicos:

`id3v2 {{--delete-v2|--delete-v1}} {{ruta/al/archivo1 ruta/al/archivo2 ...}}`

- Muestra la ayuda:

`id3v2 {{[-h|--help]}}`

- Muestra la versión:

`id3v2 {{[-v|--version]}}`
