# dolphin

> El gestor de archivos de KDE para gestionar archivos y directorios.
> Vea también: `nautilus`, `caja`, `thunar`, `ranger`.
> Más información: <https://apps.kde.org/dolphin/>.

- Inicia el gestor de archivos:

`dolphin`

- Abre directorios específicos:

`dolphin {{ruta/al/directorio1 ruta/al/directorio2 ...}}`

- Abre con archivos o directorios específicos seleccionados:

`dolphin --select {{ruta/al/archivo_o_directorio1 ruta/al/archivo_o_directorio2 ...}}`

- Abre una nueva ventana:

`dolphin --new-window`

- Abre directorios específicos en vista dividida:

`dolphin --split {{ruta/al/directorio1}} {{ruta/al/directorio2}}`

- Inicia el programa residente (solo necesario para usar la interfaz D-Bus):

`dolphin --daemon`

- Muestra la ayuda:

`dolphin {{[-h|--help]}}`
