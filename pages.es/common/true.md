# true

> Devuelve un código de estado de salida satisfactorio de 0.
> Vea también: `false`.
> Más información: <https://www.gnu.org/software/bash/manual/bash.html#index-true>.

- Devuelve un código de salida satisfactorio:

`true`

- Hace que un comando siempre finalice con 0:

`{{comando}} || true`
