# more

> Abre un archivo para lectura interactiva, permitiendo desplazar y buscar (Solamente en una dirección, adelante).

- Abre un archivo:

`more {{archivo}}`

- Página abajo:

`<Espacio>`

- Busca una termino (La tecla `n` se mueve al siguiente resultado):

`/{{termino de busqueda}}`

- Salir:

`q`
