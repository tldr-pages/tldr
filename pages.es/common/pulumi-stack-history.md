# pulumi stack history

> Muestra el historial de una pila.
> Más información: <https://www.pulumi.com/docs/iac/cli/commands/pulumi_stack_history/>.

- Muestra el historial de la pila actual:

`pulumi stack {{[hist|history]}}`

- Muestra el historial de la pila actual con fechas completas en lugar de fechas relativas:

`pulumi stack {{[hist|history]}} --full-dates`

- Muestra el historial de la pila actual en formato JSON:

`pulumi stack {{[hist|history]}} {{[-j|--json]}}`

- Muestra el historial de una pila específica:

`pulumi stack {{[hist|history]}} {{[-s|--stack]}} {{nombre_pila}}`

- Muestra la ayuda:

`pulumi stack {{[hist|history]}} {{[-h|--help]}}`
