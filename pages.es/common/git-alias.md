# git alias

> Crea accesos directos para comandos de Git.
> Parte de `git-extras`.
> Más información: <https://github.com/tj/git-extras/blob/master/Commands.md#git-alias>.

- Lista todos los alias:

`git alias`

- Crea un nuevo alias:

`git alias "{{nombre}}" "{{comando}}"`

- Busca un alias existente:

`git alias ^{{nombre}}`
