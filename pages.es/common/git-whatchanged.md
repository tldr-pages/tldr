# git whatchanged

> Este comando es un alias de `git log --raw --no-merges` (registro con diferencias sin procesar por cada confirmación).
> Nota: Este comando está en desuso y se eliminará en el futuro.
> Más información: <https://git-scm.com/docs/git-whatchanged>.

`tldr git log`
