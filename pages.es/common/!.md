# !

> Bash incorporado para sustituir con un comando encontrado en la historia.
> Más información: <https://gnu.org/software/bash/manual/bash.html#Event-Designators>.

Substitute with the previous command and run it with sudo:

sudo !!

Substitute with a command based on its line number found with history:

!{{number}}

Substitute with a command that was used a specified number of lines back:

!-{{number}}

Substitute with the most recent command that starts with a string:

!{{string}}

Print last command that starts with a string without executing it:

!{{string}}:p

Substitute with the most recent command containing a specific string:

!?{{string}}?

Print the most recent command containing a specific string without executing it:

!?{{string}}?:p

Substitute with the arguments of the latest command:

{{command}} !*

Substitute with the last argument of the latest command:

{{command}} !$

Substitute with the first argument of the latest command:

{{command}} !^

Substitute with the nth argument of the latest command:

{{command}} !:{{n}}

Substitute with a range of arguments from the latest command:

{{command}} !:{{n}}-{{m}}

Substitute with the last command but without the last argument:

!:-

Perform text substitution in the previous command and execute it:

^{{old}}^{{new}}^

Perform text substitution in a specific command from history:

!{{event}}:s/{{old}}/{{new}}/

Perform global text substitution in a specific command from history:

!{{event}}:gs/{{old}}/{{new}}/

Display the expanded form of a history command before executing (useful for debugging expansions):

set -o histexpand
!{{string}}:p

Disable history expansion temporarily:

set +H
