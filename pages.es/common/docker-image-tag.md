# docker image tag

> Asigna etiquetas a imágenes de Docker existentes.
> Más información: <https://docs.docker.com/reference/cli/docker/image/tag/>.

- Asigna un nombre y una etiqueta a un ID de imagen específico:

`docker {{[tag|image tag]}} {{id}} {{nombre}}:{{etiqueta}}`

- Asigna una etiqueta a una imagen específica:

`docker {{[tag|image tag]}} {{imagen}}:{{etiqueta_actual}} {{imagen}}:{{nueva_etiqueta}}`

- Muestra la ayuda:

`docker {{[tag|image tag]}}`
