# brew info

> Muestra información sobre la instalación de Homebrew y sus fórmulas.
> Más información: <https://docs.brew.sh/Manpage#info-abv-options-formulacask->.

- Muestra estadísticas de la instalación de Homebrew:

`brew info`

- Muestra más información sobre una fórmula o cask:

`brew info {{formula|cask}}`

- Muestra información detallada sobre una fórmula o cask:

`brew info {{[-v|--verbose]}} {{formula|cask}}`

- Muestra información sobre una fórmula o cask en formato JSON:

`brew info --json {{formula|cask}}`

- Muestra en formato JSON las fórmulas instaladas:

`brew info --json --installed`

- Muestra la ayuda:

`brew info {{[-h|--help]}}``
