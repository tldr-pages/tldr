# users

> Muestra una lista de usuarios conectados.
> Vea también: `who`, `w`, `useradd`, `userdel`, `usermod`.
> Más información: <https://www.gnu.org/software/coreutils/manual/html_node/users-invocation.html>.

- Imprime los nombres de usuario conectados:

`users`

- Muestra los nombres de usuario conectados según un archivo determinado:

`users {{/var/log/wmtp}}`
