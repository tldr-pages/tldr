# gdm-safe-restart

> Reinicia el gestor de pantalla de GNOME (GDM) solo cuando todos los usuarios hayan cerrado sesión.
> Vea también: `gdm`, `gdm-binary`, `gdmsetup`, `gdm-stop`, `gdm-restart`.
> Más información: <https://manned.org/gdm>.

- Reinicia la aplicación GNOME Display Manager cuando todos los usuarios hayan cerrado sesión:

`gdm-safe-restart`
