# age-inspect

> Inspecciona archivos cifrados con `age`.
> Vea también: `age`, `age-keygen`.
> Más información: <https://manned.org/age-inspect>.

- Muestra el tipo de destinatario, el tamaño del archivo y si utiliza cifrado poscuántico:

`age-inspect {{ruta/al/archivo}}`

- Muestra el resultado en `stdout` como JSON:

`age-inspect --json {{ruta/al/archivo}}`
