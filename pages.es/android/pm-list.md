# pm list

> Muestra una lista de usuarios, paquetes, permisos, instrumentación, grupos de permisos, funciones y bibliotecas gestionadas por el gestor de paquetes.
> Más información: <https://developer.android.com/tools/adb#pm>.

- Muestra todos los paquetes instalados:

`pm list packages`

- Muestra paquetes del [s]istema o de terceros:

`pm list packages {{-s|-3}}`

- Muestra todos los usuarios del sistema:

`pm list users`

- Muestra todos los grupos de permisos conocidos:

`pm list permission-groups`

- Muestra todos los permisos conocidos:

`pm list permissions`

- Muestra todos los paquetes de prueba:

`pm list instrumentation`

- Muestra todas las funciones del sistema:

`pm list features`

- Muestra todas las bibliotecas compatibles con el dispositivo actual:

`pm list libraries`
