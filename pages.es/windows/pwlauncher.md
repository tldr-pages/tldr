# pwlauncher

> Gestionar las opciones de inicio de Windows To Go.
> Más información: <https://learn.microsoft.com/windows-server/administration/windows-commands/pwlauncher>.

- Mostrar el estado actual de Windows To Go:

`pwlauncher`

- Habilitar o deshabilitar las opciones de inicio de Windows To Go:

`pwlauncher /{{habilitar|deshabilitar}}`
