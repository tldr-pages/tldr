# tzutil

> Herramienta para mostrar o configurar la zona horaria del sistema.
> Más información: <https://learn.microsoft.com/windows-server/administration/windows-commands/tzutil>.

- Obtiene la zona horaria actual:

`tzutil /g`

- Muestra una lista de zonas horarias disponibles:

`tzutil /l`

- Establece la zona horaria del sistema al valor específico:

`tzutil /s {{id_de_zona_horaria}}`
