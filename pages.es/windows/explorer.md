# explorer

> El Explorador de Archivos de Windows.
> Más información: <https://ss64.com/nt/explorer.html>.

- Abre el Explorador de Windows:

`explorer`

- Abre el Explorador de Windows en el directorio actual:

`explorer .`

- Abre el Explorador de Windows en un directorio específico:

`explorer {{ruta\al\directorio}}`
