# awk

> 用於處理檔案的多功能程式語言。
> 注意：各式 AWK 實作經常會將此命令作為指向其二進位檔的符號連結。
> 另請參閱：`gawk`。
> 更多資訊：<https://github.com/onetrueawk/awk>。

- 以空格為分隔符，列印檔案每行的第五欄：

`awk '{print $5}' {{路徑/至/檔案}}`

- 以空格為分隔符，列印檔案中所有包含「foo」之行的第二欄：

`awk '/{{foo}}/ {print $2}' {{路徑/至/檔案}}`

- 以逗號（而非空格）作為分隔符，列印檔案每一行的最後一欄：

`awk -F ',' '{print $NF}' {{路徑/至/檔案}}`

- 計算檔案中第一欄的數值總和並列印：

`awk '{s+=$1} END {print s}' {{路徑/至/檔案}}`

- 從第一行開始，每三行列印一行：

`awk 'NR%3==1' {{路徑/至/檔案}}`

- 根據條件列印不同的值：

`awk '{if ($1 == "foo") print "Exact match foo"; else if ($1 ~ "bar") print "Partial match bar"; else print "Baz"}' {{路徑/至/檔案}}`

- 列印第 10 欄中數值介於最小值與最大值之間的所有行：

`awk '($10 >= {{最小值}} && $10 <= {{最大值}})' {{路徑/至/檔案}}`

- 以冒號為分隔符，並以含有表頭且格式化的表格列印 UID 大於等於 1000 的使用者資訊（`%-20s`：靠左對齊的 20 個字串字元，`%6s`：靠右對齊的 6 個字串字元）：

`awk 'BEGIN {FS=":";printf "%-20s %6s %25s\n", "Name", "UID", "Shell"} $4 >= 1000 {printf "%-20s %6d %25s\n", $1, $4, $7}' /etc/passwd`
