# bat

> 可以列印並且合併檔案的命令。
> `cat` 的複製品，外加語法高亮和 Git 整合。
> 更多資訊：<https://manned.org/bat>。

- 使用美化樣式列印一個或多個檔案的內容到 `stdout`：

`bat {{路徑/至/檔案1 路徑/至/檔案2 ...}}`

- 將多個檔案連接成一個目標檔案：

`bat {{路徑/至/檔案1 路徑/至/檔案2 ...}} > {{路徑/至/目標檔案}}`

- 移除修飾並停用分頁：

`bat {{[-pp|--style plain --pager never]}} {{路徑/至/檔案}}`

- 使用不同的背景顏色突顯顯示特定行或一行範圍：

`bat {{[-H|--highlight-line]}} {{10|5:10|:10|10:|10:+5}} {{路徑/至/檔案}}`

- 顯示不可列印字元，如空格、定位字元或換行：

`bat {{[-A|--show-all]}} {{路徑/至/檔案}}`

- 在輸出中只保留行號，移除所有其他修飾：

`bat {{[-n|--number]}} {{路徑/至/檔案}}`

- 透過明確設定語言來語法高亮顯示 JSON 檔案：

`bat {{[-l|--language]}} json {{路徑/至/檔案.json}}`

- 顯示所有支援的語言：

`bat {{[-L|--list-languages]}}`
