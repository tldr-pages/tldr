# cat

> 連接檔案並印出檔案的內容。
> 更多資訊：<https://www.gnu.org/software/coreutils/cat>.

- 將檔案的內容印在標準輸出：

`cat {{檔案}}`

- 將多個檔案連接起來，輸出至目標檔案：

`cat {{檔案一}} {{檔案二}} > {{目標檔案}}`

- 將多個檔案連接起來，並將其內容加到目標檔案的結尾：

`cat {{檔案一}} {{檔案二}} >> {{目標檔案}}`

- 印出檔案的內容並顯示行號：

`cat -n {{檔案}}`

- 印出檔案的內容，且將無法顯示的字元用特殊的方式顯示出來：

`cat -v -t -e {{檔案}}`
