# PATH

> 실행 파일을 찾기 위한 검색 경로를 표시하거나 설정하는 명령어.
> 이 명령어는 `FreeDOS`에서만 사용 가능.
> 더 많은 정보: <https://help.fdos.org/en/hhstndrd/batch/path.htm>.

- 현재 검색 경로 출력:

`PATH`

- 세미콜론으로 구분된 디렉터리 목록 설정:

`PATH {{경로/대상/디렉터리1;경로/대상/디렉터리2;...}}`

- 기존 검색 경로 뒤에 디렉터리 추가:

`PATH %PATH%;{{경로/대상/디렉터리}}`

- 기존 검색 경로 앞에 디렉터리 추가:

`PATH {{경로/대상/디렉터리}};%PATH%`

- 검색 경로 초기화:

`PATH;`

- 도움말 표시:

`PATH /?`
