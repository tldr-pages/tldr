# flutterfire

> Flutter 애플리케이션에 Firebase 프로젝트 설정.
> 참고: Firebase CLI 및 FlutterFire CLI가 설치되어 있어야 함.
> 더 많은 정보: <https://firebase.google.com/docs/flutter/setup>.

- 현재 Flutter 앱에 Firebase 초기화 및 설정 수행:

`flutterfire configure`

- 지원할 플랫폼(iOS, Android, web) 지정:

`flutterfire configure --platforms {{ios,android,...}}`

- 특정 Firebase 프로젝트 사용:

`flutterfire configure --project {{프로젝트_아이디}}`
