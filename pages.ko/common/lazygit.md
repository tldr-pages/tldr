# lazygit

> Git 저장소 관리를 위한 간단한 터미널 UI.
> 더 많은 정보: <https://manned.org/lazygit>.

- 현재 저장소에서 Lazygit 실행:

`lazygit`

- 특정 Git 저장소에서 Lazygit 실행:

`lazygit {{[-p|--path]}} {{경로/대상/저장소}}`

- 특정 패널에 포커스를 두고 Lazygit 시작:

`lazygit {{status|branch|log|stash|...}}`

- 기본 Lazygit 설정 출력:

`lazygit {{[-c|--config]}}`

- Lazygit 로그 실시간 확인 (다른 터미널에서 디버그 모드 사용 시 유용):

`lazygit {{[-l|--logs]}}`

- 디버그 모드로 Lazygit 실행:

`lazygit {{[-d|--debug]}}`

- 설정 디렉터리 경로 출력:

`lazygit {{[-cd|--print-config-dir]}}`
