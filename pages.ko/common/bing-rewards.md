# bing-rewards

> Bing Rewards 포인트를 매일 자동으로 적립.
> 더 많은 정보: <https://github.com/jack-mil/bing-rewards>.

- 데스크톱 및 모바일 검색 모두 수행:

`bing-rewards`

- 모바일 사용자 에이전트로 10회 검색 수행:

`bing-rewards {{[-m|--mobile]}} {{[-c|--count]}} 10`

- 지정한 Chrome 프로필 "Profile 1"을 사용하여 모바일 및 데스크톱 검색 수행:

`bing-rewards --profile "Profile 1"`

- 여러 Chrome 프로필에서 순차적으로 검색 수행:

`bing-rewards --profile "Default" "Profile 1" "Profile 2"`

- 도움말 표시:

`bing-rewards {{[-h|--help]}}`
