# kiterunner

> 워드리스트와 kitebuilder 스키마를 사용하여 API 경로 및 웹 엔드포인트를 탐지하는 컨텍스트 기반 웹 스캐너.
> 더 많은 정보: <https://github.com/assetnote/kiterunner#usage>.

- API 경로 및 웹 엔드포인트 bruteforcing 문서 보기:

`tldr kiterunner brute`

- kitebuilder 워드리스트를 사용하여 여러 호스트를 동시 스캔하는 문서 보기:

`tldr kiterunner scan`

- kitebuilder 스키마 조작 관련 문서 보기:

`tldr kiterunner kb`

- 캐시된 워드리스트 및 원격 워드리스트 관리 관련 문서 보기:

`tldr kiterunner wordlist`
