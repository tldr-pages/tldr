# pkgz

> 여러 소스를 통합하여 관리하는 패키지 관리자.
> 더 많은 정보: <https://github.com/roguehashrate/pkgz#-usage>.

- 패키지 설치:

`pkgz install {{패키지_이름}}`

- 패키지 제거:

`pkgz remove {{패키지_이름}}`

- 패키지 검색:

`pkgz search {{패키지_이름}}`

- 소스별 설치된 패키지 개수 표시:

`pkgz info`

- 패키지가 설치된 위치 또는 사용 가능한 소스를 표시:

`pkgz info {{패키지_이름}}`

- 모든 소스의 패키지 업데이트:

`pkgz update`

- 지원되는 모든 소스의 캐시 제거:

`pkgz clean`

- 버전 정보 표시:

`pkgz --version`
