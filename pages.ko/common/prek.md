# prek

> 커밋 전에 실행되는 Git hook을 생성하고 관리.
> 더 많은 정보: <https://prek.j178.dev/reference/cli/>.

- Git hook에 prek 설치:

`prek install`

- staging된 모든 파일에 대해 prek hook 실행:

`prek run`

- Run prek hooks on all files, staged or unstaged:

`prek run {{[-a|--all-files]}}`

- Clean prek cache:

`prek cache clean`

- Update prek configuration file to the latest repos' versions:

`prek auto-update`
