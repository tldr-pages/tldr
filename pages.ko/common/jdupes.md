# jdupes

> 강력한 중복 파일 찾기 도구이자 fdupes의 향상된 포크.
> 더 많은 정보: <https://codeberg.org/jbruchon/jdupes>.

- 단일 디렉토리 검색:

`jdupes {{경로/대상/폴더}}`

- 여러 디렉토리 검색:

`jdupes {{디렉토리1}} {{디렉토리2}}`

- 모든 디렉토리를 재귀적으로 검색:

`jdupes --recurse {{경로/대상/폴더}}`

- 디렉토리를 재귀적으로 검색하고 사용자가 보존할 파일을 선택하도록 허용:

`jdupes --delete --recurse {{경로/대상/폴더}}`

- 여러 디렉토리를 검색하고 디렉토리1이 아닌 디렉토리2의 하위 디렉토리를 따라가기:

`jdupes {{디렉토리1}} --recurse: {{디렉토리2}}`

- 여러 디렉토리를 검색하고 결과에서 디렉토리 순서 유지:

`jdupes -O {{디렉토리1}} {{디렉토리2}} {{디렉토리3}}`
