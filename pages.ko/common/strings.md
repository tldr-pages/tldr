# strings

> 객체 파일이나 바이너리에서 출력 가능한 문자열을 찾습니다.
> 더 많은 정보: <https://manned.org/strings>.

- 바이너리의 모든 문자열 출력:

`strings {{경로/대상/파일}}`

- 결과를 최소 n 글자 이상의 문자열로 제한:

`strings -n {{n}} {{경로/대상/파일}}`

- 각 결과 앞에 파일 내 오프셋 접두사 추가:

`strings -t d {{경로/대상/파일}}`

- 각 결과 앞에 파일 내 오프셋을 16진수로 접두사 추가:

`strings -t x {{경로/대상/파일}}`
