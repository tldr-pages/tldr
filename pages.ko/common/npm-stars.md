# npm stars

> 즐겨찾기로 표시한 패키지를 조회하는 명령어.
> 더 많은 정보: <https://docs.npmjs.com/cli/npm-stars/>.

- 현재 로그인한 사용자의 즐겨찾기 패키지 목록 출력:

`npm stars`

- 특정 레지스트리에서 즐겨찾기 패키지 조회:

`npm stars --registry {{레지스트리_주소}}`

- 특정 사용자가 즐겨찾기한 패키지 목록 출력:

`npm stars {{사용자명}}`

- 특정 레지스트리에서 특정 사용자의 즐겨찾기 패키지 조회:

`npm stars {{사용자명}} --registry {{레지스트리_주소}}`

- 도움말 표시:

`npm stars {{[-h|--help]}}`
