# swift

> Swift 프로젝트를 생성, 실행 및 빌드.
> 더 많은 정보: <https://manned.org/swiftc>.

- REPL(대화형 셸) 시작:

`swift repl`

- 프로그램 실행:

`swift {{파일.swift}}`

- 패키지 관리자로 새 프로젝트 시작:

`swift package init`

- Xcode 프로젝트 파일 생성:

`swift package generate-xcodeproj`

- 의존성 업데이트:

`swift package update`

- 프로젝트를 릴리스용으로 컴파일:

`swift build -c release`
