# doppler projects

> Doppler 프로젝트 관리.
> 더 많은 정보: <https://docs.doppler.com/docs/cli>.

- 모든 프로젝트 가져오기:

`doppler projects`

- 프로젝트에 대한 정보 얻기:

`doppler projects get {{이름|프로젝트_아이디}}`

- 프로젝트 생성:

`doppler projects create {{이름}}`

- 프로젝트 이름과 설명 업데이트:

`doppler projects update {{이름|프로젝트_아이디}} --name "{{새로운_이름}}" --description "{{새로운_설명}}"`

- 프로젝트 삭제:

`doppler projects delete {{이름|프로젝트_아이디}}`
