# opencode web

> 웹 인터페이스가 포함된 OpenCode 서버를 실행하는 명령어.
> 더 많은 정보: <https://opencode.ai/docs/cli#web>.

- 기본 포트로 웹 인터페이스 시작:

`opencode web`

- 특정 포트로 실행:

`opencode web --port {{8080}}`

- 특정 호스트 주소로 실행:

`opencode web --hostname {{0.0.0.0}}`

- mDNS 서비스 탐색 활성화:

`opencode web --mdns`

- 도움말 표시:

`opencode web {{[-h|--help]}}`
