# cava

> 크로스 플랫폼 오디오 시각화 도구.
> 더 많은 정보: <https://github.com/karlstav/cava#usage>.

- 시각화 도구를 시작:

`cava`

- 지정한 설정 파일 사용:

`cava -p {{경로/대상/파일}}`

- 감도를 증가/감소:

`{{<ArrowUp>|<ArrowDown>}}`

- 바(bar)의 개수를 증가/감소:

`{{<ArrowRight>|<ArrowLeft>}}`

- 설정 파일을 다시 불러옴 (바 개수와 감도가 초기화):

`<r>`

- 설정 파일에서 색상 설정을 다시 불러옴:

`<c>`

- 전경/배경 색상을 순환 변경:

`{{<f>|<b>}}`

- 종료:

`<q>`
