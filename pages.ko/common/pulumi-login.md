# pulumi login

> Pulumi 클라우드에 로그인.
> 더 많은 정보: <https://www.pulumi.com/docs/cli/commands/pulumi_login/>.

- 관리되는 Pulumi Cloud 백엔드에 로그인. 기본값은 `app.pulumi.cloud`:

`pulumi login`

- 지정된 URL의 자체 호스팅된 Pulumi Cloud 백엔드에 로그인:

`pulumi login {{URL}}`

- Pulumi Cloud와 독립적으로 로컬에서 Pulumi 사용:

`pulumi login {{-l|--local}}`
