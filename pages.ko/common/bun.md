# bun

> JavaScript 런타임 및 툴킷.
> 번들러, 테스트 러너 및 패키지 관리자가 포함.
> 더 많은 정보: <https://bun.com/docs>.

- JavaScript 파일 또는 `package.json` 스크립트 실행:

`bun run {{경로/대상/파일|스크립트_이름}}`

- 단위 테스트 실행:

`bun test`

- `package.json`에 종속성으로 나열된 모든 패키지를 다운로드하고 설치:

`bun install`

- `package.json`에 의존성 추가:

`bun add {{모듈_이름}}`

- `package.json`로부터 의존성 제거:

`bun remove {{모듈_이름}}`

- 현재 디렉토리에 새로운 Bun 프로젝트 생성:

`bun init`

- REPL 시작 (대화형 쉘):

`bun repl`

- Bun을 최신버전으로 업그레이드:

`bun upgrade`
