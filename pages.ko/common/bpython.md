# bpython

> Python 인터프리터를 위한 향상된 인터페이스.
> REPL 모드에서 구문 강조와 다양한 편의 기능을 제공.
> 더 많은 정보: <https://manned.org/bpython>.

- REPL (대화형 셸)을 시작:

`bpython`

- 특정 Python 파일을 실행:

`bpython {{경로/대상/파일.py}}`

- 특정 Python 파일을 실행 한 후, REPL을 시작:

`bpython {{[-i|--interactive]}} {{경로/대상/파일.py}}`

- 기본 설정 대신 지정한 설정 파일을 사용:

`bpython --config {{경로/대상/파일.conf}}`
