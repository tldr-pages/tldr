# markdown2pdf

> markdownмқ„ PDFлЎң ліҖнҷҳ.
> лҚ” л§ҺмқҖ м •ліҙ: <https://github.com/theiskaa/markdown2pdf>.

- Markdown нҢҢмқјмқ„ PDFлЎң ліҖнҷҳ:

`markdown2pdf {{[-p|--path]}} {{кІҪлЎң/лҢҖмғҒ/мһ…л Ҙ_нҢҢмқј.md}}`

- Markdown нҢҢмқјмқ„ м§Җм •н•ң кІҪлЎңмқҳ PDFлЎң ліҖнҷҳ:

`markdown2pdf {{[-p|--path]}} {{кІҪлЎң/лҢҖмғҒ/мһ…л Ҙ_нҢҢмқј.md}} {{[-o|--output]}} {{кІҪлЎң/лҢҖмғҒ/м¶ңл Ҙ_нҢҢмқј.pdf}}`

- л¬ёмһҗм—ҙ нҳ•нғңмқҳ Markdown лӮҙмҡ©мқ„ PDFлЎң ліҖнҷҳ:

`markdown2pdf {{[-s|--string]}} {{markdown_л¬ёмһҗм—ҙ}} {{[-o|--output]}} {{кІҪлЎң/лҢҖмғҒ/м¶ңл Ҙ_нҢҢмқј.pdf}}`

- URLлЎңл¶Җн„° ліҖнҷҳ (н•ҙлӢ№ URLм—җ мһҲлҠ” Markdown нҢҢмқјмқ„ лЎңм»¬ PDF нҢҢмқјлЎң ліҖнҷҳ):

`markdown2pdf {{[-u|--url]}} {{мЈјмҶҢ}} {{[-o|--output]}} {{кІҪлЎң/лҢҖмғҒ/м¶ңл Ҙ_нҢҢмқј.pdf}}`
