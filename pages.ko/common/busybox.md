# busybox

> 하나의 실행 파일에 여러 개의 작은 시스템 유틸리티를 모아놓은 도구.
> `busybox`를 심볼릭 링크를 통해 실행하면 `busybox symlink_name`을 실행하는 것과 동일.
>  BusyBox를 사용하는 Linux 배포판은 일반적으로 각 프로그램에 대한 심볼릭 링크를 제공.
> 더 많은 정보: <https://www.busybox.net/downloads/BusyBox>.

- BusyBox 기능 실행:

`busybox {{ls|rm|mkdir|cat|...}} {{매개변수}}`

- 도움말 및 사용 가능한 기능 목록 출력:

`busybox --help`
