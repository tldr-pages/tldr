# bun init

> 빈 Bun 프로젝트를 생성(스캐폴딩).
> 더 많은 정보: <https://bun.com/docs/runtime/templating/init>.

- 현재 디렉토리에서 Bun 프로젝트 초기화:

`bun init`

- 특정 디렉토리에서 초기화:

`bun init {{경로/대상/디렉토리}}`

- 특정 프로젝트 이름으로 초기화:

`bun init {{[-n|--name]}} {{프로젝트_이름}}`

- 기본 프롬프트로 프로젝트 초기화:

`bun init {{[-y|--yes]}}`

- 최소 구성 프로젝트 초기화:

`bun init {{[-m|--minimal]}}`

- React 프로젝트 초기화:

`bun init {{[-r|--react]}}`

- Tailwind CSS가 포함된 React 프로젝트 초기화:

`bun init --react=tailwind`
