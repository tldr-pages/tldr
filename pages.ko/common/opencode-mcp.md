# opencode mcp

> MCP (Model Context Protocol) 서버를 관리하는 명령어.
> 더 많은 정보: <https://opencode.ai/docs/cli#mcp>.

- MCP 서버 및 상태 목록 출력:

`opencode mcp list`

- MCP 서버를 대화형으로 추가:

`opencode mcp add`

- OAuth 지원 MCP 서버에 인증:

`opencode mcp auth {{서버_이름}}`

- MCP 서버의 OAuth 자격 증명 제거:

`opencode mcp logout {{서버_이름}}`

- MCP 서버의 OAuth 연결 디버깅:

`opencode mcp debug {{서버_이름}}`

- 도움말 표시:

`opencode mcp {{[-h|--help]}}`
