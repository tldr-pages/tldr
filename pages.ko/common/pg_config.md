# pg_config

> 설치된 PostgreSQL 버전에 대한 정보를 표시.
> 더 많은 정보: <https://www.postgresql.org/docs/current/app-pgconfig.html>.

- PostgreSQL 실행 파일 디렉터리 표시:

`pg_config --bindir`

- 라이브러리 파일 디렉터리 표시:

`pg_config --libdir`

- 헤더 파일 (C include 파일) 디렉터리 표시:

`pg_config --includedir`

- 공유 데이터 파일 디렉터리 표시:

`pg_config --sharedir`

- 로케일 데이터 디렉터리 표시:

`pg_config --localedir`

- 도움말 표시:

`pg_config {{[-?|--help]}}`

- 버전 표시:

`pg_config --version`
