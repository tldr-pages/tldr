# linode-cli events

> Linode 이벤트 관리.
> 같이 보기: `linode-cli`.
> 더 많은 정보: <https://techdocs.akamai.com/cloud-computing/docs/cli-commands-for-viewing-events>.

- 계정의 이벤트 목록 보기:

`linode-cli events list`

- 특정 이벤트에 대한 세부 정보 보기:

`linode-cli events view {{이벤트_ID}}`

- 이벤트를 읽은 것으로 표시:

`linode-cli events mark-read {{이벤트_ID}}`
