# ng new

> 새로운 Angular 애플리케이션 생성 및 초기화.
> 더 많은 정보: <https://angular.dev/cli/new>.

- 새로운 Angular 애플리케이션 생성:

`ng {{[n|new]}} {{애플리케이션_이름}}`

- 실제 파일 생성 없이 수행 작업 미리 보기:

`ng {{[n|new]}} {{애플리케이션_이름}} {{[-d|--dry-run]}}`

- 단위 테스트 파일 (`spec.ts`) 생성 건너뛰기:

`ng {{[n|new]}} {{애플리케이션_이름}} {{[-S|--skip-tests]}}`

- 패키지 자동 설치 건너뛰기:

`ng {{[n|new]}} {{애플리케이션_이름}} --skip-install`

- Git 저장소 초기화 건너뛰기:

`ng {{[n|new]}} {{애플리케이션_이름}} {{[-g|--skip-git]}}`

- 프로젝트를 위한 AI 도구 설정 파일 생성:

`ng {{[n|new]}} {{애플리케이션_이름}} --ai-config {{claude|copilot|cursor|gemini|jetbrains|none|windsurf}}`

- 대화형 프롬프트 비활성화:

`ng {{[n|new]}} {{애플리케이션_이름}} --interactive false`
