# cargo report

> 다양한 종류의 보고서를 표시.
> 더 많은 정보: <https://doc.rust-lang.org/cargo/commands/cargo-report.html>.

- 보고서 표시:

`cargo report {{미래의-비호환성|...}}`

- 지정된 Cargo 생성 ID가 포함된 보고서를 표시:

`cargo report {{미래의-비호환성|...}} --id {{아이디}}`

- 지정된 패키지에 대한 보고서를 표시:

`cargo report {{미래의-비호환성|...}} --package {{패키지}}`
