# mullvad

> Mullvad VPN을 위한 CLI 클라이언트.
> 같이 보기: `fastd`, `ivpn`, `mozillavpn`, `warp-cli`.
> 더 많은 정보: <https://mullvad.net/>.

- 지정된 계정 번호로 Mullvad 계정 연결:

`mullvad account set {{계정_번호}}`

- VPN이 켜져 있는 동안 LAN 액세스 허용:

`mullvad lan set allow`

- VPN 터널 연결:

`mullvad connect`

- VPN 터널 상태 확인:

`mullvad status`
