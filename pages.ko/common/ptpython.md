# ptpython

> 개선된 Python REPL.
> 더 많은 정보: <https://github.com/prompt-toolkit/ptpython>.

- REPL(대화형 셸) 시작:

`ptpython`

- 특정 Python 파일 실행:

`ptpython {{경로/대상/파일.py}}`

- 특정 Python 파일을 실행하고 REPL 시작:

`ptpython -i {{경로/대상/파일.py}}`

- 메뉴 열기:

`F2`

- 기록 페이지 열기:

`F3`

- 붙여넣기 모드 전환:

`F6`

- 종료:

`<Ctrl> + D`
