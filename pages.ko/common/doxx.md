# doxx

> 터미널에서 `.docx` 파일 확인 도구.
> 더 많은 정보: <https://github.com/bgreenwell/doxx#-command-line-options>.

- 터미널에서 `.docx` 파일 보기:

`doxx {{경로/대상/파일}}.docx`

- 컬러 지원으로 `.docx` 파일 보기:

`doxx {{경로/대상/파일}}.docx --color`

- 이미지 지원으로 `.docx` 파일 보기:

`doxx {{경로/대상/파일}}.docx --images`

- 특정 페이지로 이동해, 터미널에서 `.docx` 파일 보기:

`doxx {{경로/대상/파일}}.docx {{[-p|--page]}} {{페이지_번호}}`

- `.docx` 파일에서 패턴 검색:

`doxx {{경로/대상/파일}}.docx {{[-s|--search]}} {{패턴}}`

- 문서 보기 중 패턴 검색:

`<s>{{패턴}}`

- 문서 종료:

`<q>`
