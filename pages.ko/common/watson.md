# watson

> 멋진 CLI로 시간을 추적하세요.
> 더 많은 정보: <https://github.com/TailorDev/Watson>.

- 프로젝트에서 시간 모니터링 시작:

`watson start {{프로젝트}}`

- 태그를 사용하여 프로젝트에서 시간 모니터링 시작:

`watson start {{프로젝트}} +{{태그}}`

- 현재 프로젝트에 대한 시간 모니터링 중지:

`watson stop`

- 최신 작업 세션 표시:

`watson log`

- 가장 최근의 프레임 편집:

`watson edit`

- 가장 최근의 프레임 제거:

`watson remove`
