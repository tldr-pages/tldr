# hexdump

> 파일 내용을 16진수, 10진수, 8진수 또는 ASCII 형식으로 표시.
> 덤프 파일, 바이너리 데이터 또는 디버그 출력 검사에 유용
> 관련 항목: `hexyl`, `od`, `xxd`.
> 더 많은 정보: <https://manned.org/man/freebsd/hexdump.1>.

- 파일의 16진수 표현 출력하며, 중복된 줄은 `*`로 대체:

`hexdump {{경로/대상/파일}}`

- 입력 오프셋을 16진수로 표시하고 ASCII 표현을 두 개의 열로 출력:

`hexdump -C {{경로/대상/파일}}`

- 입력의 지정한 바이트 수만 해석하며, 파일의 16진수 표현을 출력:

`hexdump -C -n {{바이트_수}} {{경로/대상/파일}}`

- `*`을 사용한 중복 줄 생략 비활성화하며 자세한 출력을 수행:

`hexdump -v {{경로/대상/파일}}`

- printf 스타일 포맷 문자열을 사용해 출력 형식을 지정:

`hexdump -e '{{요소_형식 .. 출력_형식}}' {{경로/대상/파일}}`
