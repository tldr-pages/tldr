# tlmgr shell

> 네이티브 TeX Live 관리자의 대화형 셸 시작.
> 더 많은 정보: <https://www.tug.org/texlive/doc/tlmgr.html#shell>.

- `tlmgr`의 대화형 셸 시작:

`tlmgr shell`

- 대화형 셸에서 `tlmgr` 하위 명령 실행:

`{{하위_명령}} {{인수}}`

- 대화형 셸 종료:

`quit`

- 모든 TeX Live 변수 나열:

`get`

- 특정 TeX Live 변수의 값 가져오기:

`get {{변수}}`

- 특정 TeX Live 변수의 값 설정:

`set {{변수}} {{값}}`

- 대화형 셸 재시작:

`restart`

- 현재 프로토콜의 버전 표시:

`protocol`
