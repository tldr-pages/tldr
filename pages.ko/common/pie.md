# pie

> PHP 확장 설치 도구.
> 더 많은 정보: <https://github.com/php/pie/blob/1.4.x/docs/usage.md>.

- 확장 설치 또는 업데이트:

`pie install {{벤더}}/{{확장}}`

- 설치된 확장과 버전 목록 표시:

`pie show`

- 특정 패키지 정보 표시:

`pie info {{벤더}}/{{확장}}`

- 설정된 저장소 목록 표시:

`pie repository:list`

- 저장소 추가:

`pie repository:add {{타입}} {{주소}}`

- 저장소 제거:

`pie repository:remove {{주소}}`
