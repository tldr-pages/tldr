# rabbitmq-plugins

> RabbitMQ 플러그인 관리.
> 더 많은 정보: <https://www.rabbitmq.com/docs/man/rabbitmq-plugins.8>.

- 플러그인 목록 표시:

`rabbitmq-plugins list`

- 플러그인 활성화:

`rabbitmq-plugins enable {{플러그인}}`

- 플러그인 비활성화:

`rabbitmq-plugins disable {{플러그인}}`

- 지정한 플러그인만 활성화하고 나머지 플러그인은 모두 비활성화:

`rabbitmq-plugins set {{플러그인}}`
