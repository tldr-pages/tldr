# pg_controldata

> PostgreSQL 데이터베이스 클러스터의 제어 정보를 표시.
> 더 많은 정보: <https://www.postgresql.org/docs/current/app-pgcontroldata.html>.

- 지정한 데이터 디렉터리의 제어 정보 표시:

`pg_controldata {{[-D|--pgdata]}} {{경로/대상/데이터}}`

- 도움말 표시:

`pg_controldata {{[-?|--help]}}`

- 버전 표시:

`pg_controldata {{[-V|--version]}}`
