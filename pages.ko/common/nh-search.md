# nh search

> Nixpkgs 패키지 검색.
> 더 많은 정보: <https://github.com/nix-community/nh#usage>.

- 결과를 제한하여 Nixpkgs 패키지 검색:

`nh search {{[-l|--limit]}} {{숫자}} {{이름}}`

- 지정한 Nixpkgs 채널에서 패키지 검색:

`nh search {{[-c|--channel]}} {{nixos-unstable}} {{이름}}`

- 각 패키지와 함께 지원 플랫폼 정보를 표시하며, Nixpkgs 패키지 검색:

`nh search {{[-P|--platforms]}} {{이름}}`
