# bun create

> 템플릿을 사용해 새로운 프로젝트를 생성.
> 참고: `c`는 `create`의 별칭으로 사용할 수 있습니다.
> 더 많은 정보: <https://bun.com/docs/runtime/templating/create>.

- 공식 템플릿을 사용해 대화형으로 새 프로젝트를 생성:

`bun create {{템플릿}}`

- 공식 템플릿을 사용해 새로운 디렉터리에 새 프로젝트를 생성:

`bun create {{템플릿}} {{경로/대상/목적지}}`

- GitHub 레포지토리 템플릿을 사용해 새로운 프로젝트 생성:

`bun create {{https://github.com/사용자이름/레포지토리}} {{경로/대상/목적지}}`

- 로컬 템플릿을 사용해 새로운 프로젝트 생성:

`bun create {{경로/대상/템플릿}} {{경로/대상/목적지}}`

- 기존 디렉터리가 있을 경우, 덮어쓰고 프로젝트를 생성:

`bun create {{템플릿}} {{경로/대상/목적지}} --force`

- Git 저장소를 자동으로 초기화하지 않고, 새로운 프로젝트를 생성:

`bun create {{템플릿}} {{경로/대상/목적지}} --no-git`

- 의존성을 자동으로 설치하지 않고 새로운 프로젝트를 생성:

`bun create {{템플릿}} {{경로/대상/목적지}} --no-install`
