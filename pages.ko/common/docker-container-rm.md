# docker rm

> 컨테이너 제거.
> 더 많은 정보: <https://docs.docker.com/reference/cli/docker/container/rm/>.

- 컨테이너 제거:

`docker rm {{컨테이너1 컨테이너2 ...}}`

- 컨테이너 강제 제거:

`docker rm {{[-f|--force]}} {{컨테이너1 컨테이너2 ...}}`

- 컨테이너와 그 볼륨 제거:

`docker rm {{[-v|--volumes]}} {{컨테이너}}`

- 도움말 표시:

`docker rm {{[-h|--help]}}`
