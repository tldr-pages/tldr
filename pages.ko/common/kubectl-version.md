# kubectl version

> 현재 컨텍스트의 kubectl 클라이언트 및 Kubernetes 서버 버전 정보 출력.
> 더 많은 정보: <https://kubernetes.io/docs/reference/kubectl/generated/kubectl_version/>.

- 현재 컨텍스트의 클라이언트 및 서버 버전 정보 출력:

`kubectl version`

- 현재 컨텍스트의 클라이언트 버전만 출력:

`kubectl version --client`

- 현재 컨텍스트의 클라이언트 및 서버 버전 정보를 JSON 형식으로 출력:

`kubectl version {{[-o|--output]}} json`

- 현재 컨텍스트의 클라이언트 및 서버 버전 정보를 YAML 형식으로 출력:

`kubectl version {{[-o|--output]}} yaml`
