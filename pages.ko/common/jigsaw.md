# jigsaw

> Laravel 기반의 PHP 정적 사이트 생성기.
> 더 많은 정보: <https://jigsaw.tighten.co>.

- 프로젝트 초기화:

`jigsaw init`

- 시작 템플릿을 사용하여 프로젝트 초기화:

`jigsaw init {{템플릿_이름}}`

- 개발용 사이트 빌드:

`jigsaw build`

- "build_local" 디렉토리에서 사이트 미리 보기:

`jigsaw serve`

- 프로덕션용 사이트 빌드:

`jigsaw build production`

- "build_production" 디렉토리에서 사이트 미리 보기:

`jigsaw serve {{build_production}}`
