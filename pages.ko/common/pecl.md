# pecl

> PHP Extension Community Library의 PHP 확장 관리.
> 더 많은 정보: <https://manned.org/pecl>.

- 사용 가능한 명령 목록 표시:

`pecl`

- 패키지 설치:

`pecl install {{패키지}}`

- 설치된 패키지 목록 표시:

`pecl list`

- 모든 패키지 업그레이드:

`pecl upgrade`

- 특정 패키지 업그레이드:

`pecl upgrade {{패키지}}`

- 특정 패키지 제거:

`pecl uninstall {{패키지}}`

- 특정 패키지 정보 표시:

`pecl info {{패키지}}`

- 사용 가능한 패키지 소스 업데이트:

`pecl update-channels`
