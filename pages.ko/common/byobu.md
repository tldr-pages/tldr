# byobu

> 윈도우 관리자이자 터미널 멀티플렉서.
> 관련 항목: `tmux`, `screen`.
> 더 많은 정보: <https://www.byobu.org/documentation>.

- 새로운 세션 시작:

`byobu`

- 설정 및 도움말:

`byobu-config`

- tmux(기본값) 또는 screen 백엔드 선택:

`byobu-select-backend`

- 텍스트 콘솔 로그인 시 자동 실행 활성화:

`byobu-enable`

- 텍스트 콘솔 로그인 시 자동 실행 비활성화:

`byobu-disable`

- `byobu` 세션에서 분리:

`<F6>`

- 현재 창 종료:

`<Ctrl a><k>`
