# ci

> RCS 리비전에 체크인 (리비전 버전 관리(Revision Control System)에 파일 변경사항을 저장).
> 관련 항목: `co`, `rcs`, `rcsdiff`, `rlog`.
> 더 많은 정보: <https://manned.org/ci>.

- 파일을 체크인하고 작업 파일을 잠금 해제 상태로 유지:

`ci -u {{경로/대상/파일}}`

- 파일을 체크인하고 작업 파일을 잠금 상태로 유지:

`ci -l {{경로/대상/파일}}`

- 특정 로그 메시지와 함께 파일을 체크인:

`ci -m"{{로그_메시지}}" {{경로/대상/파일}}`

- 파일을 체크인하고 잠금을 해제하지만, 작업 파일을 읽기 전용으로 유지:

`ci {{경로/대상/파일}}`

- 변경 사항이 없더라도 강제로 체크인:

`ci -f -u {{경로/대상/파일}}`
