# opencode upgrade

> OpenCode를 최신 버전 또는 특정 버전으로 업그레이드 하는 명령어.
> 더 많은 정보: <https://opencode.ai/docs/cli#upgrade>.

- 최신 버전으로 업그레이드:

`opencode upgrade`

- 특정 버전으로 업그레이드/다운그레이드:

`opencode upgrade {{1.1.10}}`

- 특정 설치 방법을 사용하여 업그레이드:

`opencode upgrade {{[-m|--method]}} {{curl|npm|pnpm|bun|brew|choco|scoop}}`

- 도움말 표시:

`opencode upgrade {{[-h|--help]}}`
