# gh pr view

> GitHub Pull Request 상세 정보 보기.
> 더 많은 정보: <https://cli.github.com/manual/gh_pr_view>.

- 현재 브랜치와 연결된 Pull Request 보기:

`gh pr view`

- 특정 Pull Request 보기:

`gh pr view {{123}}`

- 기본 웹 브라우저에서 Pull Request 열기:

`gh pr view {{[-w|--web]}}`

- 터미널에서 댓글 표시:

`gh pr view {{123}} {{[-c|--comments]}}`
