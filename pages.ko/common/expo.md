# expo

> React Native 애플리케이션을 빌드, 개발, 배포하는 도구.
> 더 많은 정보: <https://docs.expo.dev/eas/>.

- Android APK 또는 AAB 빌드:

`expo build:android`

- iOS IPA 빌드:

`expo build:ios`

- Expo Go에서 애플리케이션 실행:

`expo start`

- 의존성 설치:

`expo install {{패키지_이름}}`

- 네이티브 Android/iOS 프로젝트 사전 생성:

`expo prebuild`

- Android에서 애플리케이션 실행z:

`expo run:android`

- iOS에서 애플리케이션 실행:

`expo run:ios`
