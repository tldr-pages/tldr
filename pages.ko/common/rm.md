# rm

> 파일 혹은 폴더를 삭제.
> 참고: `rmdir`.
> 더 많은 정보: <https://www.gnu.org/software/coreutils/rm>.

- 특정 파일 삭제:

`rm {{경로/대상/파일1 경로/대상/파일2 ...}}`

- 존재하지 않는 파일은 무시하고 특정 파일 삭제:

`rm -f {{경로/대상/파일1 경로/대상/파일2 ...}}`

- 삭제하기 전에 각각의 파일 삭제 여부를 확인:

`rm -i {{경로/대상/파일1 경로/대상/파일2 ...}}`

- 각 파일 삭제 시 삭제 정보 출력:

`rm -v {{경로/대상/파일1 경로/대상/파일2 ...}}`

- 특정 파일 및 디렉터리를 재귀적으로 삭제:

`rm -r {{경로/대상/파일_혹은_디렉토리1 파일_혹은_디렉토리2 ...}}`
