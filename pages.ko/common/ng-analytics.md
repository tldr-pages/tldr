# ng analytics

> Angular CLI 분석 설정 관리.
> 더 많은 정보: <https://angular.dev/cli/analytics>.

- 현재 프로젝트에서 분석 기능 활성화:

`ng analytics {{[on|enable]}}`

- 전역 분석 기능 활성화:

`ng analytics {{[on|enable]}} {{[-g|--global]}}`

- 현재 프로젝트에서 분석 기능 비활성화:

`ng analytics {{[off|disable]}}`

- 전역 분석 기능 비활성화:

`ng analytics {{[off|disable]}} {{[-g|--global]}}`

- 현재 분석 설정 및 상태 확인:

`ng analytics info`

- 대화형으로 분석 설정을 구성:

`ng analytics prompt`
