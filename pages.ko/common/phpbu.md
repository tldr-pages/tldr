# phpbu

> PHP 백업 유틸리티 프레임워크.
> 더 많은 정보: <https://phpbu.de/manual/current/en/phpbu-manual.html#cli>.

- 기본 `phpbu.xml` 구성 파일을 사용하여 백업 실행:

`phpbu`

- 특정 구성 파일을 사용하여 백업 실행:

`phpbu --configuration={{경로/대상/구성_파일.xml}}`

- 지정된 백업만 실행:

`phpbu --limit={{백업_작업_이름}}`

- 수행될 작업을 시뮬레이션:

`phpbu --simulate`
