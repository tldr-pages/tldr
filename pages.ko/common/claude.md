# claude

> 코드베이스를 이해하고 자연어 명령을 통해 더 빠르게 코딩할 수 있도록 도와주는 에이전트 기반 코딩 도구.
> 더 많은 정보: <https://code.claude.com/docs/en/cli-reference>.

- 프롬프트로 실행:

`claude prompt`

- `claude`를 업데이트:

`claude update`

- 지정된 MCP 서버 목록을 표시:

`claude mcp list`

- 명령으로 커밋을 생성:

`claude commit`

- 설정 목록을 표시:

`claude config list`

- 가장 최근 세션을 계속 진행:

`claude {{[-c|--continue]}}`

- 계속 진행할 세션을 대화형으로 선택:

`claude {{[-r|--resume]}}`
