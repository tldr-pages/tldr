# glab auth

> GitLab 호스트로 인증.
> 더 많은 정보: <https://gitlab.com/gitlab-org/cli/-/blob/main/docs/source/auth/index.md>.

- 대화형 프롬프트로 로그인:

`glab auth login`

- 토큰으로 로그인:

`glab auth login {{[-t|--token]}} {{토큰}}`

- 인증 상태 확인:

`glab auth status`

- 특정 GitLab 인스턴스에 로그인:

`glab auth login {{[-h|--hostname]}} {{gitlab.example.com}}`
