# isutf8

> 텍스트 파일이 유효한 UTF-8 인코딩인지 확인하는 도구.
> 더 많은 정보: <https://manned.org/isutf8>.

- 지정한 파일들이 UTF-8인지 검사:

`isutf8 {{경로/대상/파일1 경로/대상/파일2 ...}}`

- 오류를 여러 줄로 상세 출력:

`isutf8 {{[-v|--verbose]}} {{경로/대상/파일1 경로/대상/파일2 ...}}`

- `stdout`로 출력 없이, 종료 코드로만 결과 반환:

`isutf8 {{[-q|--quiet]}} {{경로/대상/파일1 경로/대상/파일2 ...}}`

- UTF-8이 아닌 파일 이름만 출력:

`isutf8 {{[-l|--list]}} {{경로/대상/파일1 경로/대상/파일2 ...}}`

- `--list`와 동일하지만 결과를 반대로 출력, 즉, 유효한 UTF-8을 포함한 파일 이름만 출력:

`isutf8 {{[-i|--invert]}} {{경로/대상/파일1 경로/대상/파일2 ...}}`
