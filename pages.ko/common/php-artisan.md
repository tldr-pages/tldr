# php artisan

> Laravel의 Artisan 명령줄 인터페이스.
> 더 많은 정보: <https://laravel.com/docs/artisan>.

- 현재 Laravel 애플리케이션을 위한 PHP 내장 웹 서버 시작:

`php artisan serve`

- 대화형 PHP 명령줄 인터페이스 시작:

`php artisan tinker`

- 새 Eloquent 모델 클래스와 마이그레이션, 팩토리 및 리소스 컨트롤러 생성:

`php artisan make:model {{모델명}} --all`

- 사용 가능한 모든 명령 목록 표시:

`php artisan help`
