# pickle

> Composer 기반의 PHP 확장 프로그램 설치 도구.
> 더 많은 정보: <https://github.com/FriendsOfPHP/pickle>.

- 특정 PHP 확장 프로그램 설치:

`pickle install {{확장_이름}}`

- 기존 PECL 확장 구성 파일을 Pickle 구성 파일로 변환:

`pickle convert {{경로/대상/폴더}}`

- PECL 확장 유효성 검사:

`pickle validate {{경로/대상/폴더}}`

- PECL 확장을 릴리스용으로 패키징:

`pickle release {{경로/대상/폴더}}`
