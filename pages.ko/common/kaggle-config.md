# kaggle config

> Kaggle 설정 관리.
> 더 많은 정보: <https://github.com/Kaggle/kaggle-api/blob/main/docs/README.md#config>.

- 현재 설정 값 표시:

`kaggle config view`

- 설정 값 지정:

`kaggle config set {{[-n|--name]}} {{설정_매개변수}} {{[-v|--value]}} {{매개변수_값}}`

- 설정 값 제거:

`kaggle config unset {{[-n|--name]}} {{설정_매개변수}}`
