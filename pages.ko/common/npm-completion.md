# npm completion

> npm 명령어의 탭 자동완성을 활성화하거나 설정하는 도구.
> 더 많은 정보: <https://docs.npmjs.com/cli/npm-completion/>.

- 현재 쉘 세션에서 탭 자동완성 활성화:

`npm completion`

- Bash 프로파일에 자동완성 추가:

`npm completion >> ~/.bashrc`

- Zsh 프로파일에 자동완성 추가:

`npm completion >> ~/.zshrc`

- 자동완성 스크립트를 특정 파일로 저장:

`npm completion > {{경로/대상/파일}}`
