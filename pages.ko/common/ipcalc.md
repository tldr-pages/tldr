# ipcalc

> IP 주소와 넷마스크를 기반으로 IP 정보(서브넷, 브로드캐스트, 호스트 범위 등)를 계산하는 도구.
> 더 많은 정보: <https://manned.org/ipcalc>.

- IP 주소의 네트워크 정보 출력:

`ipcalc {{192.168.0.1}}`

- CIDR 표기법으로 네트워크 정보 출력:

`ipcalc {{192.168.0.1}}/{{24}}`

- 점 표시 넷마스크로 네트워크 정보 출력:

`ipcalc {{192.168.0.1}} {{255.255.255.0}}`

- 비트 출력 숨김:

`ipcalc {{[-b|--nobinary]}} {{192.168.0.1}}`

- 네트워크를 지정한 크기의 블록으로 분할:

`ipcalc {{[-s|--split]}} {{크기1 크기2 크기3 ...}} {{192.168.0.1}}`

- 버전 정보 출력:

`ipcalc {{[-v|--version]}}`
