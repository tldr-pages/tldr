# terragrunt

> Terraform CLI 인수를 DRY하게 유지.
> 더 많은 정보: <https://terragrunt.gruntwork.io>.

- 실행 계획 생성 및 표시:

`terragrunt plan`

- 인프라 생성 또는 변경:

`terragrunt apply`

- 현재 배포 상태 표시:

`terragrunt show`

- 모듈 출력 값 표시:

`terragrunt output`

- Terraform이 관리하는 인프라 파괴:

`terragrunt destroy`

- Terragrunt 모듈 트리(스택)에서 인프라 생성 또는 변경:

`terragrunt run-all apply`
