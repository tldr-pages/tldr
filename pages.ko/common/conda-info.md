# conda info

> conda 설치 상세 정보 표시.
> 더 많은 정보: <https://docs.conda.io/projects/conda/en/latest/commands/info.html>.

- 모든 정보 표시:

`conda info {{[-a|--all]}}`

- base 환경 경로 표시:

`conda info --base`

- 모든 conda 환경 목록 표시:

`conda info {{[-e|--envs]}}`

- 환경 변수 목록 표시:

`conda info {{[-s|--system]}}`

- 토큰이 노출된 채널 목록을 표시:

`conda info --unsafe-channels`
