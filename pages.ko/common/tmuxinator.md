# tmuxinator

> tmux 세션을 쉽게 생성하고 관리.
> 더 많은 정보: <https://github.com/tmuxinator/tmuxinator>.

- 새 프로젝트 생성:

`tmuxinator new {{프로젝트}}`

- 프로젝트 편집:

`tmuxinator edit {{프로젝트}}`

- 프로젝트 목록 나열:

`tmuxinator list`

- 프로젝트를 기반으로 tmux 세션 시작:

`tmuxinator start {{프로젝트}}`

- 프로젝트의 tmux 세션 중지:

`tmuxinator stop {{프로젝트}}`
