# dbeaver

> 여러 데이터베이스를 지원하는 GUI SQL 클라이언트.
> 더 많은 정보: <https://dbeaver.com/docs/dbeaver/Command-Line/>.

- DBeaver를 실행:

`dbeaver`

- 특정 데이터베이스에 연결하여 DBeaver를 실행:

`dbeaver {{[-con|--connect]}} {{데이터베이스}}`

- 새 인스턴스를 강제로 생성:

`dbeaver --newInstance`

- 실행 중인 인스턴스를 종료:

`dbeaver --quit`

- DBeaver에서 열려 있는 모든 탭을 닫음:

`dbeaver --closeTabs`

- DBeaver 창을 다른 애플리케이션 위로 가져옴:

`dbeaver --bringToFront`

- 도움말 표시:

`dbeaver --help`
