# lazydocker

> Docker 컨테이너, 이미지, 볼륨 등을 관리하는 터미널 UI.
> 더 많은 정보: <https://github.com/jesseduffield/lazydocker>.

- Lazydocker 열기:

`lazydocker`

- 현재 기본 설정 표시:

`lazydocker --config`

- 디버그 모드로 Lazydocker 실행:

`lazydocker --debug`

- 다른 Docker Compose 파일을 지정하여 실행:

`lazydocker --file {{경로/대상/docker-compose.yml}}`

- 도움말 표시:

`lazydocker --help`

- 버전 정보 표시:

`lazydocker --version`
