# opencode debug

> OpenCode의 디버깅 및 문제 해결 도구.
> 더 많은 정보: <https://opencode.ai/docs/cli#debug>.

- 적용된 설정 값 출략:

`opencode debug config`

- 전역 경로(데이터, 설정, 캐시, 상태) 출력:

`opencode debug paths`

- 모든 알려진 프로젝트 목록 출력:

`opencode debug scrap`

- 사용 가능한 모든 스킬 목록 출력:

`opencode debug skill`

- 특정 에이전트 설정 상세 정보 출력:

`opencode debug agent {{에이전트_이름}}`

- 도움말 표시:

`opencode debug {{[-h|--help]}}`
