# mp4decrypt

> MP4 파일 복호화.
> Bento4 도구의 일부.
> 더 많은 정보: <https://www.bento4.com/documentation/mp4decrypt/>.

- 특정 키를 사용하여 파일 복호화 (키 ID는 16진수, 키 값도 16진수):

`mp4decrypt --key {{16진수_키_아이디}}:{{16진수_키_값}} {{경로/대상/입력_파일.mp4}} {{경로/대상/출력_파일.mp4}}`

- 특정 트랙 ID에 대한 키를 사용하여 파일 복호화 (트랙 ID는 10진수, 키 값은 16진수):

`mp4decrypt --key {{트랙_아이디}}:{{16진수_키_값}} {{경로/대상/입력_파일.mp4}} {{경로/대상/출력_파일.mp4}}`

- 여러 개의 키를 사용하여 파일을 복호화하고 진행 사황을 표시:

`mp4decrypt --key {{키_아이디_1}}:{{키_1}} --key {{키_아이디_2}}:{{키_2}} --show-progress {{경로/대상/입력_파일.mp4}} {{경로/대상/출력_파일.mp4}}`
