# ng generate

> Angular Schematic을 기반으로 파일 생성 또는 수정.
> 더 많은 정보: <https://angular.dev/cli/generate>.

- Angular 프로젝트용 AI 설정 파일 생성:

`ng {{[g|generate]}} ai-config`

- 새로운 서비스 생성:

`ng {{[g|generate]}} {{[s|service]}} {{서비스_이름}}`

- 워크스페이스 내에 새로운 애플리케이션 생서:

`ng {{[g|generate]}} {{[app|application]}} {{애플리케이션_이름}}`

- 새로운 클래스 생성:

`ng {{[g|generate]}} {{[cl|class]}} {{클래스_이름}}`

- 새로운 컴포넌트 생성:

`ng {{[g|generate]}} {{[c|component]}} {{컴포넌트_이름}}`

- 새로운 디렉티브 생성:

`ng {{[g|generate]}} {{[d|directive]}} {{디렉티브_이름}}`

- 새로운 enum 생성:

`ng {{[g|generate]}} {{[e|enum]}} {{enum_이름}}`

- 새로운 파이프 생성:

`ng {{[g|generate]}} {{[p|pipe]}} {{파이프_이름}}`
