# gron

> `JSON` 데이터를 개별 할당문 형태로 변환해 관리하기 쉽게 만드는 도구.
> 더 많은 정보: <https://manned.org/gron>.

- `JSON` 파일을 개별 할당문 형태로 처리:

`gron {{경로/대상/파일|주소}}`

- 출력 데이터를 정렬하지 않음:

`gron --no-sort {{경로/대상/파일|주소}}`

- 인증서 검증 비활성화:

`gron {{[-k|--insecure]}} {{url}}`

- `gron` 할당문의 값을 표시:

`gron {{[-v|--values]}} {{경로/대상/파일|주소}}`

- `gron`으로 변환된 할당문을 다시 `JSON`으로 복원:

`gron {{[-u|--ungron]}} {{경로/대상/파일|주소}}`

- 입력의 각 줄을 개별 `JSON` 객체로 처리:

`gron {{[-s|--stream]}} {{경로/대상/파일|주소}}`

- 처리된 데이터를 `JSON` 스트림 형태로 표현:

`gron {{[-j|--json]}} {{경로/대상/파일|주소}}`
