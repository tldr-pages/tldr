# bun add

> 현대 자바스크립트 런타임, 패키지 매니저, 번들러 및 테스트 실행기.
> 참고: `a`는 `add`의 별칭으로 사용할 수 있다.
> 더 많은 정보: <https://bun.com/docs>.

- 단일 패키지 설치:

`bun add {{패키지}}`

- 여러 패키지들을 설치:

`bun add {{패키지1 패키지2 ...}}`

- Git 레포지토리로부터 패키지 설치:

`bun add {{git_레포지토리_주소}}`

- 특정 버전의 패키지 설치:

`bun add {{패키지}}@{{버전}}`

- 로컬 파일 또는 디렉터리를 통해 설치:

`bun add file:{{경로/대상/파일_또는_디렉터리}}`

- 개발 의존성으로 패키지 추가:

`bun add {{[-d|--dev]}} {{패키지}}`

- 전역으로 패키지 추가:

`bun add {{[-g|--global]}} {{패키지}}`
