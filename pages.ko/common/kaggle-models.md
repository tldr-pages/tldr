# kaggle models

> Kaggle 모델 관리.
> 더 많은 정보: <https://github.com/Kaggle/kaggle-api/blob/main/docs/README.md#models>.

- 모델 가져오기:

`kaggle {{[m|models]}} get "{{모델_이름}}"`

- 모든 모델 목록 나열:

`kaggle {{[m|models]}} list`

- 모델 생성 메타데이터 파일 초기화:

`kaggle {{[m|models]}} init`

- 새로운 모델 생성:

`kaggle {{[m|models]}} create`

- 모델 삭제:

`kaggle {{[m|models]}} delete "{{모델_이름}}"`

- 모델 업데이트:

`kaggle {{[m|models]}} update`
