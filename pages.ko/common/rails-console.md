# rails console

> Rails 애플리케이션과 대화형으로 상호작용.
> 더 많은 정보: <https://guides.rubyonrails.org/command_line.html#bin-rails-console>.

- Rails 콘솔 시작:

`rails console`

- 종료 시 모든 데이터 변경 사항을 되돌리는 샌드박스 모드로 Rails 콘솔 시작:

`rails console {{[-s|--sandbox]}}`

- 지정한 환경에서 Rails 콘솔 시작:

`rails console {{[-e|--environment]}} {{dev|test|production|...}}`

- 도움말 표시:

`rails console {{[-h|--help]}}`
