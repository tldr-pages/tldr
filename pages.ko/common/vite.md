# Vite

> Vite 프로젝트를 생성.
> JavaScript 프로젝트들을 빌드하는 데에 사용됨.
> 사용 가능 템플릿들: vanilla, vanilla-ts, vue, vue-ts, react, react-ts, react-swc, react-swc-ts, preact, preact-ts, lit, lit-ts, svelte, svelte-ts.
> 더 많은 정보: <https://vitejs.dev/guide>.

- `npm` 6.x 버전을 사용하여 설정:

`npm create vite@latest 나의-리액트-앱 --template react-ts`

- `npm` 7+ 버전을 사용하여 설정, 추가로 이중 대시 필요:

`npm create vite@latest 나의-리액트-앱 -- --template react-ts`

- `yarn`을 사용하여 설정:

`yarn create vite 나의-리액트-앱 --template react-ts`

- `pnpm`을 사용하여 설정:

`pnpm create vite 나의-리액트-앱 --template react-ts`
