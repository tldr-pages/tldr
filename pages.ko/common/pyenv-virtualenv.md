# pyenv virtualenv

> 설치된 Python 배포판을 기반으로 가상 환경을 생성합니다.
> 더 많은 정보: <https://github.com/pyenv/pyenv-virtualenv>.

- 새로운 Python 3.6.6 가상 환경 생성:

`pyenv virtualenv {{3.6.6}} {{가상환경_이름}}`

- 모든 기존 가상 환경 나열:

`pyenv virtualenvs`

- 가상 환경 활성화:

`pyenv activate {{가상환경_이름}}`

- 가상 환경 비활성화:

`pyenv deactivate`
