# biome

> 웹 프로젝트의 코드 포맷팅, 린트 및 코드 품질 검사를 위한 툴체인.
> 더 많은 정보: <https://biomejs.dev/reference/cli/>.

- 현재 디렉토리에 새로운 Biome 설정 파일 초기화:

`biome init`

- 파일을 포맷하고 원본 파일에 덮어쓰기:

`biome format --write {{경로/대상/파일_또는_디렉터리}}`

- 변경 사항을 적용하지 않고 포맷 상태 검사:

`biome format {{경로/대상/파일_또는_디렉터리}}`

- 파일을 린트하고 안전한 수정 사항 적용:

`biome lint --write {{경로/대상/파일_또는_디렉터리}}`

- 포맷, 린트 및 import 정렬을 실행하고 안전한 수정 사항 적용:

`biome check --write {{경로/대상/파일_또는_디렉터리}}`

- CI 모드애서 모든 검사 실행 (수정 없이 수행하며, 문제 발견 시 오류 코드 반환):

`biome ci {{경로/대상/파일_또는_디렉터리}}`

- 특정 린트 규칙 설명 표시:

`biome explain {{규칙_이름}}`

- 버전 표시:

`biome {{[-V|--version]}}`
