# git add

> 변경된 파일들을 인덱스에 추가합니다.
> 더 많은 정보: <https://git-scm.com/docs/git-add>.

- 인덱스에 파일 추가:

`git add {{파일/의/경로}}`

- 모든 파일 추가 (추적된 파일과 추적되지 않은 파일 모두):

`git add -A`

- 이미 추적된 파일만 추가:

`git add -u`

- 무시되는 파일 추가:

`git add -f`

- 파일의 일부분을 대화식으로 추가:

`git add -p`

- 주어진 파일의 일부분을 대화식으로 추가:

`git add -p {{파일/의/경로}}`

- 대화식으로 파일을 추가:

`git add -i`
