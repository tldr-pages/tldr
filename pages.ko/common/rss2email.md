# rss2email

> RSS 피드를 이메일 프로그램으로 전달.
> 더 많은 정보: <https://github.com/rss2email/rss2email>.

- 모든 피드 나열:

`r2e list`

- RSS 항목을 이메일로 변환:

`r2e run`

- 피드 추가:

`r2e add {{피드_주소}}`

- 특정 이메일 주소로 피드 추가:

`r2e add {{피드_주소}} {{새_이메일@example.com}}`

- 특정 피드 삭제:

`r2e delete {{목록에서의_피드_번호}}`

- 도움말 표시:

`r2e -h`
