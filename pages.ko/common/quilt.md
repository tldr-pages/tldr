# quilt

> 일련의 패치를 관리.
> 더 많은 정보: <https://savannah.nongnu.org/projects/quilt>.

- 기존 패치를 파일에서 가져오기:

`quilt import {{경로/대상/파일이름.patch}}`

- 새 패치 생성:

`quilt new {{파일이름.patch}}`

- 현재 패치에 파일 추가:

`quilt add {{경로/대상/파일}}`

- 파일을 편집한 후, 변경 사항으로 현재 패치 갱신:

`quilt refresh`

- 시리즈 파일의 모든 패치 적용:

`quilt push -a`

- 적용된 모든 패치 제거:

`quilt pop -a`
