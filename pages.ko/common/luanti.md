# luanti

> 무한 세계를 지원하는 블록 샌드박스 게임.
> 더 많은 정보: <https://manned.org/luanti>.

- 클라이언트 모드로 Luanti 실행:

`luanti`

- 다운로드된 게임 모드 목록 표시:

`luanti --gameid list`

- 특정 게임 모드를 호스팅하며 Luanti를 서버 모드로 실행:

`luanti --server --gameid {{게임_아이디}}`

- 기본 월드가 생성된 후 해당 월드로 서버 실행:

`luanti --server`

- 특정 월드로 서버 실행:

`luanti --server --world {{월드_이름}}`

- 특정 포트에서 서버 실행:

`luanti --server --port {{포트}}`

- 로그를 특정 파일에 기록:

`luanti --logfile {{경로/대상/파일}}`

- 오류 메시지만 콘솔에 출력:

`luanti --quiet`
