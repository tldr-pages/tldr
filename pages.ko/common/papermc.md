# papermc

> Minecraft 서버 관리.
> 더 많은 정보: <https://github.com/Edenhofer/minecraft-server/blob/main/minecraftd.sh.in#L480>.

- 서버 시작:

`papermc start`

- 서버 중지:

`papermc stop`

- 서버 재시작:

`papermc restart`

- 월드 백업:

`papermc backup`

- 백업에서 월드 데이터 복원:

`papermc restore {{경로/대상/파일}}`

- 도움말 표시:

`papermc`
