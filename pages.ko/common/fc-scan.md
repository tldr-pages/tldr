# fc-scan

> 글꼴 파일 또는 디렉터리를 검사해 발견된 각 글꼴의 패턴 정보를 출력.
> 더 많은 정보: <https://manned.org/fc-scan>.

- 글꼴 파일 검사:

`fc-scan {{경로/대상/글꼴_파일}}`

- 글꼴 디렉터리를 재귀적으로 검사:

`fc-scan {{경로/대상/디렉터리}}`

- 여러 글꼴 파일 또는 디렉터리 검사:

`fc-scan {{경로/대상/폰트1.ttf 경로/대상/폰트2.otf ...}}`

- 글꼴 크기 정보를 제외하고 결과 출력:

`fc-scan {{[-b|--brief]}} {{경로/대상/글꼴_파일}}`

- 사용자 지정 출력 형식으로 결과 출력:

`fc-scan {{[-f|--format]}} "{{%{family}\n}}" {{경로/대상/글꼴_파일}}`

- 도움말 표시:

`fc-scan {{[-h|--help]}}`

- 버전 정보 표시:

`fc-scan {{[-V|--version]}}`
