# rails server

> Rails에 기본 포함된 Puma 웹 서버를 사용하여 현재 디렉터리의 Rails 애플리케이션을 실행.
> 더 많은 정보: <https://guides.rubyonrails.org/command_line.html#bin-rails-server>.

- 웹 서버 실행:

`rails server`

- 지정한 포트에서 웹 서버 실행:

`rails server {{[-p|--port]}} {{포트_번호}}`

- 지정한 IP 주소에 바인딩하여 웹 서버 실행:

`rails server {{[-b|--binding]}} {{아이피_주소}}`

- 지정한 환경에서 웹 서버 실행:

`rails server {{[-e|--environment]}} {{환경}}`

- 도움말 표시:

`rails server {{[-h|--help]}}`
