# mcopy

> MSDOS 파일을 Unix와 상호 복사.
> `mtools` 패키지의 일부.
> 참고: DOS 하위 디렉터리는 백슬래시 (`\`) 대신 슬래시 (`/`) 사용.
> 더 많은 정보: <https://gnu.org/software/mtools/manual/mtools.html#mcopy>.

- Linux 파일을 MS-DOS 디스크 또는 디스크 이미지로 복사:

`mcopy {{경로/대상/소스_파일}} {{A}}:/{{경로/대상/대상_파일}}`

- MS-DOS 디스크의 파일을 현재 Linux 디렉터리로 복사:

`mcopy {{A}}:{{/경로/대상/소스_파일}} .`

- MS-DOS 드라이브의 모든 파일을 Linux 디렉터리로 복사:

`mcopy {{A}}: {{경로/대상/대상_디렉터리}}`

- 드라이브 대신 DOS 디스크 이미지([i]mage) 파일에 복사 (`::`는 DOS 디스크 이미지의 루트 디렉터리):

`mcopy -i {{경로/대상/이미지.img}} {{경로/대상/소스_파일}} ::/{{경로/대상/대상_파일}}`

- 디스크 이미지에서 현재 디렉터리로 파일 복사:

`mcopy -i {{경로/대상/이미지.img}} ::/{{경로/대상/소스_파일}} .`

- 디렉터리를 재귀적으로(recur[s]ively) MS-DOS 디스크로 복사:

`mcopy -s {{경로/대상/소스_디렉터리}} {{A}}:/{{경로/대상/대상_디렉터리}}`

- 파일 속성을 유지([p]reserve)하면서 복사:

`mcopy -p {{경로/대상/소스_파일}} {{A}}:/{{경로/대상/대상_파일}}`
