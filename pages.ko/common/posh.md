# posh

> Ruby 개발자를 위한 Gem 문서 브라우저.
> 더 많은 정보: <https://poshtui.com/docs>.

- 현재 프로젝트 문서를 로컬에 다운로드하고 (Ruby 프로젝트 안에서 실행) 전체 동기화 보고서 출력:

`posh sync --full`

- 현재 프로젝트의 대화형 문서 브라우저 실행:

`posh`

- 로컬의 모든 문서 세트 삭제:

`posh clear`

- 쉘용 자동 완성 스크립트 생성:

`posh completion {{bash|fish|powershell|zsh}}`

- 도움말 표시:

`posh {{[-h|--help]}}`

- 특정 하위 명령의 도움말 표시:

`posh {{하위_명령어}} {{[-h|--help]}}`
