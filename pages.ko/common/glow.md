# glow

> 터미널에서 마크다운을 렌더링.
> 더 많은 정보: <https://github.com/charmbracelet/glow>.

- glow 실행하고 보려는 파일을 선택:

`glow`

- Markdown 파일을 터미널에 렌더링:

`glow {{경로/대상/파일}}`

- 페이지네이터를 사용하여 Markdown 파일 보기:

`glow -p {{경로/대상/파일}}`

- URL에서 파일 보기:

`glow {{https://example.com/file.md}}`

- GitHub/GitLab README 보기:

`glow {{github.com/소유자/디렉토리}}`
