# opencode session

> OpenCode 세션 관리.
> 더 많은 정보: <https://opencode.ai/docs/cli#session>.

- 세션 목록 출력:

`opencode session list`

- 최근 N개의 세션을 지정한 형식으로 출력:

`opencode session list {{[-n|--max-count]}} {{10}} --format {{table|json}}`

- 특정 세션 삭제:

`opencode session delete {{세션_아이디}}`

- 도움말 표시:

`opencode session {{[-h|--help]}}`
