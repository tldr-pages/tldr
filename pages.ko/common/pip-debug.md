# pip debug

> 현재 pip 환경 및 설정에 대한 디버그 정보 표시.
> 디버깅 목적용이며, 옵션과 출력 형식은 예고 없이 변경될 수 있음.
> 더 많은 정보: <https://pip.pypa.io/en/stable/cli/pip_debug/>.

- 일반 디버그 정보 표시:

`pip debug`

- 특정 플랫폼에 대한 디버그 정보 표시:

`pip debug --platform {{플랫폼}}`

- 특정 Python 버전에 대한 디버그 정보 표시:

`pip debug --python-version {{버전}}`

- 특정 Python 구현체에 대한 디버그 정보 표시:

`pip debug --implementation {{구현체}}`

- 특정 ABI에 대한 디버그 정보 표시:

`pip debug --abi {{abi}}`
