# sails

> Node.js 위에 구축된 실시간 엔터프라이즈 레벨 MVC 프레임워크.
> 더 많은 정보: <https://sailsjs.com/documentation/reference/command-line-interface>.

- Sails 시작:

`sails lift`

- 새 Sails 프로젝트 생성:

`sails new {{프로젝트이름}}`

- Sails API 생성:

`sails generate {{이름}}`

- Sails 컨트롤러 생성:

`sails generate controller {{이름}}`

- Sails 모델 생성:

`sails generate model {{이름}}`
