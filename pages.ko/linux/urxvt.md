# urxvt

> Rxvt-unicode.
> 사용자 정의 가능한 터미널 에뮬레이터.
> 더 많은 정보: <https://manned.org/urxvt>.

- 새로운 urxvt 창 열기:

`urxvt`

- 특정 폴더에서 실행:

`urxvt -cd {{경로/대상/폴더}}`

- 새로운 urxvt 창에서 명령어 실행:

`urxvt -e {{명령어}}`

- 명령어 실행 후 창을 유지:

`urxvt --hold -e {{명령어}}`

- `sh` 셸 내에서 명령어 실행:

`urxvt -e {{sh}} -c {{명령어}}`
