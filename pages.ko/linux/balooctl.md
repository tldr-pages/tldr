# balooctl

> KDE Plasma의 파일 색인 및 검색 프레임워크.
> 더 많은 정보: <https://wiki.archlinux.org/index.php/Baloo>.

- 색인기의 상태 표시:

`balooctl status`

- 파일 색인기 활성화/비활성화:

`balooctl {{enable|disable}}`

- 색인 데이터베이스 정리:

`balooctl purge`

- 파일 색인기 일시 중지:

`balooctl suspend`

- 파일 색인기 재개:

`balooctl resume`

- Baloo가 사용하는 디스크 공간 표시:

`balooctl indexSize`

- 색인되지 않은 파일이 있는지 확인하고 색인:

`balooctl check`

- 도움말 표시:

`balooctl --help`
