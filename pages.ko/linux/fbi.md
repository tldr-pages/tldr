# fbi

> 터미널 프레임버퍼에 이미지를 표시.
> 더 많은 정보: <https://manned.org/fbi>.

- 이미지 표시:

`sudo fbi {{경로/대상/파일}}`

- 여러 이미지 표시:

`sudo fbi {{경로/대상/파일1 경로/대상/파일2 ...}}`

- 지정한 간격(초)으로 슬라이드 쇼 시작:

`sudo fbi {{[-t|--timeout]}} {{초}} {{경로/대상/파일/*}}`

- 화면 크기에 맞게 이미지 자동 확대/축소:

`sudo fbi {{[-a|--autozoom]}} {{경로/대상/파일}}`

- 도움말 표시:

`fbi {{[-h|--help]}}`
