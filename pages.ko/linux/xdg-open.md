# xdg-open

> 사용자 선호 응용 프로그램에서 파일이나 URL 열기.
> 더 많은 정보: <https://portland.freedesktop.org/doc/xdg-open.html>.

- 기본 파일 탐색기에서 현재 디렉터리 열기:

`xdg-open .`

- 기본 브라우저에서 URL 열기:

`xdg-open {{https://example.com}}`

- 기본 이미지 뷰어에서 이미지 열기:

`xdg-open {{경로/대상/이미지}}`

- 기본 PDF 뷰어에서 PDF 열기:

`xdg-open {{경로/대상/pdf}}`

- 도움말 표시:

`xdg-open --help`
