# protontricks

> Protonмқ„ м§Җмӣҗн•ҳлҠ” кІҢмһ„мқ„ мң„н•ҙ Winetricks лӘ…л №мқ„ мӢӨн–үн•ҳлҠ” к°„лӢЁн•ң лһҳнҚј.
> лҚ” л§ҺмқҖ м •ліҙ: <https://github.com/Matoking/protontricks#usage>.

- protontricks GUI мӢӨн–ү:

`protontricks --gui`

- нҠ№м • кІҢмһ„м—җ лҢҖн•ҙ Winetricks мӢӨн–ү:

`protontricks {{м•ұ_id}} {{winetricks_мқёмҲҳ}}`

- кІҢмһ„ м„Өм№ҳ л””л үнҶ лҰ¬ лӮҙм—җм„ң лӘ…л № мӢӨн–ү:

`protontricks -c {{лӘ…л №м–ҙ}} {{м•ұ_id}}`

- м„Өм№ҳлҗң лӘЁл“  кІҢмһ„ [l]лӮҳм—ҙ:

`protontricks -l`

- кІҢмһ„мқҳ мқҙлҰ„мңјлЎң м•ұ ID [s]кІҖмғү:

`protontricks -s {{кІҢмһ„_мқҙлҰ„}}`

- лҸ„мӣҖл§җ н‘ңмӢң:

`protontricks --help`
