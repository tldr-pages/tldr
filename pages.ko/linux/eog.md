# eog

> 이미지 보기 및 관리.
> 더 많은 정보: <https://manned.org/eog>.

- 이미지 파일 열기:

`eog {{경로/대상/이미지.png}}`

- 여러 이미지 열기:

`eog {{이미지1.jpg 이미지2.png ...}}`

- 전체 화면 모드로 이미지 열기:

`eog --fullscreen {{경로/대상/이미지.jpg}}`

- 슬라이드 쇼 시작:

`eog --slide-show {{경로/대상/디렉터리}}`

- 이미지 컬렉션을 사용하지 않고 단일 이미지 열기:

`eog --single-window {{경로/대상/이미지.jpg}}`

- 새 창에서 이미지 열기:

`eog --new-instance {{경로/대상/이미지.jpg}}`
