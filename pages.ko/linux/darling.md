# darling

> Linux에서 macOS 소프트웨어 실행.
> 더 많은 정보: <https://docs.darlinghq.org/darling-shell.html>.

- 내장 명령 실행:

`darling shell {{uname}}`

- Run a specific program with arguments:

`darling shell {{경로/대상/프로그램}} {{프로그램_인수_1 프로그램_인수_2 ...}}`

- macOS 쉘 실행:

`darling shell`

- 서비스 종료:

`darling shutdown`
