# sam

> AWS Serverless Application Model (SAM) CLI.
> 더 많은 정보: <https://github.com/awslabs/aws-sam-cli>.

- 서버리스 애플리케이션 초기화:

`sam init`

- 특정 런타임으로 서버리스 애플리케이션 초기화:

`sam init --runtime {{python3.7}}`

- SAM 애플리케이션 패키징:

`sam package`

- Lambda 함수 코드 빌드:

`sam build`

- 로컬에서 서버리스 애플리케이션 실행:

`sam local start-api`

- AWS SAM 애플리케이션 배포:

`sam deploy`
