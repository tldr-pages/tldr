# betterlockscreen

> 간단하고, 미니멀한 잠금 화면.
> 더 많은 정보: <https://github.com/betterlockscreen/betterlockscreen#usage>.

- 화면 잠금:

`betterlockscreen {{[-l|--lock]}}`

- 잠금 화면 배경 변경:

`betterlockscreen {{[-u|--update]}} {{경로/대상/이미지.png}}`

- 사용자 지정 텍스트를 표시하며 화면 잠금:

`betterlockscreen {{[-l|--lock]}} pixel {{[-t|--text]}} "{{사용자 지정 잠금 화면 텍스트}}"`

- 사용자 지정 모니터 꺼짐 시간(초)을 설정하여 화면 잠금:

`betterlockscreen --off {{5}} {{[-l|--lock]}}`
