# xed

> Cinnamon 데스크탑 환경에서 파일 편집.
> 더 많은 정보: <https://manned.org/xed>.

- 편집기 시작:

`xed`

- 특정 파일 열기:

`xed {{경로/대상/파일1 경로/대상/파일2 ...}}`

- 특정 인코딩을 사용하여 파일 열기:

`xed --encoding {{WINDOWS-1252}} {{경로/대상/파일1 경로/대상/파일2 ...}}`

- 지원되는 모든 인코딩 출력:

`xed --list-encodings`

- 특정 줄로 이동하여 파일 열기:

`xed +{{10}} {{경로/대상/파일}}`
