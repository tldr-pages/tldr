# ego

> Funtoo의 공식 시스템 성격 관리 도구.
> 더 많은 정보: <https://funtoo-ego.readthedocs.io/en/develop/>.

- Portage 트리 동기화:

`ego sync`

- 부트로더 구성 업데이트:

`ego boot update`

- 이름으로 Funtoo 위키 페이지 읽기:

`ego doc {{위키_페이지}}`

- 현재 프로필 출력:

`ego profile show`

- 믹스인 활성화/비활성화:

`ego profile mix-in +{{gnome}} -{{kde-plasma-5}}`

- 특정 패키지와 관련된 Funtoo 버그 쿼리:

`ego query bug {{패키지}}`
