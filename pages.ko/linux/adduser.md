# adduser

> 사용자 추가 도구.
> 더 많은 정보: <https://manned.org/adduser>.

- 기본 홈 디렉터리를 갖춘 새 사용자 생성 및 비밀번호 설정 안내:

`adduser {{사용자명}}`

- 홈 디렉터리 없이 새 사용자 생성:

`adduser --no-create-home {{사용자명}}`

- 지정된 경로에 홈 디렉터리를 갖춘 새 사용자 생성:

`adduser --home {{경로/대상/홈}} {{사용자명}}`

- 로그인 셸로 지정된 셸을 갖춘 새 사용자 생성:

`adduser --shell {{경로/대상/셸}} {{사용자명}}`

- 지정된 그룹에 속하는 새 사용자 생성:

`adduser --ingroup {{그룹}} {{사용자명}}`
