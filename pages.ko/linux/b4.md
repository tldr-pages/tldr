# b4

> public-inbox 아카이브에서 패치와 pull request를 관리하는 도구.
> `am`, `shazam`, `prep`, `send`와 같은 일부 하위 명령은 각각 별도의 사용 문서를 제공.
> 더 많은 정보: <https://manned.org/b4>.

- public-inbox 아카이브에서 스레드를 가져와 mbox 파일로 저장:

`b4 mbox {{메시지_아이디}}`

- 스레드에서 패치를 가져와 `git am`으로 적용할 수 있도록 준비:

`b4 am {{메시지_아이디}}`

- 패치를 가져와 현재 Git 작업 트리에 직접 적용:

`b4 shazam {{메시지_아이디}}`

- 지정한 기준점에서 분기된 새로운 패치 시리즈 브랜치 생성:

`b4 prep {{[-n|--new]}} {{시리즈_이름}} {{[-f|--fork-point]}} {{fork_포인트}}`

- 현재 브랜치의 패치를 메일링 리스트로 전송:

`b4 send`

- 현재 브랜치의 코드 리뷰(code-review) 트레일러 업데이트:

`b4 trailers {{[-u|--update]}}`

- 도움말 표시:

`b4 {{[-h|--help]}}`

- 버전 정보 표시:

`b4 --version`
