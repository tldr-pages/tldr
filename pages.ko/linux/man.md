# man

> 매뉴얼 페이지를 포맷하고 표시합니다.
> 더 많은 정보: <https://manned.org/man>.

- 명령어에 대한 매뉴얼 페이지 표시:

`man {{명령어}}`

- 브라우저에서 명령어에 대한 매뉴얼 페이지 열기 (`BROWSER` 변수가 설정되어 있어야 함):

`man --html {{명령어}}`

- 섹션 7에서 명령어에 대한 매뉴얼 페이지 표시:

`man {{7}} {{명령어}}`

- 명령어에 대한 사용 가능한 모든 섹션 나열:

`man --whatis {{명령어}}`

- 매뉴얼 페이지 검색 경로 표시:

`man --path`

- 매뉴얼 페이지 자체가 아닌 매뉴얼 페이지의 위치 표시:

`man --where {{명령어}}`

- 특정 로케일을 사용하여 매뉴얼 페이지 표시:

`man --locale {{로케일}} {{명령어}}`

- 검색 문자열을 포함하는 매뉴얼 페이지 검색:

`man --apropos "{{검색_문자열}}"`
