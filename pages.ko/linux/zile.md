# zile

> Emacs 텍스트 편집기의 경량 클론.
> 더 많은 정보: <https://www.gnu.org/software/zile/>.

- 임시 메모를 위한 버퍼 시작 (저장되지 않음):

`zile`

- 파일 열기:

`zile {{경로/대상/파일}}`

- 파일 저장:

`<Ctrl> + X, <Ctrl> + S`

- 종료:

`<Ctrl> + X, <Ctrl> + C`

- 지정된 줄 번호에서 파일 열기:

`zile +{{줄_번호}} {{경로/대상/파일}}`

- 변경 사항 실행 취소:

`<Ctrl> + X, U`
