# pluma

> MATE 데스크탑 환경에서 파일 편집.
> 더 많은 정보: <https://manned.org/pluma>.

- 편집기 시작:

`pluma`

- 특정 문서 열기:

`pluma {{경로/대상/파일1 경로/대상/파일2 ...}}`

- 특정 인코딩을 사용하여 문서 열기:

`pluma --encoding {{WINDOWS-1252}} {{경로/대상/파일1 경로/대상/파일2 ...}}`

- 지원되는 모든 인코딩 출력:

`pluma --list-encodings`

- 특정 줄로 이동하여 문서 열기:

`pluma +{{10}} {{경로/대상/파일}}`
