# port

> macOS 패키지 관리자.
> 더 많은 정보: <https://guide.macports.org/#using.port>.

- 패키지 검색:

`port search {{검색_어}}`

- 패키지 설치:

`sudo port install {{패키지}}`

- 설치된 패키지 나열:

`port installed`

- 포트를 업데이트하고 사용 가능한 패키지의 최신 목록 가져오기:

`sudo port selfupdate`

- 오래된 패키지 업그레이드:

`sudo port upgrade outdated`

- 설치된 패키지의 이전 버전 제거:

`sudo port uninstall inactive`
