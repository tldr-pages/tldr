# dvdmaker

> 비디오 DVD를 생성하고 구움.
> 참고: 이 프로그램은 더 이상 사용되지 않음.
> 더 많은 정보: <https://learn.microsoft.com/previous-versions/windows/desktop/wmmdvdm/command-line-options-for-windows-dvd-maker>.

- Windows DVD Maker 실행:

`dvdmaker`

- DVD 굽기 드라이브 지정:

`dvdmaker -drive:{{F}}`

- Windows DVD Maker 가져오기 목록 XML 파일에 정의된 미디어 가져오기:

`dvdmaker {{경로\대상\파일.xml}}`

- Windows DVD Maker 설정을 기본값으로 초기화 (경고: 애플리케이션 설정이 모두 초기화됨):

`dvdmaker _clear_`
