# excel

> Microsoft Office의 스프레드시트 애플리케이션.
> 더 많은 정보: <https://support.microsoft.com/office/command-line-switches-for-microsoft-office-products-079164cd-4ef5-4178-b235-441737deb3a6#category=excel>.

- 스프레드시트 애플리케이션 실행:

`excel`

- 매크로 사용이 가능한([m]acro-enabled) 새 통합 문서 생성:

`excel /m`

- 통합 문서를 템플릿([t]emplate)으로 열기:

`excel {{[/n|/t]}} {{경로\대상\파일.xlsx}}`

- 안전([s]afe) 모드로 Microsoft Excel 실행:

`excel {{[/s|/safemode]}}`
