# copy

> 파일을 한 위치에서 다른 위치로 복사.
> PowerShell에서는, 이 명령이 `Copy-Item`의 별칭. 이 문서는 명령 프롬프트 (`cmd`)의 `copy` 명령을 기준으로 함.
> 더 많은 정보: <https://learn.microsoft.com/windows-server/administration/windows-commands/copy>.

- 파일을 다른 위치로 복사:

`copy {{경로\대상\소스_파일}} {{경로\대상\대상_파일}}`

- 파일명을 유지한 채, 다른 디렉터리로 복사:

`copy {{경로\대상\소스_파일}} {{경로\대상\대상_디렉터리}}`

- 여러 파일을 디렉터리로 복사:

`copy {{경로\대상\파일1 경로\대상\파일2 ...}} {{경로\대상\대상_디렉터리}}`

- 한 디렉터리의 모든 파일을 다른 디렉터리로 복사:

`copy {{경로\대상\소스_디렉터리\*}} {{경로\대상\대상_디렉터리}}`

- 특정 확장자의 파일만 복사:

`copy {{경로\대상\소스_디렉터리\*.ext}} {{경로\대상\대상_디렉터리}}`

- 덮어쓰기 전 확인 메시지를 표시하며 파일 복사:

`copy /-y {{경로\대상\소스_파일}} {{경로\대상\대상_파일}}`
