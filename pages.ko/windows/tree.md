# tree

> 경로의 디렉토리 구조에 대한 그래픽 트리를 표시합니다.
> 더 많은 정보: <https://learn.microsoft.com/windows-server/administration/windows-commands/tree>.

- 현재 디렉토리에 대한 트리 표시:

`tree`

- 특정 디렉토리에 대한 트리 표시:

`tree {{경로\대상\폴더}}`

- 특정 디렉토리에 대한 트리 표시 (파일 포함):

`tree {{경로\대상\폴더}} /f`

- 트리 표시 그래픽 문자 대신 ASCII 문자 사용:

`tree {{경로\대상\폴더}} /a`
