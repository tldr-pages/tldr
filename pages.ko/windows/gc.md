# gc

> PowerShell에서는, 원래의 `gc` 프로그램 (<https://graphviz.org/pdf/gc.1.pdf>)이 설치되어 있지 않은 경우, 이 명령이 `Get-Content` (`cat`과 동일한 기능)의 별칭임.

- GraphViz의 `gc` 명령 문서 보기:

`tldr gc {{[-p|--platform]}} common`

- PowerShell의 `Get-Content` 명령 문서 보기:

`tldr get-content`

- 도움말을 출력하여 현재 `gc`가 GraphViz의 `gc`인지, PowerShell의 `Get-Content`인지 확인:

`gc -?`
