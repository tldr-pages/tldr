# GetHelpCmd

> Microsoft 365 Enterprise 설치에서 발생하는 일반적인 문제를 분석하고 해결.
> 이전 명칭은 Microsoft Support and Recovery Assistant (SaRA)이며, 명령줄 버전 (`SaRAcmd.exe`)임.
> 참고: 이 도구는 PowerShell (`Get-Help`)와는 관련이 없으며, 현재는 Windows 설정에서 제공하는 GUI 문제 해결 도구로 대체되어 더 이상 사용되지 않음.
> `OutlookCalendarCheckTask`, `ExpertExperienceAdminTask`와 같은 일부 하위 명령은 각각 별도의 사용 문서를 제공.
> 더 많은 정보: <https://learn.microsoft.com/troubleshoot/microsoft-365/admin/miscellaneous/get-help-command-line-overview>.

- PowerShell의 `Get-Help` 명령 문서 보기:

`tldr get-help`

- `ExpertExperienceAdminTask` 시나리오 문서 보기:

`tldr gethelpcmd-expertexperienceadmintask`

- `OutlookCalendarCheckTask` 시나리오 문서 보기:

`tldr gethelpcmd-outlookcalendarchecktask`
