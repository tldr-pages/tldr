# cl

> 소스 코드 파일을 컴파일하고 링크하는 Microsoft C/C++ 컴파일러.
> 더 많은 정보: <https://learn.microsoft.com/cpp/build/reference/compiler-command-line-syntax>.

- 소스 파일 컴파일:

`cl {{경로\대상\소스파일.c}}`

- 사용자 지정 이름의 실행 파일 생성:

`cl /Fe {{경로\대상\출력_실행파일}} {{경로\대상\소스파일.c}}`

- 최적화를 활성화하여 소스 파일 컴파일:

`cl /O2 {{경로\대상\소스파일.c}}`

- 디버그 정보 포함한 실행 파일 생성:

`cl /Zi {{경로\대상\소스파일.c}}`

- 여러 소스 파일 컴파일:

`cl {{경로\대상\소스파일1.c 경로\대상\소스파일2.c ...}}`

- 컴파일된 파일의 출력 디렉터리 지정:

`cl /Fo {{경로\대상\출력_디렉터리}}\ {{경로\대상\소스파일.c}}`

- 경고를 오류로 처리하여 컴파일:

`cl /WX {{경로\대상\소스파일.c}}`
