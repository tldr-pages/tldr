# whoami

> 현재 사용자에 대한 세부 정보를 표시합니다.
> 더 많은 정보: <https://learn.microsoft.com/windows-server/administration/windows-commands/whoami>.

- 현재 사용자의 사용자 이름 표시:

`whoami`

- 현재 사용자가 멤버인 그룹 표시:

`whoami /groups`

- 현재 사용자의 보안 권한 표시:

`whoami /priv`

- 현재 사용자의 이름을 사용자 계정 이름 (UPN)으로 표시:

`whoami /upn`

- 현재 사용자의 로그온 ID 표시:

`whoami /logonid`

- 현재 사용자에 대한 모든 정보 표시:

`whoami /all`
