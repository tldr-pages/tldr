# help

> 명령 프롬프트(Command Prompt)의 DOS 스타일 표준 명령에 대한 도움말을 표시.
> PowerShell에서는, 이 명령이 `Get-Help`의 별칭. 이 문서는 명령 프롬프트(`cmd`)의 `help` 명령을 기준으로 함.
> 더 많은 정보: <https://learn.microsoft.com/windows-server/administration/windows-commands/help>.

- PowerShell의 해당 명령에 대한 문서 보기:

`tldr get-help`

- 지원되는 명령 프롬프트 명령 목록 표시:

`help`

- 지정한 명령 프롬프트 명령의 도움말 표시:

`help {{명령어}}`
