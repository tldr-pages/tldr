# man

> Μορφοποίηση και εμφάνιση εγχειριδίου.
> Περισσότερες πληροφορίες: <https://manned.org/man>.

- Εμφάνιση του εγχειρίδιου για μια εντολή:

`man {{command}}`

- Άνοιγμα του εγχειρίδιου για μια εντολή σε περιηγητή (html/-[H]) (η μεταβλητή περιβάλλοντος `$BROWSER` μπορεί να αντικαταστήσει το `=browser_name`):

`man {{[-H|--html=]}}{{browser_name}} {{command}}`

- Εμφάνιση του εγχειρίδιου για μια εντολή από την ενότητα 7:

`man 7 {{command}}`

- Λίστα όλων των διαθέσιμων ενοτήτων για μια εντολή (whatis/-[f]):

`man {{[-f|--whatis]}} {{command}}`

- Εμφάνιση της διαδρομής (path/-[w]) που αναζητείται για εγχειρίδια:

`man {{[-w|--path]}}`

- Εμφάνιση της τοποθεσίας (where/-[w]) του εγχειρίδιου αντί για το ίδιο το εγχειρίδιο:

`man {{[-w|--where]}} {{command}}`

- Εμφάνιση του εγχειρίδιου χρησιμοποιώντας μια συγκεκριμένη τοπική ρύθμιση ([L]ocale):

`man {{[-L|--locale]}} {{locale}} {{command}}`

- Αναζήτηση για εγχειρίδια που περιέχουν μια συμβολοσειρά αναζήτησης (apropos/-[k]):

`man {{[-k|--apropos]}} "{{search_string}}"`
