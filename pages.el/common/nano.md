# nano

> Επεξεργαστής κειμένου. Ένας βελτιωμένος κλώνος του `pico`.
> Δείτε επίσης: `pico`, `rnano`.
> Περισσότερες πληροφορίες: <https://nano-editor.org/dist/latest/nano.html>.

- Άνοιγμα συγκεκριμένων αρχείων, μεταβαίνοντας στο επόμενο αρχείο μετά το κλείσιμο του προηγούμενου:

`nano {{path/to/file1 path/to/file2 ...}}`

- Εκκίνηση του επεξεργαστή χωρίς τη χρήση αρχείων ρυθμίσεων ([I]gnore rc files):

`nano {{[-I|--ignorercfiles]}}`

- Άνοιγμα ενός αρχείου και τοποθέτηση του δρομέα σε συγκεκριμένη γραμμή και στήλη:

`nano +{{line}},{{column}} {{path/to/file}}`

- Άνοιγμα ενός αρχείου και ενεργοποίηση της ήπιας αναδίπλωσης κειμένου([S]oft wrap):

`nano {{[-S|--softwrap]}} {{path/to/file}}`

- Άνοιγμα ενός αρχείου και αυτόματη στοίχιση νέων γραμμών στην προηγούμενη γραμμή (auto[i]ndent):

`nano {{[-i|--autoindent]}} {{path/to/file}}`

- Άνοιγμα ενός αρχείου και δημιουργία αντιγράφου ασφαλείας ([B]ackup) (`path/to/file~`) κατά την αποθήκευση:

`nano {{[-B|--backup]}} {{path/to/file}}`

- Άνοιγμα ενός αρχείου σε περιορισμένη λειτουργία ([R]estricted) (δηλ. όχι ανάγνωση/εγγραφή σε αρχεία που δεν καθορίζονται στη γραμμή εντολών):

`nano {{[-R|--restricted]}} {{path/to/file}}`

- Έξοδος από το nano:

`<Ctrl x>`
