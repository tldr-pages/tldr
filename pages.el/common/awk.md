# awk

> Μια ευέλικτη γλώσσα προγραμματισμού για εργασία με αρχεία.
> Σημείωση: Διαφορετικές υλοποιήσεις του AWK συχνά κάνουν αυτήν την εντολή συμβολικό σύνδεσμο στο εκτελέσιμο τους.
> Δείτε επίσης: `gawk`.
> Περισσότερες πληροφορίες: <https://github.com/onetrueawk/awk>.

- Εκτύπωση της πέμπτης στήλης (γνωστή και ως πεδίο) σε ένα αρχείο διαχωρισμένο με κενά:

`awk '{print $5}' {{path/to/file}}`

- Εκτύπωση της δεύτερης στήλης των γραμμών που περιέχουν "foo" σε ένα αρχείο διαχωρισμένο με κενά:

`awk '/{{foo}}/ {print $2}' {{path/to/file}}`

- Εκτύπωση της τελευταίας στήλης κάθε γραμμής σε ένα αρχείο, χρησιμοποιώντας κόμμα (αντί για κενό) ως διαχωριστικό πεδίου (field separator/-[F]):

`awk -F ',' '{print $NF}' {{path/to/file}}`

- Άθροιση των τιμών στην πρώτη στήλη ενός αρχείου και εκτύπωση του συνόλου:

`awk '{s+=$1} END {print s}' {{path/to/file}}`

- Εκτύπωση κάθε τρίτης γραμμής ξεκινώντας από την πρώτη γραμμή:

`awk 'NR%3==1' {{path/to/file}}`

- Εκτύπωση διαφορετικών τιμών με βάση συνθήκες:

`awk '{if ($1 == "foo") print "Exact match foo"; else if ($1 ~ "bar") print "Partial match bar"; else print "Baz"}' {{path/to/file}}`

- Εκτύπωση όλων των γραμμών στις οποίες η τιμή της 10ης στήλης είναι μεταξύ ενός ελαχίστου και ενός μεγίστου:

`awk '($10 >= {{min_value}} && $10 <= {{max_value}})'`

- Εκτύπωση πίνακα χρηστών με UID >=1000 με κεφαλίδα και μορφοποιημένη έξοδο, χρησιμοποιώντας άνω κάτω τελεία ως διαχωριστικό (`%-20s` σημαίνει: στοίχιση αριστερά 20 χαρακτήρων, `%6s` σημαίνει: στοίχιση δεξιά 6 χαρακτήρων):

`awk 'BEGIN {FS=":";printf "%-20s %6s %25s\n", "Name", "UID", "Shell"} $4 >= 1000 {printf "%-20s %6d %25s\n", $1, $4, $7}' /etc/passwd`
