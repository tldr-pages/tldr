# git whatchanged

> Dit commando is een alias van `git log --raw --no-merges` (log met ruwe diffs voor elke commit).
> Opmerking: dit commando wordt niet meer onderhouden en zal in de toekomst worden verwijderd.
> Meer informatie: <https://git-scm.com/docs/git-whatchanged>.

- Bekijk de documentatie van het originele commando:

`tldr git log`
