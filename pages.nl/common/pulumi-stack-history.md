# pulumi stack history

> Toon de geschiedenis van een stack.
> Meer informatie: <https://www.pulumi.com/docs/iac/cli/commands/pulumi_stack_history/>.

- Toon de geschiedenis van de huidige stack:

`pulumi stack {{[hist|history]}}`

- Toon de geschiedenis van de huidige stack, waarbij volledige data worden vertoont in plaats van relatieve data:

`pulumi stack {{[hist|history]}} --full-dates`

- Toon de geschiedenis van de huidige stack in JSON-formaat:

`pulumi stack {{[hist|history]}} {{[-j|--json]}}`

- Toon de geschiedenis van een specifieke stack:

`pulumi stack {{[hist|history]}} {{[-s|--stack]}} {{stack_naam}}`

- Toon de help:

`pulumi stack {{[hist|history]}} {{[-h|--help]}}`
