# docker image tag

> Wijs tags toe aan bestaande Docker-images.
> Meer informatie: <https://docs.docker.com/reference/cli/docker/image/tag/>.

- Wijs een naam en tag toe aan een specifiek image-ID:

`docker {{[tag|image tag]}} {{id}} {{naam}}:{{tag}}`

- Wijs een tag toe aan een specifiek image:

`docker {{[tag|image tag]}} {{image}}:{{current_tag}} {{image}}:{{new_tag}}`

- Toon de help:

`docker {{[tag|image tag]}}`
