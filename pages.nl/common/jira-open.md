# jira open

> Open een Jira issue of project in de browser.
> Meer informatie: <https://github.com/ankitpokhrel/jira-cli#other-commands>.

- Open het Jira project in de browser en toon de URL:

`jira open`

- Open een specifiek issue in de browser en toon de URL:

`jira open {{issue_id}}`

- Toon de URL zonder de browser te openen:

`jira open {{issue_id}} {{[-n|--no-browser]}}`

- Toon de help:

`jira open {{[-h|--help]}}`
