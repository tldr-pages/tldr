- Log in bij Azure:

`az login`

- Beheer azure abonnementsgegevens:

`az account`

- Toon alle Azure Managed Disks:

`az disk list`

- Toon alle Azure virtual machines:

`az vm list`

- Beheer Azure Kubernetes Services:

`az aks`

- Beheer Azure Network resources:

`az network`

- Start in interactieve modus:

`az interactive`

- Toon de help:

`az --help`
