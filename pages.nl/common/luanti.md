# luanti

> Infinite-world block sandbox game.
> Meer informatie: <https://manned.org/luanti>.

- Start Luanti in client-modus:

`luanti`

- Toon gedownloade game-modes:

`luanti --gameid list`

- Start Luanti in servermodus door een specifieke gamemodus te hosten:

`luanti --server --gameid {{game_id}}`

- Start een server met de standaard wereld zodra deze is aangemaakt:

`luanti --server`

- Start een server met een specifieke wereld:

`luanti --server --world {{wereld_naam}}`

- Start een server op een specifieke poort:

`luanti --server --port {{poort}}`

- Schrijf logs naar een specifiek bestand:

`luanti --logfile {{pad/naar/bestand}}`

- Schrijf alleen fouten naar de console:

`luanti --quiet`
