# okular

> Bekijk documenten.
> Meer informatie: <https://docs.kde.org/stable5/en/okular/okular/command-line-options.html>.

- Start de documentviewer:

`okular`

- Open specifieke documenten:

`okular {{pad/naar/bestand1 pad/naar/bestand2 ...}}`

- Open een document op een specifieke pagina:

`okular {{[-p|--page]}} {{paginanummer}} {{pad/naar/bestand}}`

- Open een specifiek document in presentatiemodus:

`okular --presentation {{pad/naar/bestand}}`

- Open een specifiek document en start een dialoog om te printen:

`okular --print {{pad/naar/bestand}}`

- Open een document en zoek voor een specifieke string:

`okular --find {{zoekstring}} {{pad/naar/bestand}}`
