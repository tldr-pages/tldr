# pkg

> FreeBSD pakketbeheer.
> Meer informatie: <https://man.freebsd.org/cgi/man.cgi?pkg>.

- Installeer een nieuw pakket:

`pkg install {{pakket}}`

- Verwijder een pakket:

`pkg delete {{pakket}}`

- Upgrade alle pakketten:

`pkg upgrade`

- Zoek naar een pakket:

`pkg search {{keyword}}`
