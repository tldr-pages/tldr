# npm star

> Отмечать пакет как избранный.
> Больше информации: <https://docs.npmjs.com/cli/npm-star/>.

- Добавить публичный пакет в избранное из реестра по умолчанию:

`npm star {{имя_пакета}}`

- Добавить пакет в избранное в определённой области видимости:

`npm star @{{область_видимости}}/{{имя_пакета}}`

- Добавить пакет в избранное из определённого реестра:

`npm star {{имя_пакета}} --registry {{url_реестра}}`

- Добавить приватный пакет в избранное, требующий аутентификации:

`npm star {{имя_пакета}} --auth-type {{legacy|oauth|web|saml}}`

- Добавить пакет в избранное, указав одноразовый пароль для двухфакторной аутентификации:

`npm star {{имя_пакета}} --otp {{одноразовый_пароль}}`

- Добавить пакет в избранное с подробным логированием:

`npm star {{имя_пакета}} --loglevel verbose`

- Вывести список всех избранных пакетов:

`npm star --list`

- Вывести список избранных пакетов из определённого реестра:

`npm star --list --registry {{url_реестра}}`
