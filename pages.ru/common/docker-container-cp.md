# docker container cp

> Копировать файлы или директории между файловыми системами хоста и контейнера.
> Больше информации: <https://docs.docker.com/reference/cli/docker/container/cp/>.

- Скопировать файл или директорию с хоста в контейнер:

`docker {{[cp|container cp]}} {{путь/к/файлу_или_директории_на_хосте}} {{имя_контейнера}}:{{путь/к/файлу_или_директории_в_контейнере}}`

- Скопировать файл или директорию из контейнера на хост:

`docker {{[cp|container cp]}} {{имя_контейнера}}:{{путь/к/файлу_или_директории_в_контейнере}} {{путь/к/файлу_или_директории_на_хосте}}`

- Скопировать файл или директорию с хоста в контейнер, следуя по символическим ссылкам (копируются целевые файлы, а не сами ссылки):

`docker {{[cp|container cp]}} {{[-L|--follow-link]}} {{путь/к/символической_ссылке_на_хосте}} {{имя_контейнера}}:{{путь/к/файлу_или_директории_в_контейнере}}`
