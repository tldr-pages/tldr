# GetADUsers.py

> Получать список пользователей из Active Directory, включая атрибуты: время последнего входа и email.
> Часть набора инструментов Impacket.
> Больше информации: <https://github.com/fortra/impacket>.

- Вывести список всех пользователей Active Directory и их атрибуты:

`GetADUsers.py -all -dc-ip {{ip_контроллера_домена}} {{домен}}/{{имя_пользователя}}:{{пароль}}`

- Получить информацию только для конкретного пользователя:

`GetADUsers.py -user {{пользователь}} -dc-ip {{ip_контроллера_домена}} {{домен}}/{{имя_пользователя}}:{{пароль}}`

- Извлечь данные пользователя с использованием аутентификации pass-the-hash:

`GetADUsers.py -all -dc-ip {{ip_контроллера_домена}} -hashes {{LM_хеш}}:{{NT_хеш}} {{домен}}/{{имя_пользователя}}`

- Сохранить результаты в файл:

`GetADUsers.py -all -dc-ip {{ip_контроллера_домена}} {{домен}}/{{имя_пользователя}}:{{пароль}} > {{путь/к/выходному_файлу.txt}}`
