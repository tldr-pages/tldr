# anki

> Мощная и умная программа для работы с флеш-картами.
> Больше информации: <https://manned.org/anki>.

- Запустить графический интерфейс:

`anki`

- Использовать указанный профиль:

`anki {{[-p|--profile]}} {{имя_профиля}}`

- Использовать указанный язык:

`anki {{[-l|--lang]}} {{язык}}`

- Использовать нестандартный каталог (по умолчанию `~/Anki`):

`anki {{[-b|--base]}} {{путь/к/каталогу}}`
