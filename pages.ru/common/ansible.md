# ansible

> Управлять группами компьютеров удалённо по SSH. (Используйте файл `/etc/ansible/hosts` для добавления новых групп/хостов).
> Некоторые подкоманды, такие как `galaxy`, имеют собственную документацию по использованию.
> Больше информации: <https://docs.ansible.com/projects/ansible/latest/cli/ansible.html>.

- Вывести список хостов, принадлежащих группе:

`ansible {{группа}} --list-hosts`

- Проверить доступность (ping) группы хостов, вызвав модуль ping:

`ansible {{группа}} {{[-m|--module-name]}} ping`

- Вывести факты о группе хостов, вызвав модуль setup:

`ansible {{группа}} {{[-m|--module-name]}} setup`

- Выполнить команду для группы хостов, вызвав модуль command с аргументами:

`ansible {{группа}} {{[-m|--module-name]}} command {{[-a|--args]}} '{{команда}}'`

- Выполнить команду с правами администратора:

`ansible {{группа}} {{[-b|--become]}} --ask-become-pass {{[-m|--module-name]}} command {{[-a|--args]}} '{{команда}}'`

- Выполнить команду, используя указанный файл инвентаря:

`ansible {{группа}} {{[-i|--inventory]}} {{файл_инвентаря}} {{[-m|--module-name]}} command {{[-a|--args]}} '{{команда}}'`

- Вывести список групп в инвентаре:

`ansible localhost {{[-m|--module-name]}} debug {{[-a|--args]}} '{{var=groups.keys()}}'`
