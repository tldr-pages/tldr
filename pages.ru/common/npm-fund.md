# npm fund

> Получать информацию о финансировании из пакетов.
> Больше информации: <https://docs.npmjs.com/cli/npm-fund/>.

- Вывести список зависимостей с URL финансирования для проекта в текущем каталоге:

`npm fund`

- Открыть URL финансирования конкретного пакета в браузере по умолчанию:

`npm fund {{пакет}}`

- Вывести список зависимостей с URL финансирования для конкретной рабочей области проекта в текущем каталоге:

`npm fund {{[-w|--workspace]}} {{рабочая_область}}`
