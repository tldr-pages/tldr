# ed

> Оригинальный текстовый редактор Unix.
> Смотрите также: `awk`, `sed`.
> Больше информации: <https://www.gnu.org/software/ed/manual/ed_manual.html>.

- Запустить интерактивную сессию редактора с пустым документом:

`ed`

- Запустить интерактивную сессию редактора с пустым документом и указанным приглашением:

`ed {{[-p|--prompt]}} '{{> }}'`

- Запустить интерактивную сессию редактора с подробными (user-friendly) ошибками:

`ed {{[-v|--verbose]}}`

- Запустить интерактивную сессию редактора с пустым документом, без диагностики, подсчета байтов и приглашения '!':

`ed {{[-q|--quiet]}} {{[-s|--script]}}`

- Запустить интерактивную сессию редактора без изменения кода выхода при сбое команды:

`ed {{[-l|--loose-exit-status]}}`

- Редактировать указанный файл (отображает количество байт загруженного файла):

`ed {{путь/к/файлу}}`

- [Интерактивно] Заменить строку указанной заменой во всех строках:

`,s/{{регулярное_выражение}}/{{замена}}/g<Enter>`

- [Интерактивно] Выйти из `ed`:

`q<Enter>`
