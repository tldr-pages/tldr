# npm rebuild

> Пересобирать нативные пакеты Node.js после изменений Node или зависимостей.
> Больше информации: <https://docs.npmjs.com/cli/npm-rebuild/>.

- Пересобрать конкретный пакет:

`npm {{[rb|rebuild]}} {{пакет}}`

- Пересобрать все установленные пакеты:

`npm {{[rb|rebuild]}}`

- Пересобрать с подробным выводом:

`npm {{[rb|rebuild]}} --verbose`

- Пересобрать пакет в указанном каталоге:

`npm {{[rb|rebuild]}} --prefix {{путь/к/каталогу}} {{пакет}}`

- Пересобрать без использования кэша npm:

`npm {{[rb|rebuild]}} --no-cache`

- Пересобрать в глобальном режиме:

`npm {{[rb|rebuild]}} {{[-g|--global]}}`
