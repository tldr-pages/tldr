# showmount

> Отображать информацию о монтировании для NFS-сервера.
> Больше информации: <https://manned.org/showmount>.

- Показать список клиентов, которые в данный момент монтируют ресурсы сервера:

`showmount {{имя_хоста}}`

- Показать список экспортируемых ресурсов NFS-сервера:

`showmount {{[-e|--exports]}} {{имя_хоста}}`

- Показать всех клиентов и каталоги, которые они примонтировали:

`showmount {{[-a|--all]}} {{имя_хоста}}`

- Показать только каталоги, примонтированные клиентами:

`showmount {{[-d|--directories]}} {{имя_хоста}}`

- Показать список экспортируемых ресурсов без заголовков:

`showmount {{[-e|--exports]}} --no-headers {{имя_хоста}}`

- Показать справку:

`showmount {{[-h|--help]}}`
