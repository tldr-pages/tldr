# npm logout

> Выходить из учётной записи пользователя реестра.
> Смотрите также: `npm login`.
> Больше информации: <https://docs.npmjs.com/cli/npm-logout/>.

- Выйти из учётной записи пользователя реестра:

`npm logout`

- Выйти с использованием пользовательского реестра:

`npm logout --registry {{url_реестра}}`
