# npm hook

> Управлять хуками реестра `npm` для пакетов.
> Примечание: эта команда устарела.
> Больше информации: <https://docs.npmjs.com/cli/v10/hook/>.

- Вывести список всех активных хуков:

`npm hook ls`

- Добавить новый хук для пакета:

`npm hook add {{название_пакета}} {{событие}} {{целевой_url}}`

- Удалить конкретный хук по его ID:

`npm hook rm {{id_хука}}`

- Обновить хук новой информацией:

`npm hook update {{id_хука}} {{целевой_url}}`
