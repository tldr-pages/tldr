# npm token

> Управлять токенами аутентификации реестра npm и генерировать их.
> Больше информации: <https://docs.npmjs.com/cli/npm-token/>.

- Создать новый токен аутентификации:

`npm token create`

- Вывести список всех токенов, связанных с учётной записью:

`npm token list`

- Отозвать определённый токен по его идентификатору:

`npm token revoke {{идентификатор_токена}}`

- Создать токен с доступом только для чтения:

`npm token create --read-only`

- Создать токен с доступом на публикацию:

`npm token create --publish`

- Автоматически настроить токен npm в глобальном файле `.npmrc` при входе:

`npm login`

- Удалить токен из глобальной конфигурации:

`npm token revoke {{идентификатор_токена}}`
