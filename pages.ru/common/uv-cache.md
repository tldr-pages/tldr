# uv cache

> Управлять глобальным каталогом кеша `uv`.
> Больше информации: <https://docs.astral.sh/uv/reference/cli/#uv-cache>.

- Показать путь к каталогу кеша:

`uv cache dir`

- Очистить весь кеш (удалить все кешированные пакеты и окружения):

`uv cache clean`

- Очистить кеш для определённых пакетов:

`uv cache clean {{пакет1 пакет2 ...}}`

- Удалить все недостижимые объекты из кеша:

`uv cache prune`

- Очистить кеш с оптимизацией для CI-окружений (например, GitHub Actions):

`uv cache prune --ci`

- Указать определённый каталог кеша:

`uv cache clean --cache-dir {{путь/к/кешу}}`

- Очистить кеш с подробным выводом:

`uv cache clean {{[-v|--verbose]}}`
