# npm uninstall

> Удалять пакет.
> Больше информации: <https://docs.npmjs.com/cli/npm-uninstall/>.

- Удалить пакет из текущего проекта:

`npm {{[r|uninstall]}} {{имя_пакета}}`

- Удалить пакет глобально:

`npm {{[r|uninstall]}} {{имя_пакета}} {{[-g|--global]}}`

- Удалить несколько пакетов сразу:

`npm {{[r|uninstall]}} {{имя_пакета1 имя_пакета2 ...}}`
