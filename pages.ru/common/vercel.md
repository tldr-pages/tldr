# vercel

> Выполнять развёртывание и управлять развёртываниями Vercel.
> Больше информации: <https://vercel.com/docs/cli>.

- Выполнить развёртывание текущего каталога:

`vercel`

- Выполнить развёртывание текущего каталога в продакшен:

`vercel --prod`

- Выполнить развёртывание каталога:

`vercel {{путь/к/проекту}}`

- Инициализировать пример проекта:

`vercel init`

- Выполнить развёртывание с переменными окружения:

`vercel {{[-e|--env]}} {{ENV}}={{var}}`

- Собрать с переменными окружения:

`vercel {{[-b|--build-env]}} {{ENV}}={{var}}`

- Задать регионы по умолчанию для развёртывания:

`vercel --regions {{идентификатор_региона}}`

- Удалить развёртывание:

`vercel remove {{имя_проекта}}`
