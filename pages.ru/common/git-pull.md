# git pull

> Получать ветку из удалённого репозитория и выполнять её слияние с локальным репозиторием.
> Больше информации: <https://git-scm.com/docs/git-pull>.

- Загрузить изменения из удалённого репозитория по умолчанию и выполнить слияние (merge):

`git pull`

- Загрузить изменения из удалённого репозитория по умолчанию и использовать перебазирование (rebase):

`git pull {{[-r|--rebase]}}`

- Загрузить изменения из указанного удалённого репозитория и ветки, затем выполнить их слияние с `HEAD`:

`git pull {{имя_удалённого_репозитория}} {{ветка}}`
