# uv remove

> Удалять зависимости из файла `pyproject.toml` проекта.
> Больше информации: <https://docs.astral.sh/uv/reference/cli/#uv-remove>.

- Удалить зависимость из проекта:

`uv remove {{пакет}}`

- Удалить несколько зависимостей:

`uv remove {{пакет1 пакет2 ...}}`

- Удалить зависимость разработки:

`uv remove --dev {{пакет}}`

- Удалить зависимость из необязательной группы:

`uv remove --optional {{имя_дополнения}} {{пакет}}`

- Удалить зависимость из определённой группы:

`uv remove --group {{имя_группы}} {{пакет}}`

- Удалить без синхронизации виртуального окружения:

`uv remove --no-sync {{пакет}}`
