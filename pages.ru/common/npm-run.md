# npm run

> Запускать скрипт.
> Больше информации: <https://docs.npmjs.com/cli/npm-run/>.

- Вывести список доступных скриптов:

`npm run`

- Запустить скрипт:

`npm run {{имя_скрипта}}`

- Передать аргументы скрипту:

`npm run {{имя_скрипта}} -- {{аргумент}} {{--опция}}`

- Запустить скрипт с именем `start`:

`npm start`

- Запустить скрипт с именем `stop`:

`npm stop`

- Запустить скрипт с именем `restart`:

`npm restart`

- Запустить скрипт с именем `test`:

`npm {{[t|test]}}`
