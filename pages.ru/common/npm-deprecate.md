# npm deprecate

> Помечать версию или диапазон версий пакета `npm` как устаревшие.
> Больше информации: <https://docs.npmjs.com/cli/npm-deprecate/>.

- Пометить конкретную версию пакета как устаревшую:

`npm deprecate {{имя_пакета}}@{{версия}} "{{сообщение_об_устаревании}}"`

- Пометить диапазон версий пакета как устаревшие:

`npm deprecate {{имя_пакета}}@"<{{диапазон_версий}}" "{{сообщение_об_устаревании}}"`

- Снять пометку устаревания с конкретной версии пакета:

`npm deprecate {{имя_пакета}}@{{версия}} ""`
