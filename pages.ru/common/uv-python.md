# uv python

> Управлять версиями и установками Python.
> Больше информации: <https://docs.astral.sh/uv/reference/cli/#uv-python>.

- Вывести список всех доступных установок Python:

`uv python list`

- Установить версию Python:

`uv python install {{версия}}`

- Удалить версию Python:

`uv python uninstall {{версия}}`

- Найти установку Python:

`uv python find {{версия}}`

- Закрепить за текущим проектом определённую версию Python:

`uv python pin {{версия}}`

- Показать каталог установок Python в `uv`:

`uv python dir`
