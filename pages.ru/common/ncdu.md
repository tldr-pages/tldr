# ncdu

> Анализировать использование диска с интерфейсом ncurses.
> Больше информации: <https://dev.yorhel.nl/ncdu/man>.

- Проанализировать текущую рабочую директорию:

`ncdu`

- Использовать цветной вывод:

`ncdu --color {{dark|off}}`

- Проанализировать указанную директорию:

`ncdu {{путь/к/каталогу}}`

- Сохранить результаты в файл:

`ncdu -o {{путь/к/файлу}}`

- Исключить файлы, соответствующие шаблону (аргумент можно указывать несколько раз для добавления шаблонов):

`ncdu --exclude '{{*.txt}}'`
