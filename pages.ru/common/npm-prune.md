# npm prune

> Удалять неиспользуемые пакеты из `node_modules`.
> Примечание: Неиспользуемые пакеты — это пакеты в каталоге node_modules, не указанные в списке зависимостей.
> Больше информации: <https://docs.npmjs.com/cli/npm-prune/>.

- Удалить все неиспользуемые пакеты, не перечисленные в зависимостях:

`npm prune`

- Удалить неиспользуемые пакеты и devDependencies (полезно для рабочих сборок):

`npm prune --production`

- Показать, что будет удалено без внесения изменений:

`npm prune --dry-run`

- Вывести изменения в формате JSON:

`npm prune --json`

- Удалить конкретные пакеты по имени:

`npm prune {{имя_пакета}}`
