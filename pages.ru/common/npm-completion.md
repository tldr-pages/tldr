# npm completion

> Включать или настраивать автодополнение по Tab для команд npm.
> Больше информации: <https://docs.npmjs.com/cli/npm-completion/>.

- Включить автодополнение по Tab для текущей сессии оболочки:

`npm completion`

- Добавить автодополнение в профиль оболочки (Bash):

`npm completion >> ~/.bashrc`

- Добавить автодополнение в профиль оболочки (Zsh):

`npm completion >> ~/.zshrc`

- Записать скрипт автодополнения в файл:

`npm completion > {{путь/к/файлу}}`
