# git status

> Показывать изменения файлов в Git-репозитории.
> Выводить список изменённых, добавленных и удалённых файлов по сравнению с текущим коммитом.
> Больше информации: <https://git-scm.com/docs/git-status>.

- Показать неотслеживаемые и изменённые файлы, которые ещё не подготовлены для коммита:

`git status`

- Вывести информацию в кратком формате:

`git status {{[-s|--short]}}`

- Вывести информацию в кратком формате вместе с данными о текущей ветке:

`git status {{[-sb|--short --branch]}}`

- Показать изменения в проиндексированных (staged) файлах (аналогично `git diff --cached`):

`git status {{[-v|--verbose]}}`

- Показать изменения во всех отслеживаемых файлах (аналогично `git diff HEAD`):

`git status {{[-vv|--verbose --verbose]}}`

- Показать количество записей в скрытых изменениях (stash):

`git status --show-stash`

- Не показывать неотслеживаемые файлы в выводе:

`git status {{[-uno|--untracked-files=no]}}`
