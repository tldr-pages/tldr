# npm edit

> Выбирать зависимость в текущем проекте и открывать папку пакета в редакторе по умолчанию (`$EDITOR`).
> После редактирования пакет пересобирается для учёта изменений в скомпилированных пакетах.
> Больше информации: <https://docs.npmjs.com/cli/npm-edit/>.

- Открыть локально установленный пакет в редакторе по умолчанию:

`npm edit {{пакет}}`
