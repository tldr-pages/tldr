# zsh

> Z SHell — командный интерпретатор, совместимый с Bash.
> Смотрите также: `bash`, `!`, `^`.
> Больше информации: <https://zsh.sourceforge.io/Doc/Release/Invocation.html#Invocation>.

- Запустить интерактивную сессию оболочки:

`zsh`

- Выполнить указанные команды ([c]ommands):

`zsh -c "{{echo Привет, мир}}"`

- Выполнить указанный скрипт:

`zsh {{путь/к/скрипту.zsh}}`

- Проверить скрипт на наличие синтаксических ошибок без его выполнения:

`zsh {{[-n|--no-exec]}} {{путь/к/скрипту.zsh}}`

- Выполнить указанные команды из `stdin`:

`{{echo Привет, мир}} | zsh`

- Выполнить указанный скрипт, выводя каждую команду перед её выполнением:

`zsh {{[-x|--xtrace]}} {{путь/к/скрипту.zsh}}`

- Запустить интерактивную сессию оболочки в подробном режиме, выводя каждую команду перед её выполнением:

`zsh {{[-v|--verbose]}}`

- Запустить Zsh без загрузки пользовательской конфигурации (например, `~/.zshrc`):

`zsh {{[-f|--no-rcs]}}`
