# npm config

> Управлять настройками конфигурации `npm`.
> Больше информации: <https://docs.npmjs.com/cli/npm-config/>.

- Показать все настройки конфигурации:

`npm {{[c|config]}} list`

- Вывести все настройки конфигурации в формате `JSON`:

`npm {{[c|config]}} list --json`

- Получить значение конкретного ключа конфигурации:

`npm {{[c|config]}} get {{ключ}}`

- Установить значение ключа конфигурации:

`npm {{[c|config]}} set {{ключ}} {{значение}}`

- Удалить ключ конфигурации:

`npm {{[c|config]}} delete {{ключ}}`

- Открыть глобальный файл конфигурации npm в редакторе по умолчанию:

`npm {{[c|config]}} edit`

- Попытаться исправить недопустимые элементы конфигурации:

`npm {{[c|config]}} fix`
