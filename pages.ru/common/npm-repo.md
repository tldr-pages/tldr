# npm repo

> Открывать страницу репозитория пакета в браузере.
> Больше информации: <https://docs.npmjs.com/cli/npm-repo/>.

- Открыть страницу репозитория текущего проекта (на основе `package.json`):

`npm repo`

- Открыть страницу репозитория конкретного пакета из реестра:

`npm repo {{пакет}}`

- Открыть страницы репозиториев для нескольких пакетов:

`npm repo {{пакет1 пакет2 ...}}`

- Вывести URL репозитория вместо открытия в браузере:

`npm repo --browser false`

- Открыть страницу репозитория пакета в указанном браузере:

`npm repo --browser {{имя_браузера}}`

- Открыть страницу репозитория пакета в конкретной рабочей области:

`npm repo --workspace {{рабочая_область}}`
