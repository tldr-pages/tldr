# autojump

> Быстро переходить по часто посещаемым каталогам.
> Для сокращения ввода доступны псевдонимы, такие как `j` или `jc`.
> Смотрите также: `bashmarks`.
> Больше информации: <https://github.com/wting/autojump#name>.

- Добавить псевдонимы `autojump` в текущую оболочку:

`source /usr/share/autojump/autojump.{{bash|fish|zsh}}`

- Перейти в каталог, содержащий указанный шаблон:

`j {{шаблон}}`

- Перейти в подкаталог (дочерний) текущего каталога, содержащий указанный шаблон:

`jc {{шаблон}}`

- Открыть каталог, содержащий указанный шаблон, в файловом менеджере операционной системы:

`jo {{шаблон}}`

- Удалить несуществующие каталоги из базы данных `autojump`:

`j --purge`

- Показать записи в базе данных `autojump`:

`j {{[-s|--stat]}}`
