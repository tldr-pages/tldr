# npm dist-tag

> Управлять тегами дистрибуции пакетов.
> Больше информации: <https://docs.npmjs.com/cli/npm-dist-tag/>.

- Вывести список всех тегов дистрибуции для пакета:

`npm dist-tag ls {{пакет}}`

- Вывести список всех тегов дистрибуции для текущего пакета:

`npm dist-tag ls`

- Добавить тег дистрибуции к определённой версии пакета:

`npm dist-tag add {{пакет}}@{{версия}} {{тег}}`

- Удалить тег дистрибуции из пакета:

`npm dist-tag rm {{пакет}} {{тег}}`

- Добавить тег, используя настроенный тег из конфигурации npm:

`npm dist-tag add {{пакет}}@{{версия}}`

- Добавить тег с двухфакторной аутентификацией:

`npm dist-tag add {{пакет}}@{{версия}} {{тег}} --otp {{одноразовый_пароль}}`
