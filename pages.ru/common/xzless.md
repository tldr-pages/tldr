# xzless

> Отображать текст из файлов, сжатых с помощью `xz` и `lzma`.
> Смотрите также: `less`.
> Больше информации: <https://manned.org/xzless>.

- Просмотреть сжатый файл:

`xzless {{путь/к/файлу}}`

- Просмотреть сжатый файл с отображением номеров строк:

`xzless --LINE-NUMBERS {{путь/к/файлу}}`

- Просмотреть сжатый файл и выйти, если файл целиком помещается на первом экране:

`xzless --quit-if-one-screen {{путь/к/файлу}}`
