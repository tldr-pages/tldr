# nslookup

> Запрашивать серверы имён для получения различных записей домена.
> Смотрите также: `dig`, `resolvectl`, `host`.
> Больше информации: <https://manned.org/nslookup>.

- Запросить DNS-сервер системы по умолчанию для получения IP-адреса (A-записи) домена:

`nslookup {{example.com}}`

- Запросить указанный сервер имён для получения NS-записи домена:

`nslookup -type=NS {{example.com}} {{8.8.8.8}}`

- Выполнить обратный DNS-запрос (PTR-запись) для IP-адреса:

`nslookup -type=PTR {{54.240.162.118}}`

- Запросить любые доступные записи, используя протокол TCP:

`nslookup -vc -type=ANY {{example.com}}`

- Запросить указанный сервер имён для получения всего файла зоны (передача зоны) домена, используя протокол TCP:

`nslookup -vc -type=AXFR {{example.com}} {{сервер_имён}}`

- Запросить почтовый сервер (MX-запись) домена с отображением деталей транзакции:

`nslookup -type=MX -debug {{example.com}}`

- Запросить указанный сервер имён на определённом порту для получения TXT-записи домена:

`nslookup -port={{номер_порта}} -type=TXT {{example.com}} {{сервер_имён}}`
