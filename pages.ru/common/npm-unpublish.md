# npm unpublish

> Удалять пакет из реестра npm.
> Больше информации: <https://docs.npmjs.com/cli/npm-unpublish/>.

- Удалить определённую версию пакета:

`npm unpublish {{имя_пакета}}@{{версия}}`

- Удалить весь пакет:

`npm unpublish {{имя_пакета}} {{[-f|--force]}}`

- Удалить пакет с областью видимости:

`npm unpublish @{{область_видимости}}/{{имя_пакета}}`

- Указать таймаут перед удалением:

`npm unpublish {{имя_пакета}} --timeout {{время_в_миллисекундах}}`

- Выполнить пробный запуск для предварительного просмотра того, что будет удалено:

`npm unpublish {{имя_пакета}} --dry-run`
