# npm cache

> Управлять кэшем пакетов npm.
> Больше информации: <https://docs.npmjs.com/cli/npm-cache/>.

- Добавить конкретный пакет в кэш:

`npm cache add {{пакет}}`

- Очистить конкретный элемент кэша по ключу:

`npm cache clean {{ключ}}`

- Очистить весь кэш npm:

`npm cache clean {{[-f|--force]}}`

- Вывести список кэшированных пакетов:

`npm cache ls`

- Вывести список кэшированных пакетов, соответствующих указанному имени и версии:

`npm cache ls {{имя}}@{{версия}}`

- Проверить целостность кэша npm:

`npm cache verify`

- Вывести список всех записей в кэше npx:

`npm cache npx ls`
