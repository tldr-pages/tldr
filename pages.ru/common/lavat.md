# lavat

> Показывать анимированный эффект лавовой лампы в терминале.
> Настроить цвета, скорость, размер, движение и стиль отображения.
> Больше информации: <https://github.com/AngelJumbo/lavat#usage>.

- Запустить анимацию лавы с настройками по умолчанию:

`lavat`

- Запустить с именованными цветами лавы и ободка:

`lavat -c {{green}} -k {{red}}`

- Запустить в режиме градиента с двумя цветами в формате hex:

`lavat -g -c {{00FF00}} -k {{FF0000}}`

- Запустить с включенным движением под действием гравитации и плавучести:

`lavat -G`

- Запустить с настраиваемыми скоростью, радиусом, размером ободка и количеством метаболов:

`lavat -s {{1..10}} -r {{1..10}} -R {{1..5}} -b {{5..20}}`

- Запустить с пользовательским набором ASCII-символов:

`lavat -F "{{.oO@}}"`

- Показать справку:

`lavat -h`
