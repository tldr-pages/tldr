# bun create

> Создавать новый проект из шаблона.
> Примечание: `c` можно использовать как сокращение для `create`.
> Больше информации: <https://bun.com/docs/runtime/templating/create>.

- Создать новый проект из шаблона в интерактивном режиме:

`bun create {{шаблон}}`

- Создать новый проект из шаблона по указанному пути:

`bun create {{шаблон}} {{путь/к/каталогу_назначения}}`

- Создать новый проект из шаблона репозитория GitHub:

`bun create {{https://github.com/username/repo}} {{путь/к/каталогу_назначения}}`

- Создать новый проект из локального шаблона:

`bun create {{путь/к/шаблону}} {{путь/к/каталогу_назначения}}`

- Создать новый проект, перезаписав каталог назначения, если он существует:

`bun create {{шаблон}} {{путь/к/каталогу_назначения}} --force`

- Создать новый проект без автоматической инициализации репозитория Git:

`bun create {{шаблон}} {{путь/к/каталогу_назначения}} --no-git`

- Создать новый проект без автоматической установки зависимостей:

`bun create {{шаблон}} {{путь/к/каталогу_назначения}} --no-install`
