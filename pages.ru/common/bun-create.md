# bun create

> Создавать новый проект из шаблона.
> Больше информации: <https://bun.com/docs/runtime/templating/create>.

- Создать новый проект из официального шаблона в интерактивном режиме:

`bun {{[c|create]}} {{шаблон}}`

- Создать новый проект из официального шаблона в новый каталог:

`bun {{[c|create]}} {{шаблон}} {{путь/к/каталогу_назначения}}`

- Создать новый проект из шаблона репозитория GitHub:

`bun {{[c|create]}} {{https://github.com/username/repo}} {{путь/к/каталогу_назначения}}`

- Создать новый проект из локального шаблона:

`bun {{[c|create]}} {{путь/к/шаблону}} {{путь/к/каталогу_назначения}}`

- Создать новый проект, перезаписав каталог назначения, если он существует:

`bun {{[c|create]}} {{шаблон}} {{путь/к/каталогу_назначения}} --force`

- Создать новый проект без автоматической инициализации репозитория Git:

`bun {{[c|create]}} {{шаблон}} {{путь/к/каталогу_назначения}} --no-git`

- Создать новый проект без автоматической установки зависимостей:

`bun {{[c|create]}} {{шаблон}} {{путь/к/каталогу_назначения}} --no-install`
