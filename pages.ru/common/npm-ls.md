# npm ls

> Выводить установленные пакеты в `stdout`.
> Больше информации: <https://docs.npmjs.com/cli/npm-ls/>.

- Вывести все версии зависимостей верхнего уровня в текущем проекте в `stdout`:

`npm {{[ls|list]}}`

- Вывести список всех установленных пакетов, включая peer-зависимости:

`npm {{[ls|list]}} {{[-a|--all]}}`

- Вывести список всех глобально установленных пакетов:

`npm {{[ls|list]}} {{[-g|--global]}}`

- Вывести зависимости с подробной информацией:

`npm {{[ls|list]}} {{[-l|--long]}}`

- Вывести зависимости в машиночитаемом формате:

`npm {{[ls|list]}} {{[-p|--parseable]}}`

- Вывести зависимости в формате JSON:

`npm {{[ls|list]}} --json`
