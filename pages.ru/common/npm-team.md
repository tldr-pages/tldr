# npm team

> Управлять командами в организации в реестре `npm`.
> Больше информации: <https://docs.npmjs.com/cli/npm-team/>.

- Добавить пользователя в команду организации:

`npm team add {{организация:команда}} {{имя_пользователя}}`

- Удалить пользователя из команды:

`npm team rm {{организация:команда}} {{имя_пользователя}}`

- Создать новую команду в организации:

`npm team create {{организация:команда}}`

- Удалить команду из организации:

`npm team destroy {{организация:команда}}`

- Вывести список всех команд в организации:

`npm team ls {{организация}}`

- Вывести список всех пользователей в команде:

`npm team ls {{организация:команда}}`
