# git format-patch

> Подготавливать файлы `.patch`. Полезно при отправке коммитов по электронной почте.
> Смотрите также: `git am`.
> Больше информации: <https://git-scm.com/docs/git-format-patch>.

- Создать автоматически именованный файл `.patch` для всех неотправленных коммитов:

`git format-patch {{origin}}`

- Записать файл `.patch` для всех коммитов между двумя ревизиями в `stdout`:

`git format-patch {{ревизия_1}}..{{ревизия_2}}`

- Записать файл `.patch` для `n` последних коммитов:

`git format-patch -{{n}}`
