# git cola

> Мощный графический интерфейс Git с удобным и интуитивно понятным пользовательским интерфейсом.
> Больше информации: <https://git-cola.readthedocs.io/en/latest/git-cola.html>.

- Запустить графический интерфейс:

`git cola`

- Запустить графический интерфейс в режиме исправления (amend):

`git cola --amend`

- Запросить выбор Git-репозитория. По умолчанию используется текущий каталог:

`git cola --prompt`

- Открыть Git-репозиторий по указанному пути:

`git cola {{[-r|--repo]}} {{путь/к/git_репозиторию}}`

- Применить фильтр пути к виджету статуса:

`git cola {{[-s|--status-filter]}} {{фильтр}}`
