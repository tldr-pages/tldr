# xz

> Сжимать или распаковывать файлы XZ и LZMA.
> Больше информации: <https://manned.org/xz>.

- Сжать файл с помощью xz:

`xz {{путь/к/файлу}}`

- Распаковать файл XZ:

`xz {{[-d|--decompress]}} {{путь/к/файлу.xz}}`

- Сжать файл с помощью lzma:

`xz {{[-F|--format]}} lzma {{путь/к/файлу}}`

- Распаковать файл LZMA:

`xz {{[-d|--decompress]}} {{[-F|--format]}} lzma {{путь/к/файлу.lzma}}`

- Распаковать файл и записать в `stdout` (означает `--keep`):

`xz {{[-d|--decompress]}} {{[-c|--stdout]}} {{путь/к/файлу.xz}}`

- Сжать файл, сохранив исходный файл:

`xz {{[-k|--keep]}} {{путь/к/файлу}}`

- Сжать файл с самым быстрым сжатием:

`xz -0 {{путь/к/файлу}}`

- Сжать файл с лучшим сжатием:

`xz -9 {{путь/к/файлу}}`
