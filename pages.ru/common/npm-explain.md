# npm explain

> Объяснять, как установлен пакет, показывая его зависимости и причины включения.
> Больше информации: <https://docs.npmjs.com/cli/explain/>.

- Объяснить, почему установлен конкретный пакет:

`npm {{[why|explain]}} {{пакет}}`

- Показать объяснение в формате JSON:

`npm {{[why|explain]}} {{пакет}} --json`

- Включить peer-зависимости в объяснение:

`npm {{[why|explain]}} {{пакет}} --include peer`

- Ограничить глубину объяснения до 2 уровней:

`npm {{[why|explain]}} {{пакет}} --depth 2`
