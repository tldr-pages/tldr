# npm-home

> Открывать страницу `npm`, страницу Yarn или репозиторий GitHub пакета в веб-браузере.
> Больше информации: <https://github.com/sindresorhus/npm-home>.

- Открыть страницу `npm` конкретного пакета в веб-браузере:

`npm-home {{пакет}}`

- Открыть репозиторий GitHub конкретного пакета в веб-браузере:

`npm-home {{[-g|--github]}} {{пакет}}`

- Открыть страницу Yarn конкретного пакета в веб-браузере:

`npm-home {{[-y|--yarn]}} {{пакет}}`
