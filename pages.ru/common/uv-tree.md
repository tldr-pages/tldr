# uv tree

> Отображать зависимости проекта в виде дерева.
> Больше информации: <https://docs.astral.sh/uv/reference/cli/#uv-tree>.

- Показать дерево зависимостей для текущего окружения:

`uv tree`

- Показать дерево зависимостей для всех окружений:

`uv tree --universal`

- Показать дерево зависимостей до определённой глубины:

`uv tree {{[-d|--depth]}} {{n}}`

- Показать последнюю доступную версию для всех устаревших пакетов:

`uv tree --outdated`

- Исключить зависимости из группы dev:

`uv tree --no-dev`

- Показать инвертированное дерево, где потомки являются зависимыми, а не зависимостями:

`uv tree --invert`
