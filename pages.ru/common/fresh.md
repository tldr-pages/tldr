# fresh

> Полнофункциональный консольный текстовый редактор.
> Больше информации: <https://getfresh.dev/docs/>.

- Открыть файл:

`fresh {{путь/к/файлу}}`

- Найти текст по шаблону:

`<Ctrl f>{{шаблон_поиска}}<Enter>`

- Заменить текст по шаблону:

`<Ctrl r>{{шаблон_поиска}}<Enter>{{замена}}<Enter>`

- Показать файловый менеджер:

`<Ctrl e>`

- Сохранить файл:

`<Ctrl s>`

- Выйти:

`<Ctrl q>`
