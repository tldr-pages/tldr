# npm link

> Создать символическую ссылку на локальный пакет в глобальный каталог `node_modules` или в другой проект для разработки.
> Больше информации: <https://docs.npmjs.com/cli/npm-link/>.

- Создать символическую ссылку на текущий пакет глобально:

`npm {{[ln|link]}}`

- Привязать глобально связанный пакет к `node_modules` другого проекта:

`npm {{[ln|link]}} {{пакет}}`

- Удалить ссылку на пакет из текущего проекта:

`npm {{[r|unlink]}} {{пакет}}`
