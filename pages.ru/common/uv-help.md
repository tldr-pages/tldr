# uv help

> Показывать подробную документацию по командам `uv`.
> Больше информации: <https://docs.astral.sh/uv/reference/cli/#uv-help>.

- Показать общую справку по `uv`:

`uv help`

- Показать справку по определённой команде:

`uv help {{команда}}`

- Показать справку по подкоманде:

`uv help {{команда}} {{подкоманда}}`

- Показать справку без использования пейджера:

`uv help --no-pager {{команда}}`
