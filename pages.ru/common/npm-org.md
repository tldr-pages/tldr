# npm org

> Управлять организациями.
> Больше информации: <https://docs.npmjs.com/cli/npm-org/>.

- Добавить нового пользователя в организацию:

`npm org set {{название_организации}} {{имя_пользователя}}`

- Изменить роль пользователя в организации:

`npm org set {{название_организации}} {{имя_пользователя}} {{developer|admin|owner}}`

- Удалить пользователя из организации:

`npm org rm {{название_организации}} {{имя_пользователя}}`

- Вывести список всех пользователей в организации:

`npm org ls {{название_организации}}`

- Вывести список всех пользователей в организации в формате JSON:

`npm org ls {{название_организации}} --json`

- Показать роль пользователя в организации:

`npm org ls {{название_организации}} {{имя_пользователя}}`
