# uv tool

> Устанавливать и запускать команды, предоставляемые пакетами Python.
> Больше информации: <https://docs.astral.sh/uv/reference/cli/#uv-tool>.

- Запустить команду из пакета без его установки:

`uv tool run {{команда}}`

- Установить пакет Python глобально:

`uv tool install {{пакет}}`

- Обновить установленный пакет Python:

`uv tool upgrade {{пакет}}`

- Удалить пакет Python:

`uv tool uninstall {{пакет}}`

- Вывести список глобально установленных пакетов Python:

`uv tool list`
