# echo

> Выводить указанные аргументы.
> Смотрите также: `printf`.
> Больше информации: <https://www.gnu.org/software/coreutils/manual/html_node/echo-invocation.html>.

- Вывести текстовое сообщение (кавычки необязательны):

`echo "{{Привет, мир}}"`

- Вывести сообщение с переменными окружения:

`echo "{{Мой путь: $PATH}}"`

- Вывести сообщение без завершающего символа новой строки:

`echo -n "{{Привет, мир}}"`

- Добавить сообщение в конец файла:

`echo "{{Привет, мир}}" >> {{файл.txt}}`

- Включить интерпретацию escape-последовательностей (специальных символов):

`echo -e "{{Столбец 1\tСтолбец 2}}"`

- Вывести код возврата последней выполненной команды (Примечание: в Windows CMD и PowerShell эквиваленты — `echo %errorlevel%` и `$lastexitcode` соответственно):

`echo $?`

- Передать текст другой программе через `stdin`:

`echo "{{Привет, мир}}" | {{программа}}`
