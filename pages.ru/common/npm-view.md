# npm view

> Просматривать информацию о пакете из реестра.
> Больше информации: <https://docs.npmjs.com/cli/npm-view/>.

- Показать информацию о последней версии пакета:

`npm {{[v|view]}} {{пакет}}`

- Показать информацию об определённой версии пакета:

`npm {{[v|view]}} {{пакет}}@{{версия}}`

- Показать все доступные версии пакета:

`npm {{[v|view]}} {{пакет}} versions`

- Показать описание пакета:

`npm {{[v|view]}} {{пакет}} description`

- Показать зависимости последней версии пакета:

`npm {{[v|view]}} {{пакет}} dependencies`

- Показать URL репозитория пакета:

`npm {{[v|view]}} {{пакет}} repository`

- Показать владельцев пакета:

`npm {{[v|view]}} {{пакет}} maintainers`
