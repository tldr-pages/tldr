# npm init

> Создать файл `package.json`.
> Больше информации: <https://docs.npmjs.com/cli/npm-init/>.

- Инициализировать новый пакет в интерактивном режиме:

`npm init`

- Инициализировать новый пакет со значениями по умолчанию:

`npm init {{[-y|--yes]}}`

- Инициализировать новый пакет с использованием конкретного инициализатора:

`npm init {{create-react-app}} {{my-app}}`
