# npm help-search

> Искать в справочной документации npm по ключевому слову.
> Больше информации: <https://docs.npmjs.com/cli/npm-help-search/>.

- Искать команды или темы, связанные с ключевым словом:

`npm help-search {{ключевое_слово}}`

- Искать по нескольким словам:

`npm help-search {{ключевое_слово1 ключевое_слово2 ...}}`
