# npm access

> Устанавливать уровень доступа для опубликованных пакетов.
> Больше информации: <https://docs.npmjs.com/cli/npm-access/>.

- Вывести список пакетов для пользователя или области видимости:

`npm access list packages {{пользователь|область|область:команда}} {{пакет}}`

- Вывести список соавторов пакета:

`npm access list collaborators {{пакет}} {{имя_пользователя}}`

- Получить статус пакета:

`npm access get status {{пакет}}`

- Установить статус пакета (публичный или приватный):

`npm access set status {{public|private}} {{пакет}}`

- Выдать доступ к пакету:

`npm access grant {{read-only|read-write}} {{область:команда}} {{пакет}}`

- Отозвать доступ к пакету:

`npm access revoke {{область:команда}} {{пакет}}`

- Настроить требование двухфакторной аутентификации:

`npm access set mfa {{none|publish|automation}} {{пакет}}`
