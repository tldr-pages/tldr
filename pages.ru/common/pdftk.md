# pdftk

> Набор инструментов для работы с PDF.
> Больше информации: <https://www.pdflabs.com/docs/pdftk-man-page/>.

- Извлечь страницы 1-3, 5 и 6-10 из PDF-файла и сохранить их в новый файл:

`pdftk {{путь/к/исходному_файлу}}.pdf cat 1-3 5 6-10 output {{путь/к/выходному_файлу}}.pdf`

- Объединить (склеить) список PDF-файлов и сохранить результат в новый файл:

`pdftk {{путь/к/файлу1}}.pdf {{путь/к/файлу2}}.pdf cat output {{путь/к/выходному_файлу}}.pdf`

- Разбить каждую страницу PDF-файла в отдельный файл, используя заданный шаблон имени выходных файлов:

`pdftk {{путь/к/исходному_файлу}}.pdf burst output {{путь/к/выходному_файлу_%d}}.pdf`

- Повернуть все страницы на 180 градусов по часовой стрелке:

`pdftk {{путь/к/исходному_файлу}}.pdf cat 1-endsouth output {{путь/к/выходному_файлу}}.pdf`

- Повернуть третью страницу на 90 градусов по часовой стрелке, оставив остальные без изменений:

`pdftk {{путь/к/исходному_файлу}}.pdf cat 1-2 3east 4-end output {{путь/к/выходному_файлу}}.pdf`

- Чередовать два PDF-файла (односторонние сканы двустороннего документа), где обратные стороны отсканированы в обратном порядке:

`pdftk A={{путь/к/лицевым_сторонам}}.pdf B={{путь/к/обратным_сторонам}}.pdf shuffle A1-end Bend-1 output {{путь/к/выходному_файлу}}.pdf`
