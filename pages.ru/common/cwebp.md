# cwebp

> Сжимать файл изображения в формат WebP.
> Больше информации: <https://developers.google.com/speed/webp/docs/cwebp>.

- Сжать WebP со стандартными настройками (сжатие с потерями, q = 75) с сохранением в выходной файл:

`cwebp {{путь/к/изображению}} -o {{путь/к/результату.webp}}`

- Сжать WebP с наилучшим качеством сжатия с потерями и наибольшим размером файла:

`cwebp {{путь/к/изображению}} -o {{путь/к/результату.webp}} -q {{100}}`

- Сжать WebP с наихудшим качеством сжатия с потерями и наименьшим размером файла:

`cwebp {{путь/к/изображению}} -o {{путь/к/результату.webp}} -q {{0}}`

- Сжать WebP без потерь с минимально возможным размером файла:

`cwebp {{путь/к/изображению}} -o {{путь/к/результату.webp}} -z 9`

- Сжать WebP с изменением размера изображения (если ширина или высота равны 0, масштабирование сохраняет пропорции):

`cwebp {{путь/к/изображению}} -o {{путь/к/результату.webp}} -resize {{ширина}} {{высота}}`

- Сжать WebP с удалением информации о прозрачности:

`cwebp {{путь/к/изображению}} -o {{путь/к/результату.webp}} -noalpha`
