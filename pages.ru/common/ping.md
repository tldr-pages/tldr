# ping

> Отправлять пакеты ICMP ECHO_REQUEST сетевым узлам.
> Смотрите также: `mtr`.
> Больше информации: <https://manned.org/ping>.

- Пропинговать хост с IP `10.0.0.1` (Примечание: `10.0.0.1` можно сократить до `10.1`):

`ping 10.0.0.1`

- Пропинговать хост определенное количество раз:

`ping -c {{количество}} {{хост}}`

- Пропинговать хост с заданным интервалом в секундах между запросами (по умолчанию 1 секунда):

`ping -i {{секунды}} {{хост}}`

- Пропинговать хост без попытки определить символьные имена для адресов:

`ping -n {{хост}}`

- Пропинговать хост со звуковым сигналом при получении пакета (если ваш терминал это поддерживает):

`ping -a {{хост}}`

- Также отобразить сообщение, если ответ не был получен:

`ping -O {{хост}}`

- Пропинговать хост с определённым количеством запросов, таймаутом ответа для каждого пакета (`-W`) и общим ограничением времени (`-w`) всего выполнения ping:

`ping -c {{количество}} -W {{секунды}} -w {{секунды}} {{хост}}`
