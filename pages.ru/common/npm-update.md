# npm update

> Обновлять пакеты в текущем проекте.
> Больше информации: <https://docs.npmjs.com/cli/npm-update/>.

- Обновить все пакеты в текущем проекте:

`npm {{[up|update]}}`

- Обновить определённый пакет в текущем проекте:

`npm {{[up|update]}} {{пакет}}`

- Обновить пакет глобально:

`npm {{[up|update]}} {{пакет}} {{[-g|--global]}}`

- Обновить несколько пакетов сразу:

`npm {{[up|update]}} {{пакет1 пакет2 ...}}`
