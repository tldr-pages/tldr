# act

> Запускать GitHub Actions локально с использованием Docker.
> Больше информации: <https://manned.org/act>.

- Вывести список доступных задач (jobs):

`act {{[-l|--list]}}`

- Запустить событие по умолчанию:

`act`

- Запустить конкретный тип события:

`act {{тип_события}}`

- Запустить конкретную задачу (job):

`act {{[-j|--job]}} {{id_задачи}}`

- Не выполнять [n] действия реально (пробный прогон):

`act {{[-n|--dryrun]}}`

- Показать подробный лог:

`act {{[-v|--verbose]}}`

- Запустить конкретный рабочий процесс (workflow) с событием push:

`act push {{[-W|--workflows]}} {{путь/к/workflow}}`
