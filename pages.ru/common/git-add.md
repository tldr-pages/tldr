# git add

> Добавлять изменённые файлы в индекс для последующего коммита.
> Больше информации: <https://git-scm.com/docs/git-add>.

- Подготовить файл для коммита (добавить в индекс):

`git add {{путь/к/файлу}}`

- Добавить все файлы (отслеживаемые и неотслеживаемые):

`git add {{[-A|--all]}}`

- Рекурсивно добавить все файлы, начиная с текущего каталога:

`git add .`

- Добавить только уже отслеживаемые файлы:

`git add {{[-u|--update]}}`

- Добавить игнорируемый файл:

`git add {{[-f|--force]}} {{путь/к/файлу}}`

- Интерактивно подготовить части файлов для коммита:

`git add {{[-p|--patch]}}`

- Интерактивно подготовить части указанного файла для коммита:

`git add {{[-p|--patch]}} {{путь/к/файлу}}`

- Интерактивно подготовить файл к коммиту:

`git add {{[-i|--interactive]}}`
