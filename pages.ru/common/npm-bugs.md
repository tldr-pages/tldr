# npm bugs

> Сообщать об ошибках пакета в веб-браузере.
> Пытаться открыть URL баг-трекера пакета или адрес электронной почты поддержки.
> Больше информации: <https://docs.npmjs.com/cli/npm-bugs/>.

- Сообщить об ошибках конкретного пакета, открыв его баг-трекер:

`npm bugs {{пакет}}`

- Открыть баг-трекер текущего пакета, используя имя из `package.json`:

`npm bugs`

- Настроить браузер для открытия URL, установив предпочтительный браузер для команд `npm`:

`npm {{[c|config]}} set browser {{имя_браузера}}`

- Управлять открытием URL: установить `browser` в `true` для системного обработчика URL или `false` для вывода URL в терминал:

`npm {{[c|config]}} set browser {{true|false}}`
