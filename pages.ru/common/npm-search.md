# npm search

> Искать пакеты в реестре `npm`.
> Больше информации: <https://docs.npmjs.com/cli/npm-search/>.

- Найти пакет по имени:

`npm {{[s|search]}} {{пакет}}`

- Найти пакеты по ключевому слову:

`npm {{[s|search]}} {{ключевое_слово}}`

- Найти пакеты с подробной информацией (описание, автор, версия и т.д.):

`npm {{[s|search]}} {{пакет}} --long`

- Найти пакеты конкретного автора:

`npm {{[s|search]}} --author {{автор}}`

- Найти пакеты в определённой области видимости:

`npm {{[s|search]}} --scope {{организация}}`

- Найти пакеты по комбинации терминов:

`npm {{[s|search]}} {{термин1 термин2 ...}}`
