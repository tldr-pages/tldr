# ls

> Выводить содержимое каталога.
> Больше информации: <https://www.gnu.org/software/coreutils/manual/html_node/ls-invocation.html>.

- Вывести список файлов по одному в строке:

`ls -1`

- Вывести список всех файлов, включая скрытые:

`ls {{[-a|--all]}}`

- Вывести список файлов с добавлением символа в конце для указания типа файла (каталог/, символическая_ссылка@, исполняемый_файл*, ...):

`ls {{[-F|--classify]}}`

- Вывести подробный [l] список всех файлов (включая скрытые) с правами доступа, владельцами, размером и датой изменения:

`ls {{[-la|-l --all]}}`

- Вывести подробный [l] список с размерами файлов в удобочитаемых единицах (КиБ, МиБ, ГиБ):

`ls {{[-lh|-l --human-readable]}}`

- Вывести подробный [l] список, отсортированный по размеру [S] (по убыванию) рекурсивно:

`ls {{[-lSR|-lS --recursive]}}`

- Вывести подробный [l] список, отсортированный по дате [t] изменения (сначала более старые):

`ls {{[-ltr|-lt --reverse]}}`

- Вывести список только каталогов:

`ls {{[-d|--directory]}} */`
