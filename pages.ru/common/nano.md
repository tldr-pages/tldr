# nano

> Текстовый редактор. Улучшенный клон `pico`.
> Смотрите также: `pico`, `rnano`.
> Больше информации: <https://nano-editor.org/dist/latest/nano.html>.

- Открыть указанные файлы, переходя к следующему после закрытия предыдущего:

`nano {{путь/к/файлу1 путь/к/файлу2 ...}}`

- Запустить редактор без использования файлов конфигурации:

`nano {{[-I|--ignorercfiles]}}`

- Открыть файл и переместить курсор на указанную строку и колонку:

`nano +{{строка}},{{колонка}} {{путь/к/файлу}}`

- Открыть файл с включённым мягким переносом строк:

`nano {{[-S|--softwrap]}} {{путь/к/файлу}}`

- Открыть файл и сделать отступы в новых строках по уровню предыдущей:

`nano {{[-i|--autoindent]}} {{путь/к/файлу}}`

- Открыть файл и создать резервную копию (`путь/к/файлу~`) при сохранении:

`nano {{[-B|--backup]}} {{путь/к/файлу}}`

- Открыть файл в ограниченном режиме (т.е. не читать и не записывать файлы, не указанные в командной строке):

`nano {{[-R|--restricted]}} {{путь/к/файлу}}`

- Выйти из nano:

`<Ctrl x>`
