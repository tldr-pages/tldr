# 2to3

> Автоматически преобразовывать код Python 2 в Python 3.
> Этот модуль объявлен устаревшим в версии 3.11 и удалён в версии 3.13.
> Для справки смотрите: <https://github.com/python/cpython/blob/8d42e2d915c3096e7eac1c649751d1da567bb7c3/Doc/whatsnew/3.13.rst?plain=1#L188>.
> Больше информации: <https://manned.org/2to3>.

- Показать изменения без их применения (пробный запуск):

`2to3 {{путь/к/файлу.py}}`

- Конвертировать файл Python 2 в Python 3:

`2to3 {{[-w|--write]}} {{путь/к/файлу.py}}`

- Конвертировать определённые языковые особенности Python 2 в Python 3:

`2to3 {{[-w|--write]}} {{путь/к/файлу.py}} {{[-f|--fix]}} {{raw_input}} {{[-f|--fix]}} {{print}}`

- Конвертировать все языковые особенности Python 2 в Python 3, кроме указанных:

`2to3 {{[-w|--write]}} {{путь/к/файлу.py}} {{[-x|--nofix]}} {{has_key}} {{[-x|--nofix]}} {{isinstance}}`

- Вывести список всех доступных языковых особенностей для преобразования из Python 2 в Python 3:

`2to3 {{[-l|--list-fixes]}}`

- Конвертировать все файлы Python 2 в каталоге в Python 3:

`2to3 {{[-o|--output-dir]}} {{путь/к/каталогу_python3}} {{[-W|--write-unchanged-files]}} {{[-n|--nobackups]}} {{путь/к/каталогу_python2}}`

- Запустить 2to3 с несколькими потоками:

`2to3 {{[-j|--processes]}} {{4}} {{[-o|--output-dir]}} {{путь/к/каталогу_python3}} {{[-w|--write]}} {{[-n|--nobackups]}} --no-diffs {{путь/к/каталогу_python2}}`
