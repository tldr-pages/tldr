# aapt

> Android Asset Packaging Tool: компилировать и упаковывать ресурсы приложений Android.
> Больше информации: <https://manned.org/aapt>.

- Вывести список файлов, содержащихся в APK-архиве:

`aapt list {{путь/к/приложению}}.apk`

- Отобразить метаданные приложения (версия, разрешения и т.д.):

`aapt dump badging {{путь/к/приложению}}.apk`

- Создать новый APK-архив с файлами из указанного каталога:

`aapt package -F {{путь/к/приложению}}.apk {{путь/к/каталогу}}`
