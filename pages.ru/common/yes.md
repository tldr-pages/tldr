# yes

> Непрерывно выводить что-либо.
> Эта команда обычно используется для автоматического подтверждения всех запросов команд установки (например, `apt-get`).
> Больше информации: <https://www.gnu.org/software/coreutils/manual/html_node/yes-invocation.html>.

- Запустить непрерывный вывод `y`:

`yes`

- Запустить непрерывный вывод указанного значения:

`yes {{значение}}`

- Подтвердить все запросы команды `apt-get`:

`yes | sudo apt-get install {{программа}}`

- Запустить непрерывный вывод символа новой строки, чтобы всегда принимать параметр по умолчанию:

`yes ''`
