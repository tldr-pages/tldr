# uv export

> Экспортировать файл блокировки проекта в альтернативный формат.
> Больше информации: <https://docs.astral.sh/uv/reference/cli/#uv-export>.

- Экспортировать зависимости в файл `requirements.txt`:

`uv export --format requirements-txt {{[-o|--output-file]}} {{requirements.txt}}`

- Экспортировать зависимости в формат `pylock.toml`:

`uv export --format pylock.toml`

- Экспортировать только рабочие зависимости (без зависимостей разработки):

`uv export --no-dev`

- Экспортировать с определённой необязательной группой зависимостей:

`uv export --extra {{имя_группы}}`

- Экспортировать со всеми необязательными зависимостями:

`uv export --all-extras`

- Экспортировать с определённой группой зависимостей:

`uv export --group {{имя_группы}}`

- Экспортировать без хешей:

`uv export --no-hashes`

- Экспортировать зависимости для определённого пакета в рабочем пространстве:

`uv export --package {{имя_пакета}}`
