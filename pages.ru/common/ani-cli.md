# ani-cli

> Искать и смотреть аниме.
> Смотрите также: `animdl`.
> Больше информации: <https://manned.org/ani-cli>.

- Искать аниме по названию:

`ani-cli "{{название_аниме}}"`

- Скачать эпизод:

`ani-cli {{[-d|--download]}} "{{название_аниме}}"`

- Скачать диапазон эпизодов:

`ani-cli {{[-d|--download]}} {{[-r|--range]}} "{{1 6}}" "{{название_аниме}}"`

- Скачать весь сериал (диапазон всех эпизодов):

`ani-cli {{[-d|--download]}} {{[-r|--range]}} "1 -1" "{{название_аниме}}"`

- Использовать VLC в качестве медиаплеера:

`ani-cli {{[-v|--vlc]}} "{{название_аниме}}"`

- Смотреть определённый эпизод:

`ani-cli {{[-e|--episode]}} {{номер_эпизода}} "{{название_аниме}}"`

- Продолжить просмотр аниме из истории:

`ani-cli {{[-c|--continue]}}`

- Обновить `ani-cli`:

`ani-cli {{[-U|--update]}}`
