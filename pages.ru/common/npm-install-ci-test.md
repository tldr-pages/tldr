# npm install-ci-test

> Установить зависимости проекта и запустить тесты одной командой, в основном используется в средах непрерывной интеграции (CI).
> Эквивалент выполнения `npm ci && npm test`.
> Больше информации: <https://docs.npmjs.com/cli/install-ci-test/>.

- Установить зависимости и запустить тесты:

`npm {{[cit|install-ci-test]}}`

- Отобразить подробные логи во время установки и тестирования:

`npm {{[cit|install-ci-test]}} --verbose`
