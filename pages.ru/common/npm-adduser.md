# npm adduser

> Добавлять учётную запись пользователя реестра.
> Больше информации: <https://docs.npmjs.com/cli/npm-adduser/>.

- Создать нового пользователя в указанном реестре и сохранить учётные данные в `.npmrc`:

`npm adduser --registry {{url_реестра}}`

- Войти в приватный реестр с определённой областью видимости:

`npm login --scope {{@организация}} --registry {{https://registry.example.com}}`

- Выйти из определённой области видимости и удалить токен аутентификации:

`npm logout --scope {{@организация}}`

- Создать пакет с областью видимости при инициализации:

`npm init --scope {{@организация}} {{[-y|--yes]}}`
