# npm login

> Входить в учётную запись пользователя реестра.
> Смотрите также: `npm logout`.
> Больше информации: <https://docs.npmjs.com/cli/npm-login/>.

- Войти в учётную запись пользователя реестра и сохранить учётные данные в `.npmrc`:

`npm login`

- Войти с использованием пользовательского реестра:

`npm login --registry {{url_реестра}}`

- Войти с использованием определённого типа аутентификации:

`npm login --auth-type {{legacy|web}}`
