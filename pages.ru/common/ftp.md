# ftp

> Утилиты для взаимодействия с сервером по протоколу передачи файлов (FTP).
> Больше информации: <https://manned.org/ftp>.

- Подключиться к FTP-серверу и запустить в интерактивном режиме:

`ftp {{ftp.example.com}}`

- Подключиться к FTP-серверу, указав IP-адрес и порт:

`ftp {{ip_адрес}} {{порт}}`

- [Интерактивно] Переключиться в бинарный режим передачи (для изображений, архивов и т.д.):

`binary`

- [Интерактивно] Передать несколько файлов без запроса подтверждения для каждого файла:

`prompt off`

- [Интерактивно] Скачать несколько файлов (glob-выражение):

`mget {{*.png}}`

- [Интерактивно] Загрузить несколько файлов (glob-выражение):

`mput {{*.zip}}`

- [Интерактивно] Удалить несколько файлов на удаленном сервере:

`mdelete {{*.txt}}`

- [Интерактивно] Переименовать файл на удаленном сервере:

`rename {{исходное_имя}} {{новое_имя}}`
