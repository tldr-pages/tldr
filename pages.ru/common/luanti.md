# luanti

> Блочная игра-песочница с бесконечным миром.
> Больше информации: <https://manned.org/luanti>.

- Запустить Luanti в режиме клиента:

`luanti`

- Вывести список загруженных игровых режимов:

`luanti --gameid list`

- Запустить Luanti в режиме сервера с определённым игровым режимом:

`luanti --server --gameid {{id_игры}}`

- Запустить сервер с миром по умолчанию (после того, как он был создан):

`luanti --server`

- Запустить сервер с указанным миром:

`luanti --server --world {{название_мира}}`

- Запустить сервер на указанном порту:

`luanti --server --port {{порт}}`

- Записывать логи в указанный файл:

`luanti --logfile {{путь/к/файлу}}`

- Выводить в консоль только ошибки:

`luanti --quiet`
