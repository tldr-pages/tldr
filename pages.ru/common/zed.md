# zed

> Быстрый, эффективный и удобный текстовый редактор.
> Больше информации: <https://zed.dev/docs/reference/cli>.

- Открыть указанные пути в Zed:

`zed {{путь/к/файлу_или_каталогу1 путь/к/файлу_или_каталогу2 ...}}`

- Открыть путь на переднем плане и отобразить логи:

`zed {{путь/к/проекту}} --foreground`

- Открыть путь в новом окне:

`zed {{путь/к/проекту}} {{[-n|--new]}}`

- Открыть файл на указанной строке и столбце:

`zed {{путь/к/файлу}}:{{номер_строки}}:{{номер_столбца}}`

- Открыть вкладку сравнения в Zed для двух версий файла:

`zed --diff {{путь/к/старому_файлу}} {{путь/к/новому_файлу}}`
