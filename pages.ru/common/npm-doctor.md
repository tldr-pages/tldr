# npm doctor

> Проверять состояние среды npm.
> Больше информации: <https://docs.npmjs.com/cli/npm-doctor/>.

- Запустить все стандартные проверки состояния `npm`:

`npm doctor`

- Проверить соединение с реестром `npm`:

`npm doctor connection`

- Проверить версии используемых Node.js и `npm`:

`npm doctor versions`

- Проверить проблемы с правами доступа к каталогам и кэшу `npm`:

`npm doctor permissions`

- Проверить кэшированные файлы пакетов и контрольные суммы:

`npm doctor cache`
