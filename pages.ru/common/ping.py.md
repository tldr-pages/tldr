# ping.py

> Проверять доступность хоста IPv4 с помощью ICMP.
> Отправляет эхо-запросы ICMP и ожидает эхо-ответы.
> Примечание: требуются права суперпользователя (root) для доступа к сырым сокетам (например, запускайте с `sudo`).
> Часть набора инструментов Impacket.
> Больше информации: <https://github.com/fortra/impacket>.

- Пропинговать хост с указанного исходного IPv4-адреса:

`ping.py {{исходный_ipv4}} {{целевой_ipv4}}`

- Пропинговать 192.168.1.100 с адреса 192.168.1.10:

`ping.py 192.168.1.10 192.168.1.100`
