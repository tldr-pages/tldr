# npm install-test

> Эквивалент последовательного выполнения `npm install` и `npm test`.
> Примечание: `it` можно использовать как сокращение для `install-test`.
> Больше информации: <https://docs.npmjs.com/cli/npm-install-test/>.

- Установить все зависимости и затем запустить тесты:

`npm {{[it|install-test]}}`

- Установить конкретный пакет и затем запустить тесты:

`npm {{[it|install-test]}} {{пакет}}`

- Установить пакет и сохранить его как зависимость перед запуском тестов:

`npm {{[it|install-test]}} {{пакет}} {{[-S|--save]}}`

- Установить зависимости глобально и затем запустить тесты:

`npm {{[it|install-test]}} {{[-g|--global]}}`
