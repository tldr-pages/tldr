# adb install

> Устанавливать пакеты на эмулятор Android или подключённое устройство Android.
> Больше информации: <https://developer.android.com/tools/adb>.

- Установить приложение Android на эмулятор/устройство:

`adb install {{путь/к/файлу}}.apk`

- Установить приложение Android на конкретный эмулятор/устройство (переопределяет `$ANDROID_SERIAL`):

`adb -s {{серийный_номер}} install {{путь/к/файлу}}.apk`

- Переустановить ([r]einstall) существующее приложение, сохранив его данные:

`adb install -r {{путь/к/файлу}}.apk`

- Установить приложение Android, разрешив понижение ([d]owngrade) версии (только для отлаживаемых пакетов):

`adb install -d {{путь/к/файлу}}.apk`

- Предоставить ([g]rant) все разрешения, перечисленные в манифесте приложения:

`adb install -g {{путь/к/файлу}}.apk`

- Быстро обновить установленный пакет, обновляя только изменившиеся части APK:

`adb install --fastdeploy {{путь/к/файлу}}.apk`
