# git init

> Инициализировать новый локальный репозиторий Git.
> Больше информации: <https://git-scm.com/docs/git-init>.

- Инициализировать новый локальный репозиторий:

`git init`

- Инициализировать репозиторий с указанным именем для начальной ветки:

`git init {{[-b|--initial-branch]}} {{имя_ветки}}`

- Инициализировать репозиторий, используя SHA256 для хеширования объектов (требуется Git версии 2.29+):

`git init --object-format sha256`

- Инициализировать bare-репозиторий, подходящий для использования в качестве удалённого через SSH:

`git init --bare`
