# npm exec

> Выполнять исполняемые файлы из пакетов `npm`.
> Больше информации: <https://docs.npmjs.com/cli/npm-exec/>.

- Выполнить команду из локального или удалённого пакета `npm`:

`npm {{[x|exec]}} {{команда}} {{аргумент1 аргумент2 ...}}`

- Явно указать пакет (полезно, если несколько команд имеют одинаковое имя):

`npm {{[x|exec]}} --package {{пакет}} {{команда}}`

- Запустить команду, если она существует в текущем пути или в `node_modules/.bin`:

`npm {{[x|exec]}} --no-install {{команда}} {{аргумент1 аргумент2 ...}}`

- Выполнить команду, подавляя любой вывод самого `npm`:

`npm {{[x|exec]}} --quiet {{команда}} {{аргумент1 аргумент2 ...}}`

- Показать справку:

`npm {{[x|exec]}} --help`
