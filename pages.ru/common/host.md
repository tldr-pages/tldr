# host

> Искать информацию на сервере доменных имён.
> Смотрите также: `dig`, `resolvectl`, `nslookup`.
> Больше информации: <https://manned.org/host>.

- Найти A-, AAAA- и MX-записи домена:

`host {{домен}}`

- Найти определённое поле (CNAME, TXT, ...) домена:

`host -t {{поле}} {{домен}}`

- Выполнить обратный поиск IP-адреса:

`host {{ip_адрес}}`

- Указать альтернативный DNS-сервер для запроса:

`host {{домен}} {{8.8.8.8}}`
