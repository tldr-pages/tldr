# npm owner

> Управлять владельцами опубликованных пакетов.
> Больше информации: <https://docs.npmjs.com/cli/npm-owner/>.

- Добавить нового пользователя в качестве владельца пакета:

`npm owner add {{имя_пользователя}} {{название_пакета}}`

- Удалить пользователя из списка владельцев пакета:

`npm owner rm {{имя_пользователя}} {{название_пакета}}`

- Вывести список всех владельцев пакета:

`npm owner ls {{название_пакета}}`
