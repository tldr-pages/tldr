# bun remove

> Удалять зависимости из файла `package.json`.
> Примечание: `rm` можно использовать как сокращение для `remove`.
> Больше информации: <https://bun.com/docs/pm/cli/remove>.

- Удалить зависимость:

`bun remove {{имя_пакета}}`

- Удалить несколько зависимостей:

`bun remove {{имя_пакета1 имя_пакета2 ...}}`

- Удалить глобально установленный пакет:

`bun remove {{[-g|--global]}} {{имя_пакета}}`

- Удалить зависимость без изменения файла `package.json`:

`bun remove --no-save {{имя_пакета}}`

- Выполнить пробный запуск без фактического удаления пакетов (симулировать удаление):

`bun remove --dry-run {{имя_пакета}}`
