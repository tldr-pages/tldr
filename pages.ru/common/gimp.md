# gimp

> Программа для редактирования изображений GNU.
> Смотрите также: `krita`.
> Больше информации: <https://docs.gimp.org/en/gimp-fire-up.html#gimp-concepts-running-command-line>.

- Запустить GIMP:

`gimp`

- Открыть указанные файлы:

`gimp {{путь/к/изображению1 путь/к/изображению2 ...}}`

- Открыть указанные файлы в новом окне:

`gimp --new-instance {{путь/к/изображению1 путь/к/изображению2 ...}}`

- Запустить без заставки:

`gimp --no-splash`

- Выводить ошибки и предупреждения в консоль вместо отображения их в диалоговом окне:

`gimp --console-messages`

- Включить обработчики сигналов отладки:

`gimp --debug-handlers`
