# kubectl attach

> Подключаться к процессу, уже работающему внутри существующего контейнера.
> Больше информации: <https://kubernetes.io/docs/reference/kubectl/generated/kubectl_attach/>.

- Получить вывод из пода:

`kubectl attach {{имя_пода}}`

- Получить вывод из контейнера в указанном поде:

`kubectl attach {{имя_пода}} {{[-c|--container]}} {{имя_контейнера}}`

- Получить вывод из первого пода набора реплик:

`kubectl attach {{[rs|replicaset]}}/{{имя_набора_реплик}}`

- Создать интерактивную сессию с подом:

`kubectl attach {{имя_пода}} {{[-it|--stdin --tty]}}`

- Создать интерактивную сессию с определённым контейнером пода:

`kubectl attach {{имя_пода}} {{[-c|--container]}} {{имя_контейнера}} {{[-it|--stdin --tty]}}`
