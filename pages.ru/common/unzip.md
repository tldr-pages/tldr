# unzip

> Извлекать файлы и каталоги из Zip-архивов.
> Смотрите также: `zip`.
> Больше информации: <https://manned.org/unzip>.

- Распаковать все файлы/каталоги из архивов в текущий каталог:

`unzip {{путь/к/архиву1.zip путь/к/архиву2.zip ...}}`

- Распаковать файлы/каталоги из архивов в указанный каталог:

`unzip {{путь/к/архиву1.zip путь/к/архиву2.zip ...}} -d {{путь/к/каталогу}}`

- Распаковать файлы/каталоги в `stdout` вместе с именами файлов:

`unzip -c {{путь/к/архиву1.zip путь/к/архиву2.zip ...}}`

- Распаковать архив, созданный в Windows, с не-ASCII именами файлов (например, с китайскими или японскими иероглифами):

`unzip -O {{gbk}} {{путь/к/архиву1.zip путь/к/архиву2.zip ...}}`

- Вывести содержимое конкретного архива без распаковки:

`unzip -l {{путь/к/архиву}}.zip`

- Извлечь конкретные файлы из архива:

`unzip -j {{путь/к/архиву}}.zip {{путь/к/файлу1_в_архиве путь/к/файлу2_в_архиве ...}}`
