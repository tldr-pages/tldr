# unzip

> Извлекать файлы и директории из Zip-архивов.
> Смотрите также: `zip`.
> Больше информации: <https://manned.org/unzip>.

- Распаковать все файлы/директории из архивов в текущую директорию:

`unzip {{путь/к/архиву1.zip путь/к/архиву2.zip ...}}`

- Распаковать файлы/директории из архивов в указанную директорию:

`unzip {{путь/к/архиву1.zip путь/к/архиву2.zip ...}} -d {{путь/к/директории}}`

- Распаковать файлы/директории в `stdout` вместе с именами файлов:

`unzip -c {{путь/к/архиву1.zip путь/к/архиву2.zip ...}}`

- Распаковать архив, созданный в Windows, с не-ASCII именами файлов (например, с Китайскими иероглифами):

`unzip -O {{gbk}} {{путь/к/архиву1.zip путь/к/архиву2.zip ...}}`

- Вывести содержимое конкретного архива без распаковки:

`unzip -l {{путь/к/архиву.zip}}`

- Извлечь конкретный файл из архива:

`unzip -j {{путь/к/архиву.zip}} {{путь/к/файлу1_в_архиве путь/к/файлу2_в_архиве ...}}`
