# dirname

> Удалять завершающую часть имени файла из пути.
> Больше информации: <https://www.gnu.org/software/coreutils/manual/html_node/dirname-invocation.html>.

- Определить родительский каталог заданного пути:

`dirname {{путь/к/файлу_или_каталогу}}`

- Определить родительский каталог для нескольких путей:

`dirname {{путь/к/файлу_или_каталогу1 путь/к/файлу_или_каталогу2 ...}}`

- Использовать символ NUL вместо символа новой строки в качестве разделителя вывода (полезно при использовании с `xargs`):

`dirname {{[-z|--zero]}} {{путь/к/файлу_или_каталогу1 путь/к/файлу_или_каталогу2 ...}}`
