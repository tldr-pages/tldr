# npm pack

> Создавать tarball из пакета.
> Больше информации: <https://docs.npmjs.com/cli/pack/>.

- Создать tarball из текущего пакета в текущем каталоге:

`npm pack`

- Создать tarball из определённого каталога пакета:

`npm pack {{путь/к/каталогу_пакета}}`

- Выполнить пробный запуск для предварительного просмотра содержимого tarball без его создания:

`npm pack --dry-run`

- Создать tarball без выполнения скриптов жизненного цикла:

`npm pack --ignore-scripts`

- Указать пользовательский реестр для получения метаданных пакета:

`npm pack --registry {{https://registry.npmjs.org/}}`
