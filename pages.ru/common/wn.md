# wn

> Искать связанные слова в базе данных WordNet.
> Больше информации: <https://wordnet.princeton.edu/documentation/wn1wn>.

- Показать доступную информацию для слова "emphatically":

`wn emphatically`

- Вывести список синонимов наречия "emphatically":

`wn emphatically -synsr`

- Вывести список антонимов прилагательного "slow":

`wn slow -antsa`
