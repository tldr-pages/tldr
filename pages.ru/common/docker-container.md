# docker container

> Управлять контейнерами Docker.
> Больше информации: <https://docs.docker.com/reference/cli/docker/container/>.

- Вывести список запущенных контейнеров Docker:

`docker {{[ps|container ls]}}`

- Запустить один или несколько остановленных контейнеров:

`docker {{[start|container start]}} {{имя_контейнера1}} {{имя_контейнера2}}`

- Принудительно остановить (kill) один или несколько запущенных контейнеров:

`docker {{[kill|container kill]}} {{имя_контейнера}}`

- Остановить один или несколько запущенных контейнеров:

`docker {{[stop|container stop]}} {{имя_контейнера}}`

- Приостановить все процессы в одном или нескольких контейнерах:

`docker {{[pause|container pause]}} {{имя_контейнера}}`

- Отобразить подробную информацию об одном или нескольких контейнерах:

`docker container inspect {{имя_контейнера}}`

- Экспортировать файловую систему контейнера в архив `.tar`:

`docker {{[export|container export]}} {{имя_контейнера}}`

- Создать новый образ из изменений контейнера:

`docker {{[commit|container commit]}} {{имя_контейнера}}`
