# hx

> Helix, постмодерновый текстовый редактор, предоставляет несколько режимов для различных видов манипулирования текстом.
> Нажатие `<i>` входит в режим вставки. `<Esc>` входит в нормальный режим, который позволяет использовать команды Helix.
> Больше информации: <https://manned.org/man/debian-forky/hx>.

- Открыть файл:

`hx {{путь/к/файлу}}`

- Открыть файлы и отобразить их рядом друг с другом:

`hx --vsplit {{путь/к/файлу1 путь/к/файлу2 ...}}`

- Показать руководство для изучения Helix (или открыть его внутри Helix, нажав `<Esc>` и введя `<:>tutor<Enter>`):

`hx --tutor`

- Изменить тему Helix:

`<:>theme {{название_темы}}`

- Сохранить и выйти:

`<:>wq<Enter>`

- Принудительно выйти без сохранения:

`<:>q!<Enter>`

- Отменить последнюю операцию:

`<u>`

- Найти шаблон в файле (нажмите `<n>`/`<N>` для перехода к следующему/предыдущему совпадению):

`</>{{шаблон_поиска}}<Enter>`
