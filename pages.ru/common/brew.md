# brew

> Homebrew - менеджер пакетов для macOS и Linux.
> Некоторые подкоманды, такие как `install`, имеют собственную документацию по использованию.
> Больше информации: <https://docs.brew.sh/Manpage>.

- Установить последнюю стабильную версию формулы или каска (cask):

`brew install {{формула|каск}}`

- Вывести список всех установленных формул и касков:

`brew list`

- Обновить установленную формулу или каск (если аргумент не указан, будут обновлены все установленные формулы/каски):

`brew upgrade {{формула|каск}}`

- Загрузить свежую версию Homebrew и сведения о всех доступных формулах и касках из репозитория Homebrew:

`brew update`

- Показать формулы и каски, для которых доступны более новые версии:

`brew outdated`

- Найти доступные формулы (пакеты) и каски (нативные пакеты `.app` для macOS):

`brew search {{текст}}`

- Отобразить информацию о формуле или каске (версия, путь установки, зависимости и т.д.):

`brew info {{формула|каск}}`

- Проверить локальную установку Homebrew на наличие потенциальных проблем:

`brew doctor`
