# npm docs

> Открывать документацию для одного или нескольких пакетов в веб-браузере по умолчанию.
> Больше информации: <https://docs.npmjs.com/cli/npm-docs/>.

- Открыть страницу документации конкретного пакета:

`npm docs {{пакет}}`

- Открыть документацию для нескольких пакетов сразу:

`npm docs {{пакет1 пакет2 ...}}`
