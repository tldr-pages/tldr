# sort

> Сортировать строки текстовых файлов.
> Больше информации: <https://www.gnu.org/software/coreutils/manual/html_node/sort-invocation.html>.

- Отсортировать файл по возрастанию:

`sort {{путь/к/файлу}}`

- Отсортировать файл по убыванию:

`sort {{[-r|--reverse]}} {{путь/к/файлу}}`

- Отсортировать файл без учёта регистра:

`sort {{[-f|--ignore-case]}} {{путь/к/файлу}}`

- Отсортировать файл по числовому, а не алфавитному порядку:

`sort {{[-n|--numeric-sort]}} {{путь/к/файлу}}`

- Отсортировать `/etc/passwd` по 3-му полю и далее по числовому значению, используя `:` как разделитель полей:

`sort {{[-t|--field-separator]}} : {{[-k|--key]}} 3n /etc/passwd`

- То же, но при равенстве 3-го поля отсортировать по 4-му полю с числами в экспоненциальной записи:

`sort {{[-t|--field-separator]}} : {{[-k|--key]}} 3,3n {{[-k|--key]}} 4,4g /etc/passwd`

- Отсортировать файл, сохранив только уникальные строки:

`sort {{[-u|--unique]}} {{путь/к/файлу}}`

- Отсортировать файл, записав результат в указанный файл (можно использовать для сортировки файла на месте):

`sort {{[-o|--output]}} {{путь/к/выходному_файлу}} {{путь/к/входному_файлу}}`
