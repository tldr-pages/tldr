# let

> Вычислять арифметические выражения в оболочке.
> Больше информации: <https://www.gnu.org/software/bash/manual/bash.html#index-let>.

- Вычислить простое арифметическое выражение:

`let "{{result = a + b}}"`

- Использовать пост-инкремент и присваивание в выражении:

`let "{{x++}}"`

- Использовать условный оператор в выражении:

`let "{{result = (x > 10) ? x : 0}}"`

- Показать справку:

`let --help`
