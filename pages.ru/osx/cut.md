# cut

> Вырезать поля из стандартного ввода или файлов.
> Больше информации: <https://keith.github.io/xcode-man-pages/cut.1.html>.

- Вывести пятый символ [c] каждой строки:

`{{команда}} | cut -c 5`

- Вывести с 5-го по 10-й символ [c] каждой строки указанного файла:

`cut -c 5-10 {{путь/к/файлу}}`

- Разбить каждую строку в файле по разделителю на поля [f] и вывести 2-е и 6-е поле (разделитель по умолчанию — `TAB`):

`cut -f 2,6 {{путь/к/файлу}}`

- Разбить каждую строку по указанному разделителю [d] и вывести всё, начиная со 2-го поля [f]:

`{{команда}} | cut -d "{{разделитель}}" -f 2-`

- Использовать пробел в качестве разделителя [d] и вывести только первые 3 поля [f]:

`{{команда}} | cut -d " " -f -3`
