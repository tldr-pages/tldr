# yaa

> Создавать YAA-архивы и управлять ими.
> Больше информации: <https://www.manpagez.com/man/1/yaa/>.

- Создать архив из каталога:

`yaa archive -d {{путь/к/каталогу}} -o {{путь/к/выходному_файлу.yaa}}`

- Создать архив из файла:

`yaa archive -i {{путь/к/файлу}} -o {{путь/к/выходному_файлу.yaa}}`

- Извлечь архив в текущий каталог:

`yaa extract -i {{путь/к/файлу_архива.yaa}}`

- Вывести список содержимого архива:

`yaa list -i {{путь/к/файлу_архива.yaa}}`

- Создать архив с определённым алгоритмом сжатия:

`yaa archive -a {{алгоритм}} -d {{путь/к/каталогу}} -o {{путь/к/выходному_файлу.yaa}}`

- Создать архив с размером блока 8 МБ:

`yaa archive -b 8m -d {{путь/к/каталогу}} -o {{путь/к/выходному_файлу.yaa}}`
