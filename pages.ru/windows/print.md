# print

> Вывод на печать текстового файла.
> Больше информации: <https://learn.microsoft.com/windows-server/administration/windows-commands/print>.

- Выполнить печать текстового файла исползуя локальный принтер:

`print {{path\to\file}}`

- Выполнить печать текстового файла исползуя сетевой принтер:

`print /d:{{printer}} {{path\to\file}}`
