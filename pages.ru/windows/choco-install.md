# choco install

> Устанавливать один или несколько пакетов с помощью Chocolatey.
> Больше информации: <https://docs.chocolatey.org/en-us/choco/commands/install/>.

- Установить один или несколько пакетов:

`choco install {{пакет1 пакет2 ...}}`

- Установить пакеты из пользовательского файла конфигурации:

`choco install {{путь\к\файлу_пакетов.config}}`

- Установить определённый файл `.nuspec` или `.nupkg`:

`choco install {{путь\к\файлу}}`

- Установить определённую версию пакета:

`choco install {{пакет}} --version {{версия}}`

- Разрешить установку нескольких версий пакета:

`choco install {{пакет}} --allow-multiple`

- Автоматически подтвердить все запросы:

`choco install {{пакет}} --yes`

- Указать пользовательский источник для получения пакетов:

`choco install {{пакет}} --source {{url_источника|псевдоним}}`

- Указать имя пользователя и пароль для аутентификации:

`choco install {{пакет}} --user {{имя_пользователя}} --password {{пароль}}`
