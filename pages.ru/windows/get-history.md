# Get-History

> Отображать историю команд PowerShell.
> Примечание: эта команда доступна только в PowerShell.
> Больше информации: <https://learn.microsoft.com/powershell/module/microsoft.powershell.core/get-history>.

- Показать историю команд с их ID:

`Get-History`

- Получить запись истории по ID:

`Get-History -Id {{id}}`

- Показать последние `n` команд:

`Get-History -Count {{n}}`
