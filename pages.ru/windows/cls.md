# cls

> Очищать экран.
> В PowerShell эта команда является псевдонимом для `Clear-Host`. Данная документация основана на версии `cls` для командной строки (`cmd`).
> Больше информации: <https://learn.microsoft.com/windows-server/administration/windows-commands/cls>.

- Посмотреть документацию для эквивалентной команды PowerShell:

`tldr clear-host`

- Очистить экран:

`cls`
