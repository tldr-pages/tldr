# cmd

> Интерпретатор командной строки Windows.
> Больше информации: <https://learn.microsoft.com/windows-server/administration/windows-commands/cmd>.

- Запустить интерактивную сессию командной оболочки:

`cmd`

- Выполнить определённые команды ([c]ommands):

`cmd /c {{echo Привет, мир}}`

- Запустить указанный скрипт:

`cmd {{путь\к\скрипту.bat}}`

- Выполнить определённые команды, а затем перейти в интерактивный режим:

`cmd /k {{echo Привет, мир}}`

- Запустить интерактивную оболочку с отключённым выводом команд (`echo` выключен):

`cmd /q`

- Запустить интерактивную оболочку с включённым или выключенным отложенным расширением переменных ([v]ariable expansion):

`cmd /v:{{on|off}}`

- Запустить интерактивную оболочку с включёнными или выключенными расширениями ([e]xtensions) команд:

`cmd /e:{{on|off}}`

- Запустить интерактивную оболочку с использованием кодировки [u]nicode:

`cmd /u`
