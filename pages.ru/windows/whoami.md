# whoami

> Отображать сведения о текущем пользователе.
> Больше информации: <https://learn.microsoft.com/windows-server/administration/windows-commands/whoami>.

- Отобразить имя текущего пользователя:

`whoami`

- Отобразить список групп, в которых состоит текущий пользователь:

`whoami /groups`

- Отобразить привилегии текущего пользователя:

`whoami /priv`

- Отобразить имя субъекта-пользователя (UPN) для текущего пользователя:

`whoami /upn`

- Отобразить идентификатор входа (logon ID) текущего пользователя:

`whoami /logonid`

- Отобразить всю доступную информацию о текущем пользователе:

`whoami /all`
