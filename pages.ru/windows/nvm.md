# nvm

> Устанавливать, удалять и переключаться между версиями Node.js.
> Поддерживает номера версий вроде "12.8" или "v16.13.1", метки вроде "stable", "system" и т.д.
> Больше информации: <https://github.com/coreybutler/nvm-windows>.

- Установить заданную версию Node.js:

`nvm install {{версия_node}}`

- Задать версию Node.js по умолчанию (требуется запускать от имени администратора):

`nvm use {{версия_node}}`

- Вывести список всех установленных версий Node.js и подсветить версию по умолчанию:

`nvm list`

- Вывести список всех доступных для установки (удаленных) версий Node.js:

`nvm ls-remote`

- Удалить указанную версию Node.js:

`nvm uninstall {{версия_node}}`
