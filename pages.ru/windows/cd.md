# cd

> Отображать текущий рабочий каталог или переходить в другой.
> В PowerShell эта команда является псевдонимом для `Set-Location`. Эта документация основана на версии для командной строки (`cmd`).
> Больше информации: <https://learn.microsoft.com/windows-server/administration/windows-commands/cd>.

- Посмотреть документацию для эквивалентной команды PowerShell:

`tldr set-location`

- Вывести путь к текущему каталогу:

`cd`

- Перейти в указанный каталог на текущем диске:

`cd {{путь\к\каталогу}}`

- Перейти в указанный каталог на другом диске ([d]rive):

`cd /d {{C}}:{{путь\к\каталогу}}`

- Перейти вверх в родительский каталог:

`cd ..`

- Перейти в домашний каталог текущего пользователя:

`cd %userprofile%`

- Перейти в корневой каталог текущего диска:

`cd \`
