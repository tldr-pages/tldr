# exit

> Выходить из текущего экземпляра командной строки (CMD) или текущего пакетного файла.
> Больше информации: <https://learn.microsoft.com/windows-server/administration/windows-commands/exit>.

- Выйти из текущего экземпляра командной строки (CMD):

`exit`

- Выйти из текущего пакетного ([b]atch) файла:

`exit /b`

- Выйти с указанным кодом завершения:

`exit {{код_завершения}}`
