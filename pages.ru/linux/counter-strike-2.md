# Counter Strike 2

> Запускать выделенный сервер Counter Strike 2.
> Больше информации: <https://developer.valvesoftware.com/wiki/Counter-Strike_2/Dedicated_Servers>.

- Запустить игру с одной картой:

`{{путь/к}}/cs2 -dedicated +map {{de_dust2}}`

- Запустить игру с указанным максимальным количеством игроков:

`{{путь/к}}/cs2 -dedicated +map {{de_dust2}} -maxplayers {{64}}`

- Запустить игру с указанными IP и портом сервера:

`{{путь/к}}/cs2 -dedicated +map {{de_dust2}} -ip {{1.2.3.4}} -port {{27015}}`

- [Интерактивно] Остановить сервер:

`quit`
