# apt install

> Установить пакет или обновить его до последней версии для дистрибутивов на базе Debian.
> Больше информации: <https://manned.org/apt.8>.

- Установить пакет или обновить его до последней версии:

`sudo apt install {{пакет}}`

- Отобразить подробную информацию о версиях пакета во время установки или обновления:

`sudo apt install {{[-V|--verbose-versions]}} {{пакет}}`
