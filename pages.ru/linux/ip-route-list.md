# ip route list

> Подкоманда для отображения таблицы IP-маршрутизации.
> Больше информации: <https://manned.org/ip-route>.

- Показать основную (`main`) таблицу маршрутизации:

`ip {{[r|route]}} {{[l|list]}}`

- Показать основную таблицу маршрутизации (то же, что и в первом примере):

`ip {{[r|route]}} {{[l|list]}} {{[t|table]}} {{main|254}}`

- Показать локальную таблицу маршрутизации:

`ip {{[r|route]}} {{[l|list]}} {{[t|table]}} {{local|255}}`

- Показать все таблицы маршрутизации:

`ip {{[r|route]}} {{[l|list]}} {{[t|table]}} {{all|unspec|0}}`

- Вывести список маршрутов только для указанного устройства:

`ip {{[r|route]}} {{[l|list]}} dev {{ethX}}`

- Вывести список маршрутов в рамках указанной области действия (scope):

`ip {{[r|route]}} {{[l|list]}} {{[s|scope]}} link`

- Показать кэш маршрутизации:

`ip {{[r|route]}} {{[l|list]}} {{[c|cache]}}`

- Показать маршруты только для IPv6 или IPv4:

`ip {{-6|-4}} {{[r|route]}}`
