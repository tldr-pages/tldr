# qm disk

> Управлять образами дисков.
> Больше информации: <https://pve.proxmox.com/pve-docs/qm.1.html#cli_qm_disk_import>.

- Добавить `n` гигабайт к виртуальному диску:

`qm {{[di|disk]}} {{[resi|resize]}} {{100}} {{имя_диска}} +{{n}}G`

- Переместить виртуальный диск:

`qm {{[di|disk]}} {{[m|move]}} {{100}} {{назначение}} {{индекс}}`

- Удалить предыдущую копию виртуального диска:

`qm {{[di|disk]}} {{[m|move]}} --delete {{100}} {{назначение}} {{индекс}}`

- Импортировать образ диска VMDK/`.qcow2`/raw, используя определённое имя хранилища:

`qm {{[di|disk]}} {{[i|import]}} {{100}} {{путь/к/диску}} {{имя_хранилища}} --format {{qcow2|raw|vmdk}}`

- Повторно просканировать все хранилища и обновить размеры дисков и неиспользуемые образы дисков:

`qm {{[di|disk]}} {{[resc|rescan]}}`

- Выполнить пробный запуск повторного сканирования без записи изменений в конфигурации:

`qm {{[di|disk]}} {{[resc|rescan]}} --dryrun`

- Указать виртуальную машину по её идентификатору:

`qm {{[di|disk]}} {{[resc|rescan]}} --vmid {{100}}`
