# flex

> Генератор лексических анализаторов.
> Генерировать код на C, реализующий лексический анализатор на основе его спецификации.
> Больше информации: <https://manned.org/flex>.

- Сгенерировать анализатор из Lex-файла, сохранив его в файл `lex.yy.c`:

`flex {{анализатор.l}}`

- Вывести анализатор в `stdout`:

`flex {{[-t|--stdout]}} {{анализатор.l}}`

- Указать выходной файл:

`flex {{анализатор.l}} {{[-o|--outfile]}} {{анализатор.c}}`

- Сгенерировать пакетный сканер вместо интерактивного:

`flex {{[-B|--batch]}} {{анализатор.l}}`

- Скомпилировать C-файл, сгенерированный Lex:

`cc {{путь/к/lex.yy.c}} -o {{исполняемый_файл}}`
