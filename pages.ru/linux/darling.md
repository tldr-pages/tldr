# darling

> Запускать программы для macOS в Linux.
> Больше информации: <https://darlinghq.org>.

- Запустить встроенную команду:

`darling shell {{uname}}`

- Запустить указанную программу из текущего каталога с аргументами:

`darling shell {{./программа}} {{аргумент_программы_1 аргумент_программы_2 ...}}`

- Открыть оболочку macOS:

`darling shell`

- Завершить работу сервиса:

`darling shutdown`
