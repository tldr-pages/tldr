# darling

> Запускать программы для macOS в Linux.
> Больше информации: <https://docs.darlinghq.org/darling-shell.html>.

- Запустить встроенную команду:

`darling shell {{uname}}`

- Запустить указанную программу с аргументами:

`darling shell {{путь/к/программе}} {{аргумент_программы_1 аргумент_программы_2 ...}}`

- Открыть оболочку macOS:

`darling shell`

- Завершить работу сервиса:

`darling shutdown`
