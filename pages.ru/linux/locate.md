# locate

> Быстро находить файлы по имени.
> Больше информации: <https://manned.org/locate>.

- Найти шаблон в базе данных. Примечание: база пересчитывается периодически (обычно еженедельно или ежедневно):

`locate "{{шаблон}}"`

- Найти шаблон без учёта регистра:

`locate {{[-i|--ignore-case]}} "{{шаблон}}"`

- Найти файл по точному имени (шаблон без спецсимволов интерпретируется как `*шаблон*`):

`locate "*/{{имя_файла}}"`

- Пересчитать базу данных. Это необходимо сделать, если вы хотите найти недавно добавленные файлы:

`sudo updatedb`
