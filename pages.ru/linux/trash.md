# trash

> Управлять корзиной.
> Больше информации: <https://github.com/andreafrancia/trash-cli>.

- Отправить файл в корзину:

`trash {{путь/к/файлу}}`

- Показать список всех файлов в корзине:

`trash-list`

- Интерактивно восстановить файл из корзины:

`trash-restore`

- Очистить корзину:

`trash-empty`

- Навсегда удалить все файлы в корзине старше 10 дней:

`trash-empty 10`

- Удалить все файлы в корзине, соответствующие определённому glob-шаблону:

`trash-rm "{{*.o}}"`

- Удалить все файлы с определённым исходным расположением:

`trash-rm /{{путь/к/файлу_или_каталогу}}`
