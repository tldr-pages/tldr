# a2enconf

> Включать конфигурационные файлы Apache в операционных системах на базе Debian.
> Больше информации: <https://manned.org/a2enconf>.

- Включить конфигурационный файл:

`sudo a2enconf {{конфигурационный_файл}}`

- Не выводить информационные сообщения:

`sudo a2enconf {{[-q|--quiet]}} {{конфигурационный_файл}}`
