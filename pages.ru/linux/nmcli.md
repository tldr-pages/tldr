# nmcli

> Управлять сетевой конфигурацией с помощью NetworkManager.
> Смотрите также: `nmtui`, `iw`, `iwctl`.
> Больше информации: <https://networkmanager.pages.freedesktop.org/NetworkManager/NetworkManager/nmcli.html>.

- Посмотреть документацию по управлению сетевыми интерфейсами и установке новых Wi-Fi подключений:

`tldr nmcli device`

- Посмотреть документацию по управлению сетевыми подключениями:

`tldr nmcli connection`

- Посмотреть документацию по запуску `nmcli` в качестве агента секретов/polkit NetworkManager:

`tldr nmcli agent`

- Посмотреть документацию по управлению общими настройками NetworkManager:

`tldr nmcli general`

- Посмотреть документацию по монитору активности NetworkManager:

`tldr nmcli monitor`

- Посмотреть документацию по включению/отключению и проверке состояния сети:

`tldr nmcli networking`

- Посмотреть документацию по управлению радиопереключателями:

`tldr nmcli radio`
