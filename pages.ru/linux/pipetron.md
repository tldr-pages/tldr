# pipetron

> Управлять узлами Electron для PipeWire.
> Больше информации: <https://codeberg.org/ponleou/pipetron#usage>.

- Запустить демон, указанный в файле конфигурации (по умолчанию запускается демон громкости):

`pipetron {{[-d|--daemon]}}`

- Запустить демон громкости (дублировать настройки громкости):

`pipetron {{[-vd|--volume-daemon]}}`

- Запустить аудио-демон (дублировать аудиоданные):

`pipetron {{[-ad|--audio-daemon]}}`
