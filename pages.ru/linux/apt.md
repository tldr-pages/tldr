# apt

> Менеджер пакетов для дистрибутивов на базе Debian.
> Предназначен как удобная альтернатива `apt-get` для интерактивного использования.
> Для эквивалентных команд в других пакетных менеджерах см. <https://wiki.archlinux.org/title/Pacman/Rosetta>.
> Больше информации: <https://manned.org/apt.8>.

- Обновить список доступных пакетов и версий (рекомендуется перед другими командами `apt`):

`sudo apt update`

- Искать пакеты по имени или описанию:

`apt search {{пакет}}`

- Искать пакеты только по имени (поддерживаются шаблоны `*`):

`apt list {{пакет}}`

- Показать подробную информацию о пакете:

`apt show {{пакет}}`

- Установить пакет или обновить его до последней версии:

`sudo apt install {{пакет}}`

- Удалить пакет (для удаления вместе с файлами конфигурации используйте `purge`):

`sudo apt remove {{пакет}}`

- Обновить все установленные пакеты до последних версий:

`sudo apt upgrade`

- Показать список всех установленных пакетов:

`apt list {{[-i|--installed]}}`
