# a2ensite

> Включать виртуальные хосты Apache в операционных системах на базе Debian.
> Больше информации: <https://manned.org/a2ensite>.

- Включить виртуальный хост:

`sudo a2ensite {{виртуальный_хост}}`

- Не выводить информационные сообщения:

`sudo a2ensite {{[-q|--quiet]}} {{виртуальный_хост}}`
