# groupadd

> Добавлять группы пользователей в систему.
> Смотрите также: `groups`, `groupdel`, `groupmod`.
> Больше информации: <https://manned.org/groupadd>.

- Создать новую группу:

`sudo groupadd {{имя_группы}}`

- Создать новую системную группу:

`sudo groupadd {{[-r|--system]}} {{имя_группы}}`

- Создать новую группу с указанным идентификатором группы (GID):

`sudo groupadd {{[-g|--gid]}} {{id}} {{имя_группы}}`
