# apt-file

> Искать файлы в пакетах `apt`, в том числе ещё не установленных.
> Больше информации: <https://manned.org/apt-file>.

- Обновить базу данных метаданных:

`sudo apt update`

- Найти пакеты, содержащие указанный файл или путь:

`apt-file {{[find|search]}} {{путь/к/файлу}}`

- Вывести содержимое конкретного пакета:

`apt-file list {{пакет}}`

- Найти пакеты, соответствующие регулярному выражению:

`apt-file {{[find|search]}} {{[-x|--regexp]}} {{регулярное_выражение}}`
