# Factorio

> Создавать и запускать выделенный сервер Factorio.
> Больше информации: <https://wiki.factorio.com/Multiplayer>.

- Создать новый файл сохранения:

`{{путь/к/}}/factorio --create {{путь/к/файлу_сохранения.zip}}`

- Запустить сервер Factorio:

`{{путь/к/}}/factorio --start-server {{путь/к/файлу_сохранения.zip}}`

- Создать новый файл сохранения с определёнными настройками:

`{{путь/к/}}/factorio --create {{путь/к/сохранению.zip}} --map-gen-settings {{путь/к/настройкам_генерации_карты.json}} --map-settings {{путь/к/настройкам_карты.json}}`

- [Интерактивно] Корректно остановить сервер:

`/quit`

- Показать справку:

`{{путь/к/}}/factorio {{[-h|--help]}}`
