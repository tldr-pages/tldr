# rfetch

> A neofetch-like tool but, faster & prettier.
> More information: <https://github.com/LinusDierheimer/fastfetch>.
> Modify ~/.config/fastfetch/config.conf for persistent flags.

- Display system information:

`rfetch`

- Display system [a]rchitecture:

`rfetch -a`

- Display system up[t]ime:

`rfetch -t`

- Display system [k]ernel:

`rfetch -k`

- Display system [c]PU:

`rfetch -c`

- Display Linux [D]istro:

`rfetch -D`

- View [d]esktop environment:

`rfetch -d`
