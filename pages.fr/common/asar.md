# asar

> Un outil d'archivage pour la plateforme Electron.
> Plus d'informations : <https://github.com/electron/asar#usage>.

- Archive un fichier ou un dossier :

`asar {{[p|pack]}} {{chemin/vers/fichier_ou_dossier}} {{archive.asar}}`

- Extrais une archive :

`asar {{[e|extract]}} {{archive.asar}}`

- Extrais un fichier spécifique d'une archive :

`asar {{[ef|extract-file]}} {{archive.asar}} {{fichier}}`

- Liste les éléments contenus dans une archive :

`asar {{[l|list]}} {{archive.asar}}`
