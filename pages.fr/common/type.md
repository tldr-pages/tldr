# type

> Afficher le type de la commande que le terminal exécuterait.
> Plus d'information: <https://manned.org/type>.

- Afficher le type d'une commande:

`type {{commande}}`

- Afficher tous les emplacements contenants l'exécutable spécifié :

`type -a {{commande}}`

- Afficher le nom du fichier disque qui serait exécuté:

`type -p {{commande}}`
