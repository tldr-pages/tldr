# cls

> Efface l'écran.
> Dans PowerShell, cette commande est un alias de `Clear-Host`. Cette documentation est basée sur la version de `cls` utilisable dans l'Invite de Commande (`cmd`).
> Plus d'informations : <https://learn.microsoft.com/windows-server/administration/windows-commands/cls>.

- Consulte la documentation relative à la commande PowerShell équivalente :

`tldr clear-host`

- Efface l'écran :

`cls`
