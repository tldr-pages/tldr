# chpass

> ব্যবহারকারীর ডাটাবেসের তথ্য যোগ বা পরিবর্তন করুন, যার মধ্যে রয়েছে লগইন শেল এবং পাসওয়ার্ড।
> দ্রষ্টব্য: Open Directory সিস্টেমে ব্যবহারকারীর পাসওয়ার্ড পরিবর্তন করা সম্ভব নয়, পরিবর্তে `passwd` ব্যবহার করুন।
> আরও দেখুন: `passwd`।
> আরও তথ্য পাবেন: <https://man.freebsd.org/cgi/man.cgi?chpass>।

- বর্তমান ব্যবহারকারীর জন্য ডাটাবেসের তথ্য ইন্টার‌্যাকটিভভাবে যোগ বা পরিবর্তন করুন:

`su -c chpass`

- বর্তমান ব্যবহারকারীর জন্য একটি নির্দিষ্ট লগইন [s]hell সেট করুন:

`chpass -s {{শেল/এর/পাথ}}`

- একজন নির্দিষ্ট ব্যবহারকারীর জন্য একটি লগইন [s]hell সেট করুন:

`chpass -s {{শেল/এর/পাথ}} {{ইউজারনেম}}`

- প্রদত্ত [l]ocation-এ থাকা directory node-এ ব্যবহারকারীর রেকর্ড এডিট করুন:

`chpass -l {{লোকেশন}} -s {{শেল/এর/পাথ}} {{ইউজারনেম}}`

- ব্যবহারকারী থাকা directory node-এ অথেন্টিকেট করার সময় প্রদত্ত [u]sername ব্যবহার করুন:

`chpass -u {{অথনেম}} -s {{শেল/এর/পাথ}} {{ইউজারনেম}}`
