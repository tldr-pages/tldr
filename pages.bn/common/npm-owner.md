# npm owner

> প্রকাশিত (published) প্যাকেজের মালিকানা ম্যানেজ করুন।
> আরও তথ্য পাবেন: <https://docs.npmjs.com/cli/npm-owner/>।

- একটি প্যাকেজের মেইনটেইনার হিসেবে একজন নতুন ইউজার যোগ করুন:

`npm owner add {{ইউজারনেম}} {{প্যাকেজের_নাম}}`

- একটি প্যাকেজের মালিক (owner) তালিকা থেকে একজন ইউজার সরান:

`npm owner rm {{ইউজারনেম}} {{প্যাকেজের_নাম}}`

- একটি প্যাকেজের সব মালিকের (owner) তালিকা দেখুন:

`npm owner ls {{প্যাকেজের_নাম}}`
