# python

> Python ল্যাঙ্গুয়েজ ইন্টারপ্রেটার।
> আরও তথ্য পাবেন: <https://docs.python.org/using/cmdline.html>।

- একটি REPL (ইন্টার‌্যাকটিভ শেল) চালু করুন:

`python`

- একটি নির্দিষ্ট Python ফাইল এক্সিকিউট করুন:

`python {{ফাইল.py/এর/পাথ}}`

- একটি নির্দিষ্ট Python ফাইল এক্সিকিউট করুন এবং একটি REPL চালু করুন:

`python -i {{ফাইল.py/এর/পাথ}}`

- একটি Python এক্সপ্রেশন এক্সিকিউট করুন:

`python -c "{{এক্সপ্রেশন}}"`

- নির্দিষ্ট লাইব্রেরি মডিউলের স্ক্রিপ্ট চালান:

`python -m {{মডিউল}} {{আর্গুমেন্ট}}`

- `pip` ব্যবহার করে একটি প্যাকেজ ইনস্টল করুন:

`python -m pip install {{প্যাকেজ}}`

- একটি Python স্ক্রিপ্ট ইন্টার‌্যাকটিভভাবে ডিবাগ করুন:

`python -m pdb {{ফাইল.py/এর/পাথ}}`

- বর্তমান ডিরেক্টরিতে বিল্ট-ইন HTTP সার্ভার পোর্ট 8000-এ চালু করুন:

`python -m http.server`
