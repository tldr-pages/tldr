# npm run

> একটি স্ক্রিপ্ট চালান।
> আরও তথ্য পাবেন: <https://docs.npmjs.com/cli/npm-run/>।

- এভেলেভেল স্ক্রিপ্টের তালিকা দেখুন:

`npm run`

- একটি স্ক্রিপ্ট চালান:

`npm run {{স্ক্রিপ্টের_নাম}}`

- একটি স্ক্রিপ্টে আর্গুমেন্ট পাস করুন:

`npm run {{স্ক্রিপ্টের_নাম}} -- {{আর্গুমেন্ট}} {{--অপশন}}`

- `start` নামের একটি স্ক্রিপ্ট চালান:

`npm start`

- `stop` নামের একটি স্ক্রিপ্ট চালান:

`npm stop`

- `restart` নামের একটি স্ক্রিপ্ট চালান:

`npm restart`

- `test` নামের একটি স্ক্রিপ্ট চালান:

`npm {{[t|test]}}`
