# bzip2

> একটি ব্লক-সর্টিং ফাইল কমপ্রেসার।
> আরও দেখুন: `bzcat`, `bunzip2`, `bzip2recover`।
> আরও তথ্য পাবেন: <https://manned.org/bzip2>।

- একটি ফাইল কমপ্রেস করুন:

`bzip2 {{কমপ্রেস_করার_ফাইল/এর/পাথ}}`

- একটি ফাইল ডিকমপ্রেস করুন:

`bzip2 {{[-d|--decompress]}} {{কমপ্রেসড_ফাইল/এর/পাথ.bz2}}`

- একটি ফাইল `stdout`-এ ডিকমপ্রেস করুন:

`bzip2 {{[-dc|--decompress --stdout]}} {{কমপ্রেসড_ফাইল/এর/পাথ.bz2}}`

- আর্কাইভ ফাইলের ভেতরের প্রতিটি ফাইলের অখণ্ডতা (integrity) যাচাই করুন:

`bzip2 {{[-t|--test]}} {{কমপ্রেসড_ফাইল/এর/পাথ.bz2}}`

- বিস্তারিত তথ্যসহ প্রতিটি প্রক্রিয়াকৃত ফাইলের কমপ্রেশন অনুপাত দেখুন:

`bzip2 {{[-v|--verbose]}} {{কমপ্রেসড_ফাইল/এর/পাথ.bz2}}`

- বিদ্যমান (existing) ফাইল ওভাররাইট করে একটি ফাইল ডিকমপ্রেস করুন:

`bzip2 {{[-f|--force]}} {{কমপ্রেসড_ফাইল/এর/পাথ.bz2}}`

- সাহায্য প্রদর্শন করুন:

`bzip2 {{[-h|--help]}}`
