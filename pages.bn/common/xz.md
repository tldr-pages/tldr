# xz

> XZ এবং LZMA ফাইল কম্প্রেস বা ডিকম্প্রেস করুন।
> আরও তথ্য পাবেন: <https://manned.org/xz>।

- xz ব্যবহার করে একটি ফাইল কম্প্রেস করুন:

`xz {{ফাইল/এর/পাথ}}`

- একটি XZ ফাইল ডিকম্প্রেস করুন:

`xz {{[-d|--decompress]}} {{ফাইল.xz/এর/পাথ}}`

- lzma ব্যবহার করে একটি ফাইল কম্প্রেস করুন:

`xz {{[-F|--format]}} lzma {{ফাইল/এর/পাথ}}`

- একটি LZMA ফাইল ডিকম্প্রেস করুন:

`xz {{[-d|--decompress]}} {{[-F|--format]}} lzma {{ফাইল.lzma/এর/পাথ}}`

- একটি ফাইল ডিকম্প্রেস করুন এবং `stdout`-এ লিখুন (`--keep` বোঝায়):

`xz {{[-d|--decompress]}} {{[-c|--stdout]}} {{ফাইল.xz/এর/পাথ}}`

- একটি ফাইল কম্প্রেস করুন, কিন্তু মূল ফাইল মুছবেন না:

`xz {{[-k|--keep]}} {{ফাইল/এর/পাথ}}`

- দ্রুততম কম্প্রেশন ব্যবহার করে একটি ফাইল কম্প্রেস করুন:

`xz -0 {{ফাইল/এর/পাথ}}`

- সর্বোত্তম কম্প্রেশন ব্যবহার করে একটি ফাইল কম্প্রেস করুন:

`xz -9 {{ফাইল/এর/পাথ}}`
