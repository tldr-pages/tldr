# npm install-test

> `npm install` চালানোর পরে `npm test` চালানোর সমতুল্য।
> দ্রষ্টব্য: `it` কে `install-test`-এর শর্টহ্যান্ড হিসেবে ব্যবহার করা যেতে পারে।
> আরও তথ্য পাবেন: <https://docs.npmjs.com/cli/npm-install-test/>।

- সব ডিপেন্ডেন্সি ইনস্টল করুন এবং তারপর টেস্ট চালান:

`npm {{[it|install-test]}}`

- একটি নির্দিষ্ট প্যাকেজ ইনস্টল করুন এবং তারপর টেস্ট চালান:

`npm {{[it|install-test]}} {{প্যাকেজের_নাম}}`

- একটি প্যাকেজ ইনস্টল করুন এবং টেস্ট চালানোর আগে এটি ডিপেন্ডেন্সি হিসেবে সংরক্ষণ করুন:

`npm {{[it|install-test]}} {{প্যাকেজের_নাম}} {{[-S|--save]}}`

- গ্লোবালি ডিপেন্ডেন্সি ইনস্টল করুন এবং তারপর টেস্ট চালান:

`npm {{[it|install-test]}} {{[-g|--global]}}`
