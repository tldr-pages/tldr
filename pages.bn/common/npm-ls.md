# npm ls

> ইনস্টল করা প্যাকেজগুলো `stdout`-এ প্রিন্ট করুন।
> আরও তথ্য পাবেন: <https://docs.npmjs.com/cli/npm-ls/>।

- বর্তমান প্রজেক্টে সরাসরি ডিপেন্ডেন্সির সব ভার্সন `stdout`-এ প্রিন্ট করুন:

`npm {{[ls|list]}}`

- পিয়ার (peer) ডিপেন্ডেন্সি সহ সব ইনস্টল করা প্যাকেজ প্রিন্ট করুন:

`npm {{[ls|list]}} {{[-a|--all]}}`

- সব গ্লোবালি ইনস্টল করা প্যাকেজ প্রিন্ট করুন:

`npm {{[ls|list]}} {{[-g|--global]}}`

- বর্ধিত (extended) তথ্য সহ ডিপেন্ডেন্সি প্রিন্ট করুন:

`npm {{[ls|list]}} {{[-l|--long]}}`

- পার্সযোগ্য (parseable) ফরম্যাটে ডিপেন্ডেন্সি প্রিন্ট করুন:

`npm {{[ls|list]}} {{[-p|--parseable]}}`

- JSON ফরম্যাটে ডিপেন্ডেন্সি প্রিন্ট করুন:

`npm {{[ls|list]}} --json`
