# npm rebuild

> Node বা ডিপেন্ডেন্সি পরিবর্তনের পরে নেটিভ Node.js প্যাকেজ পুনর্নির্মাণ করুন।
> আরও তথ্য পাবেন: <https://docs.npmjs.com/cli/npm-rebuild/>।

- একটি নির্দিষ্ট প্যাকেজ পুনর্নির্মাণ করুন:

`npm {{[rb|rebuild]}} {{প্যাকেজ}}`

- সব ইনস্টল করা প্যাকেজ পুনর্নির্মাণ করুন:

`npm {{[rb|rebuild]}}`

- ভার্বোস (verbose) আউটপুট সহ পুনর্নির্মাণ করুন:

`npm {{[rb|rebuild]}} --verbose`

- একটি নির্দিষ্ট ডিরেক্টরিতে একটি প্যাকেজ পুনর্নির্মাণ করুন:

`npm {{[rb|rebuild]}} --prefix {{ডিরেক্টরি/এর/পাথ}} {{প্যাকেজ}}`

- npm ক্যাশ ব্যবহার না করে পুনর্নির্মাণ করুন:

`npm {{[rb|rebuild]}} --no-cache`

- গ্লোবাল মোডে পুনর্নির্মাণ করুন:

`npm {{[rb|rebuild]}} {{[-g|--global]}}`
