# jira issue

> একটি Jira প্রজেক্টে ইস্যু ম্যানেজ করুন।
> আরও তথ্য পাবেন: <https://github.com/ankitpokhrel/jira-cli#issue>।

- সাম্প্রতিক ইস্যুর তালিকা দেখুন:

`jira issue {{[ls|list]}}`

- একজন নির্দিষ্ট ব্যবহারকারীকে বরাদ্দ করা ইস্যুর তালিকা দেখুন:

`jira issue {{[ls|list]}} {{[-a|--assignee]}} "{{ইমেইল_বা_প্রদর্শন_নাম}}"`

- আমার কাছে বরাদ্দ করা উচ্চ অগ্রাধিকারের ইস্যুর তালিকা দেখুন:

`jira issue {{[ls|list]}} {{[-a|--assignee]}} $(jira me) {{[-y|--priority]}} High`

- একটি ইন্টার‌্যাকটিভ প্রম্পট ব্যবহার করে একটি ইস্যু তৈরি করুন:

`jira issue create`

- একটি ইন্টার‌্যাকটিভ প্রম্পট ব্যবহার করে একটি ইস্যু এডিট করুন:

`jira issue edit`

- একটি ইন্টার‌্যাকটিভ প্রম্পট ব্যবহার করে একটি ইস্যুতে ব্যবহারকারী বরাদ্দ করুন:

`jira issue {{[asg|assign]}}`

- একটি নির্দিষ্ট অবস্থায় ইস্যু মুভ করুন:

`jira issue {{[mv|move]}} {{ইস্যু_আইডি}} "{{In Progress}}"`

- `less` ব্যবহার করে টার্মিনালে একটি ইস্যু খুলুন:

`jira issue view {{ইস্যু_আইডি}}`
