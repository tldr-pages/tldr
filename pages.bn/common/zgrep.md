# zgrep

> কম্প্রেসড ফাইলের মধ্যে টেক্সট প্যাটার্ন Grep করুন।
> আরও তথ্য পাবেন: <https://manned.org/zgrep>।

- একটি কম্প্রেসড ফাইলে একটি প্যাটার্ন Grep করুন (case-sensitive):

`zgrep {{প্যাটার্ন}} {{কম্প্রেসড_ফাইল/এর/পাথ}}`

- প্রতিটি ম্যাচের আশেপাশে, আগে বা পরে 3 লাইনের [C]ontext প্রিন্ট করুন:

`zgrep {{--context|--before-context|--after-context}} 3 {{প্যাটার্ন}} {{কম্প্রেসড_ফাইল/এর/পাথ}}`

- একটি কম্প্রেসড ফাইলে একটি প্যাটার্ন Grep করুন (case-insensitive):

`zgrep {{[-i|--ignore-case]}} {{প্যাটার্ন}} {{কম্প্রেসড_ফাইল/এর/পাথ}}`

- একটি কম্প্রেসড ফাইলে মিলে যাওয়া প্যাটার্ন সম্বলিত লাইনের সংখ্যা আউটপুট করুন:

`zgrep {{[-c|--count]}} {{প্যাটার্ন}} {{কম্প্রেসড_ফাইল/এর/পাথ}}`

- এমন লাইনগুলো প্রদর্শন করুন যেখানে প্যাটার্ন নেই (সার্চ ফাংশন উল্টো করুন):

`zgrep {{[-v|--invert-match]}} {{প্যাটার্ন}} {{কম্প্রেসড_ফাইল/এর/পাথ}}`

- একটি কম্প্রেসড ফাইলে একাধিক প্যাটার্ন Grep করুন:

`zgrep {{[-e|--regexp]}} "{{প্যাটার্ন_১}}" {{[-e|--regexp]}} "{{প্যাটার্ন_২}}" {{কম্প্রেসড_ফাইল/এর/পাথ}}`

- এক্সটেন্ডেড `regex` ব্যবহার করুন (`?`, `+`, `{}`, `()`, এবং `|` সাপোর্ট করে):

`zgrep {{[-E|--extended-regexp]}} {{regex}} {{ফাইল/এর/পাথ}}`
