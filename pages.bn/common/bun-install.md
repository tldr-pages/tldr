# bun install

> `package.json` থেকে একটি প্রজেক্টের জন্য JavaScript ডিপেন্ডেন্সি ইনস্টল করুন।
> আরও তথ্য পাবেন: <https://bun.com/docs/pm/cli/install>।

- `package.json`-এ তালিকাভুক্ত সমস্ত ডিপেন্ডেন্সি ইনস্টল করুন:

`bun {{[i|install]}}`

- একটি একক (single) প্যাকেজ ইনস্টল করুন (এটি `bun add` এর একটি উপনাম):

`bun {{[i|install]}} {{প্যাকেজ_নাম}}@{{ভার্সন}}`

- গ্লোবালি একটি প্যাকেজ ইনস্টল করুন:

`bun {{[i|install]}} {{[-g|--global]}} {{প্যাকেজ_নাম}}`

- শুধুমাত্র প্রোডাকশন ডিপেন্ডেন্সি ইনস্টল করুন (`devDependencies` স্কিপ করে):

`bun {{[i|install]}} {{[-p|--production]}}`

- `bun.lockb` লকফাইল থেকে ডিপেন্ডেন্সি ঠিক মতো ইনস্টল করুন (ফ্রোজেন লকফাইল):

`bun {{[i|install]}} --frozen-lockfile`

- ক্যাশ উপেক্ষা করে রেজিস্ট্রি থেকে সমস্ত প্যাকেজ জোর করে পুনরায় ডাউনলোড করুন:

`bun {{[i|install]}} {{[-f|--force]}}`
