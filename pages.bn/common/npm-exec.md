# npm exec

> `npm` প্যাকেজ থেকে বাইনারি এক্সিকিউট করুন।
> আরও তথ্য পাবেন: <https://docs.npmjs.com/cli/npm-exec/>।

- একটি লোকাল বা রিমোট `npm` প্যাকেজ থেকে কমান্ড এক্সিকিউট করুন:

`npm {{[x|exec]}} {{কমান্ড}} {{আর্গুমেন্ট১ আর্গুমেন্ট২ ...}}`

- প্যাকেজটি স্পষ্টভাবে নির্দিষ্ট করুন (একই নামের একাধিক কমান্ড থাকলে উপকারী):

`npm {{[x|exec]}} --package {{প্যাকেজ}} {{কমান্ড}}`

- একটি কমান্ড চালান যদি এটি বর্তমান পাথে বা `node_modules/.bin`-এ থাকে:

`npm {{[x|exec]}} --no-install {{কমান্ড}} {{আর্গুমেন্ট১ আর্গুমেন্ট২ ...}}`

- একটি নির্দিষ্ট কমান্ড এক্সিকিউট করুন, `npm` নিজে থেকে যেকোনো আউটপুট দমন (suppress) করে:

`npm {{[x|exec]}} --quiet {{কমান্ড}} {{আর্গুমেন্ট১ আর্গুমেন্ট২ ...}}`

- সাহায্য প্রদর্শন করুন:

`npm {{[x|exec]}} --help`
