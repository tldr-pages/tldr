# hugo server

> Hugo-এর বিল্ট-ইন ওয়েবসার্ভার দিয়ে একটি সাইট বিল্ড এবং সার্ভ করুন।
> আরও তথ্য পাবেন: <https://gohugo.io/commands/hugo_server/>।

- একটি সাইট বিল্ড এবং সার্ভ করুন:

`hugo server`

- একটি নির্দিষ্ট পোর্ট নম্বরে সাইট বিল্ড এবং সার্ভ করুন:

`hugo server {{[-p|--port]}} {{পোর্ট_নম্বর}}`

- সাপোর্টেড আউটপুট ফরম্যাট (HTML, XML, ইত্যাদি) মিনিফাই করার সময় সাইট বিল্ড এবং সার্ভ করুন:

`hugo server --minify`

- প্রোডাকশন এনভায়রনমেন্টে সম্পূর্ণ রি-রেন্ডার সহ সাপোর্টেড ফরম্যাট মিনিফাই করে সাইট বিল্ড এবং সার্ভ করুন:

`hugo server {{[-e|--environment]}} {{production}} --disableFastRender --minify`

- সাহায্য প্রদর্শন করুন:

`hugo server {{[-h|--help]}}`
