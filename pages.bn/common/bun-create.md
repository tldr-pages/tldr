# bun create

> টেমপ্লেট থেকে একটি নতুন প্রজেক্ট তৈরি করুন।
> দ্রষ্টব্য: `create` এর জন্য `c` উপনাম (alias) হিসেবে ব্যবহার করা যেতে পারে।
> আরও তথ্য পাবেন: <https://bun.com/docs/runtime/templating/create>।

- অফিশিয়াল টেমপ্লেট থেকে ইন্টার্যাক্টিভভাবে একটি নতুন প্রজেক্ট তৈরি করুন:

`bun create {{টেমপ্লেট}}`

- নতুন ডিরেক্টরিতে একটি অফিশিয়াল টেমপ্লেট থেকে নতুন প্রজেক্ট তৈরি করুন:

`bun create {{টেমপ্লেট}} {{পাথ/টু/ডেস্টিনেশন}}`

- একটি GitHub রিপোজিটরি টেমপ্লেট থেকে একটি নতুন প্রজেক্ট তৈরি করুন:

`bun create {{https://github.com/username/repo}} {{পাথ/টু/ডেস্টিনেশন}}`

- একটি লোকাল টেমপ্লেট থেকে একটি নতুন প্রজেক্ট তৈরি করুন:

`bun create {{পাথ/টু/টেমপ্লেট}} {{পাথ/টু/ডেস্টিনেশন}}`

- একটি নতুন প্রজেক্ট তৈরি করুন, ডেস্টিনেশন ডিরেক্টরি বিদ্যমান থাকলে তা ওভাররাইট (overwrite) করুন:

`bun create {{টেমপ্লেট}} {{পাথ/টু/ডেস্টিনেশন}} --force`

- স্বয়ংক্রিয়ভাবে Git রিপোজিটরি ইনিশিয়ালাইজ না করে একটি নতুন প্রজেক্ট তৈরি করুন:

`bun create {{টেমপ্লেট}} {{পাথ/টু/ডেস্টিনেশন}} --no-git`

- স্বয়ংক্রিয়ভাবে ডিপেন্ডেন্সি ইনস্টল না করে একটি নতুন প্রজেক্ট তৈরি করুন:

`bun create {{টেমপ্লেট}} {{পাথ/টু/ডেস্টিনেশন}} --no-install`
