# GetADUsers.py

> Active Directory থেকে ব্যবহারকারীদের একটি তালিকা সংগ্রহ করুন, যেখানে শেষ লগঅন টাইমস্ট্যাম্প এবং ইমেইলের মতো অ্যাট্রিবিউট থাকে।
> Impacket স্যুইটের অংশ।
> আরও তথ্য পাবেন: <https://github.com/fortra/impacket>।

- সব Active Directory ব্যবহারকারী এবং তাদের অ্যাট্রিবিউট তালিকাভুক্ত করুন:

`GetADUsers.py -all -dc-ip {{ডোমেইন_কন্ট্রোলার_আইপি}} {{ডোমেইন}}/{{ইউজারনেম}}:{{পাসওয়ার্ড}}`

- কেবল নির্দিষ্ট একজন ব্যবহারকারীর তথ্য সংগ্রহ করুন:

`GetADUsers.py -user {{ব্যবহারকারী}} -dc-ip {{ডোমেইন_কন্ট্রোলার_আইপি}} {{ডোমেইন}}/{{ইউজারনেম}}:{{পাসওয়ার্ড}}`

- pass-the-hash অথেন্টিকেশন ব্যবহার করে ব্যবহারকারীর তথ্য বের করুন:

`GetADUsers.py -all -dc-ip {{ডোমেইন_কন্ট্রোলার_আইপি}} -hashes {{LM_হ্যাশ}}:{{NT_হ্যাশ}} {{ডোমেইন}}/{{ইউজারনেম}}`

- আউটপুট একটি ফাইলে সংরক্ষণ করুন:

`GetADUsers.py -all -dc-ip {{ডোমেইন_কন্ট্রোলার_আইপি}} {{ডোমেইন}}/{{ইউজারনেম}}:{{পাসওয়ার্ড}} > {{আউটপুট.txt/এর/পাথ}}`
