# chpass

> ব্যবহারকারীর ডাটাবেসের তথ্য যোগ বা পরিবর্তন করুন, যার মধ্যে রয়েছে লগইন শেল এবং পাসওয়ার্ড।
> আরও দেখুন: `passwd`।
> আরও তথ্য পাবেন: <https://man.freebsd.org/cgi/man.cgi?chpass>।

- বর্তমান ব্যবহারকারীর জন্য ডাটাবেসের তথ্য ইন্টার‌্যাকটিভভাবে যোগ বা পরিবর্তন করুন:

`su -c chpass`

- বর্তমান ব্যবহারকারীর জন্য একটি নির্দিষ্ট লগইন [s]hell সেট করুন:

`chpass -s {{শেল/এর/পাথ}}`

- একজন নির্দিষ্ট ব্যবহারকারীর জন্য একটি লগইন [s]hell সেট করুন:

`chpass -s {{শেল/এর/পাথ}} {{ইউজারনেম}}`

- অ্যাকাউন্টের [e]xpire সময় পরিবর্তন করুন (epoch থেকে সেকেন্ডে, UTC):

`su -c 'chpass -e {{সময়}} {{ইউজারনেম}}'`

- একজন ব্যবহারকারীর পাসওয়ার্ড পরিবর্তন করুন:

`su -c 'chpass -p {{এনক্রিপটেড_পাসওয়ার্ড}} {{ইউজারনেম}}'`

- কুয়েরি করার জন্য একটি NIS সার্ভারের [h]ostname বা ঠিকানা নির্দিষ্ট করুন:

`su -c 'chpass -h {{হোস্টনেম}} {{ইউজারনেম}}'`

- একটি নির্দিষ্ট NIS [d]omain নির্দিষ্ট করুন (ডিফল্টভাবে সিস্টেম ডোমেইন নেম):

`su -c 'chpass -d {{ডোমেইন}} {{ইউজারনেম}}'`
