# apt

> Debian-ভিত্তিক ডিস্ট্রিবিউশনের জন্য প্যাকেজ ম্যানেজার।
> ইন্টার‌্যাকটিভভাবে ব্যবহারের জন্য `apt-get`-এর একটি সহজে ব্যবহারযোগ্য বিকল্প হিসেবে তৈরি করা হয়েছে।
> অন্যান্য প্যাকেজ ম্যানেজারের সমতুল্য কমান্ডের জন্য দেখুন: <https://wiki.archlinux.org/title/Pacman/Rosetta>।
> আরও তথ্য পাবেন: <https://manned.org/apt.8>।

- এভেলেভেল প্যাকেজ ও তাদের ভার্সনের তালিকা আপডেট করুন (অন্যান্য `apt` কমান্ড চালানোর আগে, এটি করার সুপারিশ করা হয়):

`sudo apt update`

- নাম বা বর্ণনা অনুযায়ী প্যাকেজ অনুসন্ধান করুন:

`apt search {{প্যাকেজ}}`

- শুধু নাম অনুযায়ী প্যাকেজ অনুসন্ধান করুন (`*` এর মতো ওয়াইল্ডকার্ড সমর্থিত):

`apt list {{প্যাকেজ}}`

- নির্দিষ্ট একটি প্যাকেজের বিস্তারিত তথ্য দেখান:

`apt show {{প্যাকেজ}}`

- একটি প্যাকেজ ইনস্টল করুন অথবা সেটিকে সর্বশেষ ভার্সনে আপডেট করুন:

`sudo apt install {{প্যাকেজ}}`

- একটি প্যাকেজ সরিয়ে ফেলুন (কনফিগারেশন ফাইলসহ সরাতে `purge` ব্যবহার করুন):

`sudo apt remove {{প্যাকেজ}}`

- ইনস্টল করা সব প্যাকেজকে তাদের সর্বশেষ ভার্সনে আপগ্রেড করুন:

`sudo apt upgrade`

- ইনস্টল করা সব প্যাকেজের তালিকা দেখুন:

`apt list {{[-i|--installed]}}`
