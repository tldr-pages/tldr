# apt-file

> ইনস্টল না করা প্যাকেজসহ `apt` প্যাকেজগুলোতে ফাইল অনুসন্ধান করে।
> আরও তথ্য পাবেন: <https://manned.org/apt-file>।

- মেটাডেটা ডেটাবেস আপডেট করুন:

`sudo apt update`

- নির্দিষ্ট কোনো ফাইল বা পাথ অন্তর্ভুক্ত করে এমন প্যাকেজগুলো খুঁজুন:

`apt-file {{[find|search]}} {{ফাইল/এর/পাথ}}`

- কোনো নির্দিষ্ট প্যাকেজের ফাইলগুলোর তালিকা দেখুন:

`apt-file list {{package}}`

- নির্দিষ্ট একটি `regex`-এর সাথে মিলে এমন প্যাকেজ খুঁজুন:

`apt-file {{[find|search]}} {{[-x|--regexp]}} {{regex}}`
