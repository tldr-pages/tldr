# qm disk

> ডিস্ক ইমেজ ম্যানেজ করুন।
> আরও তথ্য পাবেন: <https://pve.proxmox.com/pve-docs/qm.1.html#cli_qm_disk_import>।

- একটি ভার্চুয়াল ডিস্কে `n` গিগাবাইট যোগ করুন:

`qm {{[di|disk]}} {{[resi|resize]}} {{vm_id}} {{ডিস্কের_নাম}} +{{n}}G`

- একটি ভার্চুয়াল ডিস্ক মুভ করুন:

`qm {{[di|disk]}} {{[m|move]}} {{vm_id}} {{গন্তব্য}} {{ইনডেক্স}}`

- ভার্চুয়াল ডিস্কের পূর্ববর্তী কপি ডিলিট করুন:

`qm {{[di|disk]}} {{[m|move]}} --delete {{vm_id}} {{গন্তব্য}} {{ইনডেক্স}}`

- একটি নির্দিষ্ট স্টোরেজ নাম ব্যবহার করে একটি VMDK/`.qcow2`/raw ডিস্ক ইমেজ ইম্পোর্ট করুন:

`qm {{[di|disk]}} {{[i|import]}} {{vm_id}} {{ডিস্ক/এর/পাথ}} {{স্টোরেজ_নাম}} --format {{qcow2|raw|vmdk}}`

- সমস্ত স্টোরেজ রিস্ক্যান করুন এবং ডিস্কের সাইজ এবং অব্যবহৃত ডিস্ক ইমেজ আপডেট করুন:

`qm {{[di|disk]}} {{[resc|rescan]}}`

- একটি রিস্ক্যানের dry-run সম্পাদন করুন এবং কনফিগারেশনে কোনো পরিবর্তন লিখবেন না:

`qm {{[di|disk]}} {{[resc|rescan]}} --dryrun`

- একটি ভার্চুয়াল মেশিনকে তার ID দ্বারা নির্দিষ্ট করুন:

`qm {{[di|disk]}} {{[resc|rescan]}} --vmid {{100}}`
