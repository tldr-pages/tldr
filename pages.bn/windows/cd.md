# cd

> বর্তমান ওয়ার্কিং ডিরেক্টরি দেখাতে অথবা অন্য কোনো ডিরেক্টরিতে যেতে ব্যবহৃত হয়।
> PowerShell-এ এই কমান্ডটি `Set-Location`-এর একটি উপনাম (alias)। এই ডকুমেন্টেশনটি Command Prompt (`cmd`)–এর `cd` কমান্ডের উপর ভিত্তি করে তৈরি।
> আরও তথ্য পাবেন: <https://learn.microsoft.com/windows-server/administration/windows-commands/cd>।

- সমতুল্য PowerShell কমান্ডের ডকুমেন্টেশন দেখুন:

`tldr set-location`

- বর্তমান ডিরেক্টরির পাথ দেখান:

`cd`

- একই ড্রাইভে নির্দিষ্ট কোনো ডিরেক্টরিতে যান:

`cd {{ডিরেক্টরি\এর\পাথ}}`

- ভিন্ন ড্রাইভ-এ নির্দিষ্ট কোনো ডিরেক্টরিতে যান:

`cd /d {{C}}:{{ডিরেক্টরি\এর\পাথ}}`

- বর্তমান ডিরেক্টরির প্যারেন্ট ডিরেক্টরিতে যান:

`cd ..`

- বর্তমান ব্যবহারকারীর হোম ডিরেক্টরিতে যান:

`cd %userprofile%`

- বর্তমান ড্রাইভের রুট ডিরেক্টরিতে যান:

`cd \`
