# scoop

> Scoop প্যাকেজ ম্যানেজার।
> কিছু সাবকমান্ড (যেমন `bucket`)–এর নিজস্ব ব্যবহারসংক্রান্ত ডকুমেন্টেশন রয়েছে।
> আরও তথ্য পাবেন: <https://github.com/ScoopInstaller/Scoop/wiki/Commands>।

- একটি প্যাকেজ ইনস্টল করুন:

`scoop install {{প্যাকেজ}}`

- একটি প্যাকেজ সরিয়ে ফেলুন:

`scoop uninstall {{প্যাকেজ}}`

- ইনস্টল করা সব প্যাকেজ আপডেট করুন:

`scoop update --all`

- ইনস্টল করা সব প্যাকেজের তালিকা দেখান:

`scoop list`

- নির্দিষ্ট একটি প্যাকেজ সম্পর্কে তথ্য দেখান:

`scoop info {{প্যাকেজ}}`

- একটি প্যাকেজ অনুসন্ধান করুন:

`scoop search {{প্যাকেজ}}`

- সব প্যাকেজের পুরোনো ভার্সন সরিয়ে ফেলুন এবং ডাউনলোড ক্যাশ পরিষ্কার করুন:

`scoop cleanup --cache --all`
