# getprop

> অ্যান্ড্রয়েড সিস্টেম বৈশিষ্ট্য সম্পর্কে তথ্য দেখুন।
> আরও তথ্য পাবেন: <https://manned.org/getprop>।

- অ্যান্ড্রয়েড সিস্টেম বৈশিষ্ট্য সম্পর্কে তথ্য প্রদর্শন করুন:

`getprop`

- একটি নির্দিষ্ট বৈশিষ্ট্য সম্পর্কে তথ্য প্রদর্শন করুন:

`getprop {{বৈশিষ্ট্য}}`

- SDK API লেভেল প্রদর্শন করুন:

`getprop {{ro.build.version.sdk}}`

- অ্যান্ড্রয়েড ভার্সন প্রদর্শন করুন:

`getprop {{ro.build.version.release}}`

- অ্যান্ড্রয়েড ডিভাইস মডেল প্রদর্শন করুন:

`getprop {{ro.vendor.product.model}}`

- OEM আনলক স্ট্যাটাস প্রদর্শন করুন:

`getprop {{ro.oem_unlock_supported}}`

- অ্যান্ড্রয়েডের Wi-Fi কার্ডের MAC এড্রেস প্রদর্শন করুন:

`getprop {{ro.boot.wifimacaddr}}`
