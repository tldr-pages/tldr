# input

> একটি অ্যান্ড্রয়েড ডিভাইসে ইভেন্ট কোড বা টাচস্ক্রিন অঙ্গভঙ্গি (gestures) পাঠান।
> এই কমান্ডটি শুধুমাত্র `adb shell` এর মাধ্যমে ব্যবহার করা যাবে।
> আরও তথ্য পাবেন: <https://developer.android.com/reference/android/view/KeyEvent.html#constants_1>।

- অ্যান্ড্রয়েড ডিভাইসে একটি একক অক্ষরের জন্য ইভেন্ট কোড পাঠান:

`input keyevent {{ইভেন্ট_কোড}}`

- অ্যান্ড্রয়েড ডিভাইসে একটি টেক্সট পাঠান (`%s` দ্বারা স্পেস বোঝানো হয়):

`input text "{{টেক্সট}}"`

- অ্যান্ড্রয়েড ডিভাইসে একটি একক ট্যাপ পাঠান:

`input tap {{x_অবস্থান}} {{y_অবস্থান}}`

- অ্যান্ড্রয়েড ডিভাইসে একটি সোয়াইপ অঙ্গভঙ্গি পাঠান:

`input swipe {{x_শুরু}} {{y_শুরু}} {{x_শেষ}} {{y_শেষ}} {{সময়_মিলিসেকেন্ড}}`

- একটি সোয়াইপ অঙ্গভঙ্গি ব্যবহার করে একটি অ্যান্ড্রয়েড ডিভাইসে একটি লং প্রেস পাঠান:

`input swipe {{x_অবস্থান}} {{y_অবস্থান}} {{x_অবস্থান}} {{y_অবস্থান}} {{সময়_মিলিসেকেন্ড}}`
