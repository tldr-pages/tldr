# git add

> बदले गए फाइल्स को कमिट के लिए स्टेज करना।
> अधिक जानकारी: <https://git-scm.com/docs/git-add>।

- किसी फाइल को स्टेज करना:

`git add {{path/to/file}}`

- सभी फाइल्स (ट्रैक्ड और अनट्रैक्ड) जोड़ना:

`git add {{-A|--all}}`

- करंट डायरेक्टरी से सभी फाइल्स रेकर्सिवली जोड़ना:

`git add .`

- केवल ट्रैक्ड फाइल्स अपडेट करना:

`git add {{-u|--update}}`

- इग्नोर्ड फाइल को फोर्स करके ऐड करना:

`git add {{-f|--force}} {{path/to/file}}`

- फाइल के कुछ हिस्सों को इंटरएक्टिवली स्टेज करना:

`git add {{-p|--patch}}`

- किसी स्पेसिफिक फाइल के पार्ट्स को इंटरएक्टिवली स्टेज करना:

`git add {{-p|--patch}} {{path/to/file}}`

- इंटरएक्टिवली स्टेज करना:

`git add {{-i|--interactive}}`
