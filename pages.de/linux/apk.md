# apk

> Alpine Linux-Paketverwaltungstool.
> Weitere Informationen: <https://wiki.alpinelinux.org/wiki/Alpine_Linux_package_management>.

`apk update`

- Installiere ein neues Paket:

`apk add {{Paket}}`

- Entferne ein Paket:

`apk del {{Paket}}`
