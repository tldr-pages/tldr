# sam

> AWS Server Application Model (SAM) CLI.
> Mai multe informaţii: <https://github.com/awslabs/aws-sam-cli>

- Iniţializaţi o aplicaţie fără server:

`sam init`

- Inițializați o aplicație fără server cu o anumită execuție:

`sam init --runtime {{python3.7}}`

- Pachet o aplicație SAM:

`sam package`

- Construiește codul funcției Lambda:

`sam build`

- Rulați aplicația fără server local:

`sam local start-api`

- Implementaţi o aplicaţie AWS SAM:

`sam deploy`
