# opkg

> Un manager de pachete ușor utilizat pentru a instala pachete OpenWRT.

- Instalați un pachet:

`opkg install {{package}}`

- Elimină un pachet:

`opkg remove {{package}}`

- Actualizați lista pachetelor disponibile:

`opkg update`

- Upgrade toate pachetele instalate:

`opkg upgrade`

- Upgrade unul sau mai multe pachete specifice:

`opkg upgrade {{package(s)}}`

- Afișarea informațiilor pentru un anumit pachet:

`opkg info {{package}}`

- Listează toate pachetele disponibile:

`opkg list`
