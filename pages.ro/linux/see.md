# see

> Alias pentru vizualizarea `run-mailcap'.
> Un alias pentru o imprimare de acțiune `run-mailcap'.

- A se vedea acțiunea poate fi utilizată pentru a vizualiza orice fișier (de obicei imagine) pe exploratorul implicit mailcap:

`see {{filename}}`

- Utilizarea cu `run-mailcap`:

`run-mailcap --action=view {{filename}}`
