# getent

> Obțineți intrări din bibliotecile de comutare serviciu nume.
> Mai multe informaţii: <https://manned.org/getent>

- Obțineți lista tuturor grupurilor:

`getent group`

- A se vedea membrii unui grup:

`getent group {{group_name}}`

- Obțineți lista tuturor serviciilor:

`getent services`

- Găsiți un nume de utilizator după UID:

`getent passwd 1000`

- Efectuați o căutare DNS inversă:

`getent hosts {{host}}`
