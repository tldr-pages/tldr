# gedit

> Editor de text al proiectului GNOME Desktop.
> Mai multe informaţii: <https://help.gnome.org/users/gedit/stable/>

- Deschideţi un fişier text:

`gedit {{path/to/file}}`

- Deschideți mai multe fișiere text:

`gedit {{file1 file2 ...}}`

- Deschideți un fișier text cu o codificare specifică:

`gedit --encoding={{UTF-8}} {{path/to/file}}`

- Afișează o listă de codificări acceptate:

`gedit --list-encodings`
