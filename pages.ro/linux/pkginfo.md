# pkginfo

> Interogare baza de date pachet pe un sistem CRUX.

- Lista pachetelor instalate și versiunile acestora:

`pkginfo -i`

- Lista fișierelor deținute de un pachet:

`pkginfo -l {{package_name}}`

- Listați proprietarul (proprietarii) fișierelor care se potrivesc unui model:

`pkginfo -o {{pattern}}`

- Imprimați amprenta unui fișier:

`pkginfo -f {{file}}`
