# finch

> Client de mesagerie modulară bazată pe consolă.
> Mai multe informaţii: <https://developer.pidgin.im/wiki/Using%20Finch>

- Lansează cinteza:

`finch`

- Renunţă:

`alt + q or ctrl + c`

- Afișează meniul de acțiuni:

`alt + a`

- Sari la fereastra n-lea:

`alt + {{number_key}}`

- Închide fereastra curentă:

`alt + c`

- Începeți să mutați o fereastră, utilizați tastele săgeată pentru a vă deplasa, apăsați escape când ați terminat:

`alt + m`

- Începeți redimensionarea unei ferestre, utilizați tastele săgeată pentru a redimensiona, apăsați escape când ați terminat:

`alt + r`
