# toilet

> Un instrument pentru afişarea fonturilor ASCII-art.
> Mai multe informaţii: <http://caca.zoy.org/wiki/toilet>

- Generează arta ASCII pentru un text dat:

`toilet {{input_text}}`

- Generează arta ASCII folosind un fișier de font personalizat:

`toilet {{input_text}} -f {{font_filename}}`

- Generează arta ASCII folosind un filtru:

`toilet {{input_text}} --filter {{filter_name}}`

- Afișați filtrele de toaletă disponibile:

`toilet --filter list `
