# print

> Un alias pentru o imprimare de acțiune `run-mailcap'.
> Inițial `run-mailcap` este utilizat pentru procesarea tipului/fișierului MIME.

- Acțiunea de imprimare poate fi utilizată pentru a imprima orice fișier pe instrumentul implicit de run-mailcap:

`print {{filename}}`

- Cu `run-mailcap':

`run-mailcap --action=print {{filename}}`
