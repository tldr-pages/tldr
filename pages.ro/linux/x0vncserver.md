# x0vncserver

> Server TigerVNC pentru ecrane X.
> Mai multe informaţii: <https://tigervnc.org/doc/x0vncserver.html>

- Porniți un server VNC folosind un fișier de parole:

`x0vncserver -display {{:0}} -passwordfile {{path/to/file}}`

- Porniți un server VNC folosind un anumit port:

`x0vncserver -display {{:0}} -rfbport {{port}}`
