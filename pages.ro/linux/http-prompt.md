# http-prompt

> Un client interactiv HTTP de linie de comandă cu completare automată și evidențierea sintaxei.

- Lansarea unei sesiuni care vizează adresa URL implicită a http://localhost:8000 sau sesiunea anterioară:

`http-prompt`

- Lansarea unei sesiuni cu un URL dat:

`http-prompt {{http://example.com}}`

- Lansarea unei sesiuni cu câteva opțiuni inițiale:

`http-prompt {{localhost:8000/api}} --auth {{username:password}}`
