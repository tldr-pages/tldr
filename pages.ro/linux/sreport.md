# sreport

> Generați rapoarte despre locuri de muncă, utilizatori și clustere din date contabile.
> Mai multe informaţii: <https://slurm.schedmd.com/sreport.html>

- Afișați datele de utilizare a clusterului delimitat:

`sreport --parsable cluster utilization`

- Arată numărul de locuri de muncă executate:

`sreport job sizes printjobcount`

- Arată utilizatorii cu cea mai mare utilizare a timpului CPU:

`sreport user topuser`
