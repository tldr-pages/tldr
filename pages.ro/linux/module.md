# module

> Modificați mediul utilizatorilor utilizând comanda modulului.
> Mai multe informaţii: <https://lmod.readthedocs.io/en/latest/010_user.html>

- Afișează modulele disponibile:

`module avail`

- Căutați un modul după nume:

`module spider {{module_name}}`

- Încarcă un modul:

`module load {{module_name}}`

- Module încărcate pe ecran

`module list`

- Descărcaţi un modul încărcat specific:

`module {{module_name}}`

- Descărcaţi toate modulele încărcate:

`module purge`
