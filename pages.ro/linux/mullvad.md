# mullvad

> Client CLI pentru Mullvad VPN.
> Mai multe informaţii: <https://mullvad.net/>

- Conectați contul dvs. mullvad cu numărul de cont specificat:

`mullvad account set {{account_number}}`

- Activați accesul LAN în timp ce VPN este activat:

`mullvad lan set allow`

- Stabilirea tunelului VPN:

`mullvad connect`

- Verificaţi starea tunelului VPN:

`mullvad status`
