# zile

> Zile este o clonă ușoară a editorului de text Emacs.
> Mai multe informaţii: <https://www.gnu.org/software/zile/>

- Porniți un tampon pentru note temporare, care nu vor fi salvate:

`zile`

- Deschide un fișier:

`zile {{path/to/file}}`

- Salvează un fișier:

`Ctrl + X, Ctrl + S`

- Renunţă:

`Ctrl + X, Ctrl + C`

- Deschideți un fișier la un număr de linie specificat:

`zile +{{line_number}} {{path/to/file}}`

- Anulează modificările:

`Ctrl + X, U`
