# i3

> Un manager dinamic de ferestre tigla.
> Mai multe informaţii: <https://i3wm.org/docs/userguide.html>

- Start i3 (Rețineți că un manager de ferestre preexistent nu trebuie să fie deschis atunci când se execută această comandă. ):

`i3`

- Deschide o fereastră nouă de terminal:

`Super + Return`

- Creați un nou spațiu de lucru:

`Super + Shift + {{number}}`

- Treceți la spațiul de lucru {{number}}:

`Super + {{number}}`

- Deschide fereastra noua orizontal:

`Super + h`

- Deschide fereastra noua pe verticala:

`Super + v`

- Deschideți aplicația (tastați numele aplicației după executarea comenzii):

`Super + D`
