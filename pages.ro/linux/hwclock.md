# hwclock

> Utilizat pentru citirea sau modificarea ceasului hardware. De obicei necesită rădăcină.

- Afișează ora curentă așa cum este raportată de ceasul hardware:

`hwclock`

- Scrieți ora curentă a ceasului software la ceasul hardware (uneori folosit în timpul configurării sistemului):

`hwclock --systohc`

- Scrieți ora curentă a ceasului hardware la ceasul software:

`hwclock --hctosys`
