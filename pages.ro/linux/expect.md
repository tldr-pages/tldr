# expect

> Executor script care interacționează cu alte programe care necesită introducerea utilizatorului.
> Mai multe informaţii: <https://manned.org/expect>

- Execută un script de așteptare dintr-un fișier:

`expect {{path/to/file}}`

- Executați un script de așteptare specificat:

`expect -c "{{commands}}"`

- Introduceți un REPL interactiv (utilizați `ieșire” sau Ctrl + D pentru a ieși):

`expect -i`
