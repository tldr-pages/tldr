# fc-cache

> Scanați directoarele de fonturi pentru a construi fișiere cache de font.
> Mai multe informaţii: <https://manned.org/fc-cache>

- Generează fișiere cache de font:

`fc-cache`

- Forțați o reconstruire a tuturor fișierelor cache de font, fără a verifica dacă memoria cache este actualizată:

`fc-cache -f`

- Ștergeți fișierele cache de font, apoi generați fișiere cache de font noi:

`fc-cache -r`
