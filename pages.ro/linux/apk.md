# apk

> Alpine Linux instrument de gestionare a pachetelor.
> Mai multe informaţii: <https://wiki.alpinelinux.org/wiki/Alpine_Linux_package_management>

- Actualizați indexurile depozitelor din toate depozitele la distanță:

`apk update`

- Instalați un pachet nou:

`apk add {{package}}`

- Elimină un pachet:

`apk del {{package}}`

- Reparați pachetul sau actualizați-l fără a modifica dependențele principale:

`apk fix {{package}}`

- Pachet de căutare prin cuvinte cheie:

`apk search {{keyword}}`

- Obțineți informații despre un anumit pachet:

`apk info {{package}}`
