# ul

> Efectuează sublinierea unui text.
> Fiecare caracter dintr-un șir dat trebuie subliniat separat.
> Mai multe informaţii: <https://manned.org/ul>

- Afișează conținutul fișierului cu sublinierea, dacă este cazul:

`ul {{file.txt}}`

- Afișează conținutul fișierului cu sublinierea făcută din liniuțe `-`:

`ul -i {{file.txt}}`
