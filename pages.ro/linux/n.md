# n

> Instrument pentru gestionarea mai multor versiuni de noduri.

- Instalați o versiune dată a nodului. Dacă versiunea este deja instalată, aceasta va fi activată:

`n {{version}}`

- Afișați versiunile instalate și activați interactiv una dintre ele:

`n`

- Elimină o versiune:

`n rm {{version}}`

- Executați un fișier cu o versiune dată:

`n use {{version}} {{file.js}}`

- Ieșire cale binară pentru o versiune:

`n bin {{version}}`
