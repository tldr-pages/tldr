# ascii

> Afișează aliasuri de caractere ASCII.
> Mai multe informaţii: <http://www.catb.org/~esr/ascii/>

- Arată aliasurile ASCII ale unui caracter:

`ascii {{a}}`

- Arată aliasuri ASCII în modul scurt, script-friendly:

`ascii -t {{a}}`

- Arată aliasuri ASCII cu mai multe caractere:

`ascii -s {{tldr}}`

- Afișează tabelul ASCII în zecimală:

`ascii -d`

- Afișează tabelul ASCII în hexazecimal:

`ascii -x`

- Arată tabelul ASCII în octal:

`ascii -o`

- Arată tabelul ASCII în binar:

`ascii -b`

- Afișarea opțiunilor rezumat și tabel complet ASCII:

`ascii`
