# mdbook

> Creați cărți online scriind fișiere makrdown.
> Mai multe informaţii: <https://rust-lang.github.io/mdBook/>

- Creați un proiect mdbook în directorul curent:

`mdbook init`

- Creați un proiect mdbook într-un anumit director:

`mdbook init {{path/to/directory}}`

- Curățați directorul cu cartea generată:

`mdbook clean`

- Serviți o carte la `http://localhost:3000`, construi auto atunci când se schimbă fișierul:

`mdbook serve`

- Urmăriți un set de fișiere Markdown și construiți automat atunci când un fișier este schimbat:

`mdbook watch`
