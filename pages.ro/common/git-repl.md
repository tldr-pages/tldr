# git repl

> Git REPL (citire-evaluat-imprimantă buclă) - o coajă Git interactivă.
> Parte din `git-extras`.
> Mai multe informaţii: <https://github.com/tj/git-extras/blob/master/Commands.md#git-repl>

- Începe o coajă Git interactivă:

`git repl`

- Rulați o comandă Git în timp ce în shell-ul Git interactiv:

`{{git_subcommand}} {{command_arguments}}`

- Rulați o comandă externă (non-GIT) în timp ce în shell-ul Git interactiv:

`!{{command}} {{command_arguments}}`

- Ieșiți din carcasa interactivă Git (sau apăsați Ctrl + D):

`exit`
