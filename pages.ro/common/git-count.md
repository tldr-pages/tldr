# git count

> Imprimați numărul total de angajări.
> Parte din `git-extras`.
> Mai multe informaţii: <https://github.com/tj/git-extras/blob/master/Commands.md#git-count>

- Imprimați numărul total de angajări:

`git count`

- Imprimați numărul de angajări per contribuitor și numărul total de angajări:

`git count --all`
