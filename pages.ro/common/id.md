# id

> Afișează identitatea curentă a utilizatorului și a grupului.
> Mai multe informaţii: <https://www.gnu.org/software/coreutils/id>

- Afișează ID-ul utilizatorului curent (UID), ID-ul grupului (GID) și grupurile cărora le aparțin:

`id`

- Afișează identitatea curentă a utilizatorului ca număr:

`id -u`

- Afișează identitatea curentă a grupului ca număr:

`id -g`

- Afișează ID-ul unui utilizator arbitrar (UID), ID-ul grupului (GID) și grupurile cărora le aparțin:

`id {{username}}`
