# git missing

> Afișați angajamente care nu sunt partajate între două ramuri.
> Parte din `git-extras`.
> Mai multe informaţii: <https://github.com/tj/git-extras/blob/master/Commands.md#git-missing>

- Afișați angajările care nu sunt partajate între sucursala verificată în prezent și o altă sucursală:

`git missing {{branch}}`

- Arată angajamente care nu sunt partajate între două ramuri:

`git missing {{branch_1}} {{branch_2}}`
