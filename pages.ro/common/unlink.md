# unlink

> Eliminați un link către un fișier din sistemul de fișiere.
> Conținutul fișierului se pierde dacă linkul este ultimul la fișier.
> Mai multe informaţii: <https://www.gnu.org/software/coreutils/unlink>

- Eliminați fișierul specificat dacă acesta este ultimul link:

`unlink {{path/to/file}}`
