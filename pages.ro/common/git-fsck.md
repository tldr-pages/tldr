# git fsck

> Verificați validitatea și conectivitatea nodurilor într-un index de depozit Git.
> Nu face nici o modificare. A se vedea `git gc` pentru a curăța blobs agățat.
> Mai multe informaţii: <https://git-scm.com/docs/git-fsck>

- Verificați depozitul curent:

`git fsck`

- Listează toate etichetele găsite:

`git fsck --tags`

- Listează toate nodurile rădăcină găsite:

`git fsck --root`
