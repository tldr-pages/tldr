# kube-capacity

> Un CLI simplu care oferă o prezentare generală a solicitărilor de resurse, limitelor și utilizării într-un cluster Kubernetes.
> Combinați cele mai bune părți ale `kubectl top` și `kubectl describe` într-un CLI axat pe resursele clusterului.
> Mai multe informaţii: <https://github.com/robscott/kube-capacity>

- Ieșiți o listă de noduri cu cererile și limitele totale de resurse CPU și memorie:

`kube-capacity`

- Include păstăi:

`kube-capacity -p`

- Include utilizarea:

`kube-capacity -u`
