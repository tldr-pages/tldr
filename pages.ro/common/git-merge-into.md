# git merge-into

> Fuzionează o ramură în altă ramură.
> Face parte din `git-extras`.
> Mai multe informații: <https://github.com/tj/git-extras/blob/main/Commands.md#git-merge-into>.

- Fuzionează o ramură sursă într-o ramură destinație specifică:

`git merge-into {{source_branch}} {{destination_branch}}`

- Fuzionează ramura curentă într-o ramură destinație specifică:

`git merge-into {{destination_branch}}`
