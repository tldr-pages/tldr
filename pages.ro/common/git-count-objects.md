# git count-objects

> Numărați numărul de obiecte despachetate și consumul lor de disc.
> Mai multe informaţii: <https://git-scm.com/docs/git-count-objects>

- Numărați toate obiectele și afișați utilizarea totală a discului:

`git count-objects`

- Afișează un număr de toate obiectele și utilizarea totală a discului, afișând dimensiuni în unități lizibile umane:

`git count-objects --human-readable`

- Afișează mai multe informații detaliate:

`git count-objects --verbose`

- Afișează mai multe informații verbose, afișând dimensiuni în unități lizibile umane:

`git count-objects --human-readable --verbose`
