# pip

> Manager de pachete Python.
> Mai multe informaţii: <https://pip.pypa.io>

- Instalați un pachet (consultați `pip install` pentru mai multe exemple de instalare):

`pip install {{package_name}}`

- Upgrade un pachet:

`pip install -U {{package_name}}`

- Dezinstalați un pachet:

`pip uninstall {{package_name}}`

- Salvați pachetele instalate în fișier:

`pip freeze > {{requirements.txt}}`

- Afișează informațiile despre pachetul instalat:

`pip show {{package_name}}`
