# doctum

> Un generator de documentare API PHP.
> Mai multe informaţii: <https://github.com/code-lts/doctum>

- Antreprezi un proiect:

`doctum parse`

- Randarea unui proiect:

`doctum render`

- Parse apoi face un proiect:

`doctum update`

- Parsează și redă doar o versiune specifică a unui proiect:

`doctum update --only-version={{version}}`

- Parsează și redă un proiect utilizând o configurație specifică:

`doctum update {{path/to/config.php}}`
