# firebase

> Testați, gestionați și implementați proiecte Firebase din linia de comandă.
> Mai multe informaţii: <https://github.com/firebase/firebase-tools>

- Autentificaţi-vă la  <https://console.firebase.google.com>:

`firebase login`

- Lista proiectelor de bază existente:

`firebase projects:list`

- Porniți un expert interactiv pentru a crea un proiect Firebase în directorul curent:

`firebase init`

- Implementați codul și activele la proiectul curent Firebase:

`firebase deploy`

- Porniți un server local pentru a găzdui static activele proiectului Firebase curente:

`firebase serve`

- Porniți un expert interactiv pentru a deschide unul dintre multele linkuri ale proiectului Firebase curent în browserul web implicit:

`firebase open`
