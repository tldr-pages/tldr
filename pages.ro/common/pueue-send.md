# pueue send

> Trimitere intrare la o activitate.
> Mai multe informaţii: <https://github.com/Nukesor/pueue>

- Trimiteți intrarea la o comandă de alergare:

`pueue send {{task_id}} "{{input}}"`

- Trimiteți confirmarea unei sarcini care așteaptă Y/n (de exemplu, apt, cp):

`pueue send {{task_id}} {{y}}`
