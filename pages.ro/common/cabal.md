# cabal

> Interfață linie de comandă pentru infrastructura pachetului Haskell (Cabal).
> Gestionați proiectele Haskell și pachetele Cabal din depozitul de pachete Hackage.
> Mai multe informaţii: <https://cabal.readthedocs.io/en/latest/intro.html>

- Căutați și listați pachete din Hackage:

`cabal list {{search_string}}`

- Afișați informații despre un pachet:

`cabal info {{package_name}}`

- Descărcați și instalați un pachet:

`cabal install {{package_name}}`

- Creați un nou proiect Haskell în directorul curent:

`cabal init`

- Construiți proiectul în directorul curent:

`cabal build`

- Executați teste ale proiectului în directorul curent:

`cabal test`
