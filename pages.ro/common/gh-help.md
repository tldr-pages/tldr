# gh help

> Afișați ajutor despre comanda GitHub CLI.
> Mai multe informaţii: <https://cli.github.com/manual/gh_help>

- Afișează ajutor general:

`gh help`

- Afișează ajutor pentru subcomanda `gh help`:

`gh help --help`

- Afișează ajutor despre variabilele de mediu care pot fi utilizate cu `gh`:

`gh help environment`

- Afișează o referință de marcare a tuturor comenzilor `gh`:

`gh help reference`

- Afișează ajutor despre formatarea ieșirii JSON din `gh` folosind `jq`:

`gh help formatting`

- Afișează ajutor despre utilizarea `gh` cu Mintty:

`gh help mintty`

- Afișează ajutor pentru o subcomandă:

`gh help {{subcommand}}`

- Afișează ajutor pentru o acțiune de subcomandă:

`gh help {{pr}} {{create}}`
