# git local-commits

> Afișați angajările locale care nu au fost împinse la origine. Orice argumente suplimentare vor fi transmise direct la `git log'.
> O parte din `git-extras.
> Mai multe informaţii: <https://github.com/tj/git-extras/blob/master/Commands.md#git-local-commits>

- Arată comite care nu au fost împinse:

`git local-commits`
