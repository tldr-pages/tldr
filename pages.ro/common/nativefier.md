# nativefier

> Instrument linie de comandă pentru a crea o aplicație desktop pentru orice site web cu configurație minimă.
> Mai multe informaţii: <https://github.com/jiahaog/nativefier>

- Faceți o aplicație desktop pentru un site web:

`nativefier {{url}}`

- Creați o aplicație desktop cu un nume personalizat:

`nativefier --name {{name}} {{url}}`

- Utilizați o pictogramă personalizată, ar trebui să fie un PNG:

`nativefier --icon {{path/to/icon.png}} {{url}}`
