# glow

> Randare Markdown în terminal.
> Mai multe informaţii: <https://github.com/charmbracelet/glow>

- Rulați strălucire și selectați un fișier pentru a vizualiza:

`glow`

- Randați un fișier Markdown la terminal:

`glow {{path/to/file}}`

- Vizualizaţi un fişier Markdown utilizând un paginator:

`glow -p {{path/to/file}}`

- Vizualizați un fișier dintr-un URL:

`glow {{https://example.com/file.md}}`

- Vezi un GitHub/GitLab README:

`glow {{github.com/owner/repository}}`
