# gitmoji

> Un instrument interactiv de linie de comandă pentru utilizarea emoticonurilor la comite.
> Mai multe informaţii: <https://github.com/carloscuesta/gitmoji-cli>

- Porniți expertul de comitere:

`gitmoji --commit`

- Inițializați cârligul git (așa că `gitmoji” va fi rulat de fiecare dată când `git comitet este rulat):

`gitmoji --init`

- Scoateți cârligul git:

`gitmoji --remove`

- Listează toate emoticonurile disponibile și descrierile lor:

`gitmoji --list`

- Căutare lista de emoji pentru o listă de cuvinte cheie:

`gitmoji --search {{keyword1}} {{keyword2}}`

- Actualizați lista memorată în cache de emoticoane din depozitul principal:

`gitmoji --update`

- Configurarea preferințelor globale:

`gitmoji --config`
