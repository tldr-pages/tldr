# pip install

> Instalați pachetele Python.
> Mai multe informaţii: <https://pip.pypa.io>

- Instalați un pachet:

`pip install {{package_name}}`

- Instalați o versiune specifică a unui pachet:

`pip install {{package_name}}=={{package_version}}`

- Instalați pachetele listate într-un fișier:

`pip install -r {{requirements.txt}}`

- Instalați pachetul local în directorul curent în modul de dezvoltare (editabil):

`pip install -e .`
