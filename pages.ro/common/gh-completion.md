# gh completion

> Generați scripturi de completare a shell-ului pentru comenzile GitHub CLI.
> Mai multe informaţii: <https://cli.github.com/manual/gh_completion>

- Afișează ajutorul subcomenzii:

`gh completion`

- Imprimați un script de finalizare:

`gh completion --shell {{bash|zsh|fish|powershell}}`

- Adaugă script-ul de completare `gh` la `~/.bashrc`:

`gh completion --shell {{bash}} >> {{~/.bashrc}}`

- Adaugă script-ul de completare `gh` la `~/.zshrc`:

`gh completion --shell {{zsh}} >> {{~/.zshrc}}`
