# standard

> Instrumentul Style Standard JavaScript pentru linting și fixarea codului JavaScript.
> Mai multe informaţii: <https://standardjs.com>

- Lint toate fișierele sursă JavaScript din directorul curent:

`standard`

- Fișier (e) JavaScript specific (e):

`standard {{path/to/file(s)}}`

- Aplicați remedieri automate în timpul ninting:

`standard --fix`

- Declarați orice variabile globale disponibile:

`standard --global {{variable}}`

- Folosește un plugin personalizat eSLint atunci când ninting:

`standard --plugin {{plugin}}`

- Utilizați un parser JS personalizat atunci când ninting:

`standard --parser {{parser}}`

- Utilizaţi un mediu eSLint personalizat atunci când vă lănţui:

`standard --env {{environment}}`
