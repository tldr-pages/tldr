# act

> Executați Acțiuni GitHub local utilizând Docker.
> Mai multe informaţii: <https://github.com/nektos/act>

- Listează acțiunile disponibile:

`act -l`

- Rulați evenimentul implicit:

`act`

- Rulați un anumit eveniment:

`act {{event_type}}`

- Rulați o acțiune specifică:

`act -a {{action_id}}`

- Nu executați efectiv acțiunile (adică un termen uscat):

`act -n`

- Arată jurnalele verbose:

`act -v`
