# git browse-ci

> Deschide site-ul CI al repository-ului Git curent în browserul web implicit.
> Parte din `git-extras`.
> Mai multe informații: <https://github.com/tj/git-extras/blob/main/Commands.md#git-browse-ci>.

- Deschide configurația CI a repository-ului curent pe site-ul său upstream:

`git browse-ci`

- Deschide configurația CI a repository-ului curent pe site-ul său upstream pentru un remote specific:

`git browse-ci {{la_distanță}}`
