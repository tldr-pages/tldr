# makebuildserver

> Creați o mașină virtuală de server de compilare F-Droid.
> Mai multe informaţii: <https://f-droid.org/en/docs/Build_Server_Setup/>

- Creați o nouă mașină virtuală sau actualizați una existentă (dacă este disponibilă):

`makebuildserver`

- Forța crearea unei mașini virtuale proaspete:

`makebuildserver --clean`
