# git browse

> Vizualizează un repository upstream în browserul implicit.
> Parte din `git-extras`.
> Mai multe informații: <https://github.com/tj/git-extras/blob/main/Commands.md#git-browse>.

- Deschide primul upstream în browserul implicit:

`git browse`

- Deschide un upstream specific în browserul implicit:

`git browse {{upstream}}`
