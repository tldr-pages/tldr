# git browse

> Vizualizați un depozit în amonte în browserul implicit.
> Parte din `git-extras`.
> Mai multe informaţii: <https://github.com/tj/git-extras/blob/master/Commands.md#git-browse>

- Deschideți primul în amonte în browserul implicit:

`git browse`

- Deschideți un anumit amonte în browserul implicit:

`git browse {{upstream}}`
