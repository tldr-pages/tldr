# git whatchanged

> Afișați ce s-a modificat cu angajările recente sau fișierele.
> A se vedea, de asemenea, `git log`.
> Mai multe informaţii: <https://git-scm.com/docs/git-whatchanged>

- Afișează jurnalele și modificările pentru angajările recente:

`git whatchanged`

- Afișează jurnalele și modificările pentru angajările recente în intervalul de timp specificat:

`git whatchanged --since="{{2 hours ago}}"`

- Afișează jurnalele și modificările pentru angajamente recente pentru anumite fișiere sau directoare:

`git whatchanged {{path/to/file_or_directory}}`
