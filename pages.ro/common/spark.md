# spark

> Instrumentul de linie de comandă scânteie Laravel.
> Mai multe informaţii: <https://spark.laravel.com>

- Înregistrați-vă token-ul API:

`spark register {{token}}`

- Afișează indicativul API înregistrat în prezent:

`spark token`

- Creați un nou proiect Spark:

`spark new {{project_name}}`

- Creați un nou proiect Spark cu cioturi Braintree:

`spark new {{project_name}} --braintree`

- Creați un nou proiect Spark cu coturi de facturare bazate pe echipă:

`spark new {{project_name}} --team-billing`
