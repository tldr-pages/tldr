# git gc

> Optimizați depozitul local prin curățarea fișierelor inutile.
> Mai multe informaţii: <https://git-scm.com/docs/git-gc>

- Optimizarea depozitului:

`git gc`

- Optimizarea agresivă, durează mai mult timp:

`git gc --aggressive`

- Nu prune obiecte vrac (prune uscate în mod implicit):

`git gc --no-prune`

- Suprimă toate ieșirile:

`git gc --quiet`

- Vizualizare utilizare completă:

`git gc --help`
