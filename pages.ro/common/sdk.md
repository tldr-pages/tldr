# sdk

> Instrument pentru gestionarea versiunilor paralele ale mai multor kituri de dezvoltare software.
> Suportă Java, Groovy, Scala, Kotlin, Gradle, Maven, Vert.x și multe altele.
> Mai multe informaţii: <https://sdkman.io/usage>

- Instalați o versiune specifică a Gradle:

`sdk install {{gradle}} {{gradle_version}}`

- Treceți la o versiune specifică a Gradle:

`sdk use {{gradle}} {{gradle_version}}`

- Verificați versiunea curentă Gradle:

`sdk current {{gradle}}`

- Listează toate kituri de dezvoltare software disponibile pentru instalare:

`sdk list`

- Actualizați Gradle la cea mai recentă versiune:

`sdk upgrade {{gradle}}`

- Dezinstalați o anumită versiune a Gradle:

`sdk rm {{gradle}} {{gradle_version}}`
