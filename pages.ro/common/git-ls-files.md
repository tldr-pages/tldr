# git ls-files

> Afișați informații despre fișierele din index și arborele de lucru.
> Mai multe informaţii: <https://git-scm.com/docs/git-ls-files>

- Arată fişierele şterse:

`git ls-files --deleted`

- Afișează fișierele modificate și șterse:

`git ls-files --modified`

- Afișează fișierele ignorate și neurmărite:

`git ls-files --others`
