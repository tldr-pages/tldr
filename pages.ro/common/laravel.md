# laravel

> Un program de instalare de linie de comandă pentru cadrul Laravel.
> Mai multe informaţii: <https://laravel.com>

- Creați o nouă aplicație Laravel:

`laravel new {{name}}`

- Utilizați cea mai recentă versiune de dezvoltare:

`laravel new {{name}} --dev`

- Suprascrie dacă directorul există deja:

`laravel new {{name}} --force`

- Instalați schela Laravel Jetstream:

`laravel new {{name}} --jet`

- Instalați schela Laravel Jetstream cu o stivă specifică:

`laravel new {{name}} --jet --stack {{livewire|inertia}}`

- Instalați schela Laravel Jetstream cu suport pentru echipe:

`laravel new {{name}} --jet --teams`

- Listează comenzile de instalare disponibile:

`laravel list`
