# pastel

> Generați, analizați, convertiți și manipulați culorile.
> Mai multe informaţii: <https://github.com/sharkdp/pastel>

- Conversia culorilor dintr-un format în altul. Aici de la RGB la HSL:

`pastel format {{hsl}} {{ff8000}}`

- Afișați și analizați culorile de pe terminal:

`pastel color "{{rgb(255,50,127)}}"`

- Alege o culoare de undeva de pe ecran:

`pastel pick`

- Generează un set de N culori distincte vizual:

`pastel distinct {{8}}`

- Obțineți o listă cu toate numele de culoare X11/ CSS:

`pastel list`
