# git merge-tree

> Fuzionează ramuri fără a modifica fișierele.
> Mai multe informații: <https://git-scm.com/docs/git-merge-tree>.

- Afișează rezultatul unei fuziuni între două ramuri:

`git merge-tree {{branch1}} {{branch2}}`

- Efectuează o fuziune și scrie arborele rezultat:

`git merge-tree --write-tree {{branch1}} {{branch2}}`
