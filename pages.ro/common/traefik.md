# traefik

> Un proxy invers HTTP și echilibrator de încărcare.
> Mai multe informaţii: <https://traefik.io>

- Porniți serverul cu configurare implicită:

`traefik`

- Porniți serverul cu un fișier de configurare personalizat:

`traefik --c {{config_file}}.toml`

- Porniți serverul cu modul de cluster activat:

`traefik --cluster`

- Porniți serverul cu interfața web activată:

`traefik --web`
