# git blame

> Afișați hash comite și ultimul autor pe fiecare linie a unui fișier.
> Mai multe informaţii: <https://git-scm.com/docs/git-blame>

- Imprimați fișierul cu numele autorului și comite hash pe fiecare linie:

`git blame {{file}}`

- Imprimați fișierul cu e-mail autor și comite hash pe fiecare linie:

`git blame -e {{file}}`
