# netlify

> Implementați site-uri și configurați implementarea continuă pe platforma Netlify.
> Mai multe informaţii: <https://www.netlify.com/docs/cli/>

- Conectați-vă la contul Netlify:

`netlify login`

- Implementați conținutul unui director în Netlify:

`netlify deploy`

- Configurarea implementării continue pentru un site nou sau existent:

`netlify init`

- Porniți un server local de dev:

`netlify dev`
