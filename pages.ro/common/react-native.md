# react-native

> Un cadru pentru construirea de aplicații native cu React.
> Mai multe informaţii: <https://reactnative.dev>

- Inițializați un nou proiect React Native într-un director cu același nume:

`react-native init {{project_name}}`

- Porniţi legătura cu metroul:

`react-native start`

- Porniți pachetul de metrou cu un cache curat:

`react-native start --reset-cache`

- Construiți aplicația curentă și porniți-o pe un dispozitiv Android conectat sau emulator:

`react-native run-android`

- Construiți aplicația curentă și porniți-o pe un simulator iOS:

`react-native run-ios`

- Construiți aplicația curentă în modul `lansare” și porniți-l pe un dispozitiv Android conectat sau emulator:

`react-native run-android --variant={{release}}`

- Porniți `logkitty` și imprimați jurnalele la stdout:

`react-native log-android`

- Start `tail system.log` pentru un simulator iOS și jurnalele de imprimare la stdout:

`react-native log-ios`
