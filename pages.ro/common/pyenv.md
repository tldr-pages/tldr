# pyenv

> Comutați cu ușurință între mai multe versiuni de Python.
> Mai multe informaţii: <https://github.com/pyenv/pyenv>

- Listează toate comenzile disponibile:

`pyenv commands`

- Listează toate versiunile Python în directorul `$ {PYENV_ROOT} /versions`:

`pyenv versions`

- Instalați o versiune Python în directorul `$ {PYENV_ROOT} /versions`:

`pyenv install {{2.7.10}}`

- Dezinstalați o versiune Python în directorul `$ {PYENV_ROOT} /versions`:

`pyenv uninstall {{2.7.10}}`

- Setați versiunea Python pentru a fi utilizată la nivel global în mașina curentă:

`pyenv global {{2.7.10}}`

- Setați versiunea Python pentru a fi utilizată în directorul curent și toate directoarele de mai jos:

`pyenv local {{2.7.10}}`
