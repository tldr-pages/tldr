# tlmgr

> Gestionează pachetele şi opţiunile de configurare ale unei instalări teX Live existente.
> Mai multe informaţii: <https://www.tug.org/texlive/tlmgr.html>

- Instalați un pachet și dependențele sale:

`tlmgr install {{package}}`

- Eliminați un pachet și dependențele sale:

`tlmgr remove {{package}}`

- Afișează informații despre un pachet:

`tlmgr info {{package}}`

- Actualizaţi toate pachetele:

`tlmgr update --all`

- Afișați actualizări posibile fără a actualiza nimic:

`tlmgr update --list`

- Începe o versiune GUI de tlmgr:

`tlmgr gui`

- Listează toate configurațiile teX Live:

`tlmgr conf`
