# meteor

> Platformă JavaScript full-stack pentru construirea de aplicații web.
> Mai multe informaţii: <https://meteor.com>

- Rulați un proiect meteoriți din directorul său rădăcină în modul de dezvoltare:

`meteor`

- Creați un proiect în directorul dat:

`meteor create {{path/to/directory}}`

- Afișează lista pachetelor pe care proiectul le utilizează în prezent:

`meteor list`

- Adaugă un pachet la proiect:

`meteor add {{package_name}}`

- Eliminați un pachet din proiect:

`meteor remove {{package_name}}`

- Creați o construcție de producție a proiectului ca tarball în directorul dat:

`meteor build {{path/to/directory}}`
