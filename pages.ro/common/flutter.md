# flutter

> SDK-ul aplicației mobile gratuite, open source și cross-platform Google.
> Mai multe informaţii: <https://github.com/flutter/flutter/wiki/The-flutter-tool>

- Afișați ajutor despre o anumită comandă:

`flutter help {{command}}`

- Verificați dacă toate instrumentele externe sunt instalate corect:

`flutter doctor`

- Lista sau modificarea canalului flutter:

`flutter channel {{stable|beta|dev|master}}`

- Rulați Flutter pe toate emulatoarele pornite și dispozitivele conectate:

`flutter run -d all`

- Descărcați toate pachetele specificate în `pubspec.yaml`:

`flutter pub get`

- Rulați teste într-un terminal de la rădăcina proiectului:

`flutter test {{test/example_test.dart}}`

- Construiți o versiune APK care vizează cele mai multe smartphone-uri moderne:

`flutter build apk --target-platform {{android-arm}},{{android-arm64}}`
