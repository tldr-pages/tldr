# now

> Platformă cloud pentru implementare fără server.
> Această comandă este învechită. A se vedea `vercel`, versiunea actualizată a acestui instrument.
> Mai multe informaţii: <https://zeit.co/now>

- Implementați directorul curent:

`now`

- Afișează o listă de implementări:

`now list`

- Afișează informații legate de o implementare:

`now inspect {{deployment_url}}`

- Eliminați o implementare:

`now remove {{deployment_id}}`

- Conectați-vă la un cont sau creați unul nou:

`now login`

- Inițializați un exemplu de proiect (va fi creat un nou director):

`now init`
