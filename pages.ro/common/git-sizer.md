# git sizer

> Calculează diferite valori ale dimensiunilor depozitului Git și vă avertizează cu privire la orice lucru care ar putea cauza probleme sau inconveniente.
> Mai multe informaţii: <https://github.com/github/git-sizer>

- Raportați numai statistici care au un nivel de îngrijorare mai mare de 0:

`git sizer`

- Raportează toate statisticile:

`git sizer -v`

- A se vedea opțiunile suplimentare:

`git sizer -h`
