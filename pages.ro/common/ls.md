# ls

> Listează conținutul directorului.
> Mai multe informaţii: <https://www.gnu.org/software/coreutils/ls>

- Lista fişierelor una pe linie:

`ls -1`

- Listați toate fișierele, inclusiv fișierele ascunse:

`ls -a`

- Listează toate fișierele, cu final `/` adăugat la numele directorului:

`ls -F`

- Lista de formate lungi (permisiuni, proprietate, dimensiune și data modificării) a tuturor fișierelor:

`ls -la`

- Lista de format lung cu dimensiunea afișată folosind unități lizibile umane (KiB, MiB, GiB):

`ls -lh`

- Lista de format lung sortate după mărime (descrescător):

`ls -lS`

- Lista de format lung a tuturor fișierelor, sortate după data modificării (prima cea mai veche):

`ls -ltr`

- Doar listele de directoare:

`ls -d {{*/}}`
