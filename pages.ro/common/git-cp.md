# git cp

> Copiați un fișier existent într-o locație nouă, păstrând istoricul.
> Parte din `git-extras`.
> Mai multe informaţii: <https://github.com/tj/git-extras/blob/master/Commands.md#git-cp>

- Copiați un fișier existent într-un repo Git, rămânând în același director:

`git cp {{file}} {{new_file}}`

- Copiați un fișier existent într-un repo Git și plasați-l în altă parte:

`git cp {{path/to/file}} {{path/to/new_file}}`
