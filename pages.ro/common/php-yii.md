# php yii

> Interfața de linie de comandă a Yii Framework.
> Mai multe informaţii: <https://yiiframework.com>

- Afișează o listă a tuturor comenzilor disponibile:

`php yii {{help}}`

- Porniți serverul web încorporat al PHP pentru aplicația Yii curentă:

`php yii {{serve}}`

- Generați un controler, vizualizări și fișiere asociate pentru acțiunile CRUD pe clasa de model specificată:

`php yii {{gii/crud}} --modelClass={{ModelName}} --controllerClass={{ControllerName}}`
