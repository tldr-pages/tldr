# git rm

> Eliminați fișierele din indexul depozitului și din sistemul de fișiere local.
> Mai multe informaţii: <https://git-scm.com/docs/git-rm>

- Elimină fișierul din indexul depozitului și sistemul de fișiere:

`git rm {{file}}`

- Elimină directorul:

`git rm -r {{directory}}`

- Eliminați fișierul din indexul depozitului, dar păstrați-l neatins local:

`git rm --cached {{file}}`
