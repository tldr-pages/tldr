# git var

> Imprimă valoarea unei variabile logice Git.
> A se vedea `git config`, care este preferat peste `git var`.
> Mai multe informaţii: <https://git-scm.com/docs/git-var>

- Imprimați valoarea unei variabile logice Git:

`git var {{GIT_AUTHOR_IDENT|GIT_COMMITTER_IDENT|GIT_EDITOR|GIT_PAGER}}`

- [l] ist toate variabilele Git logice:

`git var -l`
