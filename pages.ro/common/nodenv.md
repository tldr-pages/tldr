# nodenv

> Un instrument pentru gestionarea versiunilor Node.js.
> Mai multe informaţii: <https://github.com/nodenv/nodenv>

- Instalați o versiune specifică de Node.js:

`nodenv install {{version}}`

- Afișează o listă de versiuni disponibile:

`nodenv install --list`

- Utilizați o versiune specifică de Node.js în întregul sistem:

`nodenv global {{version}}`

- Utilizați o versiune specifică de Node.js cu un director:

`nodenv local {{version}}`

- Afișează versiunea Node.js pentru directorul curent:

`nodenv version`

- Afișează locația unei comenzi instalate Node.js (de ex. `npm`):

`nodenv which {{command}}`
