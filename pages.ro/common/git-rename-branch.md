# git rename-branch

> Redenumiți o ramură Git.
> Parte din `git-extras`.
> Mai multe informaţii: <https://github.com/tj/git-extras/blob/master/Commands.md#git-rename-branch>

- Redenumiți sucursala în care vă aflați în prezent:

`git rename-branch {{new_branch_name}}`

- Redenumiți o anumită ramură:

`git rename-branch {{old_branch_name}} {{new_branch_name}}`
