# grumphp

> Un plugin PHP Composer care permite verificarea calității codului sursă.
> Mai multe informaţii: <https://github.com/phpro/grumphp>

- Înregistrați cârligele Git:

`grumphp git:init`

- Declanşaţi manual cârligul de pre-angajament:

`grumphp git:pre-commit`

- Verificați fiecare fișier versioned:

`grumphp run`
