# mocha

> Executa Mocha JavaScript test alergător.
> Mai multe informaţii: <https://mochajs.org>

- Rulați teste cu configurație implicită sau configurate în `mocha.opts`:

`mocha`

- Executați teste conținute într-o anumită locație:

`mocha {{directory/with/tests}}`

- Rulați teste care se potrivesc cu un anumit model grep:

`mocha --grep {{regular_expression}}`

- Rulați teste privind modificările fișierelor JavaScript în directorul curent și o dată inițial:

`mocha --watch`

- Rulați teste cu un anumit reporter:

`mocha --reporter {{reporter}}`
