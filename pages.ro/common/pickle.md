# pickle

> Un program de instalare a extensiei PHP bazat pe Composer.
> Mai multe informaţii: <https://github.com/FriendsOfPHP/pickle>

- Instalați o extensie PHP specifică:

`pickle install {{extension_name}}`

- Convertiți o configurație de extensie PECL existentă într-un fișier de configurare Pickle:

`pickle convert {{path/to/directory}}`

- Validează o extensie PECL:

`pickle validate {{path/to/directory}}`

- Pachet o extensie PECL pentru lansare:

`pickle release {{path/to/directory}}`
