# python

> Interpretul de limbă Python.
> Mai multe informaţii: <https://www.python.org>

- Apelați o coajă interactivă Python (REPL):

`python`

- Executați script într-un fișier Python dat:

`python {{script.py}}`

- Executa script ca parte a unui shell interactiv:

`python -i {{script.py}}`

- Execută o expresie Python:

`python -c "{{expression}}"`

- Rulați modulul bibliotecii ca un script (se termină lista de opțiuni):

`python -m {{module}} {{arguments}}`

- Instalați un pachet folosind pip:

`python -m pip install {{package_name}}`

- Depanare interactiv un script Python:

`python -m pdb {{script.py}}`
