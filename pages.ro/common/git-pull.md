# git pull

> Preluați ramura dintr-un depozit la distanță și îmbinați-o în depozitul local.
> Mai multe informaţii: <https://git-scm.com/docs/git-pull>

- Descărcați modificările din depozitul implicit la distanță și îmbinați-l:

`git pull`

- Descărcați modificările din depozitul implicit de la distanță și utilizați rapid înainte:

`git pull --rebase`

- Descărcați modificările din depozitul de la distanță și ramura dată, apoi îmbinați-le în HEAD:

`git pull {{remote_name}} {{branch}}`
