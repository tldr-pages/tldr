# ionic

> Un cadru pentru a construi aplicații mobile hibride.
> Mai multe informaţii: <https://ionicframework.com/docs/cli>

- Creați un nou proiect:

`ionic start`

- Porniți un server dev local pentru aplicații dev/testare:

`ionic serve`

- Generați o nouă componentă de aplicație, directivă, pagină, țeavă, furnizor sau file:

`ionic g {{page}}`

- Arată versiuni de ionic, cordova, mediu, etc:

`ionic info`

- Rulați aplicația pe un dispozitiv Android/ios:

`ionic cordova run {{android|ios}} --device`

- Verificați starea de sănătate a unei aplicații ionice:

`ionic doctor {{check}}`
