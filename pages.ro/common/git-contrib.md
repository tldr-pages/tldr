# git contrib

> Afișarea se angajează de la un anumit autor.
> Parte din `git-extras`.
> Mai multe informaţii: <https://github.com/tj/git-extras/blob/master/Commands.md#git-contrib>

- Afișează toate hash-urile comite și mesajele lor corespunzătoare comite de la un anumit autor:

`git contrib {{author}}`
