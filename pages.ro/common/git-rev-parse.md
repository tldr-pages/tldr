# git rev-parse

> Afișează metadate legate de anumite revizuiri.
> Mai multe informaţii: <https://git-scm.com/docs/git-rev-parse>

- Obțineți hash-ul comite unei sucursale:

`git rev-parse {{branch_name}}`

- Obțineți numele sucursalei curente:

`git rev-parse --abbrev-ref {{HEAD}}`

- Obțineți calea absolută către directorul rădăcină:

`git rev-parse --show-toplevel`
