# minetest

> Multiplayer infinit lume bloc de nisip.
> A se vedea, de asemenea, `minetestserver`, binar server numai.
> Mai multe informaţii: <https://wiki.minetest.net/Minetest>

- Pornește minetest în modul client:

`minetest`

- Porniți minetest în modul server prin găzduirea unei anumite lumi:

`minetest --server --world {{name}}`

- Scrieți jurnalele într-un anumit fișier:

`minetest --logfile {{path/to/file}}`

- Doar scrie erori la consola:

`minetest --quiet`
