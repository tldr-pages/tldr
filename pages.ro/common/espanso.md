# espanso

> Expander text încrucișat scris în Rust.
> Mai multe informaţii: <https://espanso.org>

- Verifică starea:

`espanso status`

- Editează configurația:

`espanso edit config`

- Instalați un pachet din magazinul hub ( <https://hub.espanso.org/>):

`espanso install {{package_name}}`

- Reporniți (necesar după instalarea unui pachet, util în caz de eșec):

`espanso restart`
