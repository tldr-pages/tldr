# sass

> Efectuează conversia fișierelor SCSS sau Sass în CSS.
> Mai multe informaţii: <https://sass-lang.com/documentation/cli/dart-sass>

- Convertiți un fișier SCSS sau Sass în CSS și imprimați rezultatul:

`sass {{inputfile.scss|inputfile.sass}}`

- Convertiți un fișier SCSS sau Sass în CSS și salvați rezultatul într-un fișier:

`sass {{inputfile.scss|inputfile.sass}} {{outputfile.css}}`

- Urmăriți un fișier SCSS sau Sass pentru modificări și ieșire sau actualizați fișierul CSS cu același nume de fișier:

`sass --watch {{inputfile.scss|inputfile.sass}}`

- Urmăriți un fișier SCSS sau Sass pentru modificări și ieșire sau actualizați fișierul CSS cu numele de fișier dat:

`sass --watch {{inputfile.scss|inputfile.sass}}:{{outputfile.css}}`
