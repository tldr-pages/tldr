# pio lib

> Gestionați bibliotecile PlatForMio.
> Mai multe informaţii: <https://docs.platformio.org/en/latest/core/userguide/lib/>

- Lista bibliotecilor instalate:

`pio lib list`

- Lista bibliotecilor încorporate pe baza platformelor de dezvoltare instalate și a cadrelor acestora:

`pio lib builtin`

- Caută bibliotecile existente:

`pio lib search {{keyword}}`

- Arată detalii despre o bibliotecă:

`pio lib show {{library}}`

- Instalați o bibliotecă:

`pio lib install {{library}}`

- Actualizare biblioteci instalate:

`pio lib update`

- Dezinstalați o bibliotecă:

`pio lib uninstall {{library}}`

- Arată statisticile de registru ale bibliotecii PlatForMio:

`pio lib stats`
