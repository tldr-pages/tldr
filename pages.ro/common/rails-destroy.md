# rails destroy

> Distruge resurse Rails.
> Mai multe informaţii: <https://guides.rubyonrails.org/command_line.html#bin-rails-destroy>

- Listează toate generatoarele disponibile pentru a distruge:

`rails destroy`

- Distruge un model numit Post:

`rails destroy model {{Post}}`

- Distruge un controler numit Posturi:

`rails destroy controller {{Posts}}`

- Distruge o migrare care creează Postări:

`rails destroy migration {{CreatePosts}}`

- Distruge o schelă pentru un model numit Post:

`rails destroy scaffold {{Post}}`
