# hg serve

> Porniți un server web Mercurial independent pentru navigarea depozitelor.
> Mai multe informaţii: <https://www.mercurial-scm.org/doc/hg.1.html#serve>

- Porniți o instanță de server web:

`hg serve`

- Porniți o instanță de server web pe portul specificat:

`hg serve --port {{port}}`

- Porniți o instanță de server web pe adresa de ascultare specificată:

`hg serve --address {{address}}`

- Porniți o instanță de server web cu un identificator specific:

`hg serve --name {{name}}`

- Porniți o instanță de server web utilizând tema specificată (consultați directorul șabloanelor):

`hg serve --style {{style}}`

- Porniți o instanță de server web utilizând pachetul de certificate SSL specificat:

`hg serve --certificate {{path/to/certificate}}`
