# git update-index

> Comanda Git pentru manipularea indexului.
> Mai multe informaţii: <https://git-scm.com/docs/git-update-index>

- Pretindeți că un fișier modificat este neschimbat (`git status' nu va afișa acest lucru ca fiind modificat):

`git update-index --skip-worktree {{path/to/modified_file}}`
