# pyflakes

> Verifică fișierele de cod sursă Python pentru erori.
> Mai multe informaţii: <https://pypi.org/project/pyflakes>

- Verificați un singur fișier Python:

`pyflakes check {{path/to/file}}.py`

- Verificați fișierele Python într-un anumit director:

`pyflakes checkPath {{path/to/directory}}`

- Verificați fișierele Python într-un director recursiv:

`pyflakes checkRecursive {{path/to/directory}}`

- Verificați toate fișierele Python găsite în mai multe directoare:

`pyflakes iterSourceCode {{path/to/directory_1}} {{path/to/directory_2}}`
