# supervisord

> Supervizor este un sistem client/server pentru controlul unor procese pe sistemele de operare asemănătoare UNIX.
> Supervisord este partea de server a supervizorului; acesta este gestionat în primul rând printr-un fișier de configurare.
> Mai multe informaţii: <http://supervisord.org>

- Începeți supervizorul cu fișierul de configurare specificat:

`supervisord -c {{path/to/file}}`

- Rulați supervizorul în prim-plan:

`supervisord -n`
