# jstack

> Java stivă Trace instrument.

- Imprimați urme de stivă Java pentru toate firele într-un proces Java:

`jstack {{java_pid}}`

- Imprimare modul mixt (Java/C++) stiva urme pentru toate firele într-un proces Java:

`jstack -m {{java_pid}}`

- Imprimare stiva urme din Dump de bază Java:

`jstack {{/usr/bin/java}} {{file.core}}`
