# sqlite3

> Interfața de linie de comandă pentru SQLite 3, care este un motor SQL încorporat bazat pe fișiere.
> Mai multe informaţii: <https://sqlite.org>

- Începeți o coajă interactivă cu o nouă bază de date:

`sqlite3`

- Deschideți o coajă interactivă împotriva unei baze de date existente:

`sqlite3 {{path/to/database.sqlite3}}`

- Executați o instrucțiune SQL împotriva unei baze de date și apoi ieșiți:

`sqlite3 {{path/to/database.sqlite3}} '{{SELECT * FROM some_table;}}'`
