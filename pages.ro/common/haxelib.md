# haxelib

> Managerul bibliotecii Haxe.
> Mai multe informaţii: <https://lib.haxe.org/>

- Caută o bibliotecă Haxe:

`haxelib search {{keyword}}`

- Instalați o bibliotecă Haxe:

`haxelib install {{libname}}`

- Upgrade toate bibliotecile Haxe instalate:

`haxelib upgrade`

- Instalați versiunea de dezvoltare a unei biblioteci dintr-un depozit Git:

`haxelib git {{libname}} {{git_url}}`
