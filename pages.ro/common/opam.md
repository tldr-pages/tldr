# opam

> Managerul de pachete oCl.
> Gestionați compilatoarele, instrumentele și bibliotecile OcamL.
> Mai multe informaţii: <https://opam.ocaml.org/>

- Iniţializarea opam pentru prima utilizare:

`opam init`

- Caută pachete:

`opam search {{package_name}}`

- Instalați un pachet și toate dependențele sale:

`opam install {{package_name}}`

- Afișează informații detaliate despre un pachet:

`opam show {{package_name}}`

- Listează toate pachetele instalate:

`opam list`

- Actualizați baza de date a pachetelor locale:

`opam update`

- Upgrade toate pachetele instalate:

`opam upgrade`

- Afișează toate comenzile:

`opam help`
