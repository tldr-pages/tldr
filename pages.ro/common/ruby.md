# ruby

> Interpretul limbajului de programare Ruby.
> Mai multe informaţii: <https://www.ruby-lang.org>

- Deschide un Ruby Shell interactiv (REPL):

`irb`

- Execută un script Ruby:

`ruby {{script.rb}}`

- Executați o singură comandă Ruby în linia de comandă:

`ruby -e {{command}}`

- Verificați pentru erori de sintaxă pe un anumit script Ruby:

`ruby -c {{script.rb}}`

- Afișați versiunea Ruby pe care o utilizați:

`ruby -v`
