# ohdear-cli

> Un neoficial Oh Dragă CLI scris cu Laravel Zero.
> Mai multe informaţii: <https://github.com/nunomaduro/ohdear-cli>

- Afișează detalii despre utilizatorul autentificat în prezent:

`ohdear-cli me`

- Adaugă un nou site la Oh Dragă:

`ohdear-cli sites:add {{url}}`

- Afișează o listă de site-uri și starea lor curentă:

`ohdear-cli sites:list`

- Afișează detalii despre un anumit site:

`ohdear-cli sites:show {{site_id}}`
