# swift

> Creați, rulați și construiți proiecte Swift.
> Mai multe informaţii: <https://swift.org>

- Invocați interpretul interactiv (REPL):

`swift`

- Execută un program:

`swift {{file.swift}}`

- Începeți un nou proiect cu managerul de pachete:

`swift package init`

- Generați un fișier de proiect Xcode:

`swift package generate-xcodeproj`

- Actualizare dependențe:

`swift package update`

- Proiect de compilare pentru lansare:

`swift build -c release`
