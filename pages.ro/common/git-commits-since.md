# git commits-since

> Afișarea se angajează de la o anumită oră sau dată.
> Parte din `git-extras`.
> Mai multe informaţii: <https://github.com/tj/git-extras/blob/master/Commands.md#git-commits-since>

- Afișajul se angajează de ieri:

`git commits-since {{yesterday}}`

- Afișajul se angajează de săptămâna trecută:

`git commits-since {{last week}}`

- Afișarea se angajează de luna trecută:

`git commits-since {{last month}}`

- Afișajul se angajează de ieri 2pm:

`git commits-since {{yesterday 2pm}}`
