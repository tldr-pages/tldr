# nf-core

> Instrumentele cadru nf-core, pentru a crea, verifica și dezvolta orientări privind cele mai bune practici pentru Nextflow.
> Mai multe informaţii: <https://nf-co.re/tools>

- Lista conductelor existente pe nf-core:

`nf-core list`

- Creați un nou schelet de conducte:

`nf-core create`

- Same codul conductei:

`nf-core lint {{path/to/directory}}`

- Versiuni software Bump în reţeta de conducte:

`nf-core bump-version {{path/to/directory}} {{new_version}}`

- Lansarea unei conducte nf-core:

`nf-core launch {{pipeline_name}}`

- Descărcați o conductă nf-core pentru utilizare offline:

`nf-core download {{pipeline_name}}`
