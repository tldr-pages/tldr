# git delta

> Listă fișiere care diferă de o altă ramură.
> O parte din git-extras.
> Mai multe informaţii: <https://github.com/tj/git-extras/blob/master/Commands.md#git-delta>

- Lista fişierelor din ramura curentă extrase care diferă de cea principală:

`git delta {{main}}`

- Lista fișierelor din ramura specifică care diferă de o altă ramură specifică:

`git delta {{branch_1}} {{branch_2}}`
