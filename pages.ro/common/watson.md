# watson

> Un CLI minunat pentru a urmări timpul.
> Mai multe informaţii: <https://github.com/TailorDev/Watson>

- Începeți timpul de monitorizare în proiect:

`watson start {{project}}`

- Începeți timpul de monitorizare în proiect cu tag-uri:

`watson start {{project}} +{{tag}}`

- Opriți timpul de monitorizare pentru proiectul curent:

`watson stop`

- Afișează cele mai recente sesiuni de lucru:

`watson log`

- Editează cel mai recent cadru:

`watson edit`

- Eliminați cel mai recent cadru:

`watson remove`
