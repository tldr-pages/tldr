# glances

> Un instrument de monitorizare a sistemului cross-platform.
> Mai multe informaţii: <https://nicolargo.github.io/glances/>

- Rulați în terminal:

`glances`

- Rulați în modul server web pentru a afișa rezultatele în browser:

`glances -w`

- Rulați în modul server pentru a permite conexiuni de la alți clienți Glances:

`glances -s`

- Conectează-te la un server Privirile:

`glances -c {{hostname}}`

- Necesită o parolă în modul server (web):

`glances -s --password`
