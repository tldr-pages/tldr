# IPython

> O coajă Python cu istoric automat, introspecție dinamică a obiectului, configurare mai ușoară, finalizarea comenzii, acces la carcasa sistemului și multe altele.
> Mai multe informaţii: <https://ipython.org/documentation.html>

- Începeți o sesiune iPython interactivă:

`ipython`

- Introduceți o sesiune iPython interactivă după rularea unui script Python:

`ipython -i {{script.py}}`

- Creează profilul Ipython implicit:

`ipython profile create`

- Imprimați calea către director pentru profilul implicit IPython:

`ipython locate profile`

- Ștergeți baza de date a istoricului iPython, ștergând toate intrările:

`ipython history clear`
