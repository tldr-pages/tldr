# cargo doc

> Construiți și vizualizați documentația pachetului Rust offline.
> Mai multe informaţii: <https://doc.rust-lang.org/cargo/commands/cargo-doc.html>

- Construiți și vizualizați documentația implicită a pachetului în browser:

`cargo doc --open`

- Construiți documentația fără a accesa rețeaua:

`cargo doc --offline`

- Vizualizaţi documentaţia unui anumit pachet:

`cargo doc --open --package {{package}}`

- Vizualizați documentația unui anumit pachet offline:

`cargo doc --open --offline --package {{package}}`
