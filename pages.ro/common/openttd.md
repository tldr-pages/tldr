# openttd

> Clona open source a jocului Microprose „Transport Tycoon Deluxe”.
> Mai multe informaţii: <https://www.openttd.org>

- Începe un joc nou:

`openttd -g`

- Încărcați salva joc la început:

`openttd -g {{path/to/file}}`

- Începeți cu rezoluția ferestrei specificate:

`openttd -r {{1920x1080}}`

- Începeți cu un fișier de configurare personalizat:

`openttd -c {{path/to/file}}`

- Începeți cu drivere video, sunet și muzică selectate:

`openttd -v {{video_driver}} -s {{sound_driver}} -m {{music_driver}}`

- Porniți un server dedicat, bifurcat în fundal:

`openttd -f -D {{host}}:{{port}}`

- Alăturați-vă unui server cu o parolă:

`openttd -n {{host}}:{{port}}#{{player_name}} -p {{password}}`
