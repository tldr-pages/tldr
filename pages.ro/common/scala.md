# scala

> Lansator de aplicații Scala și interpret interactiv.
> Mai multe informaţii: <https://scala-lang.org>

- Începe o coajă interactivă Scala (REPL):

`scala`

- Porniți interpretul cu o dependență în classath:

`scala -classpath {{filename.jar}} {{command}}`

- Execută un script Scala:

`scala {{script.scala}}`

- Executa un program `.jar`:

`scala {{filename.jar}}`

- Executați o singură comandă Scala în linia de comandă:

`scala -e {{command}}`
