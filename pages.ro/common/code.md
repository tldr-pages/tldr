# code

> Codul Visual Studio.
> Mai multe informaţii: <https://github.com/microsoft/vscode>

- Cod VS deschis:

`code`

- Deschideți directorul curent în Codul VS:

`code .`

- Deschideți un fișier sau un director în Codul VS:

`code {{path/to/file_or_directory}}`

- Deschideți un fișier sau un director în fereastra de cod VS deschisă în prezent:

`code --reuse-window {{path/to/file_or_directory}}`

- Comparați două fișiere în codul VS:

`code -d {{file1}} {{file2}}`

- Deschideți codul VS cu permisiuni super-utilizator (sudo):

`sudo code {{path/to/file_or_directory}} --user-data-dir`
