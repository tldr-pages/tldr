# pyenv virtualenv

> Creați medii virtuale bazate pe distribuțiile Python instalate.
> Mai multe informaţii: <https://github.com/pyenv/pyenv-virtualenv>

- Creați un nou mediu virtual Python 3.6.6:

`pyenv virtualenv {{3.6.6}} {{virtualenv_name}}`

- Listează toate mediile virtuale existente:

`pyenv virtualenvs`

- Activarea unui mediu virtual:

`pyenv activate {{virtualenv_name}}`

- Dezactivați mediul virtual:

`pyenv deactivate`
