# lynx

> Browser web în linie de comandă.
> Mai multe informaţii: <https://lynx.browser.org>

- Vizitați un site web:

`lynx {{example.com}}`

- Aplicați restricții pentru contul anonim:

`lynx -anonymous {{example.com}}`

- Activați suportul mouse-ului, dacă este disponibil:

`lynx -use_mouse {{example.com}}`

- Activarea modului de culoare, dacă este disponibil:

`lynx -color {{example.com}}`

- Deschideți un link, folosind un anumit fișier pentru a citi și scrie cookie-uri:

`lynx -cookie_file={{path/to/file}} {{example.com}}`

- Navigați înainte și înapoi prin linkurile de pe o pagină:

`Up arrow key, Down arrow key`

- Reveniți la pagina afișată anterior:

`Left arrow key or u`

- Ieşire:

`q then y`
