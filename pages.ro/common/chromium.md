# chromium

> Browser web open-source de la Google.
> Mai multe informaţii: <https://chromium.org>

- Deschide un fișier:

`chromium {{path/to/file.html}}`

- Deschide un URL:

`chromium {{example.com}}`

- Deschidere în modul incognito:

`chromium --incognito {{example.com}}`

- Deschide într-o fereastră nouă:

`chromium --new-window {{example.com}}`

- Deschidere în modul aplicație (fără bare de instrumente, bara URL, butoane etc.):

`chromium --app='{{https://example.com}}'`

- Utilizați un server proxy:

`chromium --proxy-server="{{socks5://hostname:66}}" {{example.com}}`
