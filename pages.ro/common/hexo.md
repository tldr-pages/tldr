# hexo

> Un cadru de blog rapid, simplu și puternic.
> Mai multe informaţii: <https://hexo.io/>

- Inițializarea unui site web:

`hexo init {{path/to/directory}}`

- Creați un articol nou:

`hexo new {{layout}} {{title}}`

- Generează fișiere statice:

`hexo generate`

- Porniți un server local:

`hexo server`

- Implementați site-ul web:

`hexo deploy`

- Curățați fișierul cache (`db.json`) și fișierele generate (`public/`):

`hexo clean`
