# git show-ref

> comanda Git pentru listarea referințelor.
> Mai multe informaţii: <https://git-scm.com/docs/git-show-ref>

- Afișați toate refs în depozit:

`git show-ref`

- Afișează doar referințe capete:

`git show-ref --heads`

- Afișează doar referințele etichete:

`git show-ref --tags`

- Verificați dacă există o referință dată:

`git show-ref --verify {{path/to/ref}}`
