# git root

> Imprimați directorul rădăcină al depozitului Git curent.
> Parte din `git-extras`.
> Mai multe informaţii: <https://github.com/tj/git-extras/blob/master/Commands.md#git-root>

- Imprimați calea absolută a depozitului Git curent:

`git root`

- Imprimați directorul de lucru curent în raport cu rădăcina depozitului Git curent:

`git root --relative`
