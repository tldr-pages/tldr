# nrm

> NPM manager de registry.
> Ajută la comutarea cu ușurință între diferite registre npm.
> Mai multe informaţii: <https://github.com/Pana/nrm>

- Lista tuturor registrelor:

`nrm ls`

- Trecerea la un anumit registru:

`nrm use {{registry}}`

- Afișați timpul de răspuns pentru toate registrele:

`nrm test`

- Adăugați un registru particularizat:

`nrm add {{registry}} {{url}}`

- Ștergeți un registru:

`nrm del {{registry}}`
