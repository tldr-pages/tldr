# git reflog

> Afișați un jurnal de modificări la referințe locale, cum ar fi HEAD, ramuri sau etichete
> Mai multe informaţii: <https://git-scm.com/docs/git-reflog>

- Arată reflog pentru HEAD:

`git reflog`

- Afișați reflog pentru o anumită ramură:

`git reflog {{branch_name}}`

- Afișați doar cele mai recente 5 intrări din reflog:

`git reflog -n {{5}}`
