# git authors

> Generați o listă a angajatorilor unui depozit Git.
> Parte din `git-extras`.
> Mai multe informaţii: <https://github.com/tj/git-extras/blob/master/Commands.md#git-authors>

- Tipăriți o listă completă a angajatorilor care să se retragă în loc să fie în fișierul „AUTORS”:

`git authors --list`

- Adaugati lista de angajati la fisierul „AUTORS” si deschideti-o in editorul implicit:

`git authors`

- Adaugă lista de angajatori, cu excepția e-mailurilor, în fișierul „AUTORS” și deschideți-l în editorul implicit:

`git authors --no-email`
