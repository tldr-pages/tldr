# npm-check

> Verificați pentru dependențe de pachete npm învechite, incorecte și neutilizate.
> Mai multe informaţii: <https://www.npmjs.com/package/npm-check/>

- Afișează un raport de dependențe învechite, incorecte și neutilizate:

`npm-check`

- Actualizarea interactivă a pachetelor depășite:

`npm-check --update`

- Actualizați totul fără a solicita:

`npm-check --update-all`

- Nu căutați pachete neutilizate:

`npm-check --skip-unused`
