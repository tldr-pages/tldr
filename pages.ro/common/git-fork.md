# git fork

> Furculiță repo GitHub dat. Ca „git clone „, dar furculiţele mai întâi.
> Parte din `git-extras`.
> Mai multe informaţii: <https://github.com/tj/git-extras/blob/master/Commands.md#git-fork>

- Furcă și clona un depozit GitHub prin URL-ul său:

`git fork {{https://github.com/tldr-pages/tldr}}`

- Furcă și clona un depozit GitHub de melc sale:

`git fork {{tldr-pages/tldr}}`
