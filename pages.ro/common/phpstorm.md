# phpstorm

> Un IDE cross-platform pentru PHP bazat pe platforma JetBrains IntelliJ.
> Mai multe informaţii: <https://jetbrains.com/phpstorm>

- Deschide un anumit director:

`phpstorm {{path/to/directory}}`

- Deschide un fișier:

`phpstorm {{path/to/file}}`

- Deschideți un fișier la o anumită linie:

`phpstorm --line {{line_number}} {{path/to/file}}`

- Vizualizați diferențele dintre două fișiere:

`phpstorm diff {{path/to/left_file}} {{path/to/right_file}}`
