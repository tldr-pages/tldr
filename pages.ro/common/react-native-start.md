# react-native start

> Instrumente de linie de comandă pentru a porni serverul React nativ.
> Mai multe informaţii: <https://github.com/react-native-community/cli/blob/master/docs/commands.md#start>

- Porniți serverul care comunică cu dispozitivele conectate:

`react-native start`

- Porniți pachetul de metrou cu un cache curat:

`react-native start --reset-cache`

- Porniți serverul într-un port particularizat (implicit la 8081):

`react-native start --port {{3000}}`

- Porniți serverul în modul verbose:

`react-native start --verbose`

- Specificați numărul maxim de lucrători pentru transformarea fișierelor (implicit este numărul de nuclee CPU):

`react-native start --max-workers {{count}}`

- Dezactivează modul interactiv:

`react-native start --no-interactive`
