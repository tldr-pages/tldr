# git annotate

> Afișați hash comite și ultimul autor pe fiecare linie a unui fișier.
> A se vedea `git vina ', care este preferat peste `git annotate`.
> `git annotate` este furnizat pentru cei familiarizati cu alte sisteme de control al versiunilor.
> Mai multe informaţii: <https://git-scm.com/docs/git-annotate>

- Imprimați un fișier cu numele autorului și comite hash prepended pentru fiecare linie:

`git annotate {{path/to/file}}`

- Imprimați un fișier cu e-mailul autorului și comite hash prepended pentru fiecare linie:

`git annotate -e {{path/to/file}}`
