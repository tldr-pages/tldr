# cheat

> Creați și vizualizați foi de înșelăciune interactive pe linia de comandă.
> Mai multe informaţii: <https://github.com/cheat/cheat>

- Arată exemplu de utilizare a unei comenzi:

`cheat {{command}}`

- Editați foaia de ieftin pentru o comandă:

`cheat -e {{command}}`

- Listează foile de ieftin disponibile:

`cheat -l`

- Căutare disponibile foile de ieftin pentru un nume de comandă specificat:

`cheat -s {{command}}`

- Obțineți versiunea curentă de ieftin:

`cheat -v`
