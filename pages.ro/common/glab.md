# glab

> GitLab CLI instrument pentru a ajuta la lucrul cu GitLab din linia de comandă.
> Mai multe informaţii: <https://github.com/profclems/glab>

- Crearea unei solicitări de îmbinare:

`glab mr create`

- Lista cererilor de îmbinare:

`glab mr list`

- Creați o nouă problemă:

`glab issue create`

- Vizualizați și filtrați problemele deschise ale depozitului curent:

`glab issue list`

- Lista conductelor:

`glab pipeline list`

- Clonează un depozit într-un anumit director:

`glab repo clone {{user}}/{{repository}} {{directory}}`
