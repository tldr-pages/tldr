# circup

> Actualizatorul bibliotecii CircuitPython.
> Mai multe informaţii: <https://github.com/adafruit/circup>

- Actualizarea interactivă a modulelor pe un dispozitiv:

`circup update`

- Instalați o bibliotecă nouă:

`circup install {{library_name}}`

- Caută o bibliotecă:

`circup show {{partial_name}}`

- Listează toate bibliotecile de pe un dispozitiv conectat în format `cerințe.txt`:

`circup freeze`

- Salvați toate bibliotecile pe un dispozitiv conectat în directorul curent:

`circup freeze -r`
