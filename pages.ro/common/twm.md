# twm

> Un manager de ferestre pentru sistemul X Window.
> Mai multe informaţii: <https://gitlab.freedesktop.org/xorg/app/twm>

- Conectează-te la serverul X implicit:

`twm`

- Conectează-te la un anumit server X:

`twm -display {{display}}`

- Gestionați numai ecranul implicit:

`twm -s`

- Utilizați un fișier de pornire specific:

`twm -f {{path/to/file}}`

- Activați modul verbose și imprimați erori neașteptate în X:

`twm -v`
