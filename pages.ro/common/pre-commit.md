# pre-commit

> Creați cârlige Git care se execută înainte de a comite.
> Mai multe informaţii: <https://pre-commit.com>

- Instalați pre-comite în cârligele Git:

`pre-commit install`

- Rulați cârlige pre-comite pe toate fișierele pe etape:

`pre-commit run`

- Rulați cârlige pre-comite pe toate fișierele, puse în scenă sau nepuse în scenă:

`pre-commit run --all-files`

- Curățați memoria cache pre-comite:

`pre-commit clean`
