# cordova

> Aplicații mobile cu HTML, CSS și JS.
> Mai multe informaţii: <https://cordova.apache.org/docs/en/latest/guide/cli/>

- Crearea unui proiect cordova:

`cordova create {{path}} {{package_name}} {{project_name}}`

- Afișează starea curentă a spațiului de lucru:

`cordova info`

- Adaugă o platformă cordova:

`cordova platform add {{platform}}`

- Eliminați o platformă cordova:

`cordova platform remove {{platform}}`

- Adăugați un plugin cordova:

`cordova plugin add {{pluginid}}`

- Eliminați un plugin cordova:

`cordova plugin remove {{pluginid}}`
