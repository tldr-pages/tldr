# mat2

> Anonizaţi diferite formate de fişiere eliminând metadatele.
> Mai multe informaţii: <https://0xacab.org/jvoisin/mat2>

- Lista formatelor de fișiere acceptate:

`mat2 --list`

- Eliminarea metadatelor dintr-un fișier:

`mat2 {{path/to/file}}`

- Eliminați metadatele dintr-un fișier și imprimați ieșirea detaliată a consolei:

`mat2 --verbose {{path/to/file}}`

- Afișează metadatele într-un fișier fără a le elimina:

`mat2 --show {{path/to/file}}`

- Eliminarea parțială a metadatelor dintr-un fișier:

`mat2 --lightweight {{path/to/file}}`
