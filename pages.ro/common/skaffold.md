# skaffold

> Un instrument care facilitează dezvoltarea continuă a aplicațiilor Kubernetes.
> Mai multe informaţii: <https://skaffold.dev>

- Construieşte artefactele:

`skaffold build -f {{skaffold.yaml}}`

- Construiți și implementați aplicația de fiecare dată când se schimbă codul:

`skaffold dev -f {{skaffold.yaml}}`

- Rulați un fișier de conducte:

`skaffold run -f {{skaffold.yaml}}`

- Rulați un diagnostic pe Skaffold:

`skaffold diagnose -f {{skaffold.yaml}}`

- Desfăşuraţi artefactele:

`skaffold deploy -f {{skaffold.yaml}}`
