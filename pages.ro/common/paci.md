# paci

> Un manager de pachete pentru scripturi bash.

- Actualizați lista pachetelor și versiunilor disponibile (este recomandat să rulați acest lucru înainte de alte comenzi `paci`):

`paci refresh`

- Configurarea comportamentului său:

`paci configure`

- Caută un pachet dat:

`paci search {{package}}`

- Instalați un pachet:

`paci install {{package}}`

- Actualizarea unui pachet:

`paci update {{package}}`
