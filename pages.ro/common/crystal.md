# crystal

> Instrument pentru gestionarea codului sursă Crystal.
> Mai multe informaţii: <https://crystal-lang.org/reference/using_the_compiler>

- Rulează un fișier Crystal:

`crystal {{path/to/file.cr}}`

- Compilați un fișier și toate dependențele la un singur executabil:

`crystal build {{path/to/file.cr}}`

- Porniți un server interactiv local pentru testarea limbii:

`crystal play`

- Creați un director de proiect pentru o aplicație Crystal:

`crystal init app {{application_name}}`

- Afișează toate opțiunile de ajutor:

`crystal help`
