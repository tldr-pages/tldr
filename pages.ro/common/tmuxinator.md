# tmuxinator

> Creați și gestionați cu ușurință sesiuni tmux.
> Mai multe informaţii: <https://github.com/tmuxinator/tmuxinator>

- Creați un nou proiect:

`tmuxinator new {{project}}`

- Editează un proiect:

`tmuxinator edit {{project}}`

- Lista proiectelor:

`tmuxinator list`

- Începeți o sesiune tmux bazată pe proiect:

`tmuxinator start {{project}}`

- Opriți sesiunea tmux a unui proiect:

`tmuxinator stop {{project}}`
