# git alias

> Creați comenzi rapide pentru comenzile Git.
> Parte din `git-extras`.
> Mai multe informaţii: <https://github.com/tj/git-extras/blob/master/Commands.md#git-alias>

- Listează toate pseudonimele:

`git alias`

- Creați un alias nou:

`git alias "{{name}}" "{{command}}"`

- Caută un alias existent:

`git alias ^{{name}}`
