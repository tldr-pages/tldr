# git alias

> Creează scurtături pentru comenzi Git.
> Parte din `git-extras`.
> Mai multe informații: <https://github.com/tj/git-extras/blob/main/Commands.md#git-alias>.

- Listează toate aliasurile:

`git alias`

- Creează un alias nou:

`git alias "{{nume}}" "{{comandă}}"`

- Caută un alias existent:

`git alias ^{{nume}}`
