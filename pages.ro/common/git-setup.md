# git setup

> Creați un depozit Git într-un anumit director și comite toate fișierele.
> Parte din `git-extras`.
> Mai multe informaţii: <https://github.com/tj/git-extras/blob/master/Commands.md#git-setup>

- Creați un depozit Git în directorul curent și comite toate fișierele:

`git setup`

- Creați un depozit Git într-un anumit director și comite toate fișierele:

`git setup {{path/to/directory}}`
