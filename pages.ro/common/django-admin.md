# django-admin

> utilitate Django pentru activități administrative.
> Mai multe informaţii: <https://docs.djangoproject.com/en/3.0/ref/django-admin/>

- Creați un nou proiect django:

`django-admin startproject {{project_name}}`

- Creați o aplicație nouă pentru proiectul curent:

`django-admin startapp {{app_name}}`

- Verificați versiunea curentă a Django:

`django-admin --version`

- Afișați mai multe informații pentru comanda dată:

`django-admin help {{command}}`
