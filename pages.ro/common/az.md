# az

> Instrumentul oficial CLI pentru Microsoft Azure.
> Mai multe informaţii: <https://docs.microsoft.com/cli/azure>

- Autentificare la Azure:

`az login`

- Gestionați informațiile privind abonamentul Azure:

`az account`

- Listează toate discurile Azure gestionate:

`az disk list`

- Lista toate mașinile virtuale Azure:

`az vm list`

- Gestionați Azure Kubernetes servicii:

`az aks`

- Gestionați resursele rețelei Azure:

`az network`
