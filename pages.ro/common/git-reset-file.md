# git reset-file

> Reveniți un fișier la HEAD sau o anumită comitere.
> Parte din `git-extras`.
> Mai multe informaţii: <https://github.com/tj/git-extras/blob/master/Commands.md#git-reset-file>

- Resetați un fișier la HAD:

`git reset-file {{path/to/file}}`

- Resetați un fișier la un anumit angajament:

`git reset-file {{path/to/file}} {{commit_hash}}`
