# deno

> O execuție sigură pentru JavaScript și TypeScript.
> Mai multe informaţii: <https://deno.land/>

- Rulați un fișier JavaScript sau TypeScript:

`deno run {{path/to/file.ts}}`

- Porniți un REPL (shell interactiv):

`deno`

- Rulați un fișier cu acces la rețea activat:

`deno run --allow-net {{path/to/file.ts}}`

- Rulați un fișier dintr-un URL:

`deno run {{https://deno.land/std/examples/welcome.ts}}`

- Instalați un script executabil de la un URL:

`deno install {{https://deno.land/std/examples/colors.ts}}`
