# git create-branch

> Creați o ramură Git într-un depozit.
> Parte din `git-extras`.
> Mai multe informaţii: <https://github.com/tj/git-extras/blob/master/Commands.md#git-create-branch>

- Crearea unei filiale locale:

`git create-branch {{branch_name}}`

- Creați o sucursală la nivel local și la origine:

`git create-branch --remote {{branch_name}}`

- Creați o sucursală la nivel local și în amonte (prin furci):

`git create-branch --remote upstream {{branch_name}}`
