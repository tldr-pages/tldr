# ninja

> Un sistem de compilare proiectat pentru a fi rapid.
> Mai multe informaţii: <https://ninja-build.org/manual.html>

- Construiți în directorul curent:

`ninja`

- Construiți un program într-un director dat:

`ninja -C {{path/to/directory}}`

- Arată obiective (de exemplu, „instalare” și „dezinstalare”):

`ninja -t targets`

- Arată ajutor:

`ninja -h`
