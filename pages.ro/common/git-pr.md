# git pr

> Check out GitHub trage cereri la nivel local.
> Mai multe informaţii: <https://github.com/tj/git-extras/blob/master/Commands.md#git-pr>

- Check out o cerere de tragere specifice:

`git pr {{pr_number}}`

- Check out o cerere de tragere pentru o anumită telecomandă:

`git pr {{pr_number}} {{remote}}`

- Check out o cerere de tragere de la adresa URL:

`git pr {{url}}`

- Curățați ramurile de cerere de tragere vechi:

`git pr clean`
