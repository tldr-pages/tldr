# fastlane

> Construiți și eliberați aplicații mobile din linia de comandă.
> Mai multe informaţii: <https://docs.fastlane.tools/actions/>

- Construiți și semnați aplicația iOS în directorul curent:

`fastlane run build_app`

- Rulați `pod install` pentru proiect în directorul curent:

`fastlane run cocoapods`

- Ștergeți datele derivate din Xcode:

`fastlane run clear_derived_data`

- Scoateți memoria cache pentru păstăi:

`fastlane run clean_cocoapods_cache`
