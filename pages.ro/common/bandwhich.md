# bandwhich

> Afișați utilizarea curentă a rețelei prin proces, conexiune sau IP/gazdă la distanță.
> Mai multe informaţii: <https://github.com/imsnif/bandwhich>

- Afișează doar tabelul de adrese la distanță:

`bandwhich --addresses`

- Arată interogările DNS:

`bandwhich --show-dns`

- Afișează utilizarea totală (cumulativă):

`bandwhich --total-utilization`

- Afișați utilizarea rețelei pentru o anumită interfață de rețea:

`bandwhich --interface {{eth0}}`

- Arată interogările DNS cu un anumit server DNS:

`bandwhich --show-dns --dns-server {{dns_server_ip}}`
