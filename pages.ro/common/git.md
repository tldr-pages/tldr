# git

> Sistem de control al versiunii distribuite.
> Mai multe informaţii: <https://git-scm.com/>

- Verificați versiunea Git:

`git --version`

- Arată ajutor general:

`git --help`

- Afișați ajutor pe o subcomandă Git (cum ar fi `comit`, `log`, etc.):

`git help {{subcommand}}`

- Execută o subcomandă Git:

`git {{subcommand}}`

- Executați o subcomandă Git pe o cale rădăcină de depozit particularizată:

`git -C {{path/to/repo}} {{subcommand}}`

- Executați o subcomandă Git cu un set de configurare dat:

`git -c '{{config.key}}={{value}}' {{subcommand}}`
