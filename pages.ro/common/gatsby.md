# gatsby

> Generator de site static pentru React.
> Mai multe informaţii: <https://gatsbyjs.org>

- Creați un site nou:

`gatsby new {{site_name}}`

- Creați un nou site cu un „starter” Gatsby:

`gatsby new {{site_name}} {{url_of_starter_github_repo}}`

- Porniți un server de dezvoltare locală live-reload:

`gatsby develop`

- Efectuați o construcție de producție și generați HTML statică:

`gatsby build`

- Porniţi un server local care serveşte construcţia de producţie:

`gatsby serve`
