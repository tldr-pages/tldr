# git show-tree

> Afișați un grafic decorat cu arbori cu toate ramurile unui depozit Git, afișând adnotări.
> Parte din `git-extras`.
> Mai multe informaţii: <https://github.com/tj/git-extras/blob/master/Commands.md#git-show-tree>

- Afișați un grafic decorat al arborelui pentru toate ramurile adnotate cu etichete și nume de sucursale:

`git show-tree`
