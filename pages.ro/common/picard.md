# picard

> Următoarea generație MusicBrainz aplicație de etichetare.
> Mai multe informaţii: <https://picard.musicbrainz.org/>

- Porniţi Picard:

`picard`

- Deschideți un set de fișiere:

`picard {{path/to/file1.mp3}} {{path/to/file2.mp3}}`

- Afișează versiunea Picard instalată:

`picard --long-version`
