# eslint

> Un utilitar de linting pluggable pentru JavaScript și JSX.
> Mai multe informaţii: <https://eslint.org>

- Creează eslint config:

`eslint --init`

- Same pe un set dat de fișiere:

`eslint {{filename}}.js {{filename1}}.js`

- Fix probleme de scame:

`eslint --fix`

- Same cu configurare:

`eslint -c {{path/to/config_file}} {{app/src}}`
