# plesk

> Plesk găzduirea panoului de control CLI interfață.
> Mai multe informaţii: <https://docs.plesk.com>

- Generați un link de autentificare automată pentru utilizatorul admin și imprimați-l:

`plesk login`

- Arată informații despre versiunea produsului:

`plesk version`

- Listează toate domeniile găzduite:

`plesk bin domain --list`

- Începeți să urmăriți modificările în fișierul `panel.log':

`plesk log {{panel.log}}`

- Porniți consola MySQL interactivă:

`plesk db`

- Deschideți fișierul principal de configurare Plesk în editorul implicit:

`plesk conf {{panel.ini}}`
