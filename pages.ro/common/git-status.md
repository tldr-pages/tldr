# git status

> Afișați modificările la fișierele dintr-un depozit Git.
> Liste modificate, adăugate și șterse fișiere în comparație cu angajarea extrase în prezent.
> Mai multe informaţii: <https://git-scm.com/docs/git-status>

- Afișează fișierele modificate care nu sunt încă adăugate pentru comitere:

`git status`

- Dă ieșire în [s] format hort:

`git status -s`

- Nu afișa fișierele neurmărite în ieșire:

`git status --untracked-files=no`

- Afișează ieșirea în format [s] Hort împreună cu [b] info ranch:

`git status -sb`
