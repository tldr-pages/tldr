# exenv

> Un instrument pentru instalarea ușoară a versiunilor Elixir și gestionarea mediilor de aplicații.
> Mai multe informaţii: <https://github.com/exenv/exenv>

- Afișează o listă de versiuni instalate:

`exenv versions`

- Utilizați o versiune specifică de Elixir în întregul sistem:

`exenv global {{version}}`

- Utilizați o versiune specifică de Elixir pentru directorul curent de aplicație/proiect:

`exenv local {{version}}`

- Afișează versiunea Elixir selectată în prezent:

`exenv {{version}}`

- Instalați o versiune de Elixir (necesită plugin `elixir-build`  <https://github.com/mururu/elixir-build>):

`exenv install {{version}}`
