# hg update

> Actualizați directorul de lucru la un set de modificări specificat.
> Mai multe informaţii: <https://www.mercurial-scm.org/doc/hg.1.html#update>

- Actualizare la vârful sucursalei curente:

`hg update`

- Actualizare la revizuirea specificată:

`hg update --rev {{revision}}`

- Actualizarea și eliminarea modificărilor necomise:

`hg update --clean`

- Actualizare la ultima comite care se potrivește cu o dată specificată:

`hg update --date {{dd-mm-yyyy}}`
