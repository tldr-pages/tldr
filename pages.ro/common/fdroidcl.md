# fdroidcl

> Client F-Droid CLI.
> Mai multe informaţii: <https://github.com/mvdan/fdroidcl>

- Adu indicele F-Droid:

`fdroidcl update`

- Afișează informații despre o aplicație:

`fdroidcl show {{app_id}}`

- Descărcați un fișier APK:

`fdroidcl download {{app_id}}`

- Căutați o aplicație în index:

`fdroidcl search {{search_pattern}}`

- Instalați o aplicație pe un dispozitiv conectat:

`fdroidcl install {{app_id}}`
