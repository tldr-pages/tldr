# git info

> Afișează informațiile din depozitul Git.
> Parte din `git-extras`.
> Mai multe informaţii: <https://github.com/tj/git-extras/blob/master/Commands.md#git-info>

- Afișează locații la distanță, sucursale la distanță și locale, cele mai recente date de angajament și setări `.git/config`:

`git info`

- Afișați locații la distanță, sucursale la distanță și locale și cele mai recente date de angajament:

`git info --no-config`
