# gem

> Interacţionaţi cu managerul de pachete pentru limbajul de programare Ruby.
> Mai multe informaţii: <https://rubygems.org>

- Căutați bijuterie (e) la distanță și afișați toate versiunile disponibile:

`gem search {{regular_expression}} --all`

- Instalați cea mai recentă versiune a unei bijuterii:

`gem install {{gemname}}`

- Instalați versiunea specifică a unei pietre prețioase:

`gem install {{gemname}} --version {{1.0.0}}`

- Instalați cea mai recentă versiune de potrivire (SEMver) a unei pietre prețioase:

`gem install {{gemname}} --version '~> {{1.0}}'`

- Actualizaţi o bijuterie:

`gem update {{gemname}}`

- Listează toate pietrele locale:

`gem list`

- Dezinstalați o bijuterie:

`gem uninstall {{gemname}}`

- Dezinstalați versiunea specifică a unei pietre prețioase:

`gem uninstall {{gemname}} --version {{1.0.0}}`
