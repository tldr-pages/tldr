# yes

> Exit ceva în mod repetat.
> Această comandă este utilizată în mod obișnuit pentru a răspunde afirmativ la fiecare solicitare prin comenzi de instalare (cum ar fi apt-get).
> Mai multe informaţii: <https://www.gnu.org/software/coreutils/yes>

- Exit în mod repetat „mesaj”:

`yes {{message}}`

- Ieşire repetată „y”:

`yes`

- Acceptă tot ce se cere prin comanda `apt-get`:

`yes | sudo apt-get install {{program}}`
