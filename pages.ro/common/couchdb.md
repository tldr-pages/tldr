# couchdb

> Interfață linie de comandă pentru serverul de baze de date Apache CouchDB.
> Mai multe informaţii: <https://couchdb.apache.org>

- Start couchdb:

`couchdb`

- Start couchdb coajă interactivă:

`couchdb -i`

- Start couchdb ca un proces de fundal:

`couchdb -b`

- Omoară procesul de fundal (Notă: Se va reface dacă este necesar):

`couchdb -k`

- Oprirea procesului de fundal:

`couchdb -d`
