# tslint

> Un utilitar de linting pluggable pentru TypeScript.
> Mai multe informaţii: <https://palantir.github.io/tslint>

- Creați configurația tslint:

`tslint --init`

- Same pe un set dat de fișiere:

`tslint {{filename}}.js {{filename1}}.js`

- Fix probleme de scame:

`tslint --fix`

- Sint cu fișierul de configurare în rădăcina proiectului:

`tslint --project {{path/to/project_root}}`
