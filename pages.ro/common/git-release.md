# git release

> Creați o etichetă Git pentru o versiune.
> Parte din `git-extras`.
> Mai multe informaţii: <https://github.com/tj/git-extras/blob/master/Commands.md#git-release>

- Creați și împingeți o versiune:

`git release {{tag_name}}`

- Creați și împingeți o versiune semnată:

`git release {{tag_name}} -s`

- Creați și împingeți o versiune cu un mesaj:

`git release {{{tag_name}}} -m "{{message}}"`
