# cotton

> Markdown test specificație alergător.
> Mai multe informaţii: <https://github.com/chonla/cotton>

- Utilizați un URL de bază specific:

`cotton -u {{base_url}} {{file}}.md`

- Dezactivați verificarea certificatului (modul nesigur):

`cotton -u {{base_url}} -i {{file}}.md`

- Opriți rularea atunci când un test eșuează:

`cotton -u {{base_url}} -s {{file}}.md`
