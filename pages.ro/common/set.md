# set

> Afișează, setează sau anulează valorile atributelor shell și ale parametrilor de poziție.

- Afișează numele și valorile variabilelor shell:

`set`

- Marcați variabilele care sunt modificate sau create pentru export:

`set -a`

- Anunță încetarea locului de muncă imediat:

`set -b`

- Setați diferite opțiuni, de exemplu, activați editarea liniei de stil `vi`:

`set -o {{vi}}`
