# composer-require-checker

> Un instrument CLI pentru a analiza dependențele Compozitorului pentru dependențe soft.
> Mai multe informaţii: <https://github.com/maglnet/ComposerRequireChecker>

- Analizează un fișier Compozitor JSON:

`composer-require-checker check {{path/to/composer.json}}`

- Analizați un fișier JSON Compozitor cu o configurație specifică:

`composer-require-checker check --config-file {{path/to/config.json}} {{path/to/composer.json}}`
