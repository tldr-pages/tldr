# cradle package

> Gestionați pachetele pentru o instanță Cradle.
> Mai multe informaţii: <https://cradlephp.github.io/docs/3.B.-Reference-Command-Line-Tools.html#package>

- Afișează o listă de pachete disponibile:

`cradle package list`

- Caută un pachet:

`cradle package search {{package}}`

- Instalați un pachet de la Packagist:

`cradle package install {{package}}`

- Instalați o versiune specifică a unui pachet:

`cradle package install {{package}} {{version}}`

- Actualizarea unui pachet:

`cradle package update {{package}}`

- Actualizați un pachet la o anumită versiune:

`cradle package update {{package}} {{version}}`

- Eliminați un anumit pachet:

`cradle package remove {{package}}`
