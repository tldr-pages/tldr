# serverless

> Toolkit pentru implementarea și operarea arhitecturilor fără server pe AWS, Google Cloud, Azure și IBM OpenWhisk.
> Comenzile pot fi executate fie folosind comanda `serverless`, fie alias, `sls`.
> Mai multe informaţii: <https://serverless.com/>

- Creați un proiect fără server:

`serverless create`

- Creați un proiect fără server dintr-un șablon:

`serverless create --template {{template_name}}`

- Implementarea la un furnizor de cloud:

`serverless deploy`

- Afișați informații despre un proiect fără server:

`serverless info`

- Invocați o funcție implementată:

`serverless invoke -f {{function_name}}`

- Urmați jurnalele pentru un proiect:

`serverless logs -t`
