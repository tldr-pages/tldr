# gh ssh-key

> Gestionați cheile SSH GitHub din linia de comandă.
> Mai multe informaţii: <https://cli.github.com/manual/gh_ssh-key>

- Ajutor pentru afișare:

`gh ssh-key`

- Lista cheilor SSH pentru utilizatorul autentificat în prezent:

`gh ssh-key list`

- Adăugați o cheie SSH la contul utilizatorului autentificat în prezent:

`gh ssh-key add {{path/to/key.pub}}`

- Adăugați o cheie SSH la contul utilizatorului autentificat în prezent cu un anumit titlu:

`gh ssh-key add --title {{title}} {{path/to/key.pub}}`
