# arc

> Arcanist: Un CLI pentru Phabricator.
> Mai multe informaţii: <https://secure.phabricator.com/book/phabricator/article/arcanist/>

- Trimiteți modificările la Diferențial pentru revizuire:

`arc diff`

- Afișați informațiile de revizuire în așteptare:

`arc list`

- Actualizare Git comite mesaje după revizuire:

`arc amend`

- Push Git modificări:

`arc land`
