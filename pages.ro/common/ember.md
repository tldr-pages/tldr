# ember

> Utilitarul de linie de comandă Ember.js.
> Folosit pentru crearea și întreținerea aplicațiilor Ember.js.
> Mai multe informaţii: <https://cli.emberjs.com>

- Creați o nouă aplicație Ember:

`ember new {{my_new_app}}`

- Creați un nou addon Ember:

`ember addon {{my_new_addon}}`

- Construirea proiectului:

`ember build`

- Construiți proiectul în modul de producție:

`ember build -prod`

- Rulați serverul de dezvoltare:

`ember serve`

- Rulați suita de testare:

`ember test`

- Rulați un plan pentru a genera ceva de genul un traseu sau o componentă:

`ember generate {{type}} {{name}}`

- Instalați un addon ember-cli:

`ember install {{name_of_addon}}`
