# apm

> Editorul Atom Package Manager.
> A se vedea „atom'”.
> Mai multe informaţii: <https://github.com/atom/apm>

- Instalați pachete de la http://atom.io/packages și teme din http://atom.io/themes:

`apm install {{package_name}}`

- Eliminarea pachetelor/temelor:

`apm remove {{package_name}}`

- Upgrade pachete/teme:

`apm upgrade {{package_name}}`
