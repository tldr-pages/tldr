# flask

> Un script general de utilitate pentru aplicații Flask. Încarcă aplicația definită în variabila de mediu `FLASK_APP”.
> Mai multe informaţii: <https://flask.palletsprojects.com/en/1.1.x/cli/>

- Rulați un server de dezvoltare:

`flask run`

- Afișează rutele pentru aplicație:

`flask routes`

- Rulați o coajă interactivă Python în contextul aplicației:

`flask shell`
