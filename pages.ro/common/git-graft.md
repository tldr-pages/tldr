# git graft

> Îmbinați angajările dintr-o anumită ramură într-o altă ramură și ștergeți ramura sursă.
> Parte din `git-extras`.
> Mai multe informaţii: <https://github.com/tj/git-extras/blob/master/Commands.md#git-graft>

- Mergeți toate angajările care nu sunt prezente pe ramura țintă de la ramura sursă la ramura țintă și ștergeți ramura sursă:

`git graft {{source_branch}} {{target_branch}}`
