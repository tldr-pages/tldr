# offlineimap

> Sincronizați un server IMAP la distanță cu folderele locale Maildir.
> Mai multe informaţii: <http://www.offlineimap.org>

- Sincronizați o dată, fără a activa reîmprospătarea automată:

`offlineimap -o`

- Sincronizați un anumit cont:

`offlineimap -a {{account}}`

- Sincronizarea unui anumit folder:

`offlineimap -f {{folder}}`
