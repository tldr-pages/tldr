# rails

> Un server MVC cadru scris în Ruby.
> Mai multe informaţii: <https://guides.rubyonrails.org/command_line.html>

- Crearea unui nou proiect de șine:

`rails new "{{project_name}}"`

- Porniți serverul local pentru proiectul curent pe portul 3000:

`rails server`

- Porniți serverul local pentru proiectul curent pe un port specificat:

`rails server -p "{{port}}"`

- Deschideți consola pentru a interacționa cu aplicația din linia de comandă:

`rails console`

- Verificați versiunea curentă a șinelor:

`rails --version`
