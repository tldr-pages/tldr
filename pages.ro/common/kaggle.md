# kaggle

> CLI oficial pentru Kaggle implementat în Python 3.
> Mai multe informaţii: <https://github.com/Kaggle/kaggle-api>

- Vizualizați valorile de configurare curente:

`kaggle config view`

- Descărcaţi un anumit fişier dintr-un set de date al competiţiei:

`kaggle competitions download {{competition}} -f {{filename}}`
