# git delete-branch

> Ștergeți ramurile Git locale și la distanță.
> Parte din `git-extras`. Dacă ștergeți sucursala extrase, numai ramura la distanță va fi ștearsă.
> Mai multe informaţii: <https://github.com/tj/git-extras/blob/master/Commands.md#git-delete-branch>

- Ștergeți o ramură Git locală și la distanță:

`git delete-branch {{branch_name}}`

- Ștergeți mai multe ramuri locale și la distanță Git:

`git delete-branch {{branch_name1 branch_name2 ...}}`
