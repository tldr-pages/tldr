# ignite

> Un CLI pentru react nativ boilerplates, plugin-uri, generatoare și multe altele.
> Mai multe informaţii: <https://infinite.red/ignite>

- Creați un nou proiect React nativ:

`ignite new {{project_name}}`

- Generarea fișierului dintr-un plugin:

`ignite generate {{plugin_name}} {{filename}}`

- Adăugați un plugin Ignite la proiect:

`ignite add {{plugin_name}}`

- Eliminați un plugin Ignite din proiect:

`ignite remove {{plugin_name}}`
