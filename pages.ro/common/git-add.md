# git add

> Adaugă fișiere noi sau modificate în zona de pregătire.
> Mai multe informații: <https://git-scm.com/docs/git-add>.

- Adaugă un fișier în zona de pregătire:

`git add {{cale/către/fișier}}`

- Adaugă toate fișierele (urmărite și neurmărite):

`git add {{[-A|--all]}}`

- Adaugă toate fișierele recursiv pornind de la folderul curent:

`git add .`

- Adaugă doar fișierele deja urmărite:

`git add {{[-u|--update]}}`

- Adaugă și fișierele ignorate:

`git add {{[-f|--force]}}`

- Adaugă părți din fișiere în mod interactiv:

`git add {{[-p|--patch]}}`

- Adaugă părți dintr-un fișier specificat în mod interactiv:

`git add {{[-p|--patch]}} {{cale/către/fișier}}`

- Adaugă un fișier în mod interactiv:

`git add {{[-i|--interactive]}}`
