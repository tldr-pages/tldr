# irssi

> Client IRC bazat pe text.
> Mai multe informaţii: <https://irssi.org>

- Deschideți irssi și conectați-vă la un server cu o poreclă:

`irssi -n {{nickname}} -c {{irc.example.com}}`

- Deschideți irssi și conectați-vă cu un anumit server pe un anumit port:

`irssi -c {{irc.example.com}} -p {{port}}`

- Vezi ajutorul:

`irssi --help`

- Alătură-te unui canal:

`/join {{#channelname}}`

- Schimbați fereastra activă (începe la 1):

`/win {{window_number}}`

- Ieșiți din aplicație curat și renunțând la orice server (e):

`/quit`
