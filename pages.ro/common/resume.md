# resume

> instrument CLI pentru a configura cu ușurință un nou CV.
> Mai multe informaţii: <https://github.com/jsonresume/resume-cli>

- Creați un nou fișier `resume.json` în directorul de lucru curent:

`resume init`

- Validează o „resume.json” în raport cu testele de schemă pentru a se asigura că respectă standardul:

`resume validate`

- Exportați un CV local într-un format HTML sau PDF stilizat:

`resume export {{path/to/html_or_pdf}}`

- Porniți un server web care servește un local `resume.json`:

`resume serve`
