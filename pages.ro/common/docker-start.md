# docker start

> Porniți unul sau mai multe containere oprite.
> Mai multe informaţii: <https://docs.docker.com/engine/reference/commandline/start/>

- Arată ajutor:

`docker start`

- Porniți un container de andocare:

`docker start {{container}}`

- Porniți un container, atașarea stdout și stderr și semnale de redirecționare:

`docker start --attach {{container}}`

- Porniți unul sau mai multe containere separate prin spațiu:

`docker start {{container(s)}}`
