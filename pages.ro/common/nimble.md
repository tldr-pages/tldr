# nimble

> Manager de pachete pentru limbajul de programare Nim.
> Gestionați proiectele Nim și dependențele acestora.
> Mai multe informaţii: <https://github.com/nim-lang/nimble>

- Caută pachete:

`nimble search {{search_string}}`

- Instalați un pachet:

`nimble install {{package_name}}`

- Lista pachetelor instalate:

`nimble list -i`

- Creați un nou pachet Nimble în directorul curent:

`nimble init`

- Construiește un pachet agil:

`nimble build`

- Instalați un pachet Abil:

`nimble install`
