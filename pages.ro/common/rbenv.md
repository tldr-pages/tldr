# rbenv

> Un instrument pentru instalarea cu ușurință a versiunilor Ruby și gestionarea mediilor de aplicații.
> Mai multe informaţii: <https://github.com/rbenv/rbenv>

- Instalați o versiune Ruby:

`rbenv install {{version}}`

- Afișează o listă cu cele mai recente versiuni stabile pentru fiecare Ruby:

`rbenv install --list`

- Afișează o listă de versiuni Ruby instalate:

`rbenv versions`

- Utilizați o anumită versiune Ruby în întregul sistem:

`rbenv global {{version}}`

- Utilizați o anumită versiune Ruby pentru o aplicație/director de proiect:

`rbenv local {{version}}`

- Afișează versiunea Ruby selectată în prezent:

`rbenv version`

- Dezinstalați o versiune Ruby:

`rbenv uninstall {{version}}`

- Afișează toate versiunile Ruby care conțin executabilul specificat:

`rbenv whence {{executable}}`
