# ghcup

> Instalator de lanțuri de instrumente Haskell.
> Instalați, gestionați și actualizați lanțurile de instrumente Haskell.
> Mai multe informaţii: <https://gitlab.haskell.org/haskell/ghcup-hs>

- Porniți TUI interactiv:

`ghcup tui`

- Lista de versiuni disponibile GHC/Cabal:

`ghcup list`

- Instalați versiunea recomandată GHC:

`ghcup install ghc`

- Instalați o versiune specifică GHC:

`ghcup install ghc {{version}}`

- Setați versiunea curentă „activă” GHC:

`ghcup set ghc {{version}}`

- Instalare cablu de instalare:

`ghcup install cabal`

- Actualizare `ghcu` în sine:

`ghcup upgrade`
