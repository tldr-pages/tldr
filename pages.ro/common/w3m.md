# w3m

> Un browser web bazat pe text.
> Suportă intrarea SSL și mouse-ul, chiar și prin SSH.
> Mai multe informaţii: <http://w3m.sourceforge.net>

- Deschide un URL:

`w3m {{http://example.com}}`

- Deschide un URL în modul monocrom:

`w3m {{http://example.com}} -M`

- Deschideți un URL fără suport pentru mouse:

`w3m {{http://example.com}} --no-mouse`

- Deschideți o nouă filă de browser:

`Shift + T`

- Afișează istoricul browserului:

`Ctrl + H`

- Renunţă la w3m:

`'q' then 'y'`
