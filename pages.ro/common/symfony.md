# symfony

> Componenta consolei pentru cadrul Symfony.
> Mai multe informaţii: <https://symfony.com>

- Crearea unui nou proiect Symfony:

`symfony new {{name}}`

- Rulați un server web local:

`symfony serve`

- Opriți serverul web local:

`symfony server:stop`

- Verificați dacă există probleme de securitate în dependențele proiectului:

`symfony security:check`
