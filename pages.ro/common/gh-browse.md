# gh browse

> Deschideți un depozit GitHub în browser sau imprimați URL-ul.
> Mai multe informaţii: <https://cli.github.com/manual/gh_browse>

- Deschideți pagina principală a depozitului curent în browserul web implicit:

`gh browse`

- Deschideți pagina de pornire a unui anumit depozit în browserul web implicit:

`gh browse {{owner}}/{{repository}}`

- Deschideți pagina de setări a depozitului curent în browserul web implicit:

`gh browse --settings`

- Deschideți wiki-ul depozitului curent în browserul web implicit:

`gh browse --wiki`

- Deschideți o anumită problemă sau trageți solicitarea în browserul web:

`gh browse {{issue_or_pull_request_number}}`

- Deschideți o anumită ramură în browserul web:

`gh browse --branch {{branch_name}}`

- Deschideți un anumit fișier sau director al depozitului curent în browserul web:

`gh browse {{path_from_root_of_repository}}`

- Imprimați adresa URL de destinație fără a deschide browserul web:

`gh browse --no-browser`
