# virsh-help

> Afișează informații despre comenzile sau grupurile de comenzi `virsh`.
> A se vedea, de asemenea: `virsh`.
> Mai multe informaţii: <https://manned.org/virsh>

- Listează comenzile `virsh` grupate în categorii înrudite:

`virsh help`

- Lista categoriilor de comenzi:

`virsh help | grep "keyword"`

- Listează comenzile dintr-o categorie:

`virsh help {{category_keyword}}`

- Arată ajutor pentru o comandă:

`virsh help {{command}}`
