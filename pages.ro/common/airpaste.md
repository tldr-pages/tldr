# airpaste

> Partajați mesaje și fișiere în aceeași rețea utilizând MDN.
> Mai multe informaţii: <https://github.com/mafintosh/airpaste>

- Așteptați un mesaj și afișați-l când este primit:

`airpaste`

- Trimite text:

`echo {{text}} | airpaste`

- Trimite un fişier:

`airpaste < {{path/to/file}}`

- Primiți un fișier:

`airpaste > {{path/to/file}}`

- Creează sau aderă la un canal:

`airpaste {{channel_name}}`
