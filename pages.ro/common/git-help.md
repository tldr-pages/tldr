# git help

> Afișați informații de ajutor despre Git.
> Mai multe informaţii: <https://git-scm.com/docs/git-help>

- Afișați ajutor despre o subcomandă Git specifică:

`git help {{subcommand}}`

- Afișați ajutor despre o anumită subcomandă Git într-un browser web:

`git help --web {{subcommand}}`

- Afișează o listă a tuturor subcomenzilor Git disponibile:

`git help --all`

- Listează ghidurile disponibile:

`git help --guide`

- Listează toate variabilele de configurare posibile:

`git help --config`
