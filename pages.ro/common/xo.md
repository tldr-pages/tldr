# xo

> Un utilitar pluggable, cu configurație zero pentru JavaScript.
> Mai multe informaţii: <https://github.com/xojs/xo>

- Fișierele Lint din directorul „src”:

`xo`

- Lint un set dat de fișiere:

`xo {{file1}}.js {{file2}}.js`

- Remedierea automată a oricăror probleme de scame găsite:

`xo --fix`

- Same folosind spații ca indentare în loc de file:

`xo --space`

- Sint folosind stilul de cod „mai frumos”:

`xo --prettier`
