# elinks

> Un browser bazat pe text similar cu lynx.
> Mai multe informaţii: <http://elinks.or.cz>

- Elinks de pornire:

`elinks`

- Renunţă la elinks:

`Ctrl + C`

- Dump de ieșire a paginii web la consola, colorarea textului cu codurile de control ANSI:

`elinks -dump -dump-color-mode {{1}} {{url}}`
