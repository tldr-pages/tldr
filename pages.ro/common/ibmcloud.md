# ibmcloud

> Un instrument de linie de comandă pentru gestionarea aplicaţiilor şi serviciilor IBM Cloud.
> Mai multe informaţii: <https://cloud.ibm.com/docs/cli?topic=cli-ibmcloud_cli>

- Actualizați `ibmcloud` la cea mai recentă versiune:

`ibmcloud update`

- Instalați modulul Cloud Toundry pentru accesarea serviciilor Cloud Foundry:

`ibmcloud cf install`

- Lista tuturor regiunilor IBM Cloud disponibile:

`ibmcloud regions`

- Afișează versiunea `ibmcloud`:

`ibmcloud version`

- Ajutor pentru afișare:

`ibmcloud help`

- Afișează ajutor pentru o subcomandă:

`ibmcloud help {{subcommand}}`
