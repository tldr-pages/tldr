# sails

> Sails.js este un cadru MVC nivel de întreprindere în timp real construit pe partea de sus a Node.js.
> Mai multe informaţii: <https://sailsjs.com>

- Porniţi pânzele:

`sails lift`

- Creați un nou proiect Sails:

`sails new {{projectName}}`

- Generarea Sails API:

`sails generate {{name}}`

- Generare Sails Controller:

`sails generate controller {{name}}`

- Generează Sails Model:

`sails generate model {{name}}`
