# cake

> Procesorul de linie de comandă pentru cadrul CakePHP.
> Mai multe informaţii: <https://cakephp.org>

- Afișați informații de bază despre aplicația curentă și comenzile disponibile:

`cake`

- Afișează o listă a rutelor disponibile:

`cake routes`

- Ștergeți cache-urile de configurare:

`cake cache clear_all`

- Construiți memoria cache de metadate:

`cake schema_cache build --connection {{connection}}`

- Ștergeți memoria cache a metadatelor:

`cake schema_cache clear`

- Ștergeți un singur tabel cache:

`cake schema_cache clear {{table_name}}`

- Porniți un server web de dezvoltare (implicit la portul 8765):

`cake server`

- Porniți o instanță coajă interactivă REPL:

`cake console`
