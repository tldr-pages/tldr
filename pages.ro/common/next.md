# next

> React framework care utilizează randarea pe partea de server pentru construirea de aplicații web optimizate.
> Mai multe informaţii: <https://nextjs.org/docs>

- Porniți aplicația curentă în modul de dezvoltare:

`next dev`

- Porniți aplicația curentă și ascultați pe un anumit port:

`next dev --port {{port}}`

- Construiți aplicația curentă optimizată pentru producție:

`next build`

- Porniți aplicația compilată în modul de producție:

`next start`

- Porniți aplicația compilată și ascultați pe un anumit port:

`next start --port {{port}}`

- Exportați aplicația curentă în pagini HTML statice:

`next export`

- Afișează starea de telemetrie Next.js:

`next telemetry`

- Afișează ajutor pentru o subcomandă:

`next {{build|dev|export|start|telemetry}} --help`
