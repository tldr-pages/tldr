# middleman

> Generator site-ul static scris în Ruby.
> Mai multe informaţii: <https://middlemanapp.com/>

- Crearea unui nou proiect intermediar:

`middleman init "{{project_name}}"`

- Porniți serverul local pentru proiectul curent pe portul 4567:

`middleman server`

- Porniți serverul local pentru proiectul curent pe un port specificat:

`middleman server -p "{{port}}"`

- Construiți proiectul în directorul curent pentru a vă pregăti pentru implementare:

`bundle exec middleman build`

- Implementați proiectul intermediar în directorul curent:

`middleman deploy`
