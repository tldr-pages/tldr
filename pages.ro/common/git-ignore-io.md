# git ignore-io

> Generați fișiere.gitignore din șabloane predefinite.
> Mai multe informaţii: <https://github.com/tj/git-extras/blob/master/Commands.md#git-ignore-io>

- Lista șabloanelor disponibile:

`git ignore-io list`

- Generați un șablon .gitignore:

`git ignore-io {{item_a,item_b,item_n}}`
