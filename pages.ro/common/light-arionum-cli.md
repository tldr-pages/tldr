# light-arionum-cli

> Portofelul de lumină PHP pentru criptomoneda Arionum.
> Mai multe informaţii: <https://github.com/arionum/lightWalletCLI>

- Generează o nouă pereche de chei publice/private:

`light-arionum-cli`

- Afișează soldul adresei curente:

`light-arionum-cli balance`

- Afișează soldul adresei specificate:

`light-arionum-cli balance {{address}}`

- Trimiteți o tranzacție cu un mesaj opțional:

`light-arionum-cli send {{address}} {{value}} {{optional_message}}`

- Exportați informațiile curente despre portofel:

`light-arionum-cli export`

- Afișați informații despre blocul curent:

`light-arionum-cli block`

- Afișează informații despre tranzacțiile adresei curente:

`light-arionum-cli transactions`

- Afișați informații despre o anumită tranzacție:

`light-arionum-cli transaction {{transaction_id}}`
