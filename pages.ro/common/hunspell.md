# hunspell

> Verificarea ortografiei.
> Mai multe informaţii: <https://hunspell.github.io/>

- Verificați ortografia unui fișier:

`hunspell {{path/to/file}}`

- Verificați ortografia unui fișier cu dicționarul ro_US:

`hunspell -d {{en_US}} {{path/to/file}}`

- Lista cuvintelor ortografiate greșit într-un fișier:

`hunspell -l {{path/to/file}}`
