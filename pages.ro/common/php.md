# php

> Interfață de linie de comandă PHP.
> Mai multe informaţii: <https://php.net>

- Analizaţi şi executaţi un script php:

`php {{file}}`

- Verificați sintaxa pe (adică scame) un script PHP:

`php -l {{file}}`

- Rulați PHP interactiv:

`php -a`

- Rulați codul PHP (Note: Nu utilizați <? ? > etichete; evadeze ghilimele duble cu backslash):

`php -r "{{code}}"`

- Porniți un server web încorporat în PHP în directorul curent:

`php -S {{host:port}}`

- Obțineți o listă a extensiilor PHP instalate:

`php -m`

- Afișează informații despre configurația PHP curentă:

`php -i`
