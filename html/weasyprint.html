weasyprint

Render HTML to PDF or PNG.

Homepage: home page

- Render a HTML file to PDF:
weasyprint path/to/input.html path/to/outputhome pagepdf
- Render a HTML file to PNG, including an additional user stylesheet:
weasyprint path/to/input.html path/to/outputhome pagepng --stylesheet path/to/stylesheet.css
- Output additional debugging information when rendering:
weasyprint path/to/input.html path/to/outputhome pagepdf --verbose
- Specify a custom resolution when outputting to PNG:
weasyprint path/to/input.html path/to/outputhome pagepng --resolution 300
- Specify a base url for relative urls in the input HTML file:
weasyprint path/to/input.html path/to/outputhome pagepng --base-url url_or_filename
