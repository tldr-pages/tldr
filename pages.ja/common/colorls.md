# colorls

> カラー化と Font Awesome アイコンの使用によりターミナルの ls コマンドを美しくした Ruby の gem です。
> 詳細情報: <https://github.com/athityakumar/colorls#flags>。

- ファイルを 1 行ずつ表示する:

`colorls -1`

- 隠しファイルを含めた全てのファイルを表示する:

`colorls {{[-a|--all]}}`

- 全てのファイルを長文形式（パーミッション、所有者、サイズ、更新日時）で表示する:

`colorls {{[-l|--long]}} {{[-a|--all]}}`

- ディレクトリのみ表示する:

`colorls {{[-d|--dirs]}}`
