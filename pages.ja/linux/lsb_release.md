# lsb_release

> LSB (Linux Standard Base)とディストリビューション固有の情報を取得します。
> もっと詳しく: <https://manned.org/lsb_release>。

- 利用可能なすべての情報を表示する:

`lsb_release -a`

- オペレーティングシステムの説明(通常はフルネーム)を表示する:

`lsb_release -d`

- オペレーティングシステムの名前(ID)だけを表示する:

`lsb_release -i -s`

- ディストリビューションのリリース番号とコードネームを表示する:

`lsb_release -rcs`
