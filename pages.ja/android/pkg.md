# pkg

> Termux 用パッケージ管理ユーティリティ。
> もっと詳しく: <https://wiki.termux.com/wiki/Package_Management>。

- すべてのインストールされたパッケージをアップグレードします:

`pkg upgrade`

- パッケージをインストールします:

`pkg install {{package}}`

- パッケージをアンインストールします:

`pkg uninstall {{package}}`

- パッケージを再インストールします:

`pkg reinstall {{package}}`

- パッケージを検索します:

`pkg search {{package}}`
