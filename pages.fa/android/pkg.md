# pkg

> ابزار مدیر بسته ها برای ترماکس.
> اطلاعات بیشتر: <https://wiki.termux.com/wiki/Package_Management>.

- ارتقا تمامی بسته های نصب شده :

`pkg upgrade`

- نصب یک بسته :

`pkg install {{بسته}}`

- حذف یک بسته :

`pkg uninstall {{بسته}}`

- نصب دوباره یک بسته :

`pkg reinstall {{بسته}}`

- جستجوی یک بسته :

`pkg search {{بسته}}`
