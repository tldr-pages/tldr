# apt

> ابزار مدیریت بسته‌ها برای توزیع‌های مبتنی بر دبیان.
> جایگزین توصیه شده برای `apt-get` در نسخه‌های ۱۶.۰۴ و بالاتر اوبونتو در حالت تعاملی.
> برای دستورات معادل در سایر مدیریت‌کننده‌های بسته، به <https://wiki.archlinux.org/title/Pacman/Rosetta> مراجعه کنید.
> اطلاعات بیشتر: <https://manned.org/apt.8>.

- به‌روزرسانی فهرست بسته‌ها و نسخه‌های موجود (توصیه می‌شود قبل از سایر دستورات `apt` اجرا شود):

`sudo apt update`

- جستجوی یک بسته مشخص:

`apt search {{package}}`

- نمایش اطلاعات یک بسته:

`apt show {{package}}`

- نصب یک بسته یا به‌روزرسانی آن به آخرین نسخه موجود:

`sudo apt install {{package}}`

- حذف یک بسته (استفاده از `purge` به جای آن، فایل‌های پیکربندی را نیز حذف می‌کند):

`sudo apt remove {{package}}`

- به‌روزرسانی تمام بسته‌های نصب شده به جدیدترین نسخه‌های موجود:

`sudo apt upgrade`

- نمایش تمام بسته‌ها:

`apt list`

- نمایش بسته‌های نصب شده:

`apt list --installed`
