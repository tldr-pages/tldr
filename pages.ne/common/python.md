# python

> पाइथन भाषा अनुवादक.
> थप जानकारी: <https://www.python.org>.

- REPL सुरु गर्नुहोस् (अन्तर्क्रियात्मक शेल):

`python`

- एक विशिष्ट पाइथन फाइल कार्यान्वयन गर्नुहोस्:

`python {{फाइल/को/पथ}}`

- एक विशिष्ट पाइथन फाइल कार्यान्वयन गर्नुहोस् र REPL सुरु गर्नुहोस्:

`python -i {{फाइल/को/पथ}}`

- पाइथन अभिव्यक्ति कार्यान्वयन गर्नुहोस्:

`python -c "{{एक्सप्रेसन}}"`

- निर्दिष्ट पुस्तकालय मोड्युलको स्क्रिप्ट चलाउनुहोस्:

`python -m {{मोडयुल}} {{तर्कहरू}}`

- `pip` प्रयोग गरेर प्याकेज स्थापना गर्नुहोस्:

`python -m {{pip}} install {{प्याकेजको नाम}}`

- अन्तरक्रियात्मक रूपमा पाइथन स्क्रिप्ट डिबग गर्नुहोस्:

`python -m {{pdb}} {{फाइल/को/पथ}}`

- हालको डाइरेक्टरीमा पोर्ट 8000 मा निर्मित HTTP सर्भर सुरु गर्नुहोस्:

`python -m {{http.server}}`
