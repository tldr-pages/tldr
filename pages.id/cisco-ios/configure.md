# configure

> Masuk menuju mode konfigurasi sistem.
> Informasi lebih lanjut: <https://www.cisco.com/c/en/us/td/docs/ios/fundamentals/command/reference/cf_book/cf_c1.html#configure_check_syntax>.

- Masuk mode konfigurasi dari terminal:

`{{[conf|configure]}} {{[t|terminal]}}`

- Keluar menuju tingkat konfigurasi induk:

`exit`

- Keluar dari mode konfigurasi:

`end`
