# addpart

> Beritahu kernel Linux tentang keberadaan partisi yang ditentukan.
> Pembungkus (wrapper) sederhana untuk ioctl `add partition`.
> Informasi lebih lanjut: <https://manned.org/addpart>.

- Beritahu kernel tentang keberadaan partisi yang ditentukan:

`addpart {{device}} {{partition}} {{start}} {{length}}`
