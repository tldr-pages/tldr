# fileicon

> Atur ikon kustom untuk berkas atau direktori.
> Informasi lebih lanjut: <https://github.com/mklement0/fileicon>.

- Tentukan suatu ikon kustom khusus untuk suatu berkas atau direktori:

`fileicon set {{jalan/menuju/berkas_atau_direktori}} {{jalan/menuju/berkas_ikon.png}}`

- Hapus ikon kustom untuk suatu berkas atau direktori:

`fileicon rm {{jalan/menuju/berkas_atau_direktori}}`

- Simpan ikon kustom berkas atau direktori sebagai berkas baru `.icns` dalam direktori saat ini:

`fileicon get {{jalan/menuju/berkas_atau_direktori}}`

- Cek apakah suatu berkas atau direktori sedang memiliki ikon kustom:

`fileicon test {{jalan/menuju/berkas_atau_direktori}}`
