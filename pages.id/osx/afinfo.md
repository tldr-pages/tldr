# afinfo

> Pembuka metadata berkas audio untuk OS X.
> Perintah bawaan OS X.
> Informasi lebih lanjut: <https://keith.github.io/xcode-man-pages/afinfo.1.html>.

- Tampilkan informasi tentang suatu berkas audio:

`afinfo {{jalan/menuju/berkas}}`

- Tampilkan deskripsi berkas audio dalam satu baris:

`afinfo --brief {{jalan/menuju/berkas}}`

- Tampilkan informasi metadata beserta isi data InfoDictionary dalam berkas audio:

`afinfo --info {{jalan/menuju/berkas}}`

- Tampilkan luaran dalam format XML:

`afinfo --xml {{jalan/menuju/berkas}}`

- Tampilkan peringatan atas berkas audio, bila ada:

`afinfo --warnings {{jalan/menuju/berkas}}`

- Tampilkan bantuan:

`afinfo --help`
