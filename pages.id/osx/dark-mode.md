# dark-mode

> Atur pengaturan mode gelap dalam macOS.
> Informasi lebih lanjut: <https://github.com/sindresorhus/dark-mode>.

- Alihkan mode gelap (hidupkan jika saat ini mati, matikan jika saat ini menyala):

`dark-mode`

- Nyalakan mode gelap:

`dark-mode on`

- Matikan mode gelap:

`dark-mode off`

- Cek apabila mode gelap sedang menyala:

`dark-mode status`
