# tree

> Menampilkan struktur direktori pada suatu drive atau path dengan tampilan grafis seperti hierarki pohon.
> Informasi lebih lanjut: <https://docs.microsoft.com/windows-server/administration/windows-commands/tree>.

- Tampilkan struktur direktori saat ini:

`tree`

- Tampilkan struktur direktori yang ditentukan:

`tree {{lokasi/ke/direktori}}`

- Tampilkan nama file dari direktori saat ini:

`dir /f`

- Tampilkan struktur dari direktori saat ini dalam bentuk teks alih-alih karakter grafis:

`dir /a`
