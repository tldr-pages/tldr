# Get-History

> Tampilkan riwayat perintah PowerShell.
> Catatan: Perintah ini hanya dapat digunakan dalam PowerShell.
> Informasi lebih lanjut: <https://learn.microsoft.com/powershell/module/microsoft.powershell.core/get-history>.

- Tampilkan daftar riwayat perintah beserta nomor induk (ID)-nya:

`Get-History`

- Dapatkan perintah PowerShell menurut ID entri riwayat:

`Get-History -Id {{nomor_induk}}`

- Tampilkan riwayat `jumlah` (sekian) perintah terakhir:

`Get-History -Count {{jumlah}}`
