# c99

> Lakukan kompilasi program C berdasarkan standar bahasa pemrograman ISO C.
> Informasi lebih lanjut: <https://manned.org/c99>.

- Lakukan kompilasi terhadap (beberapa) berkas kode sumber untuk membuat berkas program executable:

`c99 {{berkas.c}}`

- Lakukan kompilasi dan tentukan nama berkas program executable:

`c99 -o {{nama_executable}} {{berkas.c}}`

- Lakukan kompilasi dan buat berkas-berkas objek:

`c99 -c {{berkas.c}}`

- Lakukan kompilasi, hubungkan dengan berkas-berkas objek, dan bangun suatu berkas executable:

`c99 {{berkas.c}} {{berkas.o}}`
