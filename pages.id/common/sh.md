# sh

> Bourne shell, penerjemah standar program baris perintah.
> Lihat juga: `!`, `^`.
> Informasi lebih lanjut: <https://manned.org/sh>.

- Jalankan sesi interaktif baru:

`sh`

- Jalankan suatu perintah, kemudian keluar:

`sh -c "{{perintah}}"`

- Jalankan suatu naskah/skrip perintah:

`sh {{jalan/menuju/naskah.sh}}`

- Baca dan jalankan kumpulan perintah dari `stdin`:

`sh -s`
