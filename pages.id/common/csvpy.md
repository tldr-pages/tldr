# csvpy

> Muat file CSV ke dalam shell Python sebagai objek CSVKitReader.
> Bagian dari csvkit.
> Informasi lebih lanjut: <https://csvkit.readthedocs.io/en/latest/scripts/csvpy.html>.

- Muat sebuah file CSV ke dalam objek CSVKitReader:

`csvpy {{jalan/menuju/data.csv}}`

- Muat sebuah file CSV ke dalam objek CSVKitDictReader:

`csvpy --dict {{jalan/menuju/data.csv}}`
