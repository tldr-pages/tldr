# claude

> Alat pengodean berbasis agen yang memahami basis kode (codebase) Anda dan membantu Anda mengode lebih cepat melalui perintah bahasa alami.
> Informasi lebih lanjut: <https://code.claude.com/docs/en/cli-reference>.

- Jalankan dengan perintah (prompt):

`claude prompt`

- Perbarui `claude`:

`claude update`

- Tampilkan daftar server MCP yang ditentukan:

`claude mcp list`

- Buat komit dengan perintah:

`claude commit`

- Tampilkan daftar konfigurasi:

`claude config list`

- Lanjutkan sesi terbaru:

`claude {{[-c|--continue]}}`

- Pilih sesi secara interaktif untuk dilanjutkan:

`claude {{[-r|--resume]}}`
