# git gc

> Optimalkan repositori lokal saat ini dengan menghapus berkas-berkas tak terpakai.
> Informasi lebih lanjut: <https://git-scm.com/docs/git-gc>.

- Optimalkan repositori saat ini:

`git gc`

- Optimalkan secara agresif, akan memakan waktu lebih lama:

`git gc --aggressive`

- Jangan potong objek-objek yang longgar (program ini akan memotong secara default):

`git gc --no-prune`

- Jangan tampilkan luaran pesan apapun:

`git gc --quiet`

- Tampilkan bantuan:

`git gc --help`
