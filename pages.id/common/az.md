# az

> Program baris perintah resmi untuk Microsoft Azure.
> Beberapa subperintah seperti `login` memiliki dokumentasi terpisah.
> Informasi lebih lanjut: <https://learn.microsoft.com/cli/azure/reference-index>.

- Masuk ke dalam platform Azure:

`az login`

- Kelola informasi langganan layanan Azure:

`az account`

- Tampilkan daftar seluruh daftar instansi penyimpanan Azure Disk Storage:

`az disk list`

- Tampilkan daftar seluruh instansi mesin virtual Azure:

`az vm list`

- Kelola layanan Azure Kubernetes Services:

`az aks`

- Kelola sumber daya jaringan Azure Network:

`az network`

- Jalankan program dalam mode interaktif:

`az interactive`

- Tampilkan bantuan:

`az --help`
