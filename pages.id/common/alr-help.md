# alr help

> Tampilkan informasi bantuan untuk Alire (`alr`).
> Informasi lebih lanjut: <https://alire.ada.dev/docs/#first-steps>.

- Tampilkan bantuan tingkat atas dan tampilkan daftar perintah yang tersedia:

`alr help`

- Tampilkan bantuan untuk suatu subperintah:

`alr help {{subperintah}}`

- Tampilkan aturan tata penamaan dan pemberian nomor versi:

`alr help identifiers`
