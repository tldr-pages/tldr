# edit

> Sebuah program pengolah teks berbasis terminal dari Microsoft.
> Informasi lebih lanjut: <https://github.com/microsoft/edit>.

- Buka suatu berkas:

`edit {{jalan/menuju/berkas}}`

- Cari teks dengan pola pencarian:

`<Ctrl f>{{pola}}<Enter>`

- Cari dan ganti teks dengan pola pencarian:

`<Ctrl r>{{pola}}<Enter>`

- Salin teks:

`<Ctrl c>`

- Tempelkan teks:

`<Ctrl v>`

- Buka menu Berkas (File):

`<Alt f>`

- Keluar:

`<Ctrl q>`
