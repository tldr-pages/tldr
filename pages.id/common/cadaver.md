# cadaver

> Klien WebDAV untuk Unix.
> Informasi lebih lanjut: <https://manned.org/cadaver>.

- Hubungkan ke peladen <dav.example.com>, kemudian buka direktori koleksi tingkat pangkal:

`cadaver {{http://dav.example.com/}}`

- Hubungkan menuju suatu peladen dengan port tertentu, kemudian buka isi suatu koleksi:

`cadaver {{http://dav.example.com:8022/jalan/menuju/koleksi/}}`

- Hubungkan menuju suatu peladen menggunakan SSL:

`cadaver {{https://davs.example.com/}}`
