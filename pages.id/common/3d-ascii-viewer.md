# 3d-ascii-viewer

> Lihat berkas model 3D `.obj` sebagai animasi ASCII dalam terminal.
> Informasi lebih lanjut: <https://github.com/autopawn/3d-ascii-viewer>.

- Tayangkan suatu model:

`3d-ascii-viewer {{jalan/menuju/model}}.obj`

- Atur lebar dan tinggi layar penayangan:

`3d-ascii-viewer -w {{lebar}} -h {{tinggi}} {{jalan/menuju/model}}.obj`

- Tampilkan animasi secara berwarna-warni:

`3d-ascii-viewer --color {{jalan/menuju/model}}.obj`

- Atur kecepatan animasi dalam FPS:

`3d-ascii-viewer -f {{fps}} {{jalan/menuju/model}}.obj`

- Jalankan program dalam mode interaktif:

`3d-ascii-viewer --interactive {{jalan/menuju/model}}.obj`

- Tampilkan bantuan:

`3d-ascii-viewer --help`
