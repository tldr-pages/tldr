# git help

> Tampilkan informasi bantuan tentang Git.
> Informasi lebih lanjut: <https://git-scm.com/docs/git-help>.

- Tampilkan bantuan tentang suatu subperintah Git:

`git help {{subperintah}}`

- Tampilkan bantuan tentang subperintah dalam peramban (browser) web:

`git help {{[-w|--web]}} {{subperintah}}`

- Tampilkan daftar seluruh subperintah Git yang tersedia:

`git help {{[-a|--all]}}`

- Tampilkan daftar panduan yang tersedia:

`git help {{[-g|--guides]}}`

- Tampilkan daftar seluruh variabel konfigurasi yang dapat diatur:

`git help {{[-c|--config]}}`
