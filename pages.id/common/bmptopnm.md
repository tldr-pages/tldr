# bmptopnm

> Ubah suatu berkas BMP menjadi berkas gambar PBM, PGM, atau PNM.
> Informasi lebih lanjut: <https://netpbm.sourceforge.net/doc/bmptopnm.html>.

- Buat berkas gambar PBM, PGM, atau PNM baru dari berkas masukan BMP versi Windows atau OS/2:

`bmptopnm {{path/to/file.bmp}}`

- Tampilkan isi data kepala berkas BMP menuju `stderr`:

`bmptopnm {{[-verb|-verbose]}} {{jalan/menuju/berkas.bmp}}`

- Tampilkan versi:

`bmptopnm {{[-v|-version]}}`
