# which

> Temukan program di jalur pengguna.
> Informasi lebih lanjut: <https://manned.org/which>.

- Cari jalur/path lingkungan variabel dan tampilkan lokasi setiap eksekusi yang cocok:

`which {{eksekusi}}`

- Jika ada beberapa executable yang cocok, tampilkan semuanya:

`which {{[-a|--all]}} {{eksekusi}}`
