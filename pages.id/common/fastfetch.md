# fastfetch

> Menampilkan informasi tentang sistem operasi, perangkat lunak, dan perangkat keras.
> Informasi lebih lanjut: <https://manned.org/fastfetch>.

- Tampilkan informasi sistem:

`fastfetch`

- Tampilkan informasi sistem lengkap dengan semua modul diaktifkan:

`fastfetch {{[-c|--config]}} all`

- Muat file konfigurasi khusus:

`fastfetch {{[-c|--config]}} {{path/to/config_file}}`

- Tampilkan struktur tertentu:

`fastfetch {{[-s|--structure]}} {{os:kernel:de:cpu:gpu}}`

- Tampilkan informasi sistem dalam format tertentu:

`fastfetch --format {{default|json}}`

- Gunakan logo tertentu untuk ditampilkan:

`fastfetch {{[-l|--logo]}} {{logo}}`

- Tampilkan informasi sistem tanpa logo:

`fastfetch {{[-l|--logo]}} none`

- Gunakan warna tertentu untuk kunci dan judul:

`fastfetch --color {{blue}}`
