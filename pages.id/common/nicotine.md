# nicotine

> Program klien grafis untuk jaringan peer-to-peer Soulseek.
> Informasi lebih lanjut: <https://manned.org/nicotine>.

- Nyalakan Nicotine+:

`nicotine`

- Nyalakan Nicotine+ dengan mode headless ([n]):

`nicotine {{[-n|--headless]}}`

- Pindai ulang berkas-berkas yang dibagikan:

`nicotine {{[-r|--rescan]}}`

- Tampilkan bantuan:

`nicotine {{[-h|--help]}}`
