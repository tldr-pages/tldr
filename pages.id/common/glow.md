# glow

> Render berkas Markdown dalam terminal.
> Informasi lebih lanjut: <https://manned.org/glow>.

- Jalankan glow dan pilih berkas untuk dilihat:

`glow`

- Render suatu berkas Markdown menuju terminal:

`glow {{jalan/menuju/berkas}}`

- Lihat suatu berkas Markdown dengan fitur manajemen halaman:

`glow {{[-p|--pager]}} {{jalan/menuju/berkas}}`

- Lihat suatu berkas dari suatu URL:

`glow {{https://example.com/file.md}}`

- Lihat berkas README suatu repositori GitHub/GitLab:

`glow {{github.com/pemilik/repositori}}`
