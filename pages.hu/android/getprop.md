# getprop

> Az Android rendszer tulajdonságairól szóló információk megjelenítése. További információ: <https://manned.org/getprop>.

- Az Android rendszer tulajdonságairól szóló információk megjelenítése:

`getprop`

- Információk megjelenítése egy adott tulajdonságról:

`getprop {{prop}}`

- Az SDK API szintjének megjelenítése:

`getprop {{ro.build.version.sdk}}`

- Az Android verziójának megjelenítése:

`getprop {{ro.build.version.release}}`

- Az Android eszközmodell megjelenítése:

`getprop {{ro.vendor.product.model}}`

- Az OEM feloldási állapotának megjelenítése:

`getprop {{ro.oem_unlock_supported}}`

- Az Android Wi-Fi kártyájának MAC-címének megjelenítése:

`getprop {{ro.boot.wifimacaddr}}`
