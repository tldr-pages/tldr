# pkg

> Csomagkezelő segédprogram Termuxhoz. További információ: <https://wiki.termux.com/wiki/Package_Management>.

- Frissítse az összes telepített csomagot:

`pkg upgrade`

- Telepítsen egy csomagot:

`pkg install {{package}}`

- Egy csomag eltávolítása:

`pkg uninstall {{package}}`

- Egy csomag újratelepítése:

`pkg reinstall {{package}}`

- Csomag keresése:

`pkg search {{package}}`
