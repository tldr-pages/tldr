# choco feature

> Interakció a Chocolatey funkcióival. További információ: <https://chocolatey.org/docs/commands-feature>.

- Az elérhető funkciók listájának megjelenítése:

`choco feature list`

- Engedélyezzen egy funkciót:

`choco feature enable --name {{name}}`

- Egy funkció letiltása:

`choco feature disable --name {{name}}`
