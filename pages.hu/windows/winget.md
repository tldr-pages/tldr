# winget

> Windows Package Manager CLI. További információ: <https://learn.microsoft.com/windows/package-manager/winget>.

- Csomag telepítése:

`winget install {{package}}`

- A csomagra vonatkozó információk megjelenítése:

`winget show {{package}}`

- Csomag keresése:

`winget search {{package}}`

- Az összes csomag frissítése a legújabb verzióra:

`winget upgrade --all`

- A winget segítségével kezelhető összes telepített csomag listázása:

`winget list --source winget`
