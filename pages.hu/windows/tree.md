# tree

> Egy útvonal könyvtárszerkezetének grafikus fájának megjelenítése. További információ: <https://learn.microsoft.com/windows-server/administration/windows-commands/tree>.

- Az aktuális könyvtár fájának megjelenítése:

`tree`

- Egy adott könyvtár fájának megjelenítése:

`tree {{path/to/directory}}`

- Egy könyvtár fájt tartalmazó könyvtár fájának megjelenítése:

`tree {{path/to/directory}} /f`

- A fa megjelenítése a kiterjesztett karakterek helyett ASCII karakterek használatával:

`tree {{path/to/directory}} /a`
