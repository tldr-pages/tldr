# reg restore

> Egy kulcs és értékeinek visszaállítása egy biztonsági mentési fájlból. További információkért lásd: `reg-save`. További információk: <https://learn.microsoft.com/windows-server/administration/windows-commands/reg-restore>.

- Egy megadott kulcs felülírása egy biztonsági mentési fájlból származó adatokkal:

`reg restore {{key_name}} {{path/to/file}}`
