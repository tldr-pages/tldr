# exit

> Az aktuális CMD-példány vagy az aktuális kötegelt fájl kilépése. További információ: <https://learn.microsoft.com/windows-server/administration/windows-commands/exit>.

- Az aktuális CMD-példány kilépése:

`exit`

- Az aktuális kötegelt parancsfájl kilépése:

`exit /b`

- Kilépés egy adott kilépési kóddal:

`exit {{exit_code}}`
