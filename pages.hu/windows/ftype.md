# ftype

> A fájlkiterjesztés társításához használt fájltípusok megjelenítése vagy módosítása. További információ: <https://learn.microsoft.com/windows-server/administration/windows-commands/ftype>.

- Az összes fájltípus listájának megjelenítése:

`ftype`

- Egy adott fájltípushoz társított program megjelenítése:

`ftype {{file_type}}`

- A társított program beállítása egy adott fájltípushoz:

`ftype {{file_type}}="{{path/to/executable_command}}"`
