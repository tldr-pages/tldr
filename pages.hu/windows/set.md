# set

> A CMD aktuális példányának környezeti változóinak megjelenítése vagy beállítása. További információ: <https://learn.microsoft.com/windows-server/administration/windows-commands/set>.

- Az összes aktuális környezeti változó listázása:

`set`

- Környezeti változó beállítása egy adott értékre:

`set {{name}}={{value}}`

- A megadott karakterlánccal kezdődő környezeti változók listázása:

`set {{name}}`

- A megadott változó értékének megadására való felszólítás:

`set /p {{name}}={{prompt_string}}`
