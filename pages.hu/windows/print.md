# print

> Szövegfájl nyomtatása egy nyomtatóra. További információ: <https://learn.microsoft.com/windows-server/administration/windows-commands/print>.

- Szövegfájl nyomtatása az alapértelmezett nyomtatóra:

`print {{path/to/file}}`

- Szövegfájl nyomtatása egy adott nyomtatóra:

`print /d:{{printer}} {{path/to/file}}`
