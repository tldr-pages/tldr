# scoop

> Parancssoros telepítőprogram Windowshoz. További információ: <https://scoop.sh>.

- Csomag telepítése:

`scoop install {{package}}`

- Csomag eltávolítása:

`scoop uninstall {{package}}`

- Az összes telepített csomag frissítése:

`scoop update --all`

- Telepített csomagok listázása:

`scoop list`

- Információk megjelenítése egy csomagról:

`scoop info {{package}}`

- Csomag keresése:

`scoop search {{package}}`

- Az összes csomag régi verziójának eltávolítása és a letöltési gyorsítótár törlése:

`scoop cleanup --cache --all`
