# reg load

> A mentett alkulcsok betöltése egy másik alkulcsba a rendszerleíró adatbázisban. Ez a funkció hibaelhárításra és ideiglenes kulcsokra szolgál. További információ: <https://learn.microsoft.com/windows-server/administration/windows-commands/reg-load>.

- Biztonsági mentés betöltése a megadott kulcsba:

`reg load {{key_name}} {{path/to/file}}`
