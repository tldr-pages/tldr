# nfsstat

> Az NFS-kiszolgálóhoz intézett hívások számának megjelenítése vagy visszaállítása. További információ: <https://learn.microsoft.com/windows-server/administration/windows-commands/nfsstat>.

- Az NFS-kiszolgálóhoz intézett hívások rögzített számának megjelenítése:

`nfsstat`

- Az NFS-kiszolgálóhoz intézett hívások rögzített számának visszaállítása:

`nfsstat -z`
