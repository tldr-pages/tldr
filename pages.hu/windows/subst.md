# subst

> Hozzárendel egy elérési utat egy virtuális meghajtó betűjéhez. További információ: <https://learn.microsoft.com/windows-server/administration/windows-commands/subst>.

- Aktív társítások listázása:

`subst`

- Hozzárendelés hozzáadása:

`subst {{Z:}} {{C:\Python2.7}}`

- Társítás eltávolítása:

`subst {{Z:}} /d`
