# pwlauncher

> Parancssori eszköz a Windows To Go indítási beállításainak kezelésére. További információ: <https://learn.microsoft.com/windows-server/administration/windows-commands/pwlauncher>.

- A Windows To Go aktuális állapotának megjelenítése:

`pwlauncher`

- A Windows To Go indítási beállításainak engedélyezése vagy letiltása:

`pwlauncher /{{enable|disable}}`
