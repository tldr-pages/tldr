# more

> Megnyit egy fájlt interaktív olvasásra, lehetővé téve a görgetést és a keresést. További információ: <https://manned.org/more>.

- Fájl megnyitása:

`more {{path/to/file}}`

- Egy adott sortól megjelenítő fájl megnyitása:

`more +{{line_number}} {{path/to/file}}`

- Súgó megjelenítése:

`more --help`

- Továbblépés a következő oldalra:

`<Space>`

- Egy karakterlánc keresése (a `n` megnyomásával a következő találatra léphet):

`/{{something}}`

- Kilépés:

`q`

- Súgó megjelenítése az interaktív parancsokról:

`h`
