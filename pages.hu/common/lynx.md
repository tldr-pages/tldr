# lynx

> Parancssori webböngésző. További információ: <https://lynx.browser.org>.

- Látogasson el egy weboldalra:

`lynx {{example.com}}`

- Korlátozások alkalmazása anonim fiókra:

`lynx -anonymous {{example.com}}`

- Kapcsolja be az egértámogatást, ha rendelkezésre áll:

`lynx -use_mouse {{example.com}}`

- Színes üzemmód bekapcsolásának kikényszerítése, ha rendelkezésre áll:

`lynx -color {{example.com}}`

- Nyisson meg egy linket, egy adott fájl használatával a sütik olvasásához és írásához:

`lynx -cookie_file={{path/to/file}} {{example.com}}`

- Navigáljon előre és hátrafelé az oldalon található linkek között:

`Up arrow key, Down arrow key`

- Visszalépés a korábban megjelenített oldalra:

`Left arrow key or u`

- Kilépés:

`q then y`
