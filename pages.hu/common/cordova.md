# cordova

> Mobilalkalmazások HTML, CSS és JS segítségével. További információ: <https://cordova.apache.org/docs/en/latest/guide/cli/>.

- Cordova projekt létrehozása:

`cordova create {{path}} {{package_name}} {{project_name}}`

- Az aktuális munkaterület állapotának megjelenítése:

`cordova info`

- Cordova platform hozzáadása:

`cordova platform add {{platform}}`

- Cordova platform eltávolítása:

`cordova platform remove {{platform}}`

- Cordova plugin hozzáadása:

`cordova plugin add {{pluginid}}`

- Cordova plugin eltávolítása:

`cordova plugin remove {{pluginid}}`
