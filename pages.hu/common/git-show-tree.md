# git show-tree

> Egy Git tároló összes ágát tartalmazó, díszített fa gráfjának megjelenítése, megjegyzésekkel. A `git-extras` része. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-show-tree>.

- Mutassa meg a címkékkel és ágnevekkel megjegyzésekkel ellátott összes ág díszített fa gráfját:

`git show-tree`
