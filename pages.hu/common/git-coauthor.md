# git coauthor

> Adjon hozzá egy másik szerzőt a legutóbbi commithoz. Mivel ez a parancs átírja a Git-előzményeket, a `--force` parancsra a legközelebbi tologatáskor lesz szükség. A `git-extras` részeként. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-coauthor>.

- További szerzőt illeszt be a legutóbbi Git commithoz:

`git coauthor {{name}} {{name@example.com}}`
