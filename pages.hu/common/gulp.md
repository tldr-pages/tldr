# gulp

> JavaScript task runner és streaming build rendszer. A feladatokat a gulpfile.js fájlban definiáljuk a projekt gyökerében. További információ: <https://github.com/gulpjs/gulp-cli>.

- Futtassa az alapértelmezett feladatot:

`gulp`

- Egyedi feladatok futtatása:

`gulp {{task}} {{othertask}}`
