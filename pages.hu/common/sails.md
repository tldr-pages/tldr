# sails

> A Sails.js egy valós idejű, vállalati szintű MVC keretrendszer, amely a Node.js tetejére épül. További információ: <https://sailsjs.com>.

- A Sails elindítása:

`sails lift`

- Új Sails projekt létrehozása:

`sails new {{projectName}}`

- Sails API létrehozása:

`sails generate {{name}}`

- Sails Controller generálása:

`sails generate controller {{name}}`

- Sails Model generálása:

`sails generate model {{name}}`
