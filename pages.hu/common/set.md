# set

> A shell attribútumok és pozíciós paraméterek értékeinek megjelenítése, beállítása vagy visszavonása. További információ: <https://manned.org/set>.

- A héjváltozók nevének és értékeinek megjelenítése:

`set`

- A módosított vagy exportálásra létrehozott változók jelölése:

`set -a`

- A munka azonnali befejezéséről való értesítés:

`set -b`

- Különböző opciók beállítása, pl. a `vi` stílusú sorszerkesztés engedélyezése:

`set -o {{vi}}`

- Beállíthatja, hogy a héj az első hiba esetén azonnal kilépjen (főleg szkriptekben használatos):

`set -e`
