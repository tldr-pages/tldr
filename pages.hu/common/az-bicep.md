# az bicep

> Bicep CLI-parancscsoport. A `azure-cli` weboldal része. További információ: <https://learn.microsoft.com/cli/azure/bicep>.

- Telepítse a Bicep CLI-t:

`az bicep install`

- Bicep fájl létrehozása:

`az bicep build --file {{path/to/file.bicep}}`

- Kísérlet egy ARM sablonfájl Bicep-fájlba történő dekompilálására:

`az bicep decompile --file {{path/to/template_file.json}}`

- A Bicep CLI frissítése a legújabb verzióra:

`az bicep upgrade`

- A Bicep CLI telepített verziójának megjelenítése:

`az bicep version`

- A Bicep CLI összes elérhető verziójának listája:

`az bicep list-versions`

- A Bicep CLI eltávolítása:

`az bicep uninstall`
