# antibody

> "A leggyorsabb" shell plugin manager. További információ: <https://getantibody.github.io>.

- Csomagolja össze az összes bővítményt statikus betöltéshez:

`antibody bundle < {{~/.zsh_plugins.txt}} > {{~/.zsh_plugins.sh}}`

- Frissítse az összes csomagot:

`antibody update`

- Listázza az összes telepített bővítményt:

`antibody list`
