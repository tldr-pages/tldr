# apktool

> APK fájlok visszafejtése. További információ: <https://ibotpeaches.github.io/Apktool/>.

- APK fájl dekódolása:

`apktool d {{file.apk}}`

- Egy APK fájl felépítése egy könyvtárból:

`apktool b {{path/to/directory}}`

- Egy keretrendszer telepítése és tárolása:

`apktool if {{framework.apk}}`
