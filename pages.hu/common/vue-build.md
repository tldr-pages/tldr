# vue build

> A `@vue/cli` és a `@vue/cli-service-global` által biztosított alparancs, amely gyors prototípuskészítést tesz lehetővé. További információ: <https://cli.vuejs.org/guide/prototyping.html>.

- A `.js` vagy a `.vue` fájl készítése termelési üzemmódban, nulla konfigurációval:

`vue build {{filename}}`
