# fdroidcl

> F-Droid CLI kliens. További információ: <https://github.com/mvdan/fdroidcl>.

- Az F-Droid index lekérése:

`fdroidcl update`

- Információk megjelenítése egy alkalmazásról:

`fdroidcl show {{app_id}}`

- APK fájl letöltése:

`fdroidcl download {{app_id}}`

- Egy alkalmazás keresése az indexben:

`fdroidcl search {{search_pattern}}`

- Alkalmazás telepítése egy csatlakoztatott eszközre:

`fdroidcl install {{app_id}}`
