# flask

> Egy általános segédszkript Flask alkalmazásokhoz. Betölti a `FLASK_APP` környezeti változóban meghatározott alkalmazást. További információ: <https://flask.palletsprojects.com/en/1.1.x/cli/>.

- Egy fejlesztői szerver futtatása:

`flask run`

- Megjeleníti az alkalmazás útvonalait:

`flask routes`

- Futtat egy Python interaktív héjat az alkalmazás kontextusában:

`flask shell`
