# couchdb

> Parancssori felület az Apache CouchDB adatbázis-szerverhez. További információ: <https://couchdb.apache.org>.

- Indítsa el a CouchDB-t:

`couchdb`

- A CouchDB interaktív shell indítása:

`couchdb -i`

- A CouchDB elindítása háttérfolyamatként:

`couchdb -b`

- A háttérfolyamat leállítása (Megjegyzés: szükség esetén újraindul):

`couchdb -k`

- A háttérfolyamat leállítása:

`couchdb -d`
