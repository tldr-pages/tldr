# micro

> A Micro egy modern és intuitív, terminál alapú szövegszerkesztő. A navigáláshoz és/vagy a szöveg kijelöléséhez használhatja a billentyűzetet, de az egeret is. További információ: <https://micro-editor.github.io>.

- Fájl megnyitása:

`micro {{path/to/file}}`

- Fájl mentése:

`Ctrl + S`

- A teljes sor kivágása:

`Ctrl + K`

- Mintakeresés a fájlban (a `Ctrl + N`/`Ctrl + P` gomb megnyomásával léphet a következő/előző találatra):

`Ctrl + F "{{pattern}}" <Enter>`

- Parancs végrehajtása:

`Ctrl + E {{command}} <Enter>`

- Helyettesítés végrehajtása a teljes fájlban:

`Ctrl + E replaceall "{{pattern}}" "{{replacement}}" <Enter>`

- Kilépés:

`Ctrl + Q`
