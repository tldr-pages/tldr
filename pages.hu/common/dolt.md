# dolt

> A Dolt egy SQL adatbázis, amelyet ugyanúgy elágazhatsz, klónozhatsz, elágazhatsz, egyesíthetsz, tolhatsz és húzhatsz, mint egy Git tárolót. Néhány alparancsnak, mint például a `dolt commit`, saját használati dokumentációja van. További információ: <https://github.com/dolthub/dolt>.

- Egy dolt alparancs végrehajtása:

`dolt {{subcommand}}`

- Az elérhető alparancsok listája:

`dolt help`
