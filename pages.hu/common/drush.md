# drush

> Parancssori héj és szkriptkészítő felület a Drupalhoz. További információ: <https://www.drush.org>.

- A "foo" modul engedélyezése:

`drush en {{foo}}`

- A "foo" modul eltávolítása:

`drush pmu {{foo}}`

- Törölje az összes gyorsítótárat:

`drush cr`

- CSS és JavaScript gyorsítótárak törlése:

`drush cc css-js`
