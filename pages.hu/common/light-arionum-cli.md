# light-arionum-cli

> Az Arionum kriptopénz PHP light pénztárcája. További információ: <https://github.com/arionum/lightWalletCLI>.

- Új nyilvános/magán kulcspár generálása:

`light-arionum-cli`

- Az aktuális cím egyenlegének megjelenítése:

`light-arionum-cli balance`

- A megadott cím egyenlegének megjelenítése:

`light-arionum-cli balance {{address}}`

- Tranzakció küldése opcionális üzenettel:

`light-arionum-cli send {{address}} {{value}} {{optional_message}}`

- Exportálja az aktuális pénztárca adatait:

`light-arionum-cli export`

- Az aktuális blokk információinak megjelenítése:

`light-arionum-cli block`

- Információk megjelenítése az aktuális cím tranzakcióiról:

`light-arionum-cli transactions`

- Információk megjelenítése egy adott tranzakcióról:

`light-arionum-cli transaction {{transaction_id}}`
