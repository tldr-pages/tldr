# wrangler

> Cloudflare Workers parancssori eszköz. További információ: <https://developers.cloudflare.com/workers/>.

- Kezdje el a projektet egy vázkonfigurációval:

`wrangler init {{project_name}}`

- Hitelesítés a Cloudflare segítségével:

`wrangler login`

- Indítson el egy helyi fejlesztői kiszolgálót:

`wrangler dev --host {{hostname}}`

- Nyilvánosságra hozza a munkásszkriptet:

`wrangler publish`

- Összesítse a termelési munkás naplóit:

`wrangler tail`
