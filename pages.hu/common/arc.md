# arc

> Arkanista: További információ: <https://secure.phabricator.com/book/phabricator/article/arcanist/>.

- Küldje el a módosításokat a Differentialnak felülvizsgálatra:

`arc diff`

- Függőben lévő felülvizsgálati információk megjelenítése:

`arc list`

- Git commit üzenetek frissítése felülvizsgálat után:

`arc amend`

- Push Git változások:

`arc land`
