# react-native start

> Parancssori eszközök a React Native szerver indításához. További információ: <https://github.com/react-native-community/cli/blob/master/docs/commands.md#start>.

- Indítsa el a csatlakoztatott eszközökkel kommunikáló kiszolgálót:

`react-native start`

- Indítsa el a metróbundlert tiszta gyorsítótárral:

`react-native start --reset-cache`

- Indítsa el a kiszolgálót egy egyéni porton (alapértelmezés szerint 8081):

`react-native start --port {{3000}}`

- Indítsa el a kiszolgálót verbózus üzemmódban:

`react-native start --verbose`

- Adja meg a fájlok átalakítására szolgáló munkások maximális számát (alapértelmezett a CPU-magok száma):

`react-native start --max-workers {{count}}`

- Az interaktív mód kikapcsolása:

`react-native start --no-interactive`
