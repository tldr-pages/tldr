# kiwi-ng

> A KIWI NG egy OS image és appliance builder. További információ: <https://osinside.github.io/kiwi/>.

- Állítson össze egy készüléket:

`kiwi-ng system build --description={{path/to/directory}} --target-dir={{path/to/directory}}`

- A felépített készülék építési eredményének megjelenítése:

`kiwi-ng result list --target-dir={{path/to/directory}}`

- Súgó megjelenítése:

`kiwi-ng help`

- Verzió megjelenítése:

`kiwi-ng -v`
