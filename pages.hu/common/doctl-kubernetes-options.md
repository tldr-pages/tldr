# doctl kubernetes options

> A doctl Kubernetes-parancsokkal használható értékeket biztosít. További információ: <https://docs.digitalocean.com/reference/doctl/reference/kubernetes/options/>.

- A Kubernetes fürtöket támogató régiók listája:

`doctl kubernetes options regions`

- A Kubernetes fürtökben használható gépméretek listája:

`doctl kubernetes options sizes`

- A DigitalOcean fürtökkel használható Kubernetes-verziók listája:

`doctl kubernetes options versions`
