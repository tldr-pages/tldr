# makebuildserver

> Hozzon létre egy F-Droid build szerver virtuális gépet. További információ: <https://f-droid.org/en/docs/Build_Server_Setup/>.

- Hozzon létre egy új virtuális gépet vagy frissítsen egy meglévőt (ha van):

`makebuildserver`

- Új virtuális gép létrehozásának kikényszerítése:

`makebuildserver --clean`
