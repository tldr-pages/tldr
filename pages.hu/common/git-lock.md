# git lock

> Zárol egy fájlt a Git-repositoryban, hogy ne lehessen módosítani egy commit által. A `git-extras` része. Lásd még: `git-unlock`. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-lock>.

- Letiltja a helyi fájl módosításainak commitolását:

`git lock {{path/to/file}}`
