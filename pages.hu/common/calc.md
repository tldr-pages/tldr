# calc

> Interaktív tetszőleges pontosságú számológép a terminálban. További információ: <https://github.com/lcn2/calc>.

- Indítsa el a `calc` oldalt interaktív módban:

`calc`

- Számítás elvégzése nem interaktív módban:

`calc '{{85 * (36 / 4)}}'`

- Számítás végrehajtása kimeneti formázás nélkül (pipákkal való használatra):

`calc -p '{{4/3 * pi() * 5^3}}'`

- Számítás elvégzése, majd váltás \[i\]nteraktív módba:

`calc -i '{{sqrt(2)}}'`

- A `calc` elindítása egy adott engedélyezési \[m\]módban (0-tól 7-ig, alapértelmezett érték 7):

`calc -m {{mode}}`

- A `calc` bevezetőjének megtekintése:

`calc help intro`

- A `calc` áttekintése:

`calc help overview`

- A `calc` kézikönyv megnyitása:

`calc help`
