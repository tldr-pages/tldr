# vue

> Többcélú CLI a Vue.js számára. Néhány alparancsnak, mint például a `vue build`, saját használati dokumentációja van. További információ: <https://cli.vuejs.org>.

- Új Vue projekt létrehozása interaktívan:

`vue create {{project_name}}`

- Új projekt létrehozása webes felhasználói felülettel:

`vue ui`
