# doctum

> PHP API dokumentáció generátor. További információ: <https://github.com/code-lts/doctum>.

- Projekt elemzése:

`doctum parse`

- Egy projekt renderelése:

`doctum render`

- Egy projekt elemzése, majd renderelése:

`doctum update`

- Egy projektnek csak egy adott verzióját elemzi és rendereli:

`doctum update --only-version={{version}}`

- Projekt elemzése és renderelése egy adott konfigurációval:

`doctum update {{path/to/config.php}}`
