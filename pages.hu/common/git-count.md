# git count

> A commitok teljes számának nyomtatása. A `git-extras` része. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-count>.

- A commitok teljes számának kiírása:

`git count`

- Kiírja a commitok számát közreműködőnként és a commitok teljes számát:

`git count --all`
