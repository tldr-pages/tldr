# gpg-tui

> Terminál felhasználói felület a GNU Public Guard-hoz. További információ: <https://github.com/orhun/gpg-tui>.

- Indítsa el a `gpg-tui`:

`gpg-tui`

- A `gpg-tui` elindítása színes és ASCII páncélozott kimenettel:

`gpg-tui --style {{colored}} --armor`

- Súgó megjelenítése a `gpg-tui`:

`?`

- Kilépés a `gpg-tui`:

`q`

- Interaktívan új kulcs generálása:

`g`

- A kiválasztott kulcs exportálása:

`x`

- A kiválasztott kulcs részletességi szintjének beállítása:

`1|2|3`

- A `gpg-tui` frissítése:

`r`
