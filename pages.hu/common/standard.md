# standard

> A JavaScript Standard Style eszköz a JavaScript kód szedésére és javítására. További információ: <https://standardjs.com>.

- Az aktuális könyvtárban lévő összes JavaScript forrásfájl futtatása:

`standard`

- Konkrét JavaScript-fájl(ok) foltozása:

`standard {{path/to/file1 path/to/file2 ...}}`

- Automatikus javítások alkalmazása a linting során:

`standard --fix`

- Minden rendelkezésre álló globális változó deklarálása:

`standard --global {{variable}}`

- Egyéni ESLint plugin használata a linting során:

`standard --plugin {{plugin}}`

- Egyéni JS-elemző használata a szűretés során:

`standard --parser {{parser}}`

- Egyéni ESLint környezet használata a futtatás során:

`standard --env {{environment}}`
