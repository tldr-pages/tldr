# npm-name

> Ellenőrizze, hogy egy csomag vagy szervezet neve elérhető-e az npm-en. További információ: <https://github.com/sindresorhus/npm-name-cli>.

- Annak ellenőrzése, hogy egy adott csomag neve elérhető-e az npm nyilvántartásában:

`npm-name {{package}}`

- Hasonló csomagnevek keresése az npm regiszterben:

`npm-name --similar {{package}}`
