# pyats shell

> Indítson el egy előre betöltött pyATS interaktív Python Shell-t, hogy időt takarítson meg a prototípusok készítésénél. További információ: <https://pubhub.devnetcloud.com/media/genie-docs/docs/cli/genie_shell.html>.

- Nyissa meg a pyATS shell-t egy meghatározott Testbed-fájllal:

`pyats shell --testbed-file {{path/to/testbed.yaml}}`

- Nyissa meg a pyATS héjat egy meghatározott Pickle-fájllal:

`pyats shell --pickle-file {{path/to/pickle.file}}`

- A pyATS megnyitása IPython letiltásával:

`pyats shell --no-ipython`
