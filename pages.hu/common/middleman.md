# middleman

> Ruby nyelven írt statikus oldalgenerátor. További információ: <https://middlemanapp.com/>.

- Hozzon létre egy új Middleman projektet:

`middleman init "{{project_name}}"`

- Az aktuális projekt helyi kiszolgálójának indítása a 4567-es porton:

`middleman server`

- Helyi kiszolgáló indítása az aktuális projekthez egy megadott porton:

`middleman server -p "{{port}}"`

- Építse a projektet az aktuális könyvtárban a telepítés előkészítéséhez:

`bundle exec middleman build`

- A Middleman projekt telepítése az aktuális könyvtárban:

`middleman deploy`
