# asdf

> Parancssori felület a különböző csomagok verzióinak kezelésére. További információ: <https://asdf-vm.com>.

- Az összes elérhető bővítmény listája:

`asdf plugin list all`

- Egy bővítmény telepítése:

`asdf plugin add {{name}}`

- Egy csomag összes elérhető verziójának listázása:

`asdf list all {{name}}`

- Egy csomag egy adott verziójának telepítése:

`asdf install {{name}} {{version}}`

- Egy csomag globális verziójának beállítása:

`asdf global {{name}} {{version}}`

- Helyi verzió beállítása egy csomaghoz:

`asdf local {{name}} {{version}}`
