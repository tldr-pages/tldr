# swipl

> SWI-Prolog - Átfogó ingyenes Prolog környezet. További információ: <https://www.swi-prolog.org/>.

- Interaktív munkamenet indítása:

`swipl`

- Egy parancs végrehajtása kimenet megjelenítése nélkül:

`swipl --quiet -t "{{command}}"`

- Szkript végrehajtása:

`swipl {{path/to/file.pl}}`

- A shell összes konfigurációs változójának kiírása:

`swipl --dump-runtime-variables`

- A verzió kinyomtatása:

`swipl --version`
