# transmission-show

> Információ lekérése egy torrent fájlról. Lásd még: `transmission`. További információ: <https://manned.org/transmission-show>.

- Egy adott torrent metaadatainak megjelenítése:

`transmission-show {{path/to/file.torrent}}`

- Mágneses link generálása egy adott torrenthez:

`transmission-show --magnet {{path/to/file.torrent}}`

- Egy torrent nyomkövetőinek lekérdezése és az aktuális peerek számának kiírása:

`transmission-show --scrape {{path/to/file.torrent}}`
