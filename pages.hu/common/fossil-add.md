# fossil add

> Fájlok vagy könyvtárak Fossil verziókezelés alá helyezése. További információ: <https://fossil-scm.org/home/help/add>.

- Fájl vagy könyvtár verziókezelés alá helyezése, így az az aktuális checkoutban lesz:

`fossil add {{path/to/file_or_directory}}`

- Távolítson el minden hozzáadott fájlt az aktuális ellenőrzésből:

`fossil add --reset`
