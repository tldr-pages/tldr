# act

> A GitHub Actions helyi végrehajtása Docker segítségével. További információ: <https://github.com/nektos/act>.

- Az elérhető műveletek listája:

`act -l`

- Az alapértelmezett esemény futtatása:

`act`

- Egy adott esemény futtatása:

`act {{event_type}}`

- Egy adott művelet futtatása:

`act -a {{action_id}}`

- Valójában ne futtassa az akciókat (azaz szárazon futtassa):

`act -n`

- Szöveges naplók megjelenítése:

`act -v`
