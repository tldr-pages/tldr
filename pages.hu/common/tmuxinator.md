# tmuxinator

> Könnyen létrehozhat és kezelhet tmux munkameneteket. További információ: <https://github.com/tmuxinator/tmuxinator>.

- Új projekt létrehozása:

`tmuxinator new {{project}}`

- Projekt szerkesztése:

`tmuxinator edit {{project}}`

- Projektek listázása:

`tmuxinator list`

- Tmux munkamenet indítása projekt alapján:

`tmuxinator start {{project}}`

- Egy projekt tmux munkamenetének leállítása:

`tmuxinator stop {{project}}`
