# nrm

> npm registry manager. Segít könnyen váltani a különböző npm registry-k között. További információ: <https://github.com/Pana/nrm>.

- Az összes regisztráció listázása:

`nrm ls`

- Váltás egy adott registryre:

`nrm use {{registry}}`

- Megjeleníti az összes regiszter válaszidejét:

`nrm test`

- Egyéni registry hozzáadása:

`nrm add {{registry}} {{url}}`

- Törölni egy registry-t:

`nrm del {{registry}}`
