# git gc

> A helyi tároló optimalizálása a felesleges fájlok megtisztításával. További információ: <https://git-scm.com/docs/git-gc>.

- Optimalizálja a tárolót:

`git gc`

- Agresszív optimalizálás, több időt vesz igénybe:

`git gc --aggressive`

- Ne nyesegesse a laza objektumokat (alapértelmezés szerint nyesegeti):

`git gc --no-prune`

- Minden kimenet elnyomása:

`git gc --quiet`

- Teljes használat megtekintése:

`git gc --help`
