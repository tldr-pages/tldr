# wpm

> Typeracer-szerű konzolos alkalmazás a WPM sebesség mérésére. További információ: <https://github.com/cslarsen/wpm>.

- Indítsa el a programot:

`wpm`

- Start wpm rövid szövegekkel:

`wpm --short`

- Indítsa el a wpm-et egy adott szövegfájl segítségével:

`wpm --load {{path/to/file.txt}}`

- Jelölje meg a versenyeredményeit:

`wpm --tag {{tag_name}}`

- A pontszám-statisztikák megjelenítése címkék szerint csoportosítva:

`wpm --stats`

- A wpm indítása monokróm színekkel:

`wpm --monochrome`
