# hcloud

> A Hetzner Cloud CLI használatának bemutatása. További információ: <https://github.com/hetznercloud/cli>.

- Az elérhető parancsok és zászlók megjelenítése:

`hcloud`

- A `hcloud` súgójának megjelenítése:

`hcloud -h`

- Elérhető parancsok és zászlók megjelenítése a `hcloud` kontextusokhoz:

`hcloud context`
