# xplr

> Terminál alapú fájlrendszer-kutató. További információ: <https://xplr.dev>.

- Nyisson meg egy könyvtárat:

`xplr {{path/to/directory}}`

- Fókuszáljon egy fájlra:

`xplr {{path/to/file}}`

- Fókuszáljon egy könyvtárra:

`xplr --force-focus {{path/to/directory}}`

- Könyvtár megnyitása meghatározott fájlok vagy könyvtárak kijelölésével:

`xplr {{path/to/directory}} {{path/to/selected_file_or_directory1}} {{path/to/selected_file_or_directory2}}`
