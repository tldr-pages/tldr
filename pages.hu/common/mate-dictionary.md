# mate-dictionary

> Szavak keresése szótárakban. További információ: <https://manned.org/mate-dictionary>.

- Egy adott szó meghatározásának kinyomtatása:

`mate-dictionary --no-window --look-up '{{word}}'`

- Egy adott szóhoz hasonló szavak megjelenítése egy új ablakban:

`mate-dictionary --match '{{word}}'`
