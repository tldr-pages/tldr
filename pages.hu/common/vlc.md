# vlc

> Multimédiás lejátszó, amely több platformon is használható. További információ: <https://wiki.videolan.org/Documentation:Command_line/>.

- Fájl lejátszása:

`vlc {{path/to/file}}`

- Lejátszás teljes képernyőn:

`vlc --fullscreen {{path/to/file}}`

- Lejátszás némítva:

`vlc --no-audio {{path/to/file}}`

- Ismételt lejátszás:

`vlc --loop {{path/to/file}}`

- Videó lejátszása URL-címről:

`vlc {{https://www.youtube.com/watch?v=oHg5SJYRHA0}}`
