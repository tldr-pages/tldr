# ninja

> Gyorsnak tervezett Build rendszer. További információ: <https://ninja-build.org/manual.html>.

- Építés az aktuális könyvtárban:

`ninja`

- Program építése egy adott könyvtárban:

`ninja -C {{path/to/directory}}`

- Célok megjelenítése (pl. `install` és `uninstall`):

`ninja -t targets`

- Súgó megjelenítése:

`ninja -h`
