# valac

> Vala kódfordító. Oktatóprogram: [https://wiki.gnome.org/Projects/Vala/Tutorial. További](https://wiki.gnome.org/Projects/Vala/Tutorial) információ: <https://valadoc.org/>.

- Fordítson le egy vala fájlt, gtk+ segítségével:

`valac {{path/to/file.vala}} --pkg {{gtk+-3.0}}`

- Verzióinformáció megjelenítése:

`valac --version`

- Segédüzenet megjelenítése:

`valac --help`
