# git setup

> Git-tár létrehozása egy adott könyvtárban és az összes fájl átadása. A `git-extras` része. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-setup>.

- Git-tár létrehozása az aktuális könyvtárban és az összes fájl commitolása:

`git setup`

- Git tároló létrehozása egy adott könyvtárban és az összes fájl commitolása:

`git setup {{path/to/directory}}`
