# openssl

> OpenSSL kriptográfiai eszközkészlet. Néhány alparancsnak, mint például a `openssl req`, saját használati dokumentációja van. További információ: <https://www.openssl.org>.

- Az elérhető alparancsok listájának kinyomtatása:

`openssl help`

- Opciók nyomtatása egy adott alparancshoz:

`openssl help {{x509}}`

- Az OpenSSL verziójának kiírása:

`openssl version`
