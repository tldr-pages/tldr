# Vite

> Vite projekt létrehozása. JavaScript projektek készítésére szolgál. Elérhető sablonok: vanilla, vanilla-ts, vue, vue-ts, react, react-ts, react-swc, react-swc-ts, preact, preact-ts, lit, lit-ts, svelte, svelte-ts. További információk: [https://vitejs.dev/guide.](https://vitejs.dev/guide)

- Beállítás a `npm` 6.x használatával:

`npm create vite@latest my-react-app --template react-ts`

- Beállítás a `npm` 7+ használatával, extra dupla kötőjelre van szükség:

`npm create vite@latest my-react-app -- --template react-ts`

- Beállítás a `yarn` használatával:

`yarn create vite my-react-app --template react-ts`

- Beállítás a `pnpm` használatával:

`pnpm create vite my-react-app --template react-ts`
