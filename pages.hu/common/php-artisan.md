# php artisan

> A Laravel Artisan parancssori felülete. További információ: <https://laravel.com/docs/artisan>.

- Indítsa el a PHP beépített webkiszolgálóját az aktuális Laravel alkalmazáshoz:

`php artisan serve`

- Indítson el egy interaktív PHP parancssori felületet:

`php artisan tinker`

- Generáljon egy új Eloquent modellosztályt migrációval, gyárral és erőforrás-vezérlővel:

`php artisan make:model {{ModelName}} --all`

- Az összes elérhető parancs listájának megjelenítése:

`php artisan help`
