# paci

> Bash szkriptek csomagkezelője. További információ: <https://github.com/tradebyte/paci>.

- Frissíti az elérhető csomagok és verziók listáját (ajánlott a többi `paci` parancs előtt futtatni):

`paci refresh`

- A viselkedésének beállítása:

`paci configure`

- Egy adott csomag keresése:

`paci search {{package}}`

- Egy csomag telepítése:

`paci install {{package}}`

- Egy csomag frissítése:

`paci update {{package}}`
