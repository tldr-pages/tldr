# swift

> Swift projektek létrehozása, futtatása és építése. További információ: <https://swift.org>.

- Indítson el egy REPL-t (interaktív héj):

`swift`

- Egy program végrehajtása:

`swift {{file.swift}}`

- Új projekt indítása a csomagkezelővel:

`swift package init`

- Xcode projektfájl generálása:

`swift package generate-xcodeproj`

- Függőségek frissítése:

`swift package update`

- Projekt fordítása kiadáshoz:

`swift build -c release`
