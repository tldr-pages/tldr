# fin

> Docksal parancssori segédprogram. További információ: <https://docs.docksal.io/fin/fin/>.

- Indítsa el a projektet az aktuális könyvtárban:

`fin project start`

- A projekt leállítása az aktuális könyvtárban:

`fin project stop`

- Nyisson meg egy héjat egy adott konténerbe:

`fin bash {{container_name}}`

- Egy adott konténer naplóinak megjelenítése:

`fin logs {{container_name}}`

- Egy adott konténer naplóinak megjelenítése és a napló követése:

`fin logs -f {{container_name}}`
