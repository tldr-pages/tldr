# lspath

> CLI alkalmazás a PATH környezeti változó tartalmának listázására, opcionális lapozással. További információ: <https://github.com/clementi/lspath>.

- A rendszer PATH változójának tartalmának kiírása, soronként egy elemmel:

`lspath`

- A rendszer PATH változó aktuális tartalmának kiírása, soronként egy elemmel, a kimenet átlapozásával:

`lspath --page`
