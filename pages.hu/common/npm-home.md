# npm-home

> Nyissa meg egy csomag npm-oldalát, Yarn-oldalát vagy GitHub-tárát a webböngészőben. További információ: <https://github.com/sindresorhus/npm-home>.

- Egy adott csomag npm oldalának megnyitása a webböngészőben:

`npm-home {{package}}`

- Egy adott csomag GitHub-tárának megnyitása a webböngészőben:

`npm-home -g {{package}}`

- Egy adott csomag Yarn-oldalának megnyitása a webböngészőben:

`npm-home -y {{package}}`
