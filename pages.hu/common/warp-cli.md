# warp-cli

> A Cloudflare WARP szolgáltatásának hivatalos parancssori kliense. További információ: <https://developers.cloudflare.com/warp-client/>.

- Regisztrálja az aktuális eszközt a WARP-hez (az első kapcsolat előtt kell futtatni):

`warp-cli register`

- Csatlakozás a WARP-hoz:

`warp-cli connect`

- A WARP-hez való kapcsolódás megszakítása:

`warp-cli disconnect`

- A WARP-kapcsolat állapotának megjelenítése:

`warp-cli status`

- Súgó megjelenítése:

`warp-cli help`

- Súgó megjelenítése egy alparancshoz:

`warp-cli help {{subcommand}}`
