# grunt

> JavaScript task runner folyamatok automatizálására. További információ: <https://github.com/gruntjs/grunt-cli>.

- Az alapértelmezett feladatfolyamat futtatása:

`grunt`

- Egy vagy több konkrét, szóközzel elválasztott feladat(ok) futtatása:

`grunt {{task_name}}`

- Alternatív konfigurációs fájl megadása:

`grunt --gruntfile {{path/to/file}}`

- Alternatív alap elérési út megadása a relatív fájlok számára:

`grunt --base {{path/to/directory}}`

- Egy további könyvtár megadása a feladatok kereséséhez:

`grunt --tasks {{path/to/directory}}`

- Száraz futtatás végrehajtása fájlok írása nélkül:

`grunt --no-write`

- Az összes rendelkezésre álló opció felsorolása:

`grunt --help`
