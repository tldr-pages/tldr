# dvc init

> Új helyi DVC-tár inicializálása. További információ: <https://dvc.org/doc/command-reference/init>.

- Új helyi tárolóhely inicializálása:

`dvc init`

- DVC inicializálása Git nélkül:

`dvc init --no-scm`

- DVC inicializálása egy alkönyvtárban:

`cd {{path/to/subdir}} && dvc init --sudir`
