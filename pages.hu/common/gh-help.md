# gh help

> A GitHub CLI-paranccsal kapcsolatos súgó megjelenítése. További információ: <https://cli.github.com/manual/gh_help>.

- Általános súgó megjelenítése:

`gh help`

- A `gh help` alparancshoz tartozó súgó megjelenítése:

`gh help --help`

- Súgó megjelenítése a `gh` parancsszóval használható környezeti változókról:

`gh help environment`

- Az összes `gh` parancs markdown hivatkozásának megjelenítése:

`gh help reference`

- Súgó megjelenítése a `gh` JSON kimenetének formázásáról a `jq` használatával:

`gh help formatting`

- Súgó megjelenítése a `gh` használatáról a MinTTY-vel:

`gh help mintty`

- Segítség megjelenítése egy alparancshoz:

`gh help {{subcommand}}`

- Súgó megjelenítése egy alparancs műveletéhez:

`gh help {{pr}} {{create}}`
