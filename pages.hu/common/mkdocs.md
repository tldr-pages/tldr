# mkdocs

> Projektdokumentáció Markdown segítségével. További információ: <https://www.mkdocs.org>.

- Hozzon létre egy új mkdocs projektet:

`mkdocs new {{project_name}}`

- Kiszolgálja a projektet az aktuális könyvtárban az mkdocs dev-server segítségével:

`mkdocs serve`

- Készítse el a dokumentációt az aktuális könyvtárban:

`mkdocs build`

- Az aktuális könyvtárban lévő dokumentáció telepítése a GitHub oldalakra:

`mkdocs gh-deploy`
