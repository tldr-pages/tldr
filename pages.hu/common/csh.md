# csh

> A shell (parancsértelmező) C-szerű szintaxissal. Lásd még: `tcsh`. További információ: <https://www.mkssoftware.com/docs/man1/csh.1.asp>.

- Interaktív shell munkamenet indítása:

`csh`

- Interaktív shell munkamenet indítása az indítási konfigurációk betöltése nélkül:

`csh -f`

- Konkrét \[c\]ommandók végrehajtása:

`csh -c "{{echo 'csh is executed'}}"`

- Egy adott szkript végrehajtása:

`csh {{path/to/script.csh}}`
