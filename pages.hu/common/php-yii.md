# php yii

> A Yii Framework parancssori felülete. További információ: <https://yiiframework.com>.

- Az összes elérhető parancs listájának megjelenítése:

`php yii {{help}}`

- A PHP beépített webkiszolgálójának elindítása az aktuális Yii alkalmazáshoz:

`php yii {{serve}}`

- A megadott modellosztályon végzett CRUD-műveletekhez vezérlő, nézetek és kapcsolódó fájlok generálása:

`php yii {{gii/crud}} --modelClass={{ModelName}} --controllerClass={{ControllerName}}`
