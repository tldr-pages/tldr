# git mr

> A GitLab egyesítési kérések helyi ellenőrzése. A `git-extras` része. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-mr>.

- Egy adott egyesítési kérelem megtekintése:

`git mr {{mr_number}}`

- Egy adott távoli egyesítési kérelem ellenőrzése:

`git mr {{mr_number}} {{remote}}`

- Egy egyesítési kérelem ellenőrzése az URL-címéről:

`git mr {{url}}`

- Régi egyesítési kérelem ágak kitakarítása:

`git mr clean`
