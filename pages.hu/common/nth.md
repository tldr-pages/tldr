# nth

> Name That Hash - Azonnal megnevezheti bármely hash típusát. További információ: <https://github.com/hashpals/name-that-hash>.

- Nevezzen meg egy hash-t:

`nth -t {{5f4dcc3b5aa765d61d8327deb882cf99}}`

- Nevezze meg a hash-eket egy fájlban:

`nth -f {{path/to/hashes}}`

- Kimenet json formátumban:

`nth -t {{5f4dcc3b5aa765d61d8327deb882cf99}} -g`

- Hash dekódolása Base64-ben, mielőtt elnevezné:

`nth -t {{NWY0ZGNjM2I1YWE3NjVkNjFkODMyN2RlYjg4MmNmOTkK}} -b64`
