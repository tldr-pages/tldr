# nimble

> Csomagkezelő a Nim programozási nyelvhez. Nim projektek és függőségeik kezelése. További információ: <https://github.com/nim-lang/nimble>.

- Csomagok keresése:

`nimble search {{search_string}}`

- Csomagok telepítése:

`nimble install {{package_name}}`

- Telepített csomagok listázása:

`nimble list -i`

- Új Nimble csomag létrehozása az aktuális könyvtárban:

`nimble init`

- Nimble csomag összeállítása:

`nimble build`

- Nimble csomag telepítése:

`nimble install`
