# rfetch

> Konfigurálható program a rendszerinformációk kimenetére. További információ: <https://github.com/kamui-fin/rfetch>.

- Rendszerinformációk megjelenítése:

`rfetch`

- Rendszer \[a\]rchitektúra megjelenítése:

`rfetch -a`

- A rendszer üzemkész\[t\]ime megjelenítése:

`rfetch -t`

- Rendszer \[k\]ernel:

`rfetch -k`

- Rendszer \[c\]PU:

`rfetch -c`

- Linux \[D\]istro:

`rfetch -D`

- Megtekintés \[d\]esktop környezet:

`rfetch -d`
