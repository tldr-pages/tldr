# phpdox

> PHP dokumentáció generátor. További információ: <https://phpdox.net>.

- Megjelenít egy megjegyzésekkel ellátott vázkonfigurációs XML fájlt:

`phpdox --skel`

- Dokumentáció generálása az aktuális munkakönyvtárhoz:

`phpdox`

- Dokumentáció generálása egy adott konfigurációs fájl felhasználásával:

`phpdox --file {{path/to/phpdox.xml}}`

- Csak a metaadatgyűjtési folyamat futtatása:

`phpdox --collector`

- Csak a dokumentációgeneráló folyamat futtatása:

`phpdox --generator`
