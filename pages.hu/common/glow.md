# glow

> Markdown megjelenítése a terminálban. További információ: <https://github.com/charmbracelet/glow>.

- Futtassa a glow-t, és válasszon ki egy fájlt a megtekintéshez:

`glow`

- Egy Markdown-fájl megjelenítése a terminálon:

`glow {{path/to/file}}`

- Markdown fájl megtekintése egy paginátor segítségével:

`glow -p {{path/to/file}}`

- Egy fájl megtekintése egy URL-címről:

`glow {{https://example.com/file.md}}`

- Egy GitHub/GitLab README megtekintése:

`glow {{github.com/owner/repository}}`
