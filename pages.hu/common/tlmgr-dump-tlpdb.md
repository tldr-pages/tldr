# tlmgr dump-tlpdb

> A TeX Live csomagadatbázis ürítése. További információ: <https://www.tug.org/texlive/tlmgr.html>.

- A helyi csomagadatbázis ürítése:

`tlmgr dump-tlpdb --local`

- A távoli csomagadatbázis ürítése:

`tlmgr dump-tlpdb --remote`

- A helyi csomagadatbázis kiürítése JSON formájában:

`tlmgr dump-tlpdb --local --json`
