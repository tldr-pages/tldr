# git cola

> Nagy teljesítményű Git GUI, elegáns és intuitív felhasználói felülettel. További információ: <https://git-cola.readthedocs.io>.

- Indítsa el a git cola-t:

`git cola`

- A git cola elindítása amend módban:

`git cola --amend`

- Git-tárhely kérése. Alapértelmezés szerint az aktuális könyvtár:

`git cola --prompt`

- Megnyitja a Git-tárat az említett elérési útvonalon:

`git cola --repo {{path/to/git-repository}}`

- Alkalmazza az elérési útvonal-szűrőt az állapot widgetre:

`git cola --status-filter {{filter}}`
