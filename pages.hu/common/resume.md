# resume

> CLI eszköz új önéletrajzok egyszerű beállításához. További információ: <https://github.com/jsonresume/resume-cli>.

- Hozzon létre egy új `resume.json` fájlt az aktuális munkakönyvtárban:

`resume init`

- A `resume.json` érvényesítését a sématesztek alapján, hogy megbizonyosodjon arról, hogy megfelel a szabványnak:

`resume validate`

- Az önéletrajz helyi exportálása stilizált HTML vagy PDF formátumban:

`resume export {{path/to/html_or_pdf}}`

- A helyi `resume.json` webkiszolgáló elindítása:

`resume serve`
