# nodenv

> A Node.js verziók kezelésére szolgáló eszköz. További információ: <https://github.com/nodenv/nodenv>.

- A Node.js egy adott verziójának telepítése:

`nodenv install {{version}}`

- Az elérhető verziók listájának megjelenítése:

`nodenv install --list`

- A Node.js egy adott verziójának használata az egész rendszerben:

`nodenv global {{version}}`

- A Node.js egy adott verziójának használata egy könyvtárral:

`nodenv local {{version}}`

- A Node.js verziójának megjelenítése az aktuális könyvtárhoz:

`nodenv version`

- A telepített Node.js parancs helyének megjelenítése (pl. `npm`):

`nodenv which {{command}}`
