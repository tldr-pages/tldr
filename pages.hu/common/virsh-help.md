# virsh-help

> A `virsh` parancsokkal vagy parancscsoportokkal kapcsolatos információk megjelenítése. Lásd még: `virsh`. További információk: <https://manned.org/virsh>.

- A `virsh` parancsok felsorolása kapcsolódó kategóriákba csoportosítva:

`virsh help`

- A parancskategóriák felsorolása:

`virsh help | grep "keyword"`

- A kategóriába tartozó parancsok felsorolása:

`virsh help {{category_keyword}}`

- Súgó megjelenítése egy parancshoz:

`virsh help {{command}}`
