# apm

> Atom editor Package Manager. Lásd `atom`. További információ: <https://github.com/atom/apm>.

- Telepítsen egy csomagot a http://atom.io/packages vagy egy témát a http://atom.io/themes oldalról:

`apm install {{package_name}}`

- Egy csomag/téma eltávolítása:

`apm remove {{package_name}}`

- Egy csomag/téma frissítése:

`apm upgrade {{package_name}}`
