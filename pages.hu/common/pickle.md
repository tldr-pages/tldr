# pickle

> A Composerre épülő PHP-bővítmény telepítő. További információ: <https://github.com/FriendsOfPHP/pickle>.

- Egy adott PHP-bővítmény telepítése:

`pickle install {{extension_name}}`

- Egy meglévő PECL-bővítmény konfigurációjának átalakítása Pickle konfigurációs fájlba:

`pickle convert {{path/to/directory}}`

- Egy PECL-bővítmény hitelesítése:

`pickle validate {{path/to/directory}}`

- PECL-bővítmény csomagolása kiadásra:

`pickle release {{path/to/directory}}`
