# shift

> Bash beépített parancs, amely a hívó függvénynek vagy szkriptnek átadott argumentumokat meghatározott számú helyzettel eltolja. További információ: <https://www.gnu.org/software/bash/manual/bash.html#index-shift>.

- Az argumentumok egy hellyel történő áthelyezése az első argumentumot elhagyva:

`shift`

- Move arguments by N places dropping the first N arguments:

`shift {{N}}`
