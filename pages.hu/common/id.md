# id

> Az aktuális felhasználói és csoportidentitás megjelenítése. További információ: <https://www.gnu.org/software/coreutils/id>.

- Az aktuális felhasználó azonosítójának (UID), csoport azonosítójának (GID) és a csoportoknak a megjelenítése, amelyekhez tartozik:

`id`

- Az aktuális felhasználói azonosító számként történő megjelenítése:

`id -u`

- Az aktuális csoport azonosítójának megjelenítése számként:

`id -g`

- Egy tetszőleges felhasználó azonosítójának (UID), csoport azonosítójának (GID) és a hozzá tartozó csoportoknak a megjelenítése:

`id {{username}}`
