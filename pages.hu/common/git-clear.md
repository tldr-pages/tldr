# git clear

> A Git munkakönyvtár törlése úgy, mintha frissen klónozták volna az aktuális ággal, beleértve a `.gitignore` fájlokat is. A `git-extras` része. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-clear>.

- Az összes követett fájl visszaállítása és az összes nem követett fájl törlése, még akkor is, ha azok a `.gitignore`:

`git clear`
