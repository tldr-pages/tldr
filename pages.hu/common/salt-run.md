# salt-run

> Frontend salt-runnerek végrehajtásához minionokon. További információ: <https://docs.saltstack.com/ref/cli/salt-run.html>.

- Az összes minion állapotának megjelenítése:

`salt-run manage.status`

- Az összes olyan minion megjelenítése, amelyik nincs összekapcsolva:

`salt-run manage.up`
