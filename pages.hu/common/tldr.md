# tldr

> A tldr-pages projekt parancssori eszközeinek egyszerű súgóoldalainak megjelenítése. További információ: <https://tldr.sh>.

- Egy adott parancs tldr-oldalának kinyomtatása (tipp: így jutottál ide!):

`tldr {{command}}`

- Egy adott alparancs tldr-oldalának kinyomtatása:

`tldr {{command}}-{{subcommand}}`

- Egy adott \[p\]latform parancs tldr-oldalának nyomtatása:

`tldr -p {{android|linux|osx|sunos|windows}} {{command}}`

- \[u\]pdate the local cache of tldr pages:

`tldr -u`
