# nice

> Egy program végrehajtása egyéni ütemezési prioritással (niceness). A niceness értékek -20-tól (a legmagasabb prioritás) 19-ig (a legalacsonyabb) terjednek. További információ: <https://www.gnu.org/software/coreutils/nice>.

- Egy program indítása módosított prioritással:

`nice -n {{niceness_value}} {{command}}`
