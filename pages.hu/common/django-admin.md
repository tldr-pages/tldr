# django-admin

> Django segédprogramja az adminisztrációs feladatokhoz. További információ: <https://docs.djangoproject.com/en/4.1/ref/django-admin/>.

- Hozzon létre egy új Django projektet:

`django-admin startproject {{project_name}}`

- Új alkalmazás létrehozása az aktuális projekthez:

`django-admin startapp {{app_name}}`

- Ellenőrizze a Django aktuális verzióját:

`django-admin --version`

- További információk megjelenítése az adott parancshoz:

`django-admin help {{command}}`
