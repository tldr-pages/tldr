# roave-backward-compatibility-check

> Egy eszköz, amely egy PHP könyvtár két verziója közötti visszafelé kompatibilitási törések ellenőrzésére használható. További információ: <https://github.com/Roave/BackwardCompatibilityCheck>.

- A legutóbbi tag óta történt törésváltozások ellenőrzése:

`roave-backward-compatibility-check`

- Egy adott tag óta történt törésváltozások ellenőrzése:

`roave-backward-compatibility-check --from={{git_reference}}`

- Az utolsó tag és egy adott hivatkozás közötti törésváltozások ellenőrzése:

`roave-backward-compatibility-check --to={{git_reference}}`

- A törésváltozások ellenőrzése és Markdownba történő kimenet:

`roave-backward-compatibility-check --format=markdown > {{results.md}}`
