# pyATS

> A Cisco Systems gyártófüggetlen teszt-automatizálási keretrendszere, amelyet hálózati és rendszertesztelésre használnak. További információ: <https://developer.cisco.com/pyats/>.

- Futtasson egy `pyATS` alparancsot:

`pyats {{subcommand}}`

- Verzió megjelenítése:

`pyats version check`

- Súgó megjelenítése:

`pyats --help`

- Egy adott alparanccsal kapcsolatos súgó megjelenítése:

`pyats {{subcommand}} --help`
