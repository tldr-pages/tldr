# cd

> Az aktuális munkakönyvtár megváltoztatása. További információ: <https://manned.org/cd>.

- A megadott könyvtárba való átlépés:

`cd {{path/to/directory}}`

- Az aktuális könyvtár szülő könyvtárába való átlépés:

`cd ..`

- Menj az aktuális felhasználó kezdőkönyvtárába:

`cd`

- A megadott felhasználó otthoni könyvtárába való átlépés:

`cd ~{{username}}`

- Menet a korábban kiválasztott könyvtárba:

`cd -`

- Menj a gyökérkönyvtárba:

`cd /`
