# git add

> Hozzáadja a módosított fájlokat az indexhez. További információ: <https://git-scm.com/docs/git-add>.

- Fájl hozzáadása az indexhez:

`git add {{path/to/file}}`

- Az összes fájl hozzáadása (nyomon követett és nem nyomon követett):

`git add -A`

- Csak a már nyomon követett fájlok hozzáadása:

`git add -u`

- Figyelmen kívül hagyott fájlok hozzáadása:

`git add -f`

- A fájlok interaktív szakaszolása:

`git add -p`

- Egy adott fájl egyes részeinek interaktív beállítása:

`git add -p {{path/to/file}}`

- Egy fájl interaktív szakaszolása:

`git add -i`
