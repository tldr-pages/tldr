# ptpython

> Egy jobb Python REPL. További információ: <https://github.com/prompt-toolkit/ptpython>.

- REPL (interaktív shell) indítása:

`ptpython`

- Egy adott Python fájl végrehajtása:

`ptpython {{path/to/file.py}}`

- Egy adott Python fájl végrehajtása és egy REPL indítása:

`ptpython -i {{path/to/file.py}}`

- Nyissa meg a menüt:

`F2`

- Az előzmények oldal megnyitása:

`F3`

- Beillesztési mód váltása:

`F6`

- Kilépés:

`Ctrl + D`
