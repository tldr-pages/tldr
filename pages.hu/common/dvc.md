# dvc

> Data Version Control: mint a `git` az adatokhoz. Néhány alparancsnak, például a `dvc commit` -nek saját használati dokumentációja van. További információ: <https://dvc.org/>.

- Ellenőrizze a DVC verzióját:

`dvc --version`

- Általános súgó megjelenítése:

`dvc --help`

- Egy adott alparanccsal kapcsolatos súgó megjelenítése:

`dvc {{subcommand}} --help`

- DVC alparancs végrehajtása:

`dvc {{subcommand}}`
