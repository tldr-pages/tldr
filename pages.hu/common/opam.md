# opam

> OCaml Package Manager. OCaml fordítóprogramok, eszközök és könyvtárak kezelése. További információ: <https://opam.ocaml.org/>.

- Az opam inicializálása az első használathoz:

`opam init`

- Csomagok keresése:

`opam search {{package_name}}`

- Telepítsen egy csomagot és annak összes függőségét:

`opam install {{package_name}}`

- Részletes információk megjelenítése egy csomagról:

`opam show {{package_name}}`

- Az összes telepített csomag listázása:

`opam list`

- A helyi csomagadatbázis frissítése:

`opam update`

- Az összes telepített csomag frissítése:

`opam upgrade`

- Az összes parancs megjelenítése:

`opam help`
