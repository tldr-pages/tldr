# hunspell

> A helyesírás ellenőrzése. További információ: <https://github.com/hunspell/hunspell>.

- Egy fájl helyesírásának ellenőrzése:

`hunspell {{path/to/file}}`

- Egy fájl helyesírásának ellenőrzése az en_US szótárral:

`hunspell -d {{en_US}} {{path/to/file}}`

- Listázza a hibásan írt szavakat egy fájlban:

`hunspell -l {{path/to/file}}`
