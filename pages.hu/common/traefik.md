# traefik

> HTTP reverse proxy és terheléskiegyenlítő. További információ: <https://traefik.io>.

- Kiszolgáló indítása alapértelmezett konfigurációval:

`traefik`

- Kiszolgáló indítása egyéni konfigurációs fájllal:

`traefik --c {{config_file}}.toml`

- Kiszolgáló indítása fürt mód engedélyezésével:

`traefik --cluster`

- Kiszolgáló indítása engedélyezett webes felhasználói felülettel:

`traefik --web`
