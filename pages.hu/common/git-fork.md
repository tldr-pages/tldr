# git fork

> A megadott GitHub repo elágazása. Mint a `git clone`, de előbb forkolja. A `git-extras` része. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-fork>.

- Egy GitHub-repozitórium elágazása és klónozása az URL címe alapján:

`git fork {{https://github.com/tldr-pages/tldr}}`

- Fork és klónozás egy GitHub-repositoryból a slug alapján:

`git fork {{tldr-pages/tldr}}`
