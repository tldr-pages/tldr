# dolt add

> Egy táblázat tartalmának hozzáadása a Dolt staged táblák listájához. További információ: <https://github.com/dolthub/dolt>.

- Táblázat hozzáadása a színpadra állított táblázatok listájához (stage a table):

`dolt add {{table}}`

- Az összes táblázat színpadra állítása:

`dolt add --all`
