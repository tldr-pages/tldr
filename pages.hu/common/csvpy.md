# csvpy

> CSV fájl betöltése egy Python héjba. Csvkit része. További információ: <https://csvkit.readthedocs.io/en/latest/scripts/csvpy.html>.

- CSV fájl betöltése egy `CSVKitReader` objektumba:

`csvpy {{data.csv}}`

- Egy CSV fájl betöltése egy `CSVKitDictReader` objektumba:

`csvpy --dict {{data.csv}}`
