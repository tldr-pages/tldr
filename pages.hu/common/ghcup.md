# ghcup

> Haskell toolchain telepítő. Haskell toolchains telepítése, kezelése és frissítése. További információ: <https://gitlab.haskell.org/haskell/ghcup-hs>.

- Az interaktív TUI elindítása:

`ghcup tui`

- Az elérhető GHC/cabal verziók listázása:

`ghcup list`

- Telepítse az ajánlott GHC verziót:

`ghcup install ghc`

- Egy adott GHC-verzió telepítése:

`ghcup install ghc {{version}}`

- Az aktuálisan "aktív" GHC verzió beállítása:

`ghcup set ghc {{version}}`

- A cabal-install telepítése:

`ghcup install cabal`

- A `ghcup` frissítése:

`ghcup upgrade`
