# grafana-cli

> Egy kis futtatható program, amely a Grafana serve-hoz van csomagolva. További információ: <https://grafana.com/docs/grafana/latest/cli/>.

- Bizonyos bővítmények telepítése, frissítése vagy eltávolítása:

`grafana-cli plugins {{install|update|remove}} {{plugin_id1 plugin_id2 ...}}`

- Az összes telepített bővítmény listázása:

`grafana-cli plugins ls`
