# script

> Készítsen egy terminál munkamenetről egy gépírásos fájlt. További információ: <https://manned.org/script>.

- Indítsa el a rögzítést a "typescript" nevű fájlban:

`script`

- A felvétel leállítása:

`exit`

- A felvétel elindítása egy adott fájlban:

`script {{logfile.log}}`

- Hozzáfűzés egy meglévő fájlhoz:

`script -a {{logfile.log}}`

- Csendes végrehajtás start és done üzenetek nélkül:

`script -q {{logfile.log}}`
