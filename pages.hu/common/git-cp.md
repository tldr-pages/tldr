# git cp

> Egy meglévő fájl másolása egy új helyre, az előzmények megőrzésével. A `git-extras` része. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-cp>.

- Egy meglévő fájl másolása egy Git repóban, ugyanabban a könyvtárban maradva:

`git cp {{file}} {{new_file}}`

- Egy meglévő fájl másolása egy Git repóban, és máshová helyezése:

`git cp {{path/to/file}} {{path/to/new_file}}`
