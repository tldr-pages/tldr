# ng

> Parancssori felület (CLI) az Angular alkalmazások létrehozásához és kezeléséhez. További információ: <https://angular.io/cli>.

- Hozzon létre egy új Angular alkalmazást egy könyvtáron belül:

`ng new {{project_name}}`

- Új komponens hozzáadása az alkalmazáshoz:

`ng generate component {{component_name}}`

- Új osztály hozzáadása az alkalmazáshoz:

`ng generate class {{class_name}}`

- Új direktíva hozzáadása az alkalmazáshoz:

`ng generate directive {{directive_name}}`

- Futtassa az alkalmazást a következő paranccsal a gyökérkönyvtárában:

`ng serve`

- Építse meg az alkalmazást:

`ng build`

- Unit tesztek futtatása:

`ng test`

- Ellenőrizze az aktuális Angular telepítés verzióját:

`ng version`
