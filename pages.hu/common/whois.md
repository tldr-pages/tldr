# whois

> A WHOIS (RFC 3912) protokoll parancssori kliense. További információ: <https://github.com/rfc1036/whois>.

- Információszerzés egy domain névről:

`whois {{example.com}}`

- Információ lekérdezése egy IP-címről:

`whois {{8.8.8.8}}`

- Visszaélési kapcsolat lekérdezése egy IP-címhez:

`whois -b {{8.8.8.8}}`
