# az upgrade

> Az Azure CLI és bővítmények frissítése. A `az`, a Microsoft Azure parancssori kliensének része. További információ: <https://learn.microsoft.com/cli/azure/reference-index?view=azure-cli-latest#az-upgrade>.

- Az Azure CLI frissítése:

`az upgrade`

- Az Azure CLI és bővítmények frissítése:

`az upgrade --all`

- Az Azure CLI és bővítmények frissítése megerősítés kérése nélkül:

`az version --all --yes`
