# git diff-files

> Fájlok összehasonlítása sha1 hash-juk és módjuk alapján. További információ: <https://git-scm.com/docs/git-diff-files>.

- Az összes módosított fájl összehasonlítása:

`git diff-files`

- Csak a megadott fájlok összehasonlítása:

`git diff-files {{path/to/file}}`

- Csak a megváltozott fájlok neveinek megjelenítése:

`git diff-files --name-only`

- A kiterjesztett fejlécinformációk összefoglalójának kiadása:

`git diff-files --summary`
