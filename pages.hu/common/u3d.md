# u3d

> Eszközkészlet a Unityvel való interakcióhoz a parancssorból. További információ: <https://github.com/DragonBox/u3d>.

- Projekt megnyitása az aktuális könyvtárból a megfelelő Unity-verzióban:

`u3d`

- A Unity telepített verzióinak listája:

`u3d list`

- A Unity elérhető, letölthető verzióinak listája:

`u3d available`

- A Unity legújabb stabil verziójának letöltése és telepítése:

`u3d install latest_stable`

- Unity verzió és szerkesztő \[p\]ackages letöltése és telepítése:

`u3d install {{2021.2.0f1}} -p {{Unity,iOS,Android}}`
