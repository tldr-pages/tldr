# git info

> Git tárolóhelyi információk megjelenítése. A `git-extras` része. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-info>.

- Távoli helyek, távoli és helyi ágak, legfrissebb commit adatok és `.git/config` beállítások megjelenítése:

`git info`

- Távoli helyek, távoli és helyi ágak és legfrissebb commit adatok megjelenítése:

`git info --no-config`
