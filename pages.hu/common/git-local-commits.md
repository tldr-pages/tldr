# git local-commits

> Megjeleníti azokat a helyi commitokat, amelyeket még nem tettek át az origóba. Minden további argumentumot közvetlenül a `git log` címnek ad át. A \`git-extras része. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-local-commits>.

- Mutassa azokat a commitokat, amelyeket még nem tettek át:

`git local-commits`
