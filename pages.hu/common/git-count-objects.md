# git count-objects

> A kicsomagolt objektumok számának és lemezfogyasztásának számlálása. További információ: <https://git-scm.com/docs/git-count-objects>.

- Számolja meg az összes objektumot és jelenítse meg a teljes lemezhasználatot:

`git count-objects`

- Megjeleníti az összes objektum számlálását és teljes lemezhasználatát, a méreteket ember által olvasható mértékegységekben megjelenítve:

`git count-objects --human-readable`

- További részletesebb információk megjelenítése:

`git count-objects --verbose`

- További részletesebb információk megjelenítése, a méretek megjelenítése ember által olvasható egységekben:

`git count-objects --human-readable --verbose`
