# dolt blame

> Megjeleníti a Dolt-tábla minden egyes sorának kötelezettségvállalási adatait. További információ: <http://github.com/dolthub/dolt>.

- Megjeleníti a legutóbbi commitot egy táblázat minden egyes sorához:

`dolt blame {{table}}`

- Megjeleníti egy táblázat minden egyes sorának legutóbbi commitjait, amikor a megadott commit megtörtént:

`dolt blame {{commit}} {{table}}`

- Súgó megtekintése:

`dolt blame --help`
