# which

> Egy program keresése a felhasználó elérési útvonalán. További információ: <https://manned.org/which>.

- Keresés a PATH környezeti változóban, és a megfelelő futtatható programok helyének megjelenítése:

`which {{executable}}`

- Ha több futtatható fájl is van, akkor az összeset megjeleníti:

`which -a {{executable}}`
