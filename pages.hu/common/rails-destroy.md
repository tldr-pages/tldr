# rails destroy

> Rails-erőforrások megsemmisítése. További információ: <https://guides.rubyonrails.org/command_line.html#bin-rails-destroy>.

- Az összes elérhető generátor listája a megsemmisítéshez:

`rails destroy`

- A Post nevű modell megsemmisítése:

`rails destroy model {{Post}}`

- Destroy a controller named Posts:

`rails destroy controller {{Posts}}`

- Destroy a migráció, amely létrehozza Posts:

`rails destroy migration {{CreatePosts}}`

- Destroy a scaffold for a model named Post:

`rails destroy scaffold {{Post}}`
