# kaggle

> A Kaggle hivatalos CLI-je Python 3-ban implementálva. További információ: <https://github.com/Kaggle/kaggle-api>.

- Az aktuális konfigurációs értékek megtekintése:

`kaggle config view`

- Egy adott fájl letöltése egy versenyadatkészletből:

`kaggle competitions download {{competition}} -f {{filename}}`
