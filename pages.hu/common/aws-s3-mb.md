# aws s3 mb

> S3 vödrök létrehozása. További információ: <https://awscli.amazonaws.com/v2/documentation/api/latest/reference/s3/mb.html>.

- S3 vödör létrehozása:

`aws s3 mb s3://{{bucket_name}}`

- S3 vödör létrehozása egy adott régióban:

`aws s3 mb s3://{{bucket_name}} --region {{region}}`

- Súgó megjelenítése:

`aws s3 mb help`
