# az version

> Megjeleníti az Azure CLI modulok és bővítmények aktuális verzióját. A `az`, a Microsoft Azure parancssori kliensének része. További információ: <https://learn.microsoft.com/cli/azure/reference-index?view=azure-cli-latest#az-version>.

- Az Azure CLI modulok és bővítmények aktuális verziójának megjelenítése JSON formátumban:

`az version`

- Az Azure CLI modulok és bővítmények aktuális verziójának megjelenítése adott formátumban:

`az version --output {{json|table|tsv}}`
