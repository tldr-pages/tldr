# ignite

> CLI a React Native boilerplate-ekhez, pluginekhez, generátorokhoz és egyebekhez. További információ: <https://infinite.red/ignite>.

- Hozzon létre egy új React Native projektet:

`ignite new {{project_name}}`

- Fájl generálása egy pluginból:

`ignite generate {{plugin_name}} {{path/to/file}}`

- Ignite plugin hozzáadása a projekthez:

`ignite add {{plugin_name}}`

- Ignite plugin eltávolítása a projektből:

`ignite remove {{plugin_name}}`
