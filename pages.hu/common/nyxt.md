# nyxt

> A Nyxt egy billentyűzetvezérlésű webböngésző power usereknek. További információ: <https://nyxt.atlas.engineer>.

- Összes profil listája:

`nyxt --list-data-profiles`

- A `init.lisp` fájl elérési útvonalának beállítása:

`nyxt --init {{path/to/file}}`

- Az automatikus konfigurációs fájl elérési útvonalának módosítása:

`nyxt --auto-config {{path/to/file}}`

- Rendszerinformációk nyomtatása:

`nyxt --system-information`
