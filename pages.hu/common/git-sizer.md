# git sizer

> Kiszámítja a különböző Git-repozitóriumok méretét, és figyelmeztet minden olyanra, amely problémát vagy kellemetlenséget okozhat. További információ: <https://github.com/github/git-sizer>.

- Csak olyan statisztikákat jelent, amelyeknek az aggályossági szintje nagyobb, mint 0:

`git sizer`

- Jelentse az összes statisztikát:

`git sizer -v`

- Lásd a további opciókat:

`git sizer -h`
