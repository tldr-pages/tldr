# w3m

> Szövegalapú webböngésző. Támogatja az SSL-t és az egérrel történő bevitelt, akár SSH-n keresztül is. További információ: <http://w3m.sourceforge.net>.

- URL megnyitása:

`w3m {{http://example.com}}`

- URL megnyitása monokróm módban:

`w3m {{http://example.com}} -M`

- URL megnyitása egértámogatás nélkül:

`w3m {{http://example.com}} --no-mouse`

- Új böngészőfül megnyitása:

`Shift + T`

- A böngésző előzményeinek megjelenítése:

`Ctrl + H`

- A w3m kilépése:

`'q' then 'y'`
