# pyenv virtualenv

> Virtuális környezetek létrehozása a telepített Python disztribúciók alapján. További információ: <https://github.com/pyenv/pyenv-virtualenv>.

- Hozzon létre egy új Python 3.6.6 virtuális környezetet:

`pyenv virtualenv {{3.6.6}} {{virtualenv_name}}`

- Az összes meglévő virtuális környezet listázása:

`pyenv virtualenvs`

- Virtuális környezet aktiválása:

`pyenv activate {{virtualenv_name}}`

- Virtuális környezet deaktiválása:

`pyenv deactivate`
