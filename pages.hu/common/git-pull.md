# git pull

> Ág lekérése egy távoli tárolóból és egyesítése a helyi tárolóba. További információ: <https://git-scm.com/docs/git-pull>.

- Változások letöltése az alapértelmezett távoli tárolóból és egyesítése:

`git pull`

- Változások letöltése az alapértelmezett távoli tárolóból és a gyorstovábbítás használata:

`git pull --rebase`

- Letöltés a változtatások letöltése adott távoli tárolóból és ágból, majd egyesítés a HEAD-be:

`git pull {{remote_name}} {{branch}}`
