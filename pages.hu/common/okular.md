# okular

> Dokumentumok megtekintése. További információ: <https://docs.kde.org/stable5/en/okular/okular/command-line-options.html>.

- Dokumentumnézegető elindítása:

`okular`

- Meghatározott dokumentumok megnyitása:

`okular {{path/to/file1 path/to/file2 ...}}`

- Egy dokumentum megnyitása egy adott oldalon:

`okular --page {{page_number}} {{path/to/file}}`

- Egy adott dokumentum megnyitása prezentációs módban:

`okular --presentation {{path/to/file}}`

- Egy adott dokumentum megnyitása és nyomtatási párbeszédpanel indítása:

`okular --print {{path/to/file}}`

- Dokumentum megnyitása és egy adott karakterlánc keresése:

`okular --find {{search_string}} {{path/to/file}}`
