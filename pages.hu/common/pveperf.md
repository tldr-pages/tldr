# pveperf

> A Proxmox Server benchmarking eszköze. Gyűjtsön CPU és merevlemez teljesítményadatokat a merevlemezre vonatkozóan. További információ: <https://pve.proxmox.com/pve-docs/pveperf.1.html>.

- A `/` címre szerelt merevlemez CPU- és merevlemez-teljesítményadatainak megjelenítése:

`pveperf`
