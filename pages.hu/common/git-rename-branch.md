# git rename-branch

> Egy Git-ág átnevezése. A `git-extras` része. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-rename-branch>.

- Nevezze át a jelenleg használt ágat:

`git rename-branch {{new_branch_name}}`

- Egy adott ág átnevezése:

`git rename-branch {{old_branch_name}} {{new_branch_name}}`
