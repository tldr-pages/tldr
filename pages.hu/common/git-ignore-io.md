# git ignore-io

> .gitignore fájlok generálása előre definiált sablonokból. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-ignore-io>.

- A rendelkezésre álló sablonok listázása:

`git ignore-io list`

- .gitignore sablon generálása:

`git ignore-io {{item_a,item_b,item_n}}`
