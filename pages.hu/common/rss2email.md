# rss2email

> Eszköz az RSS hírcsatornákból származó hírek e-mail programba történő továbbítására. További információ: <https://github.com/rss2email/rss2email>.

- Az összes hírfolyam listája:

`r2e list`

- RSS-bejegyzések átalakítása e-mailbe:

`r2e run`

- Adjon hozzá egy feedet:

`r2e add {{feed_address}}`

- Adjon hozzá egy feedet egy adott e-mail címmel:

`r2e add {{feed_address}} {{new_email@example.com}}`

- Egy adott feed törlése:

`r2e delete {{number_of_feed_in_list}}`

- Súgó megjelenítése:

`r2e -h`
