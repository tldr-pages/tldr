# git unpack-file

> Ideiglenes fájl létrehozása egy blob tartalmával. További információ: <https://git-scm.com/docs/git-unpack-file>.

- Létrehoz egy olyan fájlt, amely a blob azonosítója által megadott blob tartalmát tartalmazza, majd kiírja az ideiglenes fájl nevét:

`git unpack-file {{blob_id}}`
