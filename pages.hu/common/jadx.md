# jadx

> Dex to Java dekompilátor. Java forráskódot készít Android Dex és APK fájlokból. További információ: <https://github.com/skylot/jadx>.

- Dex fájl dekompilálása egy könyvtárba:

`jadx {{path/to/file}}`

- Egy Dex fájl dekompilálása egy adott könyvtárba:

`jadx --output-dir {{path/to/directory}} {{path/to/file}}`
