# n

> Eszköz több csomópont verziójának kezelésére. További információ: <https://github.com/tj/n>.

- A node adott verziójának telepítése. Ha a verzió már telepítve van, akkor aktiválódik:

`n {{version}}`

- A telepített verziók megjelenítése és interaktívan aktiválja az egyiket:

`n`

- Egy verzió eltávolítása:

`n rm {{version}}`

- Egy adott verziójú fájl végrehajtása:

`n use {{version}} {{file.js}}`

- Kimeneti bináris elérési útvonal egy verzióhoz:

`n bin {{version}}`
