# dict

> A DICT protokollt használó parancssori szótár. További információ: <https://github.com/cheusov/dictd>.

- Az elérhető adatbázisok listája:

`dict -D`

- Információk lekérdezése egy adatbázisról:

`dict -i {{database_name}}`

- Egy szó keresése egy adott adatbázisban:

`dict -d {{database_name}} {{word}}`

- Egy szó keresése az összes elérhető adatbázisban:

`dict {{word}}`

- Információk megjelenítése a DICT-kiszolgálóról:

`dict -I`
