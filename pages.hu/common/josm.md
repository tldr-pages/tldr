# josm

> Bővíthető OpenStreetMap szerkesztő Java 8+-ra. További információ: <https://josm.openstreetmap.de/>.

- A JOSM elindítása:

`josm`

- A JOSM elindítása maximalizált módban:

`josm --maximize`

- A JOSM elindítása és egy adott nyelv beállítása:

`josm --language {{de}}`

- A JOSM elindítása és az összes beállítás visszaállítása az alapértelmezett értékekre:

`josm --reset-preferences`

- A JOSM elindítása és egy adott határoló doboz letöltése:

`josm --download {{minlat,minlon,maxlat,maxlon}}`

- A JOSM elindítása és egy adott határoló doboz letöltése nyers GPS-ként:

`josm --downloadgps {{minlat,minlon,maxlat,maxlon}}`

- A JOSM elindítása bővítmények nélkül:

`josm --skip-plugins`
