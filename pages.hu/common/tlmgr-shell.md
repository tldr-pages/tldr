# tlmgr shell

> A natív TeX Live menedzser interaktív héjának elindítása. További információ: <https://www.tug.org/texlive/tlmgr.html>.

- A `tlmgr` interaktív héjazatának elindítása:

`tlmgr shell`

- A `tlmgr` bármelyik alparancsának futtatása az interaktív héjban:

`{{sub_command}} {{arguments}}`

- Az interaktív héj kilépése:

`quit`

- Az összes TeX Live változó listázása:

`get`

- Egy TeX Live változó értékének lekérdezése:

`get {{variable}}`

- Egy TeX Live változó értékének beállítása:

`set {{variable}} {{value}}`

- Az interaktív héj újraindítása:

`restart`

- Az aktuális protokoll verziójának kinyomtatása:

`protocol`
