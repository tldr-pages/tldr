# timew

> A tevékenységek időtartamának mérésére használt időnyilvántartó eszköz. További információ: <https://timewarrior.net/docs>.

- Indítson el egy új stopperórát, és adjon egy címkénevet a nyomon követendő tevékenységnek:

`timew start {{activity_tag}}`

- Futó stopperórák megtekintése:

`timew`

- Állítsa meg a stopperórát egy adott címkénévvel:

`timew stop {{activity_tag}}`

- Az összes futó stopperóra leállítása:

`timew stop`

- Nyomon követett elemek megtekintése:

`timew summary`
