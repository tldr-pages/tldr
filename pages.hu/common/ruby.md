# ruby

> Ruby programozási nyelv értelmezője. További információ: <https://www.ruby-lang.org>.

- Indítson el egy REPL-t (interaktív héj):

`irb`

- Ruby szkript végrehajtása:

`ruby {{script.rb}}`

- Egyetlen Ruby parancs végrehajtása a parancssorban:

`ruby -e {{command}}`

- Egy adott Ruby szkript szintaktikai hibáinak ellenőrzése:

`ruby -c {{script.rb}}`

- A használt Ruby verziójának megjelenítése:

`ruby -v`
