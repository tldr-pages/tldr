# git browse-ci

> Nyissa meg az aktuális `git` repository CI weboldalát az alapértelmezett webböngészőben. A `git-extras` része. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-browse-ci>.

- Nyissa meg az aktuális adattár CI-konfigurációját az upstream weboldalán:

`git browse-ci`

- Az aktuális tárolóhely CI-konfigurációjának megnyitása az upstream weboldalán egy adott távoli elérhetőséghez:

`git browse-ci {{remote}}`
