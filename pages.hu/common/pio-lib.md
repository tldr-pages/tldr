# pio lib

> PlatformIO könyvtárak kezelése. További információ: <https://docs.platformio.org/en/latest/core/userguide/lib/>.

- Telepített könyvtárak listázása:

`pio lib list`

- Beépített könyvtárak listázása a telepített fejlesztési platformok és azok keretrendszerei alapján:

`pio lib builtin`

- Meglévő könyvtárak keresése:

`pio lib search {{keyword}}`

- Részletek megjelenítése egy könyvtárról:

`pio lib show {{library}}`

- Könyvtár telepítése:

`pio lib install {{library}}`

- Telepített könyvtárak frissítése:

`pio lib update`

- Egy könyvtár eltávolítása:

`pio lib uninstall {{library}}`

- PlatformIO könyvtárak nyilvántartási statisztikáinak megjelenítése:

`pio lib stats`
