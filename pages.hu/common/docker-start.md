# docker start

> Egy vagy több leállított konténer elindítása. További információ: <https://docs.docker.com/engine/reference/commandline/start/>.

- Segítség megjelenítése:

`docker start`

- Docker konténer indítása:

`docker start {{container}}`

- Egy konténer indítása, a `stdout` és a `stderr` csatolása és a jelek továbbítása:

`docker start --attach {{container}}`

- Egy vagy több, szóközzel elválasztott konténer indítása:

`docker start {{container(s)}}`
