# pwgen

> Kiejthető jelszavak generálása. További információ: <https://github.com/tytso/pwgen>.

- Véletlenszerű jelszó generálása s\[y\]mbols segítségével:

`pwgen -y {{length}}`

- Biztonságos, nehezen megjegyezhető jelszavak generálása:

`pwgen -s {{length}}`

- Legalább egy nagybetűs jelszó generálása:

`pwgen -c {{length}}`
