# IPython

> Python shell automatikus előzményekkel, dinamikus objektum-áttekintéssel, könnyebb konfigurációval, parancskiegészítéssel, hozzáféréssel a rendszerhéjhoz és még sok mással. További információ: <https://ipython.readthedocs.io>.

- REPL (interaktív shell) indítása:

`ipython`

- Belépés egy interaktív IPython munkamenetbe egy Python szkript futtatása után:

`ipython -i {{script.py}}`

- Alapértelmezett IPython profil létrehozása:

`ipython profile create`

- Az alapértelmezett IPython-profil könyvtárának elérési útvonalának kiírása:

`ipython locate profile`

- Az IPython history adatbázis törlése, az összes bejegyzés törlése:

`ipython history clear`
