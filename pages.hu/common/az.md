# az

> A Microsoft Azure hivatalos CLI-eszköze. Egyes alparancsok, például a `az login` saját használati dokumentációval rendelkeznek. További információk: <https://learn.microsoft.com/cli/azure>.

- Jelentkezzen be az Azure rendszerbe:

`az login`

- Azure előfizetési információk kezelése:

`az account`

- Az összes Azure kezelt lemez listázása:

`az disk list`

- Az összes Azure virtuális gép listázása:

`az vm list`

- Azure Kubernetes szolgáltatások kezelése:

`az aks`

- Azure hálózati erőforrások kezelése:

`az network`
