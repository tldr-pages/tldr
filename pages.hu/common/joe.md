# joe

> Joe saját szövegszerkesztője. További információ: <https://joe-editor.sourceforge.io>.

- Új fájl megnyitása a JOE-ban:

`joe`

- Egy adott fájl megnyitása:

`joe {{path/to/file}}`

- Egy adott fájl megnyitása, a kurzort a megadott sorra helyezve:

`joe +{{line}} {{path/to/file}}`

- Egy adott fájl megnyitása csak olvasási módban:

`joe -rdonly {{path/to/file}}`
