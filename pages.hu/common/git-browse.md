# git browse

> Egy upstream tároló megtekintése az alapértelmezett böngészőben. A `git-extras` része. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-browse>.

- Az első upstream megnyitása az alapértelmezett böngészőben:

`git browse`

- Egy adott upstream megnyitása az alapértelmezett böngészőben:

`git browse {{upstream}}`
