# haxelib

> Haxe Library Manager. További információ: <https://lib.haxe.org/>.

- Haxe könyvtár keresése:

`haxelib search {{keyword}}`

- Haxe könyvtár telepítése:

`haxelib install {{libname}}`

- Egy Haxe könyvtár egy adott verziójának telepítése:

`haxelib install {{libname}} {{version}}`

- Az összes telepített Haxe könyvtár frissítése:

`haxelib upgrade`

- Egy könyvtár fejlesztői verziójának telepítése egy Git-tárból:

`haxelib git {{libname}} {{git_url}}`

- Haxe könyvtárak eltávolítása:

`haxelib remove {{libname}}`

- A helyileg telepített Haxe könyvtárak fájának kinyomtatása:

`haxelib list`
