# xml list

> Egy könyvtár tartalmának listázása (mint a `ls`) XML formátumban. További információ: <http://xmlstar.sourceforge.net/docs.php>.

- Az aktuális könyvtár listájának kiírása egy XML dokumentumba:

`xml list > {{path/to/dir_list.xml}}`

- A megadott könyvtár listájának XML-dokumentumba írása:

`xml list {{path/to/directory}} > {{path/to/dir_list.xml}}`

- A `list` alparancs súgójának megjelenítése:

`xml list --help`
