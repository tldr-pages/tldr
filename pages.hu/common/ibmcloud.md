# ibmcloud

> Parancssori eszköz az IBM Cloud alkalmazások és szolgáltatások kezelésére. További információ: <https://cloud.ibm.com/docs/cli?topic=cli-ibmcloud_cli>.

- Frissítse a `ibmcloud` oldalt a legújabb verzióra:

`ibmcloud update`

- Telepítse a Cloud Foundry modult a Cloud Foundry szolgáltatások eléréséhez:

`ibmcloud cf install`

- Az összes elérhető IBM Cloud régió listázása:

`ibmcloud regions`

- A `ibmcloud` verziójának megjelenítése:

`ibmcloud version`

- Súgó megjelenítése:

`ibmcloud help`

- Súgó megjelenítése egy alparancshoz:

`ibmcloud help {{subcommand}}`
