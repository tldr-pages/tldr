# mullvad

> CLI kliens a Mullvad VPN-hez. További információ: <https://mullvad.net/>.

- Kapcsolja össze a mullvad fiókját a megadott számlaszámmal:

`mullvad account set {{account_number}}`

- Engedélyezze a LAN-hozzáférést, miközben a VPN be van kapcsolva:

`mullvad lan set allow`

- Létrehozza a VPN-alagutat:

`mullvad connect`

- A VPN-alagút állapotának ellenőrzése:

`mullvad status`
