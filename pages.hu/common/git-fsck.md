# git fsck

> Ellenőrzi a csomópontok érvényességét és kapcsolódását egy Git-tár indexében. Nem végez módosításokat. Lásd `git gc` a lógó csomópontok megtisztításához. További információ: <https://git-scm.com/docs/git-fsck>.

- Ellenőrzi az aktuális tárolót:

`git fsck`

- Az összes talált címke listázása:

`git fsck --tags`

- List all root nodes found:

`git fsck --root`
