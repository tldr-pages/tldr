# interdiff

> Két diff fájl közötti különbségek megjelenítése. További információ: <http://freshmeat.sourceforge.net/projects/patchutils>.

- Diff fájlok összehasonlítása:

`interdiff {{old_file}} {{new_file}}`

- Diff fájlok összehasonlítása, a szóközök figyelmen kívül hagyásával:

`interdiff -w {{old_file}} {{new_file}}`
