# phpstorm

> A JetBrains IntelliJ platformjára épülő, platformokon átívelő IDE PHP-hez. További információ: <https://jetbrains.com/phpstorm>.

- Egy adott könyvtár megnyitása:

`phpstorm {{path/to/directory}}`

- Egy fájl megnyitása:

`phpstorm {{path/to/file}}`

- Fájl megnyitása egy adott sorban:

`phpstorm --line {{line_number}} {{path/to/file}}`

- Két fájl közötti különbségek megtekintése:

`phpstorm diff {{path/to/left_file}} {{path/to/right_file}}`
