# users

> A bejelentkezett felhasználók listájának megjelenítése. Lásd még: `useradd`, `userdel`, `usermod`. További információ: <https://www.gnu.org/software/coreutils/users>.

- Bejelentkezett felhasználónevek nyomtatása:

`users`

- Bejelentkezett felhasználónevek nyomtatása egy megadott fájl alapján:

`users {{/var/log/wmtp}}`
