# scala

> Scala alkalmazásindító és interaktív értelmező. További információ: <https://scala-lang.org>.

- REPL (interaktív shell) indítása:

`scala`

- Indítsa el az értelmezőt a classpath-ban lévő függőséggel:

`scala -classpath {{filename.jar}} {{command}}`

- Egy Scala script végrehajtása:

`scala {{script.scala}}`

- Egy `.jar` program végrehajtása:

`scala {{filename.jar}}`

- Egyetlen Scala parancs végrehajtása a parancssorban:

`scala -e {{command}}`
