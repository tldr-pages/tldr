# tlmgr path

> Szimlinkek hozzáadása vagy eltávolítása a TeX Live futtatható fájljaihoz, man oldalakhoz és információs oldalakhoz. Ezt a parancsot újra kell futtatni a jövőben hozzáadott fájlok esetében. További információ: <https://www.tug.org/texlive/tlmgr.html>.

- Szimlinkek hozzáadása a TeX Live fájlokhoz:

`sudo tlmgr path add`

- Szimlinkek eltávolítása a TeX Live fájlokhoz:

`sudo tlmgr path remove`
