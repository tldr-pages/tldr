# rapper

> A Raptor RDF elemző segédprogram, a Raptor RDF Syntax Library része. További információ: <http://librdf.org/raptor/rapper.html>.

- Egy RDF/XML dokumentum Turtle formátumúvá konvertálása:

`rapper -i rdfxml -o turtle {{path/to/file}}`

- A Turtle fájlban lévő hármasok számának megszámlálása:

`rapper -i turtle -c {{path/to/file}}`
