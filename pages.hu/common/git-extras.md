# git extras

> Git bővítőcsomag. További információ: <https://github.com/tj/git-extras>.

- A `git-extras` parancsok telepítése vagy frissítése:

`git extras update`

- Súgó és alparancsok megjelenítése:

`git extras --help`

- A telepített `git-extras` verziójának kinyomtatása:

`git extras --version`
