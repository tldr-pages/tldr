# git reset-file

> Egy fájl visszaváltása a HEAD-re vagy egy adott commit-ra. A `git-extras` része. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-reset-file>.

- Egy fájl visszaállítása HEAD-re:

`git reset-file {{path/to/file}}`

- Egy fájl visszaállítása egy adott commitra:

`git reset-file {{path/to/file}} {{commit_hash}}`
