# aws codecommit

> Az AWS CodeCommit egy menedzselt forráskód-ellenőrzési szolgáltatás, amely privát Git-tárakat tárol. További információ: <https://docs.aws.amazon.com/cli/latest/reference/codecommit/>.

- Súgó megjelenítése egy adott parancshoz:

`aws codecommit {{command}} help`

- Súgó megjelenítése:

`aws codecommit help`
