# ionic

> Keretrendszer hibrid mobilalkalmazások készítéséhez. További információ: <https://ionicframework.com/docs/cli>.

- Új projekt létrehozása:

`ionic start`

- Indítson el egy helyi dev-kiszolgálót az alkalmazás fejlesztéséhez/teszteléséhez:

`ionic serve`

- Új alkalmazáskomponens, direktíva, oldal, cső, szolgáltató vagy lapok generálása:

`ionic g {{page}}`

- Az Ionic, Cordova, környezet stb. verzióinak megjelenítése:

`ionic info`

- Alkalmazás futtatása Android/iOS eszközön:

`ionic cordova run {{android|ios}} --device`

- Ellenőrizze az Ionic alkalmazás állapotát:

`ionic doctor {{check}}`
