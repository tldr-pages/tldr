# R

> R nyelvi tolmács. További információ: <https://www.r-project.org>.

- Indítson el egy REPL-t (interaktív héj):

`R`

- Az R verziójának ellenőrzése:

`R --version`

- Indítsa el az R-t vanilla módban (azaz üres munkamenetben, amely nem menti el a munkaterületet a végén):

`R --vanilla`

- Egy fájl végrehajtása:

`R -f {{path/to/file.R}}`

- Egy R-kifejezés végrehajtása, majd kilépés:

`R -e {{expr}}`

- Az R futtatása hibakeresővel:

`R -d {{debugger}}`

- R csomagok ellenőrzése a csomagforrásokból:

`R CMD check {{path/to/package_source}}`
