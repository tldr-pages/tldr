# git merge-into

> Egy ág beolvasztása egy másik ágba. A `git-extras` része. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-merge-into>.

- Egy forráság egyesítése egy adott célágba:

`git merge-into {{source_branch}} {{destination_branch}}`

- Jelenlegi ág beolvasztása egy adott célágba:

`git merge-into {{destination_branch}}`
