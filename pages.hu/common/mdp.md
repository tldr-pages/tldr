# mdp

> Parancssor alapú eszköz prezentációk készítésére Markdown fájlokból. További információ: <https://github.com/visit1985/mdp>.

- Prezentáció indítása a terminálban egy Markdown fájlból:

`mdp {{presentation.md}}`

- A halványuló átmenetek kikapcsolása:

`mdp --nofade {{presentation.md}}`

- Invertálja a betűszíneket a világos hátterű terminálokon való használathoz:

`mdp --invert {{presentation.md}}`

- Átlátszóság letiltása átlátszó terminálokon:

`mdp --notrans {{presentation.md}}`
