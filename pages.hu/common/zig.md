# zig

> A Zig fordító és toolchain. További információ: <https://ziglang.org>.

- Fordítsa le a projektet az aktuális könyvtárban:

`zig build`

- A projekt fordítása és futtatása az aktuális könyvtárban:

`zig build run`

- A `zig build` alkalmazás inicializálása:

`zig init-exe`

- A `zig build` könyvtár inicializálása:

`zig init-lib`

- Létrehoz és futtat egy teszt buildet:

`zig test {{path/to/file.zig}}`

- A Zig-forrás kanonikus formába való átformálása:

`zig fmt {{path/to/file.zig}}`

- A Zig használata C fordítóprogramként:

`zig cc {{path/to/file.c}}`

- A Zig használata C++ fordítóként:

`zig c++ {{path/to/file.cpp}}`
