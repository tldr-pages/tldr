# composer-require-checker

> CLI eszköz a Composer függőségek elemzésére a puha függőségek szempontjából. További információ: <https://github.com/maglnet/ComposerRequireChecker>.

- Composer JSON fájl elemzése:

`composer-require-checker check {{path/to/composer.json}}`

- Egy Composer JSON fájl elemzése egy adott konfigurációval:

`composer-require-checker check --config-file {{path/to/config.json}} {{path/to/composer.json}}`
