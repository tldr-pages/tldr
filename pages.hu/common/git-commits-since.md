# git commits-since

> Egy adott időpont vagy dátum óta történt commitok megjelenítése. A `git-extras` része. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-commits-since>.

- Tegnap óta történt commitok megjelenítése:

`git commits-since {{yesterday}}`

- Múlt hét óta történt commitok megjelenítése:

`git commits-since {{last week}}`

- Múlt hónap óta történt commitok megjelenítése:

`git commits-since {{last month}}`

- Tegnap délután 2 óra óta történt commitok megjelenítése:

`git commits-since {{yesterday 2pm}}`
