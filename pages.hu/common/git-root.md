# git root

> Az aktuális Git-tár gyökérkönyvtárának kiírása. A `git-extras` része. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-root>.

- Az aktuális Git-tár abszolút elérési útjának kiírása:

`git root`

- Az aktuális Git-tár gyökeréhez viszonyított aktuális munkakönyvtár kiírása:

`git root --relative`
