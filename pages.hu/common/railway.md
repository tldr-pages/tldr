# railway

> Kódot csatlakoztathat egy Railway projekthez a parancssorból. További információ: <https://railway.app/>.

- Bejelentkezés egy Railway fiókba:

`railway login`

- Kapcsolódás egy meglévő projekthez egy Railway-fiók vagy -csapat alatt:

`railway link {{projectId}}`

- Új projekt létrehozása közvetlenül a parancssorból:

`railway init`

- Helyi parancs futtatása az aktív környezet változóinak felhasználásával:

`railway run {{cmd}}`

- A linkelt projektkönyvtár telepítése (ha egy alkönyvtárból futtatja, a projekt gyökere továbbra is telepítésre kerül):

`railway up`

- Interaktív shell megnyitása egy adatbázishoz:

`railway connect`
