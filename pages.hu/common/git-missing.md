# git missing

> A két ág között nem megosztott commitok megjelenítése. A `git-extras` része. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-missing>.

- Azoknak a commitoknak a megjelenítése, amelyek nem közösek az aktuálisan kiadott ág és egy másik ág között:

`git missing {{branch}}`

- Két ág között nem megosztott commitok megjelenítése:

`git missing {{branch_1}} {{branch_2}}`
