# pyats version

> A pyATS telepítésének megtekintése és frissítése. További információ: <https://developer.cisco.com/pyats/>.

- Az összes csomag verziójának megjelenítése:

`pyats version check`

- Az elavult csomagok megjelenítése:

`pyats version check --outdated`

- A csomagok frissítése a legfrissebb verzióra:

`pyats version update`

- Csomagok frissítése vagy visszaminősítése egy adott verzióra:

`pyats version update {{version}}`
