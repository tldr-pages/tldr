# git alias

> Rövidítések létrehozása a Git parancsokhoz. A `git-extras` része. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-alias>.

- Az összes álnév listázása:

`git alias`

- Új alias létrehozása:

`git alias "{{name}}" "{{command}}"`

- Meglévő alias keresése:

`git alias ^{{name}}`
