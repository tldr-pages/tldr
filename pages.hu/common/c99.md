# c99

> C programokat fordít az ISO C szabványnak megfelelően. További információ: <https://manned.org/c99>.

- Forrásfájl(ok) lefordítása és futtatható állomány létrehozása:

`c99 {{file.c}}`

- Forrásfájl(ok) lefordítása és futtatható állomány létrehozása egyéni névvel:

`c99 -o {{executable_name}} {{file.c}}`

- Forrásfájl(ok) fordítása és objektumfájl(ok) létrehozása:

`c99 -c {{file.c}}`

- Forrásfájl(ok) lefordítása, összekapcsolása objektumfájl(ok)kal, és futtatható állomány létrehozása:

`c99 {{file.c}} {{file.o}}`
