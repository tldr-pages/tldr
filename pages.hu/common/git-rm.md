# git rm

> Fájlok eltávolítása az adattár indexéből és a helyi fájlrendszerből. További információ: <https://git-scm.com/docs/git-rm>.

- Fájl eltávolítása az adattár indexéből és a fájlrendszerből:

`git rm {{path/to/file}}`

- Könyvtár eltávolítása:

`git rm -r {{path/to/directory}}`

- Fájl eltávolítása az adattár indexéből, de helyben érintetlenül hagyja:

`git rm --cached {{path/to/file}}`
