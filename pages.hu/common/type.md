# type

> A parancs típusának megjelenítése a héj által végrehajtandó parancsok között. További információ: <https://manned.org/type>.

- A parancs típusának megjelenítése:

`type {{command}}`

- A megadott futtatható programot tartalmazó összes hely megjelenítése:

`type -a {{command}}`

- A végrehajtandó lemezfájl nevének megjelenítése:

`type -p {{command}}`
