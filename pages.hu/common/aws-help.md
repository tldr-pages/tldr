# aws help

> Az AWS CLI-vel kapcsolatos súgóinformációk megjelenítése. További információ: <https://docs.aws.amazon.com/cli/latest/userguide/cli-usage-help.html>.

- Súgó megjelenítése:

`aws help`

- Az összes elérhető téma felsorolása:

`aws help topics`

- Súgó megjelenítése egy adott témáról:

`aws help {{topic_name}}`
