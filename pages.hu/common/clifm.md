# clifm

> A parancssori fájlkezelő. További információ: <https://github.com/leo-arch/clifm>.

- Indítsa el a CliFM-et:

`clifm`

- Nyissa meg azt a fájlt vagy könyvtárat, amelynek ELN (bejegyzési listaszám) 12:

`12`

- Hozzon létre egy új fájlt és egy új könyvtárat:

`n file dir/`

- PDF-fájlok keresése az aktuális könyvtárban:

`/*.pdf`

- Az aktuális könyvtárban lévő összes PNG fájl kiválasztása:

`s *.png`

- A korábban kiválasztott fájlok eltávolítása (a `t` használatával a fájlokat a szemetesbe küldheti):

`r sel`

- Részletes súgó megjelenítése:

`?`

- CliFM kilépése:

`q`
