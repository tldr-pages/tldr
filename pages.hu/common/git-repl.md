# git repl

> Git REPL (read-evaluate-print-loop) - egy interaktív Git shell. A `git-extras` része. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-repl>.

- Interaktív Git shell indítása:

`git repl`

- Git parancs futtatása az interaktív Git héjban:

`{{git_subcommand}} {{command_arguments}}`

- Külső (nem Git) parancs futtatása az interaktív Git héjban:

`!{{command}} {{command_arguments}}`

- Kilépés az interaktív Git héjból (vagy a Ctrl + D billentyűkombináció megnyomásával):

`exit`
