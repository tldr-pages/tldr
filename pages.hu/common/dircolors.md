# dircolors

> Az LS_COLOR környezeti változó és stílus beállítására szolgáló kimeneti parancsok `ls`, `dir`, stb. További információ: <https://www.gnu.org/software/coreutils/dircolors>.

- Kimeneti parancsok az LS_COLOR alapértelmezett színek használatával történő beállításához:

`dircolors`

- Kimeneti parancsok az LS_COLOR beállításához egy fájlból származó színek használatával:

`dircolors {{path/to/file}}`

- Kimeneti parancsok Bourne shellhez:

`dircolors --bourne-shell`

- Kimeneti parancsok C shellhez:

`dircolors --c-shell`

- A fájltípusok és kiterjesztések alapértelmezett színeinek megtekintése:

`dircolors --print-data`
