# mat2

> Különböző fájlformátumok anonimizálása a metaadatok eltávolításával. További információ: <https://0xacab.org/jvoisin/mat2>.

- Támogatott fájlformátumok listája:

`mat2 --list`

- Metaadatok eltávolítása egy fájlból:

`mat2 {{path/to/file}}`

- Metaadatok eltávolítása egy fájlból és részletes kimenet kiírása a konzolra:

`mat2 --verbose {{path/to/file}}`

- Metaadatok megjelenítése egy fájlban azok eltávolítása nélkül:

`mat2 --show {{path/to/file}}`

- A metaadatok részleges eltávolítása egy fájlból:

`mat2 --lightweight {{path/to/file}}`
