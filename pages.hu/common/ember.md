# ember

> Az Ember.js parancssori segédprogram. Ember.js alkalmazások létrehozására és karbantartására szolgál. További információk: <https://cli.emberjs.com>.

- Hozzon létre egy új Ember alkalmazást:

`ember new {{my_new_app}}`

- Új Ember addon létrehozása:

`ember addon {{my_new_addon}}`

- Építse a projektet:

`ember build`

- Építse a projektet termelési üzemmódban:

`ember build -prod`

- Futtassa a fejlesztői kiszolgálót:

`ember serve`

- A tesztcsomag futtatása:

`ember test`

- Futtasson egy tervezetet, hogy létrehozzon valamit, például egy útvonalat vagy komponenst:

`ember generate {{type}} {{name}}`

- Telepítsen egy ember-cli kiegészítőt:

`ember install {{name_of_addon}}`
