# git help

> Súgóinformációk megjelenítése a Gitről. További információ: <https://git-scm.com/docs/git-help>.

- Egy adott Git alparanccsal kapcsolatos súgó megjelenítése:

`git help {{subcommand}}`

- Egy adott Git alparanccsal kapcsolatos súgó megjelenítése egy webböngészőben:

`git help --web {{subcommand}}`

- Az összes elérhető Git alparancs listájának megjelenítése:

`git help --all`

- Az elérhető útmutatók listázása:

`git help --guide`

- Az összes lehetséges konfigurációs változó felsorolása:

`git help --config`
