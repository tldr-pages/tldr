# grumphp

> PHP Composer plugin, amely lehetővé teszi a forráskód minőségének ellenőrzését. További információ: <https://github.com/phpro/grumphp>.

- Regisztrálja a Git horgokat:

`grumphp git:init`

- Indítsa el a pre-commit hookot manuálisan:

`grumphp git:pre-commit`

- Ellenőrizzen minden verziószámozott fájlt:

`grumphp run`
