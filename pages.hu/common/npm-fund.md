# npm fund

> A finanszírozási információk lekérdezése a csomagokból. További információ: <https://docs.npmjs.com/cli/v8/commands/npm-fund>.

- Függőségek listázása az aktuális könyvtárban lévő projekt finanszírozási URL-címével:

`npm fund`

- Egy adott csomag finanszírozási URL-jének megnyitása az alapértelmezett webböngészőben:

`npm fund {{package}}`

- Függőségek listázása finanszírozási URL-lel egy adott \[w\]orkspace-hez a projekthez az aktuális könyvtárban:

`npm fund -w {{workspace}}`
