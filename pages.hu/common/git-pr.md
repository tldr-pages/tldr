# git pr

> Nézze meg a GitHub pull requesteket helyben. A `git-extras` része. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-pr>.

- Egy adott pull request megtekintése:

`git pr {{pr_number}}`

- Egy adott távoli helyről származó pull request megtekintése:

`git pr {{pr_number}} {{remote}}`

- Egy pull request kipróbálása az URL-címéről:

`git pr {{url}}`

- Régi pull request ágak kitakarítása:

`git pr clean`
