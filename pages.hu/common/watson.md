# watson

> Egy csodálatos CLI az időkövetéshez. További információ: <https://github.com/TailorDev/Watson>.

- Kezdje el az idő nyomon követését a projektben:

`watson start {{project}}`

- Kezdje el az idő nyomon követését a projektben címkékkel:

`watson start {{project}} +{{tag}}`

- Az aktuális projektben az idő figyelésének leállítása:

`watson stop`

- A legutóbbi munkamenetek megjelenítése:

`watson log`

- Legutóbbi keret szerkesztése:

`watson edit`

- Legutóbbi keret eltávolítása:

`watson remove`
