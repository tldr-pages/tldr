# git delete-branch

> Helyi és távoli Git-ágak törlése. A `git-extras` része. Ha a kiadott ágat törli, csak a távoli ág kerül törlésre. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-delete-branch>.

- Helyi és távoli Git ág törlése:

`git delete-branch {{branch_name}}`

- Több helyi és távoli Git-ág törlése:

`git delete-branch {{branch_name1 branch_name2 ...}}`
