# next

> React keretrendszer, amely szerveroldali renderelést használ optimalizált webes alkalmazások építéséhez. További információ: <https://nextjs.org/docs>.

- Indítsa el az aktuális alkalmazást fejlesztési módban:

`next dev`

- Indítsa el az aktuális alkalmazást, és figyeljen egy adott porton:

`next dev --port {{port}}`

- Építse az aktuális alkalmazást termelésre optimalizálva:

`next build`

- Indítsa el a lefordított alkalmazást termelési üzemmódban:

`next start`

- Indítsa el a lefordított alkalmazást, és figyeljen egy adott porton:

`next start --port {{port}}`

- Az aktuális alkalmazás exportálása statikus HTML-oldalakba:

`next export`

- A Next.js telemetria állapotának megjelenítése:

`next telemetry`

- Súgó megjelenítése egy alparancshoz:

`next {{build|dev|export|start|telemetry}} --help`
