# avo

> Az Avo hivatalos parancssori interfésze. További információ: <https://www.avo.app/docs/implementation/cli>.

- Egy munkaterület inicializálása az aktuális könyvtárban:

`avo init`

- Jelentkezzen be az Avo platformra:

`avo login`

- Váltson egy meglévő Avo ágra:

`avo checkout {{branch_name}}`

- Húzza ki az analitikai csomagolásokat az aktuális elérési útvonalhoz:

`avo pull`

- Az Avo implementáció állapotának megjelenítése:

`avo status`

- Git konfliktusok feloldása az Avo fájlokban:

`avo conflict`

- Az aktuális Avo munkaterület megnyitása az alapértelmezett webböngészőben:

`avo edit`

- Egy alparancs súgójának megjelenítése:

`avo {{subcommand}} --help`
