# truffle

> Eszköz intelligens szerződések fejlesztésére, amelyekkel szolgáltatásokat lehet futtatni az Ethereum blokkláncon. További információ: <https://www.trufflesuite.com/docs/truffle/reference/truffle-commands>.

- Töltsön le egy előre elkészített Truffle projektet (Truffle Box):

`truffle unbox {{box_name}}`

- Fordítsa le a szerződések forrásfájljait az aktuális könyvtárban:

`truffle compile`

- JavaScript és Solidity tesztek futtatása:

`truffle test`

- Migrációk futtatása a szerződések telepítéséhez:

`truffle migrate`

- Súgó megjelenítése egy alparancshoz:

`truffle help {{subcommand}}`
