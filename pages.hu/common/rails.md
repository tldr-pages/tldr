# rails

> Ruby nyelven írt szerveroldali MVC keretrendszer. Néhány alparancsnak, mint például a `rails generate`, saját használati dokumentációja van. További információ: <https://guides.rubyonrails.org/command_line.html>.

- Hozzon létre egy új rails projektet:

`rails new "{{project_name}}"`

- Az aktuális projekt helyi kiszolgálójának elindítása a 3000-es porton:

`rails server`

- Helyi kiszolgáló indítása az aktuális projekthez egy megadott porton:

`rails server -p "{{port}}"`

- Konzol megnyitása az alkalmazással való interakcióhoz parancssorból:

`rails console`

- A rails aktuális verziójának ellenőrzése:

`rails --version`
