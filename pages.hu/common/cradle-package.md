# cradle package

> Csomagok kezelése egy Cradle-példányhoz. További információ: <https://cradlephp.github.io/docs/3.B.-Reference-Command-Line-Tools.html#package>.

- Az elérhető csomagok listájának megjelenítése:

`cradle package list`

- Csomag keresése:

`cradle package search {{package}}`

- Csomag telepítése a csomaglistáról:

`cradle package install {{package}}`

- Egy csomag egy adott verziójának telepítése:

`cradle package install {{package}} {{version}}`

- Egy csomag frissítése:

`cradle package update {{package}}`

- Egy csomag frissítése egy adott verzióra:

`cradle package update {{package}} {{version}}`

- Egy adott csomag eltávolítása:

`cradle package remove {{package}}`
