# espanso

> Rust nyelven írt, platformokon átívelő szövegbővítő. További információ: <https://espanso.org>.

- Ellenőrizze a státuszt:

`espanso status`

- A konfiguráció szerkesztése:

`espanso edit config`

- Telepítsen egy csomagot a hub store-ból[(https://hub.espanso.org/)](https://hub.espanso.org/):

`espanso install {{package_name}}`

- Újraindítás (csomag telepítése után szükséges, hasznos hiba esetén):

`espanso restart`
