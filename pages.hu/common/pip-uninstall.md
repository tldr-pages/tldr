# pip uninstall

> A Python csomagok eltávolítása. További információ: <https://pip.pypa.io>.

- Egy csomag eltávolítása:

`pip uninstall {{package_name}}`

- Egy adott fájlban felsorolt csomagok eltávolítása:

`pip uninstall --requirement {{path/to/requirements.txt}}`

- Csomag eltávolítása megerősítés kérése nélkül:

`pip uninstall --yes {{package_name}}`
