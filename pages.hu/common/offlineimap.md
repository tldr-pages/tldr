# offlineimap

> Egy távoli IMAP-kiszolgáló szinkronizálása a helyi Maildir mappákkal. További információ: <http://www.offlineimap.org>.

- Egyszeri szinkronizálás, automatikus frissítés engedélyezése nélkül:

`offlineimap -o`

- Egy adott fiók szinkronizálása:

`offlineimap -a {{account}}`

- Egy adott mappa szinkronizálása:

`offlineimap -f {{folder}}`
