# git unlock

> Egy fájl feloldása egy Git-repositoryban, hogy egy commit által módosítható legyen. A `git-extras` része. Lásd még: `git lock`. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-unlock>.

- Engedélyezi a korábban zárolt helyi fájl módosításainak commitolását:

`git unlock {{path/to/file}}`
