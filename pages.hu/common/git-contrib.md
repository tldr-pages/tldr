# git contrib

> Egy adott szerző commitjainak megjelenítése. A `git-extras` része. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-contrib>.

- Egy adott szerző összes commit hash-jának és a hozzájuk tartozó commit üzeneteknek a megjelenítése:

`git contrib {{author}}`
