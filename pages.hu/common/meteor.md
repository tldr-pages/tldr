# meteor

> Teljes körű JavaScript platform webes alkalmazások építéséhez. További információ: <https://meteor.com>.

- Futtasson egy meteor projektet a gyökérkönyvtárából fejlesztési módban:

`meteor`

- Hozzon létre egy projektet a megadott könyvtárban:

`meteor create {{path/to/directory}}`

- A projekt által jelenleg használt csomagok listájának megjelenítése:

`meteor list`

- Adjon hozzá egy csomagot a projekthez:

`meteor add {{package_name}}`

- Csomag eltávolítása a projektből:

`meteor remove {{package_name}}`

- A projekt gyártói összeállításának létrehozása tarball formájában a megadott könyvtár alatt:

`meteor build {{path/to/directory}}`
