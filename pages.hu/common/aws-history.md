# aws history

> Az AWS CLI-parancsok parancssori előzményeinek kinyomtatása (az AWS CLI-parancsok előzményeinek rögzítését engedélyezni kell). További információ: <https://docs.aws.amazon.com/cli/latest/reference/history/>.

- Parancsok előzményeinek listázása parancsazonosítókkal:

`aws history list`

- Megjeleníti egy adott parancshoz kapcsolódó eseményeket egy parancsazonosító megadásával:

`aws history show {{command_id}}`
