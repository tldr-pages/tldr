# spark

> A Laravel Spark parancssori eszköze. További információ: <https://spark.laravel.com>.

- Regisztrálja az API tokenjét:

`spark register {{token}}`

- Az aktuálisan regisztrált API-token megjelenítése:

`spark token`

- Új Spark projekt létrehozása:

`spark new {{project_name}}`

- Új Spark projekt létrehozása Braintree stubokkal:

`spark new {{project_name}} --braintree`

- Új Spark projekt létrehozása csapat alapú számlázási csonkokkal:

`spark new {{project_name}} --team-billing`
