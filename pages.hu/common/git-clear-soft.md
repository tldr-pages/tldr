# git clear-soft

> A Git munkakönyvtárának törlése, mintha frissen klónozták volna az aktuális ággal, kivéve a `.gitignore`. A `git-extras` részét. További információ: <https://github.com/tj/git-extras/blob/master/Commands.md#git-clear-soft>.

- Az összes követett fájl visszaállítása és az összes nem követett fájl törlése:

`git clear-soft`
