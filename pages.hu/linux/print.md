# print

> A `run-mailcap`'s action print alias. Eredetileg a `run-mailcap` a mime-type/file feldolgozására szolgál. További információ: <https://manned.org/print>.

- A Print művelet bármilyen fájl nyomtatására használható az alapértelmezett run-mailcap eszközön:

`print {{filename}}`

- A `run-mailcap`:

`run-mailcap --action=print {{filename}}`
