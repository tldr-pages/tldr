# cockpit-tls

> TLS termináló HTTP proxy az ügyfél és a `cockpit-ws` közötti forgalom titkosításához. További információ: <https://cockpit-project.org/guide/latest/cockpit-tls.8.html>.

- HTTP-kérések kiszolgálása egy adott portra a `9090` port helyett:

`cockpit-tls --port {{port}}`

- Súgó megjelenítése:

`cockpit-tls --help`
