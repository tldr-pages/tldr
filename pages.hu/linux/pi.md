# pi

> Archimédesz konstans Pi tizedesjegyeinek kiszámítása a parancssoron. További információ: <https://manned.org/pi>.

- A Pi archimédeszi konstans 100 tizedesjegyének megjelenítése:

`pi`

- Az archimédeszi Pi konstans meghatározott számú tizedesjegyének megjelenítése:

`pi {{number}}`

- Súgó megjelenítése:

`pi --help`

- Verzió megjelenítése:

`pi --version`

- Ajánlott olvasási értékek megjelenítése:

`pi --bibliography`
