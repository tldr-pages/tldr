# betterlockscreen

> Egyszerű, minimális záróképernyő. További információ: <https://github.com/pavanjadhaw/betterlockscreen>.

- A képernyő zárolása:

`betterlockscreen --lock`

- A képernyőzárolási képernyő hátterének módosítása:

`betterlockscreen -u {{path/to/image.png}}`

- A képernyő zárolása, néhány egyéni szöveg megjelenítésével:

`betterlockscreen -l pixel -t "{{custom lock screen text}}"`

- A képernyő zárolása, a monitor kikapcsolásának egyéni, másodpercben megadott időkorlátjával:

`betterlockscreen --off {{5}} -l`
