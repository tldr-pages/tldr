# qm help

> Egy adott parancs súgójának megjelenítése. További információ: <https://pve.proxmox.com/pve-docs/qm.1.html>.

- Egy adott parancs súgójának megjelenítése:

`qm help {{command}}`

- Egy adott parancs súgójának megjelenítése részletes információkkal:

`qm help {{command}} --verbose {{true|false}}`
