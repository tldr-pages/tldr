# checkupdates-aur

> Eszköz az Arch User Repository (AUR) függőben lévő frissítéseinek ellenőrzésére. További információ: <https://metacpan.org/dist/OS-CheckUpdates-AUR>.

- Az AUR csomagok függőben lévő frissítéseinek listája:

`checkupdates-aur`

- Az AUR csomagok függőben lévő frissítéseinek listázása hibakeresési módban:

`CHECKUPDATES_DEBUG=1 checkupdates-aur`

- Súgó megjelenítése:

`checkupdates-aur --help`
