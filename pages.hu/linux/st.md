# st

> Egy egyszerű terminál emulátor az X Window Systemhez. További információ: <https://st.suckless.org>.

- Nyisson meg egy terminált:

`st`

- Terminál megnyitása egy adott címmel:

`st -T {{title}}`

- Terminál megnyitása, egy adott parancs végrehajtása és a kimenet kiírása egy fájlba:

`st -o {{path/to/file}} -e {{command argument1 argument2}}`

- A betűméret növelése/csökkentése:

`Ctrl + Shift + {{Page Up|Page Down}}`

- Másolás/beillesztés a vágólapról:

`Ctrl + Shift + {{C|V}}`
