# photorec

> Törölt fájlok helyreállítására szolgáló eszköz. Ajánlott a helyreállított fájlokat egy olyan lemezre írni, amely elkülönül attól, amelyről a helyreállítás történik. További információ: <https://www.cgsecurity.org/wiki/PhotoRec>.

- Futtassa a PhotoRec programot egy adott eszközön:

`sudo photorec {{/dev/sdb}}`

- A PhotoRec futtatása egy lemezképen (`image.dd`):

`sudo photorec {{path/to/image.dd}}`
