# see

> Alias a `run-mailcap`'s view-hoz. Alias a `run-mailcap`'s action print-hez. További információ: <https://manned.org/see>.

- A see művelet bármely fájl (általában kép) megtekintésére használható az alapértelmezett mailcap explorerben:

`see {{filename}}`

- Használata a `run-mailcap`:

`run-mailcap --action=view {{filename}}`
