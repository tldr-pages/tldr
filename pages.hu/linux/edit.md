# edit

> A `run-mailcap`'s action edit alias. Eredetileg a `run-mailcap` a mime-type/file feldolgozására/szerkesztésére szolgál. További információ: <https://www.computerhope.com/unix/uedit.htm>.

- Az Edit művelet bármely fájl megtekintésére használható az alapértelmezett mailcap explorerben:

`edit {{filename}}`

- A `run-mailcap`:

`run-mailcap --action=edit {{filename}}`
