# dphys-swapfile

> A swap fájl kezelése Debian-alapú Linux rendszereken. További információ: <https://manpages.debian.org/latest/dphys-swapfile/dphys-swapfile.html>.

- A swap fájl letiltása:

`dphys-swapfile swapoff`

- A swap-fájl engedélyezése:

`dphys-swapfile swapon`

- Új swap fájl létrehozása:

`dphys-swapfile setup`
