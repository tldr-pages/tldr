# sensible-editor

> Nyissa meg az alapértelmezett szerkesztőt. További információ: <https://manned.org/sensible-editor>.

- Fájl megnyitása az alapértelmezett szerkesztőprogramban:

`sensible-editor {{path/to/file}}`

- Fájl megnyitása az alapértelmezett szerkesztőprogramban, a kurzor a fájl végén van:

`sensible-editor + {{path/to/file}}`

- Nyisson meg egy fájlt az alapértelmezett szerkesztőprogramban, a kurzorral a 10. sor elején:

`sensible-editor +10 {{path/to/file}}`

- Egyszerre 3 fájl megnyitása függőlegesen osztott szerkesztőablakokban:

`sensible-editor -O3 {{file_1}} {{file_2}} {{file_3}}`
