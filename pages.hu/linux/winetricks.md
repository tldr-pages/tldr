# winetricks

> A Wine virtuális Windows-környezetek kezelése. További információ: <https://wiki.winehq.org/Winetricks>.

- Indítson el egy grafikus telepítést az alapértelmezett Wine helyen:

`winetricks`

- Adjon meg egy egyéni Wine könyvtárat a Winetricks futtatásához:

`WINEPREFIX={{path/to/wine_directory}} winetricks`

- Windows DLL vagy komponens telepítése az alapértelmezett Wine könyvtárba:

`winetricks {{package}}`
