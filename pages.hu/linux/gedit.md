# gedit

> A GNOME Desktop projekt szövegszerkesztője. További információ: <https://help.gnome.org/users/gedit/stable/>.

- Szöveges fájl megnyitása:

`gedit {{path/to/file}}`

- Több szöveges fájl megnyitása:

`gedit {{file1 file2 ...}}`

- Szövegfájl megnyitása egy adott kódolással:

`gedit --encoding={{UTF-8}} {{path/to/file}}`

- A támogatott kódolások listájának megjelenítése:

`gedit --list-encodings`
