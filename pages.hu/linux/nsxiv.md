# nsxiv

> Neo Simple X Image Viewer. További információ: <https://nsxiv.codeberg.page/man>.

- Képek megnyitása:

`nsxiv {{path/to/file1 path/to/file2 ...}}`

- Képek megnyitása könyvtárakból képnézegetési módban:

`nsxiv {{path/to/directory1 path/to/directory2 ...}}`

- Könyvtárakban rekurzívan kereshet képeket a megtekintéshez:

`nsxiv -r {{path/to/directory1 path/to/directory2 ...}}`

- Az nsxiv kilépése:

`q`

- Átváltás miniatűr üzemmódba vagy kiválasztott kép megnyitása képmódban:

`Return`

- Képek továbbszámlálása képmódban:

`n`

- Képek visszaszámlálása képmódban:

`p`
