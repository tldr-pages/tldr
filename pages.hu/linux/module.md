# module

> A felhasználók környezetének módosítása a modul parancs segítségével. További információ: <https://lmod.readthedocs.io/en/latest/010_user.html>.

- Elérhető modulok megjelenítése:

`module avail`

- Modul keresése név alapján:

`module avail {{module_name}}`

- Modul betöltése:

`module load {{module_name}}`

- Betöltött modulok megjelenítése:

`module list`

- Egy adott betöltött modul kitöltése:

`module unload {{module_name}}`

- Az összes betöltött modul kitöltésének megszüntetése:

`module purge`
