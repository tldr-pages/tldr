# feedreader

> Egy GUI-s asztali RSS-kliens. További információ: <https://jangernert.github.io/FeedReader/>.

- Kiírja az olvasatlan cikkek számát:

`feedreader --unreadCount`

- Adjon hozzá egy követendő feed URL-címet:

`feedreader --addFeed={{feed_url}}`

- Egy adott cikk lekérdezése az URL-cím segítségével:

`feedreader --grabArticle={{article_url}}`

- Egy adott cikk összes képének letöltése:

`feedreader --url={{feed_url}} --grabImages={{article_path}}`

- Média lejátszása egy URL-címről:

`feedreader --playMedia={{article_url}}`
