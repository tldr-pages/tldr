# finch

> Konzol alapú moduláris üzenetküldő kliens. További információ: <https://developer.pidgin.im/wiki/Using%20Finch>.

- A finch elindítása:

`finch`

- Kilépés:

`alt + q or ctrl + c`

- Műveletek menü megjelenítése:

`alt + a`

- Ugrás az n-edik ablakra:

`alt + {{number_key}}`

- Jelenlegi ablak bezárása:

`alt + c`

- Az ablak mozgatásának megkezdése, a mozgatáshoz használd a nyílbillentyűket, nyomd meg az escape billentyűt, ha végeztél:

`alt + m`

- Egy ablak méretének megváltoztatásának megkezdése, a nyílbillentyűkkel változtassa meg a méretet, nyomja meg az escape billentyűt, ha kész:

`alt + r`
