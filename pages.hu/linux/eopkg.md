# eopkg

> Csomagkezelő a Solus számára. További információ: <https://getsol.us/articles/package-management/basics/en/>.

- Egy adott csomag telepítése:

`sudo eopkg install {{package_name}}`

- Az összes csomag frissítése:

`sudo eopkg upgrade`

- Csomagok keresése:

`sudo eopkg search {{search_term}}`
