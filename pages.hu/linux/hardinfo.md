# hardinfo

> Hardverinformációk megjelenítése a GUI ablakban. További információ: <https://github.com/lpereira/hardinfo>.

- Indítsa el a hardinfo programot:

`hardinfo`

- Jelentés nyomtatása a szabványos kimenetre:

`hardinfo -r`

- Jelentés mentése HTML fájlba:

`hardinfo -r -f html > hardinfo.html`
