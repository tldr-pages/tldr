# powertop

> Az akkumulátor energiafelhasználásának optimalizálása. További információ: <https://github.com/fenrus75/powertop>.

- Kalibrálja az energiafelhasználási méréseket:

`sudo powertop --calibrate`

- HTML energiafelhasználási jelentés készítése az aktuális könyvtárban:

`sudo powertop --html={{power_report.html}}`

- Optimális beállításokra hangolás:

`sudo powertop --auto-tune`
