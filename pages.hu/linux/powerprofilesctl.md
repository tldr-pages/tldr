# powerprofilesctl

> Teljesítményprofilok kezelése a D-Buson keresztül. További információ: <https://gitlab.freedesktop.org/hadess/power-profiles-daemon/>.

- A rendelkezésre álló teljesítményprofilok listája:

`powerprofilesctl list`

- Egy adott teljesítményprofil beállítása:

`powerprofilesctl set {{profile_name}}`
