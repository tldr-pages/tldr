# gnome-extensions

> A gnome kiterjesztések kezelése a terminálból. További információ: <https://wiki.gnome.org/Projects/GnomeShell/Extensions>.

- A verzió megjelenítése:

`gnome-extensions version`

- Az összes telepített bővítmény listázása:

`gnome-extensions list`

- Egy adott bővítményre vonatkozó információk megjelenítése:

`gnome-extensions info "{{extension_id}}"`

- Súgó megjelenítése egy alparancshoz (mint a `list`):

`gnome-extensions help {{subcommand}}`

- Egy adott bővítmény engedélyezése:

`gnome-extensions enable "{{extension_id}}"`

- Egy adott bővítmény letiltása:

`gnome-extension disable "{{extension_id}}"`

- Egy adott bővítmény eltávolítása:

`gnome-extension uninstall "{{extension_id}}"`
