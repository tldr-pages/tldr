# a2ensite

> Az Apache virtuális hoszt engedélyezése Debian-alapú operációs rendszereken. További információ: <https://manpages.debian.org/latest/apache2/a2ensite.8.en.html>.

- Virtuális állomás engedélyezése:

`sudo a2ensite {{virtual_host}}`

- Ne jelenítsen meg tájékoztató üzeneteket:

`sudo a2ensite --quiet {{virtual_host}}`
