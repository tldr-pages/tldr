# kpartx

> Eszköztérképek létrehozása partíciós táblákból. További információ: <https://manned.org/kpartx>.

- Partícióleképezések hozzáadása:

`kpartx -a {{whole_disk.img}}`

- Partícióleképezések törlése:

`kpartx -d {{whole_disk.img}}`

- Partícióleképezések listázása:

`kpartx -l {{whole_disk.img}}`
