# sam

> AWS Serverless Application Model (SAM) CLI. További információ: <https://github.com/awslabs/aws-sam-cli>.

- A szervermentes alkalmazás inicializálása:

`sam init`

- A szervermentes alkalmazás inicializálása egy adott futási idővel:

`sam init --runtime {{python3.7}}`

- Csomagoljon össze egy SAM alkalmazást:

`sam package`

- Lambda függvény kódjának összeállítása:

`sam build`

- A szerver nélküli alkalmazás helyi futtatása:

`sam local start-api`

- AWS SAM alkalmazás telepítése:

`sam deploy`
