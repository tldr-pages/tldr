# zile

> A Zile az Emacs szövegszerkesztő könnyített klónja. További információ: <https://www.gnu.org/software/zile/>.

- Indítson el egy puffert az ideiglenes jegyzetek számára, amelyek nem kerülnek elmentésre:

`zile`

- Nyisson meg egy fájlt:

`zile {{path/to/file}}`

- Egy fájl mentése:

`Ctrl + X, Ctrl + S`

- Kilépés:

`Ctrl + X, Ctrl + C`

- Egy fájl megnyitása egy megadott sorszámnál:

`zile +{{line_number}} {{path/to/file}}`

- A változtatások visszavonása:

`Ctrl + X, U`
