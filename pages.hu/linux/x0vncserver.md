# x0vncserver

> TigerVNC Server for X displays. További információ: <https://tigervnc.org/doc/x0vncserver.html>.

- VNC-kiszolgáló indítása jelszófájl használatával:

`x0vncserver -display {{:0}} -passwordfile {{path/to/file}}`

- VNC-kiszolgáló indítása egy adott port használatával:

`x0vncserver -display {{:0}} -rfbport {{port}}`
