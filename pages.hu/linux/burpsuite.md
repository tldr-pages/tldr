# burpsuite

> Egy GUI alapú alkalmazás, amelyet elsősorban webes alkalmazások behatolásának tesztelésére használnak. További információ: <https://portswigger.net/burp/documentation/desktop/getting-started/launch-from-command-line>.

- Indítsa el a Burp Suite-ot:

`burpsuite`

- A Burp Suite elindítása az alapértelmezett konfigurációval:

`burpsuite --use-defaults`

- Nyisson meg egy adott projektfájlt:

`burpsuite --project-file={{path/to/file}}`

- Egy adott konfigurációs fájl betöltése:

`burpsuite --config-file={{path/to/file}}`

- Indítás kiterjesztések nélkül:

`burpsuite --disable-extensions`
