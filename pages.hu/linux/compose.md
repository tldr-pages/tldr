# compose

> A `run-mailcap`'s action compose alias-a. Eredetileg a `run-mailcap` -t a mime-type/file-ra használják. További információ: <https://manned.org/compose>.

- A Compose művelet bármely meglévő vagy új fájl összeállítására használható az alapértelmezett mailcap szerkesztő eszközön:

`compose {{filename}}`

- A `run-mailcap`:

`run-mailcap --action=compose {{filename}}`
