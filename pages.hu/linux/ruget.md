# ruget

> A wget Rust nyelven írt alternatívája. További információ: <https://github.com/ksk001100/ruget>.

- Egy URL tartalmának letöltése egy fájlba:

`ruget {{https://example.com/file}}`

- Egy URL tartalmának letöltése egy megadott \[o\]utput fájlba:

`ruget --output {{file_name}} {{https://example.com/file}}`
