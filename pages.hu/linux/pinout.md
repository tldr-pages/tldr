# pinout

> Az aktuális Raspberry Pi GPIO pin-out információinak megtekintése a terminálon egy ASCII diagram segítségével. További információ: <https://www.raspberrypi.org/documentation/computers/os.html#gpio-pinout>.

- Tekintse meg az aktuális Raspberry Pi pinout információit és GPIO fejléc diagramját:

`pinout`

- Nyissa meg a https://pinout.xyz/ címet az alapértelmezett böngészőben:

`pinout -x`
