# getfacl

> Fájl-hozzáférésvezérlési listák lekérdezése. További információ: <https://manned.org/getfacl>.

- A fájlhozzáférés-ellenőrzési lista megjelenítése:

`getfacl {{path/to/file_or_directory}}`

- A fájlhozzáférés-ellenőrzési lista megjelenítése numerikus felhasználói és csoportazonosítókkal:

`getfacl -n {{path/to/file_or_directory}}`

- A fájlhozzáférés-ellenőrzési lista megjelenítése táblázatos kimeneti formátumban:

`getfacl -t {{path/to/file_or_directory}}`
