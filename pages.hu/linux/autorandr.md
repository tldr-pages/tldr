# autorandr

> Automatikusan megváltoztatja a képernyő elrendezését. További információ: <https://github.com/phillipberndt/autorandr>.

- Az aktuális képernyő elrendezés mentése:

`autorandr --save {{profile_name}}`

- A mentett profilok megjelenítése:

`autorandr`

- Az első felismert profil betöltése:

`autorandr --change`

- Egy adott profil betöltése:

`autorandr --load {{profile_name}}`

- Alapértelmezett profil beállítása:

`autorandr --default {{profile_name}}`
