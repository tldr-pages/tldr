# links

> Parancssori webböngésző. További információ: <http://links.twibright.com/>.

- Látogasson el egy weboldalra:

`links {{https://example.com}}`

- Korlátozások alkalmazása anonim fiókra:

`links -anonymous {{https://example.com}}`

- Engedélyezze a sütiket (`1` engedélyezéséhez):

`links -enable-cookies {{0|1}} {{https://example.com}}`

- Navigálás előre és hátrafelé az oldalon található linkek között:

`Up arrow key, Down arrow key`

- Egy oldalt előre és hátrafelé lépkedés:

`Left arrow key, Right arrow key`

- Kilépés:

`q then y`
