# retroarch

> A RetroArch egy frontend emulátorokhoz, játékmotorokhoz és médialejátszókhoz. A libretro API referencia implementációja. További információ: <https://github.com/libretro/RetroArch>.

- Indítás a menü üzemmódban:

`retroarch`

- Indítás teljes képernyős módban:

`retroarch --fullscreen`

- Az összes lefordított funkció listázása:

`retroarch --features`

- Konfigurációs fájl elérési útvonalának beállítása:

`retroarch --config={{path/to/config_file}}`

- Súgó megjelenítése:

`retroarch --help`

- Verzió megjelenítése:

`retroarch --version`
