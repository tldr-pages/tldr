# archlinux-java

> Egy segédszkript, amely funkciókat biztosít Java környezetek számára. További információ: <https://github.com/michaellass/archlinux-java-run>.

- A telepített Java-környezetek listája:

`archlinux-java status`

- Az alapértelmezett Java-környezet beállítása:

`archlinux-java set {{java_environment}}`

- Az alapértelmezett Java-környezet beállításának megszüntetése:

`archlinux-java unset`

- Az alapértelmezett Java-környezet automatikus beállítása:

`archlinux-java fix`
