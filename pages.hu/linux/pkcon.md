# pkcon

> Parancssori kliens a Discover és Gnome szoftverek által használt PackageKit konzolprogramhoz és az 'apt' alternatívája. További információ: <https://manned.org/pkcon>.

- Csomag telepítése:

`pkcon install {{package}}`

- Csomag eltávolítása:

`pkcon remove {{package}}`

- A csomagok gyorsítótárának frissítése:

`pkcon refresh`

- Csomagok frissítése:

`pkcon update`

- Egy adott csomag keresése:

`pkcon search {{package}}`

- Az összes elérhető csomag listája:

`pkcon get-packages`
