# backlight_control

> A linuxos gép háttérvilágításának vezérlése százalékos értékek segítségével. További információ: <https://github.com/Hendrikto/backlight_control>.

- A háttérvilágítás növelése/csökkentése egy adott százalékos értékkel:

`backlight_control {{+|-}}{{5}}`

- A háttérvilágítás erősségének beállítása egy adott százalékszámra:

`backlight_control {{90}}`

- A súgó kinyomtatása:

`backlight_control`
