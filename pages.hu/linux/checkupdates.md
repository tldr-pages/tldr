# checkupdates

> Az Arch Linux függőben lévő frissítéseinek ellenőrzésére szolgáló eszköz. További információ: <https://man.archlinux.org/man/checkupdates.8>.

- Függő frissítések listája:

`checkupdates`

- Függő frissítések listázása és a csomagok letöltése a pacman gyorsítótárba:

`checkupdates --download`

- Függő frissítések listázása egy adott pacman adatbázis segítségével:

`CHECKUPDATES_DB={{path/to/directory}} checkupdates`

- Súgó megjelenítése:

`checkupdates --help`
