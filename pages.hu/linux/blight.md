# blight

> A kijelző fényerejének módosítására szolgáló segédprogram. További információ: <https://github.com/gutjuri/blight>.

- Állítsa a kijelző fényerejét 50%-ra:

`blight set {{50}} -r`

- A kijelző aktuális fényerejének megjelenítése:

`blight show`

- A kijelző maximális fényerejének kinyomtatása:

`blight max`

- A kijelző fényerejének növelése %-ban:

`blight inc {{number}} -r`

- A kijelző fényerejének csökkentése belső egységekkel:

`blight dec {{number}}`
