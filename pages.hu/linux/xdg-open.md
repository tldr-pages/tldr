# xdg-open

> Megnyit egy fájlt vagy URL-címet a felhasználó által preferált alkalmazásban. További információ: <https://portland.freedesktop.org/doc/xdg-open.html>.

- Megnyitja az aktuális könyvtárat az alapértelmezett fájlkeresőben:

`xdg-open .`

- Megnyit egy URL-címet az alapértelmezett böngészőben:

`xdg-open {{https://example.com}}`

- Kép megnyitása az alapértelmezett képnézegetőben:

`xdg-open {{path/to/image}}`

- PDF megnyitása az alapértelmezett PDF-megjelenítőben:

`xdg-open {{path/to/pdf}}`

- Súgó megjelenítése:

`xdg-open --help`
