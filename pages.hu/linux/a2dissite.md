# a2dissite

> Az Apache virtuális hoszt letiltása Debian-alapú operációs rendszereken. További információ: <https://manpages.debian.org/latest/apache2/a2dissite.8.en.html>.

- Virtuális állomás letiltása:

`sudo a2dissite {{virtual_host}}`

- Ne jelenítsen meg tájékoztató üzeneteket:

`sudo a2dissite --quiet {{virtual_host}}`
