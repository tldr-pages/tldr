# toolbox rm

> Egy vagy több `toolbox` konténer eltávolítása. Lásd még: `toolbox rmi`. További információ: <https://manned.org/toolbox-rm.1>.

- Egy eszköztároló konténer eltávolítása:

`toolbox rm {{container_name}}`

- Az összes `toolbox` konténer eltávolítása:

`toolbox rm --all`

- Az éppen aktív `toolbox` konténer eltávolításának kikényszerítése:

`toolbox rm --force {{container_name}}`
