# toolbox enter

> Adja meg a `toolbox` konténert interaktív használatra. Lásd még: `toolbox run`. További információ: <https://manned.org/toolbox-enter.1>.

- Adjon meg egy `toolbox` konténert egy adott disztribúció alapértelmezett képét használva:

`toolbox enter --distro {{distribution}}`

- Adjon meg egy `toolbox` konténert az aktuális disztribúció egy adott kiadásának alapértelmezett képét használva:

`toolbox enter --release {{release}}`

- Adjon meg egy toolbox konténert a Fedora 36 alapértelmezett képét használva:

`toolbox enter --distro {{fedora}} --release {{f36}}`
