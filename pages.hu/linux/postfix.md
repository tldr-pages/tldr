# postfix

> Postfix mail transfer agent (MTA) vezérlő program. Lásd még: `dovecot`, egy postai kézbesítő ügynök (MDA), amely integrálódik a Postfix-szel. További információ: <http://www.postfix.org>.

- Ellenőrizze a konfigurációt:

`sudo postfix check`

- Ellenőrizze a Postfix démon állapotát:

`sudo postfix status`

- Indítsa el a Postfixet:

`sudo postfix start`

- A Postfix kegyes leállítása:

`sudo postfix stop`

- A levelezési várólista ürítése:

`sudo postfix flush`

- A konfigurációs fájlok újratöltése:

`sudo postfix reload`
