# termusic

> Egy Rust nyelven írt terminál zenelejátszó, amely vim-szerű billentyűkötéseket használ. További információ: <https://github.com/tramhao/termusic>.

- Termusic megnyitása egy adott könyvtárban. (Állandóan beállítható a `~/.config/termusic/config.toml`):

`termusic {path/to/directory}`

- Az albumborító megjelenítésének letiltása egy adott fájlhoz:

`termusic -c {path/to/music_file}`

- A termusic használati információinak megtekintése:

`termusic --help`
