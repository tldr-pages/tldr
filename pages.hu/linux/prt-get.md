# prt-get

> A CRUX csomagkezelő. További információ: <https://crux.nu/doc/prt-get%20-%20User%20Manual.html>.

- Csomag telepítése:

`prt-get install {{package_name}}`

- Csomag telepítése függőségkezeléssel:

`prt-get depinst {{package_name}}`

- Egy csomag manuális frissítése:

`prt-get upgrade {{package_name}}`

- Egy csomag eltávolítása:

`prt-get remove {{package_name}}`

- A rendszer frissítése a helyi portfáról:

`prt-get sysup`

- Keresés a portfában:

`prt-get search {{package_name}}`

- Egy fájl keresése egy csomagban:

`prt-get fsearch {{file}}`
