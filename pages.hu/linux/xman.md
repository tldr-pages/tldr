# xman

> Kézi lapnézegető az X Window System számára. További információ: <https://manned.org/xman>.

- Az xman indítása háromgombos ablakban:

`xman`

- Megnyitja az adott fájlban tárolt kézi oldal kimenetét:

`xman -helpfile {{filename}}`

- Mind a kézi oldal, mind a könyvtár megjelenítése:

`xman -bothshown`
