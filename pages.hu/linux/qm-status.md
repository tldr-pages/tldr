# qm status

> A virtuális gép állapotának megjelenítése. További információk: <https://pve.proxmox.com/pve-docs/qm.1.html>.

- Egy adott virtuális gép állapotának megjelenítése:

`qm status {{vm_id}}`

- Egy adott virtuális gép részletes állapotának megjelenítése:

`qm status --verbose {{true}} {{vm_id}}`
