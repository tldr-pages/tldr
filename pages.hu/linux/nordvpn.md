# nordvpn

> A NordVPN parancssori interfésze. További információ: <https://nordvpn.com/download/linux/>.

- Interaktív bejelentkezés egy NordVPN-fiókba:

`nordvpn login`

- A kapcsolat állapotának megjelenítése:

`nordvpn status`

- Csatlakozás a legközelebbi NordVPN-kiszolgálóhoz:

`nordvpn connect`

- Az összes elérhető ország listája:

`nordvpn countries`

- Csatlakozás egy adott országban található NordVPN-kiszolgálóhoz:

`nordvpn connect {{Germany}}`

- Csatlakozás egy adott országban és városban található NordVPN-kiszolgálóhoz:

`nordvpn connect {{Germany}} {{Berlin}}`

- Automatikus csatlakozási opció beállítása:

`nordvpn set autoconnect on`
