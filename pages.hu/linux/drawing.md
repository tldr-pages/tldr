# drawing

> Ingyenes alap raszteres képszerkesztő program GNOME asztali környezetben. További információk: <https://maoschanz.github.io/drawing/>.

- Rajzolás indítása:

`drawing`

- Meghatározott fájlok megnyitása:

`drawing {{path/to/image1 path/to/image2 ...}}`

- Meghatározott fájlok megnyitása új ablakban:

`drawing --new-window {{path/to/image1 path/to/image2 ...}}`
