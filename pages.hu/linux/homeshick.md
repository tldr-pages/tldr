# homeshick

> Git dotfiles szinkronizálása. További információ: <https://github.com/andsens/homeshick/wiki>.

- Új kastély létrehozása:

`homeshick generate {{castle_name}}`

- Adjon hozzá egy fájlt a kastélyához:

`homeshick track {{castle_name}} {{path/to/file}}`

- Menjen egy kastélyhoz:

`homeshick cd {{castle_name}}`

- Egy kastély klónozása:

`homeshick clone {{github_username}}/{{repository_name}}`

- Egy kastély összes fájljának szinkronizálása:

`homeshick link {{castle_name}}`
