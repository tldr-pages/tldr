# ubuntu-security-status

> A telepített Ubuntu csomagok biztonsági támogatására vonatkozó információk megjelenítése. További információ: <https://git.launchpad.net/ubuntu/+source/update-manager/tree/ubuntu-security-status>.

- A nem támogatott csomagok számának megjelenítése:

`ubuntu-security-status`

- A már nem letölthető csomagok listázása:

`ubuntu-security-status --unavailable`

- Harmadik féltől származó csomagok listázása:

`ubuntu-security-status --thirdparty`
