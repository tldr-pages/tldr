# jobs

> BASH beépített program az aktuális héj által indított folyamatokról szóló információk megtekintésére. További információ: <https://manned.org/jobs>.

- Az aktuális shell által indított munkák megtekintése:

`jobs`

- A munkák és folyamatazonosítóik listázása:

`jobs -l`

- A megváltozott státuszú munkákról szóló információk megjelenítése:

`jobs -n`

- A folyamatcsoport vezetőjének folyamatazonosítójának megjelenítése:

`jobs -p`

- Folyamatban lévő folyamatok megjelenítése:

`jobs -r`

- Leállított folyamatok megjelenítése:

`jobs -s`
