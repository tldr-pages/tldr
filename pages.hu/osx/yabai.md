# yabai

> Bináris térfelosztáson alapuló, csempézett ablakkezelő macOS-hez. További információ: <https://github.com/koekeishiya/yabai>.

- Állítsa be az elrendezést bsp-re:

`yabai -m config layout {{bsp}}`

- Állítsa be az ablakrést 10pt-re:

`yabai -m config window_gap {{10}}`

- Engedélyezze az átlátszatlanságot:

`yabai -m config window_opacity on`

- Kapcsolja ki az ablakárnyékot:

`yabai -m config window_shadow off`

- Állapotsor engedélyezése:

`yabai -m config status_bar on`
