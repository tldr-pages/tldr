# shortcuts

> A parancsikonok kezelése a parancssorból a `Shortcuts` alkalmazás helyett. További információ: <https://support.apple.com/guide/shortcuts-mac/run-shortcuts-from-the-command-line-apd455c82f02/mac>.

- A megadott parancsikon futtatása (`Count holidays`):

`shortcuts run "{{Count holidays}}"`

- Az összes parancsikon kinyomtatása:

`shortcuts list`

- Az összes parancsikon mappa kinyomtatása:

`shortcuts list --folders`

- A megadott parancsikon (`Count holidays`) megnyitása a parancsikonok szerkesztőjében:

`shortcuts view "{{Count holidays}}"`
