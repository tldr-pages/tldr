# port

> Csomagkezelő macOS-hez. További információ: <https://www.macports.org>.

- Csomag keresése:

`port search {{search_term}}`

- Csomag telepítése:

`sudo port install {{package_name}}`

- Telepített csomagok listája:

`port installed`

- Port frissítése és az elérhető csomagok legfrissebb listájának lekérése:

`sudo port selfupdate`

- Elavult csomagok frissítése:

`sudo port upgrade outdated`

- A telepített csomagok régi verzióinak eltávolítása:

`sudo port uninstall inactive`
