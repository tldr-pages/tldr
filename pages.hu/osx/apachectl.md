# apachectl

> Apache HTTP Server vezérlőfelület macOS-hez. További információ: <https://www.unix.com/man-page/osx/8/apachectl/>.

- Indítsa el a `org.apache.httpd` launchd feladatot:

`apachectl start`

- Állítsa le a launchd feladatot:

`apachectl stop`

- Állítsa le, majd indítsa el a launchd feladatot:

`apachectl restart`
