# wifi-password

> A Wi-Fi jelszavának lekérdezése. További információ: <https://github.com/rauchg/wifi-password>.

- Szerezze be az aktuálisan bejelentkezett Wi-Fi jelszavát:

`wifi-password`

- Egy adott SSID-vel rendelkező Wi-Fi jelszavának lekérdezése:

`wifi-password {{ssid}}`

- Kimenetként csak a jelszó kinyomtatása:

`wifi-password -q`
