# history

> Display terminal command line history.
> More information: <https://ss64.com/bash/history.html>.

- Display the history list with line numbers:

`history`

- Clear the history list:

`history -c`

- Append history lines entered since the beginning of the current Bash session:

`history -a`

- Append the history lines not already read from the history file to the current history list:

`history -n`

- Read the current history file and append its contents to the history list:

`history -r`
