# xcrun

> Run or locate development tools and properties.

- Get help:

`xcrun --help`

- Get version:

`xcrun --version`

- Find and run a tool from the active developer directory:

`xcrun {{tool}} {{args}}`

- Show verbose output:

`xcrun --verbose`

- Find tool for given SDK:

`xcrun --sdk {{sdk_name}}`

- Find a tool for given toolchain:

`xcrun --toolchain {{name}}`
