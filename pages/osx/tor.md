# command-name

> Open socks listener which connects to the tor network.
> More information: https://2019.www.torproject.org/docs/documentation.html.en
> For a new developer portal, see https://blog.torproject.org/meet-new-torprojectorg

- Open socks listener on 127.0.0.1:9050 (default)

`tor`

- Display tor version and exit.

`tor -v`

- Specify a new configuration file to contain Tor configuration.

`tor -f {{file}}`

- Verify configuration.

`tor --verify-config`
