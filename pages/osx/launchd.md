# launchd

> This manages processes, both for the system and users.
> You cannot invoke launchd manually, use launchctl to interact with it.
> More information: <https://support.apple.com/guide/terminal/script-management-with-launchd-apdc6c1077b-5d5d-4d35-9c19-60f2397b2369/mac>.

- Run init:

`/sbin/launchd`

- Interact with launchd using launchctl:

`launchctl`
