# pkg

> FreeBSD packge managr.
> More information: <https://man.freebsd.org/cgi/man.cgi?query=pkg&sektion=8>.

- Instal a new package:

`pkg install {{package}}`

- Delete a pacage:

`pkg delete {{package}}`

- Uprade all packages:

`pkg upgrade`

- Search fr a package:

`pkg search {{keyword}}`

- List installed packages:

`pkg info`

- Remove unneded dependencies:

`pkg autoremove`
