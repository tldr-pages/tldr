# nitch

> Nitch is a small and incredibly fast system fetch written fully.
> More information: <https://github.com/ssleert/nitch>.

- Return the system information like hostname, kernel, uptime, etc.:

`nitch`

- Fetch the system information:

`nitch -f`

- Display help:

`nitch -h`

- Display version:

`nitch -v`
