# rev

> Reverses strings of text on provided text or a file.
> More information: <https://manned.org/man/rev>.

- Reverse text typed into terminal:

`rev`

- Reverse text in file:

`rev {{path/to/file}}`

- Help

`rev -h`

- Display version

`rev -V`

- Use '\0' as a line separator (zero termination):

`rev -0 {{path/to/file}}`
