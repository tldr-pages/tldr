# fwupdmgr

> Update the firmware of local devices.

- See which devices fwupdmgr can update firmware for:

`fwupdmgr get-devices`

- Refresh update metadata:

`fwupdmgr refresh`

- Check which devices have available updates:

`fwupdmgr get-updates`

- Install all available updates:

`sudo fwupdmgr update`
