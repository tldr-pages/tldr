# wl-copy

> Wayland clipboard manipulation tool.
> See also: `wl-paste`.
> More information: <https://github.com/bugaevc/wl-clipboard>.

- Copy text to the clipboard:

`wl-copy {{text}}`

- Copy the output of a command to the clipboard:

`{{command}} | wl-copy`

- Clear the clipboard:

`wl-copy --clear`
