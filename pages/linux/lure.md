# lure

> A distro-agnostic build system and user repository for Linux.
> More information: <https://github.com/lure-sh/lure/blob/master/docs/usage.md>.

- Install a package:

`lure in {{package}}`

- Remove a package:

`lure rm {{package}}`

- Update packages:

`lure up`

- List all available packages:

`lure ls`
