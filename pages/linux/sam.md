# sam

> AWS Serverless Application Model (SAM) CLI.

- Initialize a serverless application:

`sam init`

- Initialize a serverless application with a specific runtime:

`sam init --runtime python3.7`

- Package a SAM application (Alias of *aws cloudformation package*):

`sam package`

- Build your Lambda function code:

`sam build`

- Run your serverless application locally:

`sam local start-api`

- Deploy an AWS SAM application (Alias of *aws cloudformation deploy*):

`sam deploy`
