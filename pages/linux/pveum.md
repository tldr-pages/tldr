# pveum

> Manage proxmox users.
> More information: <https://pve.proxmox.com/pve-docs/pveum.1.html>.

- List users:

`pveum {{[u|user]}} {{[l|list]}}`

- Add a user:

`pveum {{[u|user]}} {{[a|add]}} {{username}}@pve`

- Change user password:

`pveum {{[pa|passwd]}} {{[username}}@pve`

- Delete a user:

`pveum {{[u|user]}} {{[d|delete]}} {{username}}@pve`
