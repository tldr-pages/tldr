# warpd

> A modal keyboard driven pointer manipulation program.

> More information: <https://github.com/rvaiya/warpd/blob/master/man.md>.

- Run warpd in normal mode:

`warpd --normal`

- Moving cursor left:

`h`

- Moving cursor down:

`j`

- Moving cursor up:

`k`

- Moving cursor right:

`l`

- Emulate left click:

`m`

- Run warpd in hint mode:

`warpd --hint`
