# warpd

> A modal keyboard driven pointer manipulation program.

- Run warpd in normal mode:

`warpd --normal`

- Moving cursor left/down/up/right respectively:

`h`, `j`, `k`, and `l`

- Emulate left/middle/right click respectively:

`m`, `,` and `.`

- Exit:

`<Esc>`

- Emulate drag:

`v`

- Run warpd in hint mode:

`warpd --hint`
