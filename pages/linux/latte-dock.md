# latte-dock

> Replacement dock for Plasma desktops, providing an elegant and intuitive experience for your tasks and plasmoids.
> More information: <https://github.com/KDE/latte-dock>.

- Clear qml cache:

`latte-dock --cc`

- Import and load default layout on startup:

`latte-dock --default-layout`

- Load specific layout on startup:

`latte-dock --layout {{layout_name}}`

- Import and load a layout:

`latte-dock --import-layout  {{path/to/file}}`
