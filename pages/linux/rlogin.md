# rlogin

> Log in to a remote host.
> More information: <https://www.gnu.org/software/inetutils/manual/inetutils.html#rlogin-invocation>.

- Log in to a remote host:

`rlogin {{remote_host}}`

- Log in to a remote host with a specific username:

`rlogin {{[-l|--user]}} {{username}} {{remote_host}}`
