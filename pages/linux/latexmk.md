# latexmk

> Automated LaTeX document generation

 - Compile .tex file to pdf:

`latexmk -pdf file`

 - Force latexmk to generate document despite errors:

`latexmk -f file`

 - Clean up files generated by latexmk except dvi, postscript and pdf:

`latexmk -c`

 - Clean up files generated by latexmk including dvi, postscript and pdf:

`latexmk -C`
