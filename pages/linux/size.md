# size

> Displays the sizes of sections inside binary files.
> More information: <https://www.ibm.com/docs/en/aix/7.2?topic=s-size-command>.

- Display the size of sections of object or executable file:

`size {{file}}`

- To display the size of an executable file in [o]ctal:

`size -o {{file}}`

- To display the size of an executable file in [d]ecimal:

`size -d {{file}}`

- To display the size of an executable file in he[x]adecimal:

`size -x {{file}}`
