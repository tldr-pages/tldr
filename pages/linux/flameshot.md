# flameshot 

> Screenshot utility with a gui interface.
> Supports basic image editing, such as; text,shapes,colors,imgur.
> More information: <hhttps://flameshot.js.org/>.

- Launch flameshot in gui mode:

`flameshot launcher`

- Click and drag screenshot

`flameshot gui`

- Full screen screenshot:
  
`flameshot full`

- Set save path for flameshot:

`flameshot full --path ~/Pictures`

- Delay the screenshot for N milliseconds and output to clipboard:

`flameshot full --delay 2000 --clipboard`
