# slabtop

> Display kernel slab cache information in real time.
> see also: `top`, `htop`, `atop`.
> More information: <https://manned.org/slabtop>.

- Start `slabtop`:

`slabtop`

- Sort by cache size:

`slabtop {{[-s|--sort-override]}} {{cache_size}}`

- Sort by number of objects:

`slabtop {{[-s|--sort-override]}} {{number_objects}}`

- Sort by object size:

`slabtop {{[-s|--sort-override]}} {{size_object}}`

- Display once and then exit:

`slabtop {{[-o|--once]}}`

- Display help:

`slabtop {{[-h|--help]}}`
