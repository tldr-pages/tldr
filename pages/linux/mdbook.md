# mdbook

> Create online books by writing makrdown files.
> More information: <https://rust-lang.github.io/mdBook/>.

- Init mdbook files in  dir, default current dir:

`mdbook init {{dir}}`

- Clean build dir:

`mdbook clean`

- Serve a book at `http://localhost:3000`, auto build when file changes:

`mdbook serve`

- Watch markdown  files, auto build when file changes:

`mdbook watch`
