# evince

> View documents such as PDF, PostScript, and TIFF using the GNOME Document Viewer.
> More information: <https://manned.org/evince>.

- Open a PDF file:

`evince {{path/to/file.pdf}}`

- Open multiple documents:

`evince {{file1.pdf file2.pdf}}`

- Start in fullscreen mode:

`evince --fullscreen {{path/to/file.pdf}}`

- Start in presentation mode:

`evince --presentation {{path/to/file.pdf}}`

- Open a specific page number:

`evince --page-label={{5}} {{path/to/file.pdf}}`
