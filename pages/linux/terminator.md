# terminator

> Arrange multiple GNOME terminals in one window.
> More information: <https://gnome-terminator.org/>.

- Start `terminator` window:

`terminator`

- Start with a fullscreen window:

`terminator -f`

- Split terminals horizontally:

`<Ctrl Shift o>`

- Split terminals vertically:

`<Ctrl Shift e>`

- Open new tab:

`<Ctrl Shift t>`
