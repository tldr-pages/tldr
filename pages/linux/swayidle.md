# swayidle

> Idle management daemon for Wayland.
> Configuration options are documented in the man page.
> More information: <https://github.com/swaywm/swayidle/blob/master/swayidle.1.scd>.

- Listen for idle activity using the config in `$XDG_CONFIG_HOME/swayidle/config` or `$HOME/swayidle/config`:

`swayidle`

- Specify an alternative path to the config file:

`swayidle -C {{path/to/file}}`
