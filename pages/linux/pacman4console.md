# pacman4console

> A text-based console game inspired by the original Pacman.
> You play as Pacman (C); avoid the ghosts (&'s); collect all the pellets (*'s and .'s).

- Start a game (at level 1 by default):

`pacman4console`

- Start a game on a certain level (there are nine official levels):

`pacman4console --level={{#}}`

- Start the pacman4console editor with a level text-file:

`pacmanforconsoleedit {{level_file}}`

- Start a non-standard level (like a custom one):

`pacman4console --level={{level_file}}`
