# upower

> System Utility to provide battery information and statistics.
> More information: <https://upower.freedesktop.org/>.

- Dump battery information:

`upower --dump`

- List connected battery-using devices:

`upower --enumerate`

- Listen for changes in battery status:

`upower --monitor`

- Listen for changes in battery status - More info:

`upower --monitor-detail`

- Show version of software:

`upower --version`
