# nsxiv

> Neo Simple X Image Viewer.
> More information: <https://nsxiv.codeberg.page>.

- Open images:

`nsxiv {{path/to/file1 path/to/file2 ...}}`

- Open images from a directories in image mode:

`nsxiv {{path/to/directory1 path/to/directory2 ...}}`

- Search the given directories recursively for images to view:

`nsxiv -r {{path/to/directory1 path/to/directory2 ...}}`

- Quit nsxiv:

`q`

- Switch to thumbnail mode / open selected image in image mode:

`Return`

- Count images forward in Image mode:

`n, Space`

- Go count images backward(In Image mode):

`p, Backspace`
