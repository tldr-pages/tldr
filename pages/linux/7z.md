# 7z

> A file archiver with highest compression ratio
> More information: <https://7-zip.org/>.

- Add all files/directories to a specific archive with default settings:

`7z a {{path/to/compressed.zip}} {{path/to/file_or_directory1 path/to/file_or_directory2 ...}}`

- Add all files/directories to a specific archive with max compression:

`7z a -t7z -m0=lzma -mx=9 -mfb=64 -md=32m -ms=on {{path/to/compressed.zip}} {{path/to/file_or_directory1 path/to/file_or_directory2 ...}}`

- Add all files/directories to a specific archive with no compression:

`7z a -m0=copy {{path/to/compressed.zip}} {{path/to/file_or_directory1 path/to/file_or_directory2 ...}}`
