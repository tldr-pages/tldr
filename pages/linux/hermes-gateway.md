# hermes gateway

> Manage messaging gateway services for Telegram, Discord, Slack, and other platforms.
> More information: <https://hermes-agent.nousresearch.com/docs>.

- Run the messaging gateway:

`hermes gateway`

- Install the gateway as a background service:

`hermes gateway install`

- Uninstall the gateway service:

`hermes gateway uninstall`

- Restart the gateway service:

`hermes gateway restart`

- Check gateway service status:

`hermes gateway status`
