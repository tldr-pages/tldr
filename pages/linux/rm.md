# rm

> Copy files and directories.
> More information: <https://www.gnu.org/software/coreutils/rm>.

- Delete each given file:

`rm file.txt`

- Delete a file, asking for confirmation before deleting:

`rm -i file.txt`

- Delete a file, without asking for confirmation before deleting:

`rm -f file.txt`

- Delete the listed directories if they are empty:

`rm -d directory`

- Delete directories and subdirectories recursively:

`rm -rf directory`

- Delete files and show the removal result:

`rm -v file.txt`
