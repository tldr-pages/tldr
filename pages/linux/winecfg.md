# winecfg

> Configure a Wine prefix (drives, libraries, graphics, and the emulated Windows version).
> More information: <https://manned.org/winecfg>.

- Open the graphical Wine configuration tool:

`winecfg`

- Display the Windows [v]ersion the prefix currently emulates:

`winecfg /v`

- Set the Windows [v]ersion the prefix emulates:

`winecfg /v {{win11|win10|win7|winxp}}`

- Display help:

`winecfg /?`
