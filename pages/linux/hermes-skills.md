# hermes skills

> Search, install, configure, and manage Hermes skills.
> More information: <https://hermes-agent.nousresearch.com/docs>.

- List available skills:

`hermes skills list`

- Search for a skill:

`hermes skills search {{query}}`

- Install a skill:

`hermes skills install {{skill_name}}`

- Remove a skill:

`hermes skills remove {{skill_name}}`

- View a skill's content:

`hermes skills view {{skill_name}}`
