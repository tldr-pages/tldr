# zplug

> Zplug -  is a next-generation plugin manager for zsh.

- Check if the remote repositories are up to date:

`zplug status`

- Return true if all packages are installed, false otherwise:

`zplug check`

- Remove repositories which are no longer managed:

`zplug clean`

- Source installed plugins and add installed commands to $PATH:

`zplug load`

- List installed packages (more specifically, view the associative array $zplugs):

`zplug list`
