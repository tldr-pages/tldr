# ls

> List directory contents. List information about the FILEs.
> More information: <https://manned.org/ls>.

- List all files including those starting with '.' (hidden files):

`ls -a`

- List files in the long listing format:

`ls -l`

- List files and output their respective sizes when used with -l and -s:

`ls -lh`

- List files in reverse order sorting:

`ls -r`

- List files sorted by time, newest first:

`ls -t`

- List files sorted by time, newest first, with reverse order sorting, displaying sizes in human readable format, using long listing format and including all hidden files:

`ls -lahrt`
