# flash

> Flash Cards In Your Terminal.
> More information: <https://github.com/tallguyjenks/fla.sh>.

- Open a menu of your available flashcard decks for selection:

`flash`

- Current program version:

`flash -v`

- Get information about the flashcard system

`flash -i`

- Get a help menu of commands that flash can run

`flash -h`

- Change the previewer from default `bat` to common binary of `cat`

`flash -p {{cat}}`
