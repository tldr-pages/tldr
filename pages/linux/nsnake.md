# nsnake

> Snake game in the terminal.
> More information: <https://github.com/alexdantas/nsnake/#controls>.

- Start a snake game:

`nsnake`

- Navigate the snake:

`{{<ArrowUp>|<ArrowDown>|<ArrowLeft>|<ArrowRight>}}`

- Pause/unpause the game:

`<p>`

- Quit the game:

`<q>`

- Display help during the game:

`<h>`
