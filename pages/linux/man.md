# man

> the system's manual pager

- Display the manual page for the program:

`man {{program_name}}`

- Display specific entry of a manual page for a program:

`man {{number_of_entry}} {{program_name}}`

- Display all manual pages for a program:

`man -a {{name_of_program}}`

- Search manual pages containing a keyword:

`man -k {{keyword}}`

- Search manpages about a keyword:

`man -f {{keyword}}`
