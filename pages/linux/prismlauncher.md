# prismlauncher

> Prism Launcher is a launcher and instance manager for Minecraft.
> More information: <https://prismlauncher.org>.

- Launch Prism Launcher:

`prismlauncher`

- Launch a specific instance:

`prismlauncher --launch {{"Instance ID"}}`

- Join a server on launching a specific instance:

`prismlauncher --launch {{"Instance ID"}} --server {{"server.ip"}}`

- Join a world on launching a specific instance:

`prismlauncher --launch {{"Instance ID"}} --world {{"world name"}}`

- View the configuration window for a specific instance:

`prismlauncher --show {{"Instance ID"}}`

- Use a different root directory for Prism Launcher:

`prismlauncher --dir {{path/to/directory/}}`
