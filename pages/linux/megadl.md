# megadl

> Download files from mega.nz.
> Part of the megatools suite.
> More information: <https://megatools.megous.com/>.

- Download files from a mega.nz link into the current directory:

`megadl {{https://mega.nz/...}}`

- Interactively choose which files to download:

`megadl --choose-files {{https://mega.nz/...}}`

- Optionally limit speed in kb/s and specify a path:

`megadl --choose-files --limit-speed {{speed}} --path {{/path/to/file_or_directory}} {{https://mega.nz/...}}`
