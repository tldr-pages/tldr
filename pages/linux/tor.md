# tor

> Enable anonymous communication through the Tor network.
> More information: <https://manned.org/tor>.

- Connect to the Tor network:

`tor`

- View Tor configuration:

`tor --config`

- Check Tor status:

`tor --status`

- Run as client only:

`tor --client`

- Run as relay:

`tor --relay`

- Run as bridge:

`tor --bridge`

- Run as a hidden service:

`tor --hidden-service`
