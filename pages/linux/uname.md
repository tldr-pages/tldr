# uname 

> prints system information about machine and operating system it is run on.
> More information on : https://www.gnu.org/software/coreutils/manual/html_node/uname-invocation.html#uname-invocation

- Print all information, except for processor and hardware platform if unknown

`uname -a`

- Print the kernel name. Note that it might be the same as operating system name or might differ

`uname -s`

- Print the network node hostname

`uname -n`

- Print the kernel version

`uname -v`

- Print the processor type (instruction set architecture) (non-portable)

`uname -p`

- Print the name of the operating system 

`uname -o`
