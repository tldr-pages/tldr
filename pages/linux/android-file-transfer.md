# android-file-transfer

> Transfer files between Android devices and your computer using MTP.
> More information: <https://www.archlinux.org/packages/community/x86_64/android-file-transfer/>.

- List connected Android devices:

`android-file-transfer`

- Open the GUI file manager:

`android-file-transfer --device`

- Mount device (if supported by backend):

`android-file-transfer mount {{mount_point}}`

- Unmount device:

`android-file-transfer unmount {{mount_point}}`
