# kwrite

> Text editor of the KDE Desktop project.
> More information: <https://apps.kde.org/kwrite/>.

- Open a text file:

`kwrite {{path/to/file}}`

- Open multiple text files:

`kwrite {{file1 file2 ...}}`

- Open a text file with a specific encoding:

`kwrite --encoding={{UTF-8}} {{path/to/file}}`

- Navigate to a specific line and column in a text document.:

`kwrite -l {{line_number}} -c {{column_number}} {{path/to/file}}`
