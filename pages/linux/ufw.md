# ufw

> Uncomplicated Firewall.
> Frontend for iptables aiming to make configuration of a firewall easier.

- Enable ufw:

`ufw enable`

- Disable ufw:

`ufw disable`

- Add ufw allow rule:

`ufw allow {{port}} {{service_name}}`

- Add ufw deny rule:

`ufw deny {{port}} {{service_name}}`

- Show ufw rules:

`ufw status`
