# xbanish

> Hide the X11 cursor when a key is pressed and turn it back on when the mouse is used.
> More information: <https://manned.org/xbanish>.

- Hide the cursor on key press:

`xbanish`

- Start xbanish so that the cursor is always hidden, ignoring key presses:

`xbanish -a`

- Automatically hide the cursor after a certain amount of time of inactivity:

`xbanish -t {{seconds}}`
