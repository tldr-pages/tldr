# xbanish

> X utility that hides the cursor when a key is pressed, the cursor appears again when there is clicking or movement.
> More information: <http://manned.org/xbanish>.

- Hide the cursor on key press:

`xbanish`

- Start xbanish so that the cursor is always hidden, ignoring key presses:

`xbanish -a`

- Automatically hide the cursor after a certain amount of time of inactivity:

`xbanish -t {{seconds}}`
