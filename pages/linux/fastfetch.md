# fastfetch

> A neofetch-like tool but, faster & prettier.
> More information: <https://github.com/LinusDierheimer/fastfetch>.

- Display system information:

`fastfetch`

- Set the structure of the fetch:

`fastfetch --structure {{structure}}`

- Load a custom config file:

`fastfetch --load-config {{path/to/config_file}}`

- Set the logo to use:

`fastfetch --logo {{logo}}`

- Set the color of the keys:

`fastfetch --color`

- Set the logo to use:

`fastfetch --separator`
