# krdc

> Connect to a remote desktop server.
> More information: <https://invent.kde.org/network/krdc>.

- Start KRDC:

`krdc`

- Specify an host to connect to automatically:

`krdc {{rdp://user@ip_address}}`

- Start the connection in fullscreen mode:

`krdc {{rdp://user@ip_address}} --fullscreen`
