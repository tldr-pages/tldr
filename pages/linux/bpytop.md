# bpytop

> Display dynamic real-time information about running processes with graphs. Python port of `bashtop`. Similar to `gtop`.
> More at <https://github.com/aristocratos/bpytop>

- Start bpytop
`bpytop`

- Start in minimal mode without memory and networking boxes
`bpytop -m`

- Show version
`bpytop -v`

- Toggle minimal mode
`m`

- Seach for running programs or processes
`f`

- Change settings
`M`
