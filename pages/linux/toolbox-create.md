# toolbox create

> Create a new `toolbox` container.
> More information: <https://www.mankier.com/1/toolbox-create>.

- Create a `toolbox` container for a different distribution other than the host:

`toolbox create --distro {{distribution}}`

- Create a `toolbox` container for a different distribution release other than the host:

`toolbox create --release {{release}}`

- Change the name of the image used to create the `toolbox` container:

`toolbox create --image {{name}}`
