# toolbox create

> Create a new toolbox container.
> More information: <https://www.mankier.com/1/toolbox-create>.

- Create a toolbox container for a different operating system DISTRO than the host:

`toolbox create --distro {{DISTRO}}`

- Create a toolbox container for a different operating system RELEASE than the host:

`toolbox create --release {{RELEASE}}`

- Change the NAME of the image used to create the toolbox container:

`toolbox create --image {{NAME}}`
