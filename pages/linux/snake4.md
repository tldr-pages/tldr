# snake4

> Snake game in the terminal.
> More information: <https://manned.org/snake4.6>.

- Start a snake game:

`snake4`

- Choose level:

`{{1|2|3|4|5}}`

- Navigate the snake:

`{{Up|Down|Left|Right arrow key}}`

- Pause game:

`<Spacebar>`

- Quit game:

`q`

- Show the high scores:

`snake4 --highscores`
