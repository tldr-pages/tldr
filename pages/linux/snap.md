# snap

> Tool for managing the *snap* self-contained softwawre packages.
> Similar to what *apt* is for *.deb*.

- Looking for an app

`snap find someprogram`

- Installing software:

`snap install tldr`

- Find the installed snap software

`snap list`

- Removing programs:

`snap remove no-good-app`

- Check for recent system changes

`snap changes`
