# snap

> Tool for managing the *snap* self-contained softwawre packages.
> Similar to what *apt* is for *.deb*.

- Look for an app:

`snap find {{package}}`

- Install software:

`snap install {{package}}`

- Find installed *snap* software:

`snap list`

- Remove programs:

`snap remove {{package}}`

- Check for recent *snap* changes:

`snap changes`
