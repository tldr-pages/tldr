# tlp

> Advanced Power Management for Linux.

- Apply settings (according to the actual power source):

`sudo tlp start`

- Apply battery settings (_ignoring_ the actual power source):

`sudo tlp bat`

- Apply AC settings (_ignoring_ the actual power source):

`sudo tlp ac`

- Status report with configuration and all active settings:

`sudo tlp-stat`

- Show battery information:

`sudo tlp-stat -b`

- Show configuration:

`sudo tlp-stat -c`
