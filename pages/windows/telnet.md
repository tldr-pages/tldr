# telnet

> Communicates with a computer running the telnet server service.
> More information: [(https://learn.microsoft.com/en-us/windows-server/administration/windows-commands/telnet).](https://learn.microsoft.com/en-us/windows-server/administration/windows-commands/telnet).

- Connect with computer using telnet:

`telnet {{host_name}} {{port}}}`

- Assign Escape character used to enter the telnet client prompt:

`telnet /e {{escape_char}}`

- User specific user name to logon:

`telnet /l {{user_name}}`
