# format

> Formats a disk to accept Windows files. You must be a member of the Administrators group to format a hard drive.
> More information: <https://learn.microsoft.com/en-us/windows-server/administration/windows-commands/format>.

- Specifies the mount point, volume name, or drive letter (followed by a colon) of the drive that you want to format:

`volume`

- Specifies the type of file system (FAT, FAT32, NTFS, exFAT, ReFS, or UDF):

`/FS:filesystem`

- Performs a quick format:

`/Q`

- 	Displays help at the command prompt:

`/?`
