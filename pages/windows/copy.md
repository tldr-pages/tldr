# copy

> Copy file(s) to another location.
> More information: <https://docs.microsoft.com/en-us/windows-server/administration/windows-commands/copy>.

- Copy file(s) to another directory:

`copy {{path/to/source_file(s)}} {{path/to/target_directory}}`

- Copy file to another file:

`copy {{path/to/source_file}} {{path/to/destination_file}}`
