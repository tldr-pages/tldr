# netstat

> Displays active TCP connections, ports on which the computer is listening, Ethernet statistics, the IP routing table, IPv4 statistics and IPv6 statistics.
> More information: <https://learn.microsoft.com/en-us/windows-server/administration/windows-commands/netstat>.

- Displays active TCP connections:

`netstat`

- Displays a list of currently open ports and related IP addresses:

`netstat -an`
