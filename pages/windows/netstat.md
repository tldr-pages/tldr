# netstat

> Displays active TCP connections, ports on which the computer is listening, Ethernet statistics, the IP routing table, IPv4 statistics (for the IP, ICMP, TCP, and UDP protocols), and IPv6 statistics (for the IPv6, ICMPv6, TCP over IPv6, and UDP over IPv6 protocols).
> More information: <https://learn.microsoft.com/en-us/windows-server/administration/windows-commands/netstat>.

- Displays active TCP connections:

`netstat`

- Displays a list of currently open ports and related IP addresses. This command will also tell you what state the port is in; listening, established, or closed:

`netstat -an`
