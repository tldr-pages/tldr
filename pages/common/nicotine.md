# nicotine

> Graphical client for the Soulseek peer-to-peer network.
> More information: <https://manned.org/nicotine>.

- Start Nicotine+:

`nicotine`

- Start Nicotine+ in headless mode:

`nicotine {{[-n|--headless]}}`

- Rescan shared files:

`nicotine {{[-r|--rescan]}}`

- Display help:

`nicotine {{[-h|--help]}}`
