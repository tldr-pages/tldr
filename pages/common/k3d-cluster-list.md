# k3d cluster list

> List all k3d clusters.
> More information: <https://k3d.io/>.

- List all clusters:

`k3d cluster list`

- List specific cluster(s):

`k3d cluster list {{cluster_name}}`

- List clusters without table headers:

`k3d cluster list --no-headers`

- Output cluster list as JSON:

`k3d cluster list --output json`

- Output cluster list as YAML:

`k3d cluster list --output yaml`

- Show cluster token:

`k3d cluster list --token`
