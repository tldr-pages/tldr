# acli jira

> Jira Cloud commands for Atlassian CLI.
> Interact with Jira boards, dashboards, projects, sprints, and work items.
> More information: <https://developer.atlassian.com/cloud/acli/reference/commands/jira>.

- Display help:

`acli jira {{[-h|--help]}}`

- Authenticate to use Atlassian CLI:

`acli jira auth`

- Manage Jira boards:

`acli jira board`

- Manage Jira dashboards:

`acli jira dashboard`

- Manage Jira fields:

`acli jira field`

- Manage Jira filters:

`acli jira filter`

- Manage Jira projects:

`acli jira project`

- Display help for a specific Jira command:

`acli jira {{command}} {{[-h|--help]}}`
