# rnano

> A restricted version of `nano`.
> Command-line text editor with restricted drive access.
> See also: `nano`.
> More information: <https://manned.org/rnano>.

- Start the editor:

`rnano`

- Open a specific file:

`rnano {{path/to/file}}`

- Display help:

`rnano {{[-h|--help]}} {{path/to/file}}`
