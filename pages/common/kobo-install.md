# kobo-install

> Command-line tool to setup and manage KoboToolbox.
> More information: <https://github.com/kobotoolbox/kobo-install>.

- Setup KoboToolbox for the first time:

`python3 run.py --setup`

- View the status of the KoboToolbox containers:

`python3 run.py --info`

- Stop all running KoboToolbox services:

`python3 run.py --stop`

- Update the KoboToolbox installation:

`python3 run.py --update`
