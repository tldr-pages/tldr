# whois

> Command-line client for the WHOIS (RFC 3912) protocol.
> More information: <https://manned.org/whois>.

- Get information about a domain name:

`whois {{example.com}}`

- Get information about an IP address:

`whois {{8.8.8.8}}`

- Get abuse contact for an IP address:

`whois -b {{8.8.8.8}}`
