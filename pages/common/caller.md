# caller

> Print function context.
> More information: <https://www.gnu.org/software/bash/manual/bash.html#index-caller>.

- Print the line where the current function was called and its filename:

`caller`

- Print the line where the current function was called, its function name and filename:

`caller 0`

- Print the line of a function call `n` frames back, its function name and filename:

`caller {{n}}`
