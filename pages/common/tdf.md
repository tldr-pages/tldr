# tdf

> A tui PDF viewer.
> More information: <https://github.com/itsjunetime/tdf>.

- Open specific file:

`tdf {{path/to/pdf}}`

- Open in fullscreen mode:

`tdf -f {{path/to/pdf}}`

- Set custom white color:

`tdf -w "{{COLOR HEX or RGB}}"`

- Set custom black color:

`tdf -b "{{COLOR HEX or RGB}}"`
