# timg

> A terminal image and video viewer.
> More information: <https://github.com/hzeller/timg>.

- Display a static image:

`timg image.jpg`

- Display all jpg images:

`timg *.jpg`

- Display all the bitmaps in icons file:

`timg example.ico`

- Play a video:

`timg video.mp4`

- Play contents of webcam:

`timg {{/dev/videoX}}`

- Play animated gif:

`timg gif.gif`

- Print the version:

`timg --version`
