# acli

> Official Atlassian Command Line Interface.
> Interact with Atlassian products such as Jira through a command-line interface.
> Some subcommands such as `admin`, `jira`, `rovodev`, and `feedback` have their own usage documentation.
> More information: <https://developer.atlassian.com/cloud/acli/reference/commands/>.

- Display help:

`acli {{[-h|--help]}}`

- Display help for a specific command:

`acli {{command}} {{[-h|--help]}}`

- View documentation for the admin commands:

`tldr acli-admin`

- View documentation for the Jira commands:

`tldr acli-jira`

- View documentation for the Rovo Dev commands:

`tldr acli-rovodev`

- View documentation for the feedback command:

`tldr acli-feedback`
