# subl

> Sublime Text
>
> A sophisticated text editor for code, markup and prose.

- Open a file or folder:

`subl {{path/to/file_or_folder}}`

- Open a file or folder in a new window:

`subl -n {{path/to/file_or_folder}}`

- Add a folder to existing window:

`subl -a {{path/to/folder}}`

- Open a Project:

`subl --project {{path/to/project/file}}`
