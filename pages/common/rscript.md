# R

> Run a script with the R programming language.
> More information: <https://www.r-project.org>.

- Run an R script:

`Rscript {{file.R}}`

- Start an Rscript in vanilla mode (i.e. a blank session that doesn't save the workspace at the end):

`Rscript --vanilla {{file.R}}`

- Check R version:

`Rscript --version`

- Execute one or more R expressions and then exit:

`Rscript -e {{expr1}} -e {{expr2}}`
