# ls

> List directory contents.
> More information: <https://www.gnu.org/software/coreutils/ls>.

- List files one per line:

`ls -1`

- List all files including hidden ones:

`ls -a`

- List files with human readable sizes:

`ls -lh`

- List files sorted by size:

`ls -lS`
