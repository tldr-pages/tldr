# javap

> Disassemble one or more class files and view packages.
> Part of `JDK`.
> More information: <https://docs.oracle.com/en/java/javase/18/docs/specs/man/javap.html>.

- Disassemble and list a class file:

`javap {{path/to/class}}`

- Disassemble and list a built-in class file:

`javap java.{{package}}.{{class}}`

- Display Version:

`javap -version`

- Display help:

`javap -help`
