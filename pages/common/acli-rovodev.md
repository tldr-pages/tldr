# acli rovodev

> Atlassian's AI coding agent: Rovo Dev (Beta).
> Interact with Rovo Dev through the command line.
> More information: <https://developer.atlassian.com/cloud/acli/reference/commands/rovodev>.

- Display help:

`acli rovodev {{[-h|--help]}}`

- Authenticate to use Rovo Dev CLI:

`acli rovodev auth`

- Display help for a specific Rovo Dev command:

`acli rovodev {{command}} {{[-h|--help]}}`
