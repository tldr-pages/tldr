# composer

> A package-based dependency manager for PHP projects.

- Update composer to the latest version:

`composer self-update`

- Make a package a dependency for this project:

`composer require {{user/package-name}}`

- Install all dependencies for this project:

`composer install`

- Uninstall a package from this project, removing it as a dependency:

`composer remove {{user/package-name}}`

- Update all the dependencies of this project:

`composer update`
