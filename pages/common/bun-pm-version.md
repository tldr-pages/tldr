# bun pm version

> Manage the `version` field in `package.json`.
> More information: <https://bun.com/docs/pm/cli/pm#version>.

- Display the current package version:

`bun pm version`

- Set a specific version:

`bun pm version {{version}}`

- Bump to the next major version:

`bun pm version major`

- Bump to the next minor version:

`bun pm version minor`

- Bump to the next patch version:

`bun pm version patch`

- Bump to a premajor version:

`bun pm version premajor`

- Bump to a preminor version:

`bun pm version preminor`

- Set the version based on Git tags:

`bun pm version from-git`
