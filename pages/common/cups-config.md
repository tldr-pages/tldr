# cups-config

> Shows technical information about your CUPS print server installation.
> More information: <https://www.cups.org/doc/man-cups-config.html>.

- Show information about the currently-installed version of CUPS:

`cups-config --version`

- Show information about where CUPS is currently installed:

`cups-config --serverbin`

- Show information about CUPS' configuration directory:

`cups-config --serverroot`

- Show information about CUPS' data directory:

`cups-config --datadir`

- Display all available options:

`cups-config --help`
