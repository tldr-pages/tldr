# aws s3 mb

> Create S3 buckets.
> More information: <https://awscli.amazonaws.com/v2/documentation/api/latest/reference/s3/mb.html>.

- Create an S3 bucket:

`aws s3 mb s3://{{bucket_name}}`

- Create an S3 bucket in a specific region:

`aws s3 mb s3://{{bucket_name}} --region {{region}}`

- Display help:

`aws s3 mb help`
