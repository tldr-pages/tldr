# fish

> A command-line shell for the 90's

- Start interactive shell:

`fish`

- Execute a command:

`fish -c "{{command}}"`

- Run commands from a file:

`fish {{file.fish}}`

- Check a file for syntax:

`fish --no-execute {{file.fish}}`

- Display version information and exit:

`fish --version`

- Enable debug information:

`fish -D {{DEBUG_LEVEL}}`
