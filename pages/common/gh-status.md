# gh status

> Check the status of subscribed repos on GitHub.
> More information: <https://cli.github.com/manual/gh>.

- Print information about all repos you're subscribed to (issues, pull requests, reviews, mentions, activity, etc.):

`gh status`

- Exclude certain repos:

`gh status {{[-e|--exclude]}} {{repos}}`

- Print status of repos belonging to an organization :

`gh status {{[-o|--org]}} {{organization}}`
