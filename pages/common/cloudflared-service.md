# cloudflared service

> Manage cloudflared as a system service.
> More information: <https://developers.cloudflare.com/cloudflare-one/networks/connectors/cloudflare-tunnel/>.

- Install cloudflared as a system service:

`cloudflared service install`

- Install as a service using a tunnel token:

`cloudflared service install {{token}}`

- Uninstall the cloudflared service:

`cloudflared service uninstall`
