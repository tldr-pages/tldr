# cpan

> Query, download and build perl modules from CPAN sites.
> More information: <https://manned.org/man/cpan>.

- Install modules (sole -i is optional):

`cpan -i {{module_name}}`

- Force install modules (must use -i):

`cpan -fi {{module_name}}`

- Upgrade all installed modules:

`cpan -u`

- Recompile modules:

`cpan -r`
