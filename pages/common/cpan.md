# cpan

> Query, download and build perl modules from CPAN sites.
> More information: <https://manned.org/cpan>.

- Install a module (`-i` is optional):

`cpan {{-i}} {{module_name}}`

- Force install a module (`-i` is not optional):

`cpan -fi {{module_name}}`

- Upgrade all installed modules:

`cpan -u`

- Recompile modules:

`cpan -r`
