# cloudflared tunnel login

> Authenticate cloudflared and generate an account certificate for managing tunnels.
> See also: `cloudflared tunnel create`.
> More information: <https://developers.cloudflare.com/cloudflare-one/networks/connectors/cloudflare-tunnel/>.

- Authenticate and store the account certificate at the default path (`~/.cloudflared/cert.pem`):

`cloudflared tunnel login`
