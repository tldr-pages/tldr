# cline
> AI-powered command-line assistant.
> More information: <https://github.com/cline/cline-cli>

- Ask cline a question:
`cline "{{How do I list files?}}"`

- Ask cline to explain a command:
`cline explain "{{ls -la}}"`

- Generate a command for a task:
`cline generate "{{create a zip backup for this folder}}"`

- Show help:
`cline --help`
