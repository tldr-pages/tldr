\# cline



> AI-powered autonomous coding agent that runs in the terminal.

> More information: <https://github.com/cline/cline-cli>.



\- Authenticate with an AI provider (OpenAI, Anthropic, etc.):



`cline auth`



\- Start a new coding task:



`cline task "{{description of what you want to build}}"`



\- List all task instances:



`cline list`



\- Resume a specific task instance:



`cline resume {{instance\_id}}`



\- Stop a running task:



`cline stop {{instance\_id}}`



\- View current configuration:



`cline config`



\- Display help:



`cline --help`
