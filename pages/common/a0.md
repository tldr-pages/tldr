# a0

> CLI connector for Agent Zero that bridges the Docker-sandboxed AI agent to the host terminal, local files, and browser.
> More information: <https://www.agent-zero.ai/p/docs/a0-cli-connector/>.

- Launch and auto-connect to a local Agent Zero instance:

`a0`

- Connect to a specific Agent Zero host:

`a0 --host {{http://localhost:32080}}`

- Launch without auto-connecting:

`a0 --no-auto-connect`

- Launch without Docker discovery:

`a0 --no-docker-discovery`

- Connect using an environment variable:

`AGENT_ZERO_HOST={{http://localhost:32080}} a0`
