# nano

> Simple, easy to use editor. An enhanced, free Pico clone.

- Start nano in terminal with {filename}:

`nano {{filename}}`

- Enable smooth scrolling:

`nano -S {{filename}}`

- Indent new lines to the previous lines' indentation:

`nano -i {{filename}}`
- Save an edited file

`CTRL` + `o`, then `enter` or `CTRL` + `c` to cancel

- Quit and optionally save

`CTRL` + `x`, then `Y` or `N` and then `enter`
