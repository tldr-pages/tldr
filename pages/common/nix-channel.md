# nix-channel

> Manage `nix` update channels.
> More information: <https://nixos.wiki/wiki/Nix_channels>.

- List current channels:

`nix-channel --list`

- Add a channel:

`nix-channel --add {{https://nixos.org/channels/nixpkgs-unstable}}`

- Update package list of all channels:

`nix-channel --update`
