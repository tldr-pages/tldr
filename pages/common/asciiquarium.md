# asciiquarium

> Display an animated aquarium in ASCII art inside the terminal.
> Requires Perl and the `Term::Animation` module.
> More information: <https://github.com/cmatsuoka/asciiquarium>.

- Start the aquarium animation:

`asciiquarium`

- Run the aquarium in a smaller terminal window for compact view:

`asciiquarium`

- Exit the aquarium (keyboard shortcut):

`Ctrl + C`

- Pipe the output through `lolcat` for colorful fish:

`asciiquarium | lolcat`

- Run `asciiquarium` in full-screen terminal mode for best experience:

`asciiquarium`
