# asciiquarium

> Display an animated aquarium in ASCII art inside the terminal.
> More information: <https://robobunny.com/projects/asciiquarium/html/>.

- Start `asciiquarium`:

`asciiquarium`

- Exit `asciiquarium`:

`<q>`

- Toggle [p]ause:

`<p>`

- [R]edraw aquarium and all entities:

`<r>`

- Pipe the output through `lolcat` for rainbow colors:

`asciiquarium | lolcat`
