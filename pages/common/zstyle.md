# zstyle

> Set styles used by Zsh.

> More information: <https://manned.org/zshbuiltins>.

- Display current style settings:

`zstyle`

- Set a style for a specific pattern:

`zstyle {{pattern}} {{style}} {{value}}`

- List styles matching a pattern:

`zstyle -L {{pattern}}`

- Display help:

`zstyle --help`

- Display version:

`zstyle --version`
