# chime

> The Amazon Chime SDK is a set of real-time communications components that developers can use to quickly add messaging, audio, video, and screen sharing.
> More information: <https://aws.amazon.com/chime/>.

- To Install directly from the npm:

`npm install amazon-chime-sdk-js --save`

- To confirm your attachment request:

`Get my attachments ktsben`

- To confirm your account deletion request:

`Delete me aofrkq`

- To send an Amazon Chime chat message using markdown syntax:

`/md **Hello world!**`
