# czkawka-cli

> Multi functional app to find duplicates, empty folders, similar images etc. Command-line version.
> More information: <https://github.com/qarmin/czkawka>.

- Find [dup]licate files in home directory:

`czkawka-cli dup --directories ~/`

- Find [dup]licate files and delete them:

`czkawka-cli dup --directories ~/ --delete-method AEN`

- Find similar [image]s:

`czkawka-cli image --directories ~/`
