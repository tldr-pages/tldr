# postcss

> PostCSS is a tool for transforming styles with JS plugins.

- Parse and transform CSS file(s):

`postcss {{path/to/file(s)}}`

- Parse and transform CSS file(s) and output to a specific file:

`postcss {{path/to/file(s)}} --output {{path/to/file}}`

- Parse and transform CSS file(s) and output to a specific directory:

`postcss {{path/to/file(s)}} --dir {{path/to/directory}}`

- Parse and transform CSS file(s) in-line:

`postcss {{path/to/file(s)}} --replace`

- Specify a custom PostCSS parser:

`postcss {{path/to/file(s)}} --parser {{parser}}`

- Specify a custom PostCSS syntax:

`postcss {{path/to/file(s)}} --syntax {{syntax}}`

- Parse and transform CSS file(s) by watching for changes:

`postcss {{path/to/file(s)}} --watch`

- Display available options and examples:

`postcss --help`
