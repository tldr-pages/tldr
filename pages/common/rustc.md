# rustc

> The Rust compiler.
> Processes, compiles and links Rust language source files.
> More information: <https://doc.rust-lang.org/rustc>.

- Compile a single file:

`rustc {{file.rs}}`

- Compile with high optimization:

`rustc -O {{file.rs}}`

- Compile with debugging information:

`rustc -g {{file.rs}}`
