# mpd

> Music Player Daemon.
> A daemon for playing music.
> More information: <https://www.musicpd.org/>.

- Start MPD:

`mpd`

- Start MPD but don't read from the configuration file:

`mpd --no-config`

- Start MPD and don't detach from console:

`mpd --no-daemon`

- Kill the currently running MPD session:

`mpd --kill`
