# xpdf

> Portable Document Format (PDF) file viewer.
> More information: <https://www.xpdfreader.com/xpdf-man.html>.

- Open PDF file:

`xpdf {{file.pdf}}`

- Open a specific page in pdf file:

`xpdf {{file.pdf}} {{5}}`

- Open compressed PDF file:

`xpdf {{file.pdf.tar}}`

- Open PDF file in fullscreen mode:

`xpdf -fullscreen {{file.pdf}}`

- Specify initial zoom:

`xpdf -z {{75}}% {{file.pdf}}`
