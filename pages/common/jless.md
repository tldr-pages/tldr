# jless

> Terminal-based JSON viewer.
> More information: <https://github.com/PaulJuliusMartinez/jless>.

- Open and view a JSON file:

`jless {{path/to/file.json}}`

- View JSON piped from `stdin`:

`{{command}} | jless`

- Display help:

`jless {{[-h|--help]}}`

- Display version:

`jless {{[-V|--version]}}`
