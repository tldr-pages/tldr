# pip list

> List installed packages.
> More information: <https://pip.pypa.io/en/stable/cli/pip_list/>.

- List all installed packages:

`pip list`

- List outdated packages:

`pip list --outdated`

- List packages in JSON format:

`pip list --format json`

- List packages in freeze format:

`pip list --format freeze`

- List only locally installed packages:

`pip list --local`

- List only user-installed packages:

`pip list --user`

- List packages not required by others:

`pip list --not-required`
