# rails db

> Various database-related subcommands for Ruby on Rails.

- Access the database console:

`rails db`

- Create databases defined in current environment:

`rails db:create`

- Destroy databases defined in current environment:

`rails db:drop`

- Run pending migrations:

`rails db:migrate`

- View status of each migration file:

`rails db:migrate:status`

- Rollback the last migration:

`rails db:rollback`

- Fill database with data defined in db/seeds.rb:

`rails db:seed`

- Create databases, load the schema, and initialize with seed data:

`rails db:setup`
