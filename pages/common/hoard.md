# hoard

> A command organizer tool to save terminal commands.
> More information: <https://github.com/Hyde46/hoard#Usage>.

- Save a new command:

`hoard new`

- List saved commands:

`hoard list`

- Edit a saved command:

`hoard --name {{command_name}}`

- Remove a command:

`hoard remove --name {{command_name}}`

- Remove a namespace and all of the commands in it:

`hoard remove-namespace --namespace {{namespace_name}}`
