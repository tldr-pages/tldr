# screenfetch

> Display system information.

- Start screenfetch:

`screenfetch`

- Take a screenshot (requires 'scrot'):

`screenfetch -s`

- Specify distribution logo:

`screenfetch -A '{{distribution_name}}'`

- Specify distribution logo and text:

`screenfetch -D '{{distribution_name}}'`

- Strip all color:

`screenfetch -N`
