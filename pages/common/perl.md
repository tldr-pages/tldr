# perl

> The Perl 5 language interpreter 'CLI'.

- Parse and execute a file:

`perl {{file}}`

- Parse and execute a statement:

`perl -e {{statement}}`

- Check syntax error:

`perl -c {{file}}`

- Import module before execution:

`perl -M{{module}} -e {{statement}}`

- Run under perl debugger (perldebug):

`perl -d {{file}}`
