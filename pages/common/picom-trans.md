# picom-trans

> Set window opacity.
> More information: <https://github.com/yshui/picom>.

- Set opacity of current window to 90 percent:

`picom-trans --current --opacity={{90}}`

- Set opacity of window with name 'Firefox' to 90 percent:

`picom-trans --name={{Firefox}} --opacity={{90}}`

- Select window to set opacity for with mouse cursor:

`picom-trans --select --opacity={{90}}`

- Toggle opacity for window with name 'Firefox':

`picom-trans --name={{Firefox}} --toggle`
