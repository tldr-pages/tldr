# git what-changed

> Show what has changed with recent commits or files.
> See also `git log`.
> More information <https://git-scm.com/docs/git-whatchanged>.

- Display log of recent commits:

`git what-changed`

- Display log of recent commits within the specified time frame:

`git what-changed --since="2 hours ago"`

- Display changes to selected file(s):

`git what-changed {{file}}`
