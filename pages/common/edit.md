# edit

> A terminal-based text editor from Microsoft.
> More information: <https://github.com/microsoft/edit>.

- Open a file:

`edit {{path/to/file}}`

- Search for a pattern:

`<Ctrl f>{{pattern}}<Enter>`

- Find and replace a pattern:

`<Ctrl r>{{pattern}}<Enter>`

- Copy text:

`<Ctrl c>`

- Paste text:

`<Ctrl v>`

- Open the File menu:

`<Alt f>`

- Quit:

`<Ctrl q>`
