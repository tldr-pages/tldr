# koboldcpp

> An AI text-generation application for GGML and GGUF models.
> More information: <https://github.com/LostRuins/koboldcpp>.

- Load one or more model files:

`koboldcpp --model {{path/to/model_file_1}}`

- Set port to listen on (defaults to 5001):

`koboldcpp --port {{port}}`

- Set amound of threads to use:

`koboldcpp --threads {{amount_of_threads}}`

- Lauch web browser when boot is complete:

`koboldcpp --launch`

- Do not show the GUI launcher:

`koboldcpp --skiplauncher`
