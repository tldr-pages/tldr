# pm2

> Process manager for Node.js.
> Log management, monitoring, configuring processes.

- Start a process:

`pm2 start {{app.js}}`

- List proceses:

`pm2 list`

- Monitor all processes:

`pm2 monit`

- Stop process:

`pm2 stop {{process_id}}`

- Restart process:

`pm2 restart {{process_name}}`
