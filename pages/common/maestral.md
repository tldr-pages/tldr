# maestral

> Maestral is a lightweight Dropbox client for macOS and Linux.

- Start the GUI:

`maestral gui`

- Get current status of Maestral:

`maestral status`

- Pause syncing:

`maestral pause`

- Resume syncing:

`maestral resume`

- Get sync status of a specific file or folder:

`maestral filestatus {{local_path}}`
