# csvpy

> Load a CSV file into a Python shell.
> Included in csvkit.
> More information: <https://csvkit.readthedocs.io/en/latest/scripts/csvpy.html>.

- Load a CSV file into a `CSVKitReader` object:

`csvpy {{data.csv}}`

- Load a CSV file into a `CSVKitDictReader` object:

`csvpy --dict {{data.csv}}`
