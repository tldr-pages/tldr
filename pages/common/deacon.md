# deacon

> Utility that fetches daily Orthodox readings and fasting information, for both Julian and Gregorian calendars.
> More information: <https://codeberg.org/knyz/deacon>.

- Output colourized ANSI Julian calendar information to `stdout`:

`deacon`

- Read Julian calendar information in a pager:

`deacon -ji`

- Output colourized ANSI Gregorian calendar information to `stdout`:

`deacon -g`

- Display help:

`deacon -h`
