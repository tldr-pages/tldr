# osmium

> A multipurpose command line tool based on the Osmium Library.
> More information: <https://osmcode.org/osmium-tool/manual>.

- Extract a geographic region by a GeoJSON file:

`osmium extract -p {{path/to/extract-geometry}}.geojson {{path/to/input}}.pbf -o {{path/to/output}}.pbf`
