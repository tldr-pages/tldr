# npm-stars

> View packages starred by a specific npm user or by the current logged-in user.  
> More information: <https://docs.npmjs.com/cli/v10/commands/npm-stars>.

- Show all packages starred by the currently logged-in user:

`npm stars`

- Show packages starred by a specific user:

`npm stars {{username}}`

- Show starred packages in JSON format:

`npm stars --json`

- Show help for the command:

`npm stars --help`
