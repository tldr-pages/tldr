# taskwarrior

> Менеджер задач в командной строке.
> Поддерживает теги, приоритеты, зависимости, повторы и фильтрацию.
> Подробнее: <https://taskwarrior.org/docs/>.

- Добавить новую задачу:

`task add {{описание_задачи}}`

- Добавить задачу с датой выполнения и тегами:

`task add {{описание}} due:{{завтра}} +{{тег1}} +{{тег2}}`

- Просмотреть список текущих задач:

`task`

- Отметить задачу как выполненную:

`task {{номер_задачи}} done`

- Удалить задачу:

`task {{номер_задачи}} delete`

- Изменить описание существующей задачи:

`task {{номер_задачи}} modify {{новое_описание}}`

- Фильтровать задачи по тегу:

`task +{{тег}}`

- Отобразить задачи с подробностями:

`task long`
