# cloudflared update

> Update cloudflared to the latest version from the official download server.
> More information: <https://developers.cloudflare.com/cloudflare-one/networks/connectors/cloudflare-tunnel/>.

- Update to the latest release:

`cloudflared update`

- Update to the latest beta release:

`cloudflared update --beta`

- Update or downgrade to a specific version:

`cloudflared update --version {{2024.1.0}}`
