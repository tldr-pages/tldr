# kubectl version

> Prints the client and server version information for the current context.
> More information: <https://kubernetes.io/docs/reference/generated/kubectl/kubectl-commands#version>.

- Prints the client and server version information for the current context:

`kubectl version`

- Prints the client version only for the current context

`kubectl version --client`

- Outputs the client and server version information for the current context in json:

`kubectl version --json`

- Outputs the client and server version information for the current context in yaml:

`kubectl version --yaml`
