# CUPS

> Open-source printing system.
> CUPS isn't a command, but a set of commands. See the pages below.
> More information: <https://www.cups.org/index.html>.

- Show the tldr page for running the CUPS daemon:

`tldr cupsd`

- Show the tldr page for managing printers:

`tldr lpadmin`

- Show the tldr page for printing pages:

`tldr lp`

- Show the tldr page for seeing status information about the current classes, jobs, and printers:

`tldr lpstat`

- Show the tldr page for cancelling print jobs:

`tldr lprm`
