# pywal

> A tool to create colorschemes based on dominant colors of a wallpaper.
> Read more: https://github.com/dylanaraps/pywal.

- Preview colorscheme:

`wal --preview image.png`

- Create colorscheme:

`wal -i image.png`

- Create light colorscheme:

`wal -il image.png`

- Skip setting wallpaper:

`wal -in image.png`

- Skip setting terminal colors:

`wal -is image.png`

- Restore colorscheme and wallpaper:

`wal -R`
