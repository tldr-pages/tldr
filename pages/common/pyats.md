# pyATS

> A vendor agnostic test automation framework by Cisco Systems, used for network and systems testing.
> More information: <https://developer.cisco.com/pyats/>.

- Show the installed version of pyATS and libraries:

`pyats version check`

- Show general help including available subcommands:

`pyats --help`

- Show help on a pyATS subcommand (like `create`, `shell`, `diff`, `parse`, etc.):

`pyats {{subcommand}} --help`

- Execute a pyATS subcommand:

`pyats {{subcommand}} [options]`
