

# expo

> A tool for creating, developing, and building React Native apps using Expo.
> More information: https://docs.expo.dev.

- Create a new Expo project:
  `expo init {{project_name}}`

- Start the development server:
  `expo start`

- Run the project on an Android device or emulator:
  `expo run:android`

- Run the project on an iOS simulator:
  `expo run:ios`

- Install a package into the project:
  `expo install {{package_name}}`

- Build an Android APK or AAB:
  `expo build:android`

- Build an iOS IPA:
  `expo build:ios`
