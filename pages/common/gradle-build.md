# gradle build

> Build a project using Gradle.
> More information: <https://docs.gradle.org/current/userguide/command_line_interface.html#computing_all_outputs>.

- Build the project:

`gradle build`

- Perform a clean build:

`gradle clean build`

- Build the project while skipping tests:

`gradle build {{[-x|--exclude-task]}} test`

- Build with more detailed logging:

`gradle build {{[-i|--info]}}`
