# [more](http://man7.org/linux/man-pages/man1/more.1.html)
> Open a file for interactive reading, allowing scrolling and search (in forward direction only).

- Open a file:

`more {{source_file}}`

- Page down:

`<Space>`

- Search for a string (press `n` to go to the next match):

`/{{something}}`

- Exit:

`q`
