# vue-prototyping

> A feature that does not ship with `@vue/cli` in default which allows you to do prototyping painlessly.
> You need to install `@vue/cli-service-global` to use this feature.
> More information: <https://cli.vuejs.org/guide/prototyping.html>.

- Serve a `.js` or `.vue` file in development mode with zero config:

`vue serve {{filename}}`

- Build a `.js` or `.vue` file in production mode with zero config:

`vue build {{filename}}`
