# zotero

> Manage your bibliographies.
> More information: <https://www.zotero.org/support>.

- Run Zotero with the GUI:

`zotero`

- Run Zotero in headless mode:

`zotero --headless`

- Run Zotero with a specific profile:

`zotero -P {{profile}}`

- Run Zotero Migration Assistant:

`zotero --migration`
