# fnt

> A font manager for Linux and macOS.
> More information: <https://manned.org/fnt>.

- Install a font:

`fnt {{[-i|install]}} {{font}}`

- Search for a font:

`fnt {{[-s|search]}} {{font}}`

- Preview a font:

`fnt {{[-p|preview]}} {{font}}`

- List installed fonts:

`fnt {{[-l|list]}}`

- Update the font package index:

`fnt {{[-u|update]}}`

- Remove a font:

`fnt {{[-r|remove]}}`
