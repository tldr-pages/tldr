# git

> Distributed version control system.

- Reset TO the last commit in the HEAD

`git reset --hard HEAD`

- Modify existing, unpushed commits?
  
  `git commit --amend -m "New commit message"`
  
  `git commit --amend`

- Check the Git version:

`git --version`

- Call general help:

`git --help`

- Call help on a command:

`git help {{command}}`

- Execute Git command:

`git {{command}}`
