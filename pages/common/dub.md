# dub

> Package manager for D packages.
> More information: <https://dub.pm/commandline>.

- Display general or command specific help:

`dub --help`

- Update the dependencies in D project:

`dub upgrade`

- Build and run D project:

`dub`

- Install dependencies specified in D project's `dub.json|dub.sdl` file:

`dub fetch`

- Create a new Dlang project [iterative mode]:

`dub init {{Project_Name}}`

- Create a new D project [non-iterative mode]:

`dub init {{Project_Name}} -n`
