# greed

> Simple game.
> Clear as much of the board as you can.
> More information: <https://manpages.ubuntu.com/manpages/jammy/man6/greed.6.html>.

- Start the game:

`greed`

- Open help:

`<?>`

- Toggle move highlights:

`<p>`

- Quit:

`<q><y>`

- Print scores:

`greed -s`

- Play with a custom board size and seed:

`greed -w {{width}} -h {{height}} -r {{seed}}`
