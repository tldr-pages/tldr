# acli admin

> Admin commands for Atlassian CLI.
> Manage organization admin tasks.
> More information: <https://developer.atlassian.com/cloud/acli/reference/commands/admin>.

- Display help:

`acli admin {{[-h|--help]}}`

- Authenticate to use Atlassian CLI for organization admin tasks:

`acli admin auth`

- Manage a user within your Atlassian organization:

`acli admin user`

- Display help for a specific admin command:

`acli admin {{command}} {{[-h|--help]}}`
