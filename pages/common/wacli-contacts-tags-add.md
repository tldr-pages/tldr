# wacli contacts tags add

> Add a tag to a WhatsApp contact.
> See also: `wacli contacts show`, `wacli contacts search`.

- Add a tag to a contact:

`wacli contacts tags add {{phone_number}}@s.whatsapp.net {{tag}}`

- Add a tag using a custom store:

`wacli contacts tags add {{phone_number}}@s.whatsapp.net {{tag}} --store {{path/to/store}}`
