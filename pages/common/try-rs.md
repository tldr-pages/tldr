# try-rs

> Manage temporary project sandboxes from the terminal.
> More information: <https://try-rs.org/#usage>.

- Open the TUI:

`try-rs`

- Create or jump to a named experiment:

`try-rs {{experiment_name}}`

- Clone a repository into a dated folder:

`try-rs {{https://github.com/user/repository}}`

- Clone a repository into a specific folder:

`try-rs {{https://github.com/user/repository}} {{folder_name}}`

- Shallow clone a repository:

`try-rs {{[-s|--shallow-clone]}} {{https://github.com/user/repository}}`
