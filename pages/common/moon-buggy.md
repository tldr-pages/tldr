# moon-buggy

> Chrome dino-like game on the terminal.
> More information: <https://github.com/seehuhn/moon-buggy>.

- Start a moon-buggy game:

`moon-buggy`

- Start a game, skipping the title screen and displaying high scores:

`moon-buggy --no-title --show-scores`

- Jump:

`<Space> | <j>`

- Use a mining laser:

`<l> | <a>`

- Quit the game:

`{{<q>|<n>}}`
