# uptime

> Display how long the system has been running, including the current time, number of logged-in users, and load averages.
> More information: <https://www.gnu.org/software/coreutils/manual/html_node/uptime-invocation.html>.

- Print the current time, uptime duration, number of logged-in users, and load averages:

`uptime`

- Show only how long the system has been running:

`uptime {{[-p|--pretty]}}`

- Show the date and time when the system started:

`uptime {{[-s|--since]}}`

- Display the version of uptime:

`uptime {{[-V|--version]}}`
