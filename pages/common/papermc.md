# papermc

> Manage a Minecraft server.
> More information: <https://github.com/Edenhofer/minecraft-server/blob/main/minecraftd.sh.in#L480>.

- Start a server:

`papermc start`

- Stop a server:

`papermc stop`

- Restart a server:

`papermc restart`

- Backup a world:

`papermc backup`

- Restore world data from a backup:

`papermc restore {{path/to/file}}`

- Display help:

`papermc`
