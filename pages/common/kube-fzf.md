# kube-fzf

> Shell commands for command line fuzzy searching of Kubernetes Pods.

- Get pod details (searching from current namespace):

`findpod`

- Get pod details (searching from all namespaces):

`findpod -a`

- Describe a pod:

`describepod`

- Tail pod logs:

`tailpod`

- Exec into a pod's container:

`execpod`

- Port-forward a pod:

`pfpod`
