# duf

> Disk Usage/Free Utility - a better 'df' alternative.
> More information at https://github.com/muesli/duf.

- List accessible devices:

`duf`

- List everything (including pseudo, duplicate, inaccessible file systems):

`duf --all`

- Only list specific devices & mount points:

`duf /home /some/file`

- Sort the output:

`duf --sort size`
