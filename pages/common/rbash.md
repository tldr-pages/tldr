# rbash

> Restricted Bash shell.
> Restrictions implemented in rbash: the cd command is not allowed,PATH and ENV (setting/unsetting)...

- Start rbash:

`rbash`

- Execute a command:

`rbash -c "{{command}}"`

- Run commands from a file:

`rbash {{file.sh}}`

- Print the version information of rbash:

`rbash --version`
