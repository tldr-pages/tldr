# ip route list

> IP ரூட்டிங் அட்டவணை மேலாண்மைக்கான காட்சி துணைக்கட்டளை (Display subcommand).
> மேலும் விவரத்திற்கு: <https://manned.org/ip-route>.

- `main` ரூட்டிங் அட்டவணையைக் காட்டு:

`ip {{[r|route]}} {{[l|list]}}`

- முக்கிய ரூட்டிங் அட்டவணையைக் காட்டு (முதல் உதாரணத்தைப் போன்றது):

`ip {{[r|route]}} {{[l|list]}} {{[t|table]}} {{main|254}}`

- உள்ளூர் ரூட்டிங் அட்டவணையைக் காட்டு:

`ip {{[r|route]}} {{[l|list]}} {{[t|table]}} {{local|255}}`

- அனைத்து ரூட்டிங் அட்டவணைகளையும் காட்டு:

`ip {{[r|route]}} {{[l|list]}} {{[t|table]}} {{all|unspec|0}}`

- கொடுக்கப்பட்ட சாதனத்திலிருந்து உள்ள வழிகளை மட்டும் பட்டியலிடு:

`ip {{[r|route]}} {{[l|list]}} dev {{ethX}}`

- கொடுக்கப்பட்ட வரையறைக்குள் உள்ள வழிகளைப் பட்டியலிடு:

`ip {{[r|route]}} {{[l|list]}} {{[s|scope]}} link`

- ரூட்டிங் தேக்ககத்தைக் காட்டு:

`ip {{[r|route]}} {{[l|list]}} {{[c|cache]}}`

- IPv6 அல்லது IPv4 வழிகளை மட்டும் காட்டு:

`ip {{-6|-4}} {{[r|route]}}`
