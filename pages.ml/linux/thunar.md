# thunar

> എക്സ്എഫ്‌സിഇക്കു വേണ്ടിയുള്ള ഗ്രാഫിക്കൽ ഫയൽ മാനേജർ.
> കൂടുതൽ വിവരങ്ങൾ: <https://docs.xfce.org/xfce/thunar/start>.

- നിലവിലുള്ള ഡയറക്ടറിയിൽ പുതിയ തുണാർ ജാലകം ഓപ്പൺ ചെയ്യുവാൻ:

`thunar`

- തൂണാർ ഏതു പതിപ്പാണെന്നറിയുവാൻ:

`thunar --version`

- ഓപ്പൺ ആയിട്ടുള്ള എല്ലാ തുണാർ ജാലകങ്ങളും ക്ലോസ് ചെയ്യുവാൻ:

`thunar --quit`
