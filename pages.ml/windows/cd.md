# cd

> നിലവിലെ ഡയറക്ടറിയെക്കുറിച്ചുള്ള വിവരങ്ങൾ കാണിക്കുകയും മറ്റൊരു ഡയറക്ടറിയിലേക്ക് മാറാനും ഉപയോഗിക്കുന്നു.
> കൂടുതൽ വിവരങ്ങൾ: https://learn.microsoft.com/windows-server/administration/windows-commands/cd.

- നിലവിലെ ഡയറക്ടറിയുടെ പാത കാണിക്കുക:

`cd`

- മേൽ ഡയറക്ടറിയിലേക്ക് (parent directory) മാറുക:

`cd ..`

- അതേ ഡ്രൈവിലുള്ള ഒരു പ്രത്യേക ഡയറക്ടറിയിലേക്ക് മാറുക:

`cd {{percorso\della\directory}}`

- Sമറ്റൊരു ഡ്രൈവിലുള്ള ഒരു പ്രത്യേക ഡയറക്ടറിയിലേക്ക് മാറുക:

`cd /d {{C}}:{{percorso\della\directory}}`
