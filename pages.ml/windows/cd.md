
# cd

> നിലവിലെ ഡയറക്ടറിയെ മാറ്റുക.
> കൂടുതൽ വിവരങ്ങൾ: https://www.gnu.org/software/bash/manual/bash.html#index-cd

- നിർദ്ദിഷ്ട ഡയറക്ടറിയിലേക്ക് പോകുക:
`cd {{path/to/directory}}`

- നിലവിലെ ഡയറക്ടറിയുടെ parent ഡയറക്ടറിയിലേക്ക് മാറുക:
`cd ..`

- നിലവിലെ ഉപയോക്താവിന്റെ home ഡയറക്ടറിയിലേക്ക് പോകുക:
`cd`

- നിർദ്ദിഷ്ട ഉപയോക്താവിന്റെ home ഡയറക്ടറിയിലേക്ക് പോകുക:
`cd ~{{username}}`

- മുമ്പ് പോയ ഡയറക്ടറിയിലേക്ക് തിരികെ പോകുക:
`cd -`

- root ഡയറക്ടറിയിലേക്ക് പോകുക:
`cd /`
