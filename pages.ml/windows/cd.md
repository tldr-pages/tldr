# cd

> നിലവിലെ പ്രവർത്തന ഡയറക്ടറി കാണിക്കുക അല്ലെങ്കിൽ മറ്റൊരു ഡയറക്ടറിയിലേക്ക് മാറുക.
> PowerShell-ൽ, ഈ കമാൻഡ് `Set-Location` എന്നതിനുള്ള ഒരു alias ആണ്.
> കൂടുതൽ വിവരങ്ങൾ: https://learn.microsoft.com/windows-server/administration/windows-commands/cd.

- സമാനമായ PowerShell കമാൻഡിന്റെ ഡോക്യുമെന്റേഷൻ കാണുക:
`tldr set-location`

- നിലവിലെ ഡയറക്ടറിയുടെ പാത പ്രദർശിപ്പിക്കുക:
`cd`

- അതേ ഡ്രൈവിലെ ഒരു നിർദ്ദിഷ്ട ഡയറക്ടറിയിലേക്ക് മാറുക:
`cd {{path\to\directory}}`

- വേറൊരു ഡ്രൈവിലെ നിർദ്ദിഷ്ട ഡയറക്ടറിയിലേക്ക് മാറുക:
`cd /d {{C}}:{{path\to\directory}}`

- നിലവിലെ ഡയറക്ടറിയുടെ മാതൃ ഡയറക്ടറിയിലേക്ക് മാറുക:
`cd ..`

- നിലവിലെ ഉപയോക്താവിന്റെ ഹോം ഡയറക്ടറിയിലേക്ക് മാറുക:
`cd %userprofile%`

- നിലവിലെ ഡ്രൈവിന്റെ root ഡയറക്ടറിയിലേക്ക് മാറുക:
`cd \`
