# fd

> Альтернатива до `find`.
> Прагне бути швидшою і простішою альтернативою `find`.
> Більше інформації: <https://github.com/sharkdp/fd#how-to-use>.

- Рекурсивно знайти файли, які відповідають певному шаблону в поточному каталозі:

`fd "{{рядок|regex}}"`

- Знайти файли, які починаються з певного рядка:

`fd "{{^рядок}}"`

- Знайти файли з певним розширенням:

`fd {{[-e|--extension]}} {{txt}}`

- Знайти файли у певному каталозі:

`fd "{{рядок|regex}}" {{шлях/до/каталогу}}`

- Включити в пошук ігноровані та приховані файли:

`fd {{[-H|--hidden]}} {{[-I|--no-ignore]}} "{{рядок|regex}}"`

- Виконати команду для кожного результату пошуку:

`fd "{{рядок|regex}}" {{[-x|--exec]}} {{команда}}`
