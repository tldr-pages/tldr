# git blame

> Показує хеш коміту та останнього автора на кожному рядку у файлі.
> Більше інформації: <https://git-scm.com/docs/git-blame>.

- Виводить файл із іменем автора та хешем коміту на кожному рядку:

`git blame {{шлях/до/файлу}}`

- Виводить файл з електронною поштою автора та хешем коміту на кожному рядку:

`git blame {{-e|--show-email}} {{шлях/до/файлу}}`

- Виводить файл лише із хешем коміту на кожному рядку:

`git blame -s {{шлях/до/файлу}}`
