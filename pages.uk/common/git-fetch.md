# git fetch

> Завантаження об'єктів та посилань з віддаленого сховища.
> Більше інформації: <https://git-scm.com/docs/git-fetch>.

- Отримати останні зміни з віддаленого сховища за замовчуванням (якщо встановлено):

`git fetch`

- Отримати нові гілки з конкретного віддаленого сховища:

`git fetch {{назва_сховища}}`

- Отримати останні зміни з усіх віддалених сховищ:

`git fetch --all`

- Також отримати мітки з віддаленого сховища:

`git fetch --tags`

- Видалити локальні посилання на віддалені гілки, які були видалені з віддаленого сховища:

`git fetch --prune`
