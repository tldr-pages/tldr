# abbr

> Відображає або змінює скорочення в оболонці.
> Більше інформації: <https://manned.org/abbr>.

- Показати всі скорочення:

`abbr`

- Додати скорочення:

`abbr -a {{скорочення}} {{команда}}`

- Видалити скорочення:

`abbr -r {{скорочення}}`

- Змінити існуюче скорочення:

`abbr -c {{старе_скорочення}} {{нове_скорочення}}`
