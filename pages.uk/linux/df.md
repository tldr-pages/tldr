# df

> Показати огляд використання дискового простору файлової системи.
> Більше інформації: <https://www.gnu.org/software/coreutils/manual/html_node/df-invocation.html>.

- Показати всі файлові системи та їх використання диска:

`df`

- Показати всі файлові системи в зручному для читання вигляді:

`df {{[-h|--human-readable]}}`

- Показати файлову систему, що містить вказаний файл або каталог:

`df {{шлях/до/файлу_чи_каталогу}}`

- Включити статистику щодо кількості вільних inode:

`df {{[-i|--inodes]}}`

- Показати файлові системи, але виключати вказані типи:

`df {{[-x|--exclude-type]}} {{squashfs}} {{[-x|--exclude-type]}} {{tmpfs}}`

- Показати типи файлових систем:

`df {{[-T|--print-type]}}`
