# ip

> Показувати/керувати маршрутизацією, пристроями, політикою маршрутизації та тунелями.
> Деякі підкоманди, такі як `address`, мають власну документацію щодо використання.
> Більше інформації: <https://manned.org/ip.8>.

- Відобразити список інтерфейсів із детальною інформацією:

`ip {{[a|address]}}`

- Відобразити список інтерфейсів із короткою інформацією про мережевий рівень:

`ip {{[-br|-brief]}} {{[a|address]}}`

- Відобразити список інтерфейсів із короткою інформацією про канальний рівень:

`ip {{[-br|-brief]}} {{[l|link]}}`

- Відобразити таблицю маршрутизації:

`ip {{[r|route]}}`

- Показати сусідів (таблиця ARP):

`ip {{[n|neighbour]}}`

- Підняти/покласти інтерфейс:

`sudo ip {{[l|link]}} {{[s|set]}} {{ethX}} {{up|down}}`

- Додати/видалити IP-адресу до/з інтерфейсу:

`sudo ip {{[a|address]}} {{add|delete}} {{ip}}/{{mask}} dev {{ethX}}`

- Додати маршрут за замовчуванням:

`sudo ip {{[r|route]}} {{[a|add]}} default via {{ip}} dev {{ethX}}`
