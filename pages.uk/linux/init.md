# init

> Менеджер рівнів запуску Linux.
> Потрібна опція компіляції SYSVINIT, якщо використовується systemd.
> Більше інформації: <https://manned.org/man/init.8>.

- Встановити систему для запуску графічного середовища:

`sudo init 5`

- Встановити систему для запуску багатокористувацького терміналу:

`sudo init 3`

- Вимкнути систему:

`init 0`

- Перезавантажити систему:

`init 6`

- Встановити систему для запуску в терміналі, де дозволено лише користувач root і без мережі:

`sudo init 1`
