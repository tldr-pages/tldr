# haxelib

> Haxe 库管理器。
> 更多信息：<https://lib.haxe.org/>。

- 搜索 Haxe 库：

`haxelib search {{keyword}}`

- 安装 Haxe 库：

`haxelib install {{libname}}`

- 安装特定版本的 Haxe 库：

`haxelib install {{libname}} {{version}}`

- 升级所有已安装的 Haxe 库：

`haxelib upgrade`

- 从 Git 仓库安装库的开发版本：

`haxelib git {{libname}} {{git_url}}`

- 卸载 Haxe 库：

`haxelib remove {{libname}}`

- 打印本地安装的 Haxe 库树：

`haxelib list`
