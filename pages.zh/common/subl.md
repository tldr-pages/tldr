# subl

> Sublime Text 编辑器。
> 更多信息：<https://www.sublimetext.com>。

- 在 Sublime Text 中打开当前目录：

`subl {{.}}`

- 在 Sublime Text 中打开文件或目录：

`subl {{path/to/file_or_directory}}`

- 打开文件并跳转到特定行号：

`subl {{path/to/file}}:{{line_number}}`

- 在当前打开的窗口中打开文件或目录：

`subl -a {{path/to/file}}`

- 在新窗口中打开文件或目录：

`subl -n {{path/to/file}}`
