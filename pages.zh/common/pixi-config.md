# pixi config

> 管理配置文件。
> 更多信息：<https://pixi.sh/latest/reference/cli/#config>。

- 编辑配置文件：

`pixi config edit`

- 列出所有配置：

`pixi config list`

- 在列表配置键前添加一个值：

`pixi config prepend {{key}} {{value}}`

- 在列表配置键后添加一个值：

`pixi config append {{key}} {{value}}`

- 将配置键设置为一个值：

`pixi config set {{key}} {{value}}`

- 取消设置一个配置键：

`pixi config unset {{key}}`
