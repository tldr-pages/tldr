# aws s3 sync

> 在本地系统与 S3 存储桶之间，或在 S3 存储桶之间递归同步文件和目录。
> 更多信息：<https://docs.aws.amazon.com/cli/latest/reference/s3/sync.html>.

- 将本地目录中的文件同步到存储桶：

`aws s3 sync {{路径/到/目录}} s3://{{存储桶_名称}}/{{路径/到/远程位置}}`

- 将存储桶中的文件同步到本地：

`aws s3 sync s3://{{存储桶_名称}}/{{路径/到/远程位置}} {{路径/到/目录}}`

- 在两个存储桶之间同步对象：

`aws s3 sync s3://{{源存储桶_名称}}/{{路径/到/远程位置}} s3://{{目标存储桶_名称}}/{{路径/到/远程位置}}`

- 将本地文件同步到 S3，同时排除特定文件或目录：

`aws s3 sync {{路径/到/目录}} s3://{{存储桶_名称}}/{{路径/到/远程位置}} --exclude {{路径/到/文件}} --exclude {{路径/到/目录}}/*`

- 在存储桶之间同步对象，并删除目标中不存在于源中的文件：

`aws s3 sync s3://{{源存储桶_名称}}/{{路径/到/远程位置}} s3://{{目标存储桶_名称}}/{{路径/到/远程位置}} --delete`

- 使用高级选项同步到 S3（设置 ACL 和存储类别）：

`aws s3 sync {{路径/到/本地目录}} s3://{{存储桶_名称}}/{{路径/到/远程位置}} --acl {{private|public-read}} --storage-class {{STANDARD_IA|GLACIER}}`

- 将文件同步到 S3，并跳过未更改的文件（比较大小和修改时间）：

`aws s3 sync {{路径/到/目录}} s3://{{存储桶_名称}}/{{路径/到/远程位置}} --size-only`

- 显示帮助信息：

`aws s3 sync help`
