# shards

> Crystal 语言的依赖管理工具。
> 更多信息：<https://crystal-lang.org/reference/the_shards_command>.

- 创建一个基本的 `shard.yml` 文件：

`shards init`

- 从 `shard.yml` 文件安装依赖：

`shards install`

- 更新所有依赖：

`shards update`

- 列出所有已安装的依赖：

`shards list`

- 显示依赖的版本：

`shards version {{path/to/dependency_directory}}`
