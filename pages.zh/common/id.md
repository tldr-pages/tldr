# id

> 显示当前用户和组的身份。
> 更多信息：<https://www.gnu.org/software/coreutils/manual/html_node/id-invocation.html>.

- 显示当前用户的 ID（UID）、组 ID（GID）以及所属的组：

`id`

- 显示当前用户的身份：

`id {{[-un|--user --name]}}`

- 以数字形式显示当前用户的身份：

`id {{[-u|--user]}}`

- 显示当前用户的主组身份：

`id {{[-gn|--group --name]}}`

- 以数字形式显示当前用户的主组身份：

`id {{[-g|--group]}}`

- 显示任意用户的 ID（UID）、组 ID（GID）以及所属的组：

`id {{username}}`
