# v

> 用于管理 V 源代码的工具。
> 更多信息：<https://docs.vlang.io/getting-started.html>.

- 编译文件并输出其可执行文件：

`v {{path/to/file.v}}`

- 编译当前项目或目录：

`v .`

- 编译文件并运行：

`v run {{path/to/file.v}}`

- 编译文件并运行，同时输出其可执行文件：

`v crun {{path/to/file.v}}`

- 在每次修改文件时重新编译：

`v watch {{path/to/file.v}}`

- 在每次修改文件时重新运行：

`v watch run {{path/to/file.v}}`

- 打开 v 交互式命令行：

`v repl`

- 格式化文件并保存：

`v fmt -w {{path/to/file.v}}`
