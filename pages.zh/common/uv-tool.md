# uv tool

> 安装和运行由 Python 软件包提供的命令。
> 更多信息：<https://docs.astral.sh/uv/reference/cli/#uv-tool>.

- 运行一个来自软件包的命令，而不安装它：

`uv tool run {{命令}}`

- 在系统范围内安装一个 Python 软件包：

`uv tool install {{软件包}}`

- 升级已安装的 Python 软件包：

`uv tool upgrade {{软件包}}`

- 卸载一个 Python 软件包：

`uv tool uninstall {{软件包}}`

- 列出系统范围内已安装的 Python 软件包：

`uv tool list`
