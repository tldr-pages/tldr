# ps-nvm

> 基于 PowerShell 的工具，用于管理多个 Node.js 版本，灵感来源于 `nvm`。
> 此工具提供的命令都只能通过 PowerShell 运行。
> 更多信息：<https://github.com/aaronpowell/ps-nvm>。

- 查看 `Get-NodeInstallLocation` 的文档，该工具用于获取当前 Node.js 的安装位置：

`tldr get-nodeinstalllocation`

- 查看 `Get-NodeVersions` 的文档，该工具用于列出所有可用和已安装的 Node.js 版本：

`tldr get-nodeversions`

- 查看 `Install-NodeVersion` 的文档，该工具用于安装 Node.js 运行时版本：

`tldr install-nodeversion`

- 查看 `Remove-NodeVersion` 的文档，该工具用于卸载已安装的 Node.js 版本：

`tldr remove-nodeversion`

- 查看 `Set-NodeInstallLocation` 的文档，该工具用于设置 Node.js 的安装位置：

`tldr set-nodeinstalllocation`

- 查看 `Set-NodeVersion` 的文档，该工具用于设置默认的 Node.js 版本：

`tldr set-nodeversion`
