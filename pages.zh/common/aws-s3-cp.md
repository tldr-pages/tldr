# aws s3 cp

> 将本地文件或 S3 对象复制到本地或 S3 中的另一个位置。
> 更多信息：<https://docs.aws.amazon.com/cli/latest/reference/s3/cp.html>.

- 将文件从本地复制到指定的存储桶：

`aws s3 cp {{路径/到/文件}} s3://{{存储桶名称}}/{{路径/到/远程文件}}`

- 将指定的 S3 对象复制到另一个存储桶：

`aws s3 cp s3://{{存储桶名称1}}/{{路径/到/文件}} s3://{{存储桶名称2}}/{{路径/到/目标}}`

- 将指定的 S3 对象复制到另一个存储桶，并保留原始名称：

`aws s3 cp s3://{{存储桶名称1}}/{{路径/到/文件}} s3://{{存储桶名称2}}`

- 递归地将 S3 对象复制到本地目录：

`aws s3 cp s3://{{存储桶名称}} . --recursive`

- 显示帮助：

`aws s3 cp help`
