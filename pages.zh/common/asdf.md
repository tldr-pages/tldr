# asdf

> 可扩展的包版本管理器，支持Nodejs,Ruby,Elixir,Erlang等.
> 详见: <https://asdf-vm.com>.

- 可用插件清单:

`asdf plugin-list-all`

- 安装插件:

`asdf plugin-add {{name}}`

- 软件包的可用版本清单:

`asdf list-all {{name}}`

- 安装指定版本的软件包:

`asdf install {{name}} {{version}}`

- 设置软件包的全局安装版本:

`asdf global {{name}} {{version}}`

- 设置软件包的本地版本:

`asdf local {{name}} {{version}}`
