# asdf

> 可扩展的包版本管理器，支持 Nodejs、Ruby、Elixir 和 Erlang 等。
> 更多信息：<https://asdf-vm.com>.

- 可用插件清单：

`asdf plugin-list-all`

- 安装插件：

`asdf plugin-add {{插件名}}`

- 软件包的可用版本清单：

`asdf list-all {{软件包名}}`

- 安装指定版本的软件包：

`asdf install {{软件包名}} {{版本}}`

- 设置软件包的全局安装版本：

`asdf global {{软件包名}} {{版本}}`

- 设置软件包的本地版本：

`asdf local {{软件包名}} {{版本}}`
