# git commits-since

> 显示自某个时间或日期以来的提交。
> 是 `git-extras` 的一部分。
> 更多信息：<https://github.com/tj/git-extras/blob/master/Commands.md#git-commits-since>.

- 显示自昨天以来的提交：

`git commits-since {{yesterday}}`

- 显示自上周以来的提交：

`git commits-since {{last week}}`

- 显示自上个月以来的提交：

`git commits-since {{last month}}`

- 显示自昨天下午2点以来的提交：

`git commits-since {{yesterday 2pm}}`
