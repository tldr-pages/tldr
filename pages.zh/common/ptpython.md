# ptpython

> 更好的 Python 交互式解释器。
> 更多信息：<https://github.com/prompt-toolkit/ptpython>。

- 启动一个交互式 shell：

`ptpython`

- 执行特定的 Python 文件：

`ptpython {{path/to/file.py}}`

- 执行特定的 Python 文件并启动交互式 shell：

`ptpython -i {{path/to/file.py}}`

- 打开菜单：

`<F2>`

- 打开历史记录页面：

`<F3>`

- 切换粘贴模式：

`<F6>`

- 退出：

`<Ctrl d>`
