# cs

> Scala 语言的应用程序和工件管理器。
> 安装 Scala 应用程序并设置 Scala 开发环境。
> 一些子命令（如 `install`、`launch`、`java`、`fetch`、`resolve`、`complete-dep` 等）有其自己的使用文档。
> 更多信息：<https://get-coursier.io/docs/overview>.

- 列出已安装的应用程序：

`cs list`

- 安装特定的应用程序：

`cs install {{application_name}}`

- 卸载特定的应用程序：

`cs uninstall {{application_name}}`

- 配置机器以进行 Scala 开发：

`cs setup`

- 更新所有已安装的应用程序：

`cs update`

- 显示版本：

`cs version`
