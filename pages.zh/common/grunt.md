# grunt

> 用于自动化流程的 JavaScript 任务运行器。
> 更多信息：<https://github.com/gruntjs/grunt-cli>.

- 运行默认任务流程：

`grunt`

- 运行一个或多个任务：

`grunt {{task1 task2 ...}}`

- 指定替代配置文件：

`grunt --gruntfile {{path/to/file}}`

- 指定相对文件的替代基准路径：

`grunt --base {{path/to/directory}}`

- 指定一个额外的任务扫描目录：

`grunt --tasks {{path/to/directory}}`

- 执行预演（不写入任何文件）：

`grunt --no-write`

- 显示帮助：

`grunt --help`
