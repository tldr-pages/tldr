# pio lib

> 管理 PlatformIO 库。
> 更多信息：<https://docs.platformio.org/en/latest/core/userguide/lib/>.

- 列出已安装的库：

`pio lib list`

- 列出基于已安装开发平台及其框架的内置库：

`pio lib builtin`

- 搜索现有的库：

`pio lib search {{keyword}}`

- 显示库的详细信息：

`pio lib show {{library}}`

- 安装库：

`pio lib install {{library}}`

- 更新已安装的库：

`pio lib update`

- 卸载库：

`pio lib uninstall {{library}}`

- 显示 PlatformIO 库注册表统计信息：

`pio lib stats`
