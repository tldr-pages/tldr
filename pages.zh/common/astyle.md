# astyle

> 用于 C、C++、C# 和 Java 编程语言的源代码缩进、格式化和美化工具。
> 运行时，会创建原始文件的一个副本，并在原始文件名后追加 `.orig`。
> 更多信息：<https://manned.org/astyle>.

- 应用默认样式，每级缩进 4 个空格且不进行其他格式化更改：

`astyle {{源文件}}`

- 应用 Java 样式，使用附加的大括号：

`astyle {{[-A2|--style=java]}} {{路径/到/文件}}`

- 应用 Allman 样式，使用拆分的大括号：

`astyle {{[-A1|--style=allman]}} {{路径/到/文件}}`

- 使用空格应用自定义缩进，可选择 2 到 20 个空格：

`astyle {{[-s|--indent=spaces=]}}{{空格数量}} {{路径/到/文件}}`

- 使用制表符应用自定义缩进，可选择 2 到 20 个制表符：

`astyle {{[-t|--indent=tab=]}}{{制表符数量}} {{路径/到/文件}}`
