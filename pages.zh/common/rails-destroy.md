# rails destroy

> 删除 Rails 资源。
> 更多信息：<https://guides.rubyonrails.org/command_line.html#bin-rails-destroy>。

- 列出所有可删除的生成器：

`rails destroy`

- 删除名为 Post 的模型：

`rails destroy model {{Post}}`

- 删除名为 Posts 的控制器：

`rails destroy controller {{Posts}}`

- 删除创建 Posts 的迁移：

`rails destroy migration {{CreatePosts}}`

- 删除名为 Post 的模型的脚手架：

`rails destroy scaffold {{Post}}`
