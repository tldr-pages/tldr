# pulumi up

> 创建或更新堆栈中的资源。
> 更多信息：<https://www.pulumi.com/docs/iac/cli/commands/pulumi_up/>.

- 预览并部署程序和/或基础架构的更改：

`pulumi up`

- 预览后自动批准并执行更新：

`pulumi up --yes`

- 预览并部署特定堆栈的更改：

`pulumi up --stack {{stack}}`

- 不显示堆栈输出：

`pulumi up --suppress-outputs`

- 即使遇到错误也继续更新资源：

`pulumi up --continue-on-error`
