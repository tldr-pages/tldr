# yadm-config

> 传递选项给 `yadm` 的配置文件。更改由 `yadm` 管理的仓库的 `.config`。
> 更多信息：<https://github.com/TheLocehiliosan/yadm/blob/master/yadm.md#configuration>.

- 设置或更新 `yadm` 的 Git 配置：

`yadm config {{键.内键}} {{值}}`

- 从 `yadm` 的 Git 配置中获取一个值：

`yadm config --get {{键}}`

- 在 `yadm` 的 Git 配置中取消设置一个值：

`yadm config --unset {{键}}`

- 列出 `yadm` 的 Git 配置中的所有值：

`yadm config --list`
