# meteor

> 用于构建 Web 应用程序的全栈 JavaScript 平台。
> 更多信息：<https://meteor.com>.

- 以开发模式从项目根目录运行 meteor 项目：

`meteor`

- 在指定目录下创建项目：

`meteor create {{path/to/directory}}`

- 显示项目当前使用的所有包列表：

`meteor list`

- 向项目中添加一个包：

`meteor add {{package}}`

- 从项目中移除一个包：

`meteor remove {{package}}`

- 在指定目录下创建项目的生产构建 tarball：

`meteor build {{path/to/directory}}`
