# pre-commit

> 创建在提交前运行的 Git 钩子。
> 更多信息：<https://pre-commit.com>.

- 将 pre-commit 安装到你的 Git 钩子中：

`pre-commit install`

- 在所有暂存的文件上运行 pre-commit 钩子：

`pre-commit run`

- 在所有文件上运行 pre-commit 钩子，无论文件是否暂存：

`pre-commit run --all-files`

- 清除 pre-commit 缓存：

`pre-commit clean`

- 将 pre-commit 配置文件更新到最新版本的仓库：

`pre-commit autoupdate`
