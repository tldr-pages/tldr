# firebase

> 测试、管理和部署 Firebase 项目。
> 更多信息：<https://github.com/firebase/firebase-tools>.

- 登录 <https://console.firebase.google.com>:

`firebase login`

- 列出现有的 Firebase 项目:

`firebase projects:list`

- 在当前目录中启动一个交互向导来创建 Firebase 项目:

`firebase init`

- 部署代码和资源到当前的 Firebase 项目:

`firebase deploy`

- 启动本地服务器以静态托管当前 Firebase 项目的资源:

`firebase serve`

- 启动交互向导以在默认浏览器中打开当前 Firebase 项目的一个或多个链接:

`firebase open`
