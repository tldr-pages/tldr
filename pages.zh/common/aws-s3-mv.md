# aws s3 mv

> 将本地文件或 S3 对象移动到本地或 S3 中的另一个位置。
> 更多信息：<https://docs.aws.amazon.com/cli/latest/reference/s3/mv.html>.

- 将文件从本地移动到指定的存储桶：

`aws s3 mv {{路径/到/本地文件}} s3://{{存储桶名称}}/{{路径/到/远程文件}}`

- 将指定的 S3 对象移动到另一个存储桶：

`aws s3 mv s3://{{存储桶名称1}}/{{路径/到/文件}} s3://{{存储桶名称2}}/{{路径/到/目标}}`

- 将指定的 S3 对象移动到另一个存储桶，并保留原始名称：

`aws s3 mv s3://{{存储桶名称1}}/{{路径/到/文件}} s3://{{存储桶名称2}}`

- 显示帮助信息：

`aws s3 mv help`
