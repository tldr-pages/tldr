# npm run

> 运行脚本。
> 更多信息：<https://docs.npmjs.com/cli/commands/npm-run-script>.

- 运行脚本：

`npm run {{script_name}}`

- 向脚本传递参数：

`npm run {{script_name}} -- {{argument}} {{--option}}`

- 运行名为 `start` 的脚本：

`npm start`

- 运行名为 `stop` 的脚本：

`npm stop`

- 运行名为 `restart` 的脚本：

`npm restart`

- 运行名为 `test` 的脚本：

`npm test`
