# pio platform

> 管理 PlatformIO 开发平台。
> 更多信息：<https://docs.platformio.org/en/latest/core/userguide/platforms/>。

- 列出所有已安装的开发平台：

`pio platform list`

- 搜索现有开发平台：

`pio platform search {{platform}}`

- 显示开发平台的详细信息：

`pio platform show {{platform}}`

- 安装开发平台：

`pio platform install {{platform}}`

- 更新已安装的开发平台：

`pio platform update`

- 卸载开发平台：

`pio platform uninstall {{platform}}`

- 列出所有支持的框架：

`pio platform frameworks`
