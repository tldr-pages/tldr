# moro

> 跟踪工作时间。
> 更多信息：<https://moro.js.org>。

- 调用 `moro` 不带参数，将当前时间设置为工作日的开始时间：

`moro`

- 指定自定义时间作为工作日的开始时间：

`moro hi {{09:30}}`

- 再次调用 `moro` 不带参数，将当前时间设置为工作日的结束时间：

`moro`

- 指定自定义时间作为工作日的结束时间：

`moro bye {{17:30}}`

- 为当前工作日添加备注：

`moro note {{在项目 Foo 上花费 3 小时}}`

- 显示当前工作日的时间日志和备注报告：

`moro report`

- 显示所有记录工作日的时间日志和备注报告：

`moro report --all`
