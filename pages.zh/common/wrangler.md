# wrangler

> Cloudflare Workers 命令行工具。
> 更多信息：<https://developers.cloudflare.com/workers/>。

- 使用骨架配置初始化项目：

`wrangler init {{project_name}}`

- 使用 Cloudflare 进行身份验证：

`wrangler login`

- 启动本地开发服务器：

`wrangler dev --host {{hostname}}`

- 发布 Worker 脚本：

`wrangler publish`

- 汇总生产 Worker 的日志：

`wrangler tail`
