# rails server

> 使用随 Rails 捆绑的 Puma Web 服务器在当前目录中运行 Rails 应用。
> 更多信息：<https://guides.rubyonrails.org/command_line.html#bin-rails-server>。

- 运行 Web 服务器：

`rails server`

- 在指定的端口上运行 Web 服务器：

`rails server --port {{port_number}}`

- 在指定的 IP 地址上运行 Web 服务器：

`rails server --binding {{ip_address}}`

- 在指定的环境中运行 Web 服务器：

`rails server --environment {{environment}}`

- 显示帮助信息：

`rails server --help`
