# php artisan

> Laravel 的 Artisan 命令行界面。
> 更多信息：<https://laravel.com/docs/artisan>.

- 为当前的 Laravel 应用启动 PHP 内置的 Web 服务器：

`php artisan serve`

- 启动交互式的 PHP 命令行界面：

`php artisan tinker`

- 生成包含迁移、工厂和资源控制器的新 Eloquent 模型类：

`php artisan make:model {{ModelName}} --all`

- 显示所有可用命令的列表：

`php artisan help`
