# git count-objects

> 计算未打包对象的数量及其磁盘占用。
> 更多信息：<https://git-scm.com/docs/git-count-objects>.

- 计算所有对象并显示总磁盘使用量：

`git count-objects`

- 显示所有对象的数量及其总磁盘使用量，以人类可读的单位显示大小：

`git count-objects --human-readable`

- 显示更详细的信息：

`git count-objects --verbose`

- 显示更详细的信息，并以人类可读的单位显示大小：

`git count-objects --human-readable --verbose`
