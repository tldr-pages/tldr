# aws s3

> AWS S3 的 CLI - 通过 Web 服务接口提供存储。
> 此命令也有关于其子命令的文件，例如：`cp`.
> 更多信息：<https://docs.aws.amazon.com/cli/latest/reference/s3/>.

- 显示存储桶中的文件：

`aws s3 ls {{存储桶名称}}`

- 将本地目录中的文件同步到存储桶：

`aws s3 sync {{路径/到/目录}} s3://{{存储桶名称}}`

- 将存储桶中的文件和目录同步到本地：

`aws s3 sync s3://{{存储桶名称}} {{路径/到/目标}}`

- 同步目录中的文件，并排除指定内容：

`aws s3 sync {{路径/到/目录}} s3://{{存储桶名称}} --exclude {{路径/到/文件}} --exclude {{路径/到/目录}}/*`

- 从存储桶中删除文件：

`aws s3 rm s3://{{存储桶}}/{{路径/到/文件}}`

- 仅预览将要发生的更改：

`aws s3 {{任意命令}} --dryrun`
