# truffle

> 为在以太坊区块链上运行的服务开发智能合约。
> 更多信息：<https://www.trufflesuite.com/docs/truffle/reference/truffle-commands>。

- 下载预构建的 Truffle 项目（Truffle Box）：

`truffle unbox {{box_name}}`

- 编译当前目录中的合约源文件：

`truffle compile`

- 运行 JavaScript 和 Solidity 测试：

`truffle test`

- 运行迁移以部署合约：

`truffle migrate`

- 显示子命令的帮助信息：

`truffle help {{subcommand}}`
