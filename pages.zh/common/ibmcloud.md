# ibmcloud

> 用于管理和操作 IBM Cloud 应用和服务的命令行工具。
> 更多信息：<https://cloud.ibm.com/docs/cli?topic=cli-ibmcloud_cli>。

- 更新 `ibmcloud` 到最新版本：

`ibmcloud update`

- 安装 Cloud Foundry 模块以访问 Cloud Foundry 服务：

`ibmcloud cf install`

- 列出所有可用的 IBM Cloud 区域：

`ibmcloud regions`

- 显示帮助信息：

`ibmcloud help`

- 显示子命令的帮助信息：

`ibmcloud help {{subcommand}}`

- 显示版本信息：

`ibmcloud version`
