# docker cp

> 在主机和容器文件系统之间复制文件或目录。
> 更多信息：<https://docs.docker.com/reference/cli/docker/container/cp/>.

- 从主机复制文件或目录到容器：

`docker cp {{主机上的文件或目录路径}} {{容器名}}:{{容器内的文件或目录路径}}`

- 从容器复制文件或目录到主机：

`docker cp {{容器名}}:{{容器内的文件或目录路径}} {{主机上的文件或目录路径}}`

- 从主机复制文件或目录到容器，并跟随符号链接（复制符号链接指向的文件，而不是符号链接本身）：

`docker cp --follow-link {{主机上的符号链接路径}} {{容器名}}:{{容器内的文件或目录路径}}`
