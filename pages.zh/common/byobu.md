# byobu

> 窗口管理器和终端复用器。
> 另见：`tmux` 和 `screen`。
> 更多信息：<https://www.byobu.org/documentation>。

- 启动新会话：

`byobu`

- 配置和帮助：

`byobu-config`

- 选择 tmux（默认）或 screen 后端：

`byobu-select-backend`

- 启用登录文本控制台后的自动启动：

`byobu-enable`

- 禁用登录文本控制台后的自动启动：

`byobu-disable`

- 从 `byobu` 分离：

`<F6>`

- 关闭一个窗口：

`<Ctrl a><k>`
