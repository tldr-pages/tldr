# IPython

> 一个具有自动历史记录、动态对象检查、更容易配置、命令补全、访问系统 shell 等功能的 Python shell。
> 更多信息：<https://ipython.readthedocs.io>。

- 启动一个 REPL（交互式 shell）：

`ipython`

- 运行 Python 脚本后进入交互式 IPython 会话：

`ipython -i {{script.py}}`

- 创建默认的 IPython 配置文件：

`ipython profile create`

- 打印默认 IPython 配置文件目录的路径：

`ipython locate profile`

- 清除 IPython 历史记录数据库，删除所有条目：

`ipython history clear`
