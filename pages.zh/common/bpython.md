# bpython

> 一个带有高级特性的 Python 解释器界面。
> 提供语法高亮和许多其他交互式命令行中实用的功能。
> 更多信息：<https://manned.org/bpython>.

- 启动交互式 shell (REPL)：

`bpython`

- 执行特定的 Python 文件：

`bpython {{path/to/file.py}}`

- 执行特定的 Python 文件并启动交互式 shell：

`bpython {{[-i|--interactive]}} {{path/to/file.py}}`

- 使用指定的配置文件而不是默认配置：

`bpython --config {{path/to/file.conf}}`
