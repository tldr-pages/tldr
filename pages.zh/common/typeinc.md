# typeinc

> 一个基于 `ncurses` 的终端打字速度测试程序，使用 Python 编写。
> 尝试不同的难度级别，提高你的打字速度。
> 更多信息：<https://github.com/AnirudhG07/Typeinc>.

- 进入打字测试：

`typeinc`

- 显示指定难度级别的前十名排行榜：

`typeinc {{[-r|--ranklist]}} {{difficulty_level}}`

- 获取我们词库中随机的英文单词：

`typeinc {{[-w|--words]}} {{word_count}}`

- 计算假设的 Typeinc 分数：

`typeinc {{[-s|--score]}}`
