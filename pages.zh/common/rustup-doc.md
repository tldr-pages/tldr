# rustup doc

> 打开当前工具链的离线 Rust 文档。
> 还有很多未在此列出的文档页面。更多信息请参见 `rustup help doc`。
> 更多信息：<https://rust-lang.github.io/rustup>。

- 打开主页面：

`rustup doc`

- 打开特定主题（标准库中的一个模块、一个类型、一个关键字等）的文档：

`rustup doc {{std::fs|usize|fn|...}}`

- 打开《Rust 编程语言》书籍：

`rustup doc --book`

- 打开 Cargo 书籍：

`rustup doc --cargo`

- 打开 Rust 参考手册：

`rustup doc --reference`
