# nodenv

> 管理 Node.js 版本。
> 更多信息：<https://github.com/nodenv/nodenv>.

- 安装特定版本的 Node.js：

`nodenv install {{version}}`

- 显示可用版本列表：

`nodenv install --list`

- 在整个系统中使用特定版本的 Node.js：

`nodenv global {{version}}`

- 在目录中使用特定版本的 Node.js：

`nodenv local {{version}}`

- 显示当前目录的 Node.js 版本：

`nodenv version`

- 显示已安装的 Node.js 命令的位置（例如 `npm`）：

`nodenv which {{command}}`
