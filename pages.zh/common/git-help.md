# git help

> 显示 Git 的帮助信息。
> 更多信息：<https://git-scm.com/docs/git-help>.

- 显示指定 Git 子命令的帮助信息：

`git help {{subcommand}}`

- 在网络浏览器中显示指定 Git 子命令的帮助信息：

`git help {{[-w|--web]}} {{subcommand}}`

- 列出所有可用的 Git 子命令：

`git help {{[-a|--all]}}`

- 列出可用的指南：

`git help {{[-g|--guides]}}`

- 列出所有配置变量的名称：

`git help {{[-c|--config]}}`
