# fnm

> 快速的 Node.js 版本管理器。
> 安装、卸载或切换 Node.js 版本。
> 更多信息：<https://github.com/Schniz/fnm>。

- 安装特定版本的 Node.js：

`fnm install {{node_version}}`

- 列出所有可用的 Node.js 版本并高亮显示默认版本：

`fnm list`

- 在当前 shell 中使用特定版本的 Node.js：

`fnm use {{node_version}}`

- 设置默认的 Node.js 版本：

`fnm default {{node_version}}`

- 卸载指定的 Node.js 版本：

`fnm uninstall {{node_version}}`
