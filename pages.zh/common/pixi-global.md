# pixi global

> 管理全局包。
> 更多信息：<https://pixi.sh/latest/reference/cli/#global>。

- 全局安装包并添加到路径：

`pixi global install {{package1 package2 ...}}`

- 全局卸载包：

`pixi global remove {{package1 package2 ...}}`

- 列出所有已全局安装的包：

`pixi global list`

- 更新已全局安装的包：

`pixi global upgrade {{package}}`

- 更新所有已全局安装的包：

`pixi global upgrade-all`
