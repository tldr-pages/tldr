# rails console

> 与 Rails 应用程序进行交互。
> 更多信息：<https://guides.rubyonrails.org/command_line.html#bin-rails-console>。

- 启动 Rails 控制台：

`rails console`

- 启动 Rails 控制台并在退出时回滚所有数据修改：

`rails console --sandbox`

- 在指定的环境中启动 Rails 控制台：

`rails console --environment {{dev|test|production|...}}`

- 显示帮助：

`rails console --help`
