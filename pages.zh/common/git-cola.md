# git cola

> 一个具有流畅且直观用户界面的强大 Git 图形界面工具。
> 更多信息：<https://git-cola.readthedocs.io>。

- 启动图形界面：

`git cola`

- 以修正模式启动图形界面：

`git cola --amend`

- 提示选择一个 Git 仓库。默认为当前目录：

`git cola --prompt`

- 打开指定路径的 Git 仓库：

`git cola --repo {{path/to/git-repository}}`

- 对状态小部件应用路径过滤器：

`git cola --status-filter {{filter}}`
