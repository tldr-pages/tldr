# az

> Microsoft Azure 的官方 CLI 工具。
> 此命令也有关于其子命令的文件，例如：`login`.
> 更多信息：<https://learn.microsoft.com/cli/azure/reference-index>.

- 登录到 Azure：

`az login`

- 管理 Azure 订阅信息：

`az account`

- 列出所有 Azure 托管磁盘：

`az disk list`

- 列出所有 Azure 虚拟机：

`az vm list`

- 管理 Azure Kubernetes 服务：

`az aks`

- 管理 Azure 网络资源：

`az network`

- 以交互模式启动：

`az interactive`

- 显示帮助信息：

`az --help`
