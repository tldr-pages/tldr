# zinit

> 灵活且极其快速的 Zsh 插件管理器。
> 更多信息：<https://github.com/zdharma-continuum/zinit>。

- 显示所有已安装的插件：

`zinit plugins`

- 进入某个插件的目录：

`zinit cd {{仓库所有者}}/{{仓库名称}}`

- 从 GitHub 获取最新版的 `zinit`：

`zinit self-update`

- 列出所有已安装补全的状态：

`zinit completions`

- 为某个插件安装补全：

`zinit creinstall {{仓库所有者}}/{{仓库名称}}`

- 删除某个插件的补全：

`zinit cuninstall {{仓库所有者}}/{{仓库名称}}`
