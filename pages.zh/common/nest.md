# nest

> 初始化、开发和维护 Nest 应用程序。
> 更多信息：<https://docs.nestjs.com/cli/overview>.

- 显示已安装的 nest 版本信息：

`nest info`

- 在与项目同名的目录中创建一个新的 NestJS 项目：

`nest new {{project_name}}`

- 构建特定的 NestJS 项目：

`nest build {{project_name}}`

- 运行特定的 NestJS 项目：

`nest start {{project_name}}`

- 将库导入当前的 NestJS 项目：

`nest add {{library_name}}`
