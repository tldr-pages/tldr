# railway

> 将代码连接到 Railway 项目。
> 更多信息：<https://railway.app/>.

- 登录 Railway 账号：

`railway login`

- 将现有项目链接到 Railway 账号或团队：

`railway link {{projectId}}`

- 创建新项目：

`railway init`

- 使用活动环境中的变量运行本地命令：

`railway run {{cmd}}`

- 部署已链接的项目目录（如果从子目录运行，仍将部署项目根目录）：

`railway up`

- 打开与数据库的交互式 shell：

`railway connect`
