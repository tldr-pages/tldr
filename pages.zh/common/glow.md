# glow

> 在终端渲染 Markdown.
> 更多信息：<https://github.com/charmbracelet/glow>.

- 运行 glow 并选择文件查看：

`glow`

- 渲染 Markdown 文件到终端：

`glow {{路径/到/文件}}`

- 使用分页器查看 Markdown 文件：

`glow -p {{路径/到/文件}}`

- 从 URL 查看文件：

`glow {{https://example.com/file.md}}`

- 查看 GitHub 或 GitLab 的 README：

`glow {{github.com/owner/repository}}`
