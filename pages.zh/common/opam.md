# opam

> OCaml 软件包管理器。
> 管理 OCaml 编译器、工具和库。
> 更多信息：<https://opam.ocaml.org/>。

- 初始化 opam 以供首次使用：

`opam init`

- 搜索软件包：

`opam search {{query}}`

- 安装一个软件包及其所有依赖项：

`opam install {{package}}`

- 显示软件包的详细信息：

`opam show {{package}}`

- 列出所有已安装的软件包：

`opam list`

- 更新本地软件包数据库：

`opam update`

- 升级所有已安装的软件包：

`opam upgrade`

- 显示帮助：

`opam help`
