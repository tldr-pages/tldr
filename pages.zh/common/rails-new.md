# rails new

> 创建新的 Rails 应用程序。
> 更多信息：<https://guides.rubyonrails.org/command_line.html#rails-new>.

- 在当前目录中创建名为 `blog` 的 Rails 应用程序：

`rails new blog`

- 创建仅包含 API 配置的 Rails 应用程序：

`rails new {{app_name}} --api`

- 创建使用 `postgresql` 作为数据库的 Rails 应用程序：

`rails new {{app_name}} --database postgresql`

- 创建不生成 JavaScript 文件的 Rails 应用程序：

`rails new {{app_name}} --skip-javascript`

- 显示帮助：

`rails new --help`
