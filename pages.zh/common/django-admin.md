# django-admin

> Django 的管理任务工具。
> 更多信息：<https://docs.djangoproject.com/en/4.2/ref/django-admin/>.

- 创建一个新的 Django 项目：

`django-admin startproject {{project_name}}`

- 为当前项目创建一个新的应用：

`django-admin startapp {{app_name}}`

- 查看当前安装的 Django 版本：

`django-admin --version`

- 显示特定命令的帮助信息：

`django-admin help {{command}}`
