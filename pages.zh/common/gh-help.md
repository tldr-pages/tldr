# gh help

> 显示关于 GitHub CLI 命令的帮助信息。
> 更多信息：<https://cli.github.com/manual/gh_help>。

- 显示常规帮助信息：

`gh help`

- 显示 `gh help` 子命令的帮助信息：

`gh help --help`

- 显示 `gh` 可用的环境变量的帮助信息：

`gh help environment`

- 显示所有 `gh` 命令的 Markdown 参考：

`gh help reference`

- 显示使用 `jq` 格式化 `gh` 输出的 JSON 的帮助信息：

`gh help formatting`

- 显示在 MinTTY 中使用 `gh` 的帮助信息：

`gh help mintty`

- 显示子命令的帮助信息：

`gh help {{subcommand}}`

- 显示子命令操作的帮助信息：

`gh help {{pr}} {{create}}`
