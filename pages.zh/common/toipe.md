# toipe

> 又一个打字测试，但这次是蟹味的。
> 一个可靠的终端打字测试工具。
> 更多信息：<https://github.com/Samyak2/toipe>.

- 使用默认单词列表开始打字测试：

`toipe`

- 使用特定的单词列表：

`toipe {{[-w|--wordlist]}} {{wordlist_name}}`

- 使用自定义单词列表：

`toipe {{[-f|--file]}} {{path/to/file}}`

- 指定每次测试中的单词数量：

`toipe {{[-n|--num]}} {{number_of_words}}`

- 包含标点符号：

`toipe {{[-p|--punctuation]}}`
