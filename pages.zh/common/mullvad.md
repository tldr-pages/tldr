# mullvad

> Mullvad VPN 的命令行客户端。
> 参见：`fastd`, `ivpn`, `mozillavpn`, `warp-cli`。
> 更多信息：<https://mullvad.net/>。

- 使用指定的账户号码关联您的 Mullvad 账户：

`mullvad account set {{account_number}}`

- 在启用 VPN 时允许局域网访问：

`mullvad lan set allow`

- 建立 VPN 隧道连接：

`mullvad connect`

- 检查 VPN 隧道的状态：

`mullvad status`
