# pixi project

> 修改项目配置文件。
> 更多信息：<https://pixi.sh/latest/reference/cli/#project>。

- 管理项目渠道：

`pixi project channel {{command}}`

- 管理项目描述：

`pixi project description {{command}}`

- 管理项目平台：

`pixi project platform {{command}}`

- 管理项目版本：

`pixi project version {{command}}`

- 管理项目环境：

`pixi project environment {{command}}`
