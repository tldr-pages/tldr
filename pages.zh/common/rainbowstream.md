# rainbowstream

> 基于终端的 Twitter 客户端，支持实时推文流、趋势、发送、搜索、收藏和用户管理。
> 在线帮助：<`h`>，使用 <`ArrowUp`> 和 <`ArrowDown`> 浏览历史记录，<`Tab`> 用于自动补全，双击 <`Tab`> 获取建议。
> 更多信息：<https://github.com/orakaro/rainbowstream>.

- 打开 RainbowStream：

`rainbowstream`

- 显示您的时间线（可选显示最近的推文数量，默认为 5 条）：

`home [{{num_of_last_tweets}}]`

- 显示指定用户的资料：

`whois @{{user}}`

- 发布一条推文：

`t {{message}}`

- 转发指定 ID 的推文（ID 位于时间旁边）：

`rt {{tweet_id}}`

- 收藏指定 ID 的推文：

`fav {{tweet_id}}`

- 搜索指定词语（带或不带话题标签）：

`s {{word}}`
