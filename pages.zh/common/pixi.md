# pixi

> 项目开发流程和环境管理工具。
> 更多信息：<https://pixi.sh>.

- 初始化新项目：

`pixi init {{path/to/project}}`

- 添加项目依赖：

`pixi add {{dependency1 dependency2 ...}}`

- 在项目环境中启动 pixi shell：

`pixi shell`

- 在项目环境中运行任务：

`pixi run {{task}}`

- 管理项目环境中的任务：

`pixi task {{command}}`

- 打印帮助信息：

`pixi {{command}} --help`

- 清理环境和任务缓存：

`pixi clean`
