# pixi

> 针对项目的开发人员工作流与环境管理。
> 更多信息：<https://pixi.prefix.dev/latest/getting_started/>。

- 初始化一个新项目：

`pixi init {{路径/到/项目}}`

- 添加项目依赖：

`pixi add {{依赖项1 依赖项2 ...}}`

- 在项目环境中启动 pixi shell：

`pixi shell`

- 在项目环境中运行一个任务：

`pixi run {{任务}}`

- 在项目环境中管理任务：

`pixi task {{命令}}`

- 显示帮助：

`pixi {{命令}} --help`

- 清理环境和任务缓存：

`pixi clean`
