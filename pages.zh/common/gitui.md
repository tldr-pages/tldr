# gitui

> 一个轻量级的仅键盘操作的 Git 图形界面。
> 相关工具：`tig`, `git-gui`。
> 更多信息：<https://github.com/extrawurst/gitui>。

- 指定颜色主题（默认为 `theme.ron`）：

`gitui --theme {{theme}}`

- 将日志输出存储到缓存目录中：

`gitui --logging`

- 使用通知基础的文件系统监视器而不是基于定时更新：

`gitui --watcher`

- 生成一个错误报告：

`gitui --bugreport`

- 使用特定的 Git 目录：

`gitui --directory {{path/to/directory}}`

- 使用特定的工作目录：

`gitui --workdir {{path/to/directory}}`

- 显示帮助信息：

`gitui --help`

- 显示版本信息：

`gitui --version`
