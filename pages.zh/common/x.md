# x

> x-cmd：一个用于管理模块和软件包的命令行工具。
> 更多信息：<https://x-cmd.com>。

* 显示 x-cmd 的帮助信息：

`x -h`

* 使用特定模块（例如 path 模块）：

`x path`

* 安装并使用软件包（例如 jq）：

`x env use jq`

* 在当前 Shell 会话中临时尝试软件包：

`x env try jq`

* 无需安装直接使用软件包：

`x jq`

* 初始化 OpenAI API 密钥以使用 AI 功能：

`x openai init`

* 开启与 AI 助手的对话：

`@gpt`

* 使用 pixi 软件包管理器：

`x pixi`

* 查看安装方法：

`x install`
