# gatsby

> 为 React 设计的静态站点生成器。
> 更多信息：<https://gatsbyjs.com>。

- 创建新站点：

`gatsby new {{site_name}}`

- 使用 Gatsby 'starter' 创建新站点：

`gatsby new {{site_name}} {{url_of_starter_github_repo}}`

- 启动带有实时重载功能的本地开发服务器：

`gatsby develop`

- 执行生产构建并生成静态 HTML：

`gatsby build`

- 启动提供生产构建的本地服务器：

`gatsby serve`
