# git create-branch

> 在 Git 仓库中创建分支。
> 属于 `git-extras` 一部分。
> 更多信息：<https://manned.org/git-create-branch>.

- ​​创建本地分支​​：

`git create-branch {{分支名}}`

- ​​创建本地分支并推送到 origin 远程​​：

`git create-branch --remote {{分支名}}`

- ​​创建本地分支并推送到 upstream 远程（常用于 fork 的工作流）​​：

`git create-branch --remote upstream {{分支名}}`
