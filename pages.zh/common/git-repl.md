# git repl

> Git REPL（读取-求值-打印循环） - 一个交互式的 Git shell。
> 属于 `git-extras` 的一部分。
> 更多信息：<https://github.com/tj/git-extras/blob/master/Commands.md#git-repl>。

- 启动交互式的 Git shell：

`git repl`

- 在交互式的 Git shell 中运行 Git 命令：

`{{git_subcommand}} {{command_arguments}}`

- 在交互式的 Git shell 中运行外部（非 Git）命令：

`!{{command}} {{command_arguments}}`

- 退出交互式的 Git shell（或按 `<Ctrl d>`）：

`exit`
