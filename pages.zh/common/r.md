# R

> R 语言解释器。
> 更多信息：<https://www.r-project.org>。

- 启动 REPL（交互式壳）：

`R`

- 以纯模式启动 R（即空白会话，不保存工作区）：

`R --vanilla`

- 执行文件：

`R -f {{path/to/file.R}}`

- 执行 R 表达式然后退出：

`R -e {{expr}}`

- 用调试器运行 R：

`R -d {{debugger}}`

- 从包源检查 R 包：

`R CMD check {{path/to/package_source}}`

- 显示版本：

`R --version`
