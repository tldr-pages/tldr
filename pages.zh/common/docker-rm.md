# docker rm

> 移除容器。
> 更多信息：<https://docs.docker.com/reference/cli/docker/container/rm/>.

- 移除容器：

`docker rm {{container1 container2 ...}}`

- 强制移除容器：

`docker rm {{[-f|--force]}} {{container1 container2 ...}}`

- 移除容器及其卷：

`docker rm {{[-v|--volumes]}} {{container}}`

- 显示帮助：

`docker rm {{[-h|--help]}}`
