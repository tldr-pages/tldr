# tlmgr shell

> 启动 TeX Live 管理器的交互式 shell。
> 更多信息：<https://www.tug.org/texlive/tlmgr.html>.

- 启动 `tlmgr` 的交互式 shell：

`tlmgr shell`

- 在交互式 shell 中运行任何 `tlmgr` 子命令：

`{{subcommand}} {{arguments}}`

- 退出交互式 shell：

`quit`

- 列出所有 TeX Live 变量：

`get`

- 获取 TeX Live 变量的值：

`get {{variable}}`

- 设置 TeX Live 变量的值：

`set {{variable}} {{value}}`

- 重启交互式 shell：

`restart`

- 显示当前协议的版本：

`protocol`
