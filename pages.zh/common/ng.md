# ng

> 创建和管理 Angular 应用。
> 更多信息：<https://angular.io/cli>.

- 在一个目录中创建一个新的 Angular 应用：

`ng new {{project_name}}`

- 为应用添加一个新的组件：

`ng generate component {{component_name}}`

- 为应用添加一个新的类：

`ng generate class {{class_name}}`

- 为应用添加一个新的指令：

`ng generate directive {{directive_name}}`

- 在应用的根目录下运行以下命令启动应用：

`ng serve`

- 构建应用：

`ng build`

- 运行单元测试：

`ng test`

- 显示当前 Angular 安装的版本：

`ng version`
