# avo

> Avo 的官方接口。
> 更多信息：<https://www.avo.app/docs/implementation/cli>.

- 在当前目录中初始化一个工作区：

`avo init`

- 登录到 Avo 平台：

`avo login`

- 切换到一个现有的 Avo 分支：

`avo checkout {{分支名}}`

- 拉取当前路径的分析包装器：

`avo pull`

- 显示 Avo 实现的状态：

`avo status`

- 解决 Avo 文件中的 Git 冲突：

`avo conflict`

- 在默认的 Web 浏览器中打开当前的 Avo 工作区：

`avo edit`

- 显示某个子命令的帮助信息：

`avo {{子命令}} --help`
