# zapier versions

> 列出 Zapier 集成的版本。
> 更多信息：<https://github.com/zapier/zapier-platform/blob/main/packages/cli/docs/cli.md#validate>。

- 列出某个集成的版本：

`zapier versions`

- 包含已弃用的版本：

`zapier versions {{[-a|--all]}}`

- 显示额外的调试输出：

`zapier versions {{[-d|--debug]}}`

- 使用不同的输出结构列出版本：

`zapier versions {{[-f|--format]}} {{plain|json|raw|row|table}}`
