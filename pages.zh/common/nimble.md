# nimble

> Nim 编程语言的包管理器。
> 管理 Nim 项目及其依赖关系。
> 更多信息：<https://github.com/nim-lang/nimble>。

- 搜索包：

`nimble search {{search_string}}`

- 安装包：

`nimble install {{package}}`

- 列出已安装的包：

`nimble list -i`

- 在当前目录中创建一个新的 Nimble 包：

`nimble init`

- 构建 Nimble 包：

`nimble build`

- 安装 Nimble 包：

`nimble install`
