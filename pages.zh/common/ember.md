# ember

> Ember CLI：创建和管理 Ember.js 应用程序。
> 更多信息：<https://cli.emberjs.com>。

- 创建一个新的 Ember 应用程序：

`ember new {{my_new_app}}`

- 创建一个新的 Ember 插件：

`ember addon {{my_new_addon}}`

- 构建项目：

`ember build`

- 以生产模式构建项目：

`ember build -prod`

- 运行开发服务器：

`ember serve`

- 运行测试套件：

`ember test`

- 运行蓝图以生成路由或组件等：

`ember generate {{type}} {{name}}`

- 安装一个 ember-cli 插件：

`ember install {{name_of_addon}}`
