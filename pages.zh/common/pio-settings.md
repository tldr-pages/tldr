# pio settings

> 查看和修改 PlatformIO 设置。
> 更多信息：<https://docs.platformio.org/en/latest/core/userguide/cmd_settings.html>。

- 显示所有 PlatformIO 设置的名称、值和描述：

`pio settings get`

- 显示特定 PlatformIO 设置的名称、值和描述：

`pio settings get {{setting}}`

- 设置特定设置的值：

`pio settings set {{setting}} {{value}}`

- 将所有已修改的设置值重置为其出厂默认值：

`pio settings reset`
