# tag

> 在 Mac OS X 文件上编辑标签（10.9 Mavericks 及以上版本）。
> 更多信息：<https://github.com/jdberry/tag/>.

- 向文件添加标签：

`tag --add {{tag_name1,tag_name2,...}} {{path/to/file}}`

- 移除文件的标签：

`tag --remove {{tag_name}} {{path/to/file}}`

- 移除文件的所有标签：

`tag --remove \* {{path/to/file}}`

- 显示具有指定标签的所有文件：

`tag --match {{tag_name}}`
