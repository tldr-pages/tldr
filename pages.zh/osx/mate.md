# mate

> macOS 的通用文本编辑器。
> 更多信息：<https://macromates.com/>。

- 启动 TextMate:

`mate`

- 打开特定文件:

`mate {{path/to/file1 path/to/file2 ...}}`

- 指定文件的类型:

`mate --type {{filetype}} {{path/to/file}}`

- 打开文件并等待编辑完成:

`mate --wait {{path/to/file}}`

- 打开文件并将光标定位在特定的行和列:

`mate --line {{line_number}}:{{column_number}} {{path/to/file}}`
