# pkgctl

> Arch Linux 开发工具的统一命令行前端。
> 更多信息：<https://manned.org/pkgctl.1>。

- 查看有关使用 GitLab 等服务进行 `pkgctl` 身份验证的文档：

`tldr pkgctl auth`

- 查看有关在干净的 `chroot` 中构建包的文档：

`tldr pkgctl build`

- 查看有关作为最终发布步骤更新二进制仓库的文档：

`tldr pkgctl db update`

- 查看有关使用不同模式比较包文件的文档：

`tldr pkgctl diff`

- 查看有关发布构建工件的文档：

`tldr pkgctl release`

- 查看有关管理和配置 Git 包仓库的文档：

`tldr pkgctl repo`

- 显示版本：

`pkgctl version`
