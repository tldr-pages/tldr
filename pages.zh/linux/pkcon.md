# pkcon

> PackageKit 控制台程序的命令行客户端，用于 Discover 和 Gnome 软件，是 'apt' 的替代品。
> 更多信息：<https://manned.org/pkcon>.

- 安装软件包：

`pkcon install {{package}}`

- 卸载软件包：

`pkcon remove {{package}}`

- 刷新软件包缓存：

`pkcon refresh`

- 更新软件包：

`pkcon update`

- 搜索特定的软件包：

`pkcon search {{package}}`

- 列出所有可用的软件包：

`pkcon get-packages`
