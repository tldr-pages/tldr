# larasail

> 管理 Digital Ocean 服务器上的 Laravel。
> 更多信息：<https://github.com/thedevdojo/larasail>。

- 使用默认的 PHP 版本设置服务器和 Laravel 依赖项：

`larasail setup`

- 使用特定的 PHP 版本设置服务器和 Laravel 依赖项：

`larasail setup {{php71}}`

- 添加一个新的 Laravel 站点：

`larasail host {{domain}} {{path/to/site_directory}}`

- 获取 Larasail 用户密码：

`larasail pass`

- 获取 Larasail MySQL 密码：

`larasail mysqlpass`
