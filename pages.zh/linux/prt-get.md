# prt-get

> CRUX 包管理器。
> 更多信息：<https://crux.nu/doc/prt-get%20-%20User%20Manual.html>.

- 安装一个包：

`prt-get install {{package}}`

- 安装一个包并处理依赖关系：

`prt-get depinst {{package}}`

- 手动更新一个包：

`prt-get upgrade {{package}}`

- 删除一个包：

`prt-get remove {{package}}`

- 从本地端口树升级系统：

`prt-get sysup`

- 搜索端口树：

`prt-get search {{query}}`

- 在包中搜索文件：

`prt-get fsearch {{file}}`
