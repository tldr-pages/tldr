# toilet

> 显示 ASCII 艺术字体。
> 更多信息：<http://caca.zoy.org/wiki/toilet>。

- 为给定的文本生成 ASCII 艺术字：

`toilet {{input_text}}`

- 使用自定义字体文件生成 ASCII 艺术字：

`toilet {{input_text}} -f {{font_filename}}`

- 使用过滤器生成 ASCII 艺术字：

`toilet {{input_text}} --filter {{filter_name}}`

- 显示可用的 toilet 过滤器：

`toilet --filter list`
