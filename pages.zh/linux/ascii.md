# ascii

> 显示 ASCII 字符的别名。
> 更多信息：<http://www.catb.org/~esr/ascii/>.

- 显示一个 ASCII 字符的别名：

`ascii {{a}}`

- 以较短和脚本友好模式显示 ASCII 字符的别名：

`ascii -t {{a}}`

- 显示多个 ASCII 字符的别名：

`ascii -s {{tldr}}`

- 显示十进制 ASCII 字符表：

`ascii -d`

- 显示十六进制 ASCII 字符表：

`ascii -x`

- 显示八进制 ASCII 字符表：

`ascii -o`

- 显示二进制 ASCII 字符表：

`ascii -b`

- 显示选项总结和整个 ASCII 字符表：

`ascii`
