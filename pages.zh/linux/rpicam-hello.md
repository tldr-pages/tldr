# rpicam-hello

> 使用 Raspberry Pi 摄像头查看实时摄像头流。
> 更多信息：<https://www.raspberrypi.com/documentation/computers/camera_software.html#rpicam-hello>.

- 显示特定持续时间（以毫秒为单位）的摄像头预览流：

`rpicam-hello -t {{time}}`

- 调整特定摄像头传感器的配置：

`rpicam-hello --tuning-file {{/usr/share/libcamera/ipa/rpi/path/to/config.json}}`
