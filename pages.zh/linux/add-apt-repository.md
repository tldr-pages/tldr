# add-apt-repository

> apt 仓库管理。
> 更多信息：<https://manned.org/apt-add-repository>.

- 添加一个新的 apt 仓库：

`add-apt-repository {{指定仓库}}`

- 移除一个 apt 仓库：

`add-apt-repository --remove {{指定仓库}}`

- 添加一个仓库并更新缓存：

`add-apt-repository --update {{指定仓库}}`

- 允许从指定仓库下载源码：

`add-apt-repository --enable-source {{指定仓库}}`
