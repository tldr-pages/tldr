# gnome-software

> 添加和删除应用程序，更新系统。
> 更多信息：<https://apps.gnome.org/app/org.gnome.Software/>.

- 如果 GNOME Software 未运行，启动 GNOME Software 图形界面：

`gnome-software`

- 如果 GNOME Software 未打开，启动 GNOME Software 图形界面并导航到指定页面：

`gnome-software --mode {{updates|updated|installed|overview}}`

- 如果 GNOME Software 未打开，启动 GNOME Software 图形界面并查看指定软件包的详细信息：

`gnome-software --details {{package}}`

- 显示版本：

`gnome-software --version`
