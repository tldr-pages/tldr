# cmd

> Windows命令解释器.

- 开启一个新的命令行实例:

`cmd`

- 运行指定的命令然后退出:

`cmd /c "{{命令}}"`

- 执行一个指定的命令,之后进入一个交互式shell:

`cmd /k "{{命令}}"`

- 不显示命令的输出结果:

`cmd /q`

- 启用或禁用命令扩展:

`cmd /e:{{on|off}}`

- 启用或禁用文件和目录名的自动补全:

`cmd /f:{{on|off}}`

- 启用或禁用环境变量扩展:

`cmd /v:{{on|off}}`

- 强制输出内容使用Unicode编码:

`cmd /u`
