# eslint

> Uma ferramenta de linting plugável para JavaScript e JSX.
> Mais informações: <https://eslint.org>.

- Cria o arquivo de configuração do ESLint:

`eslint --init`

- Faz lint de um ou mais arquivos:

`eslint {{caminho/para/arquivo1.js caminho/para/arquivo2.js ...}}`

- Corrige problemas de lint:

`eslint --fix`

- Faz lint usando o arquivo de configuração especificado:

`eslint {{[-c|--config]}} {{caminho/para/arquivo_de_configuracao}} {{caminho/para/arquivo1.js caminho/para/arquivo2.js ...}}`
