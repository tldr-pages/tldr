# tldr

> Exibe páginas de ajuda simples para ferramentas de linha de comando do projeto tldr-pages.
> Mais informações: <https://tldr.sh>.

- Imprimir a página do tldr para um comando específico (dica: foi isso que você chegou aqui!)

`tldr {{comando}}`

- Imprimir a página do tldr para um subcomando específico:

`tldr {{comando}}-{{subcomando}}`

- Imprimir a página do tldr para um comando para uma [p]lataforma específica:

`tldr -p {{android|linux|osx|sunos|windows}} {{comando}}`

- Atualizar o cache local das páginas do tldr:

`tldr -u`
