# anki

> Um poderoso programa de flashcards.
> Mais informações em: <https://docs.ankiweb.net>.

- Inicializa a interface gráfica:

`anki`

- Use um perfil específico:

`anki {{[-p|--profile]}} {{nome_perfil}}`

- Use um idioma específico:

`anki {{[-l|--lang]}} {{idioma}}`

- Inicializa no modo seguro (sem addons nem sincronização automatica):

`anki --safemode`

- Use uma pasta diferente (o padrão é `~/Anki`):

`anki {{[-b|--base]}} {{caminho/para/pasta}}`
