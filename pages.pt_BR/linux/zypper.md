# zypper

> Utilitário de gerenciamento de pacotes SUSE e openSUSE.
> Mais informações: <https://en.opensuse.org/SDB:Zypper_manual>.

- Sincroniza a lista de pacotes e versões disponíveis:

`zypper refresh`

- Instalar um novo pacote:

`zypper install {{pacote}}`

- Remover um pacote:

`zypper remove {{pacote}}`

- Atualizar os pacotes instalados para as versões mais recentes disponíveis:

`zypper update`

- Pesquisar pacote por palavra-chave:

`zypper search {{palavra-chave}}`

- Mostrar informações relacionadas aos repositórios configurados:

`zypper repos --sort-by-priority`
