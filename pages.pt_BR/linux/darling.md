# darling

> Executa software do macOS no Linux..
> Mais informações: <https://darlinghq.org>.

- Executa um comando integrado:

`darling shell {{uname}}`

- Executa um comando específico com argumentos:

`darling shell {{caminho/para/programa}} {{argumento_do_programa_1 argumento_do_programa_2 ...}}`

- Abre um shell do macOS:

`darling shell`

- Encerra o serviço:

`darling shutdown`
