# bugreportz

> Genera un report di bug di Android in formato compresso (.zip).
> Questo comando puÃ² essere usato solo tramite `adb shell`.
> Maggiori informazioni: <https://cs.android.com/android/platform/superproject/+/main:frameworks/native/cmds/bugreportz>.

- Genera un report di bug completo in formato compresso di un dispositivo Android:

`bugreportz`

- Mostra l'avanzamento di un'operazione `bugreportz` in esecuzione:

`bugreportz -p`

- Scrive il contenuto di un report di bug Android su `stdout`:

`bugreportz -s`

- Mostra la guida:

`bugreportz -h`

- Mostra la versione:

`bugreportz -v`
