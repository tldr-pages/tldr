# pm list packages

> Elenca i pacchetti installati, conosciuti o filtrati su un dispositivo Android.
> More information: <https://developer.android.com/tools/adb#pm>.

- Elenca tutti i pacchetti installati:

`pm list packages`

- Elenca tutti i pacchetti e i percorsi associati dei file APK:

`pm list packages -f`

- Elenca solo i pacchetti [d]isabilitati:

`pm list packages -d`

- Elenca solo i pacchetti abilitati:

`pm list packages -e`

- Elenca solo i pacchetti di [s]istema:

`pm list packages -s`

- Elenca solo i pacchetti di terze parti (non di sistema):

`pm list packages -3`

- Mostra il [i]nstaller per ogni pacchetto:

`pm list packages -i`
