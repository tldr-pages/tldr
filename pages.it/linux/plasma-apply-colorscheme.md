# plasma-apply-colorscheme

> Cambia lo schema di colori di Plasma.
> Maggiori informazioni: <https://invent.kde.org/plasma/plasma-workspace>.

- Elenca gli schemi di colori disponibili:

`plasma-apply-colorscheme --list-schemes`

- Applica uno schema di colori:

`plasma-apply-colorscheme {{BreezeLight}}`

- Mostra aiuto:

`plasma-apply-colorscheme --help`
