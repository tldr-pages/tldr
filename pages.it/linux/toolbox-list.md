# toolbox list

> Elenca i contenitori e le immagini Toolbx esistenti.
> Maggiori informazioni: <https://manned.org/toolbox-list>.

- Elenca tutti i contenitori e le immagini Toolbx:

`toolbox list`

- Elenca solo i contenitori Toolbx:

`toolbox list {{[-c|--containers]}}`

- Elenca solo le immagini Toolbx:

`toolbox list {{[-i|--images]}}`
