# gnome-extensions

> Gestisce le estensioni di GNOME dal terminale.
> Maggiori informazioni: <https://wiki.gnome.org/Projects/GnomeShell/Extensions>.

- Elenca tutte le estensioni installate:

`gnome-extensions list`

- Mostra informazioni su un'estensione specifica:

`gnome-extensions info "{{extension_id}}"`

- Abilita un'estensione specifica:

`gnome-extensions enable "{{extension_id}}"`

- Disabilita un'estensione specifica:

`gnome-extensions disable "{{extension_id}}"`

- Disinstalla un'estensione specifica:

`gnome-extensions uninstall "{{extension_id}}"`

- Mostra aiuto per un sottocomando specifico (come `list`):

`gnome-extensions help {{subcommand}}`

- Mostra versione:

`gnome-extensions version`
