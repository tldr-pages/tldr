# distrobox-upgrade

> Aggiorna uno o piÃ¹ container Distrobox.
> Vedi anche: `distrobox`.
> Maggiori informazioni: <https://distrobox.it/usage/distrobox-upgrade/>.

- Aggiorna un container utilizzando il gestore di pacchetti nativo del container:

`distrobox-upgrade {{container_name}}`

- Aggiorna tutti i container utilizzando i gestori di pacchetti nativi dei container:

`distrobox-upgrade {{[-a|--all]}}`

- Aggiorna container specifici tramite il gestore di pacchetti nativo del container:

`distrobox-upgrade {{container1 container2 ...}}`
