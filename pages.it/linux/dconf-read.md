# dconf read

> Legge i valori delle chiavi dai database dconf.
> Vedi anche: `dconf`.
> Maggiori informazioni: <https://manned.org/dconf>.

- Stampa il valore di una chiave specifica:

`dconf read /{{path/to/key}}`

- Stampa il valore predefinito [d]i una chiave specifica:

`dconf read -d /{{path/to/key}}`
