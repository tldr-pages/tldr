# gnome-calculator

> La calcolatrice ufficiale per l'ambiente desktop GNOME.
> Maggiori informazioni: <https://wiki.gnome.org/Apps/Calculator>.

- Avvia l'interfaccia grafica GNOME Calculator:

`gnome-calculator`

- Risolve l'equazione specificata senza avviare l'applicazione desktop:

`gnome-calculator --solve {{2^5 * 2 + 5}}`

- Mostra versione:

`gnome-calculator --version`
