# distrobox

> Usa qualsiasi distribuzione Linux all'interno del tuo terminale in un container. Installa e usa pacchetti al suo interno integrandosi strettamente con l'OS host, condividendo storage (directory `home`) e hardware.
> `distrobox` non Ã¨ un comando, ma un insieme di comandi.
> Nota: Usa Podman o Docker per creare i tuoi container.
> Maggiori informazioni: <https://github.com/89luca89/distrobox>.

- Visualizza documentazione per creare container:

`tldr distrobox-create`

- Visualizza documentazione per elencare le informazioni dei container:

`tldr distrobox-list`

- Visualizza documentazione per entrare nel container:

`tldr distrobox-enter`

- Visualizza documentazione per eseguire un comando sull'host da dentro un container:

`tldr distrobox-host-exec`

- Visualizza documentazione per esportare app/servizio/binario dal container all'host:

`tldr distrobox-export`

- Visualizza documentazione per aggiornare i container:

`tldr distrobox-upgrade`

- Visualizza documentazione per fermare i container:

`tldr distrobox-stop`

- Visualizza documentazione per rimuovere i container:

`tldr distrobox-rm`
