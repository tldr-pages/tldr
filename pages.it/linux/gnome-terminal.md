# gnome-terminal

> L'emulatore di terminale GNOME.
> Maggiori informazioni: <https://help.gnome.org/users/gnome-terminal/stable/>.

- Apre una nuova finestra del terminale GNOME:

`gnome-terminal`

- Esegue un comando specifico in una nuova finestra del terminale:

`gnome-terminal -- {{command}}`

- Apre invece una nuova scheda nell'ultima finestra aperta:

`gnome-terminal --tab`

- Imposta il titolo della nuova scheda:

`gnome-terminal --tab {{[-t|--title]}} "{{title}}"`
