# ark

> Strumento di archiviazione di KDE.
> Maggiori informazioni: <https://docs.kde.org/stable5/en/ark/ark/>.

- Estrae un archivio specifico nella directory corrente:

`ark {{[-b|--batch]}} {{percorso/dell/archivio}}`

- Estrae un archivio in una directory specifica:

`ark {{[-b|--batch]}} {{[-o|--destination]}} {{percorso/della/directory}} {{percorso/dell/archivio}}`

- Crea un archivio se non esiste e aggiunge file specifici:

`ark {{[-t|--add-to]}} {{percorso/dell/archivio}} {{percorso/del/file1 percorso/del/file2 ...}}`
