# distrobox-list

> Elenca tutti i container Distrobox.
> I container Distrobox sono elencati separatamente dal resto dei normali container Podman o Docker.
> Vedi anche: `distrobox`.
> Maggiori informazioni: <https://distrobox.it/usage/distrobox-list>.

- Elenca tutti i container Distrobox:

`distrobox-list`

- Elenca tutti i container Distrobox con informazioni dettagliate:

`distrobox-list {{[-v|--verbose]}}`
