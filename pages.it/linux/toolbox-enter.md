# toolbox enter

> Entra in un contenitore Toolbx per uso interattivo.
> Vedi anche: `toolbox run`.
> Maggiori informazioni: <https://manned.org/toolbox-enter>.

- Entra in un contenitore Toolbx usando l'immagine predefinita di una distribuzione specifica:

`toolbox enter {{[-d|--distro]}} {{distribuzione}}`

- Entra in un contenitore Toolbx usando l'immagine predefinita di una versione specifica della distribuzione corrente:

`toolbox enter {{[-r|--release]}} {{versione}}`

- Entra in un contenitore Toolbx usando l'immagine predefinita per una versione Fedora specifica:

`toolbox enter {{[-d|--distro]}} {{fedora}} {{[-r|--release]}} f{{versione}}`
