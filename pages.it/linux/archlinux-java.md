# archlinux-java

> Passa tra gli ambienti Java installati.
> Maggiori informazioni: <https://wiki.archlinux.org/title/Java#Switching_between_JVM>.

- Elenca gli ambienti Java installati:

`archlinux-java status`

- Restituisce il nome abbreviato dell'ambiente Java predefinito attuale:

`archlinux-java get`

- Imposta l'ambiente Java predefinito:

`archlinux-java set {{java_environment}}`

- Annulla l'impostazione dell'ambiente Java predefinito:

`archlinux-java unset`

- Corregge una configurazione non valida o danneggiata dell'ambiente Java predefinito:

`archlinux-java fix`
