# pw-reserve

> Riserva un dispositivo per PipeWire.
> Maggiori informazioni: <https://docs.pipewire.org/page_man_pw-reserve_1.html>.

- Riserva un dispositivo (attualmente supporta solo dispositivi audio):

`pw-reserve {{[-n|--name]}} {{audioN}}`

- Monitora un dispositivo invece di riservarlo:

`pw-reserve {{[-n|--name]}} {{audioN}} {{[-m|--monitor]}}`

- Mostra aiuto:

`pw-reserve {{[-h|--help]}}`
