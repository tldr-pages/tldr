# cockpit-tls

> Proxy HTTP con terminazione TLS per cifrare il traffico tra un client e `cockpit-ws`.
> Maggiori informazioni: <https://cockpit-project.org/guide/latest/cockpit-tls.8.html>.

- Serve richieste HTTP su una porta specifica invece della porta `9090`:

`cockpit-tls --port {{port}}`

- Mostra aiuto:

`cockpit-tls --help`
