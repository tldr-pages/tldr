# apt install

> Installa pacchetti per distribuzioni basate su Debian.
> Maggiori informazioni: <https://manned.org/apt.8>.

- Installa un pacchetto, o lo aggiorna all'ultima versione:

`sudo apt install {{pacchetto}}`

- Mostra informazioni verbose sulla versione del pacchetto durante l'installazione o l'aggiornamento:

`sudo apt install {{[-V|--verbose-versions]}} {{pacchetto}}`
