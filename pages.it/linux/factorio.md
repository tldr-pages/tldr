# Factorio

> Crea e avvia un server Factorio headless.
> Maggiori informazioni: <https://wiki.factorio.com/Multiplayer>.

- Crea un nuovo file di salvataggio:

`{{percorso/a/factorio}} --create {{percorso/al/file_salvataggi.zip}}`

- Avvia un server Factorio:

`{{percorso/a/factorio}} --start-server {{percorso/al/file_salvataggi.zip}}`

- Crea un nuovo file di salvataggio con impostazioni specifiche:

`{{percorso/a/factorio}} --create {{percorso/a/save.zip}} --map-gen-settings {{percorso/al/map-gen-settings.json}} --map-settings {{percorso/al/map-settings.json}}`

- Arresta il server in modo elegante:

`/quit`

- Mostra informazioni di aiuto:

`{{percorso/a/factorio}} {{[-h|--help]}}`
