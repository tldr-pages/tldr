# plasma-apply-lookandfeel

> Cambia i temi globali di Plasma.
> Maggiori informazioni: <https://userbase.kde.org/System_Settings/Look_And_Feel>.

- Elenca i temi globali disponibili:

`plasma-apply-lookandfeel --list`

- Applica un tema globale:

`plasma-apply-lookandfeel --apply {{org.example.theme.desktop}}`

- Opera `plasma-apply-lookandfeel` senza un server di visualizzazione:

`plasma-apply-lookandfeel --platform offscreen`

- Mostra aiuto:

`plasma-apply-lookandfeel --help`
