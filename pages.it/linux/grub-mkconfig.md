# grub-mkconfig

> Genera un file di configurazione GRUB.
> Maggiori informazioni: <https://www.gnu.org/software/grub/manual/grub/grub.html#Invoking-grub_002dmkconfig>.

- Esegue una prova e stampa la configurazione su `stdout`:

`sudo grub-mkconfig`

- Genera il file di configurazione:

`sudo grub-mkconfig {{[-o|--output]}} {{/boot/grub/grub.cfg}}`

- Mostra aiuto:

`grub-mkconfig {{[-h|--help]}}`
