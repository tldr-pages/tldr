# tree

> Visualizza la struttura di directory di un percorso come grafico ad albero.
> Maggiori informazioni: <https://learn.microsoft.com/windows-server/administration/windows-commands/tree>.

- Mostra l'albero per la directory corrente:

`tree`

- Mostra l'albero per una directory specifica:

`tree {{percorso\della\directory}}`

- Mostra l'albero per la directory includendo i [f]iles:

`tree {{percorso\della\directory}} /f`

- Mostra l'albero usando caratteri [a]SCII anziche' caratteri estesi:

`tree {{percorso\della\directory}} /a`
