# powershell

> Interfaccia a riga di comando e linguaggio di script appositamente progettato per l'amministrazione.
> Maggiori informazioni: <https://learn.microsoft.com/windows-server/administration/windows-commands/powershell>.

- Avvia una sessione di Windows PowerShell in una finestra del prompt dei comandi:

`powershell`

- Carica un file di console PowerShell specifico:

`powershell -PSConsoleFile {{percorso/del/file}}`

- Avvia una sessione con una versione specificata di PowerShell:

`powershell -Version {{versione}}`

- Impedisce la chiusura dell'interfaccia di sistema dopo aver eseguito i comandi di avvio:

`powershell -NoExit`

- Descrive il formato dei dati inviati a PowerShell:

`powershell -InputFormat {{Testo|XML}}`

- Determina come formattare l'output di PowerShell:

`powershell -OutputFormat {{Testo|XML}}`

- Visualizza il messaggio d'aiuto:

`powershell -Help`
