# transport

> Gestisce i protocolli di trasporto delle linee.
> Accessibile in modalitÃ  configurazione sotto `line`.
> Maggiori informazioni: <https://www.cisco.com/c/en/us/td/docs/routers/sdwan/command/iosxe/qualified-cli-command-reference-guide/m-line-commands.pdf>.

- Limita i protocolli della linea a `ssh`:

`transport input ssh`

- Limita i protocolli della linea a `telnet`:

`transport input telnet`
