# show

> Mostra varie informazioni di sistema.
> Maggiori informazioni: <https://www.cisco.com/c/en/us/td/docs/switches/datacenter/nexus5000/sw/command/reference/fund/n5k-fund-cr/n5k-fund_cmds_show.html>.

- Mostra gli indirizzi IP dello switch:

`{{[sh|show]}} ip interface brief`

- Mostra la configurazione di un'interfaccia specifica:

`{{[sh|show]}} ip interface {{vlan1}}`

- Mostra la configurazione vlan:

`{{[sh|show]}} vlan`

- Mostra la configurazione attualmente in esecuzione:

`{{[sh|show]}} running-config`

- Mostra la configurazione SSH:

`{{[sh|show]}} ip ssh`
