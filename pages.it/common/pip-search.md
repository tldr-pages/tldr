# pip search

> Cerca pacchetti Python per nome o sommario.
> Non funziona con PyPI; potrebbe funzionare con altri indici di pacchetti.
> Maggiori informazioni: <https://pip.pypa.io/en/stable/cli/pip_search/>.

- Cerca pacchetti utilizzando un indice di pacchetti personalizzato:

`pip search {{[-i|--index]}} {{esempio.com}} {{query}}`
