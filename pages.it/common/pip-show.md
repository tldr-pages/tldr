# pip show

> Mostra informazioni sui pacchetti installati.
> Maggiori informazioni: <https://pip.pypa.io/en/stable/cli/pip_show/>.

- Mostra informazioni su un pacchetto:

`pip show {{pacchetto}}`

- Mostra tutte le informazioni su un pacchetto:

`pip show {{[-v|--verbose]}} {{pacchetto}}`

- Mostra tutti i file installati per un pacchetto:

`pip show {{[-f|--files]}} {{pacchetto}}`
