# 2to3

> Conversione automatica del codice Python 2 in Python 3.
> Questo modulo è stato deprecato dalla versione 3.11 ed eliminato dalla 3.13.
> Per riferimento, vedi: <https://github.com/python/cpython/blob/8d42e2d915c3096e7eac1c649751d1da567bb7c3/Doc/whatsnew/3.13.rst?plain=1#L188>.
> Maggiori informazioni: <https://manned.org/2to3>.

- Mostra le modifiche che verrebbero effettuate senza applicarle (dry-run):

`2to3 {{percorso/del/file.py}}`

- Converte un file Python 2 in Python 3:

`2to3 {{[-w|--write]}} {{percorso/del/file.py}}`

- Converte specifiche feature del linguaggio Python 2 in Python 3:

`2to3 {{[-w|--write]}} {{percorso/del/file.py}} {{[-f|--fix]}} {{raw_input}} {{[-f|--fix]}} {{print}}`

- Converte tutte le feature Python 2 tranne quelle specificate in Python 3:

`2to3 {{[-w|--write]}} {{percorso/del/file.py}} {{[-x|--nofix]}} {{has_key}} {{-x|--nofix]}} {{isinstance}}`

- Elenca tutte le feature del linguaggio disponibili per la conversione da Python 2 a Python 3:

`2to3 {{[-l|--list-fixes]}}`

- Converte tutti i file Python 2 in una directory in Python 3:

`2to3 {{[-o|--output-dir]}} {{percorso/della/directory_python3}} {{[-W|--write-unchanged-files}} {{[-n|--nobackups]}} {{percorso/della/directory_python2}}`

- Esegue 2to3 con più thread:

`2to3 {{[-j|--processes]}} {{4}} {{[-o|--output-dir]}} {{percorso/della/directory_python3}} {{[-w|--write]}} {{-n|--nobackups]}} --no-diffs {{percorso/della/directory_python2}}`
