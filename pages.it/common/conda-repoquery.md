# conda repoquery

> Ricerca avanzata pacchetti repository.
> Maggiori informazioni: <https://docs.conda.io/projects/conda/en/latest/commands/repoquery/index.html>.

- Versioni disponibili:

`conda repoquery search {{pacchetto}}`

- Dipendenze pacchetto:

`conda repoquery depends {{pacchetto}}`

- Pacchetti dipendenti:

`conda repoquery whoneeds {{pacchetto}}`
