# az version

> Mostra la versione corrente dei moduli ed estensioni Azure CLI.
> Parte di `azure-cli` (noto anche come `az`).
> Maggiori informazioni: <https://learn.microsoft.com/cli/azure/reference-index#az-version>.

- Mostra la versione corrente dei moduli ed estensioni Azure CLI in formato JSON:

`az version`

- Mostra la versione corrente dei moduli ed estensioni Azure CLI in un formato specificato:

`az version {{[-o|--output]}} {{json|table|tsv}}`
