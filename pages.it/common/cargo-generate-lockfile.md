# cargo generate-lockfile

> Genera il file `Cargo.lock` per il pacchetto corrente. Simile a `cargo update`, ma con meno opzioni.
> Se il lockfile esiste già, verrà ricreato con le ultime versioni di ogni pacchetto.
> Maggiori informazioni: <https://doc.rust-lang.org/stable/cargo/commands/cargo-generate-lockfile.html>.

- Genera un file `Cargo.lock` con le ultime versioni di ogni pacchetto:

`cargo generate-lockfile`
