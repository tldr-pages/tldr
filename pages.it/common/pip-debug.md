# pip debug

> Mostra informazioni di debug sull'ambiente e sulla configurazione corrente di pip.
> Pensato per il debug; le opzioni e l'output potrebbero cambiare senza preavviso.
> Maggiori informazioni: <https://pip.pypa.io/en/stable/cli/pip_debug/>.

- Mostra informazioni generali di debug:

`pip debug`

- Mostra informazioni di debug per una specifica piattaforma:

`pip debug --platform {{piattaforma}}`

- Mostra informazioni di debug per una specifica versione di Python:

`pip debug --python-version {{versione}}`

- Mostra informazioni di debug per una specifica implementazione di Python:

`pip debug --implementation {{implementazione}}`

- Show debug information for a specific ABI:

`pip debug --abi {{abi}}`
