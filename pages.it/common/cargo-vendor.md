# cargo vendor

> Metti in vendor tutte le dipendenze di un progetto nella directory specificata (predefinita: `vendor`).
> Maggiori informazioni: <https://doc.rust-lang.org/cargo/commands/cargo-vendor.html>.

- Metti in vendor le dipendenze e configura `cargo` per usare le sorgenti vendorizzate nel progetto corrente:

`cargo vendor {{percorso/della/directory}} > .cargo/config.toml`
