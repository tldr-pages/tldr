# brew update

> Recupera la versione più recente di Homebrew e tutte le formule da GitHub usando `git` ed esegue eventuali migrazioni necessarie.
> Per aggiornare tutte le formule installate, usa `brew upgrade`.
> Maggiori informazioni: <https://docs.brew.sh/Manpage#update-up-options>.

- Recupera la versione più recente di Homebrew e tutte le formule:

`brew {{[up|update]}}`
