# aws history

> Stampa la cronologia dei comandi per i comandi AWS CLI (la registrazione della cronologia dei comandi AWS CLI deve essere abilitata).
> Maggiori informazioni: <https://docs.aws.amazon.com/cli/latest/reference/history/>.

- Elenca la cronologia dei comandi con ID comando:

`aws history list`

- Visualizza gli eventi relativi a un comando specifico dato un ID comando:

`aws history show {{id_comando}}`
