# brew uninstall

> Disinstalla una formula o un cask Homebrew.
> Usa `brew autoremove` per rimuovere le dipendenze inutilizzate in seguito.
> Maggiori informazioni: <https://docs.brew.sh/Manpage#uninstall-remove-rm-options-installed_formulainstalled_cask->.

- Disinstalla una formula/cask:

`brew {{[rm|uninstall]}} {{formula|cask}}`

- Disinstalla un cask e rimuovi tutti i file associati:

`brew {{[rm|uninstall]}} --zap {{cask}}`
