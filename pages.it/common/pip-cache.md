# pip cache

> Ispeziona e gestisci la cache wheel di pip.
> Maggiori informazioni: <https://pip.pypa.io/en/stable/cli/pip_cache/>.

- Mostra la posizione della directory della cache di pip:

`pip cache dir`

- Elenca i nomi dei file di tutti i pacchetti attualmente memorizzati nella cache:

`pip cache list`

- Rimuovi tutti i file dalla cache di pip:

`pip cache purge`

- Rimuovi i file memorizzati nella cache che corrispondono a uno specifico nome di pacchetto:

`pip cache remove {{package_name}}`
