# aws help

> Visualizza informazioni di aiuto su AWS CLI.
> Maggiori informazioni: <https://docs.aws.amazon.com/cli/latest/userguide/cli-usage-help.html>.

- Visualizza l'aiuto:

`aws help`

- Elenca tutti gli argomenti disponibili:

`aws help topics`

- Visualizza l'aiuto su un argomento specifico:

`aws help {{nome_argomento}}`
