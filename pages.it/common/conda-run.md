# conda run

> Esegui comando in ambiente conda.
> Maggiori informazioni: <https://docs.conda.io/projects/conda/en/latest/commands/run.html>.

- Comando ambiente attivo:

`conda run {{comando}}`

- Ambiente per nome:

`conda run {{[-n|--name]}} {{nome_ambiente}} {{comando}}`

- Ambiente per percorso:

`conda run {{[-p|--prefix]}} {{percorso/dell/env}} {{comando}}`
