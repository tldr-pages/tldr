# conda rename

> Rinomina ambiente conda esistente.
> Non base né attivo corrente.
> Maggiori informazioni: <https://docs.conda.io/projects/conda/en/latest/commands/rename.html>.

- Rinomina per nome:

`conda rename {{[-n|--name]}} {{nome_vecchio}} {{nome_nuovo}}`

- Rinomina per percorso:

`conda rename {{[-p|--prefix]}} {{percorso/env}} {{nome_nuovo}}`
