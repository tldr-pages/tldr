# 7z

> Archiviatore di file con un alto fattore di compressione.
> Maggiori informazioni: <https://manned.org/7z>.

- [A]ggiungi un file o una directory a un archivio nuovo o esistente:

`7z a {{path/to/archive.7z}} {{path/to/file_or_directory}}`

- Cripta un archivio esistente (inclusi i nomi dei file):

`7z a {{path/to/encrypted.7z}} -p{{password}} -mhe=on {{path/to/archive.7z}}`

- [x]Estrai un archivio preservando la struttura originale delle directory:

`7z x {{path/to/archive.7z}}`

- [x]Etrai un archivio in una directory specifica:

`7z x {{path/to/archive.7z}} -o{{path/to/output}}`

- [x]strai un archivio su `stdout`:

`7z x {{path/to/archive.7z}} -so`

- [A]rchivia utilizzando un tipo di archivio specifico:

`7z a -t{{7z|bzip2|gzip|lzip|tar|zip}} {{path/to/archive}} {{path/to/file_or_directory}}`

- [x]Elenca i contenuti di un archivio:

`7z l {{path/to/archive.7z}}`

- Imposta il livello di compressione (più alto significa più compressione, ma più lento):

`7z a {{path/to/archive.7z}} -mx={{0|1|3|5|7|9}} {{path/to/file_or_directory}}`
