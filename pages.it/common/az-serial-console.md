# az serial-console

> Connettiti alla console seriale di una Virtual Machine.
> Parte di `azure-cli` (noto anche come `az`).
> Maggiori informazioni: <https://learn.microsoft.com/cli/azure/serial-console>.

- Connettiti alla console seriale:

`az serial-console connect {{[-g|--resource-group]}} {{Nome_Gruppo_Risorse}} {{[-n|--name]}} {{Nome_Macchina_Virtuale}}`

- Termina la connessione:

`<Ctrl ]>`
