# conda info

> Dettagli installazione conda.
> Maggiori informazioni: <https://docs.conda.io/projects/conda/en/latest/commands/info.html>.

- Tutte le informazioni:

`conda info {{[-a|--all]}}`

- Percorso ambiente base:

`conda info --base`

- Elenca ambienti:

`conda info {{[-e|--envs]}}`

- Variabili sistema:

`conda info {{[-s|--system]}}`

- Canali con token:

`conda info --unsafe-channels`
