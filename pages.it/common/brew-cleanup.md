# brew cleanup

> Rimuovi file di blocco obsoleti e download scaduti per tutte le formule e i cask.
> Maggiori informazioni: <https://docs.brew.sh/Manpage#cleanup-options-formulacask->.

- Rimuovi file di blocco obsoleti e download scaduti per tutte le formule/cask:

`brew cleanup`

- Rimuovi file di blocco obsoleti e download scaduti per una formula/cask specifica:

`brew cleanup {{formula|cask}}`

- Mostra cosa verrebbe rimosso, ma non rimuovere nulla:

`brew cleanup {{[-n|--dry-run]}}`

- Mostra l'aiuto:

`brew cleanup {{[-h|--help]}}`
